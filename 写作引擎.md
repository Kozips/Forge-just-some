# 写作引擎

> 独立通用模块。不绑定任何风格，不绑定任何 skill。
> 这里教的是**怎么写**，不是**写成什么样**。风格选择权在你。

---

## 一、场景：把读者扔进去

### 1.1 场景三要素

每个场景必须回答三个问题：

| 问题           | 怎么写                                                            |
| -------------- | ----------------------------------------------------------------- |
| **在哪**       | 用 1 个感官细节锚定空间。不是"一个房间"，是"地板踩上去会响的房间" |
| **谁在**       | 角色的身体状态——站着/坐着/靠着什么/手在干什么                     |
| **发生了什么** | 一件事情在发生，或者一件事情即将发生                              |

**不需要描写整个场景。** 选 2-3 个细节，读者的大脑会自动补完剩下的。

### 1.2 感官排序

人接收信息的优先级：视觉 > 听觉 > 触觉 > 嗅觉 > 味觉。但**写作的优先级应该反过来**——嗅觉和触觉最能让读者"进入"场景，因为它们在日常生活中最不被注意。

- 视觉：最容易写，也最容易平庸。不要写"他看到了"，直接写看到的东西。
- 听觉：最容易被忽略。一个场景里总有什么在响。
- 触觉：温度、质感、风。最能让读者"感觉到"场景。
- 嗅觉：最强烈的记忆触发器。消毒水、雨前的土味、旧书的霉味。

### 1.3 场景切换

场景切换时，第一句话就锚定新空间。不要让读者猜"这是在哪"。

## 二、角色：让读者相信这个人是真的

### 2.1 角色 = 欲望 + 恐惧 + 矛盾

- **欲望**：他想要什么？（一句话）
- **恐惧**：他怕什么？（一句话）
- **矛盾**：他的欲望和恐惧在打架吗？怎么打？

一个角色不需要复杂。一个矛盾就够了。怕死但想活、想爱但不敢信、想赢但不屑于努力。

### 2.2 角色怎么说话

不同的人，同一件事说法完全不同。差距来自：

| 维度      | 例子                                           |
| --------- | ---------------------------------------------- |
| 词汇量    | "我认为这不合理" vs "这他妈什么玩意儿"         |
| 句长      | 有人一句话三个字，有人一句话三行               |
| 直接/迂回 | "你骗我" vs "你上次说的那个事，后来我想了想……" |
| 情绪表达  | 有人越生气话越少，有人越生气话越多             |

**检验标准**：遮住角色名，能分辨谁在说话。不能→拉开词汇量、句长、直接度的差距。

### 2.3 角色怎么行动

角色的行动不是"做正确的事"，是"做他会做的事"。一个怕死的人不会突然勇敢——除非你铺垫了足够的原因。

**每个行动背后有动机。** 不需要写出来，但你（作者）要知道。

---

## 三、信息：什么时候给，怎么给

### 3.1 信息释放三原则

1. **先让读者好奇，再给答案。** 先展示"有什么不对劲"，再解释为什么。
2. **一次只给一个信息。** 不要在一个段落里塞世界观+角色背景+剧情推进。
3. **用角色过滤信息。** 读者通过角色的眼睛看世界。角色不知道的，读者也不知道。

### 3.2 怎么给信息

| 方式       | 效果               | 例子                             |
| ---------- | ------------------ | -------------------------------- |
| 角色发现   | 读者和角色同步     | 他打开抽屉，里面有一封信。       |
| 角色被告知 | 读者和角色一起知道 | "你父亲不是意外死的。"           |
| 读者先知道 | 制造紧张           | 读者看到柜子里有人，但角色不知道 |
| 暗示       | 读者自己推断       | 她从不坐靠窗的位置。             |

### 3.3 留白

留白的前提：读者已经拥有足够的信息去推断。给出具体的行为线索，不说结论。

- ✅ 好的留白："她从不坐靠窗的位置。后来崩溃那晚才说出七岁时的车祸。"
- ❌ 坏的留白："她有很深的创伤。"（什么都没给）

---

## 四、节奏：快慢是选择，不是默认

### 4.1 控制节奏的三个杠杆

| 杠杆     | 加快                     | 放慢                             |
| -------- | ------------------------ | -------------------------------- |
| 句长     | 短句。碎片。             | 长句。从句。                     |
| 段落     | 单行段。甚至一个词一行。 | 完整段落。描写+动作+内心。       |
| 信息密度 | 一个动作接一个动作。     | 动作之间有停顿、描写、内心活动。 |

### 4.2 节奏模式

- **紧张场景**：短句 → 短段 → 动作密集 → 几乎无内心独白
- **情感场景**：长句 → 描写多 → 内心独白 → 时间放慢
- **过渡场景**：中等句长 → 交代信息 → 推进剧情

### 4.3 节奏变化

**没有节奏变化 = 没有节奏。** 哪怕是最紧张的动作戏，也要有 1-2 个停顿。最安静的情感戏，也要有 1-2 个加速。变化本身才是节奏。

---

## 五、情感：让读者感受到，不是被告知

### 5.1 情感传达的三个层次

| 层次             | 方式           | 例子                             |
| ---------------- | -------------- | -------------------------------- |
| 第一层：身体反应 | 写身体在做什么 | 手在抖。嗓子发紧。眼眶烫。       |
| 第二层：行为     | 写角色在做什么 | 把杯子转了又转。走到门口又回来。 |
| 第三层：环境     | 写环境映射情绪 | 雨打在窗上。灯管嗡嗡响。         |

**第一层最直接，第二层最耐读，第三层最高级。** 三层不互斥，可以同时用。

### 5.2 不同情感的处理方式

- **悲伤**：不要写哭。写哭之前的克制——嗓子发紧、眼眶发烫、转过身去。哭声是释放，克制才是情感。
- **愤怒**：不要写"他很愤怒"。写他做了什么——攥拳、咬牙、沉默。或者相反：他笑了。
- **恐惧**：不要写"他很害怕"。写他注意到的细节——声音、影子、温度变化。恐惧让人过度关注环境。
- **喜悦**：不要写"他很开心"。写他忘了什么——忘了时间、忘了场合、忘了自己本来在干什么。

### 5.3 情感强度

**情感的强度不在于写得多重，在于反差。** 一个从不哭的人眼眶红了，比一个爱哭的人嚎啕大哭更有力。一个温和的人沉默了三秒，比一个暴躁的人摔东西更可怕。

## 六、对话：每句话都在做事

### 6.1 对话的功能

每句对话至少做三件事之一：

1. **推动剧情**：这句话让事情往前走了一步
2. **揭示角色**：这句话让你更了解说话的人
3. **埋下信息**：这句话后面回头看会有新意思

### 6.2 对话的结构

```
对话 → 反应（动作/沉默/内心） → 下一句对话
```

不要连续多句纯对话。对话之间的空隙里，角色在呼吸、在思考、在犹豫、在观察。这些空隙比对话本身更有信息量。

### 6.3 潜台词

角色说出来的话和真正想说的话之间的差距，就是潜台词。

- 说"我没事" → 可能是有事但不想说，也可能是真的没事但对方不信
- 说"你走吧" → 可能是真的想让你走，也可能是试探你会不会留
- 说"随便" → 可能是不在乎，也可能是太在乎但不敢说

**潜台词不需要写出来。** 读者自己能感受到。写出来就毁了。

---

## 七、结尾：每一段结尾都是一扇门

### 7.1 结尾的功能

一段/一章的结尾只有一个任务：**让读者想继续读。**

| 方式     | 效果                                                             |
| -------- | ---------------------------------------------------------------- |
| 悬念     | 读者想知道答案                                                   |
| 反转     | 读者想重新理解前面的内容                                         |
| 情感悬念 | 读者想知道角色接下来会怎样                                       |
| 画面     | 一个画面留在读者脑子里，不回答任何问题，但让读者想留在这个世界里 |

### 7.2 结尾不要做的事

- 不要总结（"这就是他的人生"）
- 不要升华（"这一刻，他终于明白了……"）
- 不要预告（"他不知道的是，这将改变一切"）
- 不要关闭（"从此以后，他们过上了幸福的生活"）

## 写后自检

写完一段/一章后，逐条过：

1. **场景**：读者能感受到空间吗？（至少 1 个非视觉感官细节）
2. **角色**：每个行动背后有动机吗？遮住名字能分辨谁是谁吗？
3. **信息**：读者知道的和角色知道的一致吗？有没有一次塞太多信息？
4. **节奏**：有变化吗？全是快或全是慢 = 没有节奏。
5. **情感**：读者感受到的，是你想让他们感受到的吗？还是你只是写了"他很悲伤"？
6. **对话**：每句话都在做事吗？能删掉任何一句而不影响剧情/角色/信息吗？
7. **结尾**：读者想翻下一页吗？还是这里可以停下来？
