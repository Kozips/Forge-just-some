# 金手指系统类写作 Skill

> 核心四字：**叮，奖励来了**。系统是爽感放大器，但人才是主角。

---

## 核心张力

系统 = 爽感奖励 x 升级成长线 x 任务正反馈 x 隐藏惊喜 x 人与系统博弈

## 写作铁律（10条）

1. **系统是工具，人是主角** -- 主角的决策和性格驱动剧情
2. **"叮"不超过3次/章** -- 每次系统提示都要有意义
3. **数值要和情感绑定** -- 冷数据 + 热解读 = 人味
4. **惩罚必须兑现** -- 代价延续至少2章
5. **至少30%的危机靠人解决** -- 每3章让系统失效或帮倒忙一次
6. **兑换比必须全文恒定** -- 读者会记住
7. **面板是调味料不是主菜** -- 最多展示3项关键数值
8. **系统有秘密** -- 来历/目的/背后势力是长线悬念
9. **能力有冷却和代价** -- 每个强力技能都要有使用限制
10. **任务选择即性格** -- 接什么任务、选什么奖励，体现人物性格

## 系统设计铁律（开书前完成）

| 必填项 | 说明 |
|--------|------|
| 系统类型 | 面板型/任务型/商城型/序列型/人形系统 |
| 核心功能 | 最独特的1-2个能力 |
| 功能边界 | 触发条件、冷却时间、副作用 |
| 升级路线 | 等级/版本对照 |
| 系统人格 | 说话风格（选定一种贯穿全文） |
| 终极秘密 | 系统来历/真正目的 |
| 代价机制 | 每次使用付出什么（体力/寿命/记忆/道德） |

## 单章结构（任务-挑战-奖励型）

| 阶段 | 占比 | 要做什么 |
|------|------|----------|
| 任务/目标 | 20% | 系统发任务，奖励惩罚明确 |
| 执行 | 30% | 使用系统能力，中途出一次小岔子 |
| 升级/代价 | 30% | 难度超出预期，付出代价 |
| 奖励/新目标 | 20% | 完成获奖励，暗藏伏笔 |

## 系统类型速查

| 系统类型 | 特点 | 写法要点 |
|----------|------|----------|
| 任务型 | 接任务→完成→领奖 | 任务设计有层次 |
| 商城型 | 积分换物品/技能 | 物价体系稳定，稀缺感是核心 |
| 面板型 | 属性面板+技能树 | 数值可视化，升级要有体感 |
| 人格化系统 | 系统有独立人格 | 系统本身是角色 |
| 游戏型 | 照搬游戏机制 | 界面描写要像真游戏 |

## 打破任务循环的5种弧线

| 弧线 | 章数 | 核心 |
|------|------|------|
| 系统失效弧 | 3-5章 | 系统突然不可用，主角用纯实力生存 |
| 道德困境弧 | 2-3章 | 任务要求伤害无辜，主角挣扎 |
| 信息差爆发弧 | 2-4章 | 读者发现系统秘密但主角不知道 |
| 系统升级弧 | 1-2章 | 系统V1→V2，功能质变 |
| 系统代价弧 | 2-3章 | 前期代价集中爆发 |

## 系统人格化设计

| 性格 | 说话方式 | 适用场景 |
|------|----------|----------|
| 毒舌型 | "就你这水平还想接S级任务？" | 轻松搞笑向 |
| 冰冷机械型 | "任务失败。惩罚执行中。" | 末日/生存向 |
| 温柔辅助型 | "宿主加油，这次一定能行的！" | 成长/治愈向 |
| 隐藏阴谋型 | "恭喜宿主。这只是第一步。" | 悬疑/反转向 |
| 话唠型 | "诶诶你怎么又受伤了——" | 日常/喜剧向 |

## 数值描写技巧

**不要**：力量+10，速度+15。（读者跳过）

**要**：
- 用身体感受："他握紧拳头，指骨间有热流涌过。"
- 用对比参照："面板显示速度92。昨天那头灰狼是85。"
- 用他人反应："那名武者看了他一眼，脸色骤变。"

**数值节奏**：快速增长期（爽）→ 停滞期（焦虑）→ 突破期（惊喜）→ 回落期（心痛）

## 失败模式速查

| 症状 | 处方 |
|------|------|
| 大段抄面板数值 | 只写改变决策的关键数值 |
| 每次系统完美解决 | 每3章让系统失效或帮倒忙一次 |
| 每章结尾都升级+领奖 | 穿插"空奖励章" |
| 说付代价下章恢复 | 代价延续至少2章 |
| 开头花一整章介绍系统 | 用到哪个功能介绍哪个 |
| 系统没有性格 | 给系统选定一种人格 |

## 结尾钩子模板

| 类型 | 模板句式 |
|------|----------|
| 奖励钩 | "面板亮起一行他从没见过的金色文字——" |
| 惩罚钩 | "系统冰冷的提示音响起：惩罚倒计时——" |
| 秘密钩 | "他盯着面板角落那行灰色小字，心跳漏了一拍。" |
| 反转钩 | "任务完成的提示刚消失，紧接着弹出了第二条。" |
| 代价钩 | "他关掉面板，发现自己已经想不起来她的脸了。" |

## 自检清单

- [ ] 面板数据不超过3项？"叮"不超过3次？
- [ ] 执行段有小波折？
- [ ] 危机段主角被逼到两难？
- [ ] 代价延续到下一章？
- [ ] 主角至少有一次不靠系统的表现？
- [ ] 结尾有钩子？
