# 系统流写作 Skill

> 核心四字：**叮，奖励来了**。系统是爽感放大器，但人才是主角。
> 读者跟主角一起看面板、算收益、等"叮"的一刻。"叮"太多会麻木，太少会无聊。

---

## 一句话定义

主角获得系统（面板/任务/商城/升级），用系统能力在异世界/末日/都市中升级打怪，系统提供任务、奖励、升级路径，主角负责决策和执行。

## 核心爽点

1. **数值反馈** -- 面板跳数字，属性上涨，读者跟着爽
2. **任务奖励** -- 完成任务获得宝物/技能/隐藏奖励，开盲盒感
3. **系统吐槽/人格化** -- 系统有性格，会嘲讽、毒舌、卖萌，互动本身是乐趣
4. **面板碾压** -- 查看敌人数据发现自己全面碾压，还没打就知道赢了
5. **隐藏成就** -- 触发隐藏任务/彩蛋奖励，读者也想知道触发条件
6. **选择困境** -- 多个任务/奖励只能选一个，读者跟着纠结

## 经典套路结构（六步走）

| 阶段 | 节奏 | 核心动作 |
|------|------|----------|
| 获得系统 | 快 | 穿越/意外/濒死触发，系统绑定，初始界面展示 |
| 新手任务 | 中 | 第一个简单任务，试水系统功能，获得第一桶金 |
| 小试牛刀 | 中 | 用系统能力解决实际问题，旁人震惊，主角装逼 |
| 系统升级 | 快 | 系统版本更新，解锁新功能，商城扩容，出现新货币 |
| 危机与代价 | 慢 | 系统不是万能的，任务失败有惩罚，副作用显现 |
| 更大舞台 | 渐快 | 发现系统背后更大的秘密/更高维度的存在，格局扩大 |

## 必备元素

- **系统面板**：属性、等级、技能、背包，界面描述要简洁一致
- **任务体系**：主线/支线/隐藏/限时，每种任务的奖励和惩罚不同
- **升级反馈**：每次升级有明确的数值变化和身体/精神变化描写
- **商城/兑换**：明码标价，让读者跟着算账，偶尔来个"买不起但好想要"
- **惩罚机制**：任务失败/超时有代价，不能白嫖
- **系统人格**：哪怕只是偶尔冒出一句冷嘲，也能让系统活起来
- **限制条件**：能力有冷却/消耗/副作用，不是无限使用

## 常见失败模式

| 失败模式 | 问题 | 修正方案 |
|----------|------|----------|
| 面板报账员 | 全章都在读数据 | 只展示改变决策的关键数值，其他一笔带过 |
| 万能金手指 | 系统什么都能解决 | 每3章让系统失效或帮倒忙一次 |
| 奖励疲劳 | 每章都"叮" | 穿插"空奖励章"，让读者等一等 |
| 数值通货膨胀 | 属性动辄百万 | 控制数值增长曲线，用百分比而非绝对值 |
| 系统话太多 | 系统解释占半章 | 用到哪个功能介绍哪个，禁止一次性倾倒 |
| 人被系统淹没 | 主角没有主见 | 系统给选项，主角做选择，选择体现性格 |
| 任务千篇一律 | 永远是"击杀XX" | 混入社交/探索/解谜类任务 |

| 作品 | 学什么 | 怎么学 |
|------|--------|--------|
| 《我有一座恐怖屋》 | 系统与恐怖氛围的结合 | 系统不是解药而是导航，真正吓人的是场景本身 |

## 写作铁律（10条）

1. **系统是工具，人是主角** -- 主角的决策和性格驱动剧情，系统只是放大器
2. **"叮"不超过3次/章** -- 每次系统提示都要有意义，无意义的提示是噪音
3. **数值要可视化** -- 用对比/参照物让读者感受到数值变化，不是列数字表
4. **惩罚必须兑现** -- 说了任务失败要扣血，就必须真的扣，不能嘴上说说
5. **面板展示有节奏** -- 战斗中不展示面板，安全时才看数据
6. **商城不能无限刷** -- 限制购买次数/刷新时间/货币获取，制造稀缺感
7. **系统有秘密** -- 系统的来历/目的/背后势力是长线悬念，不能第一章就交代完
8. **能力有冷却和代价** -- 每个强力技能都要有使用限制，否则无张力
9. **任务选择即性格** -- 主角接什么任务、选什么奖励，要体现人物性格
10. **系统升级要配合剧情** -- 不能为了升级而升级，每次系统变化都推动新剧情

## 对话风格速查

| 场景 | 系统语气示例 | 主角反应示例 |
|------|-------------|-------------|
| 发布任务 | "叮——支线任务已发布。奖励：未知。惩罚：死。祝你好运。" | "这个'祝你好运'怎么听起来像诅咒？" |
| 完成任务 | "恭喜宿主完成任务，奖励已发放。友情提示：你只剩下半条命了。" | "你就不能说点好听的？" |
| 商城浏览 | "积分不足。建议宿主努力搬砖。" | "……你是资本家吗？" |
| 隐藏成就 | "？？？——恭喜宿主触发隐藏成就。具体信息暂不公开。" | "喂！说清楚啊！" |
| 系统升级 | "系统升级中……预计用时：宿主有生之年。开个玩笑，三分钟。" | "你是不是越来越欠了？" |

## 自检清单

- [ ] 面板数据不超过3项被展示？"叮"不超过3次？
- [ ] 系统能力有明确的限制/冷却/代价？
- [ ] 主角的选择体现性格而非纯功利计算？
- [ ] 惩罚机制真的被兑现过（至少一次）？
- [ ] 系统不是万能的，有帮倒忙/失效的时候？
- [ ] 升级反馈有感官描写（不只列数字）？
- [ ] 章尾钩子不是简单的"叮——新任务发布"？
- [ ] 系统背后有未解之谜保持长线悬念？
- [ ] 任务类型有变化，不是永远"击杀XX"？
- [ ] 读者能感受到数值变化的实际意义？

## 系统类型速查

写之前先定好系统类型，不同类型写法完全不同：

| 系统类型 | 特点 | 代表作 | 写法要点 |
|----------|------|--------|----------|
| 任务型 | 接任务→完成→领奖 | 全球高武 | 任务设计要有层次，不能全是打怪 |
| 商城型 | 积分换物品/技能 | 斗破苍穹（部分） | 物价体系要稳定，稀缺感是核心 |
| 面板型 | 属性面板+技能树 | 大奉打更人 | 数值可视化，升级要有体感 |
| 直播型 | 观众互动+打赏 | 我有一座恐怖屋（部分） | 观众反应是第二条叙事线 |
| 人格化系统 | 系统有独立人格 | 从红月开始 | 系统本身是角色，有成长和秘密 |
| 游戏型 | 完全照搬游戏机制 | 网游之近战法师 | 界面描写要像真游戏，术语要统一 |

## 升级节奏参考

| 卷/阶段 | 系统等级 | 主角实力 | 舞台大小 | 核心矛盾 |
|---------|---------|---------|---------|---------|
| 第1卷 | Lv1-10 | 普通人→超人 | 一座城市 | 生存+认识系统 |
| 第2卷 | Lv10-30 | 城市级 | 一个国家 | 势力冲突+系统升级 |
| 第3卷 | Lv30-60 | 国家级 | 一个大陆 | 大势力博弈+系统秘密初露 |
| 第4卷 | Lv60-90 | 大陆级 | 多个世界 | 跨世界危机+系统真相 |
| 终卷 | Lv90+ | 超越系统 | 系统之上 | 系统本身是敌人/工具/命运 |

## 数值描写技巧

**不要这样写**：力量+10，速度+15，防御+20。面板数据：力量87，速度92，防御76。

**要这样写**：
- 用身体感受："他握紧拳头，感觉指骨间有一股热流涌过。上次挥拳能打碎木板，这次——他一拳砸在墙上，整面墙震了一下。"
- 用对比参照："面板上显示他的速度是92。他记得昨天那头灰狼的速度是85。所以理论上，他现在能跑赢狼。"
- 用他人反应："那名武者看了他一眼，脸色骤变。'你……你什么时候突破的？'"

## 系统人格化设计速查

| 系统性格 | 说话方式 | 与主角的关系 | 适用场景 |
|----------|---------|-------------|---------|
| 毒舌型 | "就你这水平还想接S级任务？笑死。" | 互怼搭档 | 轻松搞笑向 |
| 冰冷机械型 | "任务失败。惩罚执行中。" | 工具/枷锁 | 末日/生存向 |
| 温柔辅助型 | "宿主加油，这次一定能行的！" | 伙伴/朋友 | 成长向/治愈向 |
| 隐藏阴谋型 | "恭喜宿主。这只是第一步。" | 表面助手实则另有目的 | 悬疑/反转向 |
| 话唠型 | "诶诶诶你怎么又受伤了我不是说了要小心吗——" | 啰嗦管家 | 日常/喜剧向 |

## 角色速查（写作时直接参考）

> 完整版见 `人物/references/人物联动写作/金手指系统类.md`

### 江寒（隐忍 / 护妹 / 兽鼎蓄力）
- 语言：面对强权不开口辩解，"紧紧抿着嘴，几乎咬碎了牙齿"
- 语气：低沉压抑，每个字像从牙缝里挤出来
- 动作：强忍怒火时站定不动全身肌肉紧绷；战斗时眸子陡然睁开银色光辉闪耀
- 用法：金手指配合隐忍性格，被嘲笑铺垫越长亮出实力越爽

### 姜玄（少年老成 / 隐藏实力）
- 语言：面对羞辱不回嘴只在心里分析局势，自言自语时语速快条理清晰
- 语气：平时低沉不引人注意，爆发时刀势暴涨；对长辈恭敬诚恳
- 动作：被称为废物时眼皮跳了跳（唯一的应激反应）；拖刀式起步先慢后快
- 用法：金手指文的"计算"比"力量"更有趣，越废物的外表越反衬实力

### 萧云（乐观莽撞 / 顿悟系统）
- 语言："修炼难度大？没有关系，我有顿悟帮助，什么功法不能修炼？"
- 语气：激动时咧嘴一笑，面对犹豫时主动表态不等人劝
- 动作：十六年苦等系统完成后激动流泪；在众人争论时主动开口做决定
- 用法：系统跟性格绑定，有系统但还是要自己挤进人群——金手指是助力性格才是驱动
