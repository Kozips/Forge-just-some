# 赘婿写作 Skill

> 核心四字：**隐忍打脸**。被人踩得越低，翻盘时越爽。
> 爽感公式：受辱时间 x 打脸力度 = 读者爽感。受辱不够久没爽感，打脸不够狠没高潮。

---

## 一句话定义

主角入赘女方家庭，被看不起、羞辱、欺压，但隐藏着真实实力/身份/背景，逐步翻身打脸，最终身份暴露，所有人跪。

## 核心爽点

1. **打脸的爽感** -- 嘲笑我的人被打脸，观众越多越爽
2. **隐藏身份的反转** -- "你们知道他是谁吗？"一句话全场变脸
3. **废物到大佬的逆袭** -- 从刷马桶到坐主位，落差越大越爽
4. **妻子态度转变** -- 从看不起到震惊到崇拜到依赖，情感线是粘合剂
5. **岳家集体打脸** -- 不是一个人跪，是一群人跪，从鄙视到恐惧
6. **第三方认证** -- 大人物出场对赘婿毕恭毕敬，旁人一脸问号

## 经典套路结构（八步走）

| 阶段 | 节奏 | 核心动作 |
|------|------|----------|
| 赘婿之辱 | 慢 | 入赘背景交代，日常被岳家欺负/羞辱/使唤，读者积攒怒气值 |
| 小露锋芒 | 中 | 某个小场景展示能力，但被当成运气/巧合 |
| 妻子动摇 | 中 | 妻子开始注意到赘婿的不同，态度有细微变化 |
| 关键打脸 | 快 | 一个正式场合，赘婿出手解决大问题，围观者震惊 |
| 更大打脸 | 快 | 更高层面的问题被解决，之前嘲笑的人开始心虚 |
| 身份渐露 | 中 | 外部势力/大人物来拜访，称呼暴露端倪，岳家试探 |
| 大清算 | 快 | 所有过往的侮辱被一一清算，该跪的一个不少 |
| 身份大白 | 快 | 真实身份彻底曝光，全场下跪，妻子含泪，格局升华 |

## 必备元素

- **侮辱清单**：前期受的屈辱要具体、要有记忆点，后期要逐一清算
- **隐藏理由**：为什么这么强却当赘婿？必须有合理解释
- **妻子定位**：不能是纯工具人，要有自己的事业/困境/成长
- **打脸阶梯**：从小人物到大人物到终极boss，打脸对象要层层递进
- **旁观者视角**：围观群众的震惊反应是爽感放大器
- **红颜/兄弟**：至少一个知道主角真实身份的人，做信息桥梁
- **岳家分化**：岳家不是铁板一块，有人欺负有人同情

## 常见失败模式

| 失败模式 | 问题 | 修正方案 |
|----------|------|----------|
| 受辱太轻 | 随便被说了两句 | 受辱要具体到动作和场景，让读者有画面感 |
| 打脸太频繁 | 每章都打脸 | 3-5章蓄力一次打脸，蓄力越久爽感越强 |
| 身份暴露太早 | 第10章就知道是大佬 | 拖到至少中后期，用"差点暴露"制造张力 |
| 妻子无脑 | 妻子只会崇拜 | 妻子有自己的判断和成长，偶尔跟主角有冲突 |
| 打脸方式单一 | 永远是"叫人来" | 换着花样：经济碾压/武力碾压/人脉碾压/智商碾压 |
| 岳家全恶 | 没有一个正常人 | 给岳家1-2个善良的成员，增加复杂度 |
| 侮辱重复 | 每次都是"你不配" | 每个阶段的侮辱要升级，从语言到行动到威胁 |

## 写作铁律（10条）

1. **侮辱要具体** -- "你这个废物"不够，要具体到场景和动作（当众泼酒/扔拖把/赶出包间）
2. **打脸要对标** -- 当初怎么侮辱的，现在怎么还回来，一一对应才爽
3. **蓄力期不能短** -- 至少3-5章的压抑才换一次打脸，压缩蓄力就是压缩爽感
4. **隐藏理由要合理** -- 隐藏身份不能是因为"想低调"，必须有真正的理由
5. **妻子不能是花瓶** -- 她要有自己的困境、选择和成长线
6. **旁观者是爽感放大器** -- 围观群众的震惊、议论、后悔是打脸的核心组成部分
7. **身份暴露要层层剥** -- 不能一次性全揭露，先露出一角→更多人知道→最终大白
8. **打脸对象要升级** -- 从亲戚到商界到政界到更高级别，不能一直在同一个圈子里打
9. **清算是长线** -- 每个侮辱过主角的人都要有"还债"的时刻，或至少被提及
10. **格局要升华** -- 不能永远在家长里短里打转，最终要从家庭矛盾升到更大的格局

## 对话风格速查

| 场景 | 对方台词示例 | 主角反应示例 |
|------|-------------|-------------|
| 日常羞辱 | "你就是个吃软饭的废物，有什么资格坐这？" | 沉默，低头，不说话。（但内心：你还剩三年。） |
| 小露锋芒 | "这次算你运气好，下次没这么走运了。" | "也许吧。"（转身走，不争辩） |
| 关键打脸 | "你怎么可能……不，这不可能！" | "有什么不可能的？"（淡淡一句） |
| 身份渐露 | "林先生，您怎么在这？"（大人物毕恭毕敬） | "我住这。"（旁人全懵） |
| 大清算 | "女婿……不，林总，之前是我们有眼无珠……" | "三年前你说的那句话，我还记得。" |
| 最终翻盘 | （岳父跪下）"是我们……瞎了眼。" | 不说话，转身离开。妻子追出去。 |

## 自检清单

- [ ] 本章的侮辱/打脸在整体节奏中的位置清晰（蓄力还是释放）？
- [ ] 侮辱足够具体（有画面感、有旁观者反应）？
- [ ] 打脸方式跟之前的侮辱形成对标？
- [ ] 主角的隐忍有真实的情绪（不是永远面瘫）？
- [ ] 妻子在本章有独立的行动或选择？
- [ ] 身份暴露的风险或暗示在推进？
- [ ] 打脸对象的级别在递进（不是重复同一个对手）？
- [ ] 旁观者/围观群众的反应被写出来？
- [ ] 有新的人物或势力被引入（扩展打脸舞台）？
- [ ] 章尾有钩子让读者期待下一次打脸？

## 赘婿类型速查

写之前先定好赘婿类型，不同类型隐藏身份不同：

| 赘婿类型 | 隐藏身份 | 适合题材 | 爽点方向 |
|----------|---------|---------|---------|
| 都市兵王 | 退役特种兵/战神 | 都市 | 武力碾压+军方背景 |
| 商业大佬 | 隐世财团继承人 | 都市 | 经济碾压+商战 |
| 修仙强者 | 修仙界大能转世/封印 | 玄幻 | 修仙碾压+降维打击 |
| 古代王侯 | 失势皇族/隐居将军 | 古代 | 权谋+武力双碾压 |
| 科技天才 | 隐姓埋名的科技大佬 | 都市 | 技术碾压+智商碾压 |
| 地下王者 | 黑道/雇佣兵首领 | 都市 | 暗势力碾压+反差 |

## 受辱场景库（蓄力用）

以下是可以直接使用的受辱场景，每个场景对应一个清算时刻：

| 受辱场景 | 程度 | 典型对话 | 清算方式 |
|----------|------|---------|---------|
| 当众泼酒 | 轻 | "你配喝这酒吗？" | 后期请对方喝更贵的，对方不敢接 |
| 赶出家门 | 中 | "滚出去，别丢人现眼。" | 后期岳家求着主角回来 |
| 妻子面前羞辱 | 重 | "你老婆嫁给你真是瞎了眼。" | 后期妻子公开表态站在主角一边 |
| 动手打人 | 重 | 一巴掌扇过来，主角不还手 | 后期对方跪着道歉 |
| 威胁家人 | 极重 | "你要是不听话，你妈就别想治病了。" | 后期对方的产业被主角收购 |
| 婚礼羞辱 | 极重 | "入赘就是我们家的一条狗。" | 后期婚礼/寿宴上主角反客为主 |

## 打脸方式速查

打脸方式要多样化，不能每次都叫人来：

| 打脸方式 | 适用阶段 | 爽感等级 | 写法要点 |
|----------|---------|---------|---------|
| 经济碾压 | 前期 | 中 | "这张卡里有多少钱，你们自己查。" |
| 人脉碾压 | 前中期 | 高 | 大人物当众对主角毕恭毕敬 |
| 武力碾压 | 全程 | 中 | 不建议过多使用，容易审美疲劳 |
| 智商碾压 | 全程 | 高 | 用商业/谈判/布局让对方输得心服口服 |
| 身份碾压 | 中后期 | 极高 | 名片/电话/来人，身份一层层揭开 |
| 家族碾压 | 后期 | 极高 | 主角背后的家族/势力全面碾压岳家 |
| 格局碾压 | 终局 | 最高 | 主角关心的是天下大事，岳家只是蝼蚁 |

## 妻子角色弧线设计

妻子是赘婿文最重要的配角，她的态度变化是情感线核心：

| 阶段 | 态度 | 典型行为 | 写法要点 |
|------|------|---------|---------|
| 初期 | 冷淡/无奈 | 不主动说话，各过各的 | 不要写成厌恶，而是"无所谓" |
| 小转折 | 好奇 | 开始注意主角的反常行为 | 一次意外的关心就够了 |
| 动摇期 | 困惑 | "他到底是什么人？" | 妻子开始主动了解主角 |
| 确认期 | 震惊 | 发现真相的冰山一角 | 不是一下子全信，而是半信半疑 |
| 依赖期 | 信任 | 遇事第一时间找主角商量 | 妻子要有自己的成长，不能变成附属 |
| 深情期 | 不离不弃 | 知道全部真相后选择站在主角身边 | 理由要充分，不是因为"有钱了" |

## 岳家成员分化设计

岳家不能全恶，需要有层次：

| 角色类型 | 对主角态度 | 作用 | 结局 |
|----------|-----------|------|------|
| 核心反派（岳父/大哥） | 极度鄙视 | 制造主要冲突 | 要么跪要么出局 |
| 势利眼亲戚 | 看不起+使唤 | 日常受辱来源 | 悔不当初 |
| 中立派（岳母/二姐） | 不关心 | 反映普通人的态度 | 后来庆幸没得罪 |
| 同情派（小姨子/岳弟） | 偶尔帮忙 | 给主角喘息空间 | 后来成为盟友 |
| 关键转折人 | 识货 | 第一个发现主角不简单 | 成为内应/桥梁 |

## 开篇模板速查

以下是赘婿文最有效的三种开篇方式：

**模板一：受辱开篇（最经典）**
> 岳父把拖把扔到他脚边："把马桶刷了。"他弯腰捡起拖把，三年来第一次——嘴角微微上扬。

**模板二：回忆开篇（增加深度）**
> 他站在镜子前，看着镜中那张陌生的脸。三年前，这张脸的主人在雨夜里倒下。现在，他替他活着。

**模板三：第三方视角开篇（制造悬念）**
> 所有人都知道林家的赘婿是个废物。除了每天凌晨三点来接他的那辆黑色轿车。
