# 重生写作 Skill

> 核心四字：**信息差碾压**。读者跟主角一起"我早就知道"。重生不是开挂，是降维打击。
> 爽感来源：别人不知道的事我知道，别人做不到的事我提前布局。

---

## 一句话定义

主角死亡/失败后回到过去（几年到几十年不等），用前世记忆改变命运，弥补遗憾，碾压曾经的敌人，走上人生巅峰。

## 核心爽点

1. **信息差碾压** -- 别人还在摸黑，主角已经知道结局，抢占先机
2. **提前布局** -- 提前投资/避险/结交关键人物，几年后收益暴涨
3. **弥补遗憾** -- 前世失去的人这次要保护，前世犯的错这次不再犯
4. **蝴蝶效应** -- 主角的每个选择都改变时间线，不可预测性是张力
5. **打脸前世仇人** -- 前世欺负我的人，这辈子你没机会了
6. **先知式预言** -- 随口说出未来的事，旁人不信，事后应验

## 经典套路结构（七步走）

| 阶段 | 节奏 | 核心动作 |
|------|------|----------|
| 死亡与重生 | 快 | 前世惨死/遗憾而终，意识回到过去某个关键时间点 |
| 确认时间线 | 快 | 震惊→确认年份→回忆前世关键事件→制定计划 |
| 小试牛刀 | 中 | 利用已知信息获得第一桶金/避开第一次危机，旁人惊讶 |
| 保护关键人 | 中 | 找到前世失去的人，改变TA的命运，情感线启动 |
| 布局与积累 | 慢 | 大量伏笔播种，投资/人脉/实力同步积累，外人看不懂 |
| 改变关键节点 | 快 | 前世的转折点来了，这次主角主导局面，蝴蝶效应爆发 |
| 碾压前世敌人 | 快 | 仇人按原计划行动，发现全盘落空，主角早已棋高一着 |

## 必备元素

- **前世记忆清单**：主角记得哪些关键事件、时间节点、人物关系，要有取舍
- **蝴蝶效应规则**：主角的改变会导致未来偏移，记忆不能100%准确
- **遗憾与执念**：前世最大的遗憾是情感驱动力，不能只是"赚更多钱"
- **时间紧迫感**：有些事件有窗口期，错过就没了
- **身份矛盾**：主角拥有成年人的灵魂但困在年轻的身体/身份里
- **信息管理**：主角不能无限透露未来信息，会被当成疯子/威胁
- **宿敌体系**：前世的仇人/对手要足够强大，不能一捏就碎

## 常见失败模式

| 失败模式 | 问题 | 修正方案 |
|----------|------|----------|
| 全知全能 | 主角从不出错 | 蝴蝶效应让部分记忆失效，3次布局至少失败1次 |
| 无脑赚钱 | 重生只为炒股炒房 | 赚钱只是手段，核心冲突是人际关系和命运抗争 |
| 金手指无代价 | 重生记忆白嫖 | 记忆随时间模糊/细节出错/关键事件被蝴蝶效应改变 |
| 仇人太弱 | 一出手就碾压 | 仇人也要成长，而且仇人这次也可能因为蝴蝶效应变强 |
| 道德真空 | 用未来信息无限碾压别人 | 让主角面对道德困境：利用预知赚钱算不算欺骗？ |
| 节奏失控 | 布局太长读者等不及 | 每3-5章给一次"小兑现"，让读者看到布局的回报 |
| 情感线工具化 | 女主只是"弥补遗憾"的道具 | 女主有自己的成长线和选择权 |

## 标杆作品参考

| 作品 | 学什么 | 怎么学 |
|------|--------|--------|
| 《重生之神级学霸》 | 学业+商业的重生布局 | 每个阶段都有明确目标，知识碾压写得具体 |
| 《回到过去变成猫》 | 重生+变身的温馨日常 | 情感驱动而非纯利益驱动，温暖也是一条路 |
| 《重生之都市修仙》 | 修仙+都市重生结合 | 双体系互相配合，前世记忆+修炼金手指 |
| 《我的1979》 | 年代重生的生活质感 | 时代细节扎实，不是空降金手指而是融入时代 |
| 《完美人生》 | 商业布局的递进感 | 从小生意到帝国的阶梯清晰可循 |

## 写作铁律（10条）

1. **记忆不是百科全书** -- 主角只记得跟自己有关的大事件，具体细节会模糊
2. **蝴蝶效应必须兑现** -- 从第三章开始，部分前世记忆就开始不准了
3. **重生不等于开挂** -- 知道未来不等于有能力改变，有些悲剧可能无法避免
4. **遗憾是核心驱动力** -- 主角重生后最想改变的那件事，要贯穿全文
5. **布局要有"播种-收获"时间差** -- 今天种下的因要几十章后才结果，保持耐心
6. **信息管理要有规矩** -- 主角不能逢人就说"我知道未来"，要合理隐藏
7. **前世敌人要配得上** -- 仇人也要有智商和成长，不是一次性反派
8. **每阶段有新挑战** -- 解决了上辈子的问题，这辈子会出现新问题
9. **年龄/身份要受限** -- 穿回10岁就是10岁的身体条件和社交圈
10. **情感线要真实** -- 不能因为"前世爱过"就自动进入热恋，感情要重新建立

## 对话风格速查

| 场景 | 主角内心示例 | 外在表现示例 |
|------|-------------|-------------|
| 见到前世仇人 | "上辈子就是你害我家破人亡。这辈子，换我来了。" | 微微一笑："久仰大名。" |
| 见到失去的亲人 | "妈……你还活着。"（眼眶发红但忍住） | "妈，我饿了，想吃你做的红烧肉。" |
| 朋友不信预言 | "我知道听起来疯了，但三个月后这家公司会倒闭。" | "你们要是信我，把手里的股票卖了。" |
| 被问为什么知道 | "我做了个梦，梦到过这一幕。" | 用玩笑或模糊的话搪塞 |
| 布局收获时刻 | "终于等到这一天了。"（轻描淡写） | "运气好而已。"（前世付出了血的代价才知道的） |

## 自检清单

- [ ] 主角的记忆有明确的边界（不是什么都知道）？
- [ ] 蝴蝶效应已经导致至少一次记忆偏差？
- [ ] 核心遗憾/执念在本章有所推进？
- [ ] 布局有"播种"也有"收获"，不全是在铺垫？
- [ ] 信息差带来的爽感是具体的（不是空说"他早就知道了"）？
- [ ] 前世仇人有智商、有成长、不是纸片人？
- [ ] 主角的年龄/身份限制被尊重（不能有超出年龄的能力和资源）？
- [ ] 情感线在重新建立（不是前世记忆直接续接）？
- [ ] 有新的不可预见的挑战（不是照搬前世剧本）？
- [ ] 章尾有悬念，但不是简单的"他知道接下来会发生什么"？

## 重生类型速查

写之前先定好重生类型，不同类型节奏完全不同：

| 重生类型 | 时间跨度 | 核心玩法 | 代表作 |
|----------|---------|---------|--------|
| 短期重生 | 1-5年 | 挽回近在眼前的危机 | 完美人生 |
| 中期重生 | 5-15年 | 布局商业/事业从头再来 | 重生之神级学霸 |
| 长期重生 | 15-30年 | 从小开始，改变整个人生轨迹 | 我的1979 |
| 带系统重生 | 不定 | 重生+系统双金手指 | 重生之都市修仙 |
| 变身重生 | 不定 | 重生后换了身体/物种 | 回到过去变成猫 |

## 信息差使用节奏

| 阶段 | 信息差用法 | 频率 | 注意事项 |
|------|-----------|------|---------|
| 初期（1-10章） | 直接使用，快速建立爽感 | 每章1-2次 | 不要解释太多，先爽起来 |
| 前期（10-30章） | 布局播种，读者知道是伏笔 | 每2-3章一次 | 播种和收获要有时间差 |
| 中期（30-80章） | 伏笔开始兑现+蝴蝶效应出现 | 每3-5章一次兑现 | 信息差开始不准，增加悬念 |
| 后期（80章+） | 记忆大部分失效，靠实力和人脉 | 偶尔一次 | 主角不能永远靠记忆，要成长 |

## 重生遗憾设计速查

遗憾是重生文的灵魂，选一个贯穿全文：

| 遗憾类型 | 情感强度 | 适合题材 | 写法要点 |
|----------|---------|---------|---------|
| 失去至亲 | 极强 | 都市/年代 | 保护过程要一波三折，不能轻易成功 |
| 错过爱人 | 强 | 都市/校园 | 不能直接表白，要重新建立感情 |
| 事业失败 | 中 | 商业/官场 | 从零开始的过程要有新挑战 |
| 被人陷害 | 强 | 都市/权谋 | 复仇要有策略，不是直接动手 |
| 自己作死 | 中 | 各类型 | 最难写——要让读者相信"这次不会了" |
| 时代悲剧 | 强 | 年代/历史 | 个人力量vs时代洪流，可能改不了 |

## 前世记忆管理表（写之前先定好）

写之前把主角的记忆清单列出来，避免写到后面忘了：

| 记忆内容 | 精确度 | 蝴蝶效应风险 | 可用次数 |
|----------|--------|-------------|---------|
| 大事件日期 | 高→逐渐模糊 | 低 | 每个事件用1次 |
| 具体股票/投资 | 中 | 高（蝴蝶效应最先影响） | 2-3次后失效 |
| 人物性格/弱点 | 高 | 低 | 无限，但可能变 |
| 自己的经历细节 | 最高 | 最低 | 作为内心锚点 |
| 他人未公开的秘密 | 中 | 中 | 用1-2次，多了不真实 |
