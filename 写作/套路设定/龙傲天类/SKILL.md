# 龙傲天类写作 Skill

> 核心四字：**装逼打脸**。认知落差越大爽感越强。

---

## 核心张力

龙傲天 = 认知落差（外界低估 vs 主角碾压）。每一章的驱动力是「外界低估」与「主角碾压」之间的落差。

## 写作铁律（10条）

1. **仇恨值是燃料** -- 对方越嚣张打脸越爽
2. **信息差是核弹** -- 维持越久爆发威力越大
3. **碾压感来自反转** -- 先让对手嚣张，再一击秒杀
4. **打脸必须换花样** -- 武力/财力/身份/智力/人脉/气势轮流来
5. **爽感层层递进** -- 每次打脸的对象/场景/规模都要升级
6. **反派不能太蠢** -- "低估"必须有合理依据
7. **配角有自己的生命** -- 后宫/忠犬/反派都要有自己的故事线
8. **偶尔的失败让角色更真实** -- 永远碾压→读者麻木
9. **情感锚点比力量更重要** -- 白月光/兄弟情是持续追文的动力
10. **主角的强大必须有来源** -- 读者需要"见证"变强过程

## 单章结构（被小看-展示-打脸型）

| 阶段 | 占比 | 要做什么 |
|------|------|----------|
| 蓄力 | 30% | 对手小看/侮辱主角，主角保持低调 |
| 展示 | 35% | 被迫或主动展示实力，用最出乎意料的方式 |
| 打脸与围观 | 25% | 所有人震惊，主角淡定离场 |
| 收益 | 10% | 获得新追随者/资源，新舞台出现 |

## 打脸五步法

| 步骤 | 要做什么 | 关键点 |
|------|----------|--------|
| 1.拉仇恨 | 对方嚣张到极致 | 仇恨值越高打脸越爽 |
| 2.忍 | 主角不想打/不到时候 | 蓄积读者情绪 |
| 3.出手 | 一招/一句话/一个动作 | 快准狠 |
| 4.震惊 | 旁观者替主角装逼 | "不可能""他是什么人" |
| 5.余韵 | 主角淡定离场 | 越淡定越有力 |

## 身份层级与揭露节奏

| 章节范围 | 揭露身份层级 | 打脸对象 |
|----------|------------|----------|
| 1-30章 | 小身份 | 嘲笑主角的小人物 |
| 31-60章 | 区域级 | 地方权贵 |
| 61-100章 | 国家级 | 大势力 |
| 101-150章 | 大陆级 | 顶级势力 |
| 150章+ | 世界级 | 最终BOSS |

**身份揭露五种方式**：第三方揭露 / 被逼揭露 / 事件揭露 / 主动揭露 / 身份叠加揭露

## 对话原则：少、冷、准

| 类型 | 句式示例 |
|------|----------|
| 蔑视型 | "哦。""嗯。""所以？""说完了吗？" |
| 打脸型 | "记得吗？""这次轮到你了。" |
| 霸气型 | "一起上。""够了。""你不够格。" |
| 身份型 | "回去问你家大人。""不配知道。" |
| 温情型 | "有我在。""别怕。""吃了吗？" |

**核心法则**：话越少杀伤力越大。最强的龙傲天不需要说话——让环境和旁人替他说。

## 角色模板速查

| 角色 | 核心 | 注意事项 |
|------|------|----------|
| 龙傲天主角 | 表面低调实际碾压 | 三种分支：淡漠/热血/腹黑 |
| 舔狗反派 | 嚣张但内心自卑 | 给他嚣张的理由 |
| 后宫团 | 每个有自己的故事线 | 冰山/暴力/腹黑/天真 |
| 忠犬小弟 | 认定一人不会变 | 有自己的绝活 |
| 隐藏大佬 | 不起眼的扫地大爷 | 一两个就够 |
| 白月光 | 主角的情感锚点 | 已逝/远方/在身边 |
| 最终BOSS | 站在世界顶端 | 前期只闻其名 |

## 失败模式速查

| 症状 | 处方 |
|------|------|
| 蓄力不足直接打脸 | 加一场小压迫 |
| 打脸拖沓 | 翻盘后最多3段围观 |
| 实力通货膨胀 | 用限制条件制造张力 |
| 对手太弱 | 对手至少有一次"差点赢了" |
| 主角全程面瘫 | 内心独白给真实情绪 |
| 打脸方式重复 | 换花样：沉默/反问/行动 |
