# 写作质量等级标准

> 定义四个质量等级，让AI写作时有明确目标。
> 每个等级有具体指标，不是主观感觉。

---

## 等级一览

| 等级 | 名称 | 对标 | 一句话定义 |
|------|------|------|-----------|
| D | AI垃圾 | 模板化角色名 | 能读但满是AI味，模板化，无灵魂 |
| C | 模板网文 | 欲望开发系统 | 结构完整，节奏对，但没有意外 |
| B | 中上水平 | 有辨识度的网文 | 有技法，有辨识度，有记忆点 |
| A- | 准一流 | 旧日焚馆 | 有灵气，有风格，有文学性 |

---

## D级：AI垃圾

**特征：** 能读，但一看就知道是AI写的。

| 指标 | 标准 |
|------|------|
| 禁用词 | 大量出现"仿佛/似乎/不禁/竟然/一时间" |
| 句式 | 连续5句以上用同一句式开头 |
| 对话 | 书面化，所有人说话方式一样 |
| 描写 | 只有视觉，没有其他感官 |
| 情感 | "他感到一阵XX"直接陈述 |
| 节奏 | 匀速推进，没有波峰波谷 |
| 角色 | 脸谱化，换名字不影响理解 |
| 结构 | 起承转合标准但死板 |

**典型段落：**
> 他感到一阵心痛。仿佛有什么东西碎裂了。他不禁握紧了拳头。她似乎察觉到了什么，轻轻叹了口气。一时间，空气中弥漫着尴尬的沉默。

---

## C级：模板网文

**特征：** 结构完整，节奏对，但没有意外。读者能看完但记不住。

| 指标 | 标准 |
|------|------|
| 禁用词 | 偶尔出现但不密集 |
| 句式 | 有变化但不丰富 |
| 对话 | 有区分度但不够鲜明 |
| 描写 | 五感至少2种，但不深入 |
| 情感 | 有身体反应但模式化（"心跳加速""脸红"） |
| 节奏 | 有波峰波谷但可预测 |
| 角色 | 有标签但缺乏层次 |
| 结构 | 有悬念有钩子但套路化 |

**典型段落：**
> 她咬了咬嘴唇，把头偏向一边。手指无意识地绞着衣角。"随便你。"声音很轻，但他听得出来，那不是真的随便。

---

## B级：中上水平

**特征：** 有技法，有辨识度，有记忆点。读者会推荐给别人。

| 指标 | 标准 |
|------|------|
| 禁用词 | 基本没有 |
| 句式 | 多样，长短交替，有节奏感 |
| 对话 | 遮住名字能猜出是谁说的 |
| 描写 | 五感至少3种，有通感运用 |
| 情感 | 用行为和环境映射，不用直接陈述 |
| 节奏 | 有意外转折，不完全可预测 |
| 角色 | 有矛盾面（表面A内心B） |
| 结构 | 有留白，有物象象征 |
| 记忆点 | 至少1个场景让读者事后回想 |

**典型段落：**
> 楼梯间的声控灯又灭了。她没动，他也没动。黑暗里只有两个人的呼吸声，和窗外不知谁家的电视声。"你手好冰。"他说。她没抽回去。灯亮了。她的手已经不冰了。

---

## A-级：准一流

**特征：** 有灵气，有风格，有文学性。读者会反复读，会引用。

| 指标 | 标准 |
|------|------|
| 禁用词 | 零 |
| 句式 | 有自己的句式风格，可辨识 |
| 对话 | 每句话有潜台词，沉默比语言更有力 |
| 描写 | 五感联动，有独特的感官组合 |
| 情感 | 用留白和物象，让读者自己感受 |
| 节奏 | 打破预期，有"意料之外情理之中" |
| 角色 | 有成长弧线，有不可预测性 |
| 结构 | 多线交织，伏笔回收 |
| 风格 | 有叙事声音，读一段就知道是谁写的 |
| 灵气 | 至少1处让读者停下来反复品味 |

**典型段落：**
> 她把围巾解下来递给他。围巾上有她颈窝的温度和洗衣液的味道。他接过去，没有围上，只是攥在手里。后来他才知道，那天她把围巾给他，是因为她要走了，想在他身上留点什么。围巾他围了三年，直到毛线起球、颜色褪尽，也没洗过一次。

---

## 写作时的等级定位

| 场景 | 目标等级 | 说明 |
|------|---------|------|
| 日常过渡章 | C+ | 不需要每章都惊艳，但不能有AI味 |
| 角色建立章 | B | 角色第一次出场必须有记忆点 |
| 冲突高潮章 | B+~A- | 关键场景必须有灵气 |
| 情感爆发章 | A- | 最好的技法用在最重要的场景 |
| 开篇前3章 | B+ | 前3章决定读者是否继续 |

**铁律：任何章节都不能低于C级。D级=返工。**
