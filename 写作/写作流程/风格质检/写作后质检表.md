# 写作后质检表（V2 统一版）

> 每次生成新章节后，AI自动执行以下质检。适用于所有题材类型。

---

## 第一层：基础质检（任何题材都要过）

```
【1. 角色一致性】
□ 对话是否符合角色性格？（OOC检查）
□ 行为是否符合角色设定和当前状态？
□ 说话方式是否一致？（口头禅、语气、方言）
□ 角色关系值变化是否合理？

【2. 对话辨识度】
□ 遮住名字能否分出是谁在说话？
□ 不同角色的对话是否有区别？
□ 是否有"所有角色说话都一样"的问题？

【3. 文风统一】
□ 叙述视角是否一致？（第一/第三人称）
□ 时态是否一致？（过去/现在）
□ 语言风格是否统一？
□ 有无突然冒出的AI腔/书面腔/翻译腔？

【4. 世界观一致性】
□ 是否违反世界规则？
□ 地理、时间、科技水平是否矛盾？
□ 超凡元素使用是否符合设定？
□ 科技/游戏/平台描写是否准确？（瓦罗兰特/CS是端游不是手游，不能写成手机操作）

【5. 观察细节一致性】
□ 角色"扫一眼"能看到多少？（扫一眼=颜色/轮廓，看不清材质/品牌/文字）
□ 角色"远远看"能看到多少？（隔了大半个校园=轮廓/颜色，看不清细节）
□ 角色的幻想/回忆是否跟她实际观察到的一致？（没看到蕾丝就不该出现在回忆里）
□ 同一个生理细节（如内裤湿了）是否重复描写？（只写一次，后续用递进代替重复）

【5. 时间线检查】
□ 时间是否连贯？
□ 是否有时间矛盾？
□ 时间跨度是否合理？

【6. 伏笔检查】
□ 是否有新伏笔埋设？→ 更新伏笔追踪
□ 是否有旧伏笔回收？→ 更新伏笔追踪
□ 是否有遗忘的伏笔？（超过20章未回收）

【7. 国家/文化设定检查】
□ 政治体制描写是否正确？
□ 法律、教育、医疗等细节是否准确？
□ 文化习俗是否符合？

【8. 感官密度检查】
□ 每1000字有没有至少1个嗅觉细节？（最常被遗忘的感官）
□ 新场景开头有没有环境气味？（食堂油烟味/走廊消毒水/宿舍洗衣液）
□ 角色靠近时有没有对方身上的气味？（洗发水/汗味/香水/烟味）
□ 情绪场景有没有气味配合？（紧张=汗味，暧昧=花香，压抑=霉味）
□ 角色初见时有没有用物件暗示性格？（耳机+鼠标垫=游戏宅，保温杯+面膜=精致女孩）
```

## 第二层：题材专项质检

### 恐怖/克苏鲁类
```
□ 恐惧递进是否合理？（不能一上来就最高级）
□ 是否过度依赖jump scare而缺少氛围铺垫？
□ 感官描写是否以听觉/触觉为主？（视觉恐怖最廉价）
□ "否定描写法"是否到位？（不直接描述怪物，通过反应暗示）
□ 克苏鲁禁忌：是否避免了过于具象化的神明描写？
```

### 亲密/情感类
```
□ 是否遵循亲密类 01_核心规则/？
□ 身体描写是否有因果链？（不是突然脱光）
□ 过程中有疼痛和快感交替吗？（不是全程爽）
□ 结束后有物理后遗症描写吗？（肿胀/红痕/走路困难/声音嘶哑）
□ 后续章节有延续症状吗？（不是下一章就活蹦乱跳）
□ 对话是否有辨识度？（不能所有人呻吟都一样）
□ 亲密场景是否有情感驱动？（不是为了写而写）
□ 粗俗/文雅比例是否符合场景需要？
□ 安全分类器风险：敏感内容是否只写入文件不通过对话？
□ 情欲递进是否有层次？（同一细节如"湿了"只写一次，后续用"热流""更猛"递进）
□ 睡姿/体位是否生理合理？（手指插着睡会麻，改成"指尖搭在穴口"更合理）
□ 幻想/回忆内容是否跟角色实际观察一致？（不能凭空添加角色没看到的细节）
```

### 战斗类
```
□ 战斗节奏是否有张弛？（不能全程高能）
□ 伤害描写是否有后果链？（受伤了后面要体现）
□ 武器/能力使用是否符合设定？
□ 战斗空间利用是否合理？
□ 胜负是否符合实力对比？（不能开挂）
```

### 悬疑/推理类
```
□ 线索是否公平展示？（读者有可能推理出来）
□ 误导项是否自然？（不能太刻意）
□ 推理逻辑链是否自洽？
□ 真相揭示时是否有"恍然大悟"感？
□ 是否犯了"突然蹦出新证据"的推理大忌？
```

### 玄幻/冒险类
```
□ 力量等级使用是否一致？
□ 战斗/冒险中的消耗是否有记录？
□ 地图移动距离和时间是否合理？
□ 宝物/技能获得是否符合规则？
□ 副本/事件链是否有逻辑推进？
```

### 丧尸/末日类
```
□ 资源消耗是否有记录？（弹药、食物、医疗）
□ 感染/变异规则是否一致？
□ 团队信任值变化是否合理？
□ 生存难度是否递增？
□ 末日环境下的人性反应是否真实？
```

## 第三层：去AI味检查

```
【AI腔检测】
□ 有没有"不禁""竟然""居然"堆砌？（最多1个/千字）
□ 有没有"紧接着""然后""接着"等机械连接词？（删掉，用句子自身节奏推进因果链）
□ 有没有"他对X说。然后他对Y说"的啰嗦对话归属？（用动作一次交代两层）
□ 有没有"不是A，是B"句式堆砌？（全篇最多2-3次，直接写出B本身）
□ 有没有"宛如""仿佛""恰似"三连？（最多1个/段落）
□ 有没有"心中暗想""暗自思忖"？（用行为暗示替代）
□ 有没有大段心理独白？（改用动作/微表情/环境暗示）
□ 有没有"他/她不知道的是"这种上帝视角泄露？
□ 每段对话是否都带"说道"？（换成动作/语气/省略）
□ 有没有"XXX的XXX"名词性短语堆砌？
□ 段落长度是否多样？（长短交替才有节奏）

【真实性检测】
□ 角色反应是否符合正常人？（不能太冷静也不能太夸张）
□ 痛苦/恐惧/快感描写是否有生理基础？
□ 对话是否像真人说话？（读出声来检验）
□ 是否有信息重复？（同一事实说了两遍以上）
```

---

## 第四层：结构与节奏

```
【章节结构】
□ 开头是否有钩子？（前3句要吸引人）
□ 结尾是否有悬念或情感落点？（让人想看下一章）
□ 中段是否有"水"的部分？（可以砍掉的段落=水）
□ 场景切换是否自然？（有过渡而不是硬切）

【节奏检查】
□ 是否有"一马平川"的段落？（全程一个节奏=催眠）
□ 高潮前是否有足够的铺垫？
□ 高潮后是否有喘息/余韵？
□ 字数分配是否合理？（重要场景长，过渡场景短）
```

## 质检报告格式

```
质检报告

角色一致性: ✅ 通过
对话辨识度: ✅ 良好（3个角色各有特色）
文风统一: ✅ 通过
世界观一致性: ✅ 通过
时间线: ⚠️ 警告（第5章下雨，第6章晴天，未交代天气变化）
伏笔: ✅ 已回收F001，新增F002
国家设定: ✅ 通过
题材专项: ✅ 恐怖递进合理
去AI味: ⚠️ 警告（第3段有"宛如...仿佛..."连用）
结构节奏: ✅ 通过

总体评价: 良好，2个警告需处理
→ 修复项：时间线过渡补一句天气描写；改写第3段比喻句
```

## 人工抽查建议

以下项目建议用户人工抽查（AI难以判断）：

| 项目 | 说明 |
|------|------|
| **剧情逻辑** | AI难以判断剧情走向是否合理 |
| **情感共鸣** | AI难以判断是否感人/吓人/刺激 |
| **原创性** | AI难以判断是否抄袭或过于套路 |
| **读者体验** | AI难以判断是否好看/想看下一章 |
| **尺度把控** | AI难以判断成人内容是否过火或不够 |
