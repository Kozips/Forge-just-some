# 写作前检查表（V2 统一版）

> 每次写新章节前，AI自动执行以下检查。适用于所有题材类型。

---

## 第一层：硬性必查（任何题材都要过）

```
【1. 题材与规则加载】
□ 确认当前小说的题材组合（如：恐怖+亲密+冒险）
□ 加载对应类型的 SKILL.md
□ 加载 01_核心规则/ 全部文件
□ 确认核心规则中的红线项

【2. 世界观确认】
□ 读取小说总谱/世界观档案.md（如有）
□ 确认世界规则未被违反
□ 确认超凡/科技/魔法体系的一致性

【3. 人物确认】
□ 读取本章涉及角色的人物档案（人物/SKILL.md 路径）
□ 确认角色当前状态（身体/情绪/关系值）
□ 确认角色性格标签未冲突（是否OOC风险）
□ 加载角色的对话习惯模块

【4. 时间线确认】
□ 读取小说总谱/时间线.md
□ 确认当前时间点（日期/时辰/季节）
□ 确认上一章结尾时间，避免跳跃

【5. 伏笔确认】
□ 读取小说总谱/伏笔追踪.md
□ 检查是否有到期需回收的伏笔
□ 检查本章计划是否需要埋设新伏笔
```

---

## 第二层：题材专项（按类型勾选）

### 恐怖/克苏鲁类
```
□ 加载 02_专业知识/ 中的恐惧心理学
□ 确认本章恐惧等级（暗示/不安/惊吓/崩溃）
□ 确认感官描写的比重（听觉>视觉>触觉）
□ 确认"否定描写法"是否到位
```

### 亲密/情感类
```
□ 确认角色间亲密等级（初识/暧昧/亲密/深度）
□ 加载亲密类 01_核心规则/ 描写边界.md
□ 确认是否需要安全分类器绕过策略
□ 加载 03_素材库/感官描写素材库.md
```

### 战斗类
```
□ 确认武器/装备/能力设定
□ 加载 02_专业知识/ 中的战斗术语
□ 确认战斗空间（室内/室外/地形）
□ 确认伤害等级和后果链
```

### 悬疑/推理类
```
□ 确认线索埋设计划（本章需露出哪些线索）
□ 确认误导项是否已安排
□ 加载 04_写作模板/ 中的推理结构模板
□ 确认读者视角 vs 角色视角的信息差
```

### 玄幻/冒险类
```
□ 确认力量体系等级和角色当前等级
□ 加载 02_专业知识/ 中的设定术语
□ 确认地图/位面/区域设定
□ 确认任务/副本/事件链进度
```

### 丧尸/末日类
```
□ 确认资源状态（食物/弹药/医疗）
□ 确认安全区域状态
□ 确认感染/变异规则
□ 确认团队动态和信任值
```

---

## 第三层：对话与文风

```
【对话习惯确认】
□ 每个出场角色的说话方式已加载
□ 口头禅/语气词/方言已确认
□ 角色间对话风格差异已确认
□ 遮住名字能否分出谁在说话？（预判）

【文风确认】
□ 叙述视角确认（第一/第三人称）
□ 时态确认（过去/现在）
□ 语言风格确认（文艺/口语/混搭）
□ 本章目标情绪基调确认
```

---

## 第四层：技法预设

```
【写作技法选择】
□ 本章重点技法是什么？（如：悬念钩子/感官轰炸/对话推进）
□ 是否需要从 03_素材库/ 调用片段？
□ 是否需要从 04_写作模板/ 调用结构骨架？
□ 本章节奏定位（铺垫/过渡/高潮/收束）
```

## 常见踩坑速查

| 坑 | 怎么避 |
|---|--------|
| 角色突然说网络梗但世界观是古代 | 检查世界观+角色背景一致性 |
| 上一章结尾下雨这章晴天没交代 | 检查时间线+环境连续性 |
| 所有角色说话都一个味 | 预检对话习惯差异度 |
| 埋了伏笔但忘了写进追踪表 | 写完立刻更新伏笔追踪.md |
| 用了过强的描写但后面没后果 | 检查伤害/后果链是否闭合 |
