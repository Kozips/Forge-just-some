# 修改流程 · 五轮修改法

> 写完初稿后，按此流程逐轮修改。每轮聚焦一个维度，不要同时改多件事。
> 目标：从"能看"提升到"准一流"。

---

## 第一轮：铁律违规检查（必做）

逐条对照铁律，标记违规处：

- [ ] 是否存在"不是A，是B"句式？（叙述中禁用，对话中可用）
- [ ] 是否有"仿佛/似乎/不禁/竟然/居然/一时间"等AI废话词？
- [ ] 对话是否书面化？（"我认为我们应该"→"我觉得咱俩得"）
- [ ] 是否有"他感到一阵XX"的直接情感陈述？（改为身体反应/环境映射）
- [ ] 是否有连续3段都是描写或都是对话？（需交替）
- [ ] 角色称呼是否一致？（同一角色不要用3种以上称呼）

**修改方式：** 全文搜索禁用词，逐一替换。对话逐句检查口语化程度。

---

## 第二轮：对话辨识度检查（必做）

把每段对话的角色名遮住，只看台词，能猜出是谁说的吗？

**检查标准：**
- [ ] 每个角色的用词习惯不同（有人爱用成语，有人爱用脏话，有人爱用反问）
- [ ] 每个角色的句子长度不同（有人啰嗦，有人惜字如金）
- [ ] 每个角色的语气标签不同（有人爱嘲讽，有人爱撒娇，有人爱沉默）
- [ ] 对话中有潜台词（字面意思≠真实意思，至少30%的对话有潜台词）
- [ ] 对话中有打断/没说完/转移话题（真人不会每句话都说完整）

**修改方式：** 列出每个角色的3个语言特征标签，逐句对话对照检查。

---

## 第三轮：场景逻辑检查（必做）

逐场景检查角色行为是否合理。核心问题：**角色明明能做但没做的事，有没有理由？**

**检查标准：**
- [ ] 角色在场时有没有"明明可以做但没做"的事？（要微信/打电话/逃跑/求助）→ 没做的话，必须有理由（入迷/犹豫/被打断/没想到）
- [ ] 角色离开/到场的动机是否充分？（不能"忽然出现""忽然走了"没有原因）
- [ ] 时间线是否连贯？（上一段在食堂，下一段在操场，中间有没有交代）
- [ ] 信息差是否合理？（角色知道的信息是否和读者知道的一致？不一致时是否有意为之？）
- [ ] 物件/道具是否前后一致？（前面说拿了手机，后面不能"两手空空"）
- [ ] 角色的情绪变化是否有铺垫？（上一段开心，下一段突然生气，中间要有触发事件）

**常见逻辑漏洞：**
| 漏洞 | 例子 | 修复 |
|------|------|------|
| "为什么不直接说" | 她明明可以要微信但没要 | 加入迷/犹豫/被打断的理由 |
| "为什么不去找人帮忙" | 被困了但不打电话 | 手机没电/被没收/信号被屏蔽 |
| "为什么这么巧" | 两个人刚好在电梯遇到 | 改为其中一方故意等 |
| "为什么不用X" | 有武器但不用 | 武器坏了/够不到/不忍心 |
| "为什么不说实话" | 明明可以解释误会但不解释 | 有苦衷/来不及/对方不听 |

**修改方式：** 逐场景问自己"这个角色为什么不做X？"，答不上来就要补理由。

---

## 第四轮：情绪曲线检查（重要）

画出本章的情绪曲线，检查是否有问题：

**检查标准：**
- [ ] 是否有波谷？（连续高压读者会麻木，每2000字需要一个喘息点）
- [ ] 情绪是否有递进？（不能原地踏步，每段情绪要比上一段更强或转向）
- [ ] 高潮前是否有足够的铺垫？（直接爆发=廉价，蓄力后爆发=有力）
- [ ] 余韵是否到位？（高潮后不能马上结束，需要200-500字的余韵沉淀）
- [ ] 情绪类型是否单一？（不能全篇只有"紧张"，需要紧张→放松→紧张→恐惧→释然的交替）

**修改方式：** 在每段开头标注情绪类型和强度（1-10），画出曲线图，找到问题段落。

---

## 第五轮：描写密度检查（重要）

统计本章的描写密度：

**量化标准：**
- [ ] 每1000字至少1处五感描写（视觉/听觉/嗅觉/触觉/味觉）
- [ ] 每500字至少1处身体语言（手势/表情/姿态/眼神）
- [ ] 每个场景至少1处环境描写（光线/温度/气味/声音/空间感）
- [ ] 对话场景中至少30%的行有动作/神态穿插（不能纯对话）
- [ ] 亲密场景中五感至少写了3种（不能只有视觉）

**修改方式：** 纯对话段落插入身体语言；纯叙述段落插入感官细节；关键场景补充环境描写。

---

## 第六轮："灵气"检查（加分项）

这是从"能看"到"准一流"的关键一轮：

**检查标准：**
- [ ] 有没有至少1个"意外"？（角色做了读者没想到的事，或场景走向出乎意料）
- [ ] 有没有至少1个"留白"？（没有说破，让读者自己品味的地方）
- [ ] 有没有至少1个"物象"？（用物件承载情感，而不是直接说"他很伤心"）
- [ ] 有没有至少1个"反差"？（表面和内心的反差，或场景和情绪的反差）
- [ ] 对话有没有"节奏变化"？（不是每句都一样长，有快有慢有停顿）

**修改方式：** 找到最平淡的段落，加入一个意外/留白/物象/反差。

---

## 修改完成标准

| 等级 | 通过轮数 | 质量水平 |
|------|---------|---------|
| 能看 | 1-2轮 | 网文平均水平 |
| 中上 | 3轮 | 超过80%网文 |
| 准一流 | 4-5轮 | 有技法含量 |
| 一流 | 6轮 | 有灵气 |

---

## 快速修改清单（5分钟版）

如果时间有限，只做这些：
1. 全文搜索"仿佛/似乎/不禁"并替换
2. 检查每段对话是否有动作穿插
3. 检查是否有连续3段同类型内容
4. 在最平淡的段落加入一个反差或留白
