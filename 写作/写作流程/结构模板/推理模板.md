# 推理模板

> 适用于悬疑、推理、刑侦题材。

---

## 核心结构

```
案件发生 → 调查 → 线索收集 → 推理 → 真相揭露
```

---

## 案件设计

| 元素 | 说明 |
|------|------|
| **受害者** | 谁死了/被偷了/失踪了 |
| **嫌疑人** | 3-5个嫌疑人，每人都有动机 |
| **作案手法** | 如何做到的 |
| **关键线索** | 1-3个决定性线索 |
| **红鲱鱼** | 误导读者的假线索 |

---

## 经典结构

| 阶段 | 内容 | 占比 |
|------|------|------|
| **案件发生** | 发现尸体/犯罪现场 | 10% |
| **初步调查** | 警察到场、初步取证 | 15% |
| **嫌疑人登场** | 逐一登场，每人都有嫌疑 | 20% |
| **深入调查** | 发现更多线索和秘密 | 25% |
| **推理** | 主角推理过程 | 15% |
| **真相揭露** | 公布凶手和手法 | 10% |
| **结局** | 案件解决、凶手伏法 | 5% |

---

## 推理技巧

| 技巧 | 说明 |
|------|------|
| **公平性** | 读者应该能根据线索推理出真相 |
| **误导** | 用红鲱鱼误导读者 |
| **反转** | 真相出乎意料但合理 |
| **伏笔** | 前面埋的线索后面要用到 |

---

## 核心原则

推理小说的核心是"公平游戏"——读者应该能在作者给出的线索基础上推理出真相。每一个误导都要有对应的反误导，每一个谜底都要有对应的伏笔。
不要用读者不知道的信息作为破案关键——这是"欺骗读者"而非"挑战读者"。

## 快速参考表

| 阶段 | 占比 | 核心任务 | 常见错误 |
|------|------|---------|---------|
| 案件发生 | 10% | 建立谜团，吸引读者 | 死者太边缘，读者不关心 |
| 初步调查 | 15% | 建立调查框架 | 线索给出太乱 |
| 嫌疑人登场 | 20% | 逐一登场，动机清晰 | 嫌疑人太少或太多 |
| 深入调查 | 25% | 线索收集，红鲱鱼 | 线索没有区分度 |
| 推理过程 | 15% | 逻辑推演 | 推理太跳跃 |
| 真相揭露 | 10% | 公布凶手和手法 | 反转太牵强 |
| 结局 | 5% | 收束情感 | 匆忙收尾 |

---

## 关键规则

1. **线索必须平均分布**——每个嫌疑人在前期都要有出场机会
2. **红鲱鱼要像真线索**——读者事后回看会觉得"确实被误导了"而不是"这根本不合理"
3. **凶手动机要合理**——不能只是为了杀人而杀人，要有人性的弱点
4. **破案关键不能靠巧合**——主角发现关键证据必须有铺垫
5. **推理过程要透明**——读者应该能跟随推理，而不是最后突然揭晓

---

## 常见错误

1. **线索给得太隐蔽**——读者读完都不知道有这条线索，这不是高明的推理
2. **凶手靠动机不明**——"纯粹的恶意"作为动机是偷懒，读者不会信服
3. **侦探突然开挂**——关键证据靠"我早就注意到了"出现，没有铺垫

---

## 适用场景

- 悬疑推理、刑侦
- 密室杀人、连环杀手
- 需要智力挑战的作品
