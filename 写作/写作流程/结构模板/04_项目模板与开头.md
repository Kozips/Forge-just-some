## 小说项目模板

# 小说项目模板 · 自动生成流程

> 用户说"我要写XX类型的小说"时，自动执行以下流程。

---

## 第一步：创建项目目录结构

```
outputs/小说书写/[暂定小说名]/
├── 大纲/
│   └── 总纲.md                 ← 自动生成（基于用户说的类型）
├── 人物设定/
│   └── [主角名].md             ← 自动生成（基于类型的角色卡）
├── 世界观/
│   └── 设定.md                 ← 自动生成（基于世界观模板）
├── 番外/
│   └── README.md               ← 空目录，后续填充
├── 章节/
│   └── README.md               ← 空目录，后续填充
├── 记忆卡.md                   ← 创作用（每次写完章节后更新）
└── 记忆卡_扮演.md              ← 角色扮演用（每5-10轮对话后更新）
```

## 第三步：继续创作时的自动读取

用户说"继续《小说名》的创作"时，自动按以下顺序读取：

1. **记忆卡.md** → 了解当前进度和下一章计划
2. **大纲/总纲.md** → 了解全局结构和伏笔
3. **人物设定/*.md** → 加载角色卡
4. **世界观/设定.md** → 加载世界观

然后按照记忆卡中的"下一章要做什么"直接开始写作。

---

## 第四步：写完章节后自动更新

每写完一章，自动更新记忆卡：

1. 在"已完成章节摘要"中添加本章摘要（200字以内）
2. 更新"当前进度"（最新章节号、阶段）
3. 更新"未解决的伏笔"（新埋的伏笔、已揭示的伏笔）
4. 更新"下一章要做什么"（下一章的计划）
5. 更新"角色当前状态"（角色的情感/关系/位置变化）

## 开头三章模板

# 开头三章模板

> 网文的生死在前3章。读者给你的时间只有3000字。
> 前3章要完成：建世界、建角色、抛悬念、第一次冲突、第一个大钩子。
> **所有开篇按 写作/写作流程/质检/05_写作风格自用.md 的笔风来——克制叙事，物件代人，感官锤子收尾，不做总结。**

---

## 第1章：入局（2000-3000字）

**任务：** 30秒内抓住读者 + 建立核心设定 + 留下第一个悬念

### 结构

```
开篇钩子（前200字）→ 核心设定交代（500字）→ 角色亮相（800字）→ 第一个异常/冲突（500字）→ 章末钩子（200字）
```

### 开篇钩子（前200字必须抓住人）

| 钩子类型 | 模板 | 适用题材 |
|---------|------|---------|
| 异常开场 | 第一句话就是不对劲的事 | 恐怖/悬疑/规则怪谈 |
| 反差开场 | 用平静叙述写不平静的事 | 猎奇/文学/暗黑 |
| 悬念开场 | 抛出一个读者想知道答案的问题 | 所有题材 |
| 日常切入异常 | 前100字正常，第101字开始不对 | 都市/校园/日常 |
| 情绪锚点 | 用一个情绪场景让读者代入 | 情感/恋爱/虐心 |
| 身份反差 | 角色的表面身份vs真实身份 | 龙傲天/重生/系统 |

### 章末钩子

第1章结束时，读者必须想知道：
- "接下来会发生什么？"（悬念型）
- "这个人到底是谁？"（身份型）
- "他会怎么选？"（困境型）
- "这个世界怎么了？"（世界观型）

---

## 第2章：建人（2000-3000字）

**任务：** 让读者记住主角 + 建立第一个角色关系 + 推进悬念

### 结构

```
延续第1章悬念（300字）→ 主角日常展示（800字）→ 第一个角色互动（800字）→ 悬念加深/新信息（500字）→ 章末钩子（200字）
```

### 角色建立要点

第2章必须让读者知道主角的：
- **一个欲望**（他想要什么）
- **一个缺陷**（他缺什么/怕什么）
- **一个习惯**（让角色活起来的小动作）
- **一个秘密**（读者知道但其他角色不知道的）

### 第一个角色互动

- 不要写"介绍式"对话（"你好我叫XX，我是XX人"）
- 用**冲突或互助**建立关系（一起做一件事/吵一架/帮一个忙）
- 对话中透露信息，不要用旁白交代

---

## 第3章：起爆（2500-3500字）

**任务：** 第一次真正冲突 + 展示核心张力 + 留下最大的钩子

### 结构

```
冲突触发（500字）→ 冲突升级（1000字）→ 第一次高潮/转折（800字）→ 余波+新信息（500字）→ 大钩子（300字）
```

### 第一次冲突

- 必须是**有风险的**冲突（不是小打小闹）
- 主角必须**做出选择**（展示性格）
- 结果必须**出乎意料**（不能完全在读者预期之内）
- 冲突结果必须**改变现状**（不能回到冲突前的状态）

### 大钩子（第3章末）

第3章结束时，读者必须进入"停不下来"的状态：
- 抛出一个**更大的世界观秘密**
- 或：一个**角色身份反转**
- 或：一个**不可逆的决定**
- 或：一个**新的威胁出现**

---

## 前3章节奏模板

### 模板A：快节奏（热血/玄幻/系统）

```
第1章：异常事件 + 系统觉醒 + 爽点预告     [情绪：好奇→兴奋]
第2章：能力展示 + 第一次小冲突 + 打脸预热  [情绪：期待→爽]
第3章：第一次打脸 + 实力展示 + 更强敌人出现 [情绪：爽→紧张]
```

### 模板B：中节奏（都市/悬疑/校园）

```
第1章：日常切入 + 异常出现 + 悬念建立      [情绪：平静→不安]
第2章：角色关系建立 + 信息收集 + 暗线铺设  [情绪：好奇→焦虑]
第3章：第一次冲突 + 真相一角 + 大悬念      [情绪：紧张→震惊]
```

### 模板C：慢节奏（情感/文学/恐怖）

```
第1章：情绪锚点 + 氛围营造 + 细节暗示      [情绪：代入→微妙不安]
第2章：角色深入 + 关系推进 + 暗流涌动      [情绪：共鸣→担忧]
第3章：情感爆发/恐怖降临 + 世界观崩塌      [情绪：震撼→停不下来]
```

---

## 前3章自检清单

- [ ] 第1章前200字有没有钩子？（没有→重写开头）
- [ ] 第1章结束时读者想不想看第2章？（不想→章末钩子不够强）
- [ ] 第2章有没有让主角"活起来"？（没有→缺少习惯/欲望/缺陷）
- [ ] 第3章有没有真正的冲突？（没有→节奏太慢）
- [ ] 第3章结束时读者能不能停下来？（能→大钩子不够强）
- [ ] 前3章有没有至少1个记忆点？（没有→缺少灵气）
- [ ] 前3章的角色对话有没有辨识度？（没有→去AI味）
