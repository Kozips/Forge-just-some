# 大纲编写技法

> 大纲是长篇小说的骨架。没有骨架的肉是烂泥，没有肉的骨架是骷髅。
> 大纲负责骨架，正文负责长肉。两者缺一不可。

---

## 一、大纲的作用

### 大纲不是限制创造力，是防止写崩

很多作者抗拒大纲，觉得"写大纲会限制灵感"。这是对大纲最大的误解。

大纲不是剧本，不是逐字逐句的规划。大纲是地图——你知道目的地在哪，但路上看到风景可以随时停下来。

**没有大纲的长篇通常这样死：**
- 前30章：灵感爆发，写得飞起，读者追得爽
- 第50章：开始有点不知道往哪走，节奏开始拖
- 第80章：完全失控，要么疯狂注水，要么草草结尾
- 结局：烂尾。读者骂街，作者自己也不想回看

这不是灵感不够的问题，是**结构不够**的问题。

### 大纲的三个层次

```
总纲（一本书的整体骨架）
  └─ 分卷纲（每卷30-50章的结构）
       └─ 章节纲（每章2000-5000字的详细规划）
```

三个层次的关系：
- **总纲**决定方向——"这本书要往哪走"
- **分卷纲**决定节奏——"这一卷要完成什么任务"
- **章节纲**决定细节——"这一章具体写什么场景"

写长篇不规划这三个层次，就像盖楼不打地基——前几层还行，越往上越歪。

## 二、总纲编写（一本书的整体骨架）

### 总纲的核心三问

动笔写大纲之前，先回答三个问题：

**1. 这本书卖什么？（一句话概括）**

用一句话说清这本书的核心卖点。如果一句话说不清，说明你自己也没想清楚。

| 题材 | 一句话示例 |
|------|-----------|
| 都市重生 | 重生回到2010年，用未来十年的记忆炒股买房，顺便弥补上辈子的遗憾 |
| 玄幻升级 | 废柴觉醒隐藏血脉，从家族弃子一路打到天界至尊 |
| 悬疑推理 | 一个能看见死者临终画面的刑警，追查连环杀手却发现凶手是自己 |
| 情感言情 | 冷面总裁和他的替身妻子，假戏真做的三年婚姻 |
| 末日生存 | 末世降临前三天，我开始疯狂囤物资 |

如果这句话不能让目标读者产生"想看"的冲动，这本书的总纲需要重新想。

**2. 核心矛盾是什么？**

核心矛盾 = 主角最想要什么 + 最大的阻碍是什么。

```
核心矛盾公式：
主角想要 [目标A]，但 [阻碍B] 挡在面前。
为了达成 [目标A]，主角必须 [行动C]，
但每次行动都会引发 [后果D]，让局面更复杂。
```

示例：
- 都市：主角想报仇（目标），但仇人是他亲哥（阻碍），每次接近真相都发现更多家族秘密（后果）
- 玄幻：主角想救妹妹（目标），但需要突破到元婴期（阻碍），每次突破都引来更强敌人追杀（后果）
- 悬疑：主角想破案（目标），但凶手在警局内部（阻碍），每次查到线索都指向自己最信任的人（后果）

**3. 结局预设**

动笔之前就想好结局的大方向：

| 结局类型 | 特点 | 适用题材 |
|---------|------|---------|
| Happy Ending | 主角达成目标，所有伏笔回收 | 玄幻爽文、甜宠言情 |
| 苦涩胜利 | 赢了但付出了代价 | 都市现实、战争题材 |
| 开放式 | 留下想象空间 | 文学向、悬疑 |
| 反转结局 | 真相颠覆所有认知 | 推理、惊悚 |
| 悲剧 | 主角失败或死亡 | 虐心、史诗 |

结局不需要精确到每一步，但必须知道终点在哪。没有终点的长篇，就像没有目的地的旅行——走着走着就迷路了。

### 总纲模板

```
【书名】：____________________
【题材】：____________________
【字数目标】：____万字（约____章）
【一句话卖点】：____________________

【主角】：
  - 姓名：____
  - 起点状态：____（开局时的身份/能力/处境）
  - 终点状态：____（结局时的身份/能力/处境）
  - 核心欲望：____
  - 致命缺陷：____

【核心矛盾】：
  主角想要____，但____阻碍了他。
  为了突破阻碍，他必须____。
  但每次行动都会____，让局面更复杂。

【结局类型】：____
【结局概要】：____

【世界观关键词】：____（3-5个关键词概括世界观）

【全书节奏规划】：
  第一卷（第1-50章）：____
  第二卷（第51-100章）：____
  第三卷（第101-150章）：____
  ……
  最终卷：____
```

### 总纲示例

**都市重生：**

```
【书名】：重生之商业帝国
【题材】：都市重生 + 商战
【字数目标】：200万字（约500章）
【一句话卖点】：重生回到大学毕业那年，用二十年的记忆打造商业帝国

【主角】：
  - 起点状态：普通大学毕业，被公司裁员的中年社畜
  - 终点状态：身价千亿的商业帝国掌门人
  - 核心欲望：不再被命运摆布，活出自己想要的人生
  - 致命缺陷：前世的经历让他不信任任何人，容易错失真心

【核心矛盾】：
  主角想用重生记忆快速崛起，但前世蝴蝶效应让很多"确定的事"变得不确定。
  为了保住优势，他必须比前世更快布局，但每次激进操作都会引来更强对手。

【结局类型】：Happy Ending（事业巅峰 + 家庭圆满）
【全书节奏规划】：
  第一卷（1-50章）：创业起步，赚到第一桶金
  第二卷（51-100章）：商业扩张，打败第一个对手
  第三卷（101-150章）：资本博弈，进入大佬圈层
  第四卷（151-200章）：帝国成型，遭遇最强对手
  第五卷（201-250章）：终极商战，登顶
```

**玄幻升级：**

```
【书名】：万古第一神
【题材】：玄幻 + 升级流
【字数目标】：300万字（约700章）
【一句话卖点】：废柴觉醒上古血脉，从被退婚开始一路打穿九天

【主角】：
  - 起点状态：丹田破碎的家族废物，被未婚妻当众退婚
  - 终点状态：九天之上最强者，破碎虚空
  - 核心欲望：证明自己，保护在乎的人
  - 致命缺陷：骨子里的自卑让他容易被激怒，冲动行事

【核心矛盾】：
  主角想变强保护家人，但每次突破都引来更强势力觊觎他的血脉。
  血脉既是最大依仗也是最大祸根——越强，敌人越多。

【结局类型】：Happy Ending（登顶 + 团圆）
【全书节奏规划】：
  第一卷（1-50章）：家族篇——觉醒血脉，镇压家族内部
  第二卷（51-120章）：宗门篇——进入大宗门，崭露头角
  第三卷（121-200章）：大陆篇——走出宗门，挑战各方势力
  第四卷（201-300章）：域外篇——进入更高层世界
  第五卷（301-450章）：九天篇——终极对决
```

**悬疑推理：**

```
【书名】：第四面墙
【题材】：悬疑 + 心理惊悚
【字数目标】：80万字（约200章）
【一句话卖点】：一个能看见死者临终画面的刑警，追查连环杀手时发现自己就是凶手

【主角】：
  - 起点状态：市局刑警，有"看见死者记忆"的异能但从未告诉任何人
  - 终点状态：接受自己，与"另一个自己"和解
  - 核心欲望：找到真相，维护正义
  - 致命缺陷：过度理性导致压抑情绪，潜意识里的黑暗面是他最大的敌人

【核心矛盾】：
  主角想破案，但每个案件都指向他自己的过去。
  真相越近，他越不确定自己是猎人还是猎物。

【结局类型】：反转结局（真相反转，但逻辑自洽）
【全书节奏规划】：
  第一卷（1-30章）：三个独立案件，建立角色和能力
  第二卷（31-70章）：连环案出现，线索指向主角
  第三卷（71-120章）：真相逐渐揭开，主角崩溃边缘
  第四卷（121-180章）：终极真相——身份反转
  尾声（181-200章）：收束，与自己和解
```

**情感言情：**

```
【书名】：替身三年
【题材】：现代言情 + 先婚后爱
【字数目标】：60万字（约150章）
【一句话卖点】：替嫁给冷面总裁三年，离婚当天他却红了眼

【主角】：
  女主起点状态：普通女孩，替姐姐嫁给豪门总裁
  女主终点状态：独立女性，有自己的事业和选择权
  男主起点状态：冷面总裁，心里有白月光
  男主终点状态：学会表达爱，放下执念

【核心矛盾】：
  女主想在这个假婚姻里保全尊严，但不知不觉动了真心。
  男主以为自己只爱白月光，却发现替身妻子才是真正懂他的人。

【结局类型】：Happy Ending（双向奔赴）
【全书节奏规划】：
  第一卷（1-30章）：替嫁入豪门，冷暴力与误解
  第二卷（31-60章）：日久生情，暧昧升温
  第三卷（61-90章）：白月光回归，信任崩塌
  第四卷（91-120章）：离婚拉扯，真相揭开
  第五卷（121-150章）：破镜重圆，真心告白
```

## 三、分卷纲编写（每卷30-50章的结构）

### 一卷 = 一个完整的故事弧线

分卷不是随便切的。每一卷都应该是一个相对完整的故事，有自己的起点、发展、高潮和结尾。

打个比方：总纲是一部电视剧的剧情梗概，分卷纲是每一季的故事线，章节纲是每一集的内容。

### 分卷的节奏模板

```
铺垫期（30%）→ 发展期（40%）→ 高潮期（20%）→ 余韵期（10%）

以50章一卷为例：
  铺垫期（第1-15章）：建立本卷矛盾，引入新角色/新势力
  发展期（第16-35章）：矛盾升级，主角遇到困难，多次尝试
  高潮期（第36-45章）：本卷最大冲突爆发，生死对决/真相揭露
  余韵期（第46-50章）：解决本卷矛盾，同时埋下下一卷钩子
```

各阶段的写作重点：

| 阶段 | 章节占比 | 核心任务 | 禁忌 |
|------|---------|---------|------|
| 铺垫期 | 30% | 建立新环境、新角色、新矛盾 | 不能太无聊（读者会弃书） |
| 发展期 | 40% | 推进矛盾、展示成长、制造波折 | 不能太拖沓（反复拉扯同一件事） |
| 高潮期 | 20% | 本卷最大冲突、最爽/最虐的场面 | 不能虎头蛇尾（高潮要够劲） |
| 余韵期 | 10% | 收束本卷、留钩子、给读者喘息 | 不能拖太长（3-5章足够） |

### 每卷开头怎么接上一卷的钩子

上一卷结尾抛出的钩子，这一卷开头必须接住。接的方式有三种：

**1. 直接接续型**
上一卷结尾：主角被人一掌拍下悬崖
这一卷开头：主角在悬崖底醒来，发现自己还活着

**2. 时间跳转型**
上一卷结尾：主角说"三年后，我会回来"
这一卷开头：三年后，主角站在城门前

**3. 视角切换型**
上一卷结尾：主角被困在秘境里
这一卷开头：切换到配角视角，讲配角在主角消失后的处境

### 每卷结尾怎么抛出下一卷的钩子

**三种经典卷末钩子：**

| 钩子类型 | 模板 | 效果 |
|---------|------|------|
| 格局扩大型 | "你以为打败了大BOSS？其实他只是小弟" | 让读者期待更大的世界 |
| 危机升级型 | "赢了这场仗，但代价是……" | 让读者担心主角的命运 |
| 身份揭示型 | "他摘下面具，露出了一张所有人都认识的脸" | 让读者震惊并想看后续 |

### 分卷纲模板

```
【卷名】：第X卷 · ____
【章节数】：第__章 - 第__章（共__章）
【本卷任务】：一句话说清这一卷要完成什么
【本卷核心矛盾】：____

【出场角色】：
  - 主角（本卷状态变化）：从____到____
  - 新角色A：____（作用：____）
  - 新角色B：____（作用：____）

【阶段规划】：
  铺垫期（第__-__章）：
    - 第__-__章：____
    - 第__-__章：____
  发展期（第__-__章）：
    - 第__-__章：____
    - 第__-__章：____
  高潮期（第__-__章）：
    - 第__-__章：____
  余韵期（第__-__章）：
    - 第__-__章：____

【本卷伏笔】：
  埋设：____
  回收（来自上一卷的伏笔）：____

【卷末钩子】：____（为下一卷做什么铺垫）
```

### 分卷纲示例（玄幻 · 宗门篇）

```
【卷名】：第二卷 · 宗门篇
【章节数】：第51章 - 第120章（共70章）
【本卷任务】：主角进入大宗门，从底层弟子一路打到核心弟子
【本卷核心矛盾】：宗门内斗 + 主角血脉被觊觎

【出场角色】：
  - 主角：从家族天才→宗门底层弟子→核心弟子
  - 新角色·师姐：宗门天才，对主角态度从轻视→欣赏→暗生情愫
  - 新角色·反派师兄：天赋极高但心术不正，觊觎主角血脉
  - 新角色·师父：隐世高人，知道主角血脉的秘密

【阶段规划】：
  铺垫期（第51-70章）：
    - 第51-55章：主角进入宗门，被分到最差的外门
    - 第56-62章：外门修炼，展现天赋，引起注意
    - 第63-70章：外门大比，一鸣惊人，晋升内门
  发展期（第71-100章）：
    - 第71-80章：内门竞争，遭遇反派师兄打压
    - 第81-90章：秘境历练，获得关键资源
    - 第91-100章：师姐线推进 + 师父考验
  高潮期（第101-115章）：
    - 第101-108章：宗门大比，主角vs反派师兄
    - 第109-115章：血脉秘密暴露，宗门震动
  余韵期（第116-120章）：
    - 第116-118章：师父揭示更大世界的存在
    - 第119-120章：主角决定走出宗门，前往大陆

【本卷伏笔】：
  埋设：师父提到"远古战场"（第三卷回收）
  埋设：反派师兄逃脱，投靠域外势力（第四卷回收）
  回收：第一卷埋的"家族诅咒"——在宗门古籍中找到线索

【卷末钩子】：主角在宗门藏书阁发现一张残缺的地图，上面标注着"远古战场"的位置——那里据说封印着上古神兽的血脉
```

## 四、章节纲编写（每章2000-5000字的详细规划）

### 章节纲要包含什么

一个合格的章节纲需要包含以下要素：

| 要素 | 说明 | 示例 |
|------|------|------|
| 场景 | 在哪里发生 | 宗门演武场 / 古城茶楼 / 医院ICU |
| 角色 | 出场的角色及状态 | 主角（受伤未愈）、反派（洋洋得意） |
| 核心事件 | 本章最重要的事 | 主角带伤上台挑战反派 |
| 情绪走向 | 本章的情绪弧线 | 压抑→紧张→爆发→畅快 |
| 章末钩子 | 让读者翻页的理由 | 反派使出禁术，所有人震惊 |
| 伏笔动作 | 本章埋/收哪些伏笔 | 回收：主角的旧伤；埋设：反派的禁术来源 |

### 章节纲的"度"——不要太详细，也不要太笼统

**太详细的反面教材：**
```
第37章：
  开头：主角从演武场左侧入场，左脚先踏上台阶。
  第100字：对手出场，穿黑色长袍，腰间佩剑。
  第200字：两人对视，主角说"你来了"。
  第300字：对手冷笑回应。
  ……
```
这种写法把大纲变成了剧本，写作时完全失去乐趣和灵活性。写着写着你会觉得"我在抄大纲"而不是"我在创作"。

**太笼统的反面教材：**
```
第37章：
  主角和对手打了一架，赢了。
```
这种大纲等于没有。写的时候你会完全不知道从哪开始，写着写着就跑题了。

**合适的度：**
```
第37章：
  场景：宗门演武场
  角色：主角（带伤）、反派师兄、围观弟子、师父（暗中观察）
  核心事件：主角带伤挑战反派师兄。前半段被压制（展示实力差距），
           后半段爆发血脉力量反杀（展示成长）。
  情绪走向：悲壮→被压制的憋屈→爆发的畅快→胜利后的隐忧
  章末钩子：反派临死前使出禁术，禁术来源是域外势力——暗示更大的危机
  伏笔回收：第25章埋的"旧伤隐患"在本章爆发后被师父治好
  伏笔埋设：禁术的来源（域外势力渗透宗门）
```

这种程度的规划，既给了方向，又留了发挥空间。

### 章节纲模板

```
【第__章 · 章名】
【预计字数】：__字
【场景】：____
【出场角色】：____
【核心事件】：____
【情绪走向】：____→____→____
【章末钩子】：____
【伏笔】：埋设____ / 回收____
【备注】：____（可选，写一些特殊注意事项）
```

### 章节纲示例（悬疑推理 · 第42章）

```
【第42章 · 第三个受害者】
【预计字数】：3500字
【场景】：犯罪现场（废弃工厂）→ 警局审讯室
【出场角色】：主角刑警、搭档小赵、法医老周、嫌疑人甲
【核心事件】：发现第三个受害者。与前两个不同，这次受害者手里握着一张
            照片——照片上是主角十年前的样子。主角的过去开始浮出水面。
【情绪走向】：例行勘察的冷静→发现照片的震动→压抑的恐惧→强装镇定
【章末钩子】：搭档小赵无意中说"这个受害者的死法和十年前那个悬案一模
            一样"——而十年前那个案子，主角曾经是嫌疑人
【伏笔】：埋设——照片上的拍摄地点（后面会是关键线索）
          回收——第20章主角说"我十年前离开过这个城市"（现在知道原因了）
【备注】：本章不要让主角的情绪外露太多，他应该越害怕越冷静。读者能
         从细节（手抖、反复看照片）感受到他的恐惧，但对话中不表现。
```

### 章节纲的修改——写到中途发现规划不对怎么办

这是非常正常的事。大纲不是圣旨，是活的文档。

**修改原则：**

1. **小改随时改**：某一章的情节顺序调换、某个角色的台词方向变了——直接改，不需要犹豫
2. **中改看关联**：要删一个角色或改一条线的走向——先检查这个改动影响了哪些后续章节的大纲，全部连带修改
3. **大改回分卷**：发现整条线都不对——回到分卷纲层面重新规划，不要在章节纲上缝缝补补

**什么时候该改大纲：**
- 写着写着，某个角色突然比大纲里写得更鲜活——可以给他加戏
- 某个情节写着感觉不对——可能是大纲的逻辑有漏洞
- 读者反馈说某个线没意思——考虑砍掉或压缩
- 自己写烦了——可能是大纲规划的节奏有问题

**什么时候不该改大纲：**
- 一时冲动觉得"这样写更爽"——先冷静一天再决定
- 某个新灵感和主线完全无关——记下来当素材，不要硬塞
- 写到难处想绕过去——这是逃避不是创新

## 五、伏笔布置与大纲的关系

### 伏笔要在大纲阶段就埋好

伏笔不是写着写着突然想到"这里加个伏笔吧"。好的伏笔在总纲阶段就应该规划好。

**为什么要在大纲阶段规划伏笔？**

因为伏笔需要精确的"时间差"。埋得太早读者忘了，埋得太晚读者觉得突兀。只有在大纲层面通盘考虑，才能让伏笔的埋设和回收时机恰到好处。

### 伏笔的三种跨度

| 类型 | 跨度 | 特点 | 示例 |
|------|------|------|------|
| 短期伏笔 | 3-10章内回收 | 制造小悬念，维持日常追更兴趣 | 第15章主角捡到一把钥匙，第20章用它打开了密室 |
| 中期伏笔 | 1-3卷内回收 | 构建故事深度，让读者有"原来如此"的快感 | 第一卷提到家族秘史，第三卷揭开真相 |
| 长期伏笔 | 贯穿全书 | 构建世界观厚度，结局时集中引爆 | 第1章的神秘老人，最终章揭示是未来的主角自己 |

### 伏笔的密度控制

**每10章最多埋3个新伏笔。** 超过这个数量，读者会信息过载，记不住也懒得记。

伏笔的埋设和回收要保持平衡：

```
理想的伏笔节奏：

第1-10章：  埋设3个伏笔（A、B、C）
第11-20章：埋设2个（D、E），回收1个（A）
第21-30章：埋设2个（F、G），回收2个（B、D）
第31-40章：埋设1个（H），回收3个（C、E、F）
……
```

原则：**不要只埋不收，也不要只收不埋。** 保持进出自如的状态。

### 伏笔追踪表

建议用表格追踪所有伏笔的状态：

```
伏笔追踪表

| 编号 | 埋设位置 | 伏笔内容 | 类型 | 回收位置 | 状态 |
|------|---------|---------|------|---------|------|
| F001 | 第1章 | 主角左手腕的疤痕 | 长期 | 第180章 | 埋设中 |
| F002 | 第3章 | 神秘老人的话 | 长期 | 第200章 | 埋设中 |
| F003 | 第8章 | 配角的异常举动 | 短期 | 第14章 | 已回收 |
| F004 | 第15章 | 古老地图的残片 | 中期 | 第89章 | 埋设中 |
| F005 | 第20章 | 反派的禁术来源 | 中期 | 第105章 | 埋设中 |
```

**伏笔追踪表的使用方法：**

1. **每写完一章**，检查本章是否需要更新追踪表
2. **每10章回顾一次**，检查是否有伏笔该回收但还没回收
3. **每卷结束时**，全面检查：所有短期伏笔是否已回收？中期伏笔进度如何？
4. **全书结束前**，确认所有伏笔都已回收或明确标注"故意不回收"（开放式伏笔）

## 六、多线叙事的大纲设计

### 主线 + 副线怎么安排

一本长篇通常有：
- **1条主线**：主角的核心目标（变强/破案/追到女主）
- **1-3条副线**：推动主线或丰富世界观的支线

```
主线：主角追查连环杀手
  ├─ 副线A：主角和搭档的信任危机
  ├─ 副线B：杀手视角（每隔几章切换）
  └─ 副线C：十年前的旧案真相
```

### 副线不能比主线精彩

这是多线叙事最常见的翻车：副线写着写着太精彩了，读者反而觉得主线无聊。

**检查方法：** 把主线和副线分别独立出来，问自己——"如果只写这条线，读者会不会追？"如果副线的追读欲望比主线强，说明主线需要加强，或者副线需要弱化。

**副线的功能定位：**

| 副线功能 | 说明 | 示例 |
|---------|------|------|
| 补充世界观 | 展示主线看不到的世界 | 主线在城内打怪，副线展示城外的势力格局 |
| 推进感情线 | 主线打怪升级，副线谈恋爱 | 主线在宗门修炼，副线是和师姐的感情发展 |
| 反派视角 | 从敌人角度展示信息 | 主线追查凶手，副线展示凶手的动机和计划 |
| 配角成长 | 让配角不沦为工具人 | 主线主角冒险，副线配角在后方的成长 |

### 副线的切换节奏

```
推荐节奏：每3-5章切一次副线

示例（主线A + 副线B）：
  第1章：A
  第2章：A
  第3章：A
  第4章：B（切换视角，展示另一条线）
  第5章：A
  第6章：A
  第7章：A
  第8章：B
  ……
```

切换的时机选择：
- **主线留悬念时切走**：主角刚发现关键线索，切换到副线——读者会急着切回来
- **副线推进到关键点时切回**：副线角色发现了重要信息，切换回主线——用信息差制造张力
- **不要在高潮中间切**：主线打到最激烈的时候突然切副线，读者会骂人

### 多线交汇的时机设计

所有副线最终要和主线交汇。交汇的方式：

**1. 信息交汇型**
副线获取的关键信息，为主线的突破提供钥匙。比如副线角色发现的线索，正好是主线破案的最后一块拼图。

**2. 人物交汇型**
副线中的角色在关键时刻出现在主线。比如一直在副线中成长的配角，在主线大决战时作为援军登场。

**3. 因果交汇型**
副线中发生的事件，成为主线转折的原因。比如副线中反派暗中布局，导致主线中主角陷入陷阱。

交汇不能太早（副线还没展开就汇入主线，浪费了铺垫），也不能太晚（读者已经忘了副线在说什么）。

## 七、大纲常见错误

### 错误1：大纲太详细（变成剧本了，写作没乐趣）

**症状：** 大纲写了每一段的对话走向、每一个动作、每一句台词的大概意思。

**后果：** 写作变成了"抄大纲"，完全没有创作的乐趣和惊喜。而且写出来的东西读起来像剧本的扩写，缺乏灵气。

**改正：** 大纲只规划"发生什么"，不规划"怎么写"。留出至少30%的发挥空间。

```
错误示范：
  第17章：主角走进房间，看到桌上有封信。他拿起信，拆开，
  读了一遍，脸色变了。他说："怎么会这样？"然后把信摔在
  桌上。

正确示范：
  第17章：主角发现一封改变一切的信（内容揭示父亲的真面目）。
  情绪走向：好奇→震惊→愤怒→悲伤。章末：主角决定去质问父亲。
```

### 错误2：大纲太笼统（写着写着跑题了）

**症状：** 大纲每章只有一句话，甚至每卷只有一句话。

**后果：** 写到第10章就不知道第11章该写什么了。回头一看，前10章已经偏离了主线5章的距离。

**改正：** 大纲至少要有分卷级别的详细程度，关键章节需要有章节纲。

### 错误3：前20章规划完美，后面随缘（头重脚轻）

**症状：** 开头20章的大纲写得密密麻麻，后面100章只有几句笼统描述。

**后果：** 前20章精彩绝伦，后面逐渐失控。这是最常见的"烂尾"原因。

**改正：** 大纲的详细程度应该均匀分布。如果时间不够，宁可前面粗一点后面细一点——因为后面的变数更大，更需要规划。

### 错误4：没有留"弹性空间"（写到一半发现大纲不合理但改不了）

**症状：** 大纲精确到每章每个事件，没有任何变通余地。

**后果：** 写到中途发现某个情节逻辑不通，但因为"大纲已经定好了"不敢改，硬着头皮写下去，越写越别扭。

**改正：** 大纲只规划80%的内容，留20%的空间给即兴发挥和调整。每个分卷纲的结尾都要标注"可调整区域"。

### 错误5：主线不清楚（写着写着变成了群像剧）

**症状：** 大纲里出现了太多角色的故事线，每条线都写了几十章，分不清谁是主角。

**后果：** 读者困惑"这本书到底讲谁的故事"。每个人的线都写了一半，没有一条线写透。

**改正：** 大纲中每条线都要标注"为主线服务的方式"。如果一条副线和主线没有因果关系，砍掉或者压缩到5章以内。

### 错误6：高潮太早（前30章就打完boss了）

**症状：** 大纲把最大的高潮放在了全书的前1/3。

**后果：** 高潮之后没有更大的目标，读者觉得"已经看完了"，弃书。

**改正：** 全书的终极高潮应该在最后1/5。前面的高潮都是"阶段性高潮"，每次高潮后都要揭示更大的世界/更强的敌人/更深的真相。

```
正确的高潮分布（200章为例）：
  第30章：阶段性高潮（小BOSS战）
  第80章：阶段性高潮（中BOSS战）
  第130章：阶段性高潮（大BOSS战）
  第170章：终极高潮前奏
  第190章：终极高潮
  第200章：收束
```

### 错误7：结局没想好就开始写（越写越烂尾）

**症状：** "先写着吧，到时候自然就知道怎么结了"。

**后果：** 写到200万字还不知道怎么收，最后要么草草了事，要么强行大团圆，要么断更跑路。

**改正：** 写第一个字之前就要知道最后一个字写什么。结局可以不精确，但方向必须有。

### 错误8：伏笔埋了忘了收

**症状：** 前面埋了一堆伏笔，写着写着忘了，到结局还有一堆悬念没解释。

**后果：** 读者会翻回去找伏笔，然后发现作者自己都忘了——信任崩塌。

**改正：** 用伏笔追踪表。每埋一个伏笔就记下来，定期检查。全书结束后做一次"伏笔清查"。

## 八、不同题材的大纲特点

不同题材的大纲，骨架的搭建方式完全不同。

### 玄幻/升级流：大纲以"等级体系"为主线骨架

```
大纲骨架 = 等级阶梯

练气 → 筑基 → 金丹 → 元婴 → 化神 → 大乘 → 渡劫 → 飞升

每个等级对应：
  - 一个新地图（家族→宗门→城市→大陆→域外→仙界）
  - 一批新敌人（家族对手→宗门天才→城主→域主→仙人）
  - 一次核心能力解锁（血脉觉醒→功法突破→领悟大道）
```

**大纲重点：** 每次升级的"仪式感"要够，升级前的铺垫要足，升级后的验证场景要爽。不能平A升级——必须有代价、有风险、有对手。

**节奏公式：** 压迫（被欺负/遇强敌）→ 积累（修炼/获机缘）→ 爆发（突破/反杀）→ 更大压迫

### 悬疑/推理：大纲以"谜题设计"为主线骨架

```
大纲骨架 = 谜题层次

表层谜题：谁是凶手？
  └─ 中层谜题：凶手为什么要这么做？
       └─ 深层谜题：真相背后还有什么真相？
```

**大纲重点：** 先设计谜底，再倒推线索的分布。每条线索放在哪一章、由谁发现、以什么方式呈现，都需要提前规划。

**节奏公式：** 发现问题→收集线索→推理→碰壁→新线索→重新推理→真相

**特别注意：** 推理小说的大纲必须保证逻辑自洽。写完大纲后做一次"逻辑审查"——从谜底倒推，每一步推理是否有漏洞。

### 情感/言情：大纲以"关系阶段"为主线骨架

```
大纲骨架 = 关系阶梯

相遇 → 吸引 → 试探 → 误解 → 冲突 → 分离 → 重逢 → 表白 → 考验 → 圆满

每个阶段对应：
  - 一个关系节点（第一次见面/第一次心动/第一次吵架/第一次分开）
  - 一个情感转折（误会/真相/信任/背叛）
  - 一个配角作用（助攻/搅局/催化剂）
```

**大纲重点：** 关系推进要有"理由"——为什么这两个人会在一起，为什么他们会分开，为什么他们会重新在一起。每一步都要有因果，不能硬凑。

**节奏公式：** 甜蜜→误解→虐心→和好→更大的危机→更深的羁绊

### 都市/日常：大纲以"冲突事件"为主线骨架

```
大纲骨架 = 事件链

创业事件 → 竞争事件 → 危机事件 → 转型事件 → 帝国事件

每个事件对应：
  - 一个商业/社会矛盾
  - 一次决策（展示主角的智慧/魄力）
  - 一次格局扩大（从个体户→老板→企业家→商业帝国）
```

**大纲重点：** 都市文的核心是"真实感"——商业逻辑要合理，人际关系要真实。大纲中的每个事件都要有现实原型或合理推演。

**节奏公式：** 遇到问题→想出办法→执行→遇到新问题→更大的格局

### 末日/生存：大纲以"资源/威胁"为主线骨架

```
大纲骨架 = 生存阶梯

个人生存 → 小队生存 → 据点生存 → 区域争霸 → 文明重建

每个阶段对应：
  - 一组资源需求（食物→武器→药品→能源→人口）
  - 一组威胁升级（丧尸→变异体→敌对势力→天灾→人祸）
  - 一次组织进化（独狼→小队队长→据点领袖→区域霸主→新文明缔造者）
```

**大纲重点：** 资源的"紧张感"要贯穿始终——永远不够用，永远有新的威胁消耗资源。不能让主角太舒服。

**节奏公式：** 获得资源→消耗资源→更大的威胁→寻找更多资源→资源危机→突破

## 附录：大纲编写的实操流程

如果你是第一次写长篇大纲，按这个流程走：

```
第一步：写总纲（半天到一天）
  → 完成核心三问：卖什么、核心矛盾、结局
  → 填完总纲模板

第二步：拆分卷（每个分卷半天）
  → 确定全书分几卷
  → 每卷的核心任务和矛盾
  → 每卷的钩子设计

第三步：写分卷纲（每卷1-2天）
  → 按铺垫→发展→高潮→余韵拆分
  → 标注出场角色和伏笔
  → 确保卷与卷之间的衔接

第四步：写章节纲（每章10-15分钟）
  → 先写前30章的详细章节纲
  → 后面的章节纲可以在写到之前再补充
  → 保持"提前30章"的节奏

第五步：建立伏笔追踪表
  → 把所有伏笔录入表格
  → 每10章检查一次
```

**时间参考：**
- 200万字玄幻：总纲1天 + 分卷纲3天 + 章节纲（滚动补充）= 约1周
- 80万字悬疑：总纲半天 + 分卷纲2天 + 章节纲（全部提前写完，因为推理需要逻辑严密）= 约5天
- 60万字言情：总纲半天 + 分卷纲1天 + 章节纲（滚动补充）= 约3天

大纲不是一次性的工作。它是活的文档，随着写作的推进不断调整、修正、完善。好的大纲是和正文一起成长的。
