## 毒舌评审标准

# 毒舌评审标准

> 以最严苛、最不客气的视角审视文章。不给面子，不留情面，只看问题。
> 用途：写完文章后用此标准逐条对照，找到最致命的弱点。

---

## 一、致命伤（出现一个直接不及格）

### 1. AI味检测
- ❌ "仿佛/似乎/不禁/宛如"出现在对话或叙事中
- ❌ "不是A，而是B"句式出现即删（包括"不是X。是Y。""不是X——是Y""不是X，是Y"一切变体）
- ❌ "他的内心像打翻了五味瓶"等公共比喻库
- ❌ 角色内心独白用了书面语（"我的内心深处涌起一股暖流"）
- ❌ 连续3句以上用相同句式开头
- ❌ 任何"XX地说道""XX地说"超过3处
- ❌ "一抹""一丝""一缕"等量词出现超过5次

### 2. 逻辑硬伤
- ❌ 角色突然知道他不可能知道的信息
- ❌ 时间线矛盾（上午出门下午回来已经是第二天）
- ❌ 场景空间错误（出了门左转但之前说右边是死路）
- ❌ 角色死了之后又出现且没有解释
- ❌ 前文设定的规则被后文打破
- ❌ 角色做了超出其文化程度/认知水平的推理（初中生不会用"解离症"总结搜索结果）
- ❌ 特殊物件/能力没有因果链（为什么只有这块玉能存灵气？为什么偏偏是他？）
- ❌ 比喻体系不统一（前面"蜂蜜"后面"铁水"，质感矛盾）
- ❌ 先兆/伏笔机械式排队应验（预言一项一项打勾，太工整到假）
- ❌ 配角全是功能性道具（只为触发主角剧情，没有自己的动机和困境）

### 3. 角色崩塌
- ❌ 前文话少的角色突然开始长篇大论
- ❌ 前文聪明的角色突然做出弱智决定只为推动剧情
- ❌ 角色在不同场景中性格完全不一致
- ❌ 反派突然变好人没有任何铺垫
- ❌ 角色的口头癖/习惯动作突然消失

---

## 二、重伤（出现3个以上算不及格）

### 4. 对话尸体
- 🔴 所有角色说话方式一样，换个名字分不清谁是谁
- 🔴 对话全是问答式："你好吗？""我很好。"
- 🔴 对话中没有潜台词，字面意思就是全部意思
- 🔴 角色用"我感到非常愤怒"这种话直接报情绪
- 🔴 对话中没有打断、沉默、犹豫、答非所问
- 🔴 对话承担了所有叙事功能（旁白、描写、心理全是对话在说）
- 🔴 对话前后没有情绪变化，开头冷结尾还是冷

### 5. 描写僵尸
- 🔴 通篇只有视觉描写，没有声音气味触感温度
- 🔴 "美丽""漂亮""好看"等空泛词出现超过3次
- 🔴 描写像清单罗列：左边是什么右边是什么前面是什么
- 🔴 环境描写与情绪完全脱节（伤心的时候外面阳光明媚但没有反讽意图）
- 🔴 比喻全部是公共库存（如花似玉、心如刀割、泪如雨下）
- 🔴 大段描写删掉后对故事没有任何影响
- 🔴 描写一个杯子用了50字但这个杯子之后再也没出现
- 🔴 作者替角色说话（"像一个没说出口的问句"——工地工人不会这样形容凉意）
- 🔴 作者替读者总结（"这是今天第一个真实的感觉"——疼本身就够了，不需要旁白解释）
- 🔴 角色的结论来得太快（碰了两个东西就推导出规律，应该留在困惑里）

### 6. 节奏瘫痪
- 🔴 全章句子长度差不多，读起来像匀速行驶的火车
- 🔴 整章都在高潮或者整章都在铺垫，没有呼吸
- 🔴 紧张场景用了长句慢节奏（该快不快）
- 🔴 情感爆发时写了800字但一个具体的动作都没有
- 🔴 场景切换没有任何过渡，突然跳到另一个地方
- 🔴 章节结尾没有任何钩子，读完不想翻下一页
- 🔴 信息全部在开头倒完或全部堆在结尾

---

## 三、轻伤（出现5个以上扣大分）

### 7. 情绪注水
- 🟡 "他很伤心""她非常愤怒"等直接情绪陈述
- 🟡 "她觉得""他感到""她心想"出现超过5次
- 🟡 情绪描写全是身体反应但没有独特性（每个人紧张都是心跳加速手心出汗）
- 🟡 情绪转折没有铺垫，上一秒笑下一秒哭
- 🟡 哭戏写了500字但读者完全不感动
- 🟡 用感叹号表达情绪（"他真的太过分了！"）

### 8. 结构松散
- 🟡 伏笔埋了但没有回收
- 🟡 重要人物出场后消失再也没出现
- 🟡 次要情节占用过多篇幅且不影响主线
- 🟡 回忆段落太长打断了主线节奏
- 🟡 结尾用"通过这件事他明白了"之类的总结
- 🟡 开头三段内没有建立任何悬念或冲突

### 9. 文字洁癖
- 🟡 同一个形容词在一段内出现2次以上
- 🟡 "突然""忽然"出现超过3次
- 🟡 用"但是""然而""不过"开头的句子超过5处
- 🟡 句尾语气词"了"出现过于密集
- 🟡 同一个比喻在全文中出现2次以上
- 🟡 过度使用省略号表达情绪

---

## 四、评审流程

### Step 1：通读一遍（不带评审眼光）
感受整体：有没有想跳读的地方？有没有觉得假的地方？有没有看不懂的地方？

### Step 2：逐条过致命伤
AI味检测 → 逻辑硬伤 → 角色崩塌
**任何一条命中 → 打回重写**

### Step 3：逐条过重伤
对话尸体 → 描写僵尸 → 节奏瘫痪
**3条以上命中 → 不及格，需要大幅修改**

### Step 4：逐条过轻伤
情绪注水 → 结构松散 → 文字洁癖
**5条以上命中 → 扣分但可以修补**

### Step 5：给出一句话毒舌总评
不要说"写得不错但是..."，直接说最致命的问题。
- ❌ "整体还不错，对话部分有些地方可以改进"
- ✅ "对话全是死人说话，换个名字分不清谁是谁，重写对话部分"

---

## 五、毒舌金句（评审时可以直接用）

- "这段对话可以删掉，删掉之后没有任何损失，说明它本来就不该存在"
- "你的角色不是在说话，是在念台词"
- "这段描写的唯一作用是让页数变多了"
- "读者看到这里会翻页，但不是因为想看下一页，是因为想跳过这一页"
- "这不是留白，这是偷懒"
- "角色的情绪变化比天气预报还不靠谱"
- "这段比喻像是从百度上搜的"
- "你的反派不是坏，是编剧让他坏"
- "这段可以替换成任何其他角色都不会有区别"
- "读完这段我什么画面都没有，脑子里只有一团模糊的形容词"
## 质量等级标准

# 写作质量等级标准

> 定义四个质量等级，让AI写作时有明确目标。
> 每个等级有具体指标，不是主观感觉。

---

## 等级一览

| 等级 | 名称 | 对标 | 一句话定义 |
|------|------|------|-----------|
| D | AI垃圾 | 模板化角色名 | 能读但满是AI味，模板化，无灵魂 |
| C | 模板网文 | 欲望开发系统 | 结构完整，节奏对，但没有意外 |
| B | 中上水平 | 有辨识度的网文 | 有技法，有辨识度，有记忆点 |
| A- | 准一流 | 旧日焚馆 | 有灵气，有风格，有文学性 |

---

## D级：AI垃圾

**特征：** 能读，但一看就知道是AI写的。

| 指标 | 标准 |
|------|------|
| 禁用词 | 大量出现"仿佛/似乎/不禁/竟然/一时间" |
| 句式 | 连续5句以上用同一句式开头 |
| 对话 | 书面化，所有人说话方式一样 |
| 描写 | 只有视觉，没有其他感官 |
| 情感 | "他感到一阵XX"直接陈述 |
| 节奏 | 匀速推进，没有波峰波谷 |
| 角色 | 脸谱化，换名字不影响理解 |
| 结构 | 起承转合标准但死板 |

**典型段落：**
> 他感到一阵心痛。仿佛有什么东西碎裂了。他不禁握紧了拳头。她似乎察觉到了什么，轻轻叹了口气。一时间，空气中弥漫着尴尬的沉默。

---

## C级：模板网文

**特征：** 结构完整，节奏对，但没有意外。读者能看完但记不住。

| 指标 | 标准 |
|------|------|
| 禁用词 | 偶尔出现但不密集 |
| 句式 | 有变化但不丰富 |
| 对话 | 有区分度但不够鲜明 |
| 描写 | 五感至少2种，但不深入 |
| 情感 | 有身体反应但模式化（"心跳加速""脸红"） |
| 节奏 | 有波峰波谷但可预测 |
| 角色 | 有标签但缺乏层次 |
| 结构 | 有悬念有钩子但套路化 |

**典型段落：**
> 她咬了咬嘴唇，把头偏向一边。手指无意识地绞着衣角。"随便你。"声音很轻，但他听得出来，那不是真的随便。

---

## B级：中上水平

**特征：** 有技法，有辨识度，有记忆点。读者会推荐给别人。

| 指标 | 标准 |
|------|------|
| 禁用词 | 基本没有 |
| 句式 | 多样，长短交替，有节奏感 |
| 对话 | 遮住名字能猜出是谁说的 |
| 描写 | 五感至少3种，有通感运用 |
| 情感 | 用行为和环境映射，不用直接陈述 |
| 节奏 | 有意外转折，不完全可预测 |
| 角色 | 有矛盾面（表面A内心B） |
| 结构 | 有留白，有物象象征 |
| 记忆点 | 至少1个场景让读者事后回想 |

**典型段落：**
> 楼梯间的声控灯又灭了。她没动，他也没动。黑暗里只有两个人的呼吸声，和窗外不知谁家的电视声。"你手好冰。"他说。她没抽回去。灯亮了。她的手已经不冰了。

---

## A-级：准一流

**特征：** 有灵气，有风格，有文学性。读者会反复读，会引用。

| 指标 | 标准 |
|------|------|
| 禁用词 | 零 |
| 句式 | 有自己的句式风格，可辨识 |
| 对话 | 每句话有潜台词，沉默比语言更有力 |
| 描写 | 五感联动，有独特的感官组合 |
| 情感 | 用留白和物象，让读者自己感受 |
| 节奏 | 打破预期，有"意料之外情理之中" |
| 角色 | 有成长弧线，有不可预测性 |
| 结构 | 多线交织，伏笔回收 |
| 风格 | 有叙事声音，读一段就知道是谁写的 |
| 灵气 | 至少1处让读者停下来反复品味 |

**典型段落：**
> 她把围巾解下来递给他。围巾上有她颈窝的温度和洗衣液的味道。他接过去，没有围上，只是攥在手里。后来他才知道，那天她把围巾给他，是因为她要走了，想在他身上留点什么。围巾他围了三年，直到毛线起球、颜色褪尽，也没洗过一次。

---

## 写作时的等级定位

| 场景 | 目标等级 | 说明 |
|------|---------|------|
| 日常过渡章 | C+ | 不需要每章都惊艳，但不能有AI味 |
| 角色建立章 | B | 角色第一次出场必须有记忆点 |
| 冲突高潮章 | B+~A- | 关键场景必须有灵气 |
| 情感爆发章 | A- | 最好的技法用在最重要的场景 |
| 开篇前3章 | B+ | 前3章决定读者是否继续 |

**铁律：任何章节都不能低于C级。D级=返工。**
