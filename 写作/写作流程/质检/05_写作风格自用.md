## 写作风格_自用

# 写作风格：自用

> 本风格为我个人最爱的海明威口味(也带点我自己的笔风)，非通用写作标准(觉得好，就是对海明威作品的认同)。
> 使用者可根据自身喜好调整或替换为自己的笔风文件。
> 如需自定义，复制本文件后修改各节规则即可。
>
> **禁用词表、替换方案、标杆示范详见 写作/写作流程/质检.md（质检索引）。**

> 干净利落+神秘。表面克制平静，底下暗流涌动，结尾留白收束。

---

## 一、风格DNA

**一句话：** 每个字都得干活，不干活的字滚蛋。说七分留三分，让读者自己把那三分补上。

| 特征 | 说明 |
|------|------|
| 干净 | 没有废话句、没有装饰词、没有"然而/不禁/仿佛" |
| 利落 | 句子短、动词狠、节奏快、不拖泥带水 |
| 神秘 | 不解释、不总结、不做情绪升华、结尾留白 |
| 克制 | 写悲伤不写"悲伤"二字，写恐惧不写"害怕"二字 |
| 口语 | 对话像真人说话，有虚词（还/挺/倒是），有省略 |
| 推理 | 读者跟着主角一起推理，不提前告诉答案 |
| 反差 | 主角说话极简，其他角色正常说话，用反差制造辨识度 |

## 二、句式风格

### 2.1 句子长度

**偏短。** 平均句长12字以内。长句不超过25字。

```
❌ 他感到一阵恐惧，心跳加速，手心出汗，脑海中一片空白，仿佛整个世界都安静了下来。
✅ 他手心出了汗。心跳加速。脑子空白。
```

### 2.2 段落长度

**短段为主。** 50字以内的段落占70%以上。150字以上的段落一章不超过3个。

极短段（1-15字）是杀手锏：

```
"🌿。"
"算了。"
"他走了。"
"第三天也没来。"
```

### 2.3 句式偏好

| 偏好 | 说明 |
|------|------|
| 动作开头 | "他抄起椅子腿就砸过去了。" — 不是"他很愤怒，于是抄起椅子腿" |
| 感官开头 | "甜腻的腐烂味。" — 不是"空气中弥漫着一股腐烂的味道" |
| 无主句 | "冷。""灯灭了。""砰的一声。" — 五个字以内解决 |
| 省略收尾 | "看情况。""也就那样吧。""……算了。" |

### 2.4 句式禁忌

| 禁忌 | 原因 |
|------|------|
| "他感到一阵XXX" | 直接陈述情绪，没有画面 |
| "仿佛/似乎/不禁" | 犹豫、模糊、不干脆 |
| "不是A，是B"及其一切变体 | 全面禁用 |
| "然后"不加破折号做场景跳转 | 生硬。"然后——"加破折号可用 |
| "然而/但是/不过"连续使用 | 转折太多像辩论赛 |
| "心中暗道" | 多余的引子 |
| 连续3句以"他/她/我"开头 | AI病态重复 |

## 三、叙事风格

### 3.1 视角纪律

**严格限制在一个人的视角里。** 不写"他不知道的是"——上帝视角的标志。

```
❌ 他不知道的是，门外站着三个人。
✅ 门外有声音。他没回头。
```

### 3.2 信息控制

**读者跟着主角一起推理，不提前告诉答案。** 信息碎片化呈现。

### 3.3 时间处理

**用模糊量词，不用精确数字。**

```
❌ 他等了四十分钟。
✅ 他等了差不多四十分钟。
✅ 他等了很久。
```

### 3.4 场景切换

```
❌ 然后我也死了。（生硬跳转）
✅ 然后——黑暗。（破折号制造停顿）
✅ 我把门关上。厨房里灯还亮着。（空行+动作过渡，首选）
```

### 3.5 物件代人

**用物件承载情感，物件的位置变化=角色的心理变化。**

**铁律：物件每次出场必须有新信息或新情绪。**

### 3.6 潜台词设计

**表面说A，实际指B。** 用日常物件和动作替代情感表达。

**铁律：潜台词的前提是上下文给够线索。** 没有线索的潜台词=故弄玄虚。

### 3.7 蒙太奇

**打碎时间线，让读者自己拼。** 每句话是一个画面，画面之间不用"然后""接着""之后"连接。

### 3.8 收尾风格

**不做总结，不做升华，不做抒情。** 用具体物件或一句话收尾。

**铁律：结尾最后一句不超过20字。**

## 四、对话风格

### 4.1 核心原则

**对话是人说的话，不是写的文章。** 每句话不超过15字。沉默比语言更有力。

### 4.2 口语化规则

| 规则 | 示例 |
|------|------|
| 加虚词（还/挺/倒是） | "刀倒是挺长" — 不是"刀很长" |
| 用省略号和破折号代替解释 | "剩下的明天给。你帮我个忙——" |
| 不用"说道/问道/回答道" | 直接写对话，动作另起一句 |
| 允许语病和不完整句 | "你这灯不是还亮着吗？" |
| 对话间有动作/沉默缓冲 | 不能连续5句以上无间隔对话 |

### 4.3 人物出场

**先声后人。** 先有声音/动作/动静，再写"看见了"。

```
❌ 他看见陈茵从考场出来。
✅ 他正低头看手机，听见有人喊了一声"陈茵你等会儿"，抬头就看见了她。
```

## 五、感官描写风格

### 5.1 优先级

触觉 > 听觉 > 嗅觉 > 视觉 > 味觉

### 5.2 规则

| 规则 | 示例 |
|------|------|
| 用具体物代替形容词 | "冰得扎手，冬天那种要舔舌头的冷" |
| 用通感代替比喻 | "她的声音是透明的" |
| 感官要递进 | 冷风→皮肤→骨头（从外到内穿透） |
| 不叠加超过3个感官 | 2-3个最好 |

---

## 六、情感描写风格

### 6.1 核心铁律

**禁止直接陈述情感。** 用身体反应、环境映射、行为变化来传递。

```
❌ 她很伤心。
✅ 嗓子发紧，眼眶烫得厉害，鼻腔酸涩。
✅ 雨不知道什么时候下起来了。
✅ 鼻息贴在一起，雨水滑落他的口中，很咸。
```

### 6.2 情感节奏

**压-压-压-放。** 先压抑，再压抑，再压抑，然后在一个不经意的瞬间释放——释放的方式越小，冲击力越大。

---

## 七、节奏风格

### 7.1 段落类型交替

描写(D)、对话(T)、动作(A)、内心(I) 四种类型交替。连续3段不能同类型。

### 7.2 长短段交替

连续3段不能同一长度区间。极短段（1-15字）用来打断节奏、制造冲击。

### 7.3 节奏曲线

```
开篇：中速（建立场景和角色）
中间：慢→快→慢（情绪波浪）
高潮：快→更快→突然停止（像心跳骤停）
收尾：慢，用一个安静的细节结束
```

## 八、副笔风：编年史（世界观/大纲专用）

> 写世界观、文明史、设定大纲时切换。用半文言的编年体。

| 特征 | 说明 |
|------|------|
| 时间锚点 | 用年份/纪元锚定事件 |
| 四字句+文言残留 | 半文言制造厚重感 |
| 引号给名词加冕 | 关键概念加引号 |
| 因果链推进 | 每句是上一句的后果 |
| 短句做刀 | 极短句砍在转折处 |
| 戛然而止 | 收尾不给希望 |

## 一、禁用词表

### 1.1 一级禁用（出现即删）

| 禁用词/短语 | 为什么禁 | 替换方案 |
|------------|---------|---------|
| 不是（叙述文中） | AI最爱的"不是A，是B"情绪升级句式，详见第七节全面禁令。"还"不能加——"她刚开始还以为是手套"会改变整个句子的味道 | 确定否定判断（"不是她的"）和对话中可用；叙述文中一律改写或删除 |
| 仿佛 | AI最爱的万能连接词，削弱真实感 | 删掉"仿佛"，直接写后面的比喻体 |
| 似乎 | 犹豫不决的语气，AI用来"保险" | 用肯定句或具体细节代替 |
| 不禁 | AI写情感的拐杖，"不禁XXX"全是废话 | 直接写动作，删掉"不禁" |
| 竟然/居然 | AI用来制造"惊讶"，但频率过高导致廉价 | 用具体反应代替（愣住、手一抖、话卡在嗓子眼） |
| 不由自主 | 和"不禁"同病 | 删掉，直接写动作 |
| 下意识 | AI写条件反射的万能词 | 用具体动作描述（手伸出去、身体往后缩） |
| 一时间 | AI转场的万能胶 | 删掉，直接写下一个动作或画面 |
| 心中暗道 | AI内心独白的引子 | 直接写内心独白，不需要"心中暗道" |
| 某种 | AI模糊化描述的帮凶 | 用具体名词代替 |
| 然后（场景切换时） | AI最爱的场景跳转连接词，"然后"一出场读者就错愕。**"然后——"加破折号可用**（"然后——黑暗。"有节奏感） | 用具体动作/感官/环境过渡，不用"然后"硬接 |
| 一阵（一阵心痛/一阵寒意/一阵暖流） | AI情感描写的偷懒方式 | 用具体身体反应代替 |
| 不知为何/不知道为什么 | AI解释动机的废话 | 删掉，要么给出具体原因，要么让读者自己猜 |
| 的确/确实 | AI用来"加强语气"的口头禅 | 删掉，语境本身就够强 |

### 1.2 二级禁用（同一章最多出现1次）

| 禁用词/短语 | 控制频率 |
|------------|---------|
| 不由得 | 全章最多1次 |
| 隐隐 | 全章最多1次 |
| 缓缓 | 全章最多1次 |
| 淡淡（淡淡一笑/淡淡地说） | 全章最多1次 |
| 深深（深深吸了口气/深深地看着） | 全章最多1次 |
| 轻轻 | 全章最多1次 |
| 默默 | 全章最多1次 |
| 隐约 | 全章最多1次 |
| 紧接着 | 全章最多1次 |
| 显然 | 全章最多1次 |
| 本能地 | 全章最多1次 |

### 1.3 高频废话句式（出现即删或改写）

| 废话句式 | 问题 | 改法 |
|---------|------|------|
| "他感到一阵XXX" | 直接告诉读者情绪，AI最大通病 | 改成身体反应：冷汗、手抖、心跳加速 |
| "XXX让他感到XXX" | 同上 | 改成动作+感官描写 |
| "他的心中充满了XXX" | 直接陈述内心，没画面感 | 改成具体想法或行为 |
| "目光中充满了XXX" | AI最爱的"情绪眼"写法 | 改成眼神的物理变化（瞳孔收缩、目光凝固） |
| "一种难以言喻的XXX" | 懒得写具体就说"难以言喻" | 必须写出具体是什么 |
| "他不知道的是" | AI上帝视角的标志 | 删除，让读者自己发现 |
| "仿佛整个世界都XXX" | AI渲染氛围的万能句 | 用具体的环境细节代替 |
| 叙述者评论/说教 | AI忍不住替读者总结。"挺讽刺的"可以用（角色视角的短评），但"盐在废土里还能换东西"不行（解释性说教）。**规则：角色视角的一句话感受OK，作者视角的解释性评论删掉。** | 删掉解释，留感受。"灾前超市货架上这种罐头摆一整排，没人买。"→读者自己品出讽刺。"挺讽刺的"→角色随口一句，不啰嗦，OK。 |

## 二、句式检查

### 2.1 开头多样性

**规则：连续3句不能用同一句式开头。**

常见的AI病态重复：

```
❌ 他走向门口。
   他推开了门。
   他看见了里面的场景。
   他感到一阵恐惧。

✅ 他走向门口。
   门缝里透出微弱的光。
   推开的瞬间，一股腐烂的甜味扑面而来。
   脚下的地板吱呀一声，像是在抱怨。
```

### 2.2 句式变化清单

每100字内，至少出现以下2种句式：

| 句式类型 | 示例 |
|---------|------|
| 短句（5字以内） | "没用。""他走了。""冷。" |
| 长句（20字以上，带从句/并列） | "路明非两手抄在裤兜里，抬着头看着树荫，觉得清闲而美好。" |
| 无主句 | "灯灭了。""砰的一声。""电话响了。" |
| 对话 | "走不走？""不走。" |
| 感官开头 | "一声尖叫从走廊尽头传来。""甜腻的血腥味钻进鼻子。" |
| 环境描写开头 | "下午的阳光从楼道尽头的窗户里照进来。" |

### 2.3 主语开头限制

连续3句不能以"他/她/我"开头。

替换方式：
- 用身体部位当主语："手指不自觉地攥紧了裤缝。"
- 用环境当主语："走廊里安安静静。"
- 用声音当主语："砰的一声，书掉在了地上。"
- 用无主句："冷。"

## 三、情感描写替换

### 3.1 核心原则

**禁止直接陈述情感。用身体反应、环境映射、行为变化来传递情绪。**

### 3.2 替换速查表

| 直接陈述（禁用） | 身体反应替换 | 环境映射替换 |
|----------------|-------------|-------------|
| 他感到一阵心痛 | 胸口像被人攥住，呼吸卡在喉咙里 | 窗外的风突然大了，吹得窗帘鼓起来 |
| 她非常害怕 | 冷汗浸透了后背，手指甲掐进掌心 | 走廊尽头的灯忽明忽暗 |
| 他很愤怒 | 后牙咬得咯吱响，太阳穴上的青筋跳了跳 | 桌上的杯子被他捏得变了形 |
| 她很悲伤 | 嗓子发紧，眼眶烫得厉害，鼻腔酸涩 | 雨不知道什么时候下起来了 |
| 他很紧张 | 手心全是汗，心跳快得像要撞破胸腔 | 时钟的滴答声突然变得很响 |
| 她很尴尬 | 耳根发烫，手指绞着衣角，目光不知道往哪放 | 空气像是凝固了 |
| 他很震惊 | 脑子嗡的一声，嘴张开又合上 | 时间好像停了 |

### 3.3 标杆示范

**【来源：龙族】**
> 他猛地坐起，在下午的阳光中睁开眼睛，呼吸急促，全身都是冷汗，外面是高架轻轨经过的噪音。

分析：不写"他从噩梦中惊醒感到恐惧"，而是用"猛地坐起"+"呼吸急促"+"全身冷汗"三个身体反应传递恐惧，再用"高架轻轨的噪音"把情绪从梦中拉回现实。

**【来源：重生诡异世界BOSS】**
> 孙超大口大口地喘着粗气，冷汗浸透了后背的西装，他颤抖着手，看向那页规则的目光里，充满了极致的恐惧。

分析：身体反应链条——喘粗气→冷汗浸西装→手颤抖→目光黏住。每一个都是具体的、可感知的。

## 四、对话去书面化

### 4.1 核心原则

**对话是人说的话，不是写的文章。要用口语。**

### 4.2 替换速查表

| 书面语（禁用） | 口语替换 |
|---------------|---------|
| 我认为我们应该 | 我觉得咱俩得 |
| 这件事情非常严重 | 这事儿大了 |
| 我对此表示怀疑 | 你搁这扯呢？ |
| 请你不要这样做 | 行了行了，别闹 |
| 我完全理解你的感受 | 懂，谁摊上都这样 |
| 这是一个很好的提议 | 成啊，这主意不错 |
| 我们需要重新考虑 | 哎不对，再想想 |
| 毫无疑问 | 肯定的/那还用说 |
| 因此/所以 | 那/那不就得了 |
| 然而/但是 | 不过/可/但 |
| 关于这件事情 | 这事儿/这个嘛 |
| 我无法接受 | 凭什么？/我不干 |
| 请你冷静下来 | 先别急/你消消气 |

### 4.3 对话中禁用的AI病句

| AI病句 | 问题 | 替换 |
|-------|------|------|
| "我不知道该如何形容这种感觉" | 太文艺 | 用口语化表达或直接沉默 |
| "这种感觉很奇怪" | 废话 | 删掉或用具体描述 |
| "你听我解释" | AI最爱的对话转折 | 换个更有性格的方式（"你先别走""给我两分钟"） |
| "这到底是怎么回事" | AI最爱的悬疑过渡 | 换角色个性化的表达（"你给我一个说法""哪儿出了岔子"） |
| "我相信你一定可以的" | AI最爱的鸡汤鼓励 | 用行动代替或用角色性格化语言 |

### 4.4 标杆示范

**【来源：龙族】**
> "骗哥们可以，别骗自己。哥们被你骗了真无所谓的，打个哈哈就过了。但希望你打完这段话后擦一下眼角，别让眼泪掉在网吧键盘上就行……"

分析：完全口语化，有互联网黑话，有人物性格（损友互怼），有画面感。

## 五、描写密度标准

### 5.1 最低标准

| 检查项 | 频率 | 不达标后果 |
|-------|------|-----------|
| 五感描写（视/听/嗅/触/味） | 每1000字至少1处 | 读起来像大纲不像小说 |
| 身体语言 | 每500字至少1处 | 角色像提线木偶，没有活人感 |
| 环境细节 | 每500字至少1处 | 场景像真空，读者没法沉浸 |

### 5.2 五感描写速查

| 感官 | 常用但AI化的写法 | 更好的写法 |
|------|----------------|-----------|
| 视觉 | "他看到了XXX" | "红色的灯光从门缝里渗出来，照在地板上像一道血痕" |
| 听觉 | "他听到了XXX" | "指甲划过黑板的声音，从后脑勺一路窜到牙根" |
| 嗅觉 | "空气中弥漫着XXX" | "甜腻的腐烂味，像烂掉的芒果" |
| 触觉 | "他感到XXX" | "金属扶手冰得扎手，冬天那种要舔舌头的冷" |
| 味觉 | "嘴里泛起XXX" | "嘴巴里发苦，像嚼了一把锈钉子" |

### 5.3 身体语言速查

| 情绪 | 身体语言（至少写2个） |
|------|---------------------|
| 紧张 | 搓手/咬嘴唇/频繁眨眼/肩膀僵硬/咽口水 |
| 愤怒 | 攥拳/咬牙/太阳穴青筋跳/鼻翼翕动/下巴绷紧 |

| 尴尬 | 摸鼻子/目光游移/耳根发红/干笑/搓后脑勺 |
| 开心 | 眉毛上扬/嘴角翘/走路带风/语速变快/拍大腿 |

## 六、节奏检查

### 6.1 核心规则

**不能连续3段都是同一类型的段落。描写、对话、动作、内心独白必须交替出现。**

### 6.2 段落类型标签

| 段落类型 | 定义 | 标记 |
|---------|------|------|
| 描写段 | 环境、外貌、氛围 | D |
| 对话段 | 人物说的话 | T |
| 动作段 | 人物做的事情（走路、打架、拿东西） | A |
| 内心段 | 人物的想法、回忆、感受 | I |

### 6.3 节奏自检

```
合法排列：D-T-A-I-D-T-A
非法排列：D-D-D-D（连续4段描写，读者会跳读）
非法排列：T-T-T-T（连续4段对话，像聊天记录）
非法排列：A-A-A-A（连续4段动作，没有喘息空间）
```

**最佳节奏**：DTADTIA 或 DTAITA，每个场景至少有3种以上段落类型。

### 6.4 长短段交替

| 段落长度 | 字数 | 作用 |
|---------|------|------|
| 极短段（1-15字） | "没用。""他跑了。" | 打断节奏，制造冲击 |
| 短段（15-50字） | 一句话的动作或对话 | 推进速度 |
| 中段（50-150字） | 带细节的描写或叙述 | 基本叙事单位 |
| 长段（150-300字） | 详细的环境/回忆/复杂动作 | 沉浸感，但要少用 |

**规则：连续3段不能都是同一长度区间。**

## 七、"不是A，是B"及其一切变体——全面禁令

> 这是AI写作最标志性的恶心句式。读者看到的那一刻就厌倦得看不下去。**叙述正文中出现任何变体，一律改写。**

### 7.1 禁用句式全家桶

以下所有句式**叙述正文中全部禁用**，不论怎么变形：

| 句式模板 | 示例 |
|---------|------|
| 不是A，是B | 不是恐惧，是绝望 |
| 不是A，而是B | 不是愤怒，而是心寒 |
| 不是A，更像是B | 不是疼痛，更像是什么东西在撕裂 |
| 不是A，不是B，是C | 不是害怕，不是紧张，是一种更深的不安 |
| 不是A，不是B，更像是C | 不是愤怒，不是悲伤，更像是被掏空了 |
| 与其说是A，不如说是B | 与其说是愤怒，不如说是恐惧 |
| A不是A，B不是B | 疼不是疼，麻不是麻，说不上来 |
| 那不是A，那是B | 那不是雨水，那是泪水 |
| 这不是A，这是B | 这不是疼痛，这是某种更深层的东西 |
| 与其A，不如B | 与其说是愤怒，不如说是一种无力感 |
| A与其说是B，不如说是C | 那声音与其说是尖叫，不如说是某种动物的哀鸣 |
| 与其A，倒不如B | 与其说他在笑，倒不如说他在哭 |
| 不完全是A，更像是B | 不完全是害怕，更像是困惑 |
| A？不，是B | 恐惧？不，是绝望 |
| 不是A。是B。（句号分隔） | 不是白光。是白。 |
| 不是A——是B（破折号分隔） | 不是灵气——是更深的东西 |
| 不是A，是B（逗号分隔独立成句） | 不是倒，是碎 |

**一句话规则：只要句子的结构是"否定一个词→肯定另一个词"来制造情绪升级，就改掉。**

### 7.2 为什么禁

- **读者一眼就认出来**：这个句式已经被AI用烂了，看到"不是"开头就知道后面要"升华"，直接跳读
- **看似深刻实则空洞**：用否定A来肯定B，本质是偷懒——不写具体感受，只做词语替换
- **真实的情绪不是二选一**：人痛苦的时候不会想"这不是疼痛，这是绝望"，人只会说"操"或者沉默
- **AI特别爱用**：因为这个句式最容易生成、看起来最有"文学感"，但实际上是AI最廉价的情绪伪装

### 7.3 替换方案（14种变体全部给出替换）

| 禁用写法 | 替换方案 |
|---------|---------|
| 不是恐惧，是绝望 | 恐惧的时候还想跑，现在连跑的念头都没了 |
| 不是愤怒，而是心寒 | 胸口那股火"噗"地灭了，剩下的东西比冬天的铁扶手还冰 |
| 不是疼痛，更像是什么在撕裂 | 身体里像有根线被拽住了，每扯一下整个胸腔都跟着颤 |
| 不是害怕，不是紧张，是一种更深的不安 | 手心出了汗，心跳倒不快了，就是胃在往下坠 |
| 不是愤怒，不是悲伤，更像是被掏空了 | 嘴里发苦，眼眶干的，想说什么又忘了要说什么 |
| 与其说是愤怒，不如说是恐惧 | 拳头攥着，指甲掐进掌心，掐出来的疼比胸口那股气更容易忍 |
| 疼不是疼，麻不是麻 | 左半边身子像不是自己的了，疼和麻搅在一起分不清 |
| 那不是雨水，那是泪水 | 脸上凉了一下，他伸手去擦，手指是湿的 |
| 这不是疼痛，这是某种更深层的东西 | 说不上疼，更像是有人在胸腔里挖了个洞，风从那穿过去 |
| 与其说他在笑，倒不如说他在哭 | 嘴角是翘的，但眼睛里什么都没有 |
| 那声音与其说是尖叫，不如说是某种动物的哀鸣 | 那声音从喉咙里挤出来，像猫被踩了尾巴，又像什么东西在断 |
| 不完全是害怕，更像是困惑 | 脑子里一片空白，不是吓的，是没反应过来 |
| 恐惧？不，是绝望 | 他站在原地没动，不是不想跑，是腿不听使唤了 |

### 7.4 替换的底层逻辑

不要做"词语替换"（把A换成B），要做"具体化"（写出身体在发生什么）：

```
❌ 词语替换：不是恐惧，是绝望 → 换了个词，还是在"命名情绪"
✅ 具体化：恐惧的时候还想跑，现在连跑的念头都没了 → 写出了"绝望"的具体表现
```

**判断标准：读完替换后的句子，脑子里有没有画面？有画面就对了，没有就还在空转。**

### 7.5 例外

- **对话中可以偶尔使用**（角色性格需要时），但叙述中绝对禁止
- **同一章最多1次对话中使用**，多了也变成AI味
- **不要让所有角色都用这个句式**——只有文艺型/自省型角色偶尔说，其他角色不说

## 八、自检流程

写完一章后，按顺序过一遍：

```
□ 1. 全文搜索禁用词表中的词，逐个处理
□ 2. 检查连续3句开头是否重复
□ 3. 检查是否有直接情感陈述，替换为身体反应
□ 4. 检查对话是否口语化
□ 5. 统计五感描写密度（每1000字>=1处）
□ 6. 统计身体语言密度（每500字>=1处）
□ 7. 标记每段类型（D/T/A/I），检查是否有连续3段同类型
□ 8. 标记每段长度，检查是否有连续3段同长度区间
□ 9. 搜索"不是""而是""更像是""与其说""不如说"——所有"否定→肯定"句式，逐个改写
□ 10. 朗读对话，检查是否有书面感
```
