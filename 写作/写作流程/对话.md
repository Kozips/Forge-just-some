## 对话五维标准

> 一句对话能不能同时干五件事，决定了写作在哪个档次。
> 准一流做到3维，一流做到5维。
> **对话风格参考见 `写作/写作流程/质检/05_写作风格自用.md` 第四节。** ⚠️ 注意：只在「需要拉平风格」时按需加载05，不要养成同时读的习惯。

---

### 一、五维定义

| 维度 | 定义 | 检验方式 |
|------|------|---------|
| ① 剧情 | 这句话符不符合当前正在发生的事？ | 删掉这句话，剧情缺不缺信息？ |
| ② 性格 | 这句话是不是只有这个角色才会这么说？ | 遮住角色名，能不能猜出是谁说的？ |
| ③ 行动 | 这句话有没有暗示他接下来会做什么？ | 读到后面，有没有"原来他当时就在想这个"的感觉？ |
| ④ 场景 | 这句话符不符合当下的环境和氛围？ | 换个场景这句话还成立吗？（成立=这句话没有扎根在场景里） |
| ⑤ 回收 | 读者读到后面回头看这句话，有没有"原来如此"的震感？ | 读完全文后回翻，这句话有没有多出一层意思？ |

---

### 二、质量等级对照

| 等级 | 维度数 | 表现 |
|------|--------|------|
| D级（AI垃圾） | 0-1维 | 对话能推动剧情但没有性格，换个角色说一样的话 |
| C级（模板网文） | 2维 | 有剧情+有性格，但每句话只管当下，不管后面 |
| B级（中上水平） | 3维 | 有剧情+有性格+有场景，但缺少行动预示和回收 |
| A-级（准一流） | 3-4维 | 有剧情+有性格+有行动，偶尔有回收（"原来如此"偶尔出现） |
| A级（一流） | 4-5维 | 每句话至少4维，关键对话5维全满——读者回头看每句话都有多层意思 |
| A+级（大师） | 5维+密度 | 不是只有关键对话5维，是大部分对话都在悄悄做回收——读者第二遍读的时候会发现第一遍漏掉的东西 |

### 三、逐维拆解与示范

#### 3.1 ① 剧情维（最基础）

**标准：** 这句话在推动或回应正在发生的事。

```
❌ 脱离剧情的对话：
场景是两个人在火场里逃命。
"你觉得明天会下雨吗？" ← 和剧情无关，读者困惑

✅ 扎根剧情的对话：
"你先走。" ← 回应正在发生的事（逃命），推动下一步（谁先走）
```

**自检：** 删掉这句话，读者会不会少知道什么？如果不会，这句话没有剧情维。

---

#### 3.2 ② 性格维

**标准：** 遮住角色名，能猜出是谁说的。

```
❌ 无性格差异：
A："你为什么要骗我？"
B："我不是故意的。"
→ 换成任何两个角色都能说，没有辨识度

✅ 有性格差异：
A（暴躁型）："你他妈骗我？"
B（嘴硬型）："我骗你什么了？你给我说清楚。"
→ 遮住名字也能猜出来
```

**自检：** 把这句话放到另一个角色嘴里，读起来违和吗？不违和=没有性格维。

---

#### 3.3 ③ 行动维（准一流的分水岭）

**标准：** 这句话暗示了角色接下来会做什么——不是直说，是藏在语气/用词/反应里的暗示。

```
❌ 无行动预示：
"我知道了。" ← 干巴巴的回应，不知道接下来会怎样

✅ 有行动预示：
"我知道了。" ← 但是他说完之后把刀拿起来了
→ 读者读到后面他真的动手了，会想"他当时说'我知道了'的时候就已经决定了"
```

**示范（《打烊》中的例子）：**

> "他欠我东西。"
> "他欠我茶钱。"

茶馆老板说"他欠我茶钱"——表面是回避程四娘的问题，实际上暗示了老板的立场：他和刀客之间有某种默契，他不会轻易出卖刀客。后面老板确实没有直接告诉她刀客往哪走了（"往东"是真话但不是有用的真话）。

**自检：** 读到后面这个角色做了某件事，回头看这句话，有没有"他当时就已经在想了"的感觉？没有=缺行动维。

---

#### 3.4 ④ 场景维

**标准：** 这句话是不是只能在这个场景里说？换个场景就不成立了。

```
❌ 脱离场景：
在太平间里："今天天气真好啊。" ← 可以在任何地方说，和太平间没关系

✅ 扎根场景：
在太平间里："暖气是不是坏了？" ← 只有太平间才会注意到冷不冷，同时暗示不安
```

**示范：**

高考校门口：
> "我也还行吧。"

这句话扎根在"高考完校门口"这个场景——只有高考完才会问"考得怎么样"，只有在这个场景里"我也还行吧"的试探感才成立。换到食堂里问"考得怎么样"，味道完全不同。

**自检：** 把这句话放到另一个场景里，还成立吗？成立=没有场景维。

---

#### 3.5 ⑤ 回收维（一流的分水岭）

**标准：** 读者读到后面回头看这句话，有没有"原来如此"的震感？

这是最难的一维。它要求对话不仅服务当下，还要服务未来——而且读者读第一遍的时候察觉不到。

```
第一遍读：
"他欠我茶钱。"
→ 读者理解：老板在敷衍程四娘，不想说刀客的事

读到后面（老板确实帮刀客指了路、磨了白印子、帮刀客隐瞒）：
→ 回头再看"他欠我茶钱"：原来老板不只是在敷衍——他确实觉得刀客"欠"他东西，但那个"债"不是茶钱，是一种江湖人之间的默契和信任。老板用"茶钱"这个词来定义他和刀客的关系，比程四娘的"他欠我东西"轻描淡写得多——但恰恰是这种轻描淡写，说明老板把这段关系看得很重。
```

**自检：** 读完全文后回翻，这句话有没有多出一层意思？没有=缺回收维。

### 四、五维自检表（写完对话后逐句过）

| 检查项 | 方法 |
|--------|------|
| ① 剧情 | 这句话删掉，剧情缺信息吗？ |
| ② 性格 | 遮住名字，能猜出谁说的吗？ |
| ③ 行动 | 读到后面，回头想这句话，有没有"他当时就在想这个"？ |
| ④ 场景 | 换个场景这句话还成立吗？ |
| ⑤ 回收 | 读完全文回翻，这句话有没有多出一层意思？ |

#### 评分标准

```
0-1维：D级 → 返工
2维：C级 → 能用但不惊艳
3维：B级 → 有技法，有辨识度
4维：A-级 → 准一流，有灵气
5维：A级 → 一流，读者会反复读
5维+密度：A+级 → 大师，读者第二遍读会发现新东西
```

### 五、实战拆解：《打烊》中的五维对话

#### 对话1："你这灯不是还亮着吗？"

| 维度 | 分析 |
|------|------|
| ① 剧情 | ✅ 回应"打烊了"——老板说打烊，刀客用灯还亮着反驳 |
| ② 性格 | ✅ 嘴硬+不讲理+理直气壮——刀客的说话方式，不是普通客人 |
| ③ 行动 | ✅ 暗示他不会走——灯亮着就有理由留下，他需要一个位置待着 |
| ④ 场景 | ✅ 深夜茶馆、灯还亮着——只有这个场景才能说这句话 |
| ⑤ 回收 | ✅ 读到后面有人追他→他需要一个有灯的地方等/观察→他进门那一刻就已经在布局了 |

**5维全满。**

#### 对话2："他欠我茶钱。"

| 维度 | 分析 |
|------|------|
| ① 剧情 | ✅ 回应"你知道他在哪"——用欠钱来解释为什么关注刀客 |
| ② 性格 | ✅ 嘴硬+轻描淡写——老板的性格，重事轻说 |
| ③ 行动 | ✅ 暗示他不会出卖刀客——用"茶钱"这种小事来淡化关系，实际关系比茶钱深 |
| ④ 场景 | ✅ 茶馆里说"茶钱"——场景和用词完美嵌合 |
| ⑤ 回收 | ✅ 后面老板帮刀客指路、磨白印子、代传话→"茶钱"是老板定义他和刀客关系的方式——轻描淡写但重如千金 |

**5维全满。**

#### 对话3："看情况。"

| 维度 | 分析 |
|------|------|
| ① 剧情 | ✅ 回应"你下次还来吗"——没给确定答案 |
| ② 性格 | ✅ 不承诺、不拒绝、留余地——刀客的生存方式 |
| ③ 行动 | ✅ 暗示他的行踪取决于追他的人——他还在被追，随时可能消失 |
| ④ 场景 | ✅ 门口、即将离开——适合说这种不着地的话 |
| ⑤ 回收 | ✅ 后面他第七天回来了→"看情况"=他确实在看情况，情况允许他就回来了 |

**5维全满。**

### 六、低维对话的升级方法

#### 1维对话（只有剧情）→ 升到3维

```
1维版："你为什么要骗我？"
  ↓ 加性格维
2维版："你他妈骗我？"（暴躁型角色）
  ↓ 加场景维
3维版："你他妈骗我？"（在她生日派对上，所有人都看过来了）
```

#### 2维对话（剧情+性格）→ 升到4维

```
2维版："我不知道。"
  ↓ 加行动维（暗示接下来会做什么）
3维版："我不知道。"（说完把门关了——暗示他在回避，接下来会躲起来）
  ↓ 加回收维（后面他确实躲起来了，而且躲的位置是读者想不到的）
4维版："我不知道。"（说完把门关了。后面读者发现门关上的那一刻他就从窗户翻出去了——"我不知道"=他已经准备跑了）
```

#### 3维对话→ 升到5维

```
3维版："我不认识你。"（剧情：回应"你帮我个忙"；性格：冷漠；场景：茶馆）

  ↓ 加回收维：读者读到后面发现老板其实认识那个刀客——三年前刀客也来过这家茶馆，也坐过那张桌子，也说过同样的话。"我不认识你"不是真话，是老板拒绝介入麻烦的方式。但他还是泡了茶。
5维版："我不认识你。"
→ 读到后面：原来他认识。他只是选择了"不认识"。
```

### 七、自检流程

```
□ 1. 逐句检查对话的五维得分
□ 2. 低于3维的对话标记出来，升级
□ 3. 关键对话（转折/揭露/高潮）必须5维全满
□ 4. 读完全文后回翻前半部分——有没有"原来如此"的对话？（没有→回收维不够）
□ 5. 遮住角色名检查性格维——能猜出来吗？（猜不出→性格维不够）
□ 6. 这段对话放到另一个场景还成立吗？（成立→场景维不够）
```

## 对话场景完整示范

> 展示一个"准一流"对话场景是怎么写的。
> 包含：潜台词、打断、身体语言穿插、情绪转折、沉默运用。

---

### 示范场景：天台摊牌

**背景：** 男主（林深）和女主（苏晚）是青梅竹马。苏晚发现林深一直在隐瞒一件事。天台上，她堵住了他。

---

#### ❌ D级写法（AI垃圾）

> "你为什么要骗我？"苏晚质问道。
> 林深感到一阵心痛，他不禁低下了头。"对不起，我不是故意的。"
> 苏晚似乎很伤心，眼眶红了。"你知道我有多难过吗？"
> 林深一时间不知道该说什么。他感到无比的愧疚。
> "我只是想保护你。"他终于开口道。
> 苏晚叹了口气。"我不需要你的保护。"

**问题：** 书面化、无潜台词、无身体语言、情感直接陈述、对话节奏死板。

---

#### ❌ C级写法（模板网文）

> "你骗了我。"苏晚的声音很平静，但林深听得出来，那不是真的平静。
> 他没有说话，只是把目光移向远处的天际线。
> "多久了？"她问。
> "三年。"
> 苏晚咬了咬嘴唇。"三年。你骗了我三年。"
> "我不是故意的。"林深说。
> "不是故意的？"苏晚冷笑了一声，"三年不是故意的？"
> 林深沉默了。

**进步：** 有节奏变化、有沉默、有冷笑。但还是太"工整"，像剧本。

---

#### ✅ B级写法（中上水平）

> 天台的风很大。苏晚站在他面前，围巾被吹得乱飞，她没管。
> "几号的事。"她说。不是问句。
> 林深靠在栏杆上，手指在口袋里摸烟盒。没摸到。"三月。"
> "三月。"她重复了一遍，像在确认一个数字。"你三月就知道了。"
> "嗯。"
> 风把她的头发吹到脸上。她伸手拨开，手在发抖。
> "你——"她开口，又停了。深吸一口气。"你有没有想过告诉我？"
> "想过。"
> "然后呢？"
> "然后每次看到你笑，就说不出口了。"
> 苏晚没接话。她转过身，背对着他。肩膀在抖。林深不知道她是在哭还是在笑。
> 很久以后，她说了一句话。声音很小，风差点带走。
> "那你现在说吧。"

**进步：** 有环境描写（风）、有身体语言（手在发抖/肩膀在抖）、有潜台词（"每次看到你笑就说不出口"）、有留白（不知道是哭是笑）、有沉默。

---

#### ✅ A-级写法（准一流）

> 天台上没有人。风把晾着的床单吹得鼓起来，又瘪下去，像在呼吸。
> 苏晚把手机放在栏杆上。屏幕上是一张照片。她没说话，只是看着他。
> 林深看了一眼。是三月的。他认识那个角度——是他自己拍的。
> "你翻我手机了。"他说。
> "你先骗我的。"她说。
> 两个人都没再开口。风把床单吹过来，挡在他们中间。白布上印着楼下的洗衣店logo，褪色了，看不清是什么牌子。
> 林深伸手想掏烟。口袋是空的。他想起来，今天出门换了件外套。
> "几号的。"苏晚说。
> "三月十七。"
> "你记这么清楚。"
> "因为那天你穿了件新裙子。绿色的。"
> 苏晚的睫毛动了一下。她低下头，看着自己的手。指甲上还涂着那天的绿色指甲油，已经掉了大半。
> "你每次都这样。"她说。声音很平，像在说今天天气不错。"你每次都把事情藏起来，然后对我笑，好像什么都没发生。"
> "我不想让你——"
> "你不想让我什么？"她抬起头。眼睛是干的。"难过？害怕？你知不知道，比起你骗我，更让我害怕的是，你连骗我都懒得骗了。"
> 林深没有说话。
> 风停了。床单垂下来，露出对面的楼。有人在阳台上收衣服。
> "你身上没有烟味。"苏晚忽然说。
> "今天没抽。"
> "你每天都抽。"
> "今天换了件外套。"
> 苏晚看着他。看了很久。然后她拿起栏杆上的手机，把照片删了。
> "下次想骗我，"她说，"至少别用你自己的手机拍。"
> 她转身走了。围巾被风吹起来，挂到了栏杆的角上。她没回头。
> 林深站在天台上，看着那条围巾在风里飘。灰色的，起了球。他认识那条围巾。是他去年生日她送的。
> 他把围巾取下来，攥在手里。
> 手心很凉。

---

### 技法拆解

| 技法 | 具体运用 |
|------|---------|
| **潜台词** | "你先骗我的"→表面是翻手机的事，实际是三年的欺骗 |
| **物象** | 绿色指甲油（掉了大半=关系褪色）、围巾（起了球=旧感情） |
| **沉默** | 两人多次不说话，沉默比语言更有力 |
| **身体语言** | 睫毛动了一下/眼睛是干的/手心很凉 |
| **环境映射** | 风停了=对峙结束；床单垂下来=遮挡消失，真相裸露 |
| **打断** | "我不想让你——"被她打断，没说完的话比说完更重 |
| **情绪转折** | 她眼睛是干的（不是哭，是更深层的失望） |
| **留白** | "至少别用你自己的手机拍"→她到底在生气什么？是骗她，还是不够用心骗？ |
| **结尾余韵** | 围巾挂到栏杆上，他取下来攥在手里→没说任何话，但情感全部传达 |

---

### 对话场景速查公式

```
环境描写（建立空间感）
  ↓
一方开口（不是问句就是强势）
  ↓
另一方不直接回答（回避=有秘密）
  ↓
身体语言穿插（比台词更真实的信号）
  ↓
潜台词交换（表面说A，实际说B）
  ↓
情绪转折点（一方说了真话/做了意外的事）
  ↓
沉默（最重要的对话发生在沉默里）
  ↓
留白收尾（不总结，让读者自己品）
```

### 示范场景：青春暧昧（高考结束校门口）

**背景：** 许衡和陈茵是高中同学，暗恋但没捅破。高考完那天下午，他在校门口等她。

#### ❌ AI写法

> "你怎么来了？"陈茵感到一阵心跳加速，她不禁低下了头。
> "路过。"许衡似乎有些紧张，心中暗道：她今天真好看。
> "你家在城西，不是顺路。"陈莞尔一笑，眼中充满了温柔。
> 许衡一时间不知道该如何回应。"我来看看你考得怎么样。"
> "还行吧。你呢？"
> "也还行。"他感到一阵莫名的失落。

**问题：** "感到一阵""不禁""似乎""心中暗道""眼中充满了""一时间""感到一阵"——七个AI禁用词/句式全中。对话书面化，无画面感。

#### ✅ 准一流写法

> 高考完那天下午，许衡在校门口等了差不多四十分钟。
>
> 太阳晒得后脖子发烫，校服衬衫贴在背上，潮乎乎的。他正低头看手机，听见有人喊了一声"陈茵你等会儿"，抬头就看见她了。
>
> 她和几个女生说说笑笑地走出来。白T恤，头发剪短了，露出耳朵——这件T恤他没见过。
>
> 陈茵也看见了他。她跟身边的人说了句什么，朝他走过来。
>
> "你怎么来了？"
>
> "路过。"
>
> "你家在城西。"
>
> "绕路。"
>
> 陈茵笑了一下。嘴角翘了一点就放下来了。她低头踢了踢地上的小石子。
>
> "考得怎么样？"许衡问。
>
> "还行吧。你呢？"
>
> "我也还行吧。"
>
> 两个人站在那里，谁都没动。风把地上的准考证吹起来，转了两圈，贴在许衡的鞋面上。他弯腰捡起来看——一个叫王磊的人的，不认识。

**为什么写得好：**
- **先声后人**：先听见别人喊"陈茵你等会儿"，再抬头看见她——符合真实感知顺序
- **时间模糊化**："差不多四十分钟"——人不会精确计时，模糊量词才像真人记忆
- **对话极简**：六个来回全是短句，没有内心独白解释，潜台词全在字面下（"绕路"=我专门来看你）
- **情绪用动作传递**："嘴角翘了一点就放下来了""踢了踢地上的小石子"——不写"她害羞"，用微动作
- **用无关物件收尾**：王磊的准考证——两个暧昧的人找不到话说，正好一阵风吹来一张不属于他们的准考证，比任何总结都好
- **"我也还行吧"**：那个"吧"字带着不确定，两个人都在试探，谁都不肯先说真话

**AI对标指标：**
- 人物出场先有声音/动作，再写"看见了"
- 时间不用精确数字，用"差不多/大概/四十来分钟"
- 暗恋对话不超过8个来回，每句不超过10字
- 情绪不直接陈述，用嘴角/手指/脚步等微动作传递
- 收尾不用抒情句，用一个无关的环境细节

### 示范场景：武侠茶馆（陌生人闯入）

**背景：** 一个刀客深夜闯进打烊的茶馆，老板一个人在收拾。

#### ❌ AI写法

> "一壶茶。"刀客说道，他的眼神中充满了杀气。
> "打烊了。"我似乎感受到了一丝危险。
> "我看见灯还亮着。"刀客似乎在暗示什么。
> "灯没关不代表没打烊。"我不禁握紧了拳头。
> 他看着我，一时间空气仿佛凝固了。
> 我看见他右手的食指少了半截，心中暗道：这个人不简单。

**问题：** "说道""眼神中充满了""似乎""不禁""一时间""仿佛""心中暗道"——全是AI病句。对话没有性格差异，两个人说话方式一模一样。

#### ✅ 准一流写法

> 刀客进门的时候，茶馆里只剩我一个。
>
> 他个子不高，刀倒是挺长，拖在地上走，石板被刮出一道白印子。他在角落那张桌子坐下来，把刀竖在桌边，拍了拍桌面："一壶茶。"
>
> "打烊了。"
>
> "你这灯不是还亮着吗？"
>
> "灯没关不代表没打烊。"
>
> 他看着我，没说话。我看见他右手的食指少了半截，伤口已经长好了，圆圆的，像是之前被人用剪子齐根剪掉的。
>
> 我给他泡了壶茶。他喝了两杯，从怀里掏出一块碎银子放在桌上。
>
> "不够。"我说。
>
> "我知道。"他把刀拿起来，"剩下的明天给。你帮我个忙——今晚有人来找我，你就说没见过。"
>
> "我不认识你。"
>
> "那就更好说了。"

**为什么写得好：**
- **性格通过动作差异体现**：刀客拍桌面=习惯使唤人；老板"灯没关不代表没打烊"=嘴硬但还是泡了茶——性格在行为矛盾中显现
- **"不是"在对话中做口语反问**："你这灯不是还亮着吗"——这是活人说话的语气，不是叙述文里的"不是A，是B"升级句式
- **伤疤描写具体化**："圆圆的，像是之前被人用剪子齐根剪掉的"——不说"他看起来经历过很多"，用一个具体伤疤让读者自己推断
- **潜台词密度极高**："不够""我知道""剩下的明天给"——三句话完成了一次试探性的交易，没有一个字是废话
- **收尾反转**："我不认识你""那就更好说了"——不认识反而更好办事，一句话写出了江湖规则

**AI对标指标：**
- 对话中可以用"不是"做口语反问（但叙述文中禁止）
- 伤疤/外貌用具体形状+来源猜测，不用形容词堆砌
- 陌生人对话不超过10个来回，每句带信息量
- 老板的性格通过"嘴硬但还是泡了茶"这种行为矛盾体现

## 对比与否定写法

> 替代"不是A，是B"的偷懒句式。**叙述/描写中此句式已禁用**（对话中角色可以说，那是人话）。根据角色性格、学识、情感、情景选择最合适的替代方式。

---

### 问题："不是A，是B"为什么不好用

```
她不是害怕，是恐惧。
他不是生气，是愤怒。
这不是普通的雨，是暴风雨。
```

问题：
- 太直白，读者没有参与感
- 所有角色用同一个句式，失去辨识度
- "不是X，是Y"本质是**词典替换**，不是写作

---

### 替代技法

#### 1. 角色知识库比喻（最有辨识度）

根据角色的职业、学识、生活经历选择比喻域——同一个情绪，不同角色的比喻完全不同。

| 角色 | 情绪 | 比喻 |
|------|------|------|
| 厨师 | 紧张 | "像油锅里溅了一滴水。" |
| 程序员 | 紧张 | "像代码跑到了死循环，脑子在空转。" |
| 渔民 | 紧张 | "像船底突然漏了，水在往上涌。" |
| 赌徒 | 紧张 | "像骰子已经掷出去了，还没落桌。" |
| 军人 | 紧张 | "像子弹已经出膛，但还没听到响。" |
| 学生 | 紧张 | "像考试铃响了，但卷子还没发到手里。" |
| 医生 | 紧张 | "像心电图突然拉成一条直线。" |

**核心**：比喻不是装饰，是角色身份的延伸。角色不会用自己不懂的东西做比喻。

#### 2. 感官替代法

不写"她很害怕"，写她的身体在做什么。

| 偷懒写法 | 感官替代 |
|---------|---------|
| 她很害怕 | 她的指甲掐进了掌心，月牙形的印子一排 |
| 他很愤怒 | 他的太阳穴在跳，一下一下，像有人在里面敲鼓 |
| 她很紧张 | 她吞了一口口水，喉咙发紧，像咽了一颗石子 |
| 他很悲伤 | 他盯着手机屏幕，屏幕暗了又亮，亮了又暗，他没解锁 |
| 她很尴尬 | 她的手不知道放哪，先揣兜里，又拿出来，最后假装在摸头发 |

#### 3. 动作替代法

不写"她不是生气，是伤心"，写她做了什么。

```
偷懒：她不是生气，是伤心。
替代：她把杯子放回桌上，放得很轻。然后转身去洗碗，水开得很大。
```

```
偷懒：他不是害怕，是绝望。
替代：他把手机掏出来，看了一眼信号——没有。又放回口袋。又掏出来。又看了一眼。
```

**实战案例：替代"不是想的，是身体自己动的"**

```
偷懒：陈默侧身躲开——不是想的，是身体自己动的。
替代：陈默的身体比脑子先动了——脚掌蹬地，重心下沉，上半身往后仰了半尺。
      等他反应过来的时候，杆已经擦过去了。
      在部队练过格斗，侧闪是肌肉记忆，不需要他同意。
```

**为什么更好**：
- "不是想的，是身体自己动的"是抽象否定，读者在**读解释**
- "脚掌蹬地，重心下沉，上半身后仰"是具体动作，读者在**看画面**
- "不需要他同意"比"肌肉记忆"更有性格感——军人的身体不问脑子的意见

**核心**：动作比形容词有力十倍。读者从动作里自己判断情绪。

#### 4. 环境映射法

用环境的变化映射角色内心——角色不说话，环境替她说。

```
偷懒：她不是紧张，是恐惧。
替代：窗外的风停了。房间里只剩下空调的嗡嗡声，和她自己的心跳。
```

```
偷懒：他不是生气，是愤怒。
替代：他把车窗摇下来。外面的风灌进来，吹得后视镜上的挂件疯狂摇晃。他没关窗。
```

#### 5. 留白法

最有力的对比是**不对比**——直接写结果，让读者自己填空。

```
偷懒：她不是不爱他，是不敢爱。
替代：她把他的消息看了三遍。然后锁屏。然后又解锁。然后把手机扣在桌上。
```

```
偷懒：他不是不想活，是活不下去。
替代：他把遗书叠了四折，塞进信封。然后坐在窗台上，点了根烟。烟抽到一半，他把信封撕了。
```

#### 6. 反问法

用角色内心的反问制造对比——不是作者在对比，是角色在挣扎。

```
偷懒：她不是不想走，是走不了。
替代：走？去哪？她看了一眼门口。门没锁。她的脚没动。
```

```
偷懒：他不是不害怕，是必须上。
替代：怕吗？怕。但他看了一眼身后的人。然后他往前走了一步。
```

#### 7. 递进否定法

如果一定要用否定，至少做三层递进——不是简单的一次否定。

```
偷懒：她不是紧张，是害怕。
替代：她不紧张。她告诉自己不紧张。但她的手在抖——不是那种小抖，是那种连钥匙都插不进锁孔的抖。
```

```
偷懒：他不是生气，是愤怒。
替代：他没生气。他笑了笑。但那个笑容没到眼睛。然后他把杯子放在桌上，玻璃杯底裂了。
```

### 选技法速查

| 角色状态 | 推荐技法 |
|---------|---------|
| 角色有明确职业/学识 | 角色知识库比喻 |
| 角色在压抑情绪 | 感官替代 + 动作替代 |
| 角露骨绪爆发 | 动作替代 + 环境映射 |
| 角色在挣扎/犹豫 | 反问法 + 留白法 |
| 角色在掩饰 | 递进否定法 |
| 不确定用什么 | 动作替代法（最通用） |
