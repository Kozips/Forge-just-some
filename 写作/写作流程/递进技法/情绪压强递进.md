# 情绪压强递进参考

> 从11个类型、156篇小说参考中蒸馏而得。
> 禁用"不是A，是B"句式。全部用动作、感官、环境展示情绪。

---

## 一、恐惧

### 压强阶梯

**L1 不安** → **L2 警觉** → **L3 恐惧** → **L4 崩溃** → **L5 麻木/疯狂**

### 恐惧递进模式

#### 模式A：渐变恐怖（洛夫克拉夫特/克苏鲁系）

不给你一个jump scare，让一切慢慢变不对。等意识到不对的时候，已经太晚了。

**阶梯拆解：**
1. **日常渗入**：庄稼长得不对、井水的味道变了、人的皮肤开始发灰。读者尚可自我说服"也许没什么"
2. **异常扩大**：植物变异、动物发疯。自我说服的余地收窄
3. **核心真相**：一种"不在可见光谱上的颜色"正在侵蚀一切。真相太重，认知框架被击穿
4. **无法回头**：知道了就回不去，理智已受损。唯一的"安全"是忘记一切

**技法要点：**
- 全部通过"变化"暗示，不直接描写恐怖画面。读者脑补的画面比任何描写都恐怖
- 恐怖从"外面"转移到"自己身上"是最高级的手法——"我就是恐怖的一部分"

#### 模式B：学者崩溃弧线（理性崩塌线）

**阶梯拆解：**
1. 学术冷静："这些样本不符合任何已知生物学分类"——手还在稳稳记录
2. 困惑动摇："这不可能...一定是哪里弄错了..."——笔停了，反复翻看笔记
3. 恐惧接受："天哪...它一直在那里..."——笔从手里滑落，后退撞到桌子
4. 疯狂呓语："那些角度...你没看到吗？那些角度在动..."——抓挠自己的脸，反复擦拭眼镜

**关键**：越理性的人崩溃越恐怖。崩溃的反差感=最大压强。

#### 模式C：规则恐怖（规则怪谈系）

**阶梯拆解：**
1. **锚定期**：规则清晰——"晚上十点后不要看镜子""听到敲门声不要开门"。遵守=安全。手在发抖但还有规则可依
2. **动摇期**：出现例外——某条规则不灵了。手指反复摩挲那张规则纸条，确认自己没记错
3. **矛盾期**：规则互相打架——"必须待在房间里"同时"凌晨三点必须离开房间"。两条死路，攥紧拳头又松开，反复循环
4. **崩塌期**：规则本身是陷阱——遵守即死。手里的规则纸条被揉成一团扔掉，又捡回来，又扔掉

**技法要点：**
- 信息=武器。知道一条隐藏规则=多一条命。信息的价值被极端放大
- "遵守即死"反转：打破读者的"安全假设"
- 试错成本递增：前期违反规则受伤，后期违反规则直接死

#### 模式D：封闭空间倒计时（暴风雪山庄系）

**阶梯拆解：**
1. 十个人聚齐，发现是孤岛。互相打量，有人开始数人数
2. 第一个人死了。剩余九人围在尸体旁，不敢对视
3. 信任瓦解——人越少，互相越不信任。三个人的时候，每个人都在看另外两个人的手
4. 最后一个人。整个房子只有自己的呼吸声。门全部敞开，但不敢走出去

**技法要点：**
- 人数递减=天然倒计时。读者在数"还剩几个"
- 童谣/预言做死亡预告：提前告诉读者死亡方式，增加焦虑感
- 封闭空间=没有外援=恐惧感放大

### 恐惧的外在表现层级

| 压强等级 | 动作 | 感官 | 环境呼应 |
|---------|------|------|---------|
| L1 不安 | 频繁回头、摸口袋确认钥匙 | 听觉敏感化，任何声响都被放大 | 光线似乎暗了一点 |
| L2 警觉 | 停下脚步、屏住呼吸 | 瞳孔放大，视野变窄 | 风停了 |
| L3 恐惧 | 后退、抓握身边物体 | 冷汗、心跳声盖过一切 | 空气变得粘稠 |
| L4 崩溃 | 跌倒、捂耳朵、尖叫 | 失禁、呕吐、视线模糊 | 世界旋转/坍缩 |
| L5 麻木/疯狂 | 不动了，或者开始笑 | 感觉不到疼痛 | 一切变得安静/一切都在动 |

## 二、愤怒

### 压强阶梯

**L1 不满** → **L2 忍耐** → **L3 压抑** → **L4 爆发** → **L5 冷怒（最危险）**

### 愤怒递进模式

#### 模式A：屈辱蓄力→爆发（玄幻打脸线）

**阶梯拆解：**
1. **被轻视**：对方斜眼看过来，嘴角带着笑。主角攥紧袖子里的手
2. **忍耐**：对方变本加厉，言语侮辱升级。主角低着头，牙齿咬到腮帮子出了血味
3. **底线被踩**：涉及亲人/尊严核心。主角的眼睛抬起来了，周围人发现他的眼神变了
4. **出手**：一击。没有多余的动作。对手飞出去，撞碎了身后的墙
5. **反差释放**：全场寂静。之前嘲笑的人腿在发软。主角转身离开，脚步声在空旷的场地上回荡

**技法要点：**
- 蓄力极长，爆发极短。铺垫写几章（对峙、试探），真打就一招
- 沉默比怒吼更有杀伤力。沉默=正在积蓄的压强
- 打脸的爽感来自"反差"——之前越忍，之后越爽

#### 模式B：冷怒线（权力型愤怒）

**阶梯拆解：**
1. 听到消息，手指在桌面上轻敲了三下。停顿。又敲了三下
2. 笑了。端起茶杯喝了一口。放下时杯盖和杯身碰出一声轻响
3. "朕知道了。"——语气平静得像在说天气
4. 身边的太监跪下了。不是因为命令，是因为那种安静比任何咆哮都可怕

**技法要点：**
- 冷怒比热怒恐怖十倍。热怒说明还有回旋余地，冷怒说明判决已下
- 权力越大越不需要提高声音。一句话=一个人的命运
- 潜台词："朕知道了"可以是"朕同意"也可以是"朕记住了你这笔账"

#### 模式C：愤怒+委屈叠加

**阶梯拆解：**
1. 被冤枉。嘴唇张了张，没说出话
2. 想辩解，对方根本不听。手举起来又放下
3. 眼眶红了，但硬是没掉泪。把脸转向窗外
4. 声音开始发抖："随便你怎么想。"——嘴上是放弃，手在桌子底下攥到指节发白

**技法要点：**
- 愤怒+委屈的叠加特征：嘴硬+身体诚实。嘴上说"无所谓"，手在发抖
- 压抑的哭比嚎啕更有冲击力。咬嘴唇、转脸、深呼吸——都是在阻止眼泪
- 环境配合：突然安静下来，只有钟摆的声音

### 愤怒的外在表现层级

| 压强等级 | 动作 | 声音 | 环境 |
|---------|------|------|------|
| L1 不满 | 皱眉、手指轻敲 | 语气微沉 | 空气凝了一瞬 |
| L2 忍耐 | 深呼吸、攥拳又松开 | 声音变低 | 周围人不敢说话 |
| L3 压抑 | 咬牙、转脸、离开现场 | 不说话了 | 安静得不正常 |
| L4 爆发 | 摔东西、拍桌子、出手 | 吼叫或冷笑 | 物件碎裂/人飞出去 |
| L5 冷怒 | 静坐不动、微笑 | 极度平静 | 周围人开始发抖 |

## 三、悲伤

### 压强阶梯

**L1 隐痛** → **L2 涌上来** → **L3 崩溃** → **L4 平静（最痛）** → **L5 空洞**

### 悲伤递进模式

#### 模式A：失去递进（余华线——活着）

每次以为"不能再惨了"，来一个更惨的。情绪不是线性下降，是"谷底→短暂回升→更深的谷底"。

**阶梯拆解：**
1. 失去钱——还能挣扎。手在抖，但还能站起来
2. 失去孩子——蹲在路边，双手抱头，肩膀在抖。发不出声音
3. 失去伴侣——坐在空荡荡的屋子里，盯着她坐过的那把椅子。一天一夜没动
4. 失去最后的亲人——牵着老牛去犁地。跟牛说话。声音和平时一样

**技法要点：**
- 越惨的事用越平静的语气写。平静比哭更痛——读者的情绪反差=震撼
- 叙述者讲自己一生的时候语气是平静的。"平静"是最高级的悲伤表达
- 每次失去都让读者觉得"到底了"，但没有。失去的东西越来越重要

#### 模式B：错过型悲伤（留白虐感）

**阶梯拆解：**
1. 重逢。两个人隔着人群对视了一眼。她的手指动了一下，像是想举起来，又放下了
2. 对话。两个人说话客气得像陌生人。每个字都正确，每个字都痛
3. 分开。他走出三步，停了一下。她没叫他。他继续走了
4. 独处。回到房间，关上门，靠在门板上滑坐到地上。手捂住脸。没有声音

**技法要点：**
- 留白做虐感：七年的分离只用几段话带过。读者自己脑补比直接写更痛
- 重逢的距离感：两个人明明相爱但都装不在乎。对话越客气心越痛
- 一句话做全部重量："如果世界上曾经有那个人出现过，其他人都会变成将就"

#### 模式C：平静叙述极端事件（日常暴力线）

**阶梯拆解：**
1. 有庆去献血。排着队，还跟旁边的人说笑
2. 护士抽了一管又一管。有庆的脸开始发白。他还说"没事"
3. 有庆倒下了。护士说"血型配上了，再抽一点"
4. 有庆死了。福贵抱着他回家。路上一句话没说。天很蓝

**技法要点：**
- 用新闻报道的语气写最惨的事。"死者有庆，男，十三岁。抽血过量致死。"——越克制越残忍
- 日常暴力比战斗暴力更恐怖：死在战场上是"正常"的，死在日常中是"不应该"的
- 受害者完全无辜。没有理由的暴力比有理由的更恐怖

### 悲伤的外在表现层级

| 压强等级 | 动作 | 感官 | 反常表现 |
|---------|------|------|---------|
| L1 隐痛 | 手指摩挲某样旧物 | 喉咙发紧 | 正常社交但笑容没到眼睛 |
| L2 涌上来 | 深吸一口气、抬头看天花板 | 眼眶热了 | 找借口去洗手间 |
| L3 崩溃 | 蹲下/坐倒、捂脸 | 泪水不受控制 | 哭出声或无声抽泣 |
| L4 平静 | 坐着不动、盯着某处 | 感觉麻木了 | 反而笑了，或者开始整理东西 |
| L5 空洞 | 日常行为机械执行 | 吃不出味道 | 跟平时一样，但眼神是空的 |

## 四、绝望

### 压强阶梯

**L1 挣扎** → **L2 疲惫** → **L3 放弃抵抗** → **L4 接受** → **L5 反常平静**

### 绝望递进模式

#### 模式A：末世绝望（资源递减线）

**阶梯拆解：**
1. 找到半瓶水。两个人分。她把自己那份倒回去一点，说"我不渴"
2. 水喝完了。嚼草根。舌头上全是草的苦味和泥土的腥味
3. 三天没喝水。嘴唇裂开，说话时裂口渗出血。不再说话了
4. 看到远处有水的反光。走过去。是海市蜃楼。站在原地，没有表情

**技法要点：**
- 递进的本质是"每次以为到底了，发现还有更深的坑"
- 日常细节做锚点：末世再恐怖，角色还是会想喝可乐、想吃到热饭。这些日常渴望让绝望更扎心
- 死亡的"随机性"：重要角色可以死得很随意。不是英勇牺牲，就是被咬了一口

#### 模式B：规则崩塌型绝望

**阶梯拆解：**
1. 发现规则不灵了。反复确认。还在自我说服"是我记错了"
2. 试了所有办法，全部失败。坐在地上，手里攥着写满规则的纸条
3. 意识到规则本身是陷阱。笑了——不是觉得好笑，是那种"原来如此"的苦笑
4. 把纸条扔了。站起来。不再看规则。往黑暗里走

**技法要点：**
- 信任体系瓦解=最深的绝望。规则=安全感的最后支撑，打破规则=安全感归零
- "不可逆"：一旦知道真相就回不去。好奇心=诅咒

#### 模式C：命运不可逆型绝望（历史/穿越线）

**阶梯拆解：**
1. 穿越者知道未来会发生什么。在关键事件前拼命尝试改变
2. 每次改变一个节点，蝴蝶效应让其他地方崩塌。"救了A，B就死了"
3. 意识到历史的河流太宽太急，个人的努力连一朵浪花都算不上
4. 站在城墙上看着历史按原来的轨迹碾过去。手松开了，手里的计划书随风飘散

**技法要点：**
- 知道结局但改变不了=最高级的绝望。"知道"不等于"能改变"
- 无力感比恐惧更持久。恐惧可以逃跑，无力感无处可逃

### 绝望的外在表现层级

| 压强等级 | 动作 | 生理反应 | 语言 |
|---------|------|---------|------|
| L1 挣扎 | 翻找、奔跑、尝试 | 肾上腺素飙升 | "还有办法""再试一次" |
| L2 疲惫 | 动作变慢、坐着喘气 | 脱水、低血糖 | "让我想想...""等等..." |
| L3 放弃 | 不动了、手垂下来 | 呼吸变浅 | 不说话了 |
| L4 接受 | 开始做最后的安排 | 反而平静了 | "就这样吧" |
| L5 反常 | 笑了、唱歌、做奇怪的事 | 回光返照般的精神 | 说一些别人听不懂的话 |

## 五、心动

### 压强阶梯

**L1 注意** → **L2 在意** → **L3 心跳** → **L4 沉溺** → **L5 不顾一切**

### 心动递进模式

#### 模式A：试探型心动（暧昧递进）

**阶梯拆解：**
1. 不小心碰到手指。两个人同时缩回去。都没说话。空气里多了一点什么
2. 在一群人里，眼睛会自动找她。找到了就移开。过一会又找
3. 她靠近的时候，他手里的杯子握得太紧了。指节发白。她在旁边，他什么话都说不好了
4. 她走了。他站在原地。过了很久才发现自己在看她走过的那条路

**技法要点：**
- 身体反应比心理描写更诚实：手指、呼吸、眼神——这些不受意识控制
- "不看"比"看"更有信息量。一群人里故意不看某个人=看了无数次
- 距离感的精确描写：远一步可以假装没事，近一步就要破功

#### 模式B：克制型心动（身份/道德线）

**阶梯拆解：**
1. 知道不该看她。低头。眼睛盯着自己的手
2. 她说话了。声音很好听。他把这句话在心里重复了三遍，然后骂自己
3. 她被人欺负了。他站起来。所有人看他。他坐下了。手在桌子底下攥到发抖
4. 深夜。一个人对着窗外。她在隔壁房间。一墙之隔。他把灯关了

**技法要点：**
- 压抑的心动比放肆的心动更虐。想做但不能做=持续的低强度痛感
- 用"转移注意力"写心动：拼命工作、找别人聊天、跑步跑到累——都是在逃避
- "距离"做压强：一墙之隔、一臂之遥、擦肩而过——物理距离越近心理压力越大

#### 模式C：心死与心动的反转

**阶梯拆解：**
1. 以为已经放下了。看到她的时候，步伐没变，表情没变
2. 她笑了一下。他的脚步顿了一瞬——不仔细看看不出来
3. 她转身走了。他站在原地。手伸进口袋里，摸到了一样她曾经送的东西。指腹摩挲了一下。又放下了
4. 回到家。把那样东西拿出来放在桌上。看了很久。然后放回抽屉最深处

**技法要点：**
- "以为放下了"=最没放下的时候。刻意的正常=最大的不正常
- 旧物件做触发器：一条手帕、一张照片、一首歌——物件连接记忆
- 微反应：脚步顿一瞬、手指动一下、呼吸变了一拍——越小的反应说明压抑越深

### 心动的外在表现层级

| 压强等级 | 动作 | 生理 | 声音 |
|---------|------|------|------|
| L1 注意 | 视线多停留了一秒 | 无明显反应 | 正常 |
| L2 在意 | 会注意她的状态变化 | 心跳微微加快 | 说话稍微不自然 |
| L3 心跳 | 靠近时手不知道放哪 | 耳根发热、呼吸变浅 | 声音比平时低或者高 |
| L4 沉溺 | 不自觉朝她走过去 | 心跳声自己能听见 | 说话变少，在听她说话 |
| L5 不顾一切 | 冲过去/做出冲动的事 | 什么都感觉不到 | 说出平时绝对不会说的话 |

## 六、焦虑/紧张

### 压强阶梯

**L1 不安** → **L2 等待** → **L3 煎熬** → **L4 接近极限** → **L5 崩溃或爆发**

### 焦虑递进模式

#### 模式A：倒计时焦虑（战前/审讯线）

**阶梯拆解：**
1. 知道结果在几小时后出来。开始整理不需要整理的东西。桌面擦了三遍
2. 吃不下东西。把食物在盘子里推来推去。喝了半杯水。杯子放下时手在抖
3. 来回踱步。数地板上的砖。数错了，从头来。又数错了
4. 坐下了。不动了。盯着钟表。秒针的声音变成锤子在敲

**技法要点：**
- 等待比结果更折磨人。不确定性=持续的低强度恐惧
- 用"多余动作"写焦虑：整理、擦拭、数东西、反复检查——都是焦虑的外溢
- 时间感知变形：焦虑时一分钟像一小时。描写时间的主观感受

#### 模式B：信息差焦虑（重生者/先知线）

**阶梯拆解：**
1. 知道灾难要来。开始囤物资。别人觉得他疯了
2. 灾难来了。别人恐慌，他冷静。但他心里在计算——上次的历史里，下一个灾难什么时候来
3. 蝴蝶效应开始显现。某些事情偏移了。他预知的优势在缩小。手开始发抖——不是因为眼前的危险，是因为"不确定"
4. 预知的未来面目全非。他和所有人一样不知道接下来会发生什么。重生者的焦虑比普通人更深——因为他知道"知道未来"是什么感觉，失去它更可怕

**技法要点：**
- 先知的焦虑不是"不知道"，是"知道但不确信了"
- 信息差的递减效应=焦虑的递增

### 焦虑的外在表现层级

| 压强等级 | 动作 | 生理 | 认知 |
|---------|------|------|------|
| L1 不安 | 频繁看手机/钟表 | 轻微失眠 | 注意力不集中 |
| L2 等待 | 来回踱步、整理物品 | 食欲下降 | 反复计算/推演 |
| L3 煎熬 | 坐立不安、手抖 | 心悸、胃痛 | 思维反刍停不下来 |
| L4 接近极限 | 不动了、盯着一个方向 | 浑身发冷 | 大脑一片空白或极度清晰 |
| L5 崩溃 | 突然大喊/出手/逃跑 | 出汗、失禁 | 完全失去理性判断 |

## 七、情绪叠加写法

### 恐惧+内疚

**场景**：做了一个决定导致同伴受伤。自己安全了但同伴在流血。

**写法**：
> 他蹲在她身边，手伸出去又缩回来。不敢碰她的伤口。她的血粘在他的指缝里，温热的，粘稠的。他去洗手，洗了三遍，指甲缝里还有红色。水龙头的水流声和她微弱的呼吸声混在一起。

### 愤怒+心疼

**场景**：发现亲人为了保护自己而隐瞒了伤情。

**写法**：
> 他看着那道从肩膀延伸到腰间的伤疤。疤痕已经发白了。他的手抬起来，指尖碰到疤痕的边缘，停住了。"什么时候的事？"声音平得像刀面。她没回答。他把手收回来。转身走了出去。门关上的声音比平时重了一点。她听到走廊里传来一声闷响——他一拳砸在了墙上。

### 悲伤+释然

**场景**：在葬礼上。

**写法**：
> 雨停了。送葬的人散了。他站在墓碑前，把一束白菊放下。花瓣上有水珠，滚了一下，落在土里。他站了很久。风把他的衣角吹起来又放下。最后他把帽子戴上，转身。脚步不快不慢。走出墓园的时候他抬头看了一眼天——云层裂了一条缝，有光透下来。

### 恐惧+决心

**场景**：明知是死路但必须走。

**写法**：
> 她在门口站了很久。手放在门把上。金属的冰凉从掌心传上来。她的心跳快得像要撞断肋骨。深吸一口气。又一口。手指收紧。门把转动的声音在安静的走廊里格外清楚。她把门推开了。

### 爱意+愤怒（爱恨交织）

**阶梯拆解：**
1. 看到他做了不可原谅的事。胸口像被人捶了一拳。不是疼——是闷
2. 想骂他。嘴巴张开。看到他的脸——疲惫的、愧疚的。话又咽回去了
3. 转身不看他。手指攥着衣角。指甲陷进掌心
4. "你走吧。"声音是平的。但他走后，她靠在墙上，慢慢蹲下来。手捂住嘴，牙齿咬住手背，不让自己出声

## 八、情绪传染与隔离

### 传染模式

**个体→小群体→大群体**

1. 一个人开始哭。旁边的人眼眶红了。第三个人低下了头
2. 沉默从一个角落蔓延到整个房间。没人说话。勺子碰到碗的声音变得很响
3. 一个人站起来要走。所有人看着他。没人拦。但他走到门口停了——门外的人都在看他。恐惧传染了

### 隔离模式

**一个人在群体中完全孤立**

1. 整个房间在笑。她没笑。不是不想笑——是笑不出来。笑声在她耳边变成噪音
2. 他说话了。没人回应。不是故意不理——是根本没听到他的声音。他的声音在嘈杂中被淹没了
3. 所有人在讨论怎么分配物资。他的名字没有被提到。他坐在角落里，看着他们说话。嘴巴张了一下。又闭上了

**技法要点：**
- 传染靠"动作的复制"——一个人做了一个动作，旁边的人无意识地跟着做
- 隔离靠"对比"——群体的热闹越甚，个体的孤独越深。用环境的"满"衬托内心的"空"

## 九、冷处理 vs 热处理对比

### 同一场景：得知亲人去世

**热处理（直接爆发）：**
> 电话从手里滑落。她尖叫了一声——不是那种有意识的叫，是身体自己发出的声音。膝盖软了，整个人跪在地上。手指抓着地板，指甲劈了也没感觉。眼泪和鼻涕混在一起，嘴里发出断断续续的呜咽。"不可能——不可能——"她反复说，声音嘶哑。

**冷处理（克制压抑）：**
> 她挂了电话。把手机放在桌上。屏幕朝下。去厨房烧了一壶水。水开了。她没有去倒。站在灶台前，听着水壶的啸叫声。啸叫停了——是水溢出来了。她关了火。拿抹布把灶台擦干净。然后坐在餐桌前。一直坐着。

**两种处理的适用场景：**
- 热处理适合"第一次打击"——角色还不够成熟，情绪控制力弱
- 冷处理适合"积累性打击"——经历了太多，已经学会了把崩溃藏起来
- 冷→热的转折最具杀伤力：一个一直冷静的人突然崩溃

### 同一场景：被人背叛

**热处理：**
> 他把杯子摔在地上。碎片溅了一地。"你说！你为什么要这样做！"声音劈了。胸口剧烈起伏。他抓起桌上的文件又放下。又抓起来。最后把整张桌子掀了。

**冷处理：**
> 他听完。点了点头。"我知道了。"站起来，把椅子推回桌下。走到门口，停了一下。"茶凉了，你自己倒掉吧。"走了。门关得很轻。但从那天起，那个人再也没有收到过他的任何消息。

## 十、情绪转折触发器

### 触发器类型

| 触发器 | 从→到 | 典型场景 |
|--------|-------|---------|
| **物件** | 平静→崩溃 | 翻到一张旧照片、闻到一个熟悉的味道 |
| **一句话** | 隐忍→爆发 | "你还好意思哭？""我都是为了你好。" |
| **身体极限** | 焦虑→麻木 | 累到某个临界点，突然不焦虑了 |
| **第三方介入** | 恐惧→愤怒 | 看到别人被欺负，恐惧转为保护欲 |
| **环境突变** | 绝望→希望 | 黑暗中看到一束光、沉默中听到一个声音 |
| **时间** | 愤怒→疲惫 | 生气太久，身体自动关机 |
| **身体接触** | 崩溃→平静 | 有人握住她的手、有人从背后抱住他 |
| **反差信息** | 平静→恐惧 | 以为安全了，发现危险才刚开始 |
| **失去** | 任何→绝望 | 失去支撑自己的最后一根稻草 |
| **得到** | 绝望→希望 | 在最不可能的时候得到一丝帮助 |

### 转折写法要点

- 转折前留一个"安静的瞬间"：情绪的反转需要一个呼吸的停顿。"他站在那里。世界安静了一秒。然后——"
- 触发器越小越有力：不是惊天动地的大事，是一句话、一个眼神、一滴水。"压死骆驼的最后一根稻草"——稻草很轻，但刚好够了
- 转折后的"不适应"：情绪翻转后角色会有短暂的不适应。从恐惧到愤怒的瞬间，手还在抖，但牙齿已经咬紧了

## 附录：压强递进通用公式

### 三波递进法

任何情绪线都可以用三波递进：

1. **第一波**（建立）：建立情绪基调。读者知道角色在经历什么
2. **第二波**（加深）：比第一波强30-50%。角色以为到底了，没有
3. **第三波**（突破/反转）：比第二波强50-100%。角色突破底线，或者情绪性质发生变化

每波之间需要一个"喘息点"——不是完全放松，是压强稍微降低一点再继续加压。连续高压=麻木。有波谷的高压=痛感叠加。

### 句式与节奏配合

| 压强等级 | 句子长度 | 段落节奏 | 标点 |
|---------|---------|---------|------|
| 低 | 长句、从句多 | 慢，有铺垫 | 逗号、句号正常 |
| 中 | 中等 | 加快 | 更多顿号 |
| 高 | 短句、碎句 | 快，密集 | 句号变多，逗号变少 |
| 极高 | 单词/断句 | 一个词一行 | 破折号、省略号 |
| 崩溃后 | 恢复长句 | 突然变慢 | 回归正常标点 |

**示例——恐惧递进的句式变化：**

> 他听到走廊尽头传来声音。（长句，建立场景）
>
> 脚步声。越来越近。（中句，加快）
>
> 停了。（短句）
>
> 就在门外。（极短）
>
> 门把手——在转动。（破折号=呼吸中断）

### 写作检验清单

- [ ] 情绪线是否有明确的阶梯？（不能一步到位）
- [ ] 每个阶梯是否有动作/感官/环境支撑？（不能只有"他很害怕"）
- [ ] 是否有喘息点？（连续高压读者会麻木）
- [ ] 压抑与爆发之间是否有足够的反差？（反差=冲击力）
- [ ] 情绪转折是否有触发器？（不能突然转变）
- [ ] 是否避免了"不是A，是B"句式？（用动作和感官代替）
- [ ] 句式长度是否跟随压强变化？（高压=短句）
