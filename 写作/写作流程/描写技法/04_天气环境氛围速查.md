# 天气·季节·环境氛围速查

> 环境不是装饰，是情绪的放大器。天气和季节是免费的情绪工具。

---

# 一、天气与情绪映射

## 雨天

| 情绪 | 写法 | 感官重点 |
|------|------|---------|
| 忧伤/离别 | 细雨、雨帘、伞下的沉默 | 雨声、潮湿、冷 |
| 压抑/危机 | 暴雨、雷声、视线模糊 | 雷鸣、雨水打脸、衣服湿透 |
| 浪漫/暧昧 | 雨中奔跑、共伞、避雨 | 雨声背景、体温、靠近 |
| 净化/重生 | 雨后初晴、水珠、彩虹 | 空气清新、阳光穿透云层 |
| 孤独 | 窗外的雨、独坐、雨声 | 玻璃上的水痕、无人的街 |

**正例**：她站在屋檐下，雨帘把整条街隔成了模糊的色块。他的背影已经消失在雨里了。
**反例**：下雨了，她感到很难过。（天气+直白情绪=最偷懒的写法）

## 雪天

| 情绪 | 写法 | 感官重点 |
|------|------|---------|
| 纯洁/初见 | 初雪、雪花落在肩上 | 白色、安静、脚印 |
| 孤寂/寒冷 | 大雪封路、独行 | 呼出的白气、冻红的手指 |
| 温暖/对比 | 雪天屋内的壁炉/热茶 | 窗外雪vs屋内暖、对比感 |
| 悲壮 | 雪中战斗/牺牲 | 红色在白色上、血融雪 |
| 浪漫 | 雪中漫步、堆雪人 | 呵气成雾、手指触碰的冰凉 |

**正例**：路灯下雪花翻飞，她哈了一口气，白雾还没散，他就把围巾裹到了她脖子上。
**反例**：外面下雪了，很美，他们很开心。（三流作文水平）

## 晴天

| 情绪 | 写法 | 感官重点 |
|------|------|---------|
| 欢快/开始 | 阳光、蓝天、微风 | 光线温暖、影子清晰 |
| 残酷/反讽 | 大晴天出事（阳光下的罪恶） | 阳光刺眼、反差感 |
| 焦虑/等待 | 烈日、蝉鸣、汗 | 热、烦躁、漫长 |
| 日常/平静 | 午后的阳光、树荫 | 慵懒、微风、光斑 |

**正例**：阳光好得不像话。他接过死亡通知书的时候，外面的知了叫得正欢。
**反例**：今天天气很好，阳光明媚，万里无云。（小学生日记开头）

## 风

| 情绪 | 写法 |
|------|------|
| 不安/预兆 | 风突然停了，然后又刮起来 |
| 孤独/自由 | 风吹乱头发、衣角翻飞 |
| 威胁/紧迫 | 狂风、沙尘暴、台风前的低气压 |
| 变化/过渡 | 微风→大风→风暴（情绪递进） |

## 雾

| 情绪 | 写法 |
|------|------|
| 迷茫/不确定 | 看不清前方、方向感消失 |
| 恐怖/诡异 | 浓雾中有人影、声音从雾中传来 |
| 浪漫/梦幻 | 薄雾中的清晨、朦胧美 |
| 隐藏/秘密 | 雾中藏着什么、视线被遮挡 |

# 二、季节与情绪映射

## 春天

| 情绪 | 写法 | 典型意象 |
|------|------|---------|
| 新生/开始 | 发芽、花开、万物苏醒 | 樱花、柳絮、春雨、燕子 |
| 青春/恋爱 | 校园的春天、初恋 | 白衬衫、操场、风 |
| 希望 | 冰雪消融、第一朵花 | 绿色、温暖、光线变长 |

**适用题材**：校园恋爱、成长、新开始

## 夏天

| 情绪 | 写法 | 典型意象 |
|------|------|---------|
| 热烈/冲动 | 蝉鸣、汗水、荷尔蒙 | 短袖、冰棍、泳池 |
| 烦躁/压力 | 热得睡不着、考试季 | 烈日、空调、风扇 |
| 暧昧/亲密 | 夏夜、露台、星空 | 蚊香、啤酒、夜风 |
| 危机/爆发 | 暴风雨、雷电、洪水 | 乌云、闪电、暴雨前的闷 |

**适用题材**：青春、亲密、冒险、末日

## 秋天

| 情绪 | 写法 | 典型意象 |
|------|------|---------|
| 离别/感伤 | 落叶、枯黄、大雁南飞 | 银杏、枫叶、凉风 |
| 成熟/收获 | 果实、丰收、金色 | 柿子、稻田、夕阳 |
| 怀旧/思念 | 秋风起、旧照片 | 围巾、热饮、长袖 |

**适用题材**：情感、怀旧、离别

## 冬天

| 情绪 | 写法 | 典型意象 |
|------|------|---------|
| 温暖/亲密 | 屋内vs屋外的对比 | 火锅、热茶、围巾、拥抱 |
| 孤独/寒冷 | 独行在雪地 | 呼出的白气、空旷的街 |
| 危机/生存 | 暴风雪、冻伤、物资匮乏 | 冰、饥饿、黑暗（日照短） |
| 新年/希望 | 跨年、烟花、团圆 | 红色、灯笼、倒计时 |

**适用题材**：末日生存、情感、日常

# 三、一天中的时间与情绪

| 时间 | 情绪倾向 | 写法要点 |
|------|---------|---------|
| **黎明** | 希望、新生、转折 | 天边泛白、第一缕光、露水 |
| **上午** | 活力、行动、开始 | 阳光渐强、忙碌、节奏快 |
| **正午** | 残酷、最黑暗的时刻 | 阳光最烈=阴影最深（反讽利器） |
| **下午** | 慵懒、倦怠、日常 | 漫长、影子拉长、光线变暖 |
| **黄昏** | 离别、感伤、过渡 | 夕阳、晚霞、光线变暗 |
| **夜晚** | 秘密、亲密、危险 | 灯光、月光、黑暗、虫鸣 |
| **深夜** | 孤独、崩溃、真实 | 万籁俱寂、只有自己的心跳 |

**铁律**：黄昏是"离别"的专属意象。写分手/告别/死亡用黄昏，写重逢/希望用黎明。

# 四、环境与题材搭配速查

| 题材 | 最佳天气 | 最佳季节 | 最佳时间 | 原因 |
|------|---------|---------|---------|------|
| 恐怖 | 雾/雨/雷暴 | 秋/冬 | 深夜 | 感官受限+寒冷=不安 |
| 情感/恋爱 | 雨/雪/晴 | 春/秋 | 黄昏/夜晚 | 诗意+暧昧 |
| 悬疑/推理 | 雨/雾/阴天 | 秋/冬 | 深夜/黎明 | 压抑+视线受限 |
| 末日/生存 | 暴雨/暴雪/沙尘 | 冬 | 无特定 | 极端天气=生存压力 |
| 都市/日常 | 晴/多云 | 春/夏 | 上午/下午 | 日常感 |
| 亲密 | 雨/夏夜 | 夏 | 深夜 | 闷热+私密感 |
| 热血/战斗 | 晴/暴风雨 | 无特定 | 无特定 | 暴风雨=战斗高潮 |
| 搞笑 | 晴/极端反差天气 | 无特定 | 无特定 | 好天气+倒霉事=反差 |

# 五、环境描写长度控制

| 场景 | 建议长度 | 说明 |
|------|---------|------|
| 场景开头 | 1-3句 | 快速建立氛围，不要超过100字 |
| 情绪烘托 | 1-2句 | 穿插在对话/动作之间 |
| 转场过渡 | 1句 | "雨停了""天暗了" |
| 重点渲染 | 50-150字 | 只在关键情绪节点使用 |
| **铁律** | **不超过150字** | 超过150字的环境描写，读者会跳过 |

---

# 六、环境描写自检清单

- [ ] 天气/季节跟当前情绪匹配吗？（不能分手的时候写阳光明媚）
- [ ] 环境描写有没有超过150字？
- [ ] 环境描写是为情绪服务的吗？（不是的话就删掉）
- [ ] 有没有用至少1个感官细节？（不只是视觉，加上听觉/触觉/嗅觉）
- [ ] 同一个场景重复用了同一种天气吗？（不能每章都下雨）
- [ ] 环境变化有没有跟剧情变化同步？（情绪转折时天气也变）

# 七、环境描写常见错误

## 1. 环境跟情绪不匹配
**错误**：她失恋了，走在街上，阳光灿烂，鸟儿歌唱。
**正确**：她失恋了，走在街上，路灯一盏一盏亮起来，影子被拉得很长。

## 2. 环境描写太长
**错误**：300字描写一条街的建筑风格、行人穿着、店铺招牌、天空颜色。
**正确**：街边的梧桐树掉了一地的叶子，踩上去沙沙响。（1句话建立氛围）

## 3. 只用视觉
**错误**：红色的夕阳，黄色的树叶，蓝色的天空。
**正确**：风把枯叶吹到脚边，空气里有桂花的味道，远处有人在叫卖糖炒栗子。（视觉+嗅觉+听觉）

## 4. 每章都下雨
**错误**：第3章下雨，第5章下雨，第7章还是下雨。
**正确**：雨在第3章象征离别，在第5章换成雾（迷茫），第7章用晴天（希望）。

## 5. 环境描写放在不该放的位置
**错误**：角色正在被追杀，突然花500字描写周围的建筑。
**正确**：角色被追杀时只写"巷子尽头是死胡同"——环境为剧情服务。

## 6. 季节混乱
**错误**：第10章是秋天，第12章突然下雪了（没交代时间过了多久）。
**正确**：用季节变化暗示时间流逝——"入冬了，她还穿着秋天的外套。"
