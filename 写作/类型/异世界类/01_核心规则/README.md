# 异世界类 — 核心规则（铁律）

> 开写前必读。违反任何一条，整章重写。

## 铁律1：异世界必须有自洽的规则体系

**规则**：魔法/科技/种族/社会结构必须有内在逻辑，不能为剧情随意修改。
**正例**：火魔法消耗使用者体温，用多了失温致死——这条规则贯穿全书不变。
**反例**：上一章还怕火的怪物，这一章突然不怕了，因为剧情需要。
**检验**：如果把"异世界"换成"外国"你的故事还能成立，说明你的"异"不够深。异世界的差异要渗透到时间、空间、生命、死亡、力量、信仰每一个层面。

## 铁律2：穿越者的认知差是核心驱动力

**规则**：主角来自现代世界，其知识/习惯/价值观与异世界产生碰撞。认知差不是"我知道一切"，是"我用我的方式理解这个世界，而这个世界不买账"。
**正例**：他用现代急救知识救了一个骑士，但骑士的同伴认为他在施展邪术。
**反例**：穿越后立刻适应一切，没有任何文化冲突。
**限度**：现代知识碾压要克制——他能用杠杆原理撬门，但不能三天造出蒸汽机。他知道火药配方，但异世界缺硫磺矿。

## 铁律3：语言和文化差异不能被忽略

**规则**：异世界的度量衡、货币、社交礼仪、骂人话都应不同。
**正例**：当地不说"谢谢"，而是"愿你的影子不被风被带走"。
**反例**：异世界的人说一口流利的普通话，用"元"做货币。
**写法**：语言不通是穿越初期必须面对的障碍，不要用"突然就能听懂"偷懒。

## 铁律4：主角不能是唯一的"聪明人"

**规则**：异世界的原住民有自己的智慧体系，不能都是蠢人等着主角来教。
**正例**：当地猎人不用指南针，但靠苔藓方向从未迷路。
**反例**：所有原住民愚昧无知，主角一教就会。
**违反后果**：变成殖民者叙事，读者反感。

## 铁律5：金手指必须有限制和代价

**规则**：免费的力量不值钱。穿越者的外挂越强，代价越大。
**正例**：不死之身，但每次复活失去一段记忆；看到未来，但每次使用剧痛且不可改变。
**反例**：开局满级，无副作用，打谁都是碾压。
**设计原则**：金手指跟前世挂钩（前世厨师→味觉灵敏能判断魔物弱点），会成长（弱→强→新限制出现），是工具不是答案。

## 铁律6：情绪比战斗重要

**规则**：读者记住的是主角崩溃的那一刻，不是他赢的那一刻。
**正例**：菜月昴跪在地上哭——这是全书最真实的时刻。
**反例**：一章接一章打怪升级，没有情绪波浪。
**节奏**：每一段爽配一段虐。崛起不是直线往上爬，是波浪。

## 铁律7：穿越者的心理适应期不能跳过

**规则**：一个人到了陌生世界，前三天应该是害怕而不是兴奋。
**正例**：手心出汗、心跳加速、膝盖发软——五感先动，脑子后动。
**反例**：穿越第一天就开始装逼打脸。
**延续**：穿越的孤独感不能只在开头提一下就再也不管了。它会反复出现，只是表现形式不同。

## 铁律8：世界观要分层展开

**规则**：不要第一章就写百科全书。
- **第一层（生存层）**：前10章内——吃什么？危险来源？生存地位？
- **第二层（规则层）**：前30章内——魔法体系？社会结构？信仰？
- **第三层（秘密层）**：贯穿全书——真正的历史？穿越的真正原因？
**方法**：不要用旁白解释设定。让角色碰壁、犯错、从错误中学习。

## 铁律9：配角必须有独立于主角的动机

**规则**：每个有名字的配角都要有自己的目标、秘密和行动逻辑，即使主角不存在，他们的故事也在继续。配角不是NPC，不能只在主角需要时才"激活"。
**正例**：那个酒馆老板娘收留主角，不是因为善良，是因为她需要一个外来人帮她传一封不能被本地邮差看到的信。
**反例**：所有配角都是好人，主角一出现就无条件帮忙。
**违反后果**：世界变成主角一个人的舞台，配角沦为工具人，读者无法相信这个世界是"活的"。

## 铁律10：战斗不是万能解法

**规则**：不是所有冲突都能用打架解决。政治、经济、文化、信仰层面的矛盾，用拳头解决只会制造更大的问题。最精彩的一章可能是主角在谈判桌上被逼到绝路，而不是他打败了魔王。
**正例**：主角打赢了贵族的护卫，结果整个商会联合抵制他，比输了还惨。
**反例**：遇到问题→打架→赢→问题解决。重复三次读者就弃书。
**违反后果**：故事退化为战斗流水账，失去异世界题材"用智慧适应新规则"的核心魅力。

## 铁律11：感情线必须有阻力

**规则**：如果写感情线，不能"见面就倒贴"。喜欢的原因必须具体（他做了一件别人都不会做的事），发展必须有障碍（种族差异/阶级差异/语言不通/前世创伤）。
**正例**：她喜欢他，是因为他在所有人都逃跑的时候折回去救了她的弟弟——但她不敢承认，因为她的种族被禁止与人类通婚。
**反例**：女主第一天见面就脸红心跳，无条件支持主角所有决定。
**违反后果**：感情线沦为装饰品，读者跳过不看，浪费篇幅。

## 铁律12：异世界的"异"要渗透到日常细节

**规则**：不只是"天空是紫色的"这种大设定。细节层面的差异才是真正的沉浸感——他们怎么上厕所？打嗝在他们的文化里意味着什么？下雨的时候他们打伞还是用魔法？死了之后尸体怎么处理？
**正例**：这个世界的人不握手——他们用左手触碰对方的喉咙，表示"我对你没有敌意"。第一次经历这个的主角吓得后退了三步。
**反例**：异世界除了魔法和怪物，其他跟地球一模一样。
**违反后果**：异世界变成"换皮中世纪"，读者感受不到"异"，故事失去类型特征。

## 红线（绝对不可触碰）

1. **不要让穿越者成为"救世主模板"**——他不应该拯救全世界，他最多改变一小部分人的命运。全知全能的救世主会让所有配角、所有冲突都失去意义。
2. **不要跳过"穿越代价"的心理线**——穿越意味着永远离开家人、朋友、熟悉的一切。这个代价不能只在开头提一下就再也不管了，它要在每个重要节点反复回来。
3. **不要用系统提示替代角色思考**——如果主角每做一个决定都要靠系统分析利弊，读者看到的不是一个活人，是一个RPG玩家。
4. **不要让原住民崇拜穿越者**——可以有一个人觉得他特别，但"所有人都跪下喊勇者大人"是最懒的写法。原住民的正常反应应该是怀疑、警惕、甚至敌意。
5. **不要写"异世界没有的只有科技"**——如果异世界在道德、时间观、生死观上跟地球完全一样，那它就不是异世界，是主题公园。

## 写作边界

| 边界 | 说明 |
|------|------|
| 设定讲解比例 | 单章不超过10%用于设定说明，超过则通过角色行动展示 |
| 金手指使用频率 | 每章最多1次金手指关键使用，否则主角太强，张力消失 |
| 现代梗密度 | 每章最多2-3处现代文化引用，超过则出戏 |
| 系统提示占比 | 有系统的设定，系统文字不超过单章总字数的5% |
| 世界观展开速度 | 第一层（生存）10章内，第二层（规则）30章内，第三层（秘密）贯穿全书 |
| 战斗章占比 | 单卷战斗总篇幅不超过40%，否则情绪节奏单一 |
| 感情线推进速度 | 每卷最多1个感情关系质变（陌生人→朋友→暧昧→确认），太快则廉价 |
| 穿越者身份暴露 | 至少维持30章身份悬念，过早暴露则失去"隐藏身份"这条张力线 |
