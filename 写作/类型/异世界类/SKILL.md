# 异世界写作 Skill

> "一睁眼，发现自己变成了勇者/史莱姆/贵族废物/被召唤的工具人......"

---

## 速查表

### 核心张力公式

**熟悉的人 x 陌生的世界 x 不可回避的选择** -- "异"渗透到每个细节（时间/空间/力量/信仰）。

**五种核心张力**：穿越新鲜感（五感先动，脑子后动）/ 身份反差（行为习惯冲突，不是贴标签）/ 现代知识碾压（克制+合理+有代价）/ 异世界法则（法则=冲突来源）/ 回家渴望（早点回答想不想回去）

**一句话检验**：把"异世界"换成"外国"，故事还成立吗？如果成立，你写的是旅游攻略。

### 单章结构速查（探索-遭遇-成长型）

| 阶段 | 占比 | 关键动作 |
|------|------|----------|
| 新环境/新目标 | 20% | 到达新地点/接到任务，展示异世界独特规则 |
| 遭遇挑战 | 30% | 战斗/智力/社交困境，用现代思维碰壁 |
| 适应与突破 | 30% | 结合现代知识+异世界规则找到解法（或失败学习） |
| 成长与伏笔 | 20% | 获得新能力/盟友/信息 + 埋下更大世界谜团 |

**四幕大弧线**：穿越(10%) -> 适应(10-30%) -> 崛起(30-70%) -> 选择(70-100%)

### 角色压力点速查（Top 5）

1. **身份反差** -- 前世习惯与异世界规则冲突（鞠躬/强迫症/法医本能）
2. **知识局限** -- 现代知识不是万能的，缺材料/有副作用/不被接受
3. **归属感缺失** -- 想回家 vs 有了羁绊
4. **法则代价** -- 魔法/力量有代价（折寿/变丑/精神分裂）
5. **终极选择** -- 回家还是留下？救一人还是救所有人？

### 常见失败模式速查（Top 5）

1. **设定倾泻** -- 大段百科 -> 通过观察/犯错/被纠正建立认知
2. **全能穿越者** -- 什么都懂 -> 前世是具体的人（老师/程序员），有擅长不擅长
3. **NPC式原住民** -- 没有自己的烦恼 -> 每个角色有独立动机和秘密
4. **见怪不怪** -- 穿越后立刻适应 -> 前三天应该是害怕而非兴奋
5. **系统提示轰炸** -- 提示太频繁 -> 调味料不是主食

### 结尾类型速查

| 类型 | 特征 |
|------|------|
| 归家型 | 回到原来的世界，但已不是原来的人 |
| 留守型 | 选择留下，因有了不可舍弃的羁绊 |
| 开放选择型 | 面对终极二选一，答案留给读者 |
| 新生型 | 改变了一小部分世界，不是拯救全世界 |
| 循环型 | 发现穿越的真相，可能再次面临选择 |

### 开篇钩子模板

**A. 死后转生**：上一秒是卡车灯，下一秒是紫色的天空。
**B. 召唤仪式**：他在会议桌上打瞌睡，醒来面前是一群穿盔甲的人跪着喊他"勇者大人"。
**C. 直接险境**：他睁开眼，一只三米高的蜥蜴正盯着他，口水滴在他脸上。

### 对话禁忌速查

1. 不要写"他震惊了" -> 写他手里的杯子掉了
2. 不要写"这个世界真奇妙" -> 写他看到的具体东西
3. 不要写"她美得不可方物" -> 写他盯着她看了多久、说了什么蠢话
4. 不要写"他感到一股强大的力量" -> 写他的骨头在响、视线变红、耳鸣
5. 不要让反派解释自己的计划
6. 不要让系统提示太频繁
7. 不要跳过穿越者的心理适应期

## 角色速查（写作时直接参考）

> 完整版见 `人物/references/人物联动写作/异世界类.md`

### 范闲（腹黑 / 穿越者视角）
- 语言：现代口语吐槽和古代权谋话术自由切换，嬉皮笑脸下暗藏杀机
- 语气：表面嬉皮笑脸随意，内在冷静算计
- 动作：嬉皮笑脸时突然变脸（切换的瞬间制造反差）
- 用法：异世界穿越者的典型模板，现代感吐槽制造喜感+权谋话术推动剧情

### 萧云（乐观 / 胆大 / 系统加持）
- 语言："修炼难度大？没有关系，我有顿悟帮助，什么功法不能修炼？"
- 语气：激动时咧嘴一笑，自信到近乎莽
- 动作：挤进人群主动做决定（行动派，不等长辈安排）；系统触发后激动流泪
- 用法：异世界冒险中的热血莽撞担当，系统金手指配合乐观性格，适合开篇制造冲劲

### 江寒（隐忍 / 守护 / 暗中蓄力）
- 语言：面对强权不开口辩解，用沉默代替反驳，对话极少只在关键时刻说短句
- 语气：低沉压抑，每个字像从牙缝里挤出来
- 动作：紧紧抿着嘴几乎咬碎牙齿（忍耐标志）；战斗时眸子陡然睁开银色光辉闪耀（从忍到爆的瞬间切换）
- 用法：异世界中的隐忍型主角，被嘲笑铺垫越长，亮出实力时越爽
