# 异世界类 — 结构模板

> 本目录存放章节结构模板与叙事骨架。写作前用模板规划节奏。

## 单章结构（探索-遭遇-成长型）

```
### 新环境/新目标（前20%）
- 主角到达新地点或接到新任务
- 展示异世界的独特规则/文化

### 遭遇挑战（30%）
- 战斗/智力考验/社交困境
- 主角试图用现代思维解决，遇到规则限制

### 适应与突破（30%）
- 主角结合现代知识和异世界规则找到解法
- 或者失败——从失败中学习

### 成长与伏笔（20%）
- 获得新能力/新盟友/新信息
- 埋下更大的世界谜团
```

## 多章弧线（新手-融入-改变世界型）

```
第1-3章：穿越 → 生存 → 发现这个世界的核心矛盾
第4-6章：融入社会 → 获得第一个盟友 → 第一次真正的战斗
第7-9章：卷入更大的冲突 → 主角的现代价值观受到挑战
第10-12章：做出选择 → 改变一小部分世界（不是拯救全世界）
```

## 开篇钩子模板

**类型A — 死后转生**：上一秒是卡车灯，下一秒是紫色的天空。
**类型B — 召唤仪式**：他在会议桌上打瞌睡，醒来时面前是一群穿盔甲的人跪着喊他"勇者大人"。
**类型C — 直接身处险境**：他睁开眼，一只三米高的蜥蜴正盯着他，口水滴在他脸上。
