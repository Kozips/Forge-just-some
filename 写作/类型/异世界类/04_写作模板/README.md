# 异世界类 — 写作模板

> 章节结构模板与叙事骨架。写单章前先按模板分配篇幅，模板是骨架不是枷锁。

## 一、四幕总结构（全书级）

| 幕 | 占比 | 核心任务 | 节奏 |
|----|------|----------|------|
| 第一幕：穿越 | 前10% | 快速穿越 + 第一个抓人场景 | 快，不要超过两章还没穿越 |
| 第二幕：适应 | 10%-30% | 生存线/认知线/社交线三线并行 | 中速，建立世界观和角色关系 |
| 第三幕：崛起 | 30%-70% | 波浪式成长：爽配虐 | 有快有慢，每段胜利配一段失去 |
| 第四幕：选择 | 70%-100% | 终极二选一：两个都重要的东西之间选一个 | 收束加速，选择要有铺垫和代价 |

## 二、单章结构（探索-遭遇-成长型）

```
### 新环境/新目标（前20%）
- 主角到达新地点或接到新任务
- 展示异世界的独特规则/文化

### 遭遇挑战（30%）
- 战斗/智力考验/社交困境
- 主角试图用现代思维解决，遇到规则限制

### 适应与突破（30%）
- 主角结合现代知识和异世界规则找到解法
- 或者失败——从失败中学习

### 成长与伏笔（20%）
- 获得新能力/新盟友/新信息
- 埋下更大的世界谜团
```

## 三、日更章节结构（2000-3000字/章）

| 段落 | 字数 | 内容 |
|------|------|------|
| 上一章回顾钩 | 100-200字 | 用一个画面/声音/对话把读者拉回来 |
| 新信息 | 200-400字 | 这一章要给读者什么新东西 |
| 冲突/行动 | 800-1500字 | 核心场景，这是主菜 |
| 情绪点 | 300-500字 | 角色的内心反应 |
| 下一章钩子 | 100-200字 | 最后一句话让人"然后呢？" |

**钩子类型**：
- 悬念钩："门后面站着一个他绝对想不到的人。"
- 反转钩："但他错了。"
- 危机钩："就在这时，警报响了。"
- 情感钩："她转过身来，眼眶是红的。"

## 四、多章弧线模板

```
第1-3章：穿越 → 生存 → 发现这个世界的核心矛盾
第4-6章：融入社会 → 获得第一个盟友 → 第一次真正的战斗
第7-9章：卷入更大的冲突 → 主角的现代价值观受到挑战
第10-12章：做出选择 → 改变一小部分世界（不是拯救全世界）
```

## 五、战斗场景模板

| 阶段 | 段落数 | 内容 |
|------|--------|------|
| 对峙 | 2-3段 | 双方互相打量、试探 |
| 试探 | 3-5段 | 互相出招、摸底 |
| 爆发 | 5-8段 | 全力对决、节奏加快 |
| 转折 | 1-2段 | 意外发生、局势逆转 |
| 终结 | 2-3段 | 最后一击，不要太长 |
| 余波 | 2-3段 | 战斗结束后的身体和心理状态 |

**关键**：异世界战斗三个层次——物理（写动作不写招式名）→魔法（有施放过程和副作用）→心理（比谁先崩心态）。

## 六、开篇钩子模板

**A. 死后转生**：上一秒是卡车灯，下一秒是紫色的天空。
**B. 召唤仪式**：他在会议桌上打瞌睡，醒来时面前是一群穿盔甲的人跪着喊他"勇者大人"。
**C. 直接身处险境**：他睁开眼，一只三米高的蜥蜴正盯着他，口水滴在他脸上。
**D. 慢穿越**：墙角的影子角度变了。隔壁邻居的脸记不清了。然后——窗外的天是两个太阳。

## 七、角色成长弧线模板

设计主角成长线时参考无职转生的时间跨度法：

| 阶段 | 关键事件 | 情绪基调 |
|------|----------|----------|
| 幼年期（0-7岁） | 学基础、观察世界 | 好奇+温暖 |
| 少年期（7-12岁） | 第一次出门冒险、第一次杀人、第一次失去 | 热血→震撼 |
| 青年期（12-20岁） | 恋爱、战斗、流亡 | 节奏变快、情感浓烈 |
| 壮年期（20+） | 过去的因开始结果，年轻时的错找上门 | 沉稳+代价感 |

**原则**：时间跨度要大。异世界题材最好写长线，世界观需要时间展开。
