# 异世界经济体系设计指南

> 构建可信的异世界货币与生活成本体系。直接套用或改编。

## 一、货币层级参考

### 标准货币体系（可改编）

| 等级 | 货币名 | 购买力 | 用途 |
|------|--------|--------|------|
| 铜 | 铜币/铜钿 | 买一个馒头/一根蜡烛 | 日常零花 |
| 银 | 银币/银两 | 吃一顿饱饭/住一晚客栈 | 普通市民日用 |
| 金 | 金币/金锭 | 买一套衣服/付一个月房租 | 大额交易 |
| 铂/秘银 | 稀有金属币 | 买一匹马/一件魔法道具 | 高端市场 |

### 快速定价表

| 商品/服务 | 价格区间 | 备注 |
|-----------|----------|------|
| 一顿普通饭菜 | 铜币5-20 | 路边的便宜，餐馆的贵 |
| 一杯麦酒 | 铜币2-5 | 酒馆基本价 |
| 一夜普通客栈 | 银币1-3 | 含简单床铺 |
| 一件普通衣服 | 银币3-10 | 材质决定价格 |
| 一把铁剑 | 银币10-50 | 冒险者装备入门 |
| 一瓶初级治疗药水 | 金币1-5 | 视效果定价 |
| 驯服一匹普通马 | 金币5-20 | 品种差异大 |
| 一件魔法装备 | 金币50起 | 无上限 |

## 二、职业收入参考

### 普通人收入（月薪）

| 职业 | 收入 | 生活状态 |
|------|------|----------|
| 农民 | 勉强糊口 | 靠收成，年景差就饿死 |
| 城市工人 | 银币5-15 | 饿不死但存不下钱 |
| 店员/学徒 | 银币10-20 | 勉强维持 |
| 熟练工匠 | 银币20-50 | 中产起步 |
| 商人 | 不固定 | 有人暴富有人破产 |
| 士兵/卫兵 | 银币15-30 | 加战时补贴 |

### 冒险者收入（不稳定）

| 等级 | 任务报酬 | 生存状态 |
|------|----------|----------|
| 铜级（新手） | 银币10-100/任务 | 勉强维生，经常负伤 |
| 银级（老手） | 金币1-10/任务 | 稳定收入 |
| 金级（精英） | 金币10-50/任务 | 高收入但高风险 |
| 秘银级 | 金币50+/任务 | 社会地位高 |
| 英雄级 | 面议 | 传说级别 |

**关键写法**：冒险者收入不稳定。接不到任务时怎么办？受伤了没法干活怎么办？这些才是真实的生活压力。

## 三、生活成本与阶层差异

### 城市居民分层

**底层**（贫民窟/棚户区）：
- 房租：银币1-3/月
- 食物：黑面包、稀粥、一周吃一次肉
- 疾病率高，平均寿命短
- 典型场景：巷子里有污水、垃圾、流浪汉

**中层**（普通市民）：
- 房租：银币5-15/月
- 食物：普通面包、肉类、一周洗一次澡
- 有点小积蓄，但经不起变故
- 典型场景：街边有小店，周末有集市

**上层**（商人/小贵族）：
- 房租/房产：金币10+/月
- 食物：精制食材、调味料、每天能洗澡
- 有仆人、有社会地位
- 典型场景：三层小楼、有花园、有护卫

**顶层**（大贵族）：
- 不买房子——直接拥有整片领地
- 食物：特供、异国食材、魔法保鲜
- 仆人成群，等级森严

## 四、异世界特有经济要素

### 魔法材料市场

| 材料类型 | 价格驱动因素 |
|----------|--------------|
| 魔晶石 | 纯度决定价格，矿区垄断 |
| 药草 | 采摘季节、稀有度、种植难度 |
| 魔法兽材料 | 狩猎难度、部位价值、契约限制 |
| 魔法书/卷轴 | 作者名气、内容难度、年代 |

**写作用法**：主角在集市上看到有人在卖"月光花瓣"，价格是普通花的三倍——因为它只在月圆之夜采摘，能保存的时间只有三天。

### 奴隶市场（如果设定有）

- 奴仆种族：半兽人多用于体力劳动，精灵多用于艺术/仆从
- 价格因素：健康状况、技能、年龄、是否受过训练
- 法律问题：在某些国家是合法的，在另一些国家是重罪

**写作注意**：不要美化奴隶制，但可以写角色在道德困境中面对这个现实。

### 冒险者公会经济

- 公会抽成：任务报酬的10-20%
- 公会福利：情报、训练场、医疗折扣
- 公会等级：影响接任务权限和分成比例

## 五、快速经济设定检查表

写之前回答这些问题：

1. **这个世界的货币是什么材质？** （金属/贝壳/宝石/信用）
2. **汇率怎么算？** （铜:银:金 = 1:10:100 是常见设定）
3. **主角刚穿越时身上有多少钱？** （决定开局难度）
4. **一瓶治疗药水多少钱？** （决定主角能不能买得起）
5. **普通人一天吃几顿饭？** （决定主角饿肚子的频率）
6. **最贵的商品是什么？** （设定天花板，给读者目标感）
7. **有没有通货膨胀/货币贬值的危机？** （可以写成经济事件）

## 六、经济冲突场景素材

### 场景1：主角第一次买菜

> "这个多少钱？"
>
> "三个铜币。"
>
> 他翻了翻口袋——穿越时身上只有几枚不知道什么年代的铜板。"一个铜币行吗？"
>
> 老板娘看了他一眼。"哪来的？"
>
> 他指了指城门的方向。老板娘的眼神变了——那是看傻子的眼神。
>
> "两个铜币，最低了。"
>
> 他买下了那颗蔫了的卷心菜。这大概是他这辈子买过的最贵的一颗菜。

### 场景2：讨价还价

> "一百金币，不能再少了。"
>
> "八十。"
>
> "九十。"
>
> "成交。"
>
> 商人笑了。他也笑了。他知道这东西在王都能卖一百五，商人也知道他知道。但商人需要的是尽快脱手，他需要的是安全离开。两个人都觉得自己赚了。

### 场景3：异世界物价冲击

> 一瓶矿泉水大小的治疗药水，要十五个银币。
>
> 他在地球买一瓶矿泉水，最贵也就三块钱。
>
> 这个世界的穷人，连"三块钱"都拿不出来。
>
> 他把那瓶药水放回了架子上。

## 七、经济设定的常见错误

| 错误写法 | 正确写法 |
|----------|----------|
| 主角随便买点东西就花了几百金币 | 铜币和金币的价值差距要像元跟万元那么大 |
| 冒险者都是有钱人 | 大部分冒险者死于穷死，不是战死 |
| 物价几十年不变 | 战争/歉收/魔法材料枯竭都会影响物价 |
| 货币全是整数 | 现实生活中有找零的问题——"差三个铜币怎么办" |
| 主角从不担心钱 | 经济压力是穿越初期的核心冲突之一 |
