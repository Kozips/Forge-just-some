# 异世界类 — 专业知识库

> 构建世界观时查阅。覆盖世界类型、穿越机制、力量体系、社会结构四大板块。

## 一、常见异世界类型

| 类型 | 特征 | 代表作 | 写作要点 |
|------|------|--------|----------|
| 中世纪奇幻 | 骑士/魔法/领主/教会 | Re:Zero、无职转生 | 阶级矛盾是天然冲突源；教会可以是盟友也可以是压迫者 |
| 东方仙侠 | 修仙/宗门/飞升 | 凡人修仙传 | 等级森严的宗门体系；天劫/渡劫是生死节点 |
| 蒸汽朋克 | 机械/齿轮/维多利亚风格 | 最终幻想系列 | 魔法与科技的融合/冲突；工匠阶层崛起 |
| 废土重建 | 文明崩溃后的社会 | 盾之勇者 | 资源稀缺决定社会结构；信任是最稀缺的资源 |
| 异种族生态 | 非人种族为主的社会 | 哥布林杀手 | 种族间的生理差异导致文化差异；寿命差异影响关系 |
| 游戏化世界 | 系统/面板/等级/副本 | SAO、Solo Leveling | 规则透明但有隐藏机制；bug和漏洞是剧情来源 |
| 建国经营 | 从领地到王国的扩张 | 转生史莱姆 | 外交、经济、军事并行；"收小弟"靠利益交换而非拳头 |

## 二、穿越机制

| 方式 | 特征 | 开局张力 | 注意事项 |
|------|------|----------|----------|
| 硬穿越（猝死） | 地球死亡→异世界醒来 | 高：生死切换瞬间 | 死亡场景要快，别花三章写前世 |
| 召唤型 | 被魔法阵/仪式召唤 | 高：被利用的工具人 | 语言不通是第一道坎，不能跳过 |
| 转生型 | 死后变成婴儿/动物/物品 | 中：从零开始 | 幼年期可以写很多有趣的错位 |
| 慢穿越 | 世界渐变，慢慢发现不同 | 低开高走：似曾相识的陌生感 | 考验笔力，先写小细节不对再崩塌 |
| 游戏穿越 | 进入游戏世界/游戏变现实 | 中：规则已知 | NPC是否有意识可以挖很深 |
| 反穿越 | 异世界人穿越到地球 | 新颖：魔法在地球能用吗？ | 不懂现代生活的反差萌 |
| 双向穿越 | 两个世界可以来回 | 最复杂最有潜力 | 时间流速差、两边牵挂、穿越代价 |

## 三、力量/魔法体系设计要素

设计一个魔法/力量体系时，必须回答以下问题：

1. **来源**：魔力从哪来？（体内/环境/信仰/契约/血脉）
2. **分类**：有几种属性/流派？（攻击/治愈/召唤/无咏唱）
3. **代价**：用了会怎样？（体力/体温/寿命/记忆/精神）
4. **等级**：强弱怎么分？（学徒→正式→大师→传说）
5. **天花板**：最强能到什么程度？这个天花板是物理限制还是神的限制？
6. **破格**：有没有突破等级的方式？代价是什么？
7. **社会影响**：会魔法的人在社会里地位如何？受尊敬还是被恐惧？

**参考体系**：
- 无职转生：七大列强为天花板，魔法分攻击/治愈/召唤/无咏唱，有等级差异
- Re:Zero：魔法分六属性（火/水/风/土/阴/阳），加护是天赋而非修炼
- 转生史莱灭：技能有命名系统，"大贤者"是AI型外挂

## 四、社会结构参考

| 结构 | 特征 | 冲突来源 |
|------|------|----------|
| 封建贵族制 | 领主/骑士/农奴 | 阶级固化、贵族腐败、农民起义 |
| 城邦联盟制 | 独立城邦、松散联盟 | 城邦间贸易争端、共同敌人逼迫联合 |
| 神权制 | 教会掌权、异端审判 | 信仰vs理性、教会内部权力斗争 |
| 部落制 | 部落首领、图腾信仰 | 部落间世仇、外来文明入侵 |
| 魔法学院制 | 学院培养法师、等级分明 | 学术竞争、贵族子弟vs平民天才 |
| 冒险者公会制 | 自由冒险者、任务系统 | 公会政治、高阶冒险者的特权与责任 |

## 五、异世界经济基本参数

写之前想清楚：
- 货币体系：铜/银/金？汇率？一瓶药水多少钱？一顿饭多少钱？
- 职业收入：冒险者、农民、商人、士兵各赚多少？
- 稀缺资源：什么最贵？（魔法材料/稀有矿/特殊药草）
- 贸易路线：城市之间靠什么连接？商队有什么风险？
