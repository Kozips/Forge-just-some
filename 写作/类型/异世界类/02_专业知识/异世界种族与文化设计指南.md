# 异世界种族与文化设计指南

> 设计可信的异世界种族与文化差异。直接套用模板，按需改编。不是百科全书——是让你写的时候有东西可查。

---

## 一、种族设计模板

设计一个新种族时，必须回答以下 6 个问题：

| 问题 | 为什么必须回答 |
|------|----------------|
| 寿命多长？ | 寿命决定行为节奏。精灵活一千年，说话慢、不着急、对"死亡"的理解跟人类完全不同 |
| 生理差异？ | 尖耳朵只是外形。有没有人类没有的器官？能不能夜视？对什么过敏？ |
| 社会结构？ | 谁说了算？长老制？女王制？谁养孩子？ |
| 与人类的关系？ | 仇视？漠视？贸易伙伴？互相依赖？ |
| 核心禁忌？ | 他们最忌讳什么？说出来就是冒犯 |
| 他们怎么骂人？ | 骗人的内容暴露价值观。精灵骂人可能用"短命鬼"，矮人可能用"怕死的长耳仔" |

## 二、常用种族速查

### 精灵

| 维度 | 设定参考 |
|------|----------|
| 寿命 | 500-2000岁，300岁算成年 |
| 生理 | 尖耳、夜视、对铁过敏（设定可选）、嗅觉灵敏 |
| 社会 | 长老议会制，重传统，变化极慢 |
| 与人类 | 表面和平实际轻视。人类的"快节奏"在精灵眼里是"浮躁" |
| 核心禁忌 | 砍伐世界树、质疑长老决议、在精灵面前浪费食物（他们活得太久，见过饥荒） |
| 说话特点 | 语速慢，爱用远古典故，句子长但不啰嗦——每个词都有用 |

**写法要点**：精灵不是"人类加个尖耳朵"。他们看待时间的方式就不同——一个精灵说"最近"，可能指的是五十年前。跟人类对话时的翻译腔不是因为笨，是因为思维方式不同。

**场景示例**：
> "你多大了？"
>
> 她想了一下。"三百二十七。上个月过的生日。"
>
> "......三百二十七岁？"
>
> "怎么，你们人类不数数吗？"

### 矮人

| 维度 | 设定参考 |
|------|----------|
| 寿命 | 200-400岁 |
| 生理 | 矮壮、力量大、耐热、对毒素有抗性、酒量是人类五倍 |
| 社会 | 氏族制，按工匠行会分等级，族长选举产生 |
| 与人类 | 互相做生意，但不太看得起人类的工艺。"你们造的剑能用？" |
| 核心禁忌 | 质疑矮人的锻造手艺、偷矮人的酒、在矿洞里大声喊（怕塌方） |
| 说话特点 | 直来直去，不绕弯子，喜欢用"他妈的"做语气词 |

**写法要点**：矮人的骄傲不来自血统，来自手艺。一个矮人铁匠说"这把剑值五百金币"，不是在抬价，是在说实话。你砍价等于侮辱他的作品。

### 兽人/亚人种

| 维度 | 设定参考 |
|------|----------|
| 寿命 | 50-120岁（比人类短） |
| 生理 | 保留动物特征（狼耳/猫尾/角），体能优于人类，但对魔法抗性弱 |
| 社会 | 部落制，弱肉强食，强者的决定不需要投票 |
| 与人类 | 被人类歧视/奴役，或者反过来——取决于世界观 |
| 核心禁忌 | 摸尾巴（如果设定有尾巴的话——在很多兽人文化里，摸尾巴等于性骚扰） |
| 说话特点 | 句子短、直白、重语气轻语法。"干。""走。""你弱。" |

### 魔族/恶魔

| 维度 | 设定参考 |
|------|----------|
| 寿命 | 不确定（可能不死，但会"沉睡"） |
| 生理 | 角/翅膀/特殊瞳色，对圣光/神圣属性敏感 |
| 社会 | 等级森严，靠实力说话，背叛是家常便饭 |
| 与人类 | 敌对为主，但也有"不想打仗只想活着"的魔族个体 |
| 核心禁忌 | 说出真名（设定可选——在很多文化里，知道真名就等于掌控对方） |
| 说话特点 | 如果是贵族型魔族，说话比人类贵族还优雅；如果是底层魔族，粗鲁直接 |

## 三、文化碰撞速查表

写穿越者与原住民互动时，用这张表快速找到冲突点：

| 穿越者的认知 | 原住民的反应 | 冲突类型 |
|-------------|-------------|----------|
| "谢谢"只是礼貌 | 在某些国家"谢谢"是求婚用语 | 礼仪误解 |
| 洗澡是私密行为 | 某些种族公共沐浴是社交 | 生活习惯 |
| 男女平等是常识 | 贵族女性不能独自外出 | 价值观冲突 |
| 人死了要哀悼 | 某些种族认为死亡是回归，要庆祝 | 生死观差异 |
| 医院是救命的地方 | 这个世界治疗师要收费，穷人看不起病 | 医疗观念 |
| 孩子要上学 | 这个世界孩子十岁就当学徒 | 教育观念 |
| 粮食是基本人权 | 粮食是领主的，领主不给就得饿死 | 资源观念 |
| 宪法保护个人权利 | 领主的话就是法律 | 法律观念 |

**写法**：不要让穿越者一开始就全盘接受。让他经历愤怒（"这太不公平了"）→ 无力（"我改变不了"）→ 理解（"原来是因为......"）→ 行动（"我能做的只是......"）。

## 四、宗教/信仰体系设计

### 核心问题清单

1. **神是真的存在还是人为编造的？** 决定了信仰冲突的烈度
2. **教会扮演什么角色？** 慈善机构？情报网？军队？银行？
3. **有没有异端？** 谁定义"异端"？定义权在谁手里？
4. **神职人员的权力边界？** 教会能审判贵族吗？能收税吗？
5. **无神论者怎么活？** 在神真的存在的世界里，说"我不信神"是愚蠢还是勇气？

### 三种常见信仰体系

**A. 多神共存型**
- 多个神各管一摊（战神/丰收之神/死神）
- 冲突来源：神与神之间的对立，信徒之间的教派之争
- 写作要点：同一场战争，两边的士兵可能在祈祷同一个战神

**B. 一神独大型**
- 只有一个神，教会垄断解释权
- 冲突来源：教会腐败、异端审判、信仰vs理性的对抗
- 写作要点：穿越者的科学思维直接挑战教会根基

**C. 神已消亡型**
- 神曾经存在，但已经死了或离开了
- 冲突来源：信仰真空、旧神遗物争夺、新信仰的崛起
- 写作要点：穿越者的到来可以被解读为"新神降临"

## 五、语言与称谓设计速查

### 称呼系统

| 关系 | 常见称呼 | 注意事项 |
|------|----------|----------|
| 对贵族 | "大人""殿下""阁下" | 不同等级用不同称呼，用错是冒犯 |
| 对神职 | "神父""祭司大人""圣者" | 看教派，有些教派不喜欢被叫"大人" |
| 对冒险者 | 直呼其名或绰号 | 冒险者圈子比较随意 |
| 对长辈 | 看种族。精灵对长辈用"前辈"+"精灵语敬称" | 翻译腔不要过重 |
| 穿越者被称呼 | "异乡人""勇者大人""那个人" | 看召唤方的态度 |

### 骂人话设计

好的骂人话反映世界观：

| 骂法 | 反映的价值观 |
|------|-------------|
| "短命的人类" | 精灵视角——寿命是衡量标准 |
| "连剑都拿不稳的废物" | 战士社会——武力是衡量标准 |
| "连个铜币都算不清" | 商人社会——金钱是衡量标准 |
| "连神都不信的东西" | 神权社会——信仰是衡量标准 |
| "你妈是野兽人吧" | 种族歧视——血统是衡量标准 |

**不要用地球的骂人话**。"去你妈的"在异世界没有冲击力。用世界观内的骂法才有味道。
