# 异世界类 — 质检清单

> 完稿前逐项检查。打勾通过，未通过则修改。每章总分低于30分（满分50）需要大改。

---

## A. 通用检查项

- [ ] 无病句和错别字
- [ ] 人称一致（无视角混乱）
- [ ] 时间线连贯（无矛盾）
- [ ] 对话自然（无AI腔/书面腔）
- [ ] 每章有明确的情节推进

## B. 世界观自洽

- [ ] 魔法/能力系统的规则前后一致（前文设定不与后文矛盾）
- [ ] 金手指有限制和代价（不能无限制使用）
- [ ] 经济体系合理（货币、物价、收入自洽）
- [ ] 阶级/种族设定有内在逻辑（不为剧情随意修改）
- [ ] 异世界的"异"渗透到细节（不只是换个皮肤——时间/空间/生命/力量/信仰都要不同）

## C. 角色与认知差

- [ ] 主角不是"什么都会"——有不懂和犯错的时候
- [ ] 主角的现代知识有合理来源（不是全能百科，前世要有具体职业锚点）
- [ ] 异世界原住民有自己的智慧体系，不是等主角来教的NPC
- [ ] 配角有自己的动机和行动线（不只是为主角服务的工具人）
- [ ] 反派有合理的动机（不脸谱化）
- [ ] 穿越者的前世身份在行为习惯中有体现（口头禅/本能反应/知识盲区）

## D. 穿越感与心理真实

- [ ] 穿越的冲击感逐步消退（不能100章了还在惊叹每个新事物）
- [ ] 穿越者的孤独感有持续体现（不只是开头提一下）
- [ ] 语言/文化差异有体现（不只是换个名字说同样的话）
- [ ] 主角有"想家"或"不想家"的明确态度且有铺垫
- [ ] 心理适应期没有跳过（前三天应该是害怕而不是兴奋）

## E. 叙事节奏

- [ ] 穿越发生在前两章以内（不要花太多篇幅写前世日常）
- [ ] 设定是通过角色行动/犯错展示的，不是旁白/对话灌输
- [ ] 情绪节奏有波浪（不是一直爽或一直虐）
- [ ] 每章结尾有钩子（悬念/反转/危机/情感，让人想看下一章）
- [ ] 战斗场景不是纯打——有环境利用/身体反应/心理描写
- [ ] 最终冲突不能纯靠武力解决

## F. 对话与感官

- [ ] 穿越者的对话在不同场合有不同风格（紧张时/放松时/正式场合/独处）
- [ ] 原住民的措辞跟身份匹配（贵族文绉绉/农民直白/冒险者有行话）
- [ ] 没有"他震惊了""她美得不可方物"之类的偷懒描写
- [ ] 感官描写至少覆盖3种（视觉/听觉/触觉/嗅觉/味觉）
- [ ] 对话有"说一层藏一层"的层次感
- [ ] 粗俗和文雅的分寸把握正确
- [ ] 系统提示/面板不是调味料而是偶尔出现（不能太频繁）

## G. 按项自评打分（每项1-5分）

| 检查项 | 分数 |
|--------|------|
| 这一章有没有推进主线？ | |
| 角色的反应像正常人吗？ | |
| 有没有多余的设定讲解？ | |
| 对话有没有推进关系/冲突？ | |
| 穿越者的现代身份有没有体现？ | |
| 感官描写有没有超过2种？ | |
| 这一章结尾有没有钩子？ | |
| 有没有犯禁忌清单里的错误？ | |
| 情绪节奏是不是只有一种？ | |
| 配角有没有自己的动机和行动？ | |

**总分低于30分的章节需要大改。**
