# 猎奇类 · 质检清单

> 写完后逐项检查。不达标就改。

---

## 一、核心检查（必过）

- [ ] 猎奇元素是否有内在逻辑？（不能为了猎奇而猎奇）
- [ ] 是否有情感锚点？（没有情感的猎奇=垃圾）
- [ ] 角色对猎奇事件的反应是否写了？（不能只写事件不写反应）
- [ ] 节奏是否正确？（铺垫→暗示→揭示→余韵，不能直接上最猛的）
- [ ] 是否有留白？（不能把所有东西都写出来）

## 二、描写检查

- [ ] 连续3段以上纯猎奇描写？（必须穿插心理/环境/角色反应）
- [ ] 猎奇描写是否服务于剧情？（删掉这段剧情是否受影响？不影响就删）
- [ ] 是否有"认知颠覆"？（读者看完是否觉得"我从没想过可以这样"？）
- [ ] 黑色幽默是否适度？（太多变搞笑，太少变压抑）

## 三、红线检查

- [ ] 是否涉及未成年？（绝对不能）
- [ ] 是否有详细强奸/性暴力过程？（用留白处理）
- [ ] 是否有详细施暴过程？（用结果+反应替代过程）
- [ ] 猎奇是否过度到读者不适？（恶心≠好猎奇）

## 四、质量检查

- [ ] 猎奇场景是否有记忆点？（读者事后能否想起这个场景？）
- [ ] 是否有反差？（日常vs猎奇的反差越大越有力）
- [ ] 角色在猎奇事件后是否有变化？（不能事件过后角色毫无变化）
- [ ] 是否有"意料之外情理之中"？（最好的猎奇是事后觉得合理）
