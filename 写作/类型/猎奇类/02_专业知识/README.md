## 一、身体受虐 -- 生理常识

写身体受虐必须了解真实生理反应，否则写出来的是"编"不是"虐"。

### 饥饿时间线
| 阶段 | 感觉 |
|------|------|
| 第1-2天 | 胃在叫、嘴里发酸、注意力下降、手微微发抖 |
| 第3-5天 | 饥饿感反而减轻（省电模式）、嗜睡、反应变慢 |
| 第7天后 | 肌肉开始消耗、虚弱、头晕 |
| 第2周+ | 体重明显下降、免疫力下降、味觉退化 |

### 失眠时间线
- 24小时不睡：注意力严重下降，出现微睡眠
- 48小时：边缘视觉出现幻觉
- 72小时：无法分清现实和幻觉
- 长期：免疫力崩溃、精神错乱

### 失血时间线
- 400ml：头晕、心跳加快
- 800ml：面色苍白、手脚冰凉
- 1200ml：意识模糊、濒临休克
- 2000ml+：死亡

### 体温
- 35度（轻度失温）：发抖、手脚不听使唤
- 32度（中度）：发抖停止（身体放弃了）、意识模糊
- 28度以下（重度）：幻觉、反常脱衣（觉得热了）

### 高海拔
- 3000米以上开始反应，5000米以上严重反应，8000米以上"死亡区"

## 二、精神受虐 -- 心理控制机制

### 五大控制手段
| 类型 | 表现 | 典型话术 |
|------|------|----------|
| 情感勒索 | 用爱/愧疚绑架对方 | "我这么爱你，你怎么能这样对我" |
| 孤立控制 | 切断社交圈 | "你那些朋友都不是好人" |
| 间歇性强化 | 打一巴掌给一颗糖 | 暴力后突然极度温柔 |
| 煤气灯效应 | 否认对方的记忆和感受 | "你记错了"/"你太敏感了" |
| 经济控制 | 掌握财务 | "你赚的钱不够你花的" |

### 精神受虐四阶段
1. **警觉期**：感到哪里不对，试图"做个更好的人"来避免冲突。
2. **自我怀疑期**：开始觉得"也许真的是我的问题"，主动交出判断权。
3. **习得性无助期**：不再尝试改变，认为自己活该。长期压力导致免疫力下降、消化问题、慢性疲劳。
4. **解离期**：感觉自己在身体外面看自己，情感反应消失，严重的出现人格解体。

### 精神受虐的独特性（与身体受虐对比）
| 维度 | 身体受虐 | 精神受虐 |
|------|----------|----------|
| 证据 | 有伤痕 | 没有痕迹 |
| 持续性 | 伤会好 | 一直在 |
| 识别 | 别人看得见 | 只有自己知道 |
| 逃离 | 离开施虐者就结束 | 想法跟着你走 |
| 归因 | "他打我"（指向施虐者） | "是我不好"（指向自己） |

## 三、社会性受虐 -- 群体心理

### 群体暴力五阶段
1. **试探期**：小规模排挤、试探性嘲笑，看受害者怎么反应。
2. **确认期**：受害者没有有效反抗，确认"可以欺负"。
3. **升级期**：行为越来越过分——从言语到行动，从私下到公开。
4. **常态化期**：欺负这个人变成了"日常"，大家都觉得没什么。
5. **扩散期**：其他圈子也开始知道了——"就是那个人啊"。

### 群体施虐心理机制
- **责任分散**："又不是我一个人这么做的"
- **从众心理**：不参与反而会被排斥
- **等级维护**：欺负底层是确认自己不在底层的方式
- **去人性化**：一旦被贴上标签，就不再是"人"了

### 旁观者分类
- **沉默的大多数**：看见了但选择不介入
- **附和者**：跟着笑跟着说，参与程度不高但确实参与了
- **潜在帮助者**：想帮忙但不知道怎么做
- **完全没看见的人**：真的不知道或"选择不知道"

## 四、极限生存 -- 时间阶段

| 阶段 | 身体 | 精神 |
|------|------|------|
| 第1天 | 肾上腺素还在，体力可以 | 恐慌但有行动力 |
| 第2-3天 | 消耗开始显现 | 接受处境，焦虑但有目标 |
| 第4-7天 | 体力显著下降，小问题变大 | 疲惫、偶尔崩溃 |
| 第2周+ | 身体进入"节能模式" | 麻木、机械，或找到平衡 |
| 数月以上 | 身体适应了新的"正常" | 最大敌人是无聊和绝望 |

### 常用术语速查
- **PTSD**：创伤后应激障碍——闪回/噩梦/过度警觉
- **斯德哥尔摩综合征**：被害者对加害者产生依赖
- **习得性无助**：反复失败后放弃尝试
- **触发点**：某个细节突然唤起创伤记忆
- **解离**：感觉自己不在自己的身体里，像在看电影
