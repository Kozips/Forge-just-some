# 猎奇类 · 结构模板

> 猎奇写作的章节结构模板。写作前用模板规划节奏。

---

## 一、单章核心张力

每章必须回答一个问题：**这一章里，角色最怕什么？**

| 张力类型 | 说明 | 典型场景 |
|----------|------|---------|
| 身体张力 | 身体在失控/被改变 | 变异/改造/寄生 |
| 认知张力 | 世界不是我以为的那样 | 镜中世界/记忆篡改/规则空间 |
| 控制张力 | 我在被控制但我不知道 | 催眠/洗脑/精神控制 |
| 道德张力 | 我被迫做我不想做的事 | 生存游戏/道德实验 |
| 身份张力 | 我还是我吗 | 意识转移/记忆替换/身体改造 |

## 二、单章结构模板

### 模板A：渐进恐怖型
```
日常铺垫(15%) → 微小异常(15%) → 异常加剧(25%) → 恐怖揭露(20%) → 恐惧反应(15%) → 余韵/新悬念(10%)
```
- 关键：每一步都比上一步更不对劲，但角色试图用"合理"解释

### 模板B：认知颠覆型
```
正常世界(10%) → 发现矛盾(20%) → 尝试理解(25%) → 真相揭露(25%) → 认知崩塌(15%) → 新常态(5%)
```
- 关键：真相揭露时读者和角色一起震惊

### 模板C：控制与觉醒型
```
被控制状态(15%) → 裂缝出现(15%) → 怀疑积累(25%) → 觉醒时刻(20%) → 反抗(15%) → 代价(10%)
```
- 关键：觉醒不是瞬间的，是一个痛苦的过程

### 模板D：受虐递进型
```
初始状态(10%) → 第一次突破底线(20%) → 适应/麻木(15%) → 第二次突破(20%) → 彻底改变(20%) → 余韵(15%)
```
- 关键：每次突破都比上一次更极端，角色的反应也在变化

## 三、猎奇节奏控制

### 恐惧曲线
```
平静 → 不安 → 紧张 → 恐惧 → 极度恐惧 → 麻木 → 新的恐惧
```
- 不能一直极度恐惧，读者会麻木。需要波谷（喘息）再加压。
- 每个波峰要比上一个更高。

### 递进节奏
| 阶段 | 占比 | 关键动作 |
|------|------|---------|
| **锚定期**（前20%） | 建立"正常" | 让读者以为"我懂了" |
| **递进期**（20%-60%） | 三层递进 | 每层打破一个旧认知 |
| **崩塌期**（60%-80%） | 核心恐怖 | 一切旧认知被推翻 |
| **余韵期**（80%-100%） | 恐怖的后果 | 角色/世界的永久改变 |

## 四、开篇钩子模板

| 类型 | 模板 | 适用题材 |
|------|------|---------|
| 异常开场 | 第一句话就是不对劲的事 | 身体恐怖/认知颠覆 |
| 反差开场 | 用平静叙述写不平静的事 | 受虐/社会猎奇 |
| 规则开场 | 直接抛出荒诞的规则 | 规则空间/生存游戏 |
| 回忆开场 | 从事件后的角度回忆 | 所有猎奇 |
| 感官开场 | 用一个感官异常开头 | 身体恐怖 |

---

## 五、结尾类型

| 类型 | 特征 | 适用 |
|------|------|------|
| 永久改变型 | 角色/世界不可逆转地改变了 | 身体恐怖/认知颠覆 |
| 开放型 | 恐怖没有结束，只是暂停了 | 所有猎奇 |
| 觉醒型 | 角色从控制中觉醒，但代价巨大 | 控制/受虐 |
| 循环型 | 恐怖在重复，无法逃脱 | 规则空间/社会猎奇 |
| 余韵型 | 安静的场景，恐怖在沉默中延续 | 文学猎奇 |
