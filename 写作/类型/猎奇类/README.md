# 猎奇类 -- V2 Skill 目录

> 突破认知边界，制造"卧槽"时刻。身体变异、心理扭曲、社会极端、感官颠覆、黑色幽默。
> 覆盖：身体猎奇/心理猎奇/社会猎奇/感官猎奇/黑色幽默猎奇

| 目录 | 用途 |

## 使用流程

1. **开写前** -- 读 SKILL.md 了解全局，读 01_核心规则/ 确认描写边界和红线
2. **选型阶段** -- 从 02_专业知识/ 确定猎奇子类型，获取专业知识
3. **构建阶段** -- 从 03_素材库/ 挑选场景、角色、冲突模板
4. **写作阶段** -- 按 04_写作模板/ 搭结构，参考 05_技法研究/ 提升手法
5. **完稿阶段** -- 用 06_质检清单/ 逐项检查

## 核心张力

猎奇的核心驱动力：**认知颠覆的快感**。读者读到一个场景，大脑的既有认知框架被击穿，产生"原来还可以这样"的震撼。这种快感不来自恐惧，不来自悲伤，而来自"世界比你以为的更大/更荒诞/更不可思议"。

猎奇的五个维度：
- 身体维度：人的身体可以变成什么？（变异、改造、极限状态）
- 心理维度：人的心理可以扭曲到什么程度？（极端心理、认知颠覆）
- 社会维度：社会可以运转成什么样子？（极端制度、反常识规则）
- 感官维度：感官可以体验到什么？（超现实画面、通感、感官互换）
- 幽默维度：痛苦和荒诞可以怎样共存？（黑色幽默、苦中作乐）
