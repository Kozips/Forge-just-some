# 都市类写作 Skill

> 现代都市背景 + 真实感 + 社会切面 + 身边的戏剧。本质是"在熟悉的世界里发生不平凡的事"。

---

## 一、细分类型

| 子类型 | 核心冲突 | 代表作品 |
|--------|----------|----------|
| 职场商战 | 晋升/创业/商战博弈 | 《杜拉拉升职记》《创业时代》 |
| 校园青春 | 成长/暗恋/友情/反霸凌 | 《你好，旧时光》《最好的我们》 |
| 都市异能 | 隐藏超能力融入日常 | 《我有一座恐怖屋》《全球高武》 |
| 社会现实 | 阶层/家庭/代际冲突 | 《都挺好》《蜗居》 |
| 都市悬疑 | 连环案/阴谋/心理博弈 | 《白夜行》《沉默的真相》 |
| 都市情感 | 婚恋/出轨/三角/重逢 | 《我的前半生》《三十而已》 |
| 都市种田 | 田园生活/回乡创业/慢生活 | 《向往的生活》同人类 |
| 都市重生 | 重生回过去+信息差优势 | 《重生之神级学霸》 |

## 二、核心写作铁律

1. **真实感是生命线** — 地名可虚构但生活细节必须真实（地铁换乘、外卖时间、加班文化）
2. **对话即人设** — 职场人说话留三分，学生说话带网络梗，中年人说话绕弯子
3. **社会关系网即剧情网** — 同事/邻居/同学/前任，每条关系线都是潜在冲突源
4. **信息差驱动悬念** — 读者知道而角色不知，或角色知道而另一角色不知
5. **避免"悬浮"** — 不写你没了解过的生活，至少要采访/查阅/体验

## 三、场景引擎

### 职场场景
- 格子间工位、会议室博弈、电梯偶遇、加班外卖、年会酒局
- 五感：打印机墨粉味、键盘声、中央空调的嗡嗡、咖啡凉了

### 校园场景
- 教室后排、操场跑道、食堂窗口、图书馆靠窗、天台
- 五感：粉笔灰、课间操广播、小卖部的冰棍、黑板擦的灰尘

### 都市生活场景
- 便利店深夜、地铁早高峰、出租屋、合租厨房、深夜烧烤摊
- 五感：地铁报站声、外卖包装纸的窸窣、雨天打不到车的焦躁

## 四、人物原型库

| 原型 | 特征 | 常见冲突 |
|------|------|----------|
| 小镇做题家 | 努力但资源匮乏，自尊心强 | 职场天花板、社交壁垒 |
| 富二代 | 资源丰富但被家族期望绑架 | 自我认同、与父辈冲突 |
| 职场老油条 | 圆滑世故，深谙潜规则 | 遇到愣头青被打破平衡 |
| 独立女性 | 事业心强，不依附他人 | 职场歧视、婚育压力 |
| 社恐宅男/女 | 技术强但社交弱 | 被迫走出舒适区 |
| 斜杠青年 | 多重身份，时间碎片化 | 每个身份都半吊子 |

## 五、经典桥段模板

1. **职场逆袭** — 被看轻 → 拿到机会 → 证明自己 → 升职/打脸
2. **误会分手** — 甜蜜 → 误解产生 → 冷战 → 真相大白 → 追回
3. **创业起落** — 灵感 → 拉投资 → 团队危机 → 绝境翻盘
4. **身份暴露** — 隐藏身份 → 被人接近 → 关键时刻暴露 → 信任危机

## 角色速查（写作时直接参考）

> 完整版见 `人物/references/人物联动写作/都市类.md`

### 萧准（冷静·腹黑·杀伐果断）
- 语言："给你三分钟的时间筹钱""不好意思，我们没有继续谈下去的必要了"——全程陈述句，不用感叹号
- 语气：始终"淡淡"，越危险越平静，倒计时施压——"你已经浪费了五十八秒"
- 动作：接手机前先掂量份量——安全意识极强；勾嘴角表示钱到手了；抓人质做肉盾比对方拔枪还快
- 用法：都市硬汉、杀手型男主、冷静高手，用"淡淡"做默认语气，越平淡越有压迫感

### 彭晨（温和·细心·重义气）
- 语言："不好意思，请进请进""只要你们不嫌弃就行"——先道歉，永远谦逊
- 语气：内心"涌上一股温热的暖意，沉甸甸的，格外踏实"——不善言辞但感受力强
- 动作：发现自己失态后"连忙让开身子"——立刻调整；通过观察就推断出局势
- 用法：温和型男配、暖心角色，用"微微一愣"代替"感动"，不写感动二字感动自在其中

### 百列族长（深谋·慈父式操控）
- 语言："孩子，这么多年辛苦你了""你是上天派来带领百列家族走向繁盛之人"——先共情再施压
- 语气：用"叹息"开头降低防备，结尾"神秘的一笑"——掌控感
- 动作：先扶起对方再说目的——肢体关怀是话术的前奏；"每个夜晚我都会来倾听你的心里话"——长期监控
- 用法：操控型长辈、慈父式反派，用关怀做包装，比直接施压更恐怖
