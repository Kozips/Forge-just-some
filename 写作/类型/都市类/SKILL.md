# 都市类写作 Skill

> 现代都市背景 + 真实感 + 社会切面 + 身边的戏剧。本质是"在熟悉的世界里发生不平凡的事"。

---

## 一、细分类型

| 子类型 | 核心冲突 | 代表作品 |
|--------|----------|----------|
| 职场商战 | 晋升/创业/商战博弈 | 《杜拉拉升职记》《创业时代》 |
| 校园青春 | 成长/暗恋/友情/反霸凌 | 《你好，旧时光》《最好的我们》 |
| 都市异能 | 隐藏超能力融入日常 | 《我有一座恐怖屋》《全球高武》 |
| 社会现实 | 阶层/家庭/代际冲突 | 《都挺好》《蜗居》 |
| 都市悬疑 | 连环案/阴谋/心理博弈 | 《白夜行》《沉默的真相》 |
| 都市情感 | 婚恋/出轨/三角/重逢 | 《我的前半生》《三十而已》 |
| 都市种田 | 田园生活/回乡创业/慢生活 | 《向往的生活》同人类 |
| 都市重生 | 重生回过去+信息差优势 | 《重生之神级学霸》 |

## 二、核心写作铁律

1. **真实感是生命线** — 地名可虚构但生活细节必须真实（地铁换乘、外卖时间、加班文化）
2. **对话即人设** — 职场人说话留三分，学生说话带网络梗，中年人说话绕弯子
3. **社会关系网即剧情网** — 同事/邻居/同学/前任，每条关系线都是潜在冲突源
4. **信息差驱动悬念** — 读者知道而角色不知，或角色知道而另一角色不知
5. **避免"悬浮"** — 不写你没了解过的生活，至少要采访/查阅/体验

## 三、场景引擎

### 职场场景
- 格子间工位、会议室博弈、电梯偶遇、加班外卖、年会酒局
- 五感：打印机墨粉味、键盘声、中央空调的嗡嗡、咖啡凉了

### 校园场景
- 教室后排、操场跑道、食堂窗口、图书馆靠窗、天台
- 五感：粉笔灰、课间操广播、小卖部的冰棍、黑板擦的灰尘

### 都市生活场景
- 便利店深夜、地铁早高峰、出租屋、合租厨房、深夜烧烤摊
- 五感：地铁报站声、外卖包装纸的窸窣、雨天打不到车的焦躁

## 四、人物原型库

| 原型 | 特征 | 常见冲突 |
|------|------|----------|
| 小镇做题家 | 努力但资源匮乏，自尊心强 | 职场天花板、社交壁垒 |
| 富二代 | 资源丰富但被家族期望绑架 | 自我认同、与父辈冲突 |
| 职场老油条 | 圆滑世故，深谙潜规则 | 遇到愣头青被打破平衡 |
| 独立女性 | 事业心强，不依附他人 | 职场歧视、婚育压力 |
| 社恐宅男/女 | 技术强但社交弱 | 被迫走出舒适区 |
| 斜杠青年 | 多重身份，时间碎片化 | 每个身份都半吊子 |

## 五、经典桥段模板

1. **职场逆袭** — 被看轻 → 拿到机会 → 证明自己 → 升职/打脸
2. **误会分手** — 甜蜜 → 误解产生 → 冷战 → 真相大白 → 追回
3. **创业起落** — 灵感 → 拉投资 → 团队危机 → 绝境翻盘
4. **身份暴露** — 隐藏身份 → 被人接近 → 关键时刻暴露 → 信任危机

## 题材铁律

# 都市类 -- 题材铁律

> 开写前必读。违反任何一条，代入感崩塌。

---

## 一、现实感铁律（1-6）

1. **细节要"百度得到"** — 主角上班的地点、用的软件、吃的餐厅、开的车型、住的房型、用的手机品牌。越真实越代入。
2. **职业要"准确"** — 律师/医生/程序员/金融/公务员/销售/外卖员。每个职业有自己的语言、节奏、痛点。
3. **地理要"可定位"** — 主角住的小区、上班的地铁、聚餐的商场。读者能"去过"的地方=代入。
4. **收入/支出/存款要符合现实** — 程序员月薪 2-3 万不是 100 万。刚毕业月薪 1 万不是 5 万。错误数字=读者出戏。
5. **时间线要"现实"** — 早上 7 点起床、9 点上班、晚上 7 点下班、11 点睡觉。现实作息=真实。
6. **社会话题要"紧跟"** — 内卷/躺平/996/35 岁危机/相亲/房价/教育/医疗/外卖/电商。读者关心什么写什么。

---

## 二、情感与人际铁律（7-12）

7. **"日常摩擦"是主菜** — 同事/家人/亲戚/朋友/邻居/前任/相亲对象。这些关系中的小摩擦=剧情主轴。
8. **"三观差异"是冲突** — 主角认为对的，另一个人可能认为错。差异=对话张力。
9. **"代际冲突"是常见主题** — 父母催婚/重男轻女/工作 vs 考研/回老家 vs 留大城市。这些冲突有普遍共鸣。
10. **"职场潜规则"是加分项** — 老板画饼/同事甩锅/客户刁难/内部分裂/职场 PUA。读者爱看"太真实了"。
11. **"相亲/恋爱/婚姻"是主线** — 都市情感 = 相亲/恋爱/同居/结婚/离婚/出轨/复合。这些是都市文的核心。
12. **"原生家庭"是深层主题** — 父母关系/兄弟姐妹/亲戚/家族债务/家产。这些是人物动机的来源。

---

## 三、爽点与节奏铁律（13-18）

13. **"装逼打脸"要"小而真"** — 主角不一定要打豪门/总裁的脸。可以是"同事炫耀男朋友被主角揭穿" / "亲戚炫富被主角反杀"。
14. **"职场逆袭"是核心爽点** — 主角被同事欺负/被领导打压/被客户刁难/被新人背刺。逆袭=爽。
15. **"白手起家"是主流** — 主角没背景、没钱、没资源，靠能力/智慧/运气崛起。代入感强。
16. **"金手指"要"小而精"** — 都市文金手指不能太大（透视眼/读心术不算大）。可以是"专业知识+系统辅助"。
17. **"低调发财"是基调** — 主角可以有钱但不要张扬。低调+让别人震惊=高级爽。
18. **"日常生活"是节拍器** — 上班、下班、地铁、咖啡、外卖、健身、相亲。日常=主节奏。

---

## 四、对话与文风铁律（19-24）

19. **"职场黑话"是真实感** — OKR/KPI/PIP/拉齐/对齐/抓手/赋能/复盘。读者听到这些词=真实职场。
20. **"网络梗/流行语"是加分项** — yyds/绝绝子/emo/破防/摆烂/内卷/躺平/显眼包。读者爱看。
21. **"吐槽"是都市文灵魂** — 主角/同事/朋友之间的吐槽/嘲讽/阴阳怪气。都市人靠吐槽发泄。
22. **"微信式对话"是文风** — 短句/换行/语音条/表情包。不是长段大论，是碎片化对话。
23. **"内心独白"要简短** — 都市人没空长篇大论。要么是"他/她"片段闪念，要么是事后回忆。
24. **"接地气"是最高标准** — 不要用书面语/翻译腔/古风词。都市文的语言=口语+网络语。

---

## 五、价值与底线铁律（25-30）

25. **"三观正"是基础** — 主角不脚踏两只船/不出轨/不偷窃/不做违背底线的事。读者要能代入主角。
26. **"反派"也要有逻辑** — 同事使坏/领导打压/客户刁难——反派也有自己的压力和理由。
27. **"亲情"是软肋** — 父母/家人/亲戚/孩子的健康/教育/婚事。亲情是都市人最放不下的。
28. **"道德困境"是深度** — 主角为了升职要不要举报同事？为了买房要不要分手？道德困境=文眼。
29. **"现实主义"是基调** — 不写"霸道总裁爱上我"那种童话。写"我被房贷压得喘不过气"。
30. **"希望感"是底色** — 都市文即使很丧，最终要有一点希望。"再苦再累，明天还要上班"。

---

## 自检公式

```
都市感 = 现实细节 × 职业准确 × 情感共鸣 × 吐槽密度
```

**一句话检验**：把主角扔到一个非一线城市/不同职业/不同年代，故事还成立吗？成立=情节强，失败=靠"都市"硬撑。
