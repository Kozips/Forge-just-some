## 技法1：以物代情（物质细节替代情感抒发）

**来源**：《我的前半生》《蜗居》等都市剧文学改编
**手法拆解**：
1. 角色经历重大情感事件后，不写"他很难过"
2. 转而写一个无关的动作——擦桌子、叠衣服、反复按电梯按钮
3. 动作的节奏暗示情绪（快=焦躁，慢=麻木，停下=崩溃临界点）

> 她把衣柜里的衣服全部拿出来，一件一件叠好，又一件一件挂回去。最后她坐在地上，手里攥着一件灰色卫衣，上面还有一股烟味。她已经戒烟三个月了。

## 技法2：信息差制造焦虑

**来源**：《都挺好》《三十而已》
**手法拆解**：
1. 读者知道某个信息（如：伴侣出轨、公司裁员名单）
2. 角色不知道，还在正常生活
3. 每一个"正常"场景都变成等待炸弹爆炸的煎熬

## 技法3：对话中的"第三层意思"

**来源**：王朔、余华的都市小说对白
**手法拆解**：
1. 表面意思（说出来的）
2. 实际意思（想表达的）
3. 隐藏意思（自己都没意识到的）

> "你最近忙吗？"（表面：问候 / 实际：你怎么不联系我 / 隐藏：你还爱我吗）

## 技法4：空间对比暗示阶层

**来源**：《北京折叠》《蜗居》
**手法拆解**：
1. 同一时间，不同角色处于不同空间（A在写字楼开会，B在菜市场挑菜）
2. 不评论，只用空间对比让读者自己感受差距
