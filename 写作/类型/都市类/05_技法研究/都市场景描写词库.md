# 都市场景描写词库

> 都市写作的感官速查。按场景分类，写作时直接取用。

---

## 一、职场场景

### 办公室
- **视觉**：格子间隔板上的便利贴、显示器蓝光映在脸上、文件堆成山的桌面、转椅上磨出的屁股印
- **听觉**：键盘敲击声、打印机吞纸声、空调嗡嗡声、隔壁打电话的嗡嗡低语
- **嗅觉**：打印机墨粉味、咖啡凉了的酸味、外卖味、某人刚抽完烟回来的烟味
- **触觉**：鼠标滚轮的涩感、键盘油光的滑腻、冷气吹后颈的凉
- **物件**：工牌、打卡机、外卖袋、保温杯、充电线、会议白板

### 会议室
- **视觉**：投影仪的光柱、PPT翻页的蓝光、长桌上的矿泉水瓶、有人在桌下偷偷看手机
- **听觉**：激光笔的嗡嗡声、翻文件的沙沙声、有人清嗓子、空气突然安静
- **嗅觉**：会议室的空调味、茶水间的咖啡香飘进来、有人的香水味
- **物件**：激光笔、会议纪要本、矿泉水、遥控器、白板笔

### 电梯
- **视觉**：楼层跳动的数字、镜面里的倒影、有人按着关门键、手机屏幕的反光
- **听觉**：电梯运行的嗡嗡声、到楼层的叮声、沉默中的呼吸声
- **嗅觉**：密闭空间的混合气味（香水+汗+空气清新剂）
- **空间感**：拥挤时的近距离尴尬、独处时的密闭感、到达时的如释重负

## 二、都市生活场景

### 地铁
- **视觉**：车窗外隧道黑暗中的灯带、低头看手机的满车人、扶手上的手、车门开合
- **听觉**：报站声、车轮碾过铁轨的哐当声、耳机漏出来的音乐、关门提示音
- **嗅觉**：早高峰的汗味、有人的早餐味、空调的冷气味
- **触觉**：被挤在人群中的压迫感、扶手杆的冰凉、刹车时身体前倾
- **空间感**：早高峰的窒息、末班车的空旷、换乘通道的漫长

### 出租屋
- **视觉**：窗帘缝隙的光线、墙上的霉斑、外卖盒堆在桌角、充电线从床头垂下来
- **听觉**：隔壁的电视声、楼下便利店的音乐、水管的咕噜声、冰箱的嗡嗡
- **嗅觉**：泡面味、刚洗的衣服味、下水道的返味、窗台上的烟灰缸
- **触觉**：被子的廉价感、瓷砖地面的冰凉、水龙头的水垢粗糙
- **物件**：钥匙、外卖小票、电费单、旧手机、路由器闪灯

### 深夜加班
- **视觉**：整层楼只剩自己工位亮着、窗外的城市灯火、显示器的蓝光、外卖凉了
- **听觉**：键盘声在空旷办公室的回响、空调的低鸣、手机震动
- **嗅觉**：凉掉的外卖味、咖啡渣味、空旷办公室的灰尘味
- **情绪**：孤独感、疲惫、与城市的疏离

### 外卖/快递
- **视觉**：外卖小哥的电动车、保温箱、塑料袋上的水汽、外卖单上的字
- **听觉**：敲门声、"外卖到了"、拆塑料袋的声音
- **嗅觉**：打开外卖盖的热气和饭菜混合味
- **物件**：外卖单、一次性筷子、塑料袋、好评卡

## 三、社交场景

### 饭局
- **视觉**：圆桌上的转盘、酒杯碰在一起、有人夹菜有人说话、服务员添茶
- **听觉**：碰杯声、笑声、"来来来"、筷子碰碗声、有人在说场面话
- **嗅觉**：菜的热气、酒味、烟味（饭后）、火锅的辣味
- **潜台词**：敬酒的顺序=权力关系、谁先动筷=地位、谁买单=态度
- **物件**：酒杯、名片、烟、打火机、账单

### 酒局
- **视觉**：霓虹灯映在酒杯里、吧台的暗光、有人脸红了、烟雾缭绕
- **听觉**：碰杯声、音乐声、有人说话声音大了、笑声
- **嗅觉**：酒味、烟味、香水味混在一起
- **身体反应**：脸热、头晕、话多了、舌头大了、手不稳了
- **潜台词**：劝酒=试探、挡酒=保护、喝多了说真话

### 同学聚会
- **视觉**：发福的老同学、名牌包和普通包的对比、有人在拍照、有人在加微信
- **听觉**："你还记得吗""现在在哪发财""孩子多大了"
- **潜台词**：混得好的主动说话、混得差的沉默、炫耀=不自信
- **情绪**：怀旧、攀比、尴尬、物是人非

## 四、城市环境

### CBD
- **视觉**：玻璃幕墙反射天光、西装革履的人群、豪车停在路边、大堂的水晶灯
- **听觉**：高跟鞋敲地面、旋转门的嗡嗡、空调的均匀低鸣
- **嗅觉**：大堂的香薰、咖啡厅的咖啡香、清洁剂的味道
- **情绪**：光鲜、冷漠、竞争、压抑

### 城中村
- **视觉**：握手楼之间的窄缝、晾衣绳上的衣服、杂货店的招牌、电线乱拉
- **听觉**：炒菜声、小孩哭声、麻将声、摩托车引擎声
- **嗅觉**：油烟味、下水道味、垃圾味、廉价洗衣液味
- **触觉**：潮湿的墙壁、狭窄的楼道、不平的地面
- **情绪**：拥挤、烟火气、底层、生命力

### 夜市
- **视觉**：灯火通明的摊位、蒸汽升腾、串串在架子上、塑料凳子
- **听觉**：叫卖声、油锅的滋滋声、碰杯声、音乐声
- **嗅觉**：烧烤的烟、麻辣烫的辣味、臭豆腐的臭味混在一起
- **触觉**：塑料凳的硬、啤酒瓶的凉、纸巾擦嘴
- **情绪**：热闹、放松、烟火气、真实

### 深夜便利店
- **视觉**：日光灯的白光、货架整齐排列、收银台的关东煮、窗外的黑暗
- **听觉**：冰柜的嗡嗡声、门铃叮咚、收银机滴滴声
- **嗅觉**：关东煮的汤味、便当加热的塑料味
- **情绪**：孤独、深夜的安全感、城市的另一面

## 五、都市对话风格速查

### 职场黑话
- "这个我们对齐一下" = 我们商量一下
- "你拉个群" = 你建个群把人加进来
- "赋能" = 帮助/支持
- "闭环" = 做完整/有始有终
- "颗粒度" = 细节程度
- "对齐颗粒度" = 统一细节标准
- "抓手" = 切入点/方法
- "沉淀" = 积累/总结
- "复盘" = 总结反思
- "拉通" = 沟通协调

### 社交话术
- "改天请你吃饭" = 不会请
- "下次一定" = 不一定
- "在吗？" = 要借钱/要帮忙
- "最近忙吗" = 有事找你
- "你瘦了" = 客套
- "还好吧" = 不好但不想说

### 网络用语
- "绝绝子" = 太厉害了
- "YYDS" = 永远的神
- "破防了" = 被触动了
- "社死" = 尴尬到想消失
- "emo了" = 情绪低落
- "摆烂" = 放弃努力
- "卷" = 过度竞争
- "躺平" = 放弃竞争

## 六、都市物件情感速查

| 物件 | 承载的情感 | 使用场景 |
|------|-----------|---------|
| 钥匙 | 归属感/被锁在外面的排斥 | 出租屋/回家/分手后还钥匙 |
| 手机 | 联系/隔离/秘密 | 消息已读不回/偷看手机/手机没电 |
| 烟 | 压力/孤独/社交 | 深夜阳台/饭后/等人 |
| 咖啡 | 提神/仪式感/社交 | 加班/约会/独处 |
| 名片 | 身份/社交/野心 | 交换名片/扔掉名片/收藏名片 |
| 合同 | 承诺/利益/背叛 | 签合同/毁约/谈判 |
| 外卖 | 独居/将就/孤独 | 一个人吃饭/深夜外卖 |
| 工牌 | 身份/归属/被解雇 | 入职/离职/被收走工牌 |
| 充电线 | 依赖/焦虑/连接 | 手机没电焦虑/借充电线 |
| 耳机 | 隔离/自我/逃避 | 戴耳机=不想说话/音乐治愈 |
