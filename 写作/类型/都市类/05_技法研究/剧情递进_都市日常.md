# 剧情递进技法 —— 都市日常

> 来源：剧情递进手法_五种类型全覆盖
> 四个维度：冲突升级 · 信息解密 · 情绪压强 · 格局扩展

---

## 两种典型递进套路

**套路A：个人困境→人际纠葛→社会浪潮→时代抉择**

主角面对的不只是自己的问题，是整个时代压过来的力。从"我怎么活下去"变成"我在时代洪流里站哪边"。核心张力是个人意志vs时代裹挟。

**套路B：小目标→大目标→代价→取舍**

每次达成一个目标都伴随着失去。升职了但失去了朋友。赚到钱但失去了爱情。事业成功了但失去了初心。递进的本质是"得到和失去的螺旋"。

---

## 具体例子：《平凡的世界》递进结构

> 第一层：孙少平在农村的贫困生活。冲突是"吃饱饭"。信息是黄土高原的生存法则。情绪是压抑但有希望。格局是一个村庄。
>
> 第二层：进城打工。冲突从物质贫困变成精神孤独——城里人看不起农村人。信息是城乡差距的残酷真相。情绪是自卑与倔强并存。格局是一座县城。
>
> 第三层：时代变革。冲突是整个社会在转型——有人抓住机会飞黄腾达，有人被碾碎。信息是改革浪潮中每个人的抉择和命运。情绪是悲壮。格局是一个时代。

**递进拆解：**
- **冲突升级**：温饱 → 尊严 → 阶层 → 时代命运
- **信息解密**：农村生存法则 → 城市潜规则 → 社会运行逻辑 → 时代真相
- **情绪压强**：压抑中求生 → 孤独中倔强 → 挣扎中成长 → 苍凉中坚守
- **格局扩展**：一个村 → 一座城 → 一个省 → 一个时代

## 递进节奏

核心驱动力：**人情世故**。匀速，与现实时间感同步。格局扩展路径：家庭→职场→行业→时代。

| 阶段 | 占比 | 关键动作 |
|------|------|---------|
| **锚定期**（前20%） | 建立社会关系/生活背景 | 让读者以为"我懂了" |
| **递进期**（20%-60%） | 三层递进 | 每层打破一个旧认知，建立一个新认知 |
| **崩塌期**（60%-80%） | 核心反转 | 所有旧规则/认知被推翻，主角必须从零重建 |
| **终局期**（80%-100%） | 收束或开放 | 给出终极答案，或留下"真正的问题还没解决" |

## 写作检验清单

- [ ] 每一章是否比上一章的冲突强度更高？（不能原地踏步）
- [ ] 每次信息揭露是否同时制造了新悬念？（揭示≠结束）
- [ ] 情绪是否有波谷？（连续高压读者会麻木，需要喘息再加压）
- [ ] 格局扩展时主角的代价是否同步增加？（免费的升级=无聊）
- [ ] 读者在第N章是否还能回忆起第1章的悬念？（伏笔回收=信任）
- [ ] 递进是否有"不可逆"节点？（至少一个无法回头的转折点）
