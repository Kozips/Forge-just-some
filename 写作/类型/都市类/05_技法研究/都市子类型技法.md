# 都市子类型技法

> 都市类内部的六大子类型，每种都有独特的张力来源、节奏模板和必写场景。不是"换个背景就行"，而是"换一种让人欲罢不能的方式"。

---

# 一、都市职场（商战/办公室政治）

## 核心张力

**能力与资源的错配**——你够强，但你没有位置；你有位置，但你不够强。职场文的本质是"用有限的筹码博弈无限的可能"。

读者欲罢不能的原因：每个人都经历过"明明是我做的，功劳被别人拿走"的瞬间。职场文把这种日常愤怒戏剧化，让读者在虚构中获得"迟来的正义"。

## 两种套路

### 套路A：逆袭打脸流

**公式**：被轻视 → 隐藏实力 → 关键时刻展露 → 打脸 → 更大的敌人出现

**递进拆解**：
1. 第一层打脸：同事级别的轻视（"这个新人能干什么"）
2. 第二层打脸：中层管理的忽视（"这个项目不需要你参与"）
3. 第三层打脸：高层的误判（"这个部门可以裁掉"）
4. 每一层打脸后，主角的敌人升级，筹码也升级

**示例**：
> 她在部门会议上提了三个方案，没有人听。主管转头问老张："你觉得呢？"老张把她第三个方案换个说法复述了一遍，所有人点头。
>
> 她打开笔记本，在备忘录里写下："方案三，9月12日，原创：我。"然后关上笔记本，说了句"我没意见"。
>
> 两个月后，方案三落地出了问题。审计组调取了会议记录和邮件往来。她那份备忘录的时间戳比老张的邮件早了四天。

### 套路B：办公室政治博弈流

**公式**：站队困境 → 信息收集 → 暗中布局 → 渔翁得利

**递进拆解**：
1. 第一局：被迫选边（两个领导在争一个项目，你跟谁？）
2. 第二局：被当棋子（你以为自己在选边，其实被人利用）
3. 第三局：反客为主（利用别人对你的利用，完成自己的布局）
4. 关键技法：主角不靠"能力碾压"，靠"信息差"和"人心判断"

**示例**：
> 副总找他谈话："小林，这个项目你帮我盯一下。"
>
> 他知道副总和总监在争明年的预算分配。盯项目=站队。他笑着说"好的王总"，转头给总监发了条微信："李总，副总让我协助项目，有什么需要注意的您随时指点。"
>
> 两边都以为他是自己人。他两边的信息都在收。最后项目汇报那天，他把两方的矛盾点做进了同一份PPT里，让两个领导在CEO面前自己吵了起来。他全程没说一句话。
>
> 一个月后，他调去了新成立的战略部。直属汇报对象：CEO。

## 叙事节奏模板

```
日常压迫（3章）→ 转机出现（1章）→ 准备反击（2章）→ 关键对决（1章）→ 短暂胜利 → 新的更大压迫（2章）→ 循环升级
```

**节奏要点**：职场文的"松"不是放松，是"暂时安全但暗流涌动"；"紧"不是打斗，是"一个决策失误就满盘皆输"。

## 经典场景库（5个必写场景）

1. **述职/汇报场景**——当众展示成果，底下坐着决定你命运的人。PPT翻页声=心跳声。
2. **茶水间/走廊偶遇**——非正式场合的"不经意"对话，往往是信息战的主战场。
3. **裁员/优化谈话**——HR微笑着说"我们聊聊"，主角的工位上多了一个纸箱。
4. **酒局/饭局**——酒杯举起来的时候，真正的谈判才开始。谁给谁倒酒=权力关系。
5. **深夜加班独处**——办公室只剩键盘声和外卖凉掉的味道。孤独感+使命感+疲惫感三重叠加。

## 对话风格特征

- **留三分**：职场人不会把话说满。"这个方案不错"可能是认可，也可能是"也就那样"
- **暗藏机锋**：表面上在讨论业务，实际上在争夺资源、推卸责任、邀功
- **层级语言差异**：对上级用"您看这样是否合适"，对下级用"这个你处理一下"，平级用"咱们商量商量"
- **邮件/微信体**：正式场合用邮件（留证据），私下用微信（试探态度），口头说的不算数

## 感官描写重点

- 视觉：会议室玻璃墙后面的投影、工位上堆积的文件、电梯楼层数字的跳动
- 听觉：键盘敲击声、打印机运转、会议室里突然的沉默
- 嗅觉：打印机墨粉味、咖啡凉掉后的苦味、中央空调循环的闷
- 触觉：鼠标滚轮的阻感、签字笔在合同上的摩擦、握手时对方的力度

## 代表作品分析

| 作品 | 核心技法 | 值得学的点 |
|------|----------|-----------|
| 《杜拉拉升职记》 | 能力证明+人际周旋 | 用具体工作细节建立专业感 |
| 《创业时代》 | 创业起落+团队博弈 | 融资谈判的节奏感 |
| 《半泽直树》 | 复仇打脸+银行政治 | "以牙还牙"的爽感设计 |
| 《新闻女王》 | 媒体职场+女性博弈 | 新闻现场的紧迫感 |

# 二、都市情感（恋爱/婚姻/家庭）

## 核心张力

**亲密关系中的距离感**——越亲近的人，越知道怎么伤害你。情感文的本质是"两个不完美的人试图在同一个空间里共存"。

读者欲罢不能的原因：每个人都有一段"差一点就成了"或"明明相爱却过不下去"的经历。情感文提供的是"如果当时"的代偿。

## 两种套路

### 套路A：误会-分离-重逢流

**公式**：甜蜜期 → 误会产生 → 冷战/分手 → 各自成长 → 真相大白 → 追回/被追回

**递进拆解**：
1. 甜蜜期不能太长（1-2章），否则读者腻
2. 误会必须"合理"——不是"你为什么不解释"，而是"那个情境下，任何正常人都会误解"
3. 分离期要有"各自成长线"，不能一方等另一方
4. 重逢的爽感来自"你变了，我也变了，但我们还是选择了彼此"

**示例**：
> 她看到他手机里那条消息："昨晚很开心，下次再约。"发件人备注是"小雨"。
>
> 她没问。她把手机放回原处，去厨房把做好的菜倒进了垃圾桶。他出来的时候，桌上只剩白米饭。
>
> "怎么了？"
>
> "没什么，菜糊了。"
>
> 三天后她搬走了。他打了四十七个电话，她一个没接。
>
> 三个月后，她在公司团建的KTV里，听到同事说："林哥去年不是给你捐了骨髓吗？你俩没联系了？"她愣住了。同事继续说："就是那个小雨啊，他妹妹。"

### 套路B：婚姻围城流

**公式**：表面完美 → 裂缝出现 → 互相伤害 → 去留抉择 → 重新定义关系

**递进拆解**：
1. 开篇用"完美生活"建立期待（有房有车有孩子）
2. 裂缝从"小事"开始（谁接送孩子、谁做饭、谁的父母来住）
3. 争吵不写爆发，写积累——"第十次说'随便'的时候，她真的随便了"
4. 结局不一定是复合或离婚，可以是"达成新的相处模式"

**示例**：
> 结婚第七年，他们已经不吵架了。
>
> 不是感情好，是懒得吵。他加班到十一点回家，她已经睡了。早上他出门的时候，她还没醒。微信聊天记录停在三天前的"酱油买了吗""买了"。
>
> 周末他难得在家，她带孩子去上兴趣班。他在沙发上刷了一下午手机。晚上孩子睡了，她在客厅看剧，他在卧室打游戏。
>
> 中间隔着一堵墙和七年的沉默。

## 叙事节奏模板

```
相遇/甜蜜（2章）→ 误会/裂痕（2章）→ 分离/冷战（3章）→ 各自生活（2章）→ 重逢/转折（1章）→ 和解/新关系（2章）
```

**节奏要点**：情感文的"紧"不是冲突，是"差一点就说出口"的遗憾；"松"不是甜蜜，是"以为放下了却在某个瞬间崩溃"。

## 经典场景库（5个必写场景）

1. **雨中/深夜分手**——城市背景下的情感爆发，雨声盖过哭声，出租车尾灯消失在路口。
2. **偶遇前任**——商场/地铁/公司年会，突然看到那个人身边站着别人。
3. **深夜电话**——喝醉了/失眠了/生病了，拨出去又挂断的那个号码。
4. **家长介入**——"我不是反对你们，我只是觉得她配不上你。"
5. **同居日常**——从甜蜜到习惯到厌倦，牙刷从两支变成一支的过程。

## 对话风格特征

- **欲言又止**：情感文的对话精髓是"没说出口的话比说出口的更重要"
- **反话**：说"我没事"的时候最有事，说"随便"的时候最不随便
- **语气词即情绪**："嗯。"是冷淡，"嗯"是敷衍，"嗯~"是撒娇，"嗯？"是质疑
- **沉默比对话更重**：一句"我们聊聊"之后的三秒钟沉默，比任何台词都有力

## 感官描写重点

- 视觉：空了一半的衣柜、床头柜上成对的杯子只剩一个、微信聊天记录的最后一条
- 听觉：深夜的冰箱嗡嗡声、雨打窗户、手机震动但不是那个人
- 嗅觉：对方留下的洗衣液味道、冰箱里过期的牛奶、枕头上渐淡的气息
- 触觉：空荡荡的床另一半、冬天没人暖的被窝、握手时对方抽回的手

## 代表作品分析

| 作品 | 核心技法 | 值得学的点 |
|------|----------|-----------|
| 《我的前半生》 | 婚姻崩塌+女性独立 | 经济依附关系的描写 |
| 《三十而已》 | 群像+三种婚姻困境 | 不同阶层女性的平行叙事 |
| 《东京爱情故事》 | 错过+成长 | 永尾完治的犹豫写得真实 |
| 《父母爱情》 | 年代婚姻+性格互补 | 日常细节堆砌出深情 |

# 三、都市悬疑（都市背景的推理/犯罪）

## 核心张力

**熟悉世界中的异常**——你每天走的那条路、坐的那部电梯、住的那栋楼里，藏着你不知道的事。都市悬疑的本质是"安全感的崩塌"。

读者欲罢不能的原因：都市悬疑比古装悬疑更有代入感——"这事可能发生在我身边"。

## 两种套路

### 套路A：连环案件+日常渗透流

**公式**：日常被打破 → 第一个案件 → 发现规律 → 与自身产生关联 → 真相揭示

**递进拆解**：
1. 案件从"与我无关"逐渐变成"与我有关"
2. 每个案件独立成谜，但共享一个底层逻辑
3. 侦探/主角的日常生活被案件侵蚀（失眠、多疑、关系紧张）
4. 真相揭示时，读者发现"原来第一章就有线索了"

**示例**：
> 小区连续三周丢快递。不是丢件，是签收了但收件人说没收到。物业查监控，快递员确实放到了门口。
>
> 第四周，3号楼的王阿姨在业主群里说："我看到有人半夜在楼道里走来走去。"
>
> 第五周，5号楼的独居老人死了。警察说是意外摔倒。但她的门口，放着一个不属于她的快递盒。
>
> 盒子里是一双鞋。36码。老人穿38码。
>
> 主角翻开快递单，收件人一栏写着她的名字。

### 套路B：身边人即嫌疑人流

**公式**：信任建立 → 异常迹象 → 怀疑→自我否定 → 证据累积 → 真相崩塌

**递进拆解**：
1. 嫌疑人是读者和主角都信任的人（伴侣/好友/家人）
2. 每一条线索都可以有两种解释——"可能是巧合"vs"太巧了"
3. 主角的内心挣扎是核心——"我不想相信，但证据在那"
4. 真相不一定是"TA是坏人"，可以是"TA有不得已的苦衷"

**示例**：
> 他老婆最近总是加班。
>
> 不是那种正常的加班。是手机朝下放在桌上、洗澡时带着手机、接电话时走到阳台的那种加班。
>
> 他告诉自己别多想。结婚八年了，要有基本的信任。
>
> 直到他在她包里看到了一张酒店的发票。时间是上周三，她说在公司开会的那天。
>
> 他把发票放回原处。晚上吃饭的时候，他给她夹了一筷子菜，说"最近辛苦了"。
>
> 他注意到她夹菜的手，抖了一下。

## 叙事节奏模板

```
日常（1章）→ 异常出现（1章）→ 调查/怀疑（3章）→ 误导方向（2章）→ 真相接近（2章）→ 高潮揭示（1章）→ 余波（1章）
```

**节奏要点**：都市悬疑的"松"是日常生活的假象；"紧"是"你身边最安全的人/地方突然变得可疑"。

## 经典场景库（5个必写场景）

1. **深夜独行**——空旷的地下车库、只有脚步声回响的走廊、电梯门打开时里面已经站着一个人。
2. **监控回放**——屏幕上的画面一帧一帧过，突然在某一帧停下来，放大，看到了不该看到的东西。
3. **翻找线索**——在对方不在的时候翻TA的抽屉/手机/电脑，每一秒都可能被回来的人撞见。
4. **对峙场景**——两个人面对面坐着，一个知道真相，一个在演戏，读者不知道谁是哪一个。
5. **安全空间被侵入**——家里被人动过、车里多了不属于自己的东西、手机被翻过。

## 对话风格特征

- **信息不对称**：对话双方各掌握一部分真相，每句话都是试探
- **日常对话中的裂缝**：突然的停顿、回避某个话题、答非所问
- **反问做武器**："你确定吗？"比"你撒谎了"更有压迫感
- **沉默制造恐惧**：问了一个问题，对方没有立刻回答——那三秒钟就是恐惧

## 感官描写重点

- 视觉：监控画面的颗粒感、夜晚窗户上的反光、阴影的形状
- 听觉：脚步声的距离变化、门锁转动、手机振动、空调滴水
- 嗅觉：消毒水的味道、不该出现的香水味、密闭空间的霉味
- 触觉：门把手的温度（有人刚碰过）、抽屉里多出来的东西、后背发凉

## 代表作品分析

| 作品 | 核心技法 | 值得学的点 |
|------|----------|-----------|
| 《白夜行》 | 双视角+暗线交织 | 用日常细节暗示真相 |
| 《沉默的真相》 | 多线叙事+社会批判 | 时间线交错的节奏感 |
| 《隐秘的角落》 | 孩子视角+成人世界的恶 | 孩子眼中的"不懂"制造不安 |
| 《消失的她》 | 叙述性诡计+婚姻悬疑 | 信任崩塌的递进 |

# 四、都市异能（都市+超自然元素）

## 核心张力

**隐藏的异常**——你身边那个看起来最普通的人，有着最不普通的能力。都市异能的本质是"日常秩序下的暗流"。

读者欲罢不能的原因：代入感——"如果我有这个能力，我会怎么做？"加上"我身边的人会不会也有秘密？"

## 两种套路

### 套路A：隐藏身份+日常暴露流

**公式**：隐藏能力 → 被迫使用 → 被人发现 → 信任危机 → 能力升级

**递进拆解**：
1. 能力的使用有代价（体力/寿命/暴露风险），不能随便用
2. "被迫使用"的场景必须是读者认同的——救人、自保、保护重要的人
3. 被发现后的信任危机比能力本身更有戏剧性
4. 能力升级伴随着更大的敌人和更高的代价

**示例**：
> 他在公司做了三年行政，每天准点上下班，从不加班，从不出风头。同事们觉得他是个"没什么追求的人"。
>
> 直到年会那天，大楼突然停电，电梯卡在17楼。有人开始恐慌，有人在砸电梯门。他叹了口气，走到电梯门前，把手放在门缝上。
>
> 电梯门像被无形的力量掰开一样，缓缓分开了。
>
> 他转头对惊呆的同事们说："走吧，趁消防还没来。"
>
> 第二天，全公司都在讨论"那个行政小哥"。他递了辞职信。

### 套路B：都市修炼/升级流

**公式**：发现能力 → 学习控制 → 都市中的修炼资源争夺 → 势力冲突 → 更大的世界观揭示

**递进拆解**：
1. 能力系统要有"都市感"——不是飞剑法宝，是手机信号、地铁线路、外卖骑手的路线
2. 修炼资源融入都市生活（某个地铁站的特殊能量、凌晨三点的便利店是中转站）
3. 势力冲突用"公司/组织/协会"的外壳包装
4. 升级的爽感来自"用新能力解决都市生活问题"

**示例**：
> 他发现自己能看到每个人头上的"信用分"。
>
> 不是芝麻信用，是某种更本质的东西——诚实、善良、自私、贪婪，每种品质都有一个数值。
>
> 地铁上，他看到对面那个西装革履的男人头上，"贪婪"的数值是97。
>
> 到站后，那个男人走进了一家投资公司。他犹豫了一下，跟了进去。
>
> 三天后，那家公司暴雷了。

## 叙事节奏模板

```
能力发现（2章）→ 学习控制（3章）→ 日常使用（2章）→ 被迫暴露（1章）→ 信任危机（2章）→ 新敌人/新世界（2章）→ 升级（1章）→ 循环
```

**节奏要点**：都市异能的"松"是能力带来的日常便利；"紧"是能力暴露的风险和代价。

## 经典场景库（5个必写场景）

1. **能力首次觉醒**——在最普通的情境下发现异常（刷牙时镜子不对、地铁上看到不该看到的东西）。
2. **被迫当众使用**——危急时刻藏不住了，所有人的目光都变了。
3. **能力失控**——想停但停不下来，造成意外后果。
4. **同类相遇**——在便利店/地铁/公司电梯里，发现另一个人也不正常。
5. **代价显现**——用完能力后的虚弱、副作用、或者被"组织"盯上。

## 对话风格特征

- **双关语**：表面聊日常，实际在用"行话"交流超自然信息
- **试探性对话**："你有没有觉得这个城市有时候……不太对劲？"
- **隐瞒时的不自然**：朋友问"你最近怎么了"，回答"没什么"但眼神闪躲
- **能力者的傲慢/疲惫**：用轻描淡写的语气说"这种事我见多了"

## 感官描写重点

- 视觉：普通人看不到的东西（阴影里的人影、空气中扭曲的光线）
- 听觉：只有主角能听到的声音（低频嗡鸣、远处的呼唤）
- 嗅觉：异常气味（硫磺、腐烂、不属于这个季节的花香）
- 触觉：使用能力时的身体反应（发热、发麻、血管跳动）

## 代表作品分析

| 作品 | 核心技法 | 值得学的点 |
|------|----------|-----------|
| 《我有一座恐怖屋》 | 都市恐怖+能力系统 | 恐怖氛围的日常化渗透 |
| 《全球高武》 | 都市修炼+升级体系 | 能力与都市生活的融合 |
| 《镇魂》 | 都市灵异+双男主 | 古老设定的现代表达 |
| 《司藤》 | 精怪入世+身份冲突 | 非人类角色的"人化"过程 |

# 五、都市重生/穿越（重生回到过去/穿越到现代）

## 核心张力

**信息差带来的碾压感和孤独感**——你知道未来会发生什么，但没有人相信你。重生/穿越的本质是"带着答案做选择题"。

读者欲罢不能的原因：爽感来自"我知道答案但你们不知道"的优越感；虐感来自"我知道悲剧会发生但我改变不了/改变后付出更大代价"。

## 两种套路

### 套路A：重生复仇/逆袭流

**公式**：上辈子失败/被害 → 重生到关键节点 → 利用信息差避坑 → 打脸 → 逆袭

**递进拆解**：
1. 上辈子的"死因"必须具体——不是"过得不好"，是"被某个人害死/被某个决策毁掉"
2. 重生节点选择在"一切还来得及"但"已经开始恶化"的时间
3. 信息差的使用要分层次——先用小信息赚第一桶金，再用大信息改变命运
4. 打脸对象从身边人开始，逐步升级到更大的对手

**示例**：
> 她重生回到大学毕业那年。上辈子，她放弃了保研名额让给了男朋友，男朋友读完研后跟她分了手，和导师的女儿在一起了。
>
> 这一次，辅导员问她："保研名额你确定要让给李明吗？"
>
> 她笑着说："不让。谢谢老师。"
>
> 辅导员愣了一下："但你之前说——"
>
> "我改主意了。"
>
> 她走出办公室的时候，看到李明在走廊等她。他的表情从期待变成困惑。
>
> "名额我留着了。"她说，"你可以自己考。"
>
> 她没等他回答就走了。上辈子她在走廊里哭了两个小时，这辈子她一秒钟都不想浪费。

### 套路B：穿越现代适应流

**公式**：古代人/异世界人来到现代 → 文化冲突 → 用特殊能力/知识立足 → 身份危机 → 找到归属

**递进拆解**：
1. "不适应"的笑点和痛点并存（不会用手机、不理解网络用语、被当成神经病）
2. 能力/知识在现代的"降维打击"效果（古代医术、异世界战斗经验）
3. 身份危机是核心矛盾——"我是谁？我属于哪里？"
4. 最终不是"完全融入"而是"找到平衡"

**示例**：
> 他在皇宫当了二十年御医，醒来发现自己躺在一间白色小屋里，面前的人穿着绿色的衣服，手里拿着一根针。
>
> "别动，抽血。"
>
> 他本能地摸向腰间——药箱不在了。他看了看四周，墙上有四个字："救死扶伤"。
>
> "你们……也是医者？"
>
> 护士看了他一眼："你是从剧组跑出来的吧？"
>
> 他不知道什么是"剧组"，但他知道什么是"医者"。他站起来，对护士说："在下不才，愿效犬马之劳。"
>
> 护士按下了呼叫保安的按钮。

## 叙事节奏模板

```
上辈子/穿越前交代（1章）→ 重生/穿越（1章）→ 适应+第一次使用信息差（3章）→ 小逆袭（2章）→ 大敌人出现（2章）→ 利用关键信息翻盘（2章）→ 新格局（1章）→ 循环升级
```

**节奏要点**：重生文的"松"是信息差带来的小胜利；"紧"是"历史改变了，新的不确定性出现了"。

## 经典场景库（5个必写场景）

1. **重生第一眼**——睁开眼看到年轻的脸/陌生的环境，确认"我回来了/我到了"。
2. **关键选择重现**——上辈子做错的那个选择又来了，这次主角选了另一条路。
3. **打脸时刻**——上辈子欺负TA的人用同样的招数，被主角轻松化解。
4. **蝴蝶效应**——改变了一件事，结果另一件意想不到的事发生了。
5. **面对"已故"的人**——上辈子已经去世的人还活着，主角忍不住红了眼眶。

## 对话风格特征

- **信息差傲慢**："你放心，这件事我知道结果。"——对方以为你在吹牛
- **穿越者的格格不入**：用古语说话、不理解现代梗、对"理所当然"的事感到震惊
- **压抑的预知**：知道对方未来会怎样，但不能说，只能旁敲侧击
- **自言自语式复盘**："上辈子这个时候，我已经……"

## 感官描写重点

- 视觉：熟悉又陌生的环境（老街道还没拆迁、老朋友年轻的脸）
- 听觉：时代的标志性声音（老歌、老式手机铃声、绿皮火车的汽笛）
- 嗅觉：记忆中的味道触发上辈子的回忆
- 触觉：年轻的皮肤、还没磨损的物件、"重新来过"的实感

## 代表作品分析

| 作品 | 核心技法 | 值得学的点 |
|------|----------|-----------|
| 《重生之神级学霸》 | 信息差+逆袭打脸 | 知识碾压的爽感设计 |
| 《庆余年》 | 穿越+权谋 | 现代思维在古代的降维打击 |
| 《赘婿》 | 穿越+商战 | 经济系统的信息差运用 |
| 《我的26岁女房客》 | 重生+都市情感 | 情感线的"预知"处理 |

# 六、都市现实（底层生活/社会问题）

## 核心张力

**系统性困境中的人性闪光**——不是一个人的悲剧，是一群人的日常。都市现实的本质是"看清生活的真相之后，依然热爱生活"。

读者欲罢不能的原因：真实感带来的共情——"这就是我/我身边人的生活"。不提供爽感，提供"被看见"的感觉。

## 两种套路

### 套路A：底层挣扎求存流

**公式**：困境呈现 → 挣扎求存 → 遇到转机 → 转机破灭 → 在废墟中找到意义

**递进拆解**：
1. 困境不是单一的——经济压力、家庭矛盾、健康问题同时存在
2. 挣扎的过程要有"小确幸"——不能全程压抑，要给读者喘息的缝隙
3. 转机破灭不是为了虐读者，是为了展示"即使没有奇迹，人也能活下去"
4. 意义不一定是"逆袭成功"，可以是"接受现状并找到尊严"

**示例**：
> 她在电子厂的流水线上站了十一个小时，脚肿得像馒头。
>
> 食堂的饭菜三块钱一份，她打了一份米饭和一个素菜。旁边的工友打了一份红烧肉，她咽了咽口水。
>
> 晚上回到八人间的宿舍，她给老家的女儿打了个电话。
>
> "妈妈你什么时候回来？"
>
> "过年就回来。"
>
> "你去年也是这么说的。"
>
> 她挂了电话，把脸埋在枕头里。上铺的姐妹翻了个身，轻声说："别哭了，明天发工资。"
>
> 她从枕头里抬起头，问："你说明年……我能把她接过来吗？"
>
> 上铺没说话。窗外是工厂的灯火，二十四小时不灭。

### 套路B：社会问题切入流

**公式**：具体人物的命运 → 映射社会问题 → 多方视角呈现 → 不给答案，给思考

**递进拆解**：
1. 不要写"社会问题"，写"一个人的问题"——读者共情的是人，不是议题
2. 多方视角呈现——受害者、施害者、旁观者、制度执行者，每方都有自己的"合理性"
3. 不预设立场——让读者自己判断
4. 结尾开放——不是"问题解决了"，是"问题还在，但有人在面对"

**示例**：
> 拆迁办的人第三次来了。
>
> 老张泡了茶，递过去："喝点水。"
>
> "张叔，补偿方案您看了吧？真不能再高了。"
>
> 老张看了看墙上老伴的照片。她去年走的，就在这张床上。
>
> "我不是为了钱，"老张说，"我就想问一件事——我老伴的坟，迁到哪？"
>
> 对面的人沉默了一会："公墓的事我们协调——"
>
> "协调了三个月了，还没协调好。"老张的语气很平静，"我不搬。"
>
> 他把茶杯收了，送客。门关上后，他坐在老伴照片对面，说："他们不懂。你走了，这里就是你的地方。我走了，谁给你上香？"

## 叙事节奏模板

```
困境呈现（2章）→ 挣扎中找缝隙（3章）→ 小转机（1章）→ 转机破灭/更大困境（2章）→ 低谷（1章）→ 在废墟中重建（2章）→ 开放式结尾
```

**节奏要点**：现实文的"松"是底层人物之间的温暖（工友的一碗面、邻居的一句问候）；"紧"是"下个月的房租怎么办"这种永远不会消失的焦虑。

## 经典场景库（5个必写场景）

1. **深夜的出租屋**——隔音很差，能听到隔壁的电视声。天花板上有一块水渍，形状像一张脸。
2. **医院排队**——挂号窗口的长队、缴费时看余额、在走廊的塑料椅上等化验结果。
3. **菜市场/早市**——凌晨四点的批发、为了五毛钱讲价、菜叶子上的露水和泥土。
4. **工地板房/集体宿舍**——上下铺、公共浴室、工友的鼾声和手机外放的短视频。
5. **过年回家/不回家**——绿皮火车上的泡面味、抢不到票的焦虑、或者一个人在出租屋看春晚。

## 对话风格特征

- **朴实直接**：不说"我觉得这个方案不太合适"，说"这样不行"
- **方言/口音标记身份**：适当的方言让角色立体（但不要过度）
- **沉默比对话更重**：底层人物不善于表达情感，沉默和行动比语言更有力量
- **幽默中的苦涩**：用自嘲化解困境——"我这叫啥？叫穷开心"

## 感官描写重点

- 视觉：旧物件的磨损痕迹、逼仄空间的堆叠感、廉价食物的色彩
- 听觉：工厂机器的轰鸣、隔壁的争吵、凌晨的垃圾车
- 嗅觉：潮湿的霉味、公共厨房的油烟、廉价洗衣粉的味道
- 触觉：粗糙的手、硬板床、冬天没有暖气的房间

## 代表作品分析

| 作品 | 核心技法 | 值得学的点 |
|------|----------|-----------|
| 《蜗居》 | 房价压力+小人物命运 | 经济压力的具象化 |
| 《都挺好》 | 家庭矛盾+重男轻女 | 原生家庭创伤的呈现 |
| 《我不是药神》 | 医疗困境+小人物抗争 | 真实事件改编的张力 |
| 《人世间》 | 时代变迁+群像叙事 | 多家庭的命运交织 |

# 跨子类型通用技法速查

| 技法 | 适用子类型 | 操作要点 |
|------|-----------|---------|
| 信息差 | 全部 | 读者知道而角色不知，或角色知道而另一角色不知 |
| 消费符号人设 | 职场/情感/现实 | 用消费选择代替性格描写 |
| 并行蒙太奇 | 全部 | 同一时间不同空间，反差越大越好 |
| 日常暴力 | 职场/现实 | 微小的、持续的、被合理化的伤害 |
| 微信对话叙事 | 情感/职场 | 聊天记录格式、时间标记、消息长度暗示态度 |
| 未接来电 | 情感/悬疑 | 不写谁打来的，只写"3个未接来电" |
| 天气映射 | 全部 | 梅雨季=内耗，雾霾=看不到未来 |
| 经济周期节奏 | 职场/现实 | 发工资松、月底紧，自然形成呼吸感 |
