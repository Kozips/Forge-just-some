# 都市类专业知识

## 社交潜台词词库

> 都市人最擅长欲言又止。写对话时查此表，确保角色"说人话"。

### 高频回避类

| 说出的话 | 真正的意思 | 使用场景 |
|----------|-----------|----------|
| "没事。" | 有事，但不想说 | 情侣冷战、被领导批评后 |
| "随便你。" | 我已经生气了，你自己看着办 | 意见分歧 |
| "挺好的。" | 不好，但我懒得解释 | 被问近况 |
| "再说吧。" | 我不想，但不想直接拒绝 | 拒绝邀约 |
| "你想多了。" | 被说中了，但不想承认 | 被拆穿小心思 |
| "我没事，你去吧。" | 你敢去就完了 | 吃醋 |
| "不用管我。" | 快来哄我 | 撒娇 |
| "你是不是觉得我很烦？" | 我需要你否认 | 安全感缺失 |

### 职场潜台词类

| 说出的话 | 真正的意思 | 使用场景 |
|----------|-----------|----------|
| "这个方向有意思" | 方向错了，重做 | 领导给反馈 |
| "我们回头再对齐一下" | 你做得不对 | 跨部门协作 |
| "我考虑一下" | 我不想，之后找个理由拒绝 | 被推活 |
| "你的优势在别的方向" | 你不适合这个岗位 | 委婉劝退 |
| "年轻人很有想法" | 想法不切实际 | 否定方案 |
| "先这样吧" | 不满意但不想再讨论了 | 会议结束信号 |

### 亲密关系中

| 说出的话 | 真正的意思 |
|----------|-----------|
| "你和谁去的？" | 我要知道所有细节 |
| "没谁。" | 就是不想告诉你 |
| "她要来也行的。" | 她如果敢来我当场翻脸 |
| "你喜欢就好。" | 我不喜欢但我不说 |
| "你觉得呢？" | 你的答案应该是"你" |

### 使用规则
1. **不要解释** — 角色说"没事"就是"没事"，作者不跳出来翻译
2. **用行动补充** — 嘴上说"没事" + 手上使劲掐自己 = 读者自己理解
3. **错位才有张力** — A说潜台词，B听不懂/假装听不懂，冲突自然产生
4. **潜台词要符合人设** — 社恐的回避和职场老油条的回避用词不同

## 职场生态词库

### 职场权力层级速查

| 层级 | 典型年龄 | 核心焦虑 | 典型台词 | 被谁欺负 |
|------|---------|---------|---------|---------|
| 实习生 | 20-22 | 转正/被看见 | "这个我来做吧" | 所有人 |
| 应届生 | 22-24 | 入行/被PUA | "好的我改一下" | 老员工、领导 |
| 3-5年 | 26-29 | 晋升/买房 | "这个需求不太合理" | 中层、HR |
| 中层 | 30-35 | 裁员/站队/背锅 | "我跟领导再对齐一下" | 高层、同级 |
| 总监 | 35-40 | 部门预算/派系 | "这个季度目标要保" | 老板、同级 |

### 职场空间符号

| 空间 | 象征意义 | 典型场景 |
|------|---------|---------|
| 工位 | 个人领地 | 调座位=边缘化、靠窗=受宠 |
| 会议室 | 权力展演场 | 谁坐主位、谁说话、谁记笔记 |
| 茶水间 | 信息集散地 | 八卦、抱怨、拉帮结派 |
| 消防通道 | 逃避空间 | 抽烟、打电话、偷哭 |
| 老板办公室 | 审判场 | 被叫进去=要么升职要么被裁 |
| 地下车库 | 情绪缓冲带 | 坐车里发呆五分钟再上楼 |

### 职场压力来源

| 压力类型 | 触发词 | 角色反应模式 |
|----------|--------|-------------|
| 绩效压力 | KPI、末位淘汰 | 焦虑→加班→战或逃 |
| 人际关系 | 站队、派系、暗箭 | 被迫表态→内心挣扎→选边或独善其身 |
| 经济压力 | 房贷、降薪 | 不敢辞职→忍气吞声→积压爆发 |
| 年龄压力 | 35岁定律、被优化 | 自我怀疑→拼命证明→焦虑幻灭 |
| 信息压力 | 微信群轰炸、已读不回 | 手机依赖→无法放松→精神内耗 |

### 职场日常细节速查

**加班**：外卖盒堆在工位角落、空调自动关了、末班地铁的报站声
**开会**：领导说"我简单说两句"=至少15分钟、会议桌下偷偷回微信、散会后各自回工位没人多说话
**团建**：被迫唱K、领导讲笑话时大家一起笑但不知道笑点在哪
**离职**：工位突然空了、工作群里被移出、"有时间聚聚"=再也不会有时间

### 职场关系

| 关系类型 | 典型冲突 |
|----------|---------|
| 导师-学徒 | 导师抢徒弟的成果/徒弟翅膀硬了 |
| 同级竞争 | 竞聘一个岗位/抢一个项目 |
| 上下级 | 老板画饼、下属摸鱼、中层夹心 |
| 空降兵-老员工 | 老员工联合抵制新领导 |

**使用建议**：
1. 写职场前先确定角色在哪个层级，压力和焦虑来源不同
2. 空间即权力——角色所处空间暗示其处境，不需要旁白解释
3. 细节不求多，求准——"工位调到厕所旁边"胜过十段模糊描述
4. 避免爽文化——职场大多数时候是暗流暗涌，不是明刀明枪

## 都市生活细节库

> 写都市类最怕"悬浮"。按需抽取，一个章节挑1-2个细节就够。细节要跟情节相关——地铁拥挤本身不是剧情，角色在拥挤地铁里接到分手电话才是。

### 通勤与交通
- 地铁早高峰7:30-9:00最挤；限流铁栏杆排20分钟；到站挤不下去要喊"让一下"
- 打车高峰期加价1.5-2倍；定位不准在马路对面；拼车时沉默的尴尬
- 开车早高峰堵在高架上下不来；找停车位要提前20分钟；违停罚200扣3分

### 外卖与饮食
- 满减套路（满30减15但起送价25）；备注"多放辣"从没被实现过
- 便利店饭团是加班标配；"第二杯半价"一个人不好意思买
- 深夜烧烤摊凌晨一点还在；泡面加火腿肠是最朴素的安慰

### 住房与租房
- 押一付三；中介费一个月房租；看房时中介说"这个房子很抢手"
- 合租：公共区域卫生是永恒矛盾；外卖被偷吃不敢问；凌晨室友打游戏的键盘声
- 房东突然说要卖房让你一个月内搬走；退房押金被扣因为"墙面有污渍"

### 数字生活
- 微信工作群置顶超过10个；发朋友圈要分组（同事/家人/朋友）
- 打字打到一半删掉重写；"在吗"开场白让人紧张；语音消息60秒是灾难
- 刷抖音到凌晨两点停不下来；花呗账单日是每月9号；基金跌了不敢看

### 消费锚点（一线城市2025）

**用法**：角色月薪5000时，点外卖会犹豫配送费6元；月薪15000时，会犹豫要不要打车而不是坐地铁。消费决策直接反映经济水平。

- 房租：合租单间1500-3000元/月；整租一居3500-6000元/月
- 外卖单均20-35元；瑞幸9-15元；星巴克30-40元
- 电影票30-60元；密室逃脱人均100-200元
- 同事结婚红包200-600元

### 节日与时间节点
- 春节前：抢火车票、年终奖发了多少是社交话题、回家被催婚催生
- 月底：花呗还款焦虑、"这个月又月光了"
- 周五晚上：效率最低，约饭约酒约电影，回家躺平也是选择

### 城市特质速查

**用法**：设定在具体城市时，挑2-3个细节嵌入即可。

| 城市 | 独特细节 |
|------|---------|
| 北京 | 五环内外的房价断崖；胡同里的大爷和国贸的白领共存 |
| 上海 | 精致穷的法租界咖啡文化；阿姨妈妈的菜场砍价 |
| 深圳 | 全是年轻人没有本地人；城中村和CBD一墙之隔 |
| 成都 | 下午茶搓麻将；火锅店排队两小时；"巴适得板" |

**使用建议**：
1. 按需抽取，一个章节挑1-2个细节就够
2. 价格会过时，写的时候注意时间线
3. 一个精准细节胜过十段泛泛描写
