# 都市类 — 结构模板

> 本目录存放章节结构模板与叙事骨架。写作前用模板规划节奏。

## 单章结构（日常-冲突-暗涌型）

```
### 日常锚定（前15%）
- 用一个生活细节开场（通勤/点外卖/刷手机/等人）
- 不急着推进剧情，让读者"进入"角色的一天

### 暗流涌动（30%）
- 一个微小的异常信号（同事的奇怪眼神/对象多说了一句"随便你"/领导突然@你）
- 角色选择忽略或自我安慰

### 冲突爆发（35%）
- 压力累计到临界点：一封邮件、一句话、一个截图
- 人物关系出现裂痕或转折

### 余波（20%）
- 角色独自消化情绪（发呆/洗澡时突然哭了/删了又打的微信消息）
- 生活继续——但有什么已经不一样了
```

## 多章弧线（生活崩塌-重建型）

```
第1-3章：稳定日常（工作、社交、恋情，一切"正常"）
第4-6章：裂缝出现（被裁/发现伴侣的秘密/家里出事）
第7-9章：全面崩塌（失去控制，角色做出冲动决定）
第10-12章：触底反弹（不是完美结局，而是接受现实后重新出发）
```

## 开篇钩子模板

**类型A — 从结局倒叙**："删掉最后一条聊天记录的时候，她的手没有抖。三个月前可不是这样。"
**类型B — 日常中的异常**："外卖小哥在门口站了三秒，说了句'你家门锁好像被撬过'就走了。"
**类型C — 一句话定调**："周二下午两点，她收到了两样东西：一份辞退通知，一条前任的婚礼请柬。"
