# 都市类 — 写作模板

> 本目录存放章节结构模板与叙事骨架。写作前用模板规划节奏。

## 本目录内容

| 文件 | 用途 |
|------|------|
| 结构模板.md | 单章结构 + 多章弧线模板 |
| 本文档 | 目录索引与使用说明 |

## 核心节奏

日常-冲突-暗涌型（日常铺垫→人际冲突→暗涌潜伏）

## 使用方法

1. 写单章前，打开 结构模板.md 找到"单章结构"，按比例分配篇幅
2. 规划多章弧线时，参考"多章弧线"模板安排起承转合
3. 模板是骨架，不是枷锁——根据情节需要灵活调整比例

## 扩展

如需新增模板，在本目录下新建 .md 文件并在上方表格中登记。
