# 都市类 — 质检清单

> 完稿前逐项检查。打勾通过，未通过则修改。

---

## 一、通用检查项

- [ ] 无病句和错别字
- [ ] 人称一致（无视角混乱）
- [ ] 时间线连贯（无矛盾）
- [ ] 对话自然（无AI腔/书面腔）
- [ ] 每章有明确的情节推进

---

## 二、真实感检查项

- [ ] 背景细节符合当代中国城市生活（地铁换乘/外卖时间/加班文化/房租/微信）
- [ ] 地名可虚构但生活细节必须真实——不写没了解过的生活
- [ ] 经济水平与生活方式匹配（月薪5千不会住大平层/开豪车）
- [ ] 季节/天气/时间与情节氛围匹配
- [ ] 社交媒体元素（微信聊天/朋友圈/短视频/外卖app）用法准确
- [ ] 有"无意义"的生活片段（通勤/等外卖/刷手机/排队）维持真实感
- [ ] 没有出现超自然元素（系统/穿越/异能）——除非子类型设定允许
- [ ] 职业描写不悬浮——至少查阅过该职业的日常节奏和术语

---

## 三、对话与人设检查项

- [ ] 对话即人设——职场人说话留三分，学生带网络梗，中年人绕弯子
- [ ] 对话有潜台词，不是所有情绪都直说
- [ ] 不同社会身份的说话方式有差异（老板/员工/外卖员/出租车司机）
- [ ] 网络用语和口头禅符合角色年龄段（00后/90后/80后/70后用语不同）
- [ ] 不写"悬浮对话"——底层角色不说上流社会的话，反之亦然

---

## 四、社会关系检查项

- [ ] 社会关系网即剧情网——同事/邻居/同学/前任，每条关系线都有冲突潜力
- [ ] 职场/学校/家庭场景的权力关系合理（领导不会对实习生平等对话）
- [ ] 角色的一次性决定有铺垫——不会突然"性格大变"
- [ ] 阶层差异有体现——不同阶层的生活方式/思维方式/消费习惯有区别
- [ ] 家庭关系有代际冲突的合理性——父母的"为你好"有时代背景支撑
- [ ] 信息差驱动悬念——读者知道而角色不知，或角色知道而另一角色不知

---

## 五、桥段与类型检查项

- [ ] 桥段有类型匹配——职场逆袭/误会分手/创业起落/身份暴露，至少踩中一个
- [ ] 人物原型清晰（小镇做题家/富二代/职场老油条/独立女性/社恐宅男/斜杠青年）
- [ ] 角色有自己的成长弧线——不是被动接受命运，有主动选择
- [ ] 结局不是"所有人都变好了"——而是有得有失，更真实
- [ ] 高潮场景有具体的社会场景支撑（会议室/酒局/天台/急诊室/法庭）
- [ ] 感情线有独立起承转合——不是为推进主线而存在的附属品

---

## 六、节奏与场景检查项

- [ ] 场景五感到位——不只是视觉，至少覆盖听觉（打印机声/地铁报站/键盘声）
- [ ] 城市场景有烟火气——深夜烧烤摊/便利店/地铁早高峰/合租厨房
- [ ] 职场/校园节奏与真实时间匹配——开会不会开到凌晨三点（除非有特殊情况）
- [ ] 章末有钩子（误会加深/新信息揭露/关系突变/身份暴露预兆）
- [ ] 有足够的"日常"场景稀释冲突——不能每章都是大事件

---

## 七、常见失败模式检测

- [ ] 没有"悬浮感"——角色的生活方式和经济水平匹配？
- [ ] 没有"说教"——主题通过情节和对话自然传递，不通过作者旁白教育读者？
- [ ] 没有"所有人都是好人"——现实中没有纯善纯恶，角色有灰色面？
- [ ] 没有"误会太弱智"——误会的产生有合理性，不是一句话就能解释清楚却不解释？
- [ ] 没有"反派脸谱化"——对手有自己的立场和苦衷？

---

## 八、红线验证

- [ ] 不涉及敏感政治话题
- [ ] 不美化违法行为
- [ ] 不歧视任何群体（性别/地域/职业/阶层）
- [ ] 涉及医疗/法律/金融等专业领域时基本准确
- [ ] 不抄袭其他都市作品的关键情节
