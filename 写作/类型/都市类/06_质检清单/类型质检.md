# 都市类 · 类型质检清单

> 写完本类型内容后，逐项检查。

---

## 一、类型特征检查

| 检查项 | 标准 | 达标 | 不达标 |
|--------|------|------|--------|
| 细节"百度得到"吗 | 上班地点/用的软件/吃的餐厅/开的车型/住房——越真实越代入 | ✅ | ❌ |
| 职业准确吗 | 律师/医生/程序员/公务员——每个职业有自己的语言/节奏/痛点 | ✅ | ❌ |
| 收入/支出符合现实吗 | 程序员月薪2-3万不是100万，刚毕业月薪1万不是5万 | ✅ | ❌ |
| 社会话题跟得上吗 | 内卷/躺平/996/35岁危机/相亲/房价/教育——读者关心什么写什么 | ✅ | ❌ |
| 对话接地气吗 | 口语+网络语，不是书面语/翻译腔/古风词 | ✅ | ❌ |
| "日常"是节拍器吗 | 上班/地铁/咖啡/外卖/健身/相亲——日常是主节奏 | ✅ | ❌ |

---

## 二、红线检查

- [ ] 有没有"霸道总裁爱上我"式童话？（写"我被房贷压得喘不过气"更有代入感）
- [ ] 有没有"主角三观不正"？（不脚踏两只船/不出轨/不做违背底线的事）
- [ ] 有没有"数字不合理"？（月薪/房价/消费水平要符合现实）
- [ ] 有没有"地理位置乱编"？（真实地名要准确，不能"北京的浦东新区"）
- [ ] 有没有"网络梗过时"？（流行语要用当年/近一年的，不是5年前的）
- [ ] 有没有"原生家庭工具化"？（父母催婚/重男轻女要写得有深度，不是标签）
- [ ] 有没有"职场只写勾心斗角"？（职场有合作/成长/温暖，不只是权斗）
- [ ] 有没有"完全没有希望感"？（都市文即使很丧，最终要有一点希望）

---

## 三、逻辑自洽检查

| 检查项 | 问题 |
|--------|------|
| 时间线 | 上班/下班/约会的时间合理吗？不是"下午3点下班"？ |
| 人物关系 | 人物之间的关系变化有铺垫吗？不是"忽然熟悉/忽然疏远"？ |
| 经济逻辑 | 主角的消费水平和收入匹配吗？ |
| 职业细节 | 职业描写前后一致吗？有没有"不懂装懂"的细节？ |
| 社会背景 | 时代背景前后一致吗？（不能前文2024年后文2030年） |

## 质检结论模板

```
都市类质检报告

类型特征：✅ 细节真实 / ⚠️ 警告 / ❌ 失败
红线检查：✅ 无踩线 / ❌ 踩线（具体哪条）
逻辑自洽：✅ 通过 / ⚠️ 警告 / ❌ 失败

总体评价：[良好/及格/需要重写]
修复项：[需要修改的具体问题]
```
