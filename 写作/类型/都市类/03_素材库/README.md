## 加班深夜素材

**适用**：角色处于职场压力/生活疲惫的状态

> 办公室只剩她一个人。空调早就关了，空气闷得像蒸笼。电脑屏幕的蓝光照在脸上，她盯着Excel表格看了十分钟，发现自己一个字也没读进去。楼下便利店的灯还亮着，她想：要不要去买一根烤肠。算了，太远了。她把头埋进胳膊里，就这么趴了五分钟。

**五感**：键盘敲击的回声 / 咖啡凉了的苦涩 / 空调停转后的闷热

## 出租房独处素材

**适用**：角色独居、内心活动较多的段落

> 隔壁在吵架。她听不清具体内容，只知道男人在吼，女人在哭，然后是一声摔门。安静了大概两分钟，手机响了——是妈妈打来的。"吃了吗？""吃了。""吃的什么？""外卖。""少吃外卖。""嗯。"挂了电话，她把手机扔到床的另一边，翻了个身，盯着天花板上那块水渍发呆。

## 地铁通勤素材

**适用**：日常过渡/角色心理状态描写

> 地铁里人贴着人，她被挤在角落，面前是一个男人的后背，西装上有一股洗衣液的味道，混着汗味。她把头转向车窗，玻璃里映出自己的脸——黑眼圈，口红褪了一半，头发因为静电翘起来。她伸手去压，手还没到，地铁进站了，人群一涌，她的手按在了陌生人的肩膀上。"不好意思。"对方没听见。

## 吵架对话素材

**适用**：情侣/室友/家人之间的冲突场景

> "你就不能把碗洗了？" "我今天加班到十点。" "我加到十一点。" "那你还不是回来了就躺着？" "我躺着怎么了？你洗个碗要死啊？" "……" "你又不说话了。每次一说到你就不说话。" "那你想让我说什么？" "我想让你——算了。" 她把碗端到水池里，打开水龙头，水声盖住了后面的话。
