# 都市场景库

> 每个场景提供五感细节 + 氛围关键词 + 常见情节触发点。写场景时直接查用，不用从零想象。

---

## 一、写字楼/办公区

**五感**：中央空调的嗡嗡声 / 打印机墨粉的涩味 / 键盘敲击的密集节奏 / 茶水间咖啡机的咕噜声 / 荧光灯的惨白光

**氛围关键词**：压抑、表面和谐、暗流涌动

| 子场景 | 氛围 | 适合发生的事件 |
|--------|------|---------------|
| 开放工位 | 透明感、无隐私 | 被人偷看屏幕；隔壁同事打电话声音太大；领导从背后走过 |
| 会议室 | 权力展演 | 方案被否、被当众批评、暗中站队、假装记笔记 |
| 茶水间 | 松弛但八卦 | 听到不该听的、假装倒水实则等某人、被拉去"聊聊" |
| 消防通道/楼梯间 | 逃离 | 偷哭、打电话给家人、抽烟、和某人密谈 |
| 电梯 | 尴尬共处 | 和领导同乘30秒、和竞争对手目光交汇、电梯门开看到不该看的 |
| 老板办公室 | 审判 | 被叫进去——升职或被裁，二选一 |
| 地下车库 | 情绪缓冲 | 坐车里发呆5分钟、接到坏电话、和某人在车里谈话 |

## 二、地铁/公交

**五感**：报站的机械女声 / 人群的体温和混合气味 / 手机外放的短视频声 / 刹车时身体前倾的惯性 / 车窗玻璃上自己的倒影

**氛围关键词**：拥挤、孤独、流动的城市血液

| 子场景 | 氛围 | 适合发生的事件 |
|--------|------|---------------|
| 早高峰车厢 | 窒息、被裹挟 | 手机被挤掉、和陌生人身体接触的不适、被迫听旁边人打电话 |
| 深夜末班车 | 疲惫、空旷 | 加班到最后一班；车厢里只有几个人；看窗外隧道灯光一闪一闪 |
| 换乘通道 | 匆忙、迷失 | 跟丢了人、看到墙上的广告发呆、在人流中突然停下来 |
| 站台等候 | 等待、犹豫 | 差一步没赶上；想打电话但忍住了；看对面站台的陌生人 |
| 公交后排 | 私密、发呆 | 靠窗看城市后退；耳机里放着某首歌；错过了站 |

## 三、出租屋/合租房

**五感**：隔壁传来的模糊说话声 / 冰箱压缩机突然启动的嗡鸣 / 浴室花洒水压不足的滋滋声 / 外面马路上凌晨的洒水车音乐 / 阳台晾的衣服还没干的潮气

**氛围关键词**：暂时的、不完全属于自己的、孤独但自由

| 子场景 | 氛围 | 适合发生的事件 |
|--------|------|---------------|
| 合租客厅 | 领土争夺 | 冰箱里的东西被吃了；公共区域卫生争吵；带人回来过夜 |
| 独处卧室 | 私密空间 | 深夜刷手机停不下来；接到家人电话报喜不报忧；失眠 |
| 厨房 | 生活感 | 做了一顿不像样的饭；煮泡面时接到分手电话 |
| 阳台 | 窥视与放空 | 看楼下的人来人往；抽烟；晒太阳发呆；接到前任消息 |
| 搬家当天 | 过渡感 | 打包到最后发现角落里的旧物；告别住了两年的房间 |

## 四、咖啡厅/奶茶店

**五感**：咖啡机蒸汽的嘶嘶声 / 磨豆的颗粒香气 / 木质桌面的触感 / 背景音乐若有若无 / 奶茶封口机"刺啦"一声

**氛围关键词**：第三空间、暧昧、假装精致

| 子场景 | 氛围 | 适合发生的事件 |
|--------|------|---------------|
| 独自办公 | 伪装的自由职业感 | 点一杯坐一下午；假装在工作其实在刷手机；被邻桌面试的声音吸引 |
| 约会 | 试探 | 第一次见面的紧张；AA还是买单的犹豫；聊到冷场时搅咖啡 |
| 偶遇 | 命运感 | 在咖啡厅碰到前任/老同学/竞争对手；目光交汇的那一秒 |
| 排队等奶茶 | 日常琐碎 | 选了半天口味还是点了最普通的；等待时刷手机看到坏消息 |

## 五、医院

**五感**：消毒水的刺鼻味 / 叫号屏幕的电子音 / 长椅的冰凉触感 / 走廊里轮椅的吱嘎声 / 医生办公室里键盘的快速敲击

**氛围关键词**：焦虑、脆弱、直面生死

| 子场景 | 氛围 | 适合发生的事件 |
|--------|------|---------------|
| 挂号大厅 | 拥挤、无助 | 排了两小时队看诊3分钟；不知道挂哪个科；黄牛号300块 |
| 候诊区 | 等待的煎熬 | 看到旁边的人病情更重；刷手机分散注意力；听到叫号心跳加速 |
| 住院病房 | 被迫暂停生活 | 隔床的鼾声；护士凌晨查房；家人轮流陪护的疲惫 |
| 急诊室 | 紧张、混乱 | 深夜送朋友/家人来；等检查结果的焦虑；看到走廊加床 |
| 取药窗口 | 机械、麻木 | 拿到处方单看到金额；药袋里一长串注意事项 |

**氛围关键词**：消费、选择焦虑、社交试炼

| 子场景 | 氛围 | 适合发生的事件 |

## 使用建议

1. **场景不是布景**：场景要服务于情节和情绪，不是走马灯式展示
2. **五感挑两个就够**：一个章节里不用把五感全写出来，挑最能烘托情绪的两个
3. **子场景表是触发器**：不知道这一章写什么的时候，翻子场景表找"适合发生的事件"
4. **跨场景切换要有过渡**：从办公室到地铁到出租屋，用一两句衔接，不要硬切
