# 职场生态词库

> 都市类近半剧情发生在职场。以下词库覆盖职场权力结构、空间符号、压力来源和日常细节，确保写职场场景不"悬浮"。

## 一、职场权力层级速查

| 层级 | 典型年龄 | 工资范围(一线) | 核心焦虑 | 典型台词 | 被谁欺负 | 欺负谁 |
|------|---------|---------------|---------|---------|---------|--------|
| 实习生 | 20-22 | 3k-5k | 转正/被看见 | "这个我来做吧" | 所有人 | 无 |
| 应届生 | 22-24 | 6k-10k | 入行/学东西/被PUA | "好的我改一下" | 老员工、领导 | 实习生 |
| 3-5年 | 26-29 | 12k-20k | 晋升/跳槽/买房 | "这个需求不太合理" | 中层、HR | 新人 |
| 中层 | 30-35 | 25k-40k | 裁员/站队/背锅 | "我跟领导再对齐一下" | 高层、同级 | 下属 |
| 总监 | 35-40 | 40k-60k | 部门预算/派系/裁员名单 | "这个季度目标要保" | 老板、同级 | 中层 |
| VP/老板 | 40+ | 股权+分红 | 融资/现金流/对手 | "今年要活下去" | 市场、投资人 | 所有人 |

## 二、职场空间符号

| 空间 | 象征意义 | 典型场景 |
|------|---------|---------|
| 工位 | 个人领地、安全感 | 调座位=边缘化、靠窗=受宠 |
| 会议室 | 权力展演场 | 谁坐主位、谁说话、谁记笔记 |
| 茶水间 | 信息集散地、非正式社交 | 八卦、抱怨、拉帮结派 |
| 消防通道/楼梯间 | 逃避空间 | 抽烟、打电话、偷哭 |
| 电梯 | 尴尬的平等空间 | 和领导同乘、和竞争对手同乘 |
| 老板办公室 | 审判场 | 被叫进去=要么升职要么被裁 |
| 天台 | 自由/绝望 | 放空、打电话给家里、崩溃 |
| 地下车库 | 情绪缓冲带 | 坐车里发呆五分钟再上楼 |

## 三、职场压力来源词库

| 压力类型 | 触发词 | 角色反应模式 |
|----------|--------|-------------|
| 绩效压力 | KPI、OKR、季度考核、述职、末位淘汰 | 焦虑→加班→战或逃 |
| 人际关系压力 | 站队、派系、办公室政治、暗箭 | 被迫表态→内心挣扎→选边或独善其身 |
| 经济压力 | 房贷、房租、降薪、年终奖缩水 | 不敢辞职→忍气吞声→积压爆发 |
| 年龄压力 | 35岁定律、被优化、年轻人 | 自我怀疑→拼命证明→焦虑幻灭 |
| 性别压力 | 婚育歧视、应酬酒局、着装审视 | 隐忍→反击→或妥协 |
| 信息压力 | 微信群轰炸、已读不回、钉钉消息 | 手机依赖→无法放松→精神内耗 |

## 四、职场日常细节词库

### 加班场景
- 外卖盒堆在工位角落、键盘敲击声在空荡的办公室回响
- 空调自动关了、闷热中闻到自己身上的汗味
- 末班地铁的报站声、打车要等40分钟
- 朋友圈发"加班狗"配图，其实已经累到刷不动手机

### 开会场景
- PPT翻页、投影仪偏色、有人假装记笔记其实在刷手机
- 领导说"我简单说两句"=至少15分钟
- 会议桌下偷偷回微信、被点名时"啊"一声
- 散会后各自回工位，没有人多说话

### 团建/应酬
- 被迫唱K、被迫敬酒、被迫说"团队有凝聚力"
- 领导讲笑话时大家一起笑，但不知道笑点在哪
- 酒局结束后各自叫代驾，第二天假装什么都没发生

### 离职/被裁
- 工位突然空了、电脑已经被IT收走
- 工作群里被移出、私聊窗口突然冷清
- "有时间聚聚"=再也不会有时间
- 收拾东西时最怕同事看到，更怕同事没看到

## 五、职场关系词库

| 关系类型 | 关键词 | 典型冲突 |
|----------|--------|---------|
| 导师-学徒 | 带教、站台、画饼、抢功 | 导师抢徒弟的成果/徒弟翅膀硬了 |
| 同级竞争 | 暗斗、信息差、抢资源、表功 | 竞聘一个岗位/抢一个项目 |
| 跨部门 | 甩锅、扯皮、邮件战争、CC领导 | 交付延迟互相推诿 |
| 上下级 | 服从、信任、邀功、替罪 | 老板画饼、下属摸鱼、中层夹心 |
| 办公室恋情 | 隐蔽、被发现、站队、离职 | 分手后还在一个部门 |
| 空降兵-老员工 | 不服、架空、示好、暗算 | 老员工联合抵制新领导 |

## 使用建议

1. **写职场前定级**：先确定角色在哪个层级，压力和焦虑来源不同
2. **空间即权力**：角色所处空间暗示其处境，不需要旁白解释
3. **细节不求多，求准**：一个准确的细节（如"工位调到厕所旁边"）胜过十段模糊描述
4. **避免爽文化**：职场不是打脸剧，大多数时候是暗流暗涌，不是明刀明枪
