# 都市生活细节库

> 写都市类最怕"悬浮"。本库提供2020年代中国城市生活的具体细节，写角色日常时直接查用，确保读者一看就知道"这是真正在城市里生活过的人写的"。

## 一、通勤与交通

| 细节类型 | 具体内容 |
|----------|---------|
| 地铁早高峰 | 7:30-9:00 最挤；限流铁栏杆排20分钟；安检传送带上的包被后面的人压住；到站挤不下去要喊"让一下" |
| 地铁晚高峰 | 17:30-19:30；座位抢不到但站位有讲究——门口最挤，车厢连接处稍好；有人用iPad占座位假装没看见 |
| 地铁冷知识 | 手机信号在隧道断断续续；空调车厢和非空调车厢体感差5度；安检口排长队时有人翻栏杆 |
| 打车/网约车 | 高峰期加价1.5-2倍；司机打电话"你在哪呢"；定位不准在马路对面；拼车时沉默的尴尬 |
| 骑行 | 共享单车月卡15-20元；找车扫码发现是坏的；冬天骑车手冻僵；逆行被罚款20-50元 |
| 公交 | 老年卡免费时段挤满大爷大妈；报站声和自己听歌的冲突；前门上后门下的默契或混乱 |
| 开车通勤 | 早高峰堵在高架上下不来；找停车位要提前20分钟；停车费一天50-80元写字楼；违停罚200扣3分 |

| 深夜觅食 | 楼下烧烤摊凌晨一点还在；泡面加火腿肠是最朴素的安慰；"最后一口"说了三次 |

## 三、住房与租房

| 细节类型 | 具体内容 |
|----------|---------|
| 租房市场 | 押一付三/押一付一（后者多收100元/月）；中介费一个月房租；看房时中介说"这个房子很抢手" |
| 合租生活 | 公共区域卫生是永恒矛盾；洗手间排队；外卖被偷吃不敢问；凌晨室友打游戏的键盘声 |

| 房东与中介 | 房东突然说要卖房让你一个月内搬走；退房时押金被扣因为"墙面有污渍"；二房东跑路 |
| 装修与家电 | 空调不制冷要自己找人修；马桶堵了打电话给房东不接；WiFi信号到卧室就只剩一格 |
| 搬家 | 货拉拉预约；打包到凌晨三点；搬完发现少了一个箱子；新家第一晚没有窗帘 |

## 六、节日与时间节点

| 节点 | 城市人的典型行为 |
|------|----------------|
| 春节前 | 抢火车票/机票；年终奖发了多少是社交话题；回家被催婚催生 |
| 情人节/七夕 | 朋友圈晒礼物大赛；没对象的人屏蔽朋友圈；餐厅涨价20% |
| 618/双11 | 提前一个月加购物车；凌晨付尾款；快递站排队一小时 |
| 周五晚上 | "周五综合症"——效率最低；约饭约酒约电影；回家躺平也是选择 |
| 月底 | 花呗/信用卡还款焦虑；看工资到账后余额；"这个月又月光了" |
| 换季 | 衣柜里永远少一件；冬天起床是最难的事；夏天办公室空调温度是战争 |

## 七、城市特有的生活质感

| 城市 | 独特细节 |
|------|---------|
| 北京 | 五环内外的房价断崖；冬天雾霾出门戴N95；胡同里的大爷和国贸的白领共存 |
| 上海 | 精致穷的法租界咖啡文化；阿姨妈妈的菜场砍价；"侬好"的本地味 |
| 深圳 | 全是年轻人、没有本地人；城中村和CBD一墙之隔；来了就是深圳人 |
| 成都 | 下午茶搓麻将；火锅店排队两小时；"巴适得板"的生活哲学 |
| 杭州 | 互联网公司扎堆带来的"996之城"标签；西湖边散步是免费治愈 |

**用法**：如果故事设定在具体城市，从表中挑2-3个细节嵌入即可。不用面面俱到，一个精准细节胜过十段泛泛描写。

## 使用建议

1. **按需抽取**：不需要全部用，一个章节挑1-2个细节就够
2. **配合01_核心规则**：细节是为了服务"真实感"规则，不是为了堆砌
3. **价格会过时**：价格锚点每2-3年更新一次，写的时候注意时间线
4. **细节要跟情节相关**：地铁拥挤本身不是剧情，角色在拥挤地铁里接到分手电话才是
