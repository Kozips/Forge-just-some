# 社交潜台词词库

> 都市人最擅长欲言又止。以下词句表面意思和实际意思往往是反的。写对话时查此表，确保角色"说人话"。

## 高频回避类

| 说出的话 | 真正的意思 | 使用场景 |
|----------|-----------|----------|
| "没事。" | 有事，但不想说/说了你也不懂 | 情侣冷战、被领导批评后 |
| "随便你。" | 我已经生气了，你自己看着办 | 意见分歧、被忽视后 |
| "挺好的。" | 不好，但我懒得解释 | 被问近况、敷衍社交 |
| "再说吧。" | 我不想，但不想直接拒绝 | 拒绝邀约、回避承诺 |
| "你想多了。" | 被说中了，但不想承认 | 被拆穿小心思 |
| "我没事，你去吧。" | 你敢去就完了 | 吃醋/不满但不说穿 |
| "不用管我。" | 快来哄我 | 撒娇/寻求关注 |
| "你是不是觉得我很烦？" | 我需要你否认，不是说"是" | 安全感缺失 |

## 职场潜台词类

| 说出的话 | 真正的意思 | 使用场景 |
|----------|-----------|----------|
| "这个方向有意思" | 方向错了，重做 | 领导给反馈 |
| "我们回头再对齐一下" | 你做得不对，我来告诉你怎么做 | 跨部门协作 |
| "我考虑一下" | 我不想，但之后会找个理由拒绝 | 被提需求/被推活 |
| "你的优势在别的方向" | 你不适合现在这个岗位 | 委婉劝退 |
| "这个需要更多人支持" | 我不想一个人背锅 | 给风险打预防针 |
| "辛苦了" | 知道了，可以走了 | 领导结束对话 |
| "年轻人很有想法" | 想法不切实际 | 否定方案但不直说 |
| "先这样吧" | 不满意但不想再讨论了 | 会议结束信号 |

## 社交场合类

| 说出的话 | 真正的意思 | 使用场景 |
|----------|-----------|----------|
| "下次一起吃饭啊" | 没有下次，客气话 | 偶遇旧识/结束寒暄 |
| "哎呀你瘦了" | 社交开场白，不一定是真的 | 聚会见面 |
| "改天聊" | 今天到此为止 | 微信/面聊结束 |
| "我也是刚看到" | 我看到了但没回 | 微信已读不回被抓 |
| "信号不好" | 不想聊了 | 挂电话借口 |
| "最近太忙了" | 拒绝见面的万能理由 | 逃避社交 |

## 亲密关系中

| 说出的话 | 真正的意思 | 使用场景 |
|----------|-----------|----------|
| "你和谁去的？" | 我要知道所有细节 | 吃醋时的审问开场 |
| "没谁。" | 就是不想告诉你 | 被问行踪时的防御 |
| "她要来也行的。" | 她如果敢来我当场翻脸 | 假装大方 |
| "你喜欢就好。" | 我不喜欢但我不说 | 对伴侣的选择不满 |
| "你觉得呢？" | 你的答案应该是"你" | 需要确认被爱 |

## 词库使用规则

1. **不要解释** — 角色说"没事"就是"没事"，作者不跳出来翻译
2. **用行动补充** — 嘴上说"没事" + 手上使劲掐自己 = 读者自己理解
3. **错位才有张力** — A说潜台词，B听不懂/假装听不懂，冲突自然产生
4. **潜台词要符合人设** — 社恐的回避和职场老油条的回避用词不同
