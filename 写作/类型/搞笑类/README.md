| 目录 | 内容文件 | 用途 |
|------|----------|------|

## 使用流程

1. **开写前** — 读 SKILL.md 了解全局，读 01_核心规则/核心规则.md 确认铁律
2. **构建阶段** — 从 02_专业知识/专业知识库.md 获取领域知识，03_素材库/素材库.md 挑选素材
3. **写作阶段** — 按 04_写作模板/写作模板.md 搭结构，参考 05_技法研究/技法研究.md 提升手法
4. **完稿阶段** — 用 06_质检清单/质检清单.md 逐项检查
