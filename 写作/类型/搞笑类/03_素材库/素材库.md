# 搞笑类 -- 素材库

> 可直接复用的素材片段：角色卡、场景模板、对话范例、感官/身体语言描写。

---

## 一、角色卡模板

每张卡包含：外貌标签(3个)、搞笑属性(1核心+1隐藏)、口头禅、最尴尬的往事、最佳搭档。

### 吐槽役

**社畜型**：黑眼圈快掉到下巴 | 保温杯不离手 | 衬衫永远皱的
- 属性："已经看透了人生"型吐槽 / 隐藏：年轻时比谁都热血
- 口头禅："然后呢？"
- 往事：入职第一天把咖啡洒在社长裤子上，用自己领带擦
- 最佳搭档：天然呆（所有吐槽被自动过滤）

**学生型**：普通到人群里找不到
- 属性："旁观者清"型 / 隐藏：其实很羡慕做蠢事的人
- 口头禅："不，等等。"
- 往事：唯一一次鼓起勇气做"不普通"的事，全班都看到了
- 最佳搭档：戏精（戏精每次演，他负责拆台）

### 装傻役

**真天然傻**：圆脸 | 大眼睛 | 永远在微笑
- 属性：真的不懂社交规则 / 隐藏：对某些事的直觉比谁都准
- 口头禅："诶？是这样吗？"
- 往事：把老师叫成"妈妈"，而且不是小学时候的事
- 最佳搭档：毒舌（毒舌说了刻薄话，他真心觉得"你说得好有道理"）

**自作聪明型**：戴眼镜 | 头发整齐 | 表情自信
- 属性：以为自己很聪明，但聪明每次用错地方 / 隐藏：不笨，方向感和判断力为零
- 口头禅："根据我的分析——"
- 往事：做了一个很详细的计划，执行了三天才发现方向搞反了
- 最佳搭档：倒霉蛋（霉运+"聪明"，破坏力翻倍）

### 毒舌

**冷面毒舌**：瘦高 | 面无表情 | 打字速度比说话快
- 属性：每句话都像在发讣告 / 隐藏：不会表达善意，毒舌是唯一会的交流方式
- 口头禅："你的逻辑有一个致命的错误。"
- 往事：试图夸人，用的词听起来像在骂人，对方哭了
- 最佳搭档：戏精（戏精的所有表演在他面前撑不过三秒）

**损友型毒舌**：看起来随和 | 笑嘻嘻 | 嘴里永远在嚼东西
- 属性：嘴上不饶人行动上最靠谱 / 隐藏：夸人的方式是先损一顿再偷偷帮忙
- 口头禅："不是我要说你啊——"
- 往事：在别人婚礼上说了一句话，全场安静五秒，新娘妈妈开始哭
- 最佳搭档：热血型主角（满怀激情说完一段话，他一句话浇灭）

### 倒霉蛋

**天选倒霉人**：脸上总有创可贴 | 衣服有昨天的污渍 | 发际线后退
- 属性：运气永远在最低点 / 隐藏：把"倒霉"变成哲学——"反正不会更糟了"（然后更糟了）
- 口头禅："应该不会出什么问题吧。"（他说这话时，所有人都在找掩体）
- 往事：去面试走进了隔壁殡仪馆，还鞠了三个躬
- 最佳搭档：所有人（霉运就是他最好的搭档）

### 戏精

**表演型人格**：五官表现力强 | 手势比意大利人还夸张 | 走到哪都是舞台
- 属性：把所有场景当舞台剧 / 隐藏：学过表演，因太夸张被学校劝退
- 口头禅："你不懂。"（然后开始演）
- 往事：在地铁上演了一段独角戏，旁边的人偷偷报了警
- 最佳搭档：冷面笑匠（越演越冷静，反差越来越大）

### 冷面笑匠

**数据型冷面**：戴眼镜 | 不苟言笑 | 笔记本不离手
- 属性：用数据和事实说话，但说的事实每次让人爆笑 / 隐藏：感情表达方式是"记录在案"
- 口头禅："根据统计——"
- 往事：在别人生日派送上做了一份"寿星过去一年失误总结报告"
- 最佳搭档：装傻役（装傻役做出离谱的事，他冷静给出数据化分析）

## 三、名字谐音梗对话模板

> 角色名字自带谐音，日常对话里自然产生笑点——不需要刻意搞笑，名字本身就是梗。

### 原理

角色叫"蔡道"，别人说"才到啊"，他回"你怎么知道我叫蔡道"——对话里的笑点来自**名字和日常用语的重叠**，角色自己不知道自己在搞笑，读者笑得更厉害。

### 实战组合

| 角色名 | 谐音 | 自然对话示例 |
|--------|------|------------|
| 蔡道 | 才到 | "你**才到**啊？" "你怎么知道我叫**蔡道**？" |
| 秦狩 | 禽兽 | "你真是个**禽兽**！" "谢谢，我叫**秦狩**。" |
| 梅良心 | 没良心 | "你这人**没良心**的！" "准确来说，我姓**梅**，叫**良心**。" |
| 甄好 | 真好 | "这饭**真好**吃。" "嗯？叫我干嘛？" |
| 朱逸群 | 猪一群 | "你看那边，**猪一群**。" "别叫我全名。" |
| 范统 | 饭桶 | "你就是个**饭桶**！" "是**范统**，谢谢。" |
| 贾正经 | 假正经 | "你能不能**正经**点？" "我一直很**正经**啊，**贾正经**。" |
| 杜子腾 | 肚子疼 | "我**肚子疼**。" "我也**杜子腾**。" |
| 魏生津 | 卫生巾 | "帮我拿一下那个……" "什么？" "……算了我自己来。" |
| 高富帅 | 高富帅 | "你叫什么？" "**高富帅**。" "……你爸妈对你期望很高啊。" |
| 白富美 | 白富美 | "你好，我叫**白富美**。" "你好，我叫**矮穷矬**。我们天生一对。" |
| 沈京兵 | 神经病 | "你是不是**神经病**？" "你怎么知道我叫**沈京兵**？" |
| 初芯 | 初心 | "不忘初心。" "嗯，我就是**初芯**。" |
| 熊初墨 | 熊出没 | "注意！**熊出没**！" "我叫**熊初墨**，不是熊。" |
| 蔡道 | 才到 | 老师："你**才到**？" 蔡道："老师你认识我？" |

### 写法要点

1. **角色自己不知道自己在搞笑**——笑点来自读者的认知差，不是角色在说相声
2. **配角先说日常用语，角色再接名字**——自然触发，不刻意
3. **同一个梗不要反复用**——同一个谐音出现2-3次就够了，多了变烂梗
4. **可以用来做"初见梗"**——角色第一次自我介绍时触发，后面偶尔带一笔
5. **名字谐音可以推动剧情**——比如角色因为名字谐音被误解、被嘲笑、被记住

### 进阶：谐音名+性格反差

| 角色名 | 谐音 | 性格 | 笑点来源 |
|--------|------|------|---------|
| 秦狩 | 禽兽 | 温柔内向的图书管理员 | 别人骂"禽兽"，他委屈地报名字——反差萌 |
| 高富帅 | 高富帅 | 矮穷矬但性格超好 | 自我介绍时全场沉默——然后他用人格魅力翻盘 |
| 沈京兵 | 神经病 | 冷静理性的数学天才 | 别人说"你神经病吧"，他推了推眼镜："准确来说，是**沈京兵**。" |

## 四、场景模板

### 社死现场

> 他在超市里对着一个背影喊了三声"妈"。那个人转过来——是个留着胡子的大叔。整个收银区安静了两秒。大叔看了看他，又看了看自己手里的菜，说："我不是你妈。但我可以给你打折。"

### 冷笑话/吐槽

> "我觉得我最近变聪明了。""怎么说？""我昨天算了一道数学题，一秒就算出来了。""什么题？""1+1等于几。""……""答案是2！你猜到了吗？你也没那么聪明嘛。"

### 一本正经的荒唐

> 他把地图铺在桌上，用激光笔指着各个区域："根据我的分析，敌人的弱点在这里——厨房。""为什么是厨房？""因为他们肯定会饿。饿了就去厨房。""你确定这是作战计划，不是做饭计划？"

### 告白翻车

**模板**：
1. 准备阶段：排练了二十遍，觉得万无一失
2. 出发：路上就开始出事（下雨/堵车/衣服弄脏）
3. 到达：对方在跟别人聊天（那人条件比他好十倍）
4. 告白：说到一半被打断/忘词/对方理解错了意思
5. 结果：意外的——但一定不是他预想的那种

### 谣言传播

**关键写法**：每传一个人，信息扭曲一点。扭曲的点要具体——不是"变了"，是"某一个细节被夸大/误解"。传到最后一环跟原版对比是最搞笑的。

## 三、同场景多角色对话范例

**场景：他们发现仓库门被锁了**

| 角色 | 台词 |
|------|------|
| 吐槽役 | "我就知道。每次都说'不会出问题的'，然后每次都出问题。" |
| 装傻役 | "诶？门为什么打不开？是不是它今天心情不好？" |
| 毒舌 | "你们谁能解释一下，为什么一个仓库的门会比你们的脑子还难打开？" |
| 倒霉蛋 | "果然。我就说了不该走这条路。" |
| 冷面笑匠 | "根据门锁的生锈程度，它至少有三个月没有被打开过。也就是说，我们白来了。" |
| 戏精 | （单膝跪地，手贴在门上）"......你不是一个人。" |
| 疯批 | "让开。"（然后一脚踹开了门） |
| 天然呆 | "锁了？那我们从窗户进去不就好了？" |

**原则：通过对话，读者不需要看名字就知道是谁在说话。**

## 四、搞笑对话的十种句式

### 1. 三段递进
> "我学了三年剑道，两年拳击，一年综合格斗。""然后呢？""然后我在门口的台阶上摔断了腿。"

### 2. 精准吐槽
> "你刚才做的事情用一句话总结就是——花了两小时做了一个可以花五分钟完成的事，而且还没完成。"

### 3. 答非所问
> A："你觉得这个方案怎么样？" B："我觉得你的领带跟上衣不搭。"

### 4. 过度合理化
> "我不是在偷吃你的蛋糕。我是在进行一项关于食品安全的非正式实验。结论是：它是安全的。已经吃完了。"

### 5. 自我拆台
> "我的计划万无一失。"（两秒后）"等等，我算错了。"

### 6. 翻译腔吐槽
> "经本人慎重考虑，现正式宣布：我钱包丢了。"

### 7. 打断型
> "我有一件非常重要的事要告诉你们——""你鞋带松了。""......谢谢。我刚才想说什么来着？"

### 8. 旁白入侵
> 他信心满满地走进了那扇门。（我建议你现在就翻到后面几页看看他出来的样子，做好心理准备。）

### 9. 集体沉默
> 老师："这是谁干的？"全班十四个人同时看向了天花板。其中有三个人抬头的方向跟其他人相反。

### 10. 反复横跳
> "我要去！""那里很危险。""我不去了。""但是不去会后悔。""那我还是去吧。""可能会死。""......你帮我请假吧。"

## 五、搞笑中的感官描写

### 视觉
- 不要写"他很狼狈"，写"头发像被雷劈过，衬衫扣子扣错了三个位置，一只鞋的鞋带拖在地上"
- 利用不协调：奢华场景+狼狈角色 / 严肃场合+不合时宜的道具

### 听觉
- 安静的教室里，他的肚子叫了。不是小声的，是"全班都能听到而且像打雷一样"的
- 手机在最不该响的时候响了，而且来电铃声是那首很丢人的歌

### 触觉
- 他拍了拍同事的肩膀，力气没控制好，同事往前踉跄了两步
- 他坐在椅子上往后仰——然后椅子断了

### 嗅觉
- 他在重要面试前吃了大蒜。嚼了三颗口香糖。没有用。

### 通感
- "他的歌声让我想到了车祸现场——不是比喻，是真的一样惨烈"
- "这个菜的味道就像有人把鞋油和草莓酱搅在了一起"

## 六、身体语言（替代偷懒写法）

| 偷懒写法 | 替换成 |
|----------|--------|
| 全场都笑了 | 小明把头埋进胳膊里。肩膀在抖。旁边的人假装没看到 |
| 他觉得很好笑 | 他试图忍住。嘴唇咬得发白了。然后发出了一声很奇怪的"噗" |
| 这真是太尴尬了 | 他开始假装看手机。手机屏幕是锁着的 |
| 他的笑话不好笑 | 全场安静了三秒。有人在清嗓子。有人在假装咳嗽 |
| 她气得说不出话 | 她笑了。但那个笑容没有到达眼睛 |
| 他摔倒了很好笑 | 他以一种违反人体力学的姿势倒在了地上。旁边有人在掏手机 |

### 紧张的身体反应
手指在桌子底下互相绞 | 笑容僵硬，嘴角在抖 | 眼神往左上方飘 | 不自觉摸后脑勺

### 生气但不能发作
笑容不变但太阳穴在跳 | 捏杯子的力气越来越大 | 说话速度变慢（每个字都在压着火气）| 突然变得非常有礼貌（越礼貌越可怕）

### 用身体语言写"笑"
他咬住嘴唇。眼睛眯了起来。鼻子在抽动。然后发出一个很奇怪的声音——像是从喉咙里挤出来的。"噗。"然后用手捂住了嘴。但肩膀在抖。"我没有笑。"他说。声音是抖的。
