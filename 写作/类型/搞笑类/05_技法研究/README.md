# 搞笑类 -- 技法研究

> 本目录存放写作技法的分析与借鉴。用于提升手法而非直接复制。

## 本目录内容

| 文件 | 用途 |
|------|------|
| 技法研究.md | 10种对话句式 + 7大技法拆解(三段式/画外音/预期违背/文字游戏/设定即笑点/心理战/保鲜) + 避坑指南 |

## 核心技法速览

| 技法 | 一句话 |
|------|--------|
| 三段式笑话 | 建立模式->强化模式->打破模式 |
| 画外音吐槽 | 正常旁观者+疯狂事件+冷静吐槽 |
| 预期违背 | 严肃预期+荒唐细节，差异越大越好笑 |
| 设定即笑点 | 给故事自带喜感的框架，角色行为自然产生笑点 |

## 使用方法

写作阶段参考提升手法。技法是工具箱，不是检查单——根据场景需要选用。
