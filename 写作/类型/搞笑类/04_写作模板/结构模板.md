# 搞笑类 — 结构模板

> 本目录存放章节结构模板与叙事骨架。写作前用模板规划节奏。

## 单章结构（铺垫-反转-爆笑型）

```
### 正常开局（前20%）
- 用一个看似普通的场景开场
- 角色做出一个"正常"的决定

### 铺垫升级（35%）
- 事情开始往奇怪的方向发展
- 角色试图维持"正常"——越努力越好笑
- 读者已经知道要出事了，角色还不知道

### 反转/包袱（30%）
- 关键信息揭露，预期被打破
- 角色的反应（崩溃/淡定/试图挽回）

### 余韵/下一个包袱（15%）
- 角色试图从尴尬中恢复
- 或者——在读者以为结束的时候，又来一个
```

## 多章弧线（日常-误会-真相-更大的误会型）

```
第1-3章：建立角色的搞笑人设和日常
第4-6章：一个误会引发连锁反应
第7-8章：试图解决误会，越解决越乱
第9-10章：真相大白——但新的误会已经开始了
```

## 开篇钩子模板

**类型A — 荒唐的第一印象**：面试官问他有什么特长。他说："我能一边吃拉面一边哭。"面试官："……为什么这个是特长？"他："因为每次面试都会用到。"
**类型B — 身处窘境**：他被卡在电梯里了。和他一起被卡住的，是他的前任、前任的新对象，以及前任新对象的妈妈。
