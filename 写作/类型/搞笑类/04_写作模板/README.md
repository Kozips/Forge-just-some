# 搞笑类 -- 写作模板

> 本目录存放章节结构模板与叙事骨架。写作前用模板规划节奏。

## 本目录内容

| 文件 | 用途 |
|------|------|
| 结构模板.md | 单章结构(铺垫-反转-爆笑) + 多章弧线 + 开篇钩子模板 |
| 写作模板.md | 五拍结构详解 + 长篇章节公式 + 多章弧线 + 对话节奏模板 |

## 核心节奏

五拍结构：铺垫(正常化) -> 误导(往A引) -> 反转(突然去了B) -> 笑点爆发 -> 余韵

## 使用方法

1. 写单章前，打开 写作模板.md 找"五拍结构"，按比例分配篇幅
2. 规划多章弧线时，参考"多章弧线模板"安排起承转合
3. 模板是骨架，不是枷锁——根据情节需要灵活调整比例
