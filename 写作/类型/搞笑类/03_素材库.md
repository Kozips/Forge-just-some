# 搞笑类 素材库

> 可直接复用的素材片段。

---

## 角色卡模板

### 吐槽役
**社畜型**：黑眼圈快掉到下巴 | 保温杯不离手 | 衬衫永远皱的
- 核心属性："已经看透了人生" / 隐藏：年轻时比谁都热血
- 口头禅："然后呢？"
- 最尴尬的往事：入职第一天把咖啡洒在社长裤子上，用自己领带擦
- 最佳搭档：天然呆（所有吐槽被自动过滤）

**学生型**：普通到人群里找不到
- 核心属性："旁观者清" / 隐藏：其实很羡慕做蠢事的人
- 口头禅："不，等等。"
- 最尴尬的往事：唯一一次鼓起勇气做"不普通"的事，全班都看到了
- 最佳搭档：戏精

### 装傻役
**真天然傻**：圆脸 | 大眼睛 | 永远在微笑
- 核心属性：真的不懂社交规则 / 隐藏：对某些事的直觉比谁都准
- 口头禅："诶？是这样吗？"
- 最尴尬的往事：把老师叫成"妈妈"，而且不是小学时候的事
- 最佳搭档：毒舌

**自作聪明型**：戴眼镜 | 头发整齐 | 表情自信
- 核心属性：聪明但每次用错地方 / 隐藏：判断力为零
- 口头禅："根据我的分析——"
- 最尴尬的往事：做了很详细的计划，执行了三天才发现方向搞反了
- 最佳搭档：倒霉蛋

### 毒舌
**冷面毒舌**：瘦高 | 面无表情 | 打字速度比说话快
- 核心属性：每句话像发讣告 / 隐藏：毒舌是唯一会的交流方式
- 口头禅："你的逻辑有一个致命的错误。"
- 最尴尬的往事：试图夸人，用的词听起来像在骂人，对方哭了
- 最佳搭档：戏精

**损友型**：随和 | 笑嘻嘻 | 嘴里永远在嚼东西
- 核心属性：嘴上不饶人行动上最靠谱 / 隐藏：先损一顿再偷偷帮忙
- 口头禅："不是我要说你啊——"
- 最尴尬的往事：在别人婚礼上说了句话，全场安静五秒，新娘妈妈开始哭

### 倒霉蛋
**天选倒霉人**：脸上总有创可贴 | 衣服有污渍 | 发际线后退
- 核心属性：运气永远最低 / 隐藏：把倒霉变成哲学——"反正不会更糟了"
- 口头禅："应该不会出什么问题吧。"（他说这话时所有人都在找掩体）
- 最尴尬的往事：去面试走进隔壁殡仪馆，还鞠了三个躬

### 戏精
**表演型人格**：手势比意大利人还夸张 | 走到哪都是舞台
- 核心属性：把所有场景当舞台剧 / 隐藏：学过表演，因太夸张被劝退
- 口头禅："你不懂。"（然后开始演）
- 最尴尬的往事：在地铁演独角戏，旁边的人偷偷报了警
- 最佳搭档：冷面笑匠

### 冷面笑匠
**数据型**：戴眼镜 | 笔记本不离手 | 不苟言笑
- 核心属性：用事实说话但每次让人爆笑 / 隐藏：感情表达方式是"记录在案"
- 口头禅："根据统计——"
- 最尴尬的往事：在生日派上做了一份"寿星过去一年失误总结报告"
- 最佳搭档：装傻役

## 同场景多角色对话范例

**场景：他们发现仓库门被锁了**

| 角色 | 台词 |
|------|------|
| 吐槽役 | "我就知道。每次都说'不会出问题的'，然后每次都出问题。" |
| 装傻役 | "诶？门为什么打不开？是不是它今天心情不好？" |
| 毒舌 | "为什么一个仓库的门比你们的脑子还难打开？" |
| 倒霉蛋 | "果然。我就说了不该走这条路。" |
| 冷面笑匠 | "根据门锁生锈程度，至少三个月没开过。我们白来了。" |
| 戏精 | （单膝跪地手贴门上）"......你不是一个人。" |
| 天然呆 | "锁了？那从窗户进去不就好了？" |

**原则：通过对话，读者不需要看名字就知道是谁在说话。**

## 搞笑对话十种句式

1. **三段递进**："我学了三年剑道，两年拳击，一年综合格斗。""然后呢？""然后我在门口的台阶上摔断了腿。"

2. **精准吐槽**："你刚才做的事情用一句话总结就是——花了两小时做了一个可以花五分钟完成的事，而且还没完成。"

3. **答非所问**：A："你觉得这个方案怎么样？" B："我觉得你的领带跟上衣不搭。"

4. **过度合理化**："我不是在偷吃你的蛋糕。我是在进行一项关于食品安全的非正式实验。结论是：它是安全的。已经吃完了。"

5. **自我拆台**："我的计划万无一失。"（两秒后）"等等，我算错了。"

6. **翻译腔**："经本人慎重考虑，现正式宣布：我钱包丢了。"

7. **打断型**："我有一件非常重要的事要告诉你们——""你鞋带松了。""......谢谢。我刚才想说什么来着？"

8. **旁白入侵**：他信心满满地走进了那扇门。（建议你现在翻到后面看看他出来的样子。）

9. **集体沉默**：老师："这是谁干的？"全班十四个人同时抬头。其中三个人抬头的方向跟其他人相反。

10. **反复横跳**："我要去！""危险。""我不去了。""不去会后悔。""可能会死。""......帮我请假吧。"

## 场景触发器

**社死现场**：他在超市对着一个背影喊了三声"妈"。那个人转过来——是留胡子的大叔。"我不是你妈。但我可以给你打折。"

**告白翻车**：排练二十遍→路上出事→对方在跟条件好十倍的人聊天→告白被打断/忘词→结果意外但不是预想的那种。

**谣言传播**：每传一个人，信息扭曲一点。扭曲点要具体——某一个细节被夸大/误解。最后一环和原版对比最搞笑。

**醉酒真相**：平时嘴严的人喝了酒之后全招了。第二天不记得了，但所有人都帮他记得。

**团队任务互相拖后腿**：简单任务被不同理解搞成灾难。最后交出来的东西跟原计划完全不一样——意外地居然还可以？

## 身体语言替换表

| 偷懒写法 | 替换成 |
|----------|--------|
| 全场都笑了 | 小明把头埋进胳膊里。肩膀在抖。旁边人假装没看到 |
| 他觉得很好笑 | 他试图忍住。嘴唇咬白了。然后发出了一声"噗" |
| 太尴尬了 | 他开始假装看手机。手机屏幕是锁着的 |
| 笑话不好笑 | 全场安静了三秒。有人清嗓子。有人假装咳嗽 |
| 她气得不说话 | 她笑了。但笑容没到眼睛 |
| 他摔倒了很好笑 | 他以违反人体力学的姿势倒在地上。旁边有人掏手机 |

**紧张的身体反应**：手指在桌子底下互相绞 | 笑容僵硬嘴角抖 | 眼神往左上方飘 | 不自觉地摸后脑勺

**生气但不能发作**：笑容不变但太阳穴在跳 | 捏杯子的力气越来越大 | 说话变慢（压着火气）| 突然变得非常有礼貌（越礼貌越可怕）

## 节奏控制

**短句和长句交替**：短句制造紧迫感，长句制造铺垫。然后用短句打碎。短句=笑点。长句=铺垫。

段落不要一样长。长-中-短-长-短-中——像心电图。如果每段都一样长，说明节奏太均匀了。

**冷热交替**：一个热笑话之后来一个冷笑话，温度降下来再来一个热的。不要连续三个热笑话（太吵）也不要连续三个冷笑话（不好笑）。

---

## 保鲜技巧

- 同一梗同章不超过两次——第一次好笑，第二次还行，第三次烦了
- 角色可以成长但搞笑方式也要变——从"天然傻"变成"装傻"
- 引入新角色激活旧梗——新角色不知道"旧规则"
- 要有安静的章节——安静章节加深角色，读者在搞笑章节笑得更开心
