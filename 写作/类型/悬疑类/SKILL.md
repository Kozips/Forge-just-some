# 悬疑类写作 Skill

> 写悬疑的核心就一个字：**藏**。让读者以为自己猜到了，然后狠狠打脸。

---

## 速查表

### 核心张力公式

**悬疑 = 读者"以为快懂了"与"其实还差得远"之间的落差**。每章回答一个问题的同时制造更大的新问题。

**五种核心张力**：未知（到底发生了什么？）/ 反转（居然不是他？！）/ 推理（让我理一理线索）/ 时间压力（必须在XX之前）/ 信任崩塌（原来你一直在骗我）

### 单章结构速查（线索-推理-新疑型）

| 阶段 | 占比 | 要做什么 |
|------|------|----------|
| 线索获取 | 25% | 新证据/新证人/新矛盾出现，初步解读 |
| 推理展开 | 30% | 梳理已知信息，排除部分可能性，锁定嫌疑人 |
| 转折/新疑 | 30% | 新信息推翻之前的推理，或看似无关的细节暗藏关键 |
| 悬念留扣 | 15% | 发现无法解释的矛盾，为下一章埋钩子 |

**经典五步节奏**：设谜(20%) -> 误导(25%) -> 线索(25%) -> 反转(20%) -> 揭晓(10%)

### 角色压力点速查（Top 5）

1. **认知压力** -- 推理矛盾、信息过载
2. **人际压力** -- 同伴质疑、嫌疑人反问
3. **时间压力** -- 截止期限、二次作案风险
4. **道德压力** -- 证据指向亲近的人
5. **自我怀疑** -- 质疑自己的判断力和记忆力

### 常见失败模式速查（Top 5）

1. **线索倾倒** -- 一章3条以上 -> 每章主线索1条，辅助不超2条
2. **神探附体** -- 无推理直接得出结论 -> 补写推理链+排除法
3. **假反转** -- 转折靠隐瞒信息 -> 转折信息必须前文出现过
4. **空悬念** -- 留扣与主线无关 -> 留扣必须指向下一章核心冲突
5. **情绪真空** -- 角色无心理反应 -> 每条关键线索后补1-2句内心波澜

### 结尾类型速查

| 类型 | 结构 | 适用场景 |
|------|------|----------|
| 断崖式 | 关键动作/发现戛然而止 | 高潮章、欲罢不能型 |
| 余震式 | 揭示小真相，角露骨绪震动更大 | 情感转折章 |
| 拼图式 | 新线索到位，读者可自行拼出部分真相 | 中段推进章 |
| 逆转式 | 章末最后一句推翻全章推论 | 关键反转章 |
| 静默式 | 角色独处消化信息，暗示内心变化 | 过渡章、喘息章 |

### 开篇钩子模板

**A. 现场开局**：警察到场时，门是反锁的。窗户从里面钉死了。死者面朝下趴在客厅中央。
**B. 倒叙悬念**：我杀了人。但警察说我不是凶手。他们说得对——但也不全对。
**C. 日常异常**：她走进书房，发现丈夫倒在书桌前。桌上的咖啡还冒着热气。但她丈夫昨天——已经死了。

## 写作自检清单

- [ ] 前200字内有让读者想继续读的钩子？
- [ ] 读者知道的和角色知道的信息是否不同？（信息差）
- [ ] 这一章有没有为后面埋的伏笔种子？
- [ ] 每个嫌疑人都有"说不通"的地方吗？
- [ ] 对话有没有表面意思之外的潜台词？
- [ ] 有没有调动至少三种感官？
- [ ] 节奏有变化？（快到慢/慢到快）
- [ ] 有没有把信息"喂"给读者的地方？（直给答案）
- [ ] 转折所用的证据是否在前文已经出现？
- [ ] 结尾紧张度是否高于开篇？（翻页冲动）

## 题材铁律

> 开写前必读。违反任何一条，悬疑感崩塌。

---

### 一、信息差铁律（1-6）

1. **读者要"差一点能猜到"** — 难度太低=无聊，难度太高=泄气。真相要放在"读者差点就猜到"的位置。
2. **线索要公平展示** — 凶手是园丁——之前写过他浇花/他手套上有泥/他和死者有过对话。读者回头看才发现原来线索都在。
3. **"误导项"必须合理** — 给 3 个像 A 的假证据，让读者暂时相信 A，最后揭示真凶是 B。误导不是欺骗。
4. **"信息时间差"是核心** — 主角知道 X，凶手知道 Y+1，读者知道主角知道 X。读者比主角多一点=最舒服的悬疑视角。
5. **"视角限制"是常用手法** — 凶手视角少给（只给行为/思维片段，不给全部动机）。读者和主角一起拼凶手。
6. **"反叙述"高级技法** — 让看似可靠的叙述者撒谎/不完整/视角偏差。读者要怀疑一切。

---

### 二、案件设计铁律（7-12）

7. **案件要"看似动机清晰"** — 死者被刺 12 刀，主角怀疑是情夫。再给情夫一个"完美不在场证明"。这就是悬疑。
8. **现场要有"标记"** — 一根头发 / 一根烟 / 一个脚印 / 一句留言。标记要具体到能成为证据。
9. **法医/物证/口供三层证据** — 物证定死时间，口供暴露动机，法医展示死因。三层一起发力=硬核悬疑。
10. **"密室杀人"是经典** — 门反锁、窗关着、人在里面死了。要给出至少 2 种"看似可能"的解释，最后揭穿一个。
11. **"暴风雪山庄"是经典** — 所有人困在一个地方，凶手就在其中。封闭环境+密闭人群+必须有人是凶手。
12. **"没有尸体"也可以悬疑** — 一个人失踪，没人见过、没找到尸体。是不是真的失踪了？这种"无证据"的悬疑最难。

---

### 三、人物铁律（13-18）

13. **嫌疑人有"自己的故事"** — 每个嫌疑人都有动机、过去、秘密、说谎的理由。不能工具人。
14. **"主角的过去"是支线** — 警察/侦探/普通人主角都有自己的创伤/执念/灰色过去。案件与主角内心互为镜像。
15. **"同伴"的怀疑链** — 主角的同伴可能是凶手/卧底/有秘密。读者要为主角担心"我身边这个人是不是有鬼"。
16. **受害者要"立体"** — 死者不是单纯被害人。他/她有自己的秘密、矛盾、敌友。死者是案件的中心。
17. **"双面人"角色** — 一个人白天是好人，晚上是杀人犯/卧底/复仇者。读者要能感觉到"他不太对"。
18. **"没有绝对的反派"** — 即使凶手也有自己的逻辑、动机、甚至值得同情的过去。读者要理解他为什么这么做。

---

### 四、节奏与结构铁律（19-24）

19. **3 幕式经典** — 案发→调查→揭示。中间埋入 3-5 个误导项，最后 3-5 章快速揭示。
20. **"大反转"不超过 1-2 个** — 多于 2 个 = 烂尾。读者会觉得自己被骗。核心反转 1 个 + 情感反转 1 个 = 最好。
21. **"小反转"3-5 个** — 每个章节 1 个小反转（"原来他是死者弟弟"、"原来他口袋里还有一把钥匙"）。小反转维持张力。
22. **章尾要有钩子** — 新证据 / 新嫌疑人 / 主角陷入危险 / 一个不祥的细节。章尾"啊？"是悬疑标配。
23. **"主角猜对了"不要立刻验证** — 主角推测 X 是凶手，下 2-3 章先让读者怀疑是不是真的这样，再揭示。
24. **"主角猜错了"是必要** — 至少让主角猜错 1-2 次。读者跟着主角一起错，再一起恍然大悟。

---

### 五、对话与细节铁律（25-30）

25. **对话要"藏信息"** — 嫌疑人不直接说"我没杀人"，说"我那天没去那里"。读者要听出弦外之音。
26. **"我没看到/我不记得"是金句** — 嫌疑人的"选择性失忆"是经典悬疑笔法。
27. **细节要"可回溯"** — 第 1 章描写的某个细节（如主角喝咖啡时窗外的车），第 20 章回头看是关键证据。
28. **"重复细节"是铺垫** — 凶手每次出场都戴着同款手表/同款帽子/同款香烟。这是给读者的最后通牒。
29. **"沉默"和"停顿"是台词** — 嫌疑人被问问题时停顿 3 秒，意味深长。沉默比回答更可怕。
30. **"潜台词"是悬疑文眼** — 一个人说"你相信命运吗"，另一人没回答。这种潜台词=深度。

---

### 自检公式

```
悬疑感 = 公平线索 × 误导项 × 视角差 × 反转精度
```

**一句话检验**：把凶手是 A 改成凶手是 B，整个故事还成立吗？成立=人物写活了，失败=靠"凶手是谁"硬撑。
