# 悬疑类写作 Skill

> 写悬疑的核心就一个字：**藏**。让读者以为自己猜到了，然后狠狠打脸。

---

## 速查表

### 核心张力公式

**悬疑 = 读者"以为快懂了"与"其实还差得远"之间的落差**。每章回答一个问题的同时制造更大的新问题。

**五种核心张力**：未知（到底发生了什么？）/ 反转（居然不是他？！）/ 推理（让我理一理线索）/ 时间压力（必须在XX之前）/ 信任崩塌（原来你一直在骗我）

### 单章结构速查（线索-推理-新疑型）

| 阶段 | 占比 | 要做什么 |
|------|------|----------|
| 线索获取 | 25% | 新证据/新证人/新矛盾出现，初步解读 |
| 推理展开 | 30% | 梳理已知信息，排除部分可能性，锁定嫌疑人 |
| 转折/新疑 | 30% | 新信息推翻之前的推理，或看似无关的细节暗藏关键 |
| 悬念留扣 | 15% | 发现无法解释的矛盾，为下一章埋钩子 |

**经典五步节奏**：设谜(20%) -> 误导(25%) -> 线索(25%) -> 反转(20%) -> 揭晓(10%)

### 角色压力点速查（Top 5）

1. **认知压力** -- 推理矛盾、信息过载
2. **人际压力** -- 同伴质疑、嫌疑人反问
3. **时间压力** -- 截止期限、二次作案风险
4. **道德压力** -- 证据指向亲近的人
5. **自我怀疑** -- 质疑自己的判断力和记忆力

### 常见失败模式速查（Top 5）

1. **线索倾倒** -- 一章3条以上 -> 每章主线索1条，辅助不超2条
2. **神探附体** -- 无推理直接得出结论 -> 补写推理链+排除法
3. **假反转** -- 转折靠隐瞒信息 -> 转折信息必须前文出现过
4. **空悬念** -- 留扣与主线无关 -> 留扣必须指向下一章核心冲突
5. **情绪真空** -- 角色无心理反应 -> 每条关键线索后补1-2句内心波澜

### 结尾类型速查

| 类型 | 结构 | 适用场景 |
|------|------|----------|
| 断崖式 | 关键动作/发现戛然而止 | 高潮章、欲罢不能型 |
| 余震式 | 揭示小真相，角露骨绪震动更大 | 情感转折章 |
| 拼图式 | 新线索到位，读者可自行拼出部分真相 | 中段推进章 |
| 逆转式 | 章末最后一句推翻全章推论 | 关键反转章 |
| 静默式 | 角色独处消化信息，暗示内心变化 | 过渡章、喘息章 |

### 开篇钩子模板

**A. 现场开局**：警察到场时，门是反锁的。窗户从里面钉死了。死者面朝下趴在客厅中央。
**B. 倒叙悬念**：我杀了人。但警察说我不是凶手。他们说得对——但也不全对。
**C. 日常异常**：她走进书房，发现丈夫倒在书桌前。桌上的咖啡还冒着热气。但她丈夫昨天——已经死了。

## 写作自检清单

- [ ] 前200字内有让读者想继续读的钩子？
- [ ] 读者知道的和角色知道的信息是否不同？（信息差）
- [ ] 这一章有没有为后面埋的伏笔种子？
- [ ] 每个嫌疑人都有"说不通"的地方吗？
- [ ] 对话有没有表面意思之外的潜台词？
- [ ] 有没有调动至少三种感官？
- [ ] 节奏有变化？（快到慢/慢到快）
- [ ] 有没有把信息"喂"给读者的地方？（直给答案）
- [ ] 转折所用的证据是否在前文已经出现？
- [ ] 结尾紧张度是否高于开篇？（翻页冲动）

## 角色速查（写作时直接参考）

> 完整版见 `人物/references/人物联动写作/悬疑类.md`

### 傅则谌（冷漠威压·玉面佛子阎罗王）
- 语言："嗯。有事先走了。"（极少说话，一开口就是结论或命令，不带任何亲昵）
- 语气：冷淡平静，声音不高不低像陈述天气，越平淡越有压迫感
- 动作：冷白腕骨上的黑色佛珠反复摩挲——唯一暴露情绪的习惯；碰一下茶盏暗示话题结束；推门离开时冷檀香气息消散只留压迫
- 用法：冷面权威型角色，离场比在场更有信息量，一个"嗯"就能让全场安静

### 棠梨（表面无害·内心坚韧·善于观察）
- 语言："小叔叔，我不懂您的意思。"（表面恭敬实则试探，用澄澈眼神配合平静语气）
- 语气：清丽柔婉，像在撒娇又不是，被拒后不纠缠转身就走——利落果断
- 动作：乌润清凌的眸子直视对方不回避；咬了咬唇——内心做决定的下意识动作；脚步极快消失在走廊尽头
- 用法：柔婉伪装型悬疑角色，"柔"是探测对方底线的工具，来去如风增加读者好奇心
