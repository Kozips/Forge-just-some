# 推理悬疑

> 推理小说不是灵感驱动的，是设计驱动的。先有蓝图再砌砖。

## 逆向设计法

先确定真相 → 设计诡计 → 设计线索 → 设计误导。

**好诡计**：简单到"我怎么没想到"；揭晓时恍然大悟；和角色动机紧密结合。
**坏诡计**：需要大量巧合；依赖读者不知道的专业知识；纯粹炫技，和情感脱节。

## 线索三层法

- **表层线索**（所有人可见）→ 案件发现时投放
- **中层线索**（需仔细观察）→ 调查中期
- **深层线索**（需跨信息关联）→ 分散在各场景

**藏线索手法**：
- 混入无关信息：十句话中三句是线索七句是日常
- 情绪遮蔽：角色处于强烈情绪时，关键细节被过滤掉
- 重复但变形：同一线索出现两次，第一次误导语境，第二次真实语境

## 误导四法

1. **确认偏误**：先给一个"合理"假设，用支持性线索强化
2. **锚定效应**：第一发现人（如果是凶手）给出的第一印象误导全程
3. **可得性偏误**：故事里提到"某人有暴力前科"，读者自然注意他
4. **红鲱鱼**：动机最强但非凶手

## 侦探与助手

**弱点是必需的**——没有弱点的侦探是无聊的。
**助手核心功能**（华生是读者的代理人）：
- 提出错误假设 → 被侦探纠正 → 偶尔提出关键问题（推动剧情）

## 开场/结尾钩子

**开篇**：以死亡开场 / 以不可能事件开场 / 以结果倒叙开场
**章尾**：发现新线索未解读 / 角色说了半句话 / 侦探发现推论有漏洞

## 失败模式

1. 线索给不够（读者觉得你耍赖）→ 2. 给太多太明显（第三章就知道凶手）→ 3. 靠读者不知道的信息破案 → 4. 杀人动机是"踩了我的鞋" → 5. 调查拖拖拉拉 → 6. 角色全是工具人

## 三幕结构

第一幕（20-25%）：发现案件、初步调查、建立核心谜题。
第二幕（50-60%）：逐个排查、线索与误导浮现、假解答、中盘反转。
第三幕（15-20%）：推理秀、真凶揭露、情感余韵。

## 关系网设计

每个角色至少和另外两个角色有关系；每条关系线上至少有一个秘密；必须存在至少一个三角关系。

## 经典案例技法

- **叙述性诡计**（《罗杰疑案》）：叙述者就是凶手。精髓是选择性呈现，不是撒谎。只能用一次。
- **伪死亡**（《无人生还》）：真凶伪装成最早死者，利用"死人=排除嫌疑"的心理盲区。
- **诡计为情感服务**（《嫌疑人X的献身》）：日常细节赋予双重意义——初读是生活，重读是线索。
- **暗线叙事**（《白夜行》）：没有一个场景是主角同时出现并对话的。

## 自检清单

1. 核心诡计能用一句话说清吗？2. 中等聪明的读者能独立推出答案吗？3. 有假解答吗？4. 动机能说服普通人吗？5. 侦探有什么独特之处？6. 有没有章节在拖时间？
