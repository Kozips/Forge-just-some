# 悬疑类 — 质检清单（V2）

> 完稿前逐项检查。打勾通过，未通过则修改。

---

## 一、通用检查项

- [ ] 无病句和错别字
- [ ] 人称一致（无视角混乱）
- [ ] 时间线连贯（无矛盾）
- [ ] 对话自然（无AI腔/书面腔）
- [ ] 每章有明确的情节推进
- [ ] 感官描写覆盖至少三种感官（不要只用视觉）

---

## 二、铁律检查（违反任何一条都要重写）

- [ ] **线索可追溯**：所有推理线索在前文有伏笔（不能凭空出现证据）
- [ ] **真相后置**：读者可在中途猜到部分，但完整真相在最后一刻才揭露
- [ ] **误导公平**：红鲱鱼（误导线索）公平合理（不靠角色说谎骗读者）
- [ ] **角色有秘密**：每个重要角色都有秘密（增加嫌疑人池）
- [ ] **动机合理**：凶手的动机经得起推敲（"如果是我也许也会这么做"）
- [ ] **推理可复现**：侦探的推理过程读者能跟着走（不靠直觉、不靠运气）
- [ ] **悬念递增**：每章结尾的悬念强度高于上一章
- [ ] **密室公平**（如适用）：解法在物理上可行、能一句话概括、提前暗示、角色能力匹配

---

## 三、线索布局检查

- [ ] 每条关键线索至少出现两次（第一次是噪音，第二次成证据）
- [ ] 深层线索（指向真相的）分散在至少3个不同场景
- [ ] 信息投放比例正确：前25%放70%线索/中段放20%线索+误导/最后放10%线索
- [ ] 红鲱鱼有5种之一：嫌疑人/物证/行为/情感/思维定式
- [ ] 至少两条独立的推理路径交叉验证

---

## 四、角色检查

- [ ] 每个嫌疑人有：基本信息、动机、不在场证明、秘密、可疑行为、排除理由
- [ ] 侦探有明确的认知风格+必须有的弱点
- [ ] 受害者有立体设定（不是"工具人"）
- [ ] 至少有一个角色读者同时同情又厌恶
- [ ] 配角有差异化（不能所有人说话风格一样）

---

## 五、结构检查

- [ ] 前500字有钩子
- [ ] 每章结尾紧张度高于开篇（翻页冲动）
- [ ] 至少有一个假高潮/中盘反转
- [ ] 至少有一个"假死点"（节奏短暂回落）
- [ ] 结局的反转有铺垫（不是为了反转而反转）

---

## 六、对话检查

- [ ] 暗示而非直说（没有人说"我有杀他的动机"）
- [ ] 话里有话（每句有表面意思+真实意思）
- [ ] 沉默被使用（沉默比语言更有力）
- [ ] 每人有语言特征（不混淆谁是谁）
- [ ] 审讯有节奏（试探→施压→致命一击）

---

## 七、感官描写检查

| 场景类型 | 感官字数占比 | 重点感官 |
|---------|------------|---------|
| 犯罪现场发现 | 30-40% | 视觉+嗅觉 |
| 审讯/对话 | 10-15% | 听觉+视觉（肢体语言） |
| 密室/封闭空间 | 35-45% | 全感官（压迫型） |
| 恐慌/心理崩溃 | 40-50% | 触觉+听觉 |
| 推理秀/真相揭晓 | 15-20% | 视觉（细节+反应） |
| 日常/过渡 | 5-10% | 偶尔一个"不对劲"细节 |

---

## 八、密室/不可能犯罪专项检查

- [ ] 诡计在物理上可行（做过实验验证）
- [ ] 解法能用一句话概括
- [ ] 解法需要的所有元素都在揭晓前出现过
- [ ] 凶手的能力和身份匹配
- [ ] 满足七种解法之一：不是谋杀/凶手未离开/隐藏通道/凶器误判/时间误导/从外部锁门/组合型

---

## 九、影视改编前置检查

如果你的作品可能被改编为影视，需额外检查：

- [ ] 每集有独立钩子（不是连续剧就忽略）
- [ ] 内心独白可以外化（行为和对话表达）
- [ ] 视觉化呈现（线索要能"拍"出来）
- [ ] 节奏比纯文学快（影视节奏 ≠ 文学节奏）

---

## 十、写作前自检清单（开写前问自己）

1. 核心诡计/悬念能用一句话说清楚吗？
2. 线索布局公平吗？中等聪明的读者能独立推出答案吗？
3. 至少有一个假解答/中盘反转吗？
4. 动机够强吗？能说服普通人吗？
5. 侦探/主角有辨识度吗？
6. 节奏对吗？有没有纯粹在拖时间的章节？
7. 结尾的揭秘够震撼吗？
8. 角色是人还是工具？有情感厚度吗？
9. 感官描写用了五感中的几种？有没有只用视觉？
10. 你的故事属于哪个类型？类型特征体现了多少？

---

## 评分

- 全部勾选 → 优秀，可以发布
- 漏1-2项 → 良好，修一修再发
- 漏3-5项 → 及格，需要大改
- 漏5项以上 → 重写
