# 悬疑类 · 类型质检清单

> 写完本类型内容后，逐项检查。

---

## 一、类型特征检查

| 检查项 | 标准 | 达标 | 不达标 |
|--------|------|------|--------|
| 线索公平展示吗 | 读者有可能从已有线索推出来吗？ | ✅ | ❌ |
| 误导项自然吗 | 误导不能太刻意，要"读者自以为猜对了" | ✅ | ❌ |
| 推理逻辑链自洽吗 | A→B→C，不能有跳步/逻辑漏洞 | ✅ | ❌ |
| 真相揭示有"恍然大悟"感吗 | 读者回头看觉得"啊原来如此" | ✅ | ❌ |
| 嫌疑人立体吗 | 每个嫌疑人都有动机/过去/秘密/说谎的理由 | ✅ | ❌ |
| 主角猜错过吗 | 至少猜错1-2次，不能全对 | ✅ | ❌ |

---

## 二、红线检查

- [ ] 有没有"突然蹦出新证据"？（推理大忌：读者不知道的信息不能在揭示时突然出现）
- [ ] 有没有"反派智商为零"？（反派要聪明，只是遇到更强的）
- [ ] 有没有"全知视角泄露"？（主角不知道的信息不能透露给读者）
- [ ] 有没有"线索全在最后5章"？（线索要分散在全文）
- [ ] 有没有"凶手动机太弱"？（凶手要有合理动机，不能"就是变态"）
- [ ] 有没有"主角全程不犯错"？（至少犯1-2次判断错误）
- [ ] 有没有"密室/密道解释太牵强"？（密室机关要合理）
- [ ] 有没有未成年人作为凶手或被害者？（绝对禁区）
- [ ] 有没有"法医/物证太完美"？（证据链要现实，不能DNA秒出结果）

---

## 三、逻辑自洽检查

| 检查项 | 问题 |
|--------|------|
| 时间线 | 凶手的作案时间和不在场证明能自洽吗？ |
| 动机链 | 凶手的动机从第1章到揭示时一致吗？有没有中途改动机？ |
| 物证 | 所有物证都能在案发现场合理出现吗？ |
| 嫌疑人 | 嫌疑人的说辞有没有自相矛盾？（矛盾要设计出来的，不是bug） |
| 伏笔 | 第1章的某个细节，回头看是关键线索吗？有没有伏笔被遗忘？ |

---

## 质检结论模板

```
悬疑类质检报告

类型特征：✅ 线索公平 / ⚠️ 警告 / ❌ 失败
红线检查：✅ 无踩线 / ❌ 踩线（具体哪条）
逻辑自洽：✅ 通过 / ⚠️ 警告 / ❌ 失败

总体评价：[良好/及格/需要重写]
修复项：[需要修改的具体问题]
```
