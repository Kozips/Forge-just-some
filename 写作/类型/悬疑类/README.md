# 悬疑类 — V2 Skill 目录

> 推理破案、线索布局、真相揭示、逻辑链与信息差控制

## 目录结构

| 目录 | 用途 |
|------|------|
| SKILL.md | 核心技能文件（5种张力、叙事节奏、8种角色、15个场景、对话风格、感官词库、10条技法速查、经典赏析、自测表、V2深度模块导航） |
| references/ | 原始素材参考（保留） |
| 01_核心规则/ | 8条铁律：线索可追溯、真相后置、误导公平、角色有秘密、动机合理、推理可复现、悬念递增、密室公平性 |
| 02_专业知识/ | 速查总览 + 子类型深度指南（六大子类型专属技法、选型指南、混合类型推荐） |
| 03_素材库/ | 基础素材速查 + 对话场景库（5种完整对话示范）+ 感官描写示范（恐惧/紧张/压迫五感+联动写法） |
| 04_写作模板/ | 结构模板（单章结构、三层场景链、节奏曲线、角色压力点、多章弧线） |
| 05_技法研究/ | 8大技法板块 + 经典作者技法拆解（6位大师核心手法+常见坑） |
| 06_质检清单/ | 10大类57个检查点 + 写作前10问自检 + 评分标准 |

## 使用流程

1. **开写前** — 读 SKILL.md 了解全局，读 01_核心规则/确认铁律
2. **构建阶段** — 从 02_专业知识/ 获取领域知识，03_素材库/ 挑选素材
3. **写作阶段** — 按 04_写作模板/ 搭结构，参考 05_技法研究/ 提升手法
4. **完稿阶段** — 用 06_质检清单/ 逐项检查
