# 剧情递进技法 —— 悬疑推理

> 来源：剧情递进手法_五种类型全覆盖
> 四个维度：冲突升级 · 信息解密 · 情绪压强 · 格局扩展

---

## 两种典型递进套路

**套路A：案件→线索→误导→反转→真相**

先给读者一个"确定的结论"。然后用线索一点点动摇这个确定性。在读者以为接近真相时给出反转。最终真相揭开，但代价远超预期。

**套路B：表面案件→深层关联→系统性真相→道德困境**

每个案件看似独立，其实互相关联。调查越深牵扯越广。最终发现真相本身是一个道德炸弹——揭发它会伤害更多人，隐瞒它又对不起受害者。

---

## 具体例子：《白夜行》递进结构

> 表层：一起凶杀案。嫌疑人明确。读者以为这是一个"抓凶手"的故事。
>
> 中层：时间跳跃，新案件不断出现。每个案件独立但都有一条暗线连接到最初的凶杀案。读者开始怀疑"凶手到底是谁"。
>
> 深层：揭示两个主角的关系——不是简单的"凶手和帮凶"，是两个被原生家庭毁掉的孩子在黑暗中互相依存。真相揭开时，读者的愤怒指向的不是凶手，而是制造凶手的那个世界。

**递进拆解：**
- **冲突升级**：追查凶手 → 信任崩塌 → 真相vs保护 → 正义的代价
- **信息解密**：现场线索 → 人物关系 → 历史真相 → 全景拼图
- **情绪压强**：好奇 → 焦虑 → 震惊 → 悲悯
- **格局扩展**：一个案件 → 一条暗线 → 两个人的一生 → 社会性悲剧

---

## 递进节奏

核心驱动力：**信息差**。慢→快，前期铺垫后期密集。格局扩展路径：案件→关系网→社会→人性。

| 阶段 | 占比 | 关键动作 |
|------|------|---------|
| **锚定期**（前20%） | 建立案件/世界观 | 让读者以为"我懂了" |
| **递进期**（20%-60%） | 三层递进 | 每层打破一个旧认知，建立一个新认知 |
| **崩塌期**（60%-80%） | 核心反转 | 所有旧规则/认知被推翻，主角必须从零重建 |
| **终局期**（80%-100%） | 收束或开放 | 给出终极答案，或留下"真正的问题还没解决" |

## 写作检验清单

- [ ] 每一章是否比上一章的冲突强度更高？（不能原地踏步）
- [ ] 每次信息揭露是否同时制造了新悬念？（揭示≠结束）
- [ ] 情绪是否有波谷？（连续高压读者会麻木，需要喘息再加压）
- [ ] 格局扩展时主角的代价是否同步增加？（免费的升级=无聊）
- [ ] 读者在第N章是否还能回忆起第1章的悬念？（伏笔回收=信任）
- [ ] 递进是否有"不可逆"节点？（至少一个无法回头的转折点）
