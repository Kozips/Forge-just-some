# 悬疑类素材库 · 03 角色与场景库

## 一、角色设计模板

### 侦探/调查者
- **个人执念**：为什么执着于破案？（旧案未破/亲人被害/对真相有瘾）
- **致命弱点**：什么会让他判断失误？（情感干扰/过往创伤/身体健康/道德困境）
- **破案风格**：靠推理？直觉？关系网？科技？逼供？
- **出场方式**：在"不起眼日常"中展现观察力

### 嫌疑人（第一嫌疑人）
- **真正的秘密**：不是杀人，而是另一件不能被发现的事
- **困境**：说出秘密就能洗清嫌疑，但秘密比杀人罪更不敢开口
- **要点**："可疑行为"要有合理解释，但这个解释他自己不能说

### 红鲱鱼
- **为什么可疑**：因为他在做另一件"不该做的事"
- **真实身份**：受害者/被利用的人/无辜的局外人
- **揭示时刻**：侦探发现他的秘密不是犯罪而是"另一件丑事"

### 真凶
- **动机层次**：表面（钱/情/仇）→ 深层（扭曲的爱/被压迫的反弹/自以为正义的私刑）
- **伪装策略**：伪装成受害者/主动帮忙调查/装作毫不相关
- **致命破绽**：一个他自己都没想到的小细节
- **关键**：破绽要"回头看很明显，当时看不出来"。隐藏型在之前章节安排"中性"戏份——不太多也不太少。

### 受害者
- **不为人知的另一面**：生活远比表面复杂
- **自身的罪**：可能不是好人——让案件道德维度更复杂
- **留下的线索**：生前做了什么、去了哪里、见了谁

### 帮凶
- **加入原因**：被胁迫/利益共同体/情感依附
- **内心撕裂**：知道自己在犯罪，但走不掉了
- **暴露契机**：做了一件"多余的事"

### 目击者
- **诚实但错误型**：真的以为自己看到了，但记忆被环境扭曲
- **被收买型**：说的每个字都是别人教的
- **本身就是凶手型**：伪装成无辜路人，说得太流利

### 叙述者（不可靠叙述者）
- **话术**：用真实的陈述掩盖关键的省略
- **留给读者的破绽**：不经意透露的信息和想让你相信的版本有矛盾
- **揭露方式**：另一个角色/证据/叙述者自己打破第四面墙

## 二、场景设计

### 核心场景要素

| 场景 | 关键要素 | 氛围关键词 |
|------|---------|---------|
| 密室 | 空间大小、门窗锁具、室内物品、灯光 | 封闭、窒息、不可能 |
| 审讯室 | 头顶一盏灯、铁桌固定椅、偏冷温度、时钟声 | 逼迫、审视、时间流逝 |
| 犯罪现场 | 第一印象、异常之处、气味、声音 | 震撼、不协调、倒计时 |

## 三、角色关系网

```
          [死者]
         /  |  \
   [配偶] [情人] [商业伙伴]
      |      |       |
   [帮凶?] [红鲱鱼?] [真凶?]
         \   |   /
        [侦探]
              |
         [叙述者?]
```

**设计要点**：
1. 每个角色和死者至少有一条"利益线"
2. 角色之间也要有关系（不只是都认识死者）
3. 至少有一个关系"表面正常，实际扭曲"
4. 帮凶和真凶之间的关系是后期大反转点
