# 悬疑类 — 结构模板

> 本目录存放章节结构模板与叙事骨架。写作前用模板规划节奏。

---

## 核心张力公式

每章必须回答一个问题的同时，制造一个更大的新问题。

```
已知线索 A + 新发现 B → 推理方向 C
但 C 带来的新矛盾 D > 原问题 A
```

核心张力 = 读者「以为快懂了」与「其实还差得远」之间的落差。维持此落差是悬疑章的唯一引擎。

---

## 单章结构（线索-推理-新疑型）

```
### 线索获取（前25%）
- 新证据/新证人/新矛盾出现
- 角色对线索的初步解读

### 推理展开（30%）
- 角色梳理已知信息
- 排除部分可能性，锁定嫌疑人

### 转折/新疑（30%）
- 一个新信息推翻了之前的推理
- 或：一个看似无关的细节暗藏关键

### 悬念留扣（15%）
- 角色发现了一个无法解释的矛盾
- 为下一章的调查方向埋钩子
```

---

## 三层场景链（开篇/中段/翻转）

| 层级 | 功能 | 必备元素 | 时长占比 |
|------|------|----------|----------|
| **开篇场景** | 抓人、定调 | 感官冲击 or 信息落差，3句内进入冲突 | 15-20% |
| **中段场景** | 推进、误导 | 角色主动行动 + 遭遇阻力或假线索 | 50-60% |
| **翻转场景** | 颠覆、留扣 | 至少一条信息与中段推论矛盾 | 20-30% |

三层之间用「信息断裂」衔接：场景切换时，上一场的结论在下一场开头即被质疑或补充。

## 章节节奏曲线

```
紧张度
  ▲
  │        ╭─╮        ╭──╮
  │    ╭──╯  │   ╭──╯    ╰──╮
  │ ╭─╯      ╰─╯            ╰─  ← 留扣（微升收尾）
  │─╯
  └───────────────────────────→ 章节进度
   开篇钩  线索1  假死点  转折   留扣
```

- **假死点**（50-60%处）：紧张度短暂回落，给读者「喘口气」的错觉，随后立即拉高。没有假死点，读者会疲劳；假死点太长，读者会走神。
- **收尾微升**：最后一句的紧张度必须高于章节开头，确保翻页冲动。

## 角色压力点

| 压力类型 | 触发条件 | 表现方式 |
|----------|----------|----------|
| **认知压力** | 推理矛盾、信息过载 | 反复翻看证物、自言自语、失眠 |
| **人际压力** | 同伴质疑、嫌疑人反问 | 争吵、沉默对抗、信任裂缝 |
| **时间压力** | 截止期限、二次作案风险 | 加快行动节奏、做出不理智判断 |
| **道德压力** | 证据指向亲近的人 | 内心独白、延迟汇报、试探性询问 |

每章至少激活一种压力。压力叠加（如认知+道德）可制造高潮场景。

---

## 常见失败模式

| 失败模式 | 症状 | 修复方法 |
|----------|------|----------|
| **线索倾倒** | 一章塞入3条以上新线索 | 每章主线索1条，辅助线索不超过2条 |
| **神探附体** | 主角无推理过程直接得出正确结论 | 补写推理链，至少展示排除法 |
| **假反转** | 转折靠隐瞒信息而非逻辑翻转 | 转折所用信息必须在前文出现过 |
| **空悬念** | 留扣与主线无关 | 留扣必须指向下一章的核心冲突 |
| **情绪真空** | 角色面对线索无心理反应 | 每条关键线索后补1-2句内心波澜 |

## 结尾类型

| 类型 | 结构 | 适用场景 |
|------|------|----------|
| **断崖式** | 在关键动作/发现的瞬间戛然而止 | 高潮章、欲罢不能型 |
| **余震式** | 揭示一个小真相，但角露骨绪震动更大 | 情感转折章 |
| **拼图式** | 新线索到位，读者可以自行拼出部分真相 | 中段推进章 |
| **逆转式** | 章末最后一句推翻全章推论 | 关键反转章 |
| **静默式** | 角色独处，消化信息，暗示内心变化 | 过渡章、喘息章 |

---

## 多章弧线（发案-调查-突破-真相型）

```
第1-2章：案件发生 → 发现第一个矛盾
第3-5章：调查推进 → 每章一个新线索+一个新疑问
第6-8章：陷入僵局 → 偶然发现突破口
第9-10章：真相揭露 → 反转（不是你以为的那个人/原因）
```

## 开篇钩子模板

**类型A — 现场开局**：警察到场时，门是反锁的。窗户从里面钉死了。死者面朝下趴在客厅中央。
**类型B — 倒叙悬念**：我杀了人。但警察说我不是凶手。他们说得对——但也不全对。

## 写作检查清单

- [ ] 本章核心张力是什么？（一句话写清「以为/其实」落差）
- [ ] 三层场景链是否完整？（开篇抓人 → 中段推进 → 翻转留扣）
- [ ] 节奏曲线是否有假死点？（50-60%处短暂回落）
- [ ] 每条关键线索后，角色是否有心理反应？
- [ ] 转折所用的证据，是否在前文已经出现？
- [ ] 留扣是否直接指向下一章的核心冲突？
- [ ] 结尾紧张度是否高于开篇？（翻页冲动）
- [ ] 是否避开了「线索倾倒」「神探附体」「假反转」？
