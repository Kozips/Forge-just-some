# 恐怖类 — 结构模板

> 本目录存放章节结构模板与叙事骨架。写作前用模板规划节奏。

## 核心张力公式

**恐怖 = 熟悉感被撕裂 + 无法逃离 + 未知持续扩大**
每章必须锚定一个核心恐惧源（空间封闭/认知崩塌/身份侵蚀/存在威胁），避免多源混杂导致恐惧失焦。

---

## 单章结构（日常-异常-恐惧型）

```
### 日常锚定（前15%）
- 用普通生活场景开场（起床/上班/做饭/等人）
- 一个几乎不被注意的细节（窗外的声音/物品的位置）

### 异常积累（30%）
- 第二个异常出现，角色开始注意
- 角色试图合理化（风吹的/我记错了/是猫）

### 恐惧升级（35%）
- 异常变得无法合理化
- 角色的感官开始"出错"——看到/听到不该存在的东西

### 悬念/恐惧延续（20%）
- 不给答案——异常暂时消失
- 角色和读者一起"等待"下一次出现
```

## 三级场景链（每章必须包含）

| 层级 | 功能 | 示例动作 |
|------|------|----------|
| **开场场景** | 建立安全感/日常锚点，植入一个微小裂缝 | 角色做晚饭，水龙头滴水声节奏"不对" |
| **中途场景** | 异常升级，角色主动行动（调查/对抗/逃避），恐惧从被动承受到被迫面对 | 角色追踪滴水声到浴室，水龙头关着 |
| **反转场景** | 恐惧源发生位移或质变，打破角色此前的认知框架 | 水滴声从天花板传来——楼上已经搬空三年 |

## 章节节奏曲线

```
恐惧感
  ▲
  │        ╱╲
  │      ╱    ╲    ╱╲（二波）
  │    ╱        ╲╱    ╲___
  │  ╱
  │╱
  └─────────────────────────── 章节进度
  日常锚定  异常积累  恐惧升级  余韵悬挂
  (15%)     (30%)     (35%)    (20%)
```

**节奏要点**：恐惧不是直线上升，而是"波浪推进"——每波峰值必须高于上一波；章节结尾的恐惧水平应高于开篇。

## 角色压力点（逐层挤压）

| 阶段 | 心理状态 | 典型行为 |
|------|----------|----------|
| 否认期 | "没什么大不了" | 合理化异常，回避 |
| 警觉期 | "哪里不对" | 开始留意细节，改变路线 |
| 崩裂期 | "这不是真的" | 感官失调，自我怀疑 |
| 应战期 | "不管是什么我要面对" | 主动调查/对抗，代价最高 |

每章推进一个压力层级，不可原地踏步。

## 常见失败模式

| 失败 | 症状 | 修正 |
|------|------|------|
| 恐惧通胀 | 异常出现太频繁，读者脱敏 | 每波异常间隔必须递减，但峰值递增 |
| 解释过早 | 角色立刻理解异常原因 | 推迟合理化至少一个场景层级 |
| 感官单一 | 全靠"看到"吓人 | 至少覆盖视觉+听觉+触觉中的两种 |
| 旁观者视角 | 读者置身事外 | 限制叙事视角，让读者和角色共享信息盲区 |
| 氛围空转 | 场景描写很恐怖但角色无行动/无变化 | 每个恐怖场景必须改变角色的一个认知或行为 |

## 多章弧线（好奇-恐惧-真相-代价型）

```
第1-2章：日常中的微小异常（不被注意）
第3-4章：异常加重，角色开始调查
第5-7章：发现线索，深入危险区域
第8-9章：真相揭露（但真相本身更恐怖）
第10章：付出代价，异常停止——但留下了痕迹
```

## 结局类型速查

| 类型 | 核心 | 适用场景 |
|------|------|----------|
| **余毒型** | 恐惧源消除，但角色/世界已不可逆地改变 | 强调创伤的不可修复性 |
| **循环型** | 异常重启，暗示角色成为下一个受害者 | 短篇恐怖、都市传说型 |
| **代价型** | 角色"赢了"，但失去了比恐惧更重要的东西 | 人性恐怖、伦理困境型 |
| **开放型** | 恐惧悬而未决，角色选择继续面对 | 长篇连载、续集铺垫型 |

## 开篇钩子模板

**类型A — 日常中的裂缝**：她打开家门，发现鞋柜上的照片框倒了。她扶起来。第二天，照片框又倒了。这次照片面朝下。
**类型B — 倒叙恐惧**：医生问我为什么半夜三点站在马路中间不动。我说不出来。因为我不记得。

## 写作检查清单

- [ ] 核心恐惧源是否单一且贯穿本章？
- [ ] 开场/中途/反转三个场景是否都已铺设？
- [ ] 节奏曲线是否有至少两个恐惧峰值且递增？
- [ ] 角色压力是否推进了一个层级（不能原地踏步）？
- [ ] 感官描写是否覆盖至少两种（视/听/触）？
- [ ] 异常是否比角色的合理化"快一步"？
- [ ] 结尾是否留有"等待下一次"的余韵，而非给出完整解答？
- [ ] 上一章的悬念/恐惧残余是否在本章延续？
