# 恐怖创作速查

## 速查表

**核心张力公式**：恐惧 = 未知 x 无力感 x 沉浸感 / 预期 x 控制感

**单章结构（日常-异常-恐惧型）**：日常锚定 15% -> 异常积累 30% -> 恐惧升级 35% -> 悬念延续 20%

**节奏铁律**：恐惧是波浪推进，每波峰值高于上一波，章末恐惧水平高于开篇。

**角色压力点**：否认期 -> 警觉期 -> 崩裂期 -> 应战期 -> 解离期

**常见失败**：恐惧通胀 / 解释过早 / 感官单一 / 旁观者视角 / 氛围空转

**结尾类型**：余毒型 / 循环型 / 代价型 / 开放型 / 真相型

**开篇钩子**：A. 日常裂缝（照片框倒了，第二天又倒了，这次面朝下） B. 倒叙恐惧（半夜三点站在马路中间，我不记得为什么） C. 声音先行（有人敲门三下，节奏很稳，但她住十八楼）

**粗俗/文雅铁律**：雅是蓄力，粗是释放。先雅后粗，效果翻倍。全程一个调，读者就麻了。
- 文雅（暗示>展示）：心理恐怖、氛围铺垫、暗示伏笔
- 粗俗（直接描写）：怪物描写、身体恐怖、猎奇向
- 先雅后粗：高潮爆发

**自检清单**：恐惧源单一？三层场景链？双峰值递增？压力递进？感官覆盖两种？留白？余韵？

## 角色速查（写作时直接参考）

> 完整版见 `人物/references/人物联动写作/恐怖类.md`

### 薄妄（病娇·优雅施虐·财阀疯批）
- 语言："好可怜，伤了呢，我来给你心疼。"（用怜爱口吻说要命的话，一个字一个字往外蹦）
- 语气：嗓音温柔且磁性，内容与语气形成强烈反差，偶尔一声短促的"呵"让人脊背发凉
- 动作：用指尖或刀刃慢慢滑过对方皮肤像鉴赏艺术品；单手插裤袋，姿态散漫压迫感极强
- 用法：病娇操控型反派，语气越温柔内容越恐怖，反差感是核心恐怖手段

### 鹿绫（隐忍坚韧·绝境求生）
- 语言："我要他们的玩具，你们能买给我，我就签。"（被逼到绝路时的荒诞反击）
- 语气：平淡到近乎冷漠，不哭喊不求饶，声音很低但每个字都很清楚
- 动作：赤脚踩碎石地跑鞋掉了也不捡；被钳住下巴时不吭声，越逆境越冷静
- 用法：隐忍型受害者/幸存者，用沉默代替拒绝，签字时手很稳——克制比呐喊更有力

### 白七（老好人·被迫黑化）
- 语言："没的选啊，原来这就是鬼杀人的理由吗？都是为了生存吗？"（自嘲式追问）
- 语气：平时平淡温和带着服务行业的热心，决意做事时语气变冷，是无奈的决绝
- 动作：下意识凑到后视镜看自己——变成鬼了还在乎长相；勾了勾唇角自嘲——用笑化解恐惧
- 用法：普通人被迫卷入恐怖事件，自嘲幽默是恐惧的出口，从底层求生者到诡异BOSS的蜕变
