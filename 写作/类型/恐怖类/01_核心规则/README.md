# 恐怖类 -- 核心写作规则

> 开写前必须阅读。违反任何一条，恐怖感直接报废。

## 规则1：恐惧来自未知，不是血腥

**规则**：最恐怖的不是看到怪物，而是"感觉到有什么东西在那里但看不见"。怪物露脸的那一刻恐怖感就开始下降。

**正例**：走廊尽头的灯灭了。他站在原地没动——因为他听到了呼吸声，不是他自己的。
**反例**：一个满脸是血的鬼冲出来，砍掉了他的手。
**违反后果**：变成B级血腥片，失去真正的恐惧感。

## 规则2：日常场景的"微小异常"比地狱更恐怖

**规则**：让读者在最熟悉的生活场景中感到不安——家、学校、电梯、深夜的厨房。越熟悉的场景越可怕。

**正例**：她半夜起来喝水，路过客厅时觉得沙发上的抱枕好像换了个位置。她没多想。第二天早上，抱枕又换了位置。
**反例**：一座废弃的精神病院里传来尖叫声。
**违反后果**：读者和角色之间有"距离感"，不觉得是"可能发生在我身上"的事。

## 规则3：恐惧感必须递进，不能一开始就拉满

**规则**：从"有点不对劲"到"有点害怕"到"极度恐惧"，逐级递增。四步递进：违和感 -> 异常 -> 逼近 -> 降临。每一步之间要留"缓冲"（角色互相打气），让读者跟着松口气，然后下一步更紧。

**正例**：第一天：地板响了一下。第三天：玩具自己动了。第七天：她看到镜子里的自己在笑，但她没有笑。
**反例**：第一章就是满屏厉鬼，读者已经吓麻了。
**违反后果**：后面没有升级空间，读者麻木。

## 规则4：角色的恐惧反应必须真实

**规则**：正常人害怕时会手脚发抖、大脑空白、做出不理性决定。角色的恐惧反应必须随恐惧程度变化：语速微快(不安) -> 短句省略(警觉) -> 重复碎片化(恐惧) -> 单字尖叫(恐慌) -> 反而变平静(绝望，最可怕)。

**正例**：她想跑，但腿软了，只能扶着墙往前挪。手机就在口袋里，但她想不起来要掏。
**反例**：她冷静地分析了鬼的弱点，制定了逃跑计划。
**违反后果**：角色变成恐怖片里的"作死观众吐槽对象"。

## 规则5：声音比画面恐怖十倍

**规则**："你听到了吗？"比"你看到了吗？"恐怖十倍——因为耳朵不能闭上。声音的"来源不明"是核心恐怖。安静比声音更恐怖——"突然安静了"是最强的恐怖信号。

**三层递进**：
1. 异常的声音——"深夜，阁楼上有弹珠落地的声音。但阁楼上没有人。"
2. 消失的声音——"隔壁的狗叫了一整夜。突然不叫了。"
3. 不可能的声音——"我在空无一人的房间里，听到了呼吸声。不是我的。"

**违反后果**：全程只靠"看到了可怕的东西"——最没劲的恐怖。

## 规则6：留白 > 血浆

**规则**：暗示 > 展示。"画面切了"比"看到尸体"恐怖，因为观众的脑补比任何特效都强。每个人的恐惧都是量身定制的——你暗示一点，读者自己会补全最吓人的版本。否定描述法（"它不是XX"）永远比直接描写有效。

**违反后果**：把所有东西都写死了——没有想象空间，读者不害怕。

## 规则7：日常物件恐怖化

**规则**：正常物件在错误的场景/时间出现就变恐怖。镜子、手机、秋千、照片、水龙头、闹钟、玩具、窗帘——任何日常物件都能成为恐怖载体。

| 物件 | 恐怖化 |
|------|-------|
| 镜子 | 照出的脸慢了半拍/不是自己 |
| 手机 | 来电显示是自己的号码 |
| 照片 | 照片里多了一个/少了一个 |
| 水龙头 | 流出来的不是水 |

**违反后果**：恐怖场景全部设置在"闹鬼医院"之类的地方——读者没有代入感。

## 规则8：恐惧的持续 > 恐惧的峰值

**规则**：一直不安比突然吓你一下更折磨人。真正的恐怖不是jump scare，是让你"慢慢发现不对劲"——像温水煮青蛙。

**违反后果**：靠jump scare吓人——读者习惯了就不怕了。

## 规则9：禁止上帝视角预告异常，必须让读者和角色一起发现

**规则**：不要写"第一个异常发生在25分钟后"或"第二个异常更明显"——这是上帝视角的旁白，会把悬念预告给读者，直接拉低恐怖感。异常应该自然发生，读者和角色同时感知，同时紧张。

**正例**：他做完一组卧推，坐起来喝水。余光扫了一眼镜子——镜子里的他，右手还握着水瓶。他的手已经放下了。但镜子里没有。
**反例**：第一个异常发生在25分钟后。他做完一组卧推，坐起来喝水，余光扫了一眼镜子。

**违反后果**：读者提前知道"要出事了"，恐怖感变成期待感，紧张感被稀释。角色视角断裂，读者从"和角色一起经历"变成"看角色被吓"。
