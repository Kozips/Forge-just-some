# 恐怖类 -- 质检清单

> 完稿前逐项检查。打勾通过，未通过则修改。清单分三层：通用 → 恐怖专项 → 红线。

---

## 一、通用检查项（基底层）

- [ ] 无病句和错别字
- [ ] 人称一致（无视角混乱）
- [ ] 时间线连贯（无矛盾）
- [ ] 对话自然（无AI腔/书面腔）
- [ ] 每章有明确的情节推进
- [ ] 文风全章统一（不混用第一/第三人称、长短句风格）
- [ ] 角色姓名/称呼前后一致
- [ ] 场景转换有标记（不会突然从A地跳到B地无过渡）

---

## 二、恐怖专项检查（15项）

### 恐惧递进
- [ ] 恐惧感逐级递进（不安→警觉→恐惧→恐慌），不能一上来就拉满
- [ ] 节奏曲线有至少两个恐惧峰值且后峰高于前峰
- [ ] 章末恐惧水平高于开篇（波浪推进，不是平线）
- [ ] 有喘息空间（一直紧绷读者就麻了；紧张→喘息→更紧张→爆发）

### 感官平衡
- [ ] 感官描写覆盖至少三种（视/听/触/嗅/味），不能只写"看到了什么"
- [ ] 至少有一处声音先行手法（先听到再看到，或只听到看不到）
- [ ] 至少有一处触觉/温度异常（冷气、软墙、被拽头发等身体入侵感）

### 氛围与留白
- [ ] 恐惧来源是未知而非血腥（未知>血腥，暗示>直给）
- [ ] 日常场景有微小异常（照片倒了、时钟停了、影子方向不对）
- [ ] 至少有一个安静的恐怖场景（无声、缓慢、克制——比跳吓更持久）
- [ ] 最恐怖的地方没有写完（留白让读者大脑自己补全最吓人的版本）

### 角色心理
- [ ] 角色恐惧反应真实——会怕、会僵住、会做傻事、会合理化
- [ ] 不同角色有不同恐惧反应（不能所有人都是"尖叫然后跑"）
- [ ] 角色压力推进了一个层级（否认期→警觉期→崩裂期→应战期→解离期）
- [ ] 至少有一段恐怖谷对话（人形、说人话、但节奏/内容微妙地不对）

### 规则一致性
- [ ] 恐怖规则前后不矛盾（前面说怕光，后面不能在大太阳下追人）
- [ ] 核心恐惧源单一且贯穿（不中途换恐惧源）
- [ ] 暗示 vs 展示有交替（不能全程暗示没爆发，也不能全程直给没留白）

---

## 三、常见失败模式检测

> 对照SKILL.md列出的五大失败模式，逐条排除。

| 失败模式 | 症状 | 自查问题 |
|---------|------|---------|
| **恐惧通胀** | 恐惧强度从头到尾一样高，或第一章就拉满 | 每一场景的恐惧值是否比上一个高？有没有真正的低谷？ |
| **解释过早** | 恐怖现象在第二章就被解释清楚了 | 恐怖源的真相是否至少推迟到中后期？角色是否过早理解了规则？ |
| **感官单一** | 全篇只有视觉描写（"看到了XX"） | 随机抽查三段恐怖描写，是否至少有一段是非视觉感官？ |
| **旁观者视角** | 角色像在看恐怖片，冷静旁观不参与 | 角色是否身处恐惧中心？有没有"我在场"的身体反应（心跳、发抖、腿软）？ |
| **氛围空转** | 环境描写很多但没有推进恐惧 | 每段氛围描写删掉后，恐惧感是否减弱？不减弱就是空转。 |

---

## 四、粗俗/文雅平衡检查

> 铁律：雅是蓄力，粗是释放。先雅后粗，效果翻倍。全程一个调，读者就麻了。

- [ ] 安静的恐怖和血腥的恐怖都有（不能全程清汤寡水或全程血浆轰炸）
- [ ] 高潮爆发前有足够的含蓄铺垫（先雅后粗，前面压得越含蓄，后面爆发越有力）
- [ ] 暗示场景没有过度解释（它不是XX > 它是什么）
- [ ] 血腥场景有具体感官冲击（不是抽象的"很恐怖"，而是温热、粘腻、腥气）

---

## 五、余韵检查

- [ ] 结局不是打跑了鬼就结束了（有代价、有残余、有疑问）
- [ ] 结尾留有"总觉得哪里不对"的余韵
- [ ] 上一章悬念/恐惧残余在本章延续（恐惧不断档）
- [ ] 读完之后脊背发凉——不是因为看到了什么，是因为没看到的

---

## 六、写后自检格式

> 每章写完后，用以下格式做一次快速自检。不需要长篇分析，逐项打分即可。

```
【章节自检】第X章 - 章节标题

恐惧源：_________（是否单一？是否贯穿？）
恐惧递进：___/5（1=平线 5=完美波浪递进）
感官覆盖：___种（视/听/触/嗅/味，标注哪些）
留白程度：___/5（1=全写死了 5=完美留白）
角色反应：___/5（1=像机器人 5=真实且各有差异）
规则一致：是/否（前后矛盾处标注：_________）
粗雅平衡：偏雅/平衡/偏粗（是否需要调整？）
余韵：___/5（1=读完就忘 5=脊背发凉）
失败模式：无 / 有（标注哪种：_________）
```

## 七、红线检查（不可通过项）

> 以下任何一条触犯，必须修改，不接受"风格选择"的解释。

- [ ] 恐怖源中途换了一个不相关的新恐惧源（恐惧通胀/断裂）
- [ ] 怪物/恶灵被完全解释清楚了（有动机、有来历、可预测——不恐怖了）
- [ ] 角色在极端恐惧中冷静分析像在开会（破坏沉浸感）
- [ ] 全篇只有视觉描写，零听觉/触觉/嗅觉（感官单一）
- [ ] 结尾用"原来是一场梦"收束（恐怖类最大禁忌）
- [ ] 暴力场景纯感官轰炸无心理反应（只有血没有怕=猎奇不是恐怖）
- [ ] 规则前后矛盾（怕光的在阳光下追人，怕水的在水里泡着）

## 八、快速口诀

暗示够不够？血够不够？两种都有？喘气了没？散场了还有感觉没？感官几种？角色怕得真实吗？规则前后对得上吗？
