# 恐怖对话数据库

> 核心原则：恐怖不是"说了什么恐怖的话"，是"正常话被不对的人/不对的时间/不对的方式说出来"。

---

## 1. 深夜独居——隔壁不该有的声音

**场景**：凌晨三点，独居女生。隔壁——她没有邻居。

```
像有人在搬家具。隔壁那间是空的。房东说了，半年没人租。

声音停了。她翻了个身。又响了。这次是脚步声。从墙那边传过来，一步一步。
走到墙边——停了。

手机震了一下。房东发来的：隔壁那个租客说认识你，让我把你微信给他，我发了啊。

她打字的手在发抖：隔壁没有人住。

房东秒回：不可能，他刚搬进去，行李箱都是新的。

墙那边又响了。这次是敲墙声。咚。咚。咚。三下。像在确认她在不在。

手机又震了。好友申请。备注写着：你家WiFi密码是什么？

通过验证之后，对方发来一张照片。是她的卧室。从上往下拍的。角度——从天花板。
```

**技法**：信息逐步揭露——声音→脚步→敲墙→照片。WiFi密码用日常感反衬恐怖。天花板角度暗示"它"一直在上面。

## 2. 镜子里的自己做了你没做的动作

**场景**：公共洗手间，加班到深夜的程序员。

```
他转身往门口走。走到一半——他没回头，但他听见水龙头开了。

门上也有镜子，他不需要回头就能看到洗手台。
镜子里的他还在洗手。他没有在洗手。他站在门口。
镜子里的那个他，洗完手，抬起头，对着他——笑了一下。

他转身跑了。口袋里的手机亮了。备忘录提醒。
写着：不要在洗手间里照镜子。你在镜子里的不是你。
创建时间——十分钟前。他不记得自己设过这条备忘录。
```

**技法**：镜子恐怖核心——"用你的脸对你做你没做的事"。备忘录的自设性质暗示意识层面的替换已经开始。

## 3. 朋友聚会——有人说了一个不能说的名字

**场景**：五个人在家吃火锅，席间有人说了一个"不存在的"第六个人。

```
"你来过那个空位的对吧？""上周聚餐她就坐那个位置——"她指了指唯一的空位。

赵楠的声音突然很平。"晓晓，上次聚餐我们只有五个人。""不对，六个人。"
赵楠看着她。表情像在看一个不该醒的人。"没什么。你把那个空椅子收了吧。"
"没有空椅子啊。"王凯说。

李晓晓看向她指过的位置。椅子确实不在了。但她刚才明明指着它说话的。
她低头看了一眼手机。上次聚餐的合照——五个人。
照片放大。她坐的位置旁边，有半截手搭在椅子靠背上。没有身体。只有半截手。
```

**技法**：集体记忆的"消失"比个人恐怖更具冲击力。"半截手"用视觉碎片比完整的鬼影更诡异。

## 4. 废弃医院——对讲机呼救（双声源不同步）

**场景**：两个探险博主在废弃医院直播，对讲机收到信号。

```
对讲机滋滋响了两声。然后传来第三个人的声音。很小。像隔着很远。
"有人吗……救救我……""阿诚，你那边有人？""没有啊。就我一个。"

他们在走廊尽头找到一扇门。门上写着"手术室-3"。门缝里有光。
对讲机又响了。"我在里面。门没锁。请你们——快一点。"

小陈的手放在门把上。对讲机贴着耳朵。
他忽然注意到——
对讲机里的声音——和门里面传出来的声音——不同步。
门里面的呼吸声比对讲机里的快。快很多。

阿诚的脸白了。"那对讲机里说话的……是谁？"

门从里面打开了。手术台上躺着一个人。盖着白布。
白布下面的形状——不太像人。
```

**技法**：双声源不同步是核心恐怖——同一个"人"在对讲机里和门后面同时说话但节奏不同，意味着至少有一个不是人。

## 5. 童年房间——一切都大了一号

**场景**：成年后回到老家，推开童年卧室门，所有家具大了一号。

```
抽屉的把手在他下巴的高度。他小时候每天够这个把手写作业。
拉开抽屉。小学作业本。翻开。字迹是小时候的。但本子——A4纸那么大。
他记得这个作业本。手掌大小。现在上面的字还是手掌大小。但本子变大了。

走到床边坐下。脚够不到地。他一米七八。这张床让他像个小孩坐在大人的床上。

枕头底下露出一角东西。是他小时候的照片。五岁。站在这个房间里。
照片里的家具跟他记忆中一样——小小的，适合五岁小孩的。
但照片背景里——窗户外面站着一个人。

背面写着一行字。他自己的字迹。但不是小时候的字迹。
是现在的字迹。写着：回来就好。
```

**技法**：比例失调的恐怖来自"空间认知被颠覆"——不是你在变大，是空间在变化。照片背面的字迹是"现在的自己写给过去的自己"——时间线已经乱了。

## 对话写作核心原则

| 原则 | 说明 |
|------|------|
| 日常感 > 怪异感 | "WiFi密码是什么"比"你是鬼吗"恐怖十倍 |
| 用声音设计节奏 | 咚咚咚的敲墙、滋滋的对讲机——声音是最原始的恐惧触发器 |
| 身体反应比心理描写诚实 | "后背开始冒汗""手在发抖"比"他感到非常恐惧"更真实 |
| 照片/镜子/手机屏幕 | 反射类媒介天然恐怖——因为你在看"另一个自己" |
| 结尾不给答案 | 最后一击永远是开放式的——门开了、照片里有人、字迹是自己的 |
