## 一、场景素材（精选6段）

### 深夜独处
> 凌晨两点，她被一个声音惊醒。不是外面的声音——是厨房里的。水龙头在滴水。她记得睡前关紧了。她没动，侧着耳朵听。滴答，滴答，滴答——停了。安静了大概十秒，然后柜门关上了。她一个人住。

### 镜中异常
> 她在卫生间洗脸，抬起头看镜子的时候——自己的倒影好像慢了半拍。她抬起右手，镜子里的她也抬起了右手。但她总觉得那个动作比自己慢了一点点。然后她笑了。镜子里的她也笑了。但她不记得自己笑过。

### 日常被打破
> 小区的路灯每天晚上七点亮。今天六点就亮了。他没太在意，直到他注意到——路灯下面站着一个人。穿着和他一模一样的衣服，面朝他的窗户。他拉开窗帘想看清楚，那个人不见了。路灯灭了。

### 废弃空间探索
> 走廊很长。墙皮脱落，地上全是碎玻璃和灰尘。一股霉味混着什么说不清的腐臭。护士站的台面上还有一本打开的记录簿。"1998年11月7日。入院三人。出院零人。死亡——"她没念下去。

### 电话恐怖
> 手机亮了。来电显示：奶奶。奶奶去世两年了。她接了。"丫头啊。"是奶奶的声音。一模一样。电话那头传来了炒菜的声音。香味——从手机里飘了出来。红烧肉的味道。她挂了电话。厨房里没有灯。但有红烧肉的味道。

### 监控异常
> 走廊尽头有影子。但走廊里没有人。光源在哪？没有光源。影子在墙上慢慢移动。从左边到右边。影子停了。影子的"头"——转向了摄像头。就像它知道有人在看。

## 三、五感描写词库精选

### 视觉（5条）
- 手电筒扫过去，光柱尽头什么都没有；扫回来，有东西站在那里
- 走廊比昨天长了三倍。她确定自己没有走错
- 所有时钟都停在了同一个时间
- 照片里所有人都在看镜头，除了角落里那个——它在看你
- 她的影子在她没有动的情况下，自己转了个方向

### 听觉（5条）
- 楼上的脚步声停在了正上方。它没有离开。它在等
- 电话突然响了。来电显示是她自己的号码
- 所有的声音都消失了。不是安静——是声音被吞掉了
- 她屏住呼吸，房间里安静到能听到自己的心跳——然后她听到了第二颗
- 有人在叫她的名字。声音是从衣柜里传出来的。很轻，很温柔

### 触觉（3条）
- 被子下面有另一只脚。冰凉的湿的，正在慢慢靠近她的腿
- 手摸到墙壁的那一刻，墙壁摸起来是软的——像皮肤
- 躺在床上的时候，床垫另一边凹下去了，像有人坐了下来

### 嗅觉（3条）
- 潮湿的泥土味，从不应该有泥土的地方——卧室的墙壁里
- 花香。但这个季节不应该有花。而且味道是从地下室传上来的
- 空气里有烟味，但找不到烟的来源。像有什么刚烧完
