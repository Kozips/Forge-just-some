# 恐怖常识

> 写恐怖不需要你真的见过鬼，但你需要知道在哪些场景下人类最容易害怕、什么规则能让恐惧可信、什么细节能让恐怖"落地"。

---

## 一、封闭空间设定

### 1.1 医院/精神病院

| 要素 | 恐怖来源 | 写作细节 |
|------|---------|---------|
| 白色走廊 | 视觉单调导致感知麻木 | 荧光灯的嗡嗡声、消毒水的气味、回声 |
| 住院部夜晚 | 病人的呻吟、护士的脚步、未知的病痛 | 监控器的滴滴声突然变成持续长音 |
| 手术室 | 无影灯下完全暴露 | 麻药的甜味、金属器械碰撞声、意识清醒但无法动弹 |
| 地下太平间 | 低温、死物聚集 | 冷柜抽屉的拉合声、温度骤降、排出的白雾 |
| 废弃病栋 | 历史暴力的残留 | 生锈的约束带、墙上的抓痕、病历档案 |

**特别注意**：精神病院恐怖的核心张力是"我到底是病人还是正常人"——如果你的角色被关进去，读者会自动质疑他的认知可靠性。

### 1.2 学校

| 要素 | 恐怖来源 | 写作细节 |
|------|---------|---------|
| 放学后的教室 | 人去楼空的反差 | 桌椅排列整齐但有一张歪了、黑板上有没擦干净的字 |
| 深夜操场 | 空旷+黑暗+回声 | 操场中央的声音传到边缘会延迟 |
| 地下室/储藏室 | 学校的"隐藏空间" | 旧器材、体育用品的诡异形状、积水 |
| 老旧厕所 | 水滴声、瓷砖回声、镜子 | 最后一隔的门锁着但从外面看不到脚 |
| 顶楼/天台 | 高处+风+坠落恐惧 | 生锈的栏杆、风声像哭声 |

**特别注意**：学校是儿童和青少年的"领地"——当这个地方变得不安全，读者感受到的是"连保护孩子的地方都不安全了"。

### 1.3 公寓/住宅

| 要素 | 恐怖来源 | 写作细节 |
|------|---------|---------|
| 独居住宅 | 孤立+安全感被破坏 | 钥匙转了两圈但门没锁/有人用过你的牙刷 |
| 老旧公寓楼 | 结构噪音、隔音差、邻居未知 | 隔壁的笑声凌晨三点传过来/楼上没人住但有拖拽声 |
| 阁楼/地下室 | 不常进入的"暗区" | 门框上有新的划痕/空气中有一种没有理由的潮气 |
| 隔壁房间 | 近在咫尺但看不到 | 从墙那边传来的声音——手指划过墙壁的声音 |
| 镜子/窗户 | 自我映射的异常 | 镜中的倒影延迟了一秒/窗外的人在看你但楼层很高 |

**特别注意**：家是最后的安全堡垒。当家变得恐怖，角色无处可逃——这是最大的绝望。

### 1.4 电梯

**核心恐怖要素**：
- 密闭（2平方米左右，无法躲避）
- 失控（你不知道电梯把你带到哪一层）
- 强制共处（和陌生人困在一起）
- 下坠恐惧（电梯自由落体）
- 监控（有人在看但你看不到他们）

**操作要点**：电梯恐怖的关键是"数字"——楼层显示的数字不对，或者数字在减少但电梯没有动。

### 1.5 森林

| 要素 | 恐怖来源 | 写作细节 |
|------|---------|---------|
| 迷路 | 方向感消失 | 太阳的位置和指南针不一致 |
| 视线被遮挡 | 不知道周围有什么 | 树干后面露出一个不应该在那里的人影 |
| 声音环境 | 树叶摩擦声掩盖其他声音 | 突然所有鸟都不叫了 |
| 天黑 | 光线消失的速度比你想的快 | 从"还能看清"到"完全看不见"只用了十分钟 |
| 路径消失 | 来时的路不见了 | 你明明是直着走的，但回到了原点 |

## 二、超自然规则设计

### 2.1 鬼怪的限制条件

**好的超自然存在都有规则和限制——无限制的存在不可怕，因为它"什么都行"，读者没法思考，也没法和角色一起想办法。**

| 限制类型 | 示例 | 叙事价值 |
|---------|------|---------|
| **时间限制** | 只在凌晨3-4点出现/只在特定节日活动 | 角色可以有"安全时间"，增加倒计时感 |
| **空间限制** | 只在这栋房子里/不能进入有十字架的房间 | 角色有"安全区"，但安全区可能逐渐缩小 |
| **触发条件** | 必须被看到才能行动/必须被叫名字才能找到你 | 角色可以通过行为控制危险（不看/不说名字） |
| **能力限制** | 只能影响情绪/不能直接伤害身体但能操控 | 恐怖更心理化，角色必须对抗自己的心智 |
| **传播规则** | 只能通过特定方式传播（看录像/念名字/搬进房子） | 角色必须阻止传播，增加了主动对抗的可能性 |

### 2.2 诅咒的触发条件设计

| 诅咒类型 | 触发条件 | 传播方式 | 解除条件 | 代表作 |
|---------|---------|---------|---------|--------|
| **行为诅咒** | 做了某件事（打开禁忌之门/念了某个名字） | 不传播，只针对触发者 | 找到触发条件的逆向操作 | 《午夜凶铃》 |
| **位置诅咒** | 进入某个地方（搬进闹鬼的房子） | 离开也不行，已沾染 | 满足怨灵的需求 | 《咒怨》 |
| **传播诅咒** | 接触了某个物品/信息 | 通过特定载体传播 | 找到源头 | 《午夜凶铃》 |
| **遗传诅咒** | 出生在这个家族 | 通过血缘传播 | 几乎无法解除 | 《遗传厄运》 |
| **时间诅咒** | 在特定时间做了特定的事 | 倒计时式 | 在时间到期前完成某个条件 | 《死神来了》 |

### 2.3 规则设计原则

**原则一：规则必须简单**
一条到三条规则就够了。规则太多，读者记不住，你也会在后面的故事中自相矛盾。

**原则二：规则必须一致**
一旦建立规则，不能随意打破。打破了规则=打破了读者的信任。

**原则三：规则本身就是恐怖的一部分**
当角色发现"只要不看它的眼睛它就不能动"时，角色会在接下来的每一秒都和"不要看它眼睛"这个念头搏斗——这种持续的自我控制本身就是恐怖。

**原则四：给角色（和读者）一线希望**
纯粹的绝望会让读者放弃——"反正怎么都会死，我不看了"。给一个解除条件，即使这个条件极其困难，也能让读者保持参与感。

## 四、都市传说结构

### 4.1 都市传说的标准结构

```
可信的开头 → 奇怪的中间 → 恐怖的结尾 → 警示性收尾
```

**可信的开头**：一个正常人在一个正常场景中。"我朋友的朋友……""去年在这个城市……"

**奇怪的中间**：一个无法解释的事件开始发生。但角色试图用常理解释。

**恐怖的结尾**：真相揭晓——但往往太晚了。

**警示性收尾**（可选）：不要做某某事/不要去某某地方/不要在凌晨几点看镜子。

### 4.2 经典都市传说模板

| 模板 | 结构 | 经典案例 |
|------|------|---------|
| **保姆型** | 保姆在雇主家接到恐怖电话→电话来自屋内 | 《惊声尖叫》原型 |
| **消失型** | 某人在旅途中消失→留下的线索暗示"不是自愿离开" | 各种"消失的搭车客" |
| **替代型** | 你注意到身边的人"变了"→他/她不是原来那个人 | 复制品/替换者 |
| **传染型** | 看了某个东西/知道某个信息就会被"感染" | 《午夜凶铃》 |
| **循环型** | 恐怖事件在特定条件下重复发生 | 每年同一天同一条路上的车祸 |

### 4.3 都市传说的写作价值

都市传说的核心价值是**可信度**——它发生在你身边，发生在普通人身上。写恐怖时借用都市传说的结构，能让恐怖更"接地气"：
- 不要设定"古老城堡里的吸血鬼"，要设定"你楼下那户从来不开灯的人家"
- 不要设定"被诅咒的法器"，要设定"二手网站上买到的那个有点旧的布娃娃"

## 五、规则怪谈的规则设计原则

### 5.1 什么是规则怪谈

角色必须遵守一组特定的规则才能存活。规则本身暗示了恐怖的存在——通过"不能做什么"来让读者想象"违反规则会发生什么"。

### 5.2 好规则的设计

**规则必须暗示一个更恐怖的现实**：
> "如果有人在凌晨3点敲你的门，不要开门。不要从猫眼往外看。不要回应。第二天早上你会发现门口有一束花。把花收进屋里。不要闻它。"

——这条规则暗示了：有什么东西会在凌晨3点来，它能从猫眼看到你，它会留下花，花有某种危险。但规则没有解释"为什么"——这就是恐怖来源。

### 5.3 规则设计的五条原则

| 原则 | 说明 | 好的例子 | 坏的例子 |
|------|------|---------|---------|
| **规则暗示危险但不解释** | 规则告诉你怎么做，但不告诉你为什么 | "不要看电梯里的镜子超过3秒" | "因为镜子里有一个鬼" |
| **规则之间有逻辑但不完全** | 规则和规则之间可以拼凑出一部分真相，但拼不完整 | 规则1和规则3放在一起暗示了某种东西 | 规则之间完全随机，没有关联 |
| **至少有一条规则让你不舒服** | 有一条规则的遵守方式会让你做"不正常"的事 | "如果有陌生人在对你笑，你必须也对他笑，但不要露出牙齿" | 所有规则都只是"不要做某事" |
| **违反规则的后果留白** | 不直接描写违反规则会怎样——用前人的痕迹暗示 | "上一个住户留下的笔记本上，最后一条只写了一半" | "违反规则会被怪物吃掉" |
| **规则的来源可疑** | 这些规则是谁写的？他们怎么知道的？ | "房东给你的入住须知，但手写字迹越到后面越潦草" | 规则就是规则，没有来源 |

### 5.4 规则怪谈的结构

```
正常场景建立 → 获得规则（通过什么方式得到的？）→ 角色遵守规则 → 某个规则变得难以遵守 → 角色面临选择（遵守还是违反？）→ 后果
```

## 六、恐怖的时间设定

### 6.1 深夜（凌晨0-3点）

**为什么恐怖**：
- 人类本能地在这个时间段降低警觉（本应是深度睡眠）
- 环境噪音最低——任何声音都被放大
- 社会活动停止——找不到人帮忙
- 身体的褪黑素分泌高峰——反应变慢

**写作利用**：
- 凌晨2点醒来，发现本应安静的公寓里有声音
- 凌晨1点走在空无一人的街上
- 凌晨3点，所有电子设备的时间同时变了

### 6.2 凌晨3点（"恶魔时间"/"死亡时间"）

**文化背景**：
- 西方传统认为凌晨3点是"反基督时间"——耶稣在下午3点去世，凌晨3点是其颠倒
- 民间传说认为凌晨3-4点是"阴阳交界"最薄弱的时刻
- 人类在这个时间点的体温最低、警觉性最低

**写作利用**：
- 每天凌晨3:03被同一个声音吵醒
- 闹钟显示3:00但窗外完全不像3点——天色不对

### 6.3 黄昏（"逢魔时刻"）

**文化背景**：
- 日本的"黄昏"（たそがれ）原意为"谁在那里"——因为天色暗到看不清人，要问"谁"
- 光线变化导致视觉不确定性——东西在阴影中可以是任何形状
- 白天和黑夜的过渡——世界在"切换模式"

**写作利用**：
- 黄昏时走一条熟悉的路，发现路变长了
- 夕阳把影子拉得很长——但你的影子旁边多了一个

### 6.4 节日/纪念日

| 时间 | 恐怖来源 | 写作利用 |
|------|---------|---------|
| **除夕/春节** | 团聚vs孤独的反差；驱邪仪式本身就暗示"有东西要驱" | 独自在空荡荡的城市过年；鞭炮声掩盖了另一个声音 |
| **清明节** | 和死者直接相关的节日 | 去扫墓但坟前的花是新鲜的——谁来过？ |
| **七月半** | 鬼门开 | 那个月的所有异常——没有反常才是反常 |
| **万圣节** | 化妆/面具/身份模糊 | 来敲门的"讨糖小孩"摘下面具后——不是小孩 |
| **生日** | 年龄+时间流逝+孤独 | 没有人记得你的生日/有一个你没邀请的人来了 |
| **忌日** | 和死者的时间联系 | 去年今日——同样的事情在重演 |

### 6.5 时间异常的写法

| 异常类型 | 写法 | 恐怖效果 |
|---------|------|---------|
| 时间循环 | 同一个早晨反复发生，每次有微小差异 | 第三次循环时发现有人也在循环 |
| 时间跳跃 | 角色"失去"了一段时间 | 两小时前还在下午，现在天黑了 |
| 时间倒流 | 时钟在倒转 | 但只有角色注意到 |
| 时间停滞 | 天一直不亮 | 过了"应该天亮的时间"很久，还是黑夜 |
| 时间膨胀 | 几秒钟感觉像几小时 | 恐惧状态下的主观时间拉伸 |

## 七、恐怖环境速查

| 环境 | 恐怖要素 | 关键细节 |
|------|---------|---------|
| 停车场地下层 | 混凝土回声、日光灯闪烁、空旷 | 车灯照亮的范围之外全是黑的 |
