# 恐怖常识

> 写恐怖不需要你真的见过鬼，但你需要知道在哪些场景下人类最容易害怕、什么规则能让恐惧可信。

---

## 一、封闭空间设定

### 医院/精神病院
- 白色走廊：荧光灯嗡嗡声、消毒水气味、回声
- 手术室：无影灯下完全暴露、麻药的甜味、意识清醒但无法动弹
- 地下太平间：冷柜抽屉的拉合声、温度骤降

**特别注意**：精神病院恐怖的核心张力是"我到底是病人还是正常人"——如果你的角色被关进去，读者会自动质疑他的认知可靠性。

### 学校
- 放学后的教室：桌椅排列整齐但有一张歪了、黑板上有没擦干净的字
- 老旧厕所：最后一隔的门锁着但从外面看不到脚

**特别注意**：学校是儿童的"领地"——当这个地方变得不安全，读者感受到的是"连保护孩子的地方都不安全了"。

### 公寓/住宅
- 独居：钥匙转了两圈但门没锁/有人用过你的牙刷
- 老旧公寓：隔壁的笑声凌晨三点传过来/楼上没人住但有拖拽声
- 镜子/窗户：镜中的倒影延迟了一秒/窗外的人在看你但楼层很高

**特别注意**：家是最后的安全堡垒。当家变得恐怖，角色无处可逃。

### 电梯
密闭（2平方米）+ 失控（不知道带到哪一层）+ 强制共处 + 下坠恐惧。
**操作要点**：电梯恐怖的关键是"数字"——楼层显示的数字不对，或者数字在减少但电梯没有动。

### 森林
- 迷路：太阳的位置和指南针不一致
- 声音：突然所有鸟都不叫了
- 天黑：从"还能看清"到"完全看不见"只用了十分钟

## 二、超自然规则设计

### 鬼怪的限制条件

**好的超自然存在都有规则和限制——无限制的存在不可怕，因为它"什么都行"，读者没法思考，也没法和角色一起想办法。**

| 限制类型 | 示例 | 叙事价值 |
|---------|------|---------|
| 时间限制 | 只在凌晨3-4点出现 | 角色有"安全时间"，增加倒计时感 |
| 空间限制 | 只在这栋房子里 | 安全区可能逐渐缩小 |
| 触发条件 | 必须被看到才能行动 | 角色可以通过行为控制危险 |
| 传播规则 | 只能通过特定方式传播 | 角色必须阻止传播 |

### 诅咒触发条件

| 诅咒类型 | 触发条件 | 解除条件 |
|---------|---------|---------|
| 行为诅咒 | 做了某件事（打开禁忌之门） | 找到触发条件的逆向操作 |
| 位置诅咒 | 进入某个地方 | 满足怨灵的需求 |
| 传播诅咒 | 接触了某个物品/信息 | 找到源头 |
| 遗传诅咒 | 出生在这个家族 | 几乎无法解除 |
| 时间诅咒 | 在特定时间做了特定的事 | 在时间到期前完成某个条件 |

### 规则设计四原则

1. **规则必须简单**：一到三条就够了。太多读者记不住，你也会自相矛盾。
2. **规则必须一致**：一旦建立，不能随意打破。打破规则=打破读者信任。
3. **规则本身就是恐怖的一部分**："只要不看它的眼睛它就不能动"——角色在接下来的每一秒都和这个念头搏斗。
4. **给角色一线希望**：纯粹的绝望会让读者放弃。给一个解除条件，即使极其困难。

## 三、民俗恐怖素材

### 中国民俗

| 素材 | 恐怖核心 | 写作利用 |
|------|---------|---------|
| 冥婚 | 给死者配婚——活人参与死者的婚礼 | 被选为冥婚对象的恐惧 |
| 养小鬼 | 用死婴的灵魂作为"工具" | 小鬼失控反噬 |
| 湘西赶尸 | 让尸体自己"走"回家乡 | 尸体在夜间列队行走的画面 |
| 头七回魂 | 死后第七天灵魂回家 | 为死者留门，但回来的可能不只有死者 |
| 扎纸人 | 给死者烧的纸扎物品 | 纸人在另一边"活了"——但在某些情况下在这一边也活了 |

### 日本民俗

| 素材 | 恐怖核心 | 写作利用 |
|------|---------|---------|
| 怨灵 | 死于非命的灵魂持续复仇 | 无差别复仇——和死因无关的人也会被波及 |
| 付丧神 | 使用超过百年的物品获得灵魂 | 你家的旧物件有意识 |
| 裂口女 | 戴口罩的女人问"我漂亮吗" | 回答错误就死——但正确答案有时也是错的 |
| 人形（人偶） | 承载了主人情感的物品 | 烧掉人偶时的尖叫/人偶在转头 |

### 欧美民俗

| 素材 | 恐怖核心 | 写作利用 |
|------|---------|---------|
| 女巫/猎巫 | 社区集体疯狂——每个人都是嫌疑人 | 最安全的地方变成最危险的地方 |
| 吸血鬼 | 不死的存在——你需要永远逃离 | 逃到哪里都没用 |
| 温迪戈 | 冬季食人——人变成温迪戈 | 最恐怖的不是怪物，是你也可能会变成它 |

## 四、都市传说结构

### 标准结构
```
可信的开头 → 奇怪的中间 → 恐怖的结尾 → 警示性收尾
```

**写作价值**：都市传说的核心价值是**可信度**——不要设定"古老城堡里的吸血鬼"，要设定"你楼下那户从来不开灯的人家"。

### 经典模板

| 模板 | 结构 |
|------|------|
| 保姆型 | 保姆接到恐怖电话→电话来自屋内 |
| 消失型 | 某人在旅途中消失→留下的线索暗示"不是自愿离开" |
| 替代型 | 你注意到身边的人"变了"→他/她不是原来那个人 |
| 传染型 | 看了某个东西/知道某个信息就会被"感染" |
| 循环型 | 恐怖事件在特定条件下重复发生 |

## 五、规则怪谈设计

角色必须遵守一组特定的规则才能存活。规则本身暗示了恐怖的存在——通过"不能做什么"来让读者想象"违反规则会发生什么"。

**好的规则**：
> "如果有人在凌晨3点敲你的门，不要开门。不要从猫眼往外看。第二天早上你会发现门口有一束花。把花收进屋里。不要闻它。"

这条规则暗示了：有什么东西会在凌晨3点来，它能从猫眼看到你，它会留下花，花有某种危险。但规则没有解释"为什么"——这就是恐怖来源。

### 规则设计五原则

| 原则 | 好的例子 | 坏的例子 |
|------|---------|---------|
| 暗示危险但不解释 | "不要看电梯里的镜子超过3秒" | "因为镜子里有一个鬼" |
| 规则之间有逻辑但不完全 | 规则1和规则3暗示了某种东西 | 规则之间完全随机 |
| 至少有一条让不舒服 | "如果有陌生人在对你笑，你必须也对他笑，但不要露出牙齿" | 所有规则都只是"不要做某事" |
| 违反后果留白 | "上一个住户的笔记本上，最后一条只写了一半" | "违反规则会被怪物吃掉" |
| 规则来源可疑 | "房东给你的入住须知，但手写字迹越到后面越潦草" | 规则就是规则，没有来源 |

### 规则怪谈结构
```
正常场景建立 → 获得规则 → 角色遵守规则 → 某个规则变得难以遵守 → 角色面临选择 → 后果
```

## 六、恐怖的时间设定

### 深夜（凌晨0-3点）
人类本能降低警觉、环境噪音最低、社会活动停止、反应变慢。

### 凌晨3点（"恶魔时间"）
西方传统认为凌晨3点是"反基督时间"。人类在这个时间点体温最低、警觉性最低。
**写作利用**：每天凌晨3:03被同一个声音吵醒。闹钟显示3:00但窗外完全不像3点。

### 黄昏（"逢魔时刻"）
日本的"黄昏"原意为"谁在那里"——天色暗到看不清人。光线变化导致视觉不确定性。
**写作利用**：夕阳把影子拉得很长——但你的影子旁边多了一个。

### 时间异常写法

| 异常类型 | 写法 | 恐怖效果 |
|---------|------|---------|
| 时间循环 | 同一个早晨反复发生，每次有微小差异 | 第三次循环时发现有人也在循环 |
| 时间跳跃 | 两小时前还在下午，现在天黑了 | "我失去了这段时间" |
| 时间停滞 | 天一直不亮 | 过了"应该天亮的时间"很久，还是黑夜 |

## 七、恐怖环境速查

| 环境 | 恐怖要素 | 关键细节 |
|------|---------|---------|
| 停车场地下层 | 混凝土回声、日光灯闪烁 | 车灯照亮的范围之外全是黑的 |
| 教堂/寺庙 | 宗教符号的双面性 | 深夜的宗教建筑——神圣变成了诡异 |
| 隧道 | 两端都看不到出口 | 隧道比你以为的长/出口消失了 |
| 废弃游乐园 | 欢乐的残留物变得扭曲 | 旋转木马在没人的情况下自己在转 |
| 公共厕所 | 瓷砖回声、镜子、隔间 | 旁边的隔间有人——但那个隔间的门是从外面锁着的 |
| 地铁末班车 | 封闭+移动+乘客稀少 | 车厢那头坐着一个人，但每一站都没有人上下 |
