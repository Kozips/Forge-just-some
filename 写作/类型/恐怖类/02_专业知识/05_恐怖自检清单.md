# 恐怖自检清单

> 写完恐怖场景后逐条过。命中就改，没问题就跳。

---

## 一、恐惧来源

- [ ] 恐惧来源是否具体？（不是"她很害怕"，而是具体的感官异常/环境异变/行为异常）
- [ ] 恐惧是否有明确的触发点？（一个声音/一个画面/一个动作，而不是"突然觉得不对"）
- [ ] 恐惧是否递进？（不安→焦虑→恐惧→恐慌，不能上来就是最高级）
- [ ] 恐惧释放后是否有余震？（危险过去后，角色仍在发抖/不敢回头/反复确认）

---

## 二、感官描写

- [ ] 是否有3个以上感官描写？（视/听/嗅/触/味，至少用3种）
- [ ] 是否有"剥夺一个感官"的设计？（灯灭了看不见/耳朵嗡了听不见/麻痹了感觉不到）
- [ ] 感官描写是否具体？（不是"很冷"，而是"冷得像把手伸进冰柜里"）
- [ ] 是否有"不对劲"的感官细节？（正常场景中一个细节不对——温度/声音/气味/光线）

---

## 三、节奏控制

- [ ] 恐惧场景的句式是否匹配？（长句=慢恐惧/短句=快恐惧/无主句=突然袭击）
- [ ] 是否有"沉默"的使用？（最恐怖的不是声音，是声音突然消失）
- [ ] 段落长度是否有变化？（不能全是短段或全是长段，要交替）
- [ ] 高潮前是否有"安静段"？（暴风雨前的宁静比暴风雨本身更恐怖）

---

## 四、日常异化

- [ ] 日常场景是否只扭曲了1-2个细节？（扭曲太多就不恐怖了，变成奇幻）
- [ ] 异化是否符合逻辑？（不能一会儿是鬼一会儿是幻觉，规则要统一）
- [ ] 异化是否从"小"开始？（先是门响了一声，再是地板上有脚印，最后是身后有人）
- [ ] 日常物件是否被恐怖化？（镜子/楼梯/电梯/电话/窗户——越日常越恐怖）

---

## 五、留白与暗示

- [ ] 是否有"不描写怪物，描写怪物经过的痕迹"？（地上的水渍/门上的刮痕/空气中的腐味）
- [ ] 留白是否给够了线索？（读者能推断出大概是什么，但不确定细节）
- [ ] 是否避免了"原来是一场梦"的结局？
- [ ] 是否避免了"原来是精神病"的解释？（除非整本书就是写精神病的）

---

## 六、角色反应

- [ ] 角色的恐惧反应是否符合其性格？（胆小的人和勇敢的人反应不同）
- [ ] 是否有"不合理行为"？（明知道有危险还往前走——需要给出合理动机）
- [ ] 角色是否有"冻结"反应？（不是所有人都会跑，有些人会愣在原地）
- [ ] 恐惧后是否有恢复过程？（不能吓完就没事了，要有PTSD般的余震）

---

## 七、禁忌

- [ ] 是否有"突然"出现超过3次？（用具体声音/画面替代）
- [ ] 是否有"仿佛/似乎/不禁"？（删掉，直接写后面的比喻或动作）
- [ ] 是否有"不是A，是B"句式？（全面禁用，改写）
- [ ] 是否有"她感到一阵恐惧"？（改成身体反应：冷汗/手抖/瞳孔放大/后退半步）
- [ ] 是否有"整个世界都安静了"？（用具体的安静——时钟不响了/风停了/心跳声很响）
