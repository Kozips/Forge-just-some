# 剧情递进技法 —— 规则怪谈

> 来源：剧情递进手法_规则怪谈与末世生存 + 剧情递进手法_五种类型全覆盖
> 四个维度：冲突升级 · 信息解密 · 情绪压强 · 格局扩展

---

## 两种典型递进套路

**套路A：规则叠加→规则矛盾→规则崩塌→重建规则**

前三层副本各有一套独立规则，玩家学会遵守。第四层开始出现"上一层规则在这里是错的"。最终揭示所有规则只是某个更高层存在的"游戏设定"，真正的规则从未被写下。

**套路B：表面规则→隐藏规则→反向规则→无规则区**

每条表面规则背后都有一条隐藏条件。玩家以为掌握了全部规则时，发现某些规则本身就是陷阱——遵守即死，违反反而存活。最终进入"无规则区"，一切旧逻辑失效。

---

## 具体例子：《全球诡异时代》递进结构

> 早期：规则明确写在纸上——"晚上十点后不要看镜子""听到敲门声不要开门"。遵守=安全。冲突来自记不住规则和执行失误。
>
> 中期：出现矛盾规则——A说"必须待在房间里"，B说"凌晨三点必须离开房间"。两条同时生效。冲突从遵守vs违反变成遵守A还是遵守B。
>
> 后期：揭示规则本身是某个存在的"意志投射"，规则不是保护你，是筛选你。整个规则区是养蛊场。

**递进拆解：**
- **冲突升级**：遵守规则的难度 → 规则之间的矛盾 → 规则本身的恶意
- **信息解密**：明规则 → 暗规则 → 规则背后的真相
- **情绪压强**：紧张（怕违反）→ 绝望（无法同时遵守）→ 恐惧（规则本身就是敌人）
- **格局扩展**：单个房间 → 整栋建筑 → 城市 → 全球规则区

## 递进节奏

核心驱动力：**规则博弈**。快节奏，3-5章一层。格局扩展路径：房间→建筑→城市→世界。

| 阶段 | 占比 | 关键动作 |
|------|------|---------|
| **锚定期**（前20%） | 建立规则/世界观 | 让读者以为"我懂了" |
| **递进期**（20%-60%） | 三层递进 | 每层打破一个旧认知，建立一个新认知 |
| **崩塌期**（60%-80%） | 核心反转 | 所有旧规则/认知被推翻，主角必须从零重建 |
| **终局期**（80%-100%） | 收束或开放 | 给出终极答案，或留下"真正的问题还没解决" |

## 写作检验清单

- [ ] 每一章是否比上一章的冲突强度更高？（不能原地踏步）
- [ ] 每次信息揭露是否同时制造了新悬念？（揭示≠结束）
- [ ] 情绪是否有波谷？（连续高压读者会麻木，需要喘息再加压）
- [ ] 格局扩展时主角的代价是否同步增加？（免费的升级=无聊）
- [ ] 读者在第N章是否还能回忆起第1章的悬念？（伏笔回收=信任）
- [ ] 递进是否有"不可逆"节点？（至少一个无法回头的转折点）
