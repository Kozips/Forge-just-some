# 恐怖类 -- 日常物件恐怖化速查表

> 正常物件在错误的场景/时间出现就变恐怖。万能技法，适用于所有恐怖场景。
> 来源：references/research/01-04/02_写作技法提炼.md 蒸馏。

## 物件恐怖化对照表

| 物件 | 正常 | 恐怖化 |
|------|------|-------|
| 镜子 | 照出自己的脸 | 照出的脸慢了半拍/不是自己 |
| 手机 | 有人打来 | 来电显示是自己的号码 |
| 秋千 | 风吹在动 | 没有风也在动 |
| 照片 | 记录瞬间 | 照片里多了一个/少了一个 |
| 水龙头 | 流水 | 流出来的不是水 |
| 闹钟 | 正常响 | 响了三天了这栋楼没人住 |
| 玩具 | 给孩子玩的 | 自己在动 |
| 窗帘 | 遮光 | 窗帘后面站着东西但没有脚 |
| 冰箱 | 保鲜食物 | 里面的东西全变成了红色 |
| 售货机 | 卖商品 | 停电三天还亮着，屏幕显示倒计时 |

## 直接抄的句子

> "冰箱门是开的。灯亮着。但冰箱里的东西——全变成了红色。不是坏了。是——从一开始就是红色的。但他记得昨天买的是绿色的蔬菜。"

> "她的手机收到了一条短信。发件人是她自己。内容是'别回头'。发送时间是——三小时前。三小时前她在睡觉。手机放在枕边。"

> "售货机还亮着灯。在停电三天的城区。他走过去。屏幕上显示的不是商品列表。是一个倒计时。"

## 创作原则

1. **日常物件恐怖化的关键是"错位"**——不是物件本身恐怖，是它出现在不该出现的场景
2. **越常见的物件越恐怖**——手机比古镜恐怖，因为每个人都有手机
3. **物件要承载"信息"**——照片里的多出来的人、短信里的警告、冰箱里的颜色变化——物件是恐怖信息的载体
