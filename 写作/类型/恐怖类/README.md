# 恐怖类 -- V2 Skill 目录

> 心理恐惧、氛围营造、未知威胁、Jump Scare与慢节奏压迫
> 覆盖：心理恐怖/宇宙恐怖/身体恐怖/民俗恐怖/生存恐怖/科技恐怖/规则怪谈/网络恐怖

## 目录结构

| 目录 | 用途 |
|------|------|
| SKILL.md | 主技能文件（速查引擎 + 50条速查 + 感官词库 + 角色模板 + 自测表） |
| references/ | 素材参考（对话库、剧本库、研究资料） |
| 01_核心规则/ | 8条写作铁律与不可违反的规则 |
| 03_素材库/ | 15经典场景 + 8角色对话模板 + 特殊对话技法 + 视觉描写词库 |
| 04_写作模板/ | 单章结构 + 三级场景链 + 多章弧线 + 结局类型 + 开篇钩子 |
| 05_技法研究/ | 12种写作技法 + 日常物件恐怖化速查 |
| 06_质检清单/ | 完稿前检查项（通用/结构/技法/角色/余韵） |

## 使用流程

1. **开写前** -- 读 SKILL.md 了解全局，读 01_核心规则/ 确认铁律
2. **构建阶段** -- 从 03_素材库/ 挑选场景和角色模板
3. **写作阶段** -- 按 04_写作模板/ 搭结构，参考 05_技法研究/ 提升手法
4. **完稿阶段** -- 用 06_质检清单/ 逐项检查
