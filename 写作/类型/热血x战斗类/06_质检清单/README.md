# 热血类 — 质检清单

> 完稿前逐项检查。打勾通过，未通过则修改。
> 整合自 SKILL.md 自测表、references/04 对话感官自测表、结构模板检查清单。

---

## A. 通用基础（5项）

- [ ] 无病句和错别字
- [ ] 人称一致（无视角混乱）
- [ ] 时间线连贯（无矛盾）
- [ ] 对话自然（无AI腔/书面腔）
- [ ] 每章有明确的情节推进

---

## B. 热血铁律（7项）

- [ ] 角色面对的是"不可能赢"的对手（不是碾压弱鸡）
- [ ] 成长通过失败实现（不是修炼三天就变强）
- [ ] 同伴羁绊是核心驱动力（不是个人英雄主义）
- [ ] 对手值得尊敬（不是单纯的反派工具人）
- [ ] 突破有铺垫（不能突然领悟终极奥义）
- [ ] 能力使用有代价（不能无限爆发）
- [ ] 没有假死（如果设定了有人会死，就真的让他死）

---

## C. 战斗结构（8项）

- [ ] 开头50字内建立了明确的"赢/输"赌注
- [ ] 中场有至少2次攻守反转（不能一路碾压）
- [ ] 突破的代价具体可见（不是"很累"，而是"右手已经握不住刀"）
- [ ] 角色"凭什么赢"有逻辑支撑（新技巧/新理解/代价换来的）
- [ ] 回忆杀控制在1-2句且由动作触发（不超过两段）
- [ ] 对手有一个让读者能理解的动机
- [ ] 结尾埋了下一步冲突的种子
- [ ] 全章至少一个"呼吸节拍"（非战斗描写）

---

## D. 对话质量（5项）

- [ ] 战斗前对话够简洁（没有废话和演说）
- [ ] 战斗中对话有"被打断""喘气"的感觉
- [ ] 战斗后对话有留白（不急于总结）
- [ ] 导师台词像聊天而不是念鸡汤
- [ ] 口号/宣言不超过全章对话的10%（用行动证明而非喊口号）

---

## E. 感官与节奏（6项）

- [ ] 力量感描写覆盖声音+视觉+触觉
- [ ] 疼痛感分了类型和层次（不是只写"痛"）
- [ ] 环境描写参与了战斗（地面/天气/建筑不是背景板）
- [ ] 愤怒/恐惧/决心的描写分了层次
- [ ] 用到了留白（对峙/战后/名场面前）
- [ ] 战斗节奏有快慢变化（不是匀速）

---

## F. 角色检查（4项）

- [ ] 团队中每个成员都有存在感（不是只有主角）
- [ ] 突破极限有足够的情感铺垫
- [ ] 牺牲者有"未完成的心愿"（不是白死的）
- [ ] 配角的战斗方式跟性格挂钩（冲动型打法大开大合，冷静型精准无多余动作）

---

## G. 禁忌扫描（8项）

- [ ] 无感叹号堆砌（"啊啊啊啊！！！"）
- [ ] 无打不赢就开挂（金手指有铺垫和代价）
- [ ] 无反派智商掉线来衬托主角
- [ ] 无跳过训练直接变强
- [ ] 无每场战斗都赢（至少每5场输1场）
- [ ] 无配角变成空气
- [ ] 无在热血场景中间插入搞笑打断节奏
- [ ] 无胜后立刻欢呼（应该喘息和沉默）

---

## 评分标准

| 通过项数 | 评级 | 建议 |
|----------|------|------|
| 40-43 | 优秀 | 可直接发布 |
| 35-39 | 良好 | 修补后发布 |
| 28-34 | 合格 | 需要修改，重点看未通过项 |
| <28 | 不合格 | 热血不是喊出来的，是写出来的。重写。 |
