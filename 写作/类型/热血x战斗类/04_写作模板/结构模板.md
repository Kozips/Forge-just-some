# 热血类 — 结构模板

> 本目录存放章节结构模板与叙事骨架。写作前用模板规划节奏。

## 单章结构（挑战-苦战-突破型）

```
### 挑战出现（前15%）
- 新的对手/障碍/任务出现
- 角色主动迎战（不退缩）

### 苦战（35%）
- 角色陷入苦战——对手比想象中强
- 展示角色的技巧和毅力
- 同伴/旁观者的反应（紧张/加油/担忧）

### 突破/转折（30%）
- 角色在极限中找到突破口
- 可能是新技巧、新理解、或纯粹的意志力

### 余波（20%）
- 胜利后不是狂欢——是喘息和反思
- 新的目标/更强的对手出现
```

## 多章弧线（新手-挫折-成长-超越型）

```
第1-3章：角色初出茅庐 → 第一次胜利（小胜）
第4-6章：遭遇强敌 → 大败 → 意识到差距
第7-8章：修炼/领悟 → 突破瓶颈
第9-10章：再战强敌 → 险胜 → 角色已不是当初的自己
```

## 核心张力公式

> **热血 = 不对等对抗 × 不屈意志**

每一章必须回答一个问题：「他凭什么赢？」——答案不能是"因为主角光环"，必须来自代价、技巧或领悟的其中之一。

## 三级场景链

| 层级 | 功能 | 节奏 | 示例 |
|------|------|------|------|
| **开场场（Opening）** | 锁定冲突，建立赌注 | 快，1-2段入局 | 对手亮出底牌 / 赌注被加码 |
| **中场（Mid）** | 拉扯，消耗，展示苦战 | 慢-渐快交替 | 攻防节奏变换 / 旁观者视角插入 |
| **翻转场（Turn）** | 打破僵局的决定性瞬间 | 突然加速→凝固 | 意志觉醒 / 新理解 / 代价性爆发 |

> 写作技巧：中场至少安排 2 次"小翻转"（一度占优→再被压制），避免直线叙事。

## 章节节奏曲线

```
情绪强度
  ▲
  │        ╱╲      ╱╱╱（高潮突破）
  │      ╱    ╲  ╱╱
  │    ╱   拉扯 ╲╱  ← 小翻转/低谷
  │  ╱ 开局冲劲
  │╱
  └───────────────────────────→ 场景推进
          前15%  中场  翻转  余波20%
```

- **前15%**：开局冲劲，快速建立"赢/输"的赌注
- **15%-50%**：拉扯区，层层加压，制造"可能赢不了"的错觉
- **50%-80%**：翻转突破区，主角以代价换转折
- **80%-100%**：余波，喘息+埋下新冲突种子

## 角色压力点（至少覆盖3个）

| 压力类型 | 写法 |
|----------|------|
| **体力极限** | 具体化：手在发抖 / 视线模糊 / 旧伤复发 |
| **心理动摇** | 内心独白闪过"我做不到"——然后压制它 |
| **信任危机** | 同伴质疑 / 观众倒戈 / 队友受伤因"我"而起 |
| **代价预兆** | 突破前必须交代"用了会怎样"——让读者紧张 |
| **对手尊重** | 对手不是纯恶，展现其实力让胜利更有分量 |

## 常见失败模式（写热血章的坑）

| 失败模式 | 症状 | 修正 |
|----------|------|------|
| **无代价胜利** | 赢得太轻松，读者没感觉 | 每次突破必须失去点什么（体力/信念/关系） |
| **解说式回忆杀** | 战斗中插入3段回忆 | 回忆必须是闪回式（1-2句），且由战斗动作触发 |
| **对手纸片化** | 反派只有"邪恶"一个标签 | 给对手一个让读者能理解的动机 |
| **意志=万能** | "我绝不放弃"解决一切 | 意志只能撑住不倒，突破靠具体的新理解/技巧 |
| **节奏平直** | 从头打到尾没有呼吸 | 每打300字插一个非战斗节拍（旁观者反应/环境/内心） |

## 结尾类型

| 类型 | 情绪基调 | 适用场景 |
|------|----------|----------|
| **险胜余韵** | 疲惫中的满足 | 单次对决，角色证明了自己 |
| **惨胜代价** | 赢了但失去了重要之物 | 提升故事重量，转折章 |
| **败中藏胜** | 输了但突破了某个心障 | 成长弧中段，为后续蓄力 |
| **新敌登场** | 短暂喘息被打断 | 章末钩子，推动读者翻页 |

## 写作检查清单

- [ ] 开头50字内是否建立了明确的"赢/输"赌注？
- [ ] 中场是否有至少2次攻守反转（不能一路碾压）？
- [ ] 突破的代价是否具体可见（不是"很累"，而是"右手已经握不住刀"）？
- [ ] 角色的"凭什么赢"是否有逻辑支撑（新技巧/新理解/代价换来的）？
- [ ] 回忆杀是否控制在1-2句且由动作触发？
- [ ] 对手是否有一个让读者能理解的动机？
- [ ] 结尾是否埋了下一步冲突的种子？
- [ ] 全章是否有至少一个"呼吸节拍"（非战斗描写）？

## 开篇钩子模板

**类型A — 站在绝境中**：他面对一百个人。手里只有一把断刀。他说："来吧。"
**类型B — 从失败开始**：他输了。在所有人面前。但第二天，他又站在了擂台上。
**类型C — 赌注前置**：赢了，她就能走。输了，他得留下。他说："赌。"
