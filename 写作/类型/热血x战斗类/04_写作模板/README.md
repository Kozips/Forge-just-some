# 热血类 — 写作模板

> 章节结构模板与叙事骨架。写作前用模板规划节奏。

## 本目录内容

| 文件 | 用途 |
|------|------|
| 结构模板.md | 单章结构 + 多章弧线 + 三级场景链 + 节奏曲线 + 失败模式 + 结尾类型 + 开篇钩子 |
| 本文档 | 目录索引与使用说明 |

## 快速上手

1. **写单章前** → 打开 结构模板.md 的"单章结构"，按比例分配篇幅
2. **规划多章弧线** → 参考"多章弧线"模板安排起承转合
3. **检查节奏** → 用"章节节奏曲线"确认情绪强度有起伏
4. **避开陷阱** → 参照"常见失败模式"逐项自查
5. **选结尾类型** → 根据本章在弧线中的位置选择合适的结尾

## 核心节奏公式

> **热血 = 不对等对抗 x 不屈意志**

每一章必须回答一个问题：「他凭什么赢？」——答案不能是"因为主角光环"，必须来自代价、技巧或领悟的其中之一。

## 五幕长弧结构（整本书级）

| 幕 | 占比 | 内容 | 节奏 |
|----|------|------|------|
| 第一幕：训练/准备 | 前20% | 目标+训练+挫折 | 稳，不急着战斗 |
| 第二幕：挫折/低谷 | 20-40% | 第一次大败/受伤/自我怀疑 | 低，2-3章内爬出 |
| 第三幕：突破/觉醒 | 40-65% | 领悟新东西+小战斗验证 | 渐升 |
| 第四幕：决战 | 65-90% | 全书最重要部分（详见结构模板） | 波浪式高-低-最高 |
| 第五幕：升华/告别 | 90-100% | 心态变了+最后对话+余韵 | 降落但留余味 |
