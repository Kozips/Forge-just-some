## 一、对话素材库

### 战斗前对话

**无声型**
> 两个人站在对面。谁都没说话。风吹过去了。他把手放在了剑柄上。对面那个人也动了。

**简短确认型**
> "准备好了？""嗯。"没有别的话了。

**最后交代型**
> "我要是回不来——""你会回来的。""......你让我说完。""不用说完。你会回来的。"

### 战斗中对话

**垃圾话**（不是侮辱，是互相挑衅）
> "就这？你练了多久？"
> "你那一拳打偏了。"

**喘气型**
> "你......变强了。""你......也是。"（两个人都笑了。然后继续打。）

**被打断型**
> "你——"（被打飞）"......够狠的。"（爬起来）"但我——"（又被打倒）"......还没——"

**沉默型**
> 每一招就是对话——他用了一招新的=说"我进步了"，对手换打法=说"我认可你了"。

**临终型**
> "你知道吗......"血从他嘴角流下来。"我一直想跟你说......""别说。""......我从来没后悔跟你打这一场。"

### 战斗后对话

**感谢型**（不用"谢谢"，太正式）
> "你打的那一拳真疼。""你踢的那一脚也不轻。""......下次还来？""嗯。"

**告别型**
> "你要走了？""嗯。""......注意安全。""你也是。"（走了大概一百步停了一下，然后继续走。）

**无言型**
> 走到同伴身边坐下，递一瓶水，看着远处，什么都没说。但什么都说了。

### 训练中对话

> "你这一刀砍下去连树皮都破不了。""你来示范啊。""看好了。"（一刀劈开）"看到区别了吗？""......你的刀比较贵。"

### 导师台词

> "你这一刀，劈柴都嫌歪。""......师父，我在练杀招。""杀招？你先学会劈柴。柴都劈不齐，你劈什么人？"

**导师语言特征**：
- 不讲大道理，用具体的事打比方
- 说的话听起来像在损人，但意思是好的
- 偶尔说出一句很深刻的话，自己可能都没意识到
- 教的最多的不是技术，是态度

### 经典台词模式

**热血主角**：
- "我不知道能不能赢。但我不能不去。"
- "你先走。"（已做好回不来的准备）
- "......还没结束。"（所有人以为他已经输了）

**对手之间**：
- "你很强。可惜我们是敌人。"
- "正因如此，我才要全力以赴。"

## 二、感官描写速查

### 力量感

**不要写**："他出了一拳，力量很大。"

**要写**：他的拳头穿过了空气——声音像撕布。然后是接触。对手的肋骨发出了一声闷响。对手的身体弯了。脚下的地面碎了一圈。

**要素清单**：声音（碰撞/撕裂/骨头响）+ 视觉（冲击波/碎裂地面）+ 触觉（震动传到手臂）+ 后果（被打飞/弯/入地）

### 速度感

**不要写**："他的速度很快。"

**要写**：他动了。对手的瞳孔收缩了。但身体来不及反应。一道残影。然后对手发现他已经站在自己身后了。剑已经出了鞘。

**要素清单**：短句（一个动作一句话）+ 对手反应来不及 + 残影/风声/地面脚印

### 疼痛感（5种类型）

| 类型 | 描写 |
|------|------|
| 钝痛 | 被打中腹部，像被巨石压着，喘不上气 |
| 刺痛 | 先是凉的（刀太快），然后才是热的（血在流） |
| 撕裂痛 | 一动就疼，不动也隐隐作痛 |
| 灼烧痛 | 不是被火烧，是骨头里着火 |
| 麻痹 | 没知觉了——没知觉比疼更可怕 |

### 疼痛层次（4层）

1. 感到了疼（表面）
2. 在忍（心理）
3. 忽视了疼继续打（意志）
4. 身体撑不住了（极限）

### 环境参与战斗

| 地面 | 效果 |
|------|------|
| 泥地 | 脚底打滑，泥浆摔跤 |
| 岩石 | 碎石飞溅 |
| 水面 | 水花/阻力/看不清脚下 |
| 冰面 | 站不稳，碎冰可当武器 |

| 天气 | 效果 |
|------|------|
| 雨 | 视线模糊/手滑/泥泞 |
| 雷 | 闪电照亮战场一瞬间又暗 |
| 风 | 影响远程/风沙迷眼 |
| 雪 | 冷/脚陷进去/血在白雪上刺眼 |

| 地形 | 效果 |
|------|------|
| 废墟 | 随时会塌的墙壁/碎玻璃/弹坑 |
| 城墙 | 高处对低处的优势/劣势 |
| 森林 | 视野受限/树木可被利用 |
| 地下 | 光源有限/空间狭小/回声影响判断 |

## 三、情绪描写分层

### 愤怒

| 层次 | 表现 |
|------|------|
| 表面（控制中） | 声音变低 / 咬牙 / 指甲嵌进肉里 |
| 爆发（失控） | 理智断 / 招式暴力不计后果 |
| 冷却（之后的空虚） | 手在抖 / 低头看手上有血 |

### 恐惧

| 层次 | 表现 |
|------|------|
| 身体反应 | 膝盖抖 / 手心出汗 / 想吐 |
| 极点反应 | 反而安静 / 极度专注 / 可能笑 |

**恐惧的身体反应写法**：
- 膝盖在抖——他用意志力让它不抖
- 手心出汗——握不住武器
- 想吐——胃在翻涌
- 眼睛不自觉地寻找退路——然后强迫自己不去看

### 决心

| 层次 | 表现 |
|------|------|
| 表现 | 不看退路 / 呼吸变稳 / 话变少 |
| 之后 | 眼神让人觉得"已做好死的准备" |

**写决心的误区**：不要让角色说出来"我下定决心了"——太直白。
用行为暗示：他把一个东西收好了（遗物？）、他回头看了一眼（最后一眼？）、他深吸了一口气（然后冲了上去）。
