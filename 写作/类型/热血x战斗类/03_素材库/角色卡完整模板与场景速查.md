## 一、角色卡完整模板（8类）

每张卡包含：外貌x3 + 战斗风格x1 + 性格x3 + 驱动力x1 + 弱点x1 + 搭档x1

### 热血主角

**A. 笨鸟先飞型**
- 外貌：手上全是老茧 / 身上有很多旧伤疤 / 个子不高但肩膀很宽
- 战斗：硬打，被打倒十次站起来十一次
- 性格：死心眼 / 不服输 / 对朋友掏心掏肺
- 驱动力：有一个很重要的人在等他变强
- 弱点：太在意别人，为了救一个人可能把自己搭进去
- 搭档：冷静军师

**B. 背负过去型**
- 外貌：左眼有伤疤 / 不怎么笑 / 穿着朴素
- 战斗：前期压制、后期爆发，忍到最后才出全力
- 性格：沉默 / 克制 / 偶尔温柔
- 驱动力：过去犯了一个错，在赎罪
- 弱点：觉得自己"不配赢"，关键时刻可能手下留情
- 搭档：搞笑担当

**C. 天赋不足型**
- 外貌：瘦 / 看起来不像战士 / 但眼神很亮
- 战斗：不靠力量，靠脑子和出其不意
- 性格：爱观察 / 爱琢磨 / 一根筋
- 驱动力：所有人都说他不行，要用结果打脸
- 弱点：太想证明自己，容易急躁犯错
- 搭档：热血导师

### 天才对手

**A. 天生强者**
- 外貌：五官完美 / 站姿随意 / 武器看起来都不需要保养
- 战斗：轻松随意，随手一招=别人全力
- 性格：骄傲但不恶毒 / 有点无聊感 / 对弱者不屑但不侮辱
- 驱动力：想找一个对手，太强了所有人不够打
- 弱点：没体验过失败，一旦输了不知道怎么处理
- 与主角关系：一开始不在意，后来被"韧性"吸引

**B. 天才走偏了**
- 外貌：黑眼圈深 / 嘴角有咬痕 / 衣服不修边幅
- 战斗：暴力、不计后果、有时连自己都伤
- 性格：偏执 / 聪明 / 内心有很深的恐惧
- 驱动力：变强不被抛弃
- 弱点：心态不稳定，事情没按预想走会崩
- 与主角关系：有相似的过去，走了不同的路

### 热血导师

**A. 退休老兵**
- 外貌：身上的伤比主角见过的所有人加起来还多 / 一只耳朵听不太清 / 喜欢喝茶
- 战斗：已经不打了，但站着就让人安心
- 性格：话不多 / 爱笑 / 笑的时候皱纹很深
- 教学：不讲大道理，直接扔进实战里

**B. 年轻引路人**
- 外貌：比主角大不了几岁 / 装备精良 / 气质成熟
- 战斗：精准高效，没有多余动作
- 性格：表面冷淡 / 实际上在意后辈 / 表达关心的方式是严厉
- 教学：先放任犯错，再告诉你错在哪里

### 冷静军师

**A. 书生型**
- 外貌：瘦 / 戴眼镜 / 手指上有墨渍
- 战斗：布阵设陷阱指挥，不正面打
- 性格：冷静 / 说话刻薄 / 关键时刻比谁都稳
- 弱点：太依赖计划，计划被打乱会慌

**B. 老练副官**
- 外貌：壮实 / 面相凶 / 手上有搬货老茧
- 战斗：防守为主，团队之盾
- 性格：稳重 / 不表达情绪 / 关心人的方式是直接把饭端到你面前
- 弱点：太忠诚，主人做错决定可能不会反对

### 搞笑担当

**A. 嘴炮型**
- 外貌：普通 / 表情丰富 / 嘴永远在动
- 战斗：远程（弓箭/暗器），不近战
- 性格：话多 / 爱吐槽 / 关键时刻突然安静让人担心
- 隐藏面：其实很胆小，害怕就笑

**B. 身体搞笑型**
- 外貌：体型夸张（太胖/太瘦/太高）/ 走路都会绊倒
- 战斗：力量型，看起来笨拙但重击
- 性格：憨厚 / 单纯 / 被骗过一次就不会有第二次
- 隐藏面：脑子转得慢

### 隐藏强者

**A. 闲人型**
- 外貌：看不清脸（帽子/刘海/面具）/ 存在感极低
- 战斗：平时不打，出手即结束
- 揭示：被迫的，非主动展示

**B. 伪装型**
- 外貌：看起来弱小不起眼 / 说话声音小
- 战斗：装弱，绝境才露真面目
- 揭示：绝境中保护他人时

## 二、场景素材速查（10个高频场景）

### 场景1：入门/入队

**核心画面**：主角第一次加入团队

**细节清单**：
- [ ] 主角的装备跟其他人不在一个档次
- [ ] 老成员的眼神——有人好奇、有人轻视、有人无所谓
- [ ] 第一次任务分配：主角被分到了最差的位置

### 场景2：训练场

**核心画面**：日常训练中的冲突和进步

**细节清单**：
- [ ] 环境——泥地/沙地/水里/悬崖上
- [ ] 身体感受——肌肉酸痛、手掌水泡、摔倒在泥巴里
- [ ] 有人比他练得轻松——这个人的存在就是他最大的刺激

### 场景3：第一次实战

**核心画面**：训练到战场的落差

**细节清单**：
- [ ] 对手跟假人完全不同——会动、会反击、会流血
- [ ] 实战中犯了训练时不会犯的错
- [ ] 实战后的反思——"训练了那么久，就这？"

### 场景4：挫败/失败

**核心画面**：被碾压式击败

**细节清单**：
- [ ] 输的过程干净利落——连对手衣服都没碰到
- [ ] 反应层次：不信→愤怒→沉默
- [ ] 周围人反应——安慰/嘲笑/不说话但眼神变了

### 场景5：突破/悟道

**核心画面**：某个特殊时刻，碎片拼在一起

**细节清单**：
- [ ] 触发器要具体——一句话/一个画面/一个动作
- [ ] 不是突然的，是之前所有碎片在这一刻拼在一起
- [ ] 悟道之后第一个想告诉的人是谁？

### 场景6：宿命对决

**核心画面**：跟宿敌的正面碰撞

**细节清单**：
- [ ] 战前安静——谁都没有先动
- [ ] 第一击——双方都在试探
- [ ] 中盘——越打越快，环境在崩塌
- [ ] 转折——被困的人想到了什么
- [ ] 终结——最后一击简短有力

### 场景7：同伴的战斗

**核心画面**：不只主角在打

**细节清单**：
- [ ] 给每个同伴1-2个场景
- [ ] 战斗风格跟性格匹配
- [ ] 有人赢了有人输了，不是所有人都能全身而退

### 场景8：休息/营火

**核心画面**：战斗之间的安静时刻

**细节清单**：
- [ ] 有人包扎伤口、有人吃东西、有人发呆
- [ ] 对话不需要深入——"明天打谁？""还在想。""嗯。"
- [ ] 这是写角色关系最好的时机

### 场景9：牺牲

**核心画面**：有人付出了代价

**细节清单**：
- [ ] 牺牲前给他一个"最后的温柔"——最后一句话/最后一个动作
- [ ] 主角反应不只有愤怒——还有不敢相信、自责、后悔
- [ ] 其他人反应——沉默/哭了/握紧武器

### 场景10：最终决战前的集结

**核心画面**：所有人准备最后一战

**细节清单**：
- [ ] 每个人带了什么？有些人换了武器，有些人带了纪念品
- [ ] 有人说了句不太合适的话，全场笑了——最后一次轻松的笑
- [ ] 出发的画面——背影。一个接一个。最后一个是主角

## 三、配角战斗风格与性格匹配

| 性格类型 | 战斗风格 | 弱点 | 最佳搭配 |
|----------|----------|------|----------|
| 冲动型 | 大开大合，威力大但破绽多 | 需要别人补漏洞 | 冷静军师 |
| 冷静型 | 精准无多余动作 | 不按套路出牌的不适应 | 热血主角 |
| 搞笑型 | 看起来滑稽但意外有效 | 脑子转得慢 | 热血主角 |
| 天才型 | 华丽精准让人嫉妒 | 太依赖天赋，不习惯苦战 | 背负过去型主角 |
| 防御型 | 盾，为主角挡伤害 | 太忠诚，不会反对错误决定 | 任何热血主角 |
| 辅助型 | 远程支援/布阵 | 近战能力差 | 冲动型队友 |
