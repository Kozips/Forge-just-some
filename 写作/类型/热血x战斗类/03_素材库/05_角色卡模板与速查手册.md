# 热血类 -- 角色卡完整模板与场景速查

> 每张角色卡可直接套用到写作中，场景素材可按需组合。

---

## 一、角色卡完整模板（8类）

每张卡包含：外貌x3 + 战斗风格x1 + 性格x3 + 驱动力x1 + 弱点x1 + 搭档x1

### 热血主角

**A. 笨鸟先飞型**
- 外貌：手上全是老茧 / 身上有很多旧伤疤 / 个子不高但肩膀很宽
- 战斗：硬打，被打倒十次站起来十一次
- 性格：死心眼 / 不服输 / 对朋友掏心掏肺
- 驱动力：有一个很重要的人在等他变强
- 弱点：太在意别人，为了救一个人可能把自己搭进去
- 搭档：冷静军师

**B. 背负过去型**
- 外貌：左眼有伤疤 / 不怎么笑 / 穿着朴素
- 战斗：前期压制、后期爆发，忍到最后才出全力
- 性格：沉默 / 克制 / 偶尔温柔
- 驱动力：过去犯了一个错，在赎罪
- 弱点：觉得自己"不配赢"，关键时刻可能手下留情
- 搭档：搞笑担当

**C. 天赋不足型**
- 外貌：瘦 / 看起来不像战士 / 但眼神很亮
- 战斗：不靠力量，靠脑子和出其不意
- 性格：爱观察 / 爱琢磨 / 一根筋
- 驱动力：所有人都说他不行，要用结果打脸
- 弱点：太想证明自己，容易急躁犯错
- 搭档：热血导师

### 天才对手

**A. 天生强者**
- 外貌：五官完美 / 站姿随意 / 武器看起来都不需要保养
- 战斗：轻松随意，随手一招=别人全力
- 性格：骄傲但不恶毒 / 有点无聊感 / 对弱者不屑但不侮辱
- 弱点：没体验过失败，一旦输了不知道怎么处理

**B. 天才走偏了**
- 外貌：黑眼圈深 / 嘴角有咬痕 / 衣服不修边幅
- 战斗：暴力、不计后果、有时连自己都伤
- 驱动力：变强不被抛弃
- 弱点：心态不稳定，事情没按预想走会崩

### 热血导师

**A. 退休老兵**
- 外貌：身上的伤比主角见过的所有人加起来还多 / 一只耳朵听不太清 / 喜欢喝茶
- 战斗：已经不打了，但站着就让人安心
- 教学：不讲大道理，直接扔进实战里

**B. 年轻引路人**
- 战斗：精准高效，没有多余动作
- 性格：表面冷淡 / 表达关心的方式是严厉
- 教学：先放任犯错，再告诉你错在哪里

### 冷静军师

**A. 书生型**
- 外貌：瘦 / 戴眼镜 / 手指上有墨渍
- 战斗：布阵设陷阱指挥，不正面打
- 弱点：太依赖计划，计划被打乱会慌

**B. 老练副官**
- 外貌：壮实 / 面相凶 / 手上有搬货老茧
- 战斗：防守为主，团队之盾
- 性格：关心人的方式是直接把饭端到你面前
- 弱点：太忠诚，主人做错决定可能不会反对

### 搞笑担当

**A. 嘴炮型**
- 战斗：远程（弓箭/暗器），不近战
- 性格：话多 / 爱吐槽 / 关键时刻突然安静让人担心
- 隐藏面：其实很胆小，害怕就笑

**B. 身体搞笑型**
- 外貌：体型夸张 / 走路都会绊倒
- 战斗：力量型，看起来笨拙但重击
- 隐藏面：脑子转得慢

### 隐藏强者

**A. 闲人型** — 存在感极低，平时不打，出手即结束。揭示：被迫的，非主动展示

**B. 伪装型** — 看起来弱小不起眼，装弱，绝境才露真面目。揭示：保护他人时

## 二、场景素材速查（10个高频场景）

| 场景 | 核心画面 | 关键细节 |
|------|---------|---------|
| 入门/入队 | 主角第一次加入团队 | 装备档次差距 / 老成员眼神 / 被分到最差位置 |
| 训练场 | 日常训练中的冲突和进步 | 身体感受要具体 / 有人练得轻松=最大刺激 |
| 第一次实战 | 训练到战场的落差 | 对手会动会流血 / 犯训练时不会犯的错 |
| 挫败/失败 | 被碾压式击败 | 输得干净利落 / 反应层次：不信→愤怒→沉默 |
| 突破/悟道 | 碎片拼在一起 | 触发器要具体 / 不是突然的，是积累 |
| 宿命对决 | 跟宿敌正面碰撞 | 战前安静→试探→越打越快→转折→终结 |
| 同伴的战斗 | 不只主角在打 | 战斗风格跟性格匹配 / 有人赢有人输 |
| 休息/营火 | 战斗之间的安静 | "明天打谁？""还在想。""嗯。" |
| 牺牲 | 有人付出代价 | 牺牲前的"最后的温柔" / 主角不只有愤怒 |
| 最终集结 | 准备最后一战 | 背影。一个接一个。最后一个是主角 |

## 三、配角战斗风格与性格匹配

| 性格类型 | 战斗风格 | 弱点 | 最佳搭配 |
|----------|----------|------|----------|
| 冲动型 | 大开大合，威力大但破绽多 | 需要别人补漏洞 | 冷静军师 |

| 防御型 | 盾，为主角挡伤害 | 太忠诚，不会反对错误决定 | 任何热血主角 |

## 对话素材与感官速查

### 战斗前对话

**无声型**：两个人站在对面。谁都没说话。风吹过去了。他把手放在了剑柄上。对面那个人也动了。

**最后交代型**："我要是回不来——""你会回来的。""......你让我说完。""不用说完。你会回来的。"

### 战斗中对话

**垃圾话**："就这？你练了多久？"

**喘气型**："你......变强了。""你......也是。"（两个人都笑了。然后继续打。）

**被打断型**："你——"（被打飞）"......够狠的。"（爬起来）"但我——"（又被打倒）"......还没——"

**沉默型**：每一招就是对话——他用了一招新的=说"我进步了"。

### 战斗后对话

**感谢型**："你打的那一拳真疼。""你踢的那一脚也不轻。""......下次还来？""嗯。"

**告别型**："你要走了？""嗯。""......注意安全。"（走了大概一百步停了一下，然后继续走。）

### 导师台词

> "你这一刀，劈柴都嫌歪。""......师父，我在练杀招。""杀招？你先学会劈柴。"

## 感官描写速查

### 力量感
**不要写**："他出了一拳，力量很大。"
**要写**：声音像撕布。肋骨发出闷响。地面碎了一圈。

**要素**：声音+视觉+触觉+后果

### 速度感
**不要写**："他的速度很快。"
**要写**：短句。一个动作一句话。对手反应来不及。残影。

### 疼痛感（5种类型）

| 类型 | 描写 |
|------|------|
| 钝痛 | 被打中腹部，像被巨石压着，喘不上气 |
| 刺痛 | 先是凉的（刀太快），然后才是热的（血在流） |
| 撕裂痛 | 一动就疼，不动也隐隐作痛 |
| 麻痹 | 没知觉了——没知觉比疼更可怕 |

### 疼痛层次（4层）

1. 感到了疼（表面）→ 2. 在忍（心理）→ 3. 忽视了疼继续打（意志）→ 4. 身体撑不住了（极限）

## 情绪描写分层

### 愤怒

| 层次 | 表现 |
|------|------|
| 表面（控制中） | 声音变低 / 咬牙 / 指甲嵌进肉里 |
| 爆发（失控） | 理智断 / 招式暴力不计后果 |
| 冷却（空虚） | 手在抖 / 低头看手上有血 |

### 恐惧

身体反应：膝盖抖 / 手心出汗 / 想吐 / 眼睛不自觉寻找退路——然后强迫自己不去看

极点反应：反而安静 / 极度专注 / 可能笑

### 决心

**写决心的误区**：不要让角色说出来"我下定决心了"——太直白。用行为暗示。
