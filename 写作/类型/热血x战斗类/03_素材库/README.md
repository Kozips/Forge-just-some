## 二、场景素材片段

### 绝境反击

> 他趴在地上，视线模糊。嘴里全是铁锈的味道——牙齿磕破了嘴唇。对手站在三米外，没有追击："还要打吗？"他没回答。手指扣着地面的裂缝，指甲断了。膝盖在发抖。他想：站起来。腿没动。他又想：站起来。这次不是想——是吼出来的。膝盖直了。对手的眉头皱了一下。

### 同伴羁绊爆发

> 她倒在血泊里，手里还攥着那面旗——队旗。对方说："投降吧，你们已经输了。"他看着她的手。旗杆已经断了，旗面破了一半，但她没有松手。他转过身，面对对手，说了四个字："还没结束。"

### 认可对手

> 他赢了。对手跪在地上，抬头看着他："再打一次。"他伸出手："站起来再说。"对手愣了一下，握住了他的手。

### 第一次展示实力

> 先写对手的轻视→主角出手（一招就够）→所有人反应分层写（有人惊讶/有人不信/有人开始认真）→主角自己的反应要克制（一个眼神够了）

### 突破极限

> 他的意识开始模糊。眼前是黑的。耳朵里只剩下心跳声。然后他听到了一个声音。很远。但很清楚。他在等我。他的手指动了。然后是手。然后是手臂。他撑着地面，把身体一寸一寸地抬起来。骨头在响。肌肉在撕裂。他不在乎。他站起来了。

### 战后安静

> 胜者的状态：不是兴高采烈，是累、是沉默、是"终于完了"。败者的状态：如果有尊严的败者，给他一个画面。安静本身就是一种力量。不需要对话，一个画面就够了。

## 三、对话素材

### 战斗前

**无声型**：两人对视，风吹过，手放剑柄，对面也动了。
**简短确认型**："准备好了？""嗯。"
**最后交代型**："我要是回不来——""你会回来的。""......你让我说完。""不用说完。你会回来的。"

### 战斗中

**喘气型**："你......变强了。""你......也是。"（两个人都笑了。然后继续打。）
**被打断型**："你——"（被打飞）"......够狠的。"（爬起来）"但我——"（又被打倒）"......还没——"
**沉默型**：每一招就是对话——他用了一招新的=说"我进步了"，对手换打法=说"我认可你了"。

### 战斗后

**感谢型**（不用"谢谢"）："你打的那一拳真疼。""你踢的那一脚也不轻。""......下次还来？""嗯。"
**告别型**："你要走了？""嗯。""......注意安全。""你也是。"（走了大概一百步停了一下，然后继续走。）
**无言型**：走到同伴身边坐下，递一瓶水，看着远处，什么都没说。但什么都说了。

### 训练中

> "你这一刀砍下去连树皮都破不了。""你来示范啊。""看好了。"（一刀劈开）"看到区别了吗？""......你的刀比较贵。"

### 导师台词

> "你这一刀，劈柴都嫌歪。""......师父，我在练杀招。""杀招？你先学会劈柴。柴都劈不齐，你劈什么人？"

## 四、感官描写速查

### 力量感
- 不写"他挥了一拳"，写"他的拳头穿过空气的声音，像布被撕开"
- 要素：声音（碰撞/撕裂/骨头响）+ 视觉（冲击波/碎裂地面）+ 触觉（震动传到手臂）+ 后果（被打飞/弯/入地）

### 速度感
- 不写"他速度很快"，写"他动了。对手的瞳孔收缩了。但身体来不及反应。一道残影。"
- 要素：短句（一个动作一句话）+ 对手反应来不及 + 残影/风声/地面脚印

### 疼痛感（分类型）

| 类型 | 描写 |
|------|------|
| 钝痛 | 被打中腹部，像被巨石压着，喘不上气 |
| 刺痛 | 先是凉的（刀太快），然后才是热的（血在流） |
| 撕裂痛 | 一动就疼，不动也隐隐作痛 |
| 灼烧痛 | 不是被火烧，是骨头里着火 |
| 麻痹 | 没知觉了——没知觉比疼更可怕 |

### 疼痛层次
1. 感到了疼（表面）
2. 在忍（心理）
3. 忽视了疼继续打（意志）
4. 身体撑不住了（极限）

### 环境参与战斗

| 地面 | 效果 |
|------|------|
| 泥地 | 脚底打滑，泥浆摔跤 |
| 岩石 | 碎石飞溅 |
| 水面 | 水花/阻力/看不清脚下 |
| 冰面 | 站不稳，碎冰可当武器 |

| 天气 | 效果 |
|------|------|
| 雨 | 视线模糊/手滑/泥泞 |
| 雷 | 闪电照亮战场一瞬间又暗 |
| 风 | 影响远程/风沙迷眼 |
| 雪 | 冷/脚陷进去/血在白雪上刺眼 |

### 情绪描写速查

| 情绪 | 表面（控制中） | 爆发（失控） | 冷却（之后的空虚） |
|------|----------------|--------------|---------------------|
| 愤怒 | 声音变低/咬牙/指甲嵌进肉里 | 理智断/招式暴力不计后果 | 手在抖，低头看手上有血 |
| 恐惧 | 膝盖抖/手心出汗/想吐 | 反而安静/极度专注/可能笑 | — |
| 决心 | 不看退路/呼吸变稳/话变少 | — | 眼神让人觉得"已做好死的准备" |

## 五、经典台词模式

### 热血主角
- "我不知道能不能赢。但我不能不去。"
- "你先走。"（已做好回不来的准备）
- "......还没结束。"（所有人以为他已经输了）
- "还没结束。"（四个字版本，更有力）

### 对手之间
- "你很强。可惜我们是敌人。"
- "正因如此，我才要全力以赴。"
