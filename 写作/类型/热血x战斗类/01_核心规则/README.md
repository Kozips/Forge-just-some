# 热血类 — 核心写作规则

> 本目录存放该类型的铁律与不可违反的规则。开写前必须阅读。

---

## 铁律（违反即崩）

### 铁律1：热血来自"明知不可为而为之"

**规则**：角色面对明显不可能赢的局面，选择战斗——这种选择本身就是燃点。
**正例**：对方是三个人，他只有一个。他说："一个就够了。"
**反例**：他比对手强十倍，轻松碾压。
**违反后果**：变成龙傲天爽文，失去热血感。

### 铁律2：成长必须通过失败实现

**规则**：角色在真正的成长之前，必须经历一次令人信服的失败。
**正例**：他第一次挑战Boss，三招就被打飞了。但他记住了那三招。
**反例**：他修炼了三天，实力暴涨，轻松反杀。
**违反后果**：成长没有重量，读者不买账。

### 铁律3：同伴的羁绊比个人力量更重要

**规则**：关键时刻的爆发不是因为"主角更强了"，而是因为"不能让同伴失望"。
**正例**：他已经站不起来了。但身后的人还在看着他。他咬着牙站起来。
**反例**：他突然领悟了终极奥义，一招秒杀。
**违反后果**：热血变成个人秀，失去情感共鸣。

### 铁律4：对手必须值得尊敬

**规则**：最好的热血对手是"如果立场不同，你会希望他是你的同伴"。
**正例**：对手说："你很强。可惜我们是敌人。"主角说："正因如此，我才要全力以赴。"
**反例**：对手又蠢又弱又坏，被打是活该。
**违反后果**：胜利没有重量，读者觉得"这不是理所当然的吗"。

### 铁律5：突破必须有代价

**规则**：任何能力突破、极限爆发、领悟升级，都必须付出可感知的代价（寿命/身体/记忆/关系）。
**正例**：他用了一招禁术。赢了。但右手再也握不了剑了。
**反例**：他在濒死时突然领悟了最强奥义，满血满蓝复活。
**违反后果**：力量体系崩塌，后续战斗再无紧张感。

### 铁律6：热血靠行动证明，不靠口号堆砌

**规则**：删掉所有感叹号、省略号和"啊啊啊"，你的故事还热血吗？如果还热血，你写对了。
**正例**：他擦了一下嘴角的血。没说话。把剑举起来了。
**反例**："我一定会打败你的！因为我要保护我的同伴！这就是我的忍道！"
**违反后果**：读者跳过对白直接翻页。

### 铁律7：输比赢重要

**规则**：读者记住的不是主角赢了多少次，是他输了之后怎么站起来。每5场战斗至少输1场。
**正例**：他输了。跪在地上，血从嘴角流下来。第二天他又站在了擂台上。
**反例**：主角从头赢到尾，百战百胜。
**违反后果**：战斗变成走过场，读者不再紧张。

## 强化规则（建议遵守）

### 规则8：守护是终极驱动力

热血的终极驱动力不是"变强"，是"守护"。守护的三个层次：
- **表层**：保护同伴的身体安全——"我不会让你死在这里"
- **深层**：保护同伴的梦想——"你想做的事，我帮你做到"
- **最深层**：保护同伴的纯真——"你不用变成跟我一样的人"

### 规则9：回忆杀不超过两段

战斗中的回忆杀是调味料，不是主菜。一段短回忆用来触发情感就够了，两段是极限。回忆必须由战斗动作触发（闪回式1-2句），不能是长段独白。

### 规则10：每场战斗的套路必须有变化

第一次"体力不支→回忆→突破"，读者热血。第五次用同样的套路，读者翻页。至少每3场战斗换一种突破方式（量变型/悟道型/突破型/换道型/传承型交替使用）。

### 规则11：胜利后不开香槟

战斗结束后，胜者的反应是喘息和沉默，不是欢呼。庆祝放在下一章或跳过。给败者尊严的镜头。

### 规则12：不回避死亡

如果设定了有人会死，就真的让他死。假死是最大的败笔。死后不要立刻进入下一章，留一章的空间给悲伤。

---

## 一句话检验公式

> **热血 = 有理由的变强 x 有重量的代价 x 有温度的羁绊**

每一章必须回答一个问题：「他凭什么赢？」——答案不能是"因为主角光环"，必须来自代价、技巧或领悟的其中之一。
