# 武器与格斗知识深度库

> 写武器和格斗不是写说明书，是写"拿着这件东西的人会变成什么样"。

---

## 关键生理限制（违反会让读者觉得假）

- 被掐脖子4-10秒就昏迷（不像电影里挣扎几分钟）
- 骨折后肾上腺素可暂时忽略疼痛但活动受限
- 打人的手也会受伤——额骨比掌骨硬，打额头可能先断手骨
- 肝脏一击就倒——职业拳手也扛不住
- 后脑被重击=极度危险，可能直接死亡

**拳击四大拳法**：刺拳（试探，距离最远）→直拳（主力输出）→勾拳（打太阳穴和肝脏）→上勾拳（打下巴，最容易KO）。

**要害部位效果**：
| 部位 | 效果 | 程度 |
|------|------|------|
| 下巴 | 脑震荡/直接KO | 致命级 |
| 太阳穴 | 嗡鸣/视野模糊 | 致命级 |
| 肝脏 | 弯腰喘不上气 | 非致命但有效 |
| 胃部 | 干呕/呼吸暂停 | 非致命 |
| 肋骨 | 可能断裂 | 中-高 |
| 膝盖后方 | 失去站立能力 | 非致命但决定性 |

## 受伤后"还能打"的窗口

| 伤害类型 | 肾上腺素窗口 | 写法 |
|---------|------------|------|
| 浅刀伤 | 30秒-2分钟 | 先写"不疼了"，窗口过后断崖式崩溃 |
| 骨折 | 即时-数分钟 | 先写不疼，但活动幅度受限 |
| 中枪（非要害） | 数秒-数十秒 | 被"推"了一下，然后血压下降 |
| 失血过多 | 逐渐衰退 | 越来越冷，越来越困，视野边缘发黑 |

## 体力消耗曲线

**体力递减是天然的叙事计时器**：

| 阶段 | 时间 | 身体状态 | 句子节奏 |
|------|------|---------|---------|
| 满状态 | 0-30秒 | 动作流畅 | 正常长度 |
| 轻度消耗 | 30秒-2分钟 | 出汗，呼吸加深 | 开始变短 |
| 中度消耗 | 2-5分钟 | 手抖，腿软 | 短句为主 |
| 严重消耗 | 5-10分钟 | 几乎站不住 | 断续的词 |
| 极限 | 10分钟+ | 身体关机 | 一个字一段 |

## 地形影响

| 地形 | 关键变量 |
|------|---------|
| 窄巷子 | 无法围攻，靠墙反弹 |
| 开阔地 | 距离决定一切 |
| 楼梯 | 高处优势，失足=摔落 |
| 室内 | 门=伏击点 |
| 泥地 | 容易滑倒 |
| 黑暗 | 声音比视觉重要 |
| 雨中 | 武器打滑 |

## 枪战关键细节

**子弹计数——紧张度调节器**：满弹匣→安全感 → 减少→紧张上升 → 最后几发→极致紧张 → 空枪声→最大转折点

**换弹匣脆弱窗口（2-3秒）**：拔出空弹匣"叮"——手指滑——插歪了——拍"咔嗒"——上膛。这三秒什么都做不了。

**被枪击层次**：
| 枪型 | 击中感觉 |
|------|---------|
| 手枪弹 | "推"→热→钝痛 |
| 步枪弹 | "被锤子砸了"，四肢可能碎骨 |
| 擦伤 | 偏两厘米就是生死之别 |
