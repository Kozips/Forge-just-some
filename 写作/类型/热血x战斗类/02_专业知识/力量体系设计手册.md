# 热血类 — 力量体系设计手册

> 力量体系是骨架。体系不好，战斗就像两个人在比数字大小。

## 体系自检四问

1. **力量有没有上限？** — 无上限=后期必然崩盘。第一卷就暗示天花板。
2. **变强有没有代价？** — 代价要具体：寿命、记忆、身体部位、理智、人际关系。
3. **体系规则有没有被打破的可能？** — 允许"极端代价下突破规则"。
4. **读者能不能理解？** — 三句话说不清楚的体系太复杂。

## 六种体系模板

**1. 等级制**（F→E→D→C→B→A→S→SS→SSS）
防膨胀：第一卷定最高等级且全书不改；等级差距是指数的；主角提升前期快中期慢后期靠质变。等级只是参考——低等级可以靠技巧/配合/代价赢高等级。

**2. 属性制**（力量/速度/技巧/耐力/特殊能力）
防面板感：属性只在幕后使用；通过场景展示而非数字对比。

**3. 技术制**（剑法/呼吸法/格斗术）
每种流派有核心理念。进步不是"学新招"而是"理解旧招的真义"。不同流派的对决=不同理念的碰撞。

**4. 器械/装备制**
装备只是放大器——最热血的时刻是"装备毁了，他用最基础的方式赢了"。

**5. 传承/血脉制**
血脉只是起点不是终点；有代价/副作用；同一血脉有多个选中者，赢是因为更努力。

**6. 混合制**
核心只用一种，其他做辅助。要有一个"统一公式"让读者能判断强弱。

## 体系与叙事的绑定

- 主角目标是否跟等级有关？
- 代价是否改变了角色的人生选择？
- 天花板是否跟世界观核心矛盾有关？
- 强弱差距是否制造了社会冲突？

**脱节症状**：角色变强但生活没变化；体系升级像游戏打怪；代价设定了但从未兑现。

## 突破类型

| 类型 | 体系解释 | 叙事表现 |
|------|---------|---------|
| 量变到质变 | 长期积累触发跃升 | 练了三百天同一招，第三百天质变 |
| 悟道突破 | 对规则的新理解 | "不是力量更大，是用在了更精准的位置" |
| 情感突破 | 极端情感下暂时超越等级 | "等级没变，但那一击超过了A级"——代价是事后崩溃 |
| 代价突破 | 禁忌手段强行突破 | "用了禁术，那一击是S级。但只剩三个月寿命" |
| 传承突破 | 从更高等级前辈获得力量 | "师父把剩余力量传给了他" |

突破不是永久的。量变→新等级稳固但需适应；悟道→理解是新的但身体还需训练；情感→事后回原等级甚至更低；代价→永久但代价持续生效；传承→需磨合。

## 常见陷阱

| 陷阱 | 修正 |
|------|------|
| 等级膨胀 | 第一卷定最高等级，全书不改 |
| 代价失信 | 设定后3章内兑现，否则删掉 |
| 体系歧视 | 每种体系都有克制关系 |
| 唯一主角光环 | 主角走代价路线突破 |
| 数字感 | 用场景展示强弱，不报数字 |
| 随意加规则 | 新规则出现前必须有暗示 |
| 无代价继承 | 继承要附带前辈的负担/记忆/责任 |
