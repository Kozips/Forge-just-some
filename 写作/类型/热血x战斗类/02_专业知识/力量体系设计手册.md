# 热血类 — 力量体系设计手册

> 构建热血世界观时，力量体系是骨架。体系不好，战斗就像两个人在比数字大小。
> 本文件提供可直接套用的体系框架、平衡规则和常见陷阱。

---

## 一、体系自检四问

写任何力量体系之前，先回答这四个问题：

1. **力量有没有上限？** — 无上限体系 = 后期必然崩盘。必须在第一卷就暗示天花板。
2. **变强有没有代价？** — 如果代价只是"多练几天"，读者不会心疼。代价要具体：寿命、记忆、身体部位、理智、人际关系。
3. **体系规则有没有被角色打破的可能？** — 体系太死，角色是棋子；体系太活，规则就形同虚设。好的做法：体系有明确规则，但允许"极端代价下突破规则"。
4. **读者能不能理解这个体系？** — 三句话说不清楚的体系太复杂。热血文的体系要让读者直觉上能判断"谁更强"。

## 二、六种常见体系模板

### 1. 等级制（最常见）

阶梯式等级：F→E→D→C→B→A→S→SS→SSS

**适用**：修仙、武道、超能力

**优点**：读者直觉判断强弱，不需要解释
**缺点**：后期等级膨胀（第一卷A是天花板，第三卷A是起步价）

**防膨胀规则**：
- 从第一卷就设定最高等级，且这个等级在全书后半段才有人达到
- 等级之间的差距不是线性的，是指数的（B比A强10倍，A比S强100倍）
- 主角的等级提升不是匀速的——前期快，中期慢，后期靠质变

**写作时**：等级只是参考。战斗胜负不完全看等级——低等级可以靠技巧/配合/代价赢高等级，但代价要真实。

### 2. 属性制

力量/速度/技巧/耐力/特殊能力 — 五维属性表

**适用**：体育、格斗、游戏化世界

**优点**：每个角色有明确的强项和弱项，战斗像棋局
**缺点**：容易写成"看面板"的感觉，缺乏叙事感

**防面板感规则**：
- 属性只在幕后使用，不要让角色说出"他的力量值是800"
- 属性差距要通过具体场景展示，不是数字对比
- 属性的"隐性维度"比显性更重要——经验、意志、配合、临场应变

### 3. 技术制

不是比谁强，是比谁技术更好。剑法、呼吸法、格斗术。

**适用**：武侠、体育、写实格斗

**优点**：战斗有战术深度，不是数字碾压
**缺点**：需要作者真的懂一些技术知识（或做足够的功课）

**写法要点**：
- 每个技术流派有核心理念（不是"更快更强"，是"用对手的力量打对手"）
- 技术的进步不是"学了个新招"，是"理解了旧招的真正含义"
- 不同流派之间的对决 = 不同理念的碰撞

### 4. 器械/装备制

力量来自武器/装备/搭档。人本身不变，但装备变了。

**适用**：机甲、魔法道具、搭档系（人+兽/人+武器）

**优点**：装备升级是可见的成长感，搭配变化带来战术变化
**缺点**：容易变成"装备决定一切"，人的努力被忽略

**防器械依赖规则**：
- 装备只是放大器——没有装备他依然有基础实力
- 装备跟使用者的契合度有关——同样的武器，不同人用效果不同
- 最热血的时刻是"装备毁了/失效了，他用最基础的方式赢了"

### 5. 传承/血脉制

力量来自血脉/传承/命运的选中。

**适用**：神话系、家族系、宿命系

**优点**：有天然的"为什么他特殊"的解释
**缺点**：读者会觉得"不公平"——凭什么他天生就强？

**防不公平感规则**：
- 血脉/传承只是起点，不是终点——有血脉但不用 = 废物
- 血脉有代价/副作用——比如使用后失控/消耗寿命/引来敌人
- 同一血脉的人有多个，主角不是唯一的"选中者"——他赢不是因为被选中，是因为他比别人更努力

### 6. 混合制

以上几种混合。比如等级+属性+传承。

**适用**：大型世界观

**写法要点**：
- 核心只用一种，其他做辅助——不能三种体系同时主导
- 混合体系要有一个"统一公式"让读者能判断强弱
- 每种体系之间的交互要有规则（传承等级能不能覆盖技术等级？）

## 三、体系与叙事的绑定

力量体系不能游离在故事之外。它要和角色动机、世界观设定、冲突结构深度绑定。

### 绑定检查表

| 体系特征 | 是否绑定到叙事 | 怎么绑定 |
|----------|----------------|----------|
| 体系等级 | 主角的目标是否跟等级有关？ | "我要达到A级才能加入正规军" / "师父是S级，我要超越他" |
| 体系代价 | 代价是否改变了角色的人生选择？ | "用禁术会缩短十年寿命" → 他用不用取决于他觉得自己还有多少年值得活 |
| 体系天花板 | 天花板是否跟世界观的核心矛盾有关？ | "人类的力量上限是被神设定的" → 打破天花板 = 质疑神的权威 |
| 体系差距 | 强弱差距是否制造了社会冲突？ | "A级以上的人统治世界，F级的人是劳工" → 主角从F级起步 = 从底层挑战秩序 |

### 体系绑定的反面教材

力量体系跟叙事脱节的症状：
- 角色变强了但生活没有变化
- 体系升级像游戏打怪，没有叙事意义
- 主角为了"变强"而变强，没有一个具体的目标
- 代价设定了但从来没真正兑现过

## 四、突破场景的体系设计

每次"突破"都要在体系内有逻辑依据。不是"他突然变强了"，是"体系允许这种突破，但条件极端"。

### 五种突破在体系中的对应

| 突破类型 | 体系解释 | 叙事表现 |
|----------|----------|----------|
| 量变到质变 | 同一等级内的长期积累触发跃升 | 练了三百天同一招，第三百天质变 |
| 悟道突破 | 对体系规则的新理解，用不同的方式使用现有力量 | "不是力量更大，是力量用在了更精准的位置" |
| 情感突破 | 体系允许极端情感下暂时超越等级 | "他的等级没变，但那一击的力量超过了A级"——代价是事后崩溃 |
| 代价突破 | 用禁忌手段强行突破体系限制 | "他用了禁术，那一击是S级水平。但他只剩三个月寿命了" |
| 传承突破 | 从更高等级的前辈那里获得力量 | "师父把他剩余的力量传给了他。但师父再也上不了战场了" |

### 突破后的体系状态

突破不是永久的。每次突破之后，体系状态要变化：

- **量变突破**：新等级稳固，但需要时间适应新力量
- **悟道突破**：理解是新获得的，但身体还需要训练跟上
- **情感突破**：事后回到原等级，甚至更低（身体损伤）
- **代价突破**：突破是永久的，但代价持续生效
- **传承突破**：获得的力量需要磨合，一开始使用效率低

## 五、常见体系陷阱

| 陷阱 | 症状 | 修正方案 |
|------|------|----------|
| 等级膨胀 | 第一卷天花板到了第三卷变地板 | 第一卷就定好最高等级，全书不改 |
| 代价失信 | 设定了代价但从未兑现 | 代价在设定后3章内兑现，否则删掉 |
| 体系歧视 | 某种力量类型永远碾压另一种 | 每种体系都有克制关系，无绝对碾压 |
| 唯一主角光环 | 体系规则对主角不适用 | 主角遵守体系规则，突破走代价路线 |
| 数字感 | 读者感觉在看游戏面板 | 用场景展示强弱，不直接报数字 |
| 随意加规则 | 打不过就加新体系规则 | 新规则在第一次出现前必须有暗示 |
| 无代价继承 | 主角继承前辈力量没有代价 | 继承要附带前辈的负担/记忆/责任 |
| 体系无天花板 | 角色永远可以变强 | 设定天花板，触及天花板就是故事终局 |

## 六、体系速建流程（30分钟）

1. **选模板**（5分钟）：从六种模板中选核心体系，最多加一种辅助体系
2. **定等级**（5分钟）：写5-8个等级名，最高等级永远不改
3. **定代价**（5分钟）：每个等级/每种能力都有代价，代价要具体到身体/时间/情感
4. **定天花板**（5分钟）：最高等级意味着什么？谁能达到？主角什么时候达到？
5. **绑定叙事**（10分钟）：主角的目标、世界观的核心矛盾、社会结构，都跟体系挂钩

产出物：一张体系总表 + 一段"主角从F级到天花板的路程"概要。
