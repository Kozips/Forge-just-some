## 一、力量体系设计

### 通用五阶模型

| 阶段 | 特征 | 突破条件 | 叙事功能 |
|------|------|----------|----------|
| 基础 | 掌握基本技能 | 系统训练 | 主角入场券 |
| 进阶 | 能应对同级别对手 | 实战经验 | 展示努力有回报 |
| 瓶颈 | 遇到无法超越的对手 | 认知/信念的转变 | 低谷/挫败段 |
| 突破 | 超越原有极限 | 关键战斗中的领悟 | 高潮燃点 |
| 巅峰 | 达到该体系的极限 | 不再追求"更强" | 角色弧线收束 |

### 五种"变强"路径（写作时交替使用）

| 路径 | 机制 | 适合场景 | 注意 |
|------|------|----------|------|
| 量变型 | 练了无数次，终于质变 | 基础→进阶 | 避免流水账，只写第一次/中间/最后一次 |
| 悟道型 | 某个瞬间突然理解了什么 | 瓶颈期 | 触发器要具体（一句话/一个画面），不能"闭眼冥想就悟了" |
| 突破型 | 被逼到极限，靠情感驱动爆发 | 高潮战斗 | 前面必须有"差一点就突破"的铺垫 |
| 换道型 | 原方法行不通，走新路 | 遭遇克制型对手 | 新路不是"更好的路"，是"更适合他的路" |
| 传承型 | 从前辈那里继承种子 | 师父退场后 | 主角用出来的方式跟前辈不一样 |

### 突破的三个必要条件

1. **铺垫够不够**：读者要先知道他的极限在哪
2. **代价够不够大**：用寿命换的、用理智换的、用未来换的
3. **情感够不够足**：他为什么非突破不可？不是"为了赢"，是"因为那个人在等他回来"

## 二、战斗类型学

### 按功能分

| 战斗类型 | 叙事目的 | 篇幅建议 | 胜负 |
|----------|----------|----------|------|
| 初战/首秀 | 展示实力，建立期待 | 中 | 胜（但不轻松） |
| 训练战 | 展示成长过程 | 短 | 输或险胜 |
| 宿命对决 | 核心对手的正面碰撞 | 长 | 险胜或惨胜 |
| 守护战 | 为保护同伴/信念而战 | 长 | 胜（代价大） |
| 绝境反击 | 被打到极限后的爆发 | 长 | 胜（突破性） |
| 传承战 | 新一代超越旧一代 | 中 | 胜（情感重于胜负） |
| 群像战 | 团队各自面对敌人 | 长 | 有赢有输 |
| 决战 | 全书最重要的战斗 | 极长 | 险胜 |

### 按对手类型分

| 对手类型 | 对战策略 | 热血点 |
|----------|----------|--------|
| 力量碾压型 | 速度/技巧/配合 | 以巧胜拙 |
| 技巧碾压型 | 意志力/换道 | 找到破绽 |
| 理念冲突型 | 用行动证明自己的路 | 信念碰撞 |
| 情感纠葛型 | 放下执念/理解对方 | 惺惺相惜 |
| 数量优势型 | 团队配合 | 并肩作战 |

## 三、热血场景类型速查

| 场景 | 核心画面 | 写法要点 |
|------|----------|----------|
| 第一次展示实力 | 对手轻视→主角出手→全场震惊 | 一招就够，反应分层写 |
| 修炼/训练 | 最苦的一天 | 不写流水账，写身体具体反应 |
| 宿敌重逢 | 两个人站在那里，空气变了 | 先写眼神对撞，不一定打起来 |
| 绝境求生 | 真的没有退路 | 写恐惧——怕了但还是要上 |
| 同伴倒下 | 伏笔→倒下→主角反应 | 主角反应不只有愤怒 |
| 团队合力 | 各司其职 | 最后一击是所有人一起完成的 |
| 师父退场 | 最后一课 | 教的不是技术，是态度 |
| 突破极限 | 意识模糊→声音拉回→站起来 | 有身体代价 |
| 赛后安静 | 没有人说话 | 安静本身就是力量 |
| 决战前夜 | 每个人的最后安静夜晚 | "明天见"三个字就够了 |

## 四、常用术语

| 术语 | 含义 | 使用注意 |
|------|------|----------|
| 杀意 | 对手的真正实力释放 | 不要滥用，只在关键对手 |
| 觉醒/爆种 | 极限状态下突破 | 必须有代价，不能无限爆种 |
| 气场/威压 | 强者的存在感 | 通过环境和弱者反应来写，不直接说"好强的气场" |
| 领域/结界 | 专属战斗空间 | 有进入条件和限制 |
| 禁术/代价 | 使用禁忌能力的副作用 | 代价必须真实兑现 |
| 悟道 | 瞬间的领悟 | 触发器要具体，不能"闭眼就悟" |

## 五、角色关系图谱

### 八种核心角色定位

| 角色 | 功能 | 与其他角色的关系 |
|------|------|------------------|
| 热血主角 | 冲锋+情感核心 | 需要军师拉住、需要导师引导 |
| 天才对手 | 镜像+上限标杆 | 跟主角一体两面 |
| 热血导师 | 传承+退场催化 | 自己可能已过巅峰 |
| 冷静军师 | 刹车+策略 | 跟主角化学反应最好 |
| 搞笑担当 | 呼吸+隐藏面 | 不笑的时候才是危险信号 |
| 隐藏强者 | 惊艳+安全感 | 前期暗示，被迫揭示 |
| 反派BOSS | 理念碰撞+故事重量 | 有自己的道理，不是脸谱化恶人 |
| 牺牲者 | 重量+驱动力 | 未完成的心愿成为后续动力 |

### 配角战斗风格与性格匹配

| 性格类型 | 战斗风格 | 弱点 |
|----------|----------|------|
| 冲动型 | 大开大合，有威力但破绽多 | 需要别人补漏洞 |
| 冷静型 | 精准，无多余动作 | 遇到不按套路出牌的不适应 |
| 搞笑型 | 看起来滑稽但意外有效 | 脑子转得慢 |
| 天才型 | 华丽精准，让人嫉妒 | 太依赖天赋，不习惯苦战 |
