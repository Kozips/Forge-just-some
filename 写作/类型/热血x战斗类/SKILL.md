# 热血写作 Skill

> "打不过也要打。不是因为能赢，是因为不能退。"

---

## 速查表

**核心公式**：热血 = 不对等对抗 x 不屈意志 x 有温度的羁绊 — 每章回答"他凭什么赢？"答案不能是主角光环。

### 单章结构（挑战-苦战-突破型）

| 阶段 | 占比 | 动作 |
|------|------|------|
| 挑战出现 | 15% | 新对手/障碍，角色主动迎战 |
| 苦战 | 35% | 对手比想象中强 + 展示技巧毅力 + 旁观者反应 |
| 突破/转折 | 30% | 极限中找到突破口（新技巧/新理解/意志力） |
| 余波 | 20% | 喘息反思 + 新目标/更强对手出现 |

**铁律**：开局冲劲(15%) → 拉扯加压(15-50%) → 翻转突破(50-80%) → 余波埋种(80-100%)

### 角色压力点 Top 5

1. **体力极限** — 手在发抖/视线模糊/旧伤复发
2. **心理动摇** — "我做不到"闪过然后压制
3. **信任危机** — 同伴质疑/观众倒戈/队友因"我"受伤
4. **代价预兆** — 突破前交代"用了会怎样"
5. **对手尊重** — 对手不是纯恶，展现实力让胜利有分量

### 失败模式 Top 5

1. **无代价胜利** → 每次突破必须失去点什么
2. **解说式回忆杀** → 闪回式1-2句，由动作触发
3. **对手纸片化** → 给对手一个读者能理解的动机
4. **意志=万能** → 意志只能撑住不倒，突破靠新理解
5. **感叹号热血** → 删掉感叹号还热血才对

### 结尾类型

| 类型 | 情绪基调 |
|------|----------|
| 险胜余韵 | 疲惫中的满足——单次对决 |
| 惨胜代价 | 赢了但失去重要之物 |
| 败中藏胜 | 输了但突破心障 |
| 新敌登场 | 短暂喘息被打断 |
| 告别升华 | 沉默——"终于结束了" |

### 开篇钩子

**A**. 他面对一百个人。手里只有一把断刀。他说："来吧。"
**B**. 他输了。在所有人面前。但第二天，他又站在了擂台上。
**C**. 赢了她就能走，输了他得留下。他说："赌。"

### 5步检查表

- [ ] 开头50字内建立了明确的"赢/输"赌注？
- [ ] 中场有至少2次攻守反转？
- [ ] 突破的代价具体可见？（不是"很累"，是"右手握不住刀"）
- [ ] 角色的"凭什么赢"有逻辑支撑？
- [ ] 对手有读者能理解的动机？

### 一句话检验

删掉所有感叹号、省略号和"啊啊啊"，你的故事还热血吗？还热血=写对了。不热血=靠标点撑场面。

## 角色速查（写作时直接参考）

> 完整版见 `人物/references/人物联动写作/热血类.md`

### 萧云（乐观 / 胆大 / 脸皮厚）
- 语言："这他妈的爽！""修炼难度大？没有关系，我有顿悟帮助"
- 语气：激动时咧嘴一笑，面对犹豫时主动表态不等人劝
- 动作：十六年苦等系统完成后激动流泪；挤进人群朝测试台走去（行动派）
- 用法：热血冲锋手，内心独白要直白粗犷，自信到近乎莽，适合开篇制造冲劲

### 陆川（沉默 / 坚韧 / 护人）
- 语言：几乎不说话，用行动回答一切，沉默是最高效的回应
- 语气：低沉简短，被打伤后擦嘴角的血，一个字不说继续战斗
- 动作：短刀劈开荆棘开路永远走在前面；擦嘴角的血（标志性动作）；绝境中站直身体
- 用法：热血中的沉默力量，"被打倒后站起来"比十句话更有力量，适合断后和绝境场景

### 萧乐（冷面军师 / 惜字如金）
- 语言："无论如何，王爷要的是玉佩。陆川要抓，玉佩也要拿。"
- 语气：平稳不带情绪，微微侧头嘴角扯一下就算回应
- 动作：嘴角扯一下（最微小的回应）；沉默一会儿才开口（每句经过思考）；掀帐帘大步走出（干脆利落）
- 用法：热血团队中的冷静指挥，与冲锋手形成一冲一收的层次感
