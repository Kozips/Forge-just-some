# 地下城探险

## 核心：可控的恐惧 — 你知道下面有危险——但你选择下去。因为奖励在下面。

底层公式：分层地下空间（越深越危险）+ 清晰规则 + 分工队伍 + 逐层推进 + Boss和终极宝藏

### 六大元素

**分层结构：** 垂直型（一路向下越深越核心）/ 环形（外围向中心）/ 分支（主线+支线）/ 随机（每次不一样）/ 主题型（火层/冰层/毒层）。

**怪物生态：** 每种有明确弱点和攻击模式。怪物之间有互动（A吃B、B怕C）。Boss是生态链顶端。

**陷阱分类：** 即时型/延迟型（虚假安全感）/ 连锁型/环境型（不是机关是环境本身有毒）/ 心理型（幻觉）。

**宝物：** 越深层越强但风险越大。可能有副作用。真正的宝物有时不是"东西"而是"信息"。

**补给与休息：** 法力有限、药水有限、休息点是稀缺资源。核心博弈——继续深入还是回城？

### 节奏模型

入口 → 前哨区（小怪试探）→ 资源消耗区（陷阱+中怪）→ 安全区（休息/宝箱/信息）→ 精英区 → Boss房前 → Boss战 → 战利品

### 避坑：失败模式

- **数值碾压：** "我比你数值高所以我赢"——故事变成Excel表格。
- **每层没新东西：** 第三层只是"怪更强了宝更好了"——读者会腻。
- **死亡没重量：** 要么不能复活，要么复活有巨大代价。
- **忘了地城外面：** 偶尔切回地面——让读者记得地城跟更大故事有关。
- **浪费休息点：** 篝火旁对话比战斗更能塑造角色。

### 跨作品规律

- 地城必须有自己的"历史"——谁建的？为什么被遗弃？
- 每层"主题"要比上一层不同。视觉和机制都要变。
- Boss是"这层故事的高潮"——是这层地城的"主人"。
- 战利品要配得上付出。
- 篝火/安全区是叙事黄金时刻。

### 写作技法

**"向下"的叙事力量：** 用物理上的"下"制造心理上的"沉"。

第一层石阶光滑有灯光 → 第二层粗糙岩石昏黄油灯 → 第三层台阶断了 → 第四层没有台阶了——一个垂直竖井。

**分层叙事：** 每一层讲一个故事。第一层告诉读者"规则"→ 第二层引入新规则 → 第三层打破规则 → Boss层回应所有铺垫。

**陷阱三段式：**
- 发现："别动。"他指着一块低了一毫米的石板。"压力板。看到头顶那些孔了吗？"
- 触发：地板消失。手抓住边缘。手电掉下去——照到下面的东西——密密麻麻的生锈尖刺。
- 应对："别挣扎——石缝撑不了两个人——我拉你——脚踩旁边石板——两厘米间隔——安全的。"

**陷阱叠加：** 毒箭躲过去了——毒箭射中的墙壁暗格打开——一只蜘蛛爬出来——踩死蜘蛛——体液腐蚀地板——露出第二层地板——上面有压力板。

**Boss战斗四阶段：** 试探（找预兆：右肩抖0.3秒）→ 反击（利用硬直：三秒够了）→ 变招（新机制）→ 极限（最后弱点）。

### 篝火/休息

"篝火火焰在四个脸上跳。四个不同表情。噼啪声是唯一的声音。'出去之后做什么？'矮人嚼肉干：'喝啤酒。'沉默。'我没有出去之后。'盗贼说。'我没有家。'矮人把肉干递过去。'别说了。吃东西。'盗贼看了一眼。接了。没说话。但吃了。"

### 战斗节奏

快慢交替。快速用短句——"闪。格。反击。"慢速用长句。

"它冲过来——石拳砸地。（快）他后退。喘。左腿不稳。需要侧面。（慢）绕圈——转向左边慢了半拍。（快）冲。"

### 角色模板

**战士/坦克：** 40%坚韧 + 20%沉默 + 20%保护欲 + 10%笨拙 + 10%温柔。"站我后面。""这点伤——回去喝碗酒就好了。"

**法师/术士：** 35%智慧 + 25%好奇心 + 20%骄傲 + 10%脆弱 + 10%疯狂。"省着点用——法力不多了。"

**盗贼：** 30%机灵 + 25%贪财 + 20%谨慎 + 15%怕死 + 10%忠诚。"这扇锁我能开——给我十秒。"

**牧师：** 30%慈悲 + 25%坚定 + 20%智慧 + 15%固执 + 10%脆弱。"我能治——但只有两次了。"

**Boss：** 守关者型（被创造/被命令——打败是"通过考验"）/ 失落者型（曾经英雄变成怪物——打败时感到悲伤）。

**队伍化学反应：** "莽+智" / "冷+热" / "敌+友"。

**资源不足时的分裂（叙事最人性化的时刻）：** "药水剩一瓶了。""给战士——前排。""给我——我死了你们都得死。""给盗贼——他排陷阱——他倒了我们谁也走不出去。"

### 战利品写法

"锁咔嚓开了。掀开盖子。不是金子。是一本书。'一本破书？' '比金子值钱。'牧师翻开第一页——猛地合上。脸白了。'走。现在。' '怎么了？' '它在对我说话。'"

### 写作示范

**手电筒在地下：** "光在石壁上切出一个圆形光斑。之外是纯粹的黑——不是城市夜晚那种还有路灯余光的黑。这种黑是实体的——像棉花填满所有空间。光移到哪里黑暗退到哪里。光移开黑暗立刻回来——像一个活的东西。"

**进入Boss房：** "石门自己开了。手电照进去——干净石板——没有灰尘。有人在打扫这里。石柱上有蛇浮雕。对面墙上——五米人形——脸是平的没有五官。手——六根手指——每根末端是一个张开的嘴。底部是一堆骨头。中央在动——不，在呼吸。那不是浮雕。它从墙上走下来了。空气变冷——六根手指嘴同时开口——不是尖叫。不是咆哮。是笑声。"
