# 冒险类 — 环境威胁与生存法则

> 写生存冒险/探险寻宝时参考。七大环境的威胁机制和角色身体反应逻辑。

---

## 一、七大环境威胁速查

### 1. 热带雨林

| 要素 | 内容 |
|------|------|
| 核心威胁 | 方向迷失、瘴气疾病、毒虫毒蛇 |
| 次要威胁 | 野兽、暴雨导致的河流暴涨 |
| 生存优先级 | 庇护所 > 火 > 食物 > 水源 |
| 标志性感官 | 闷热潮湿的窒息感、腐叶味、虫鸣震耳 |

**写作要点**：雨林最可怕的是"重复感"——走了三小时看起来和出发时一模一样，树木看起来都认识你，但你永远不确定自己在哪。方向感崩溃是心理崩溃的前兆。

**身体反应链**：闷热→大量出汗→电解质流失→头晕→意识模糊→幻觉

---

### 2. 沙漠

| 要素 | 内容 |
|------|------|
| 核心威胁 | 脱水、中暑、沙暴、流沙 |
| 次要威胁 | 方向迷失（沙丘会移动）、海市蜃楼 |
| 生存优先级 | 水 > 防晒 > 夜间保暖 |
| 标志性感官 | 嘴唇干裂到出血、阳光刺眼、沙子灌进每个缝隙 |

**温度规律**：白天40°C+，夜晚可能到0°C。一天经历四季。

**身体反应链**：口渴→嘴唇干裂→嗓子冒烟→少尿→深色尿→器官衰竭

**海市蜃楼写法**：前方出现一汪水，走近发现是干燥的沙地。再走一小时，又出现一汪水。这种循环会让角色崩溃。

---

### 3. 高山/雪山

| 要素 | 内容 |
|------|------|
| 核心威胁 | 失温、冻伤、雪崩、高原反应 |
| 次要威胁 | 雪盲、坠崖、缺氧 |
| 生存优先级 | 庇护所 > 火 > 保暖 > 减少活动 |
| 标志性感官 | 呼出的气瞬间凝结成白雾、手指脚趾先失去知觉 |

**高原反应**：海拔3000米以上开始。头痛→恶心→判断力下降→可能致命。

**冻伤发展**：刺痛→麻木→失去痛感（最危险——感觉不到自己在冻伤）→变黑→需要截肢

**雪崩写法**：先是"轰"的一声，然后是铺天盖地的白色。挣扎时雪会像流沙一样把你往下吸。停止后一片死寂。

---

### 4. 洞穴/地下

| 要素 | 内容 |
|------|------|
| 核心威胁 | 迷路（没有光源就失去方向感）、缺氧、塌方 |
| 次要威胁 | 暗河、毒气、地下生物 |
| 生存优先级 | 光源 > 标记路线 > 空气检测 |
| 标志性感官 | 手电照不到边的黑暗、潮湿的石壁、滴水声在封闭空间放大 |

**迷路机制**：洞穴里没有参照物，走直线也会绕圈。标记路线是生死问题。

**缺氧警告信号**：火苗变小、头疼、困倦感。突然不困了反而更危险——那是意识下降的前兆。

---

### 5. 海洋/孤岛

| 要素 | 内容 |
|------|------|
| 核心威胁 | 淡水耗尽、食物来源不稳定、孤独 |
| 次要威胁 | 风暴、鲨鱼、晒伤 |
| 生存优先级 | 淡水 > 庇护所 > 火 > 食物 |
| 标志性感官 | 盐分让嘴唇干裂更严重、海风让皮肤脱水 |

**漂流时**：渴但不能喝海水（越喝越脱水）。下雨时用一切容器接雨水。

**孤独的写法**：角色开始对不会说话的物品说话、和影子说话、出现幻听。

---

### 6. 太空/真空

| 要素 | 内容 |
|------|------|
| 核心威胁 | 真空暴露（15秒意识丧失）、氧气耗尽、温度极端 |
| 次要威胁 | 辐射、陨石碎片、微重力健康问题 |
| 生存优先级 | 气密性 > 氧气供应 > 温度调节 |
| 标志性感官 | 肺部空气被抽走的窒息感、体液沸腾（低压下） |

**真空暴露阶段**：15秒内意识丧失。但在此之前有约9秒窗口——这9秒是写作的黄金时间。

**太空最独特的感官**：绝对的寂静。没有风、没有水声、没有任何来自自然的声音。只有生命维持系统的嗡嗡声。

---

### 7. 城市废墟/废土

| 要素 | 内容 |
|------|------|
| 核心威胁 | 建筑坍塌风险、辐射/污染、变异生物 |
| 次要威胁 | 食物和水源被污染、找不到遮蔽处 |
| 生存优先级 | 安全检查 > 洁净水源 > 食物 |
| 标志性感官 | 灰尘和腐朽味、偶尔的金属嘎吱声、时间静止的感觉 |

**废墟的恐怖感**：不是"有东西会杀你"，而是"这里曾经有人活着"。墙上褪色的照片、地上的玩具、窗台上的花盆——都是人的痕迹。

## 二、身体状态反应速查表（无补给情况）

| 时间 | 身体反应 | 心理反应 | 叙事价值 |
|------|---------|---------|---------|
| 6小时 | 口渴、饥饿 | 焦虑开始 | 可以克服 |
| 24小时 | 头晕、注意力下降、头痛 | 开始抱怨 | 决策力下降的伏笔 |
| 48小时 | 肌肉酸痛、判断力下降 | 愤怒、争吵 | 开始出错的时机 |
| 72小时 | 幻觉出现、虚弱 | 放弃的念头 | 心理最低点 |
| 5天+ | 器官衰竭 | 谵妄、淡漠 | 生死边缘 |

**写作应用**：不要让角色在72小时前做出"理性决策"——让他们的判断随着身体状态一起下降。

## 三、生存技能速查（写实向参考）

### 取火

| 方法 | 条件 | 难度 |
|------|------|------|
| 钻木取火 | 需要干燥软木，手速要快 | 极高 |
| 凸透镜 | 晴天用放大镜/镜片 | 取决于天气 |
| 打火石 | 需要引火物 | 中等 |
| 电池短路 | 现代环境，电线+钢丝棉 | 低（但需要材料） |

### 净水

| 方法 | 效果 | 局限 |
|------|------|------|
| 煮沸 | 最可靠，杀菌 | 需要火源 |
| 过滤（沙石+布） | 去除杂质 | 不杀菌 |
| 蒸馏 | 能去除盐分 | 费时费力 |
| 药片消毒 | 便携 | 数量有限 |

### 导航

| 方法 | 条件 | 精度 |
|------|------|------|
| 北极星（北半球） | 需要夜间、晴朗天空 | 高 |
| 太阳方位 | 日出东、日落西 | 中 |
| 树木年轮（南疏北密） | 只适用于北半球 | 中 |
| 河流走向 | 沿河通常能到达人烟 | 中 |
| 标记法 | 出发时归零里程 | 取决于记录 |

### 急救

| 情况 | 处理方法 | 常见误区 |
|------|---------|---------|
| 止血 | 压迫/抬高/止血带 | 止血带不能绑太久 |
| 骨折 | 固定（就地取材） | 不要复位 |
| 蛇咬 | 限制活动、标记咬点时间 | 切勿吸毒 |
| 脱水 | 补液（少量多次） | 一次喝太多会吐 |

## 四、"不舒适"的写作感官库

### 脱水

- 嘴唇干到舌头舔一下就裂开了
- 嗓子像在冒烟，吞口水都是疼的
- 三天没喝水，嘴里已经没有口水了，舌头发硬
- 咬了一口压缩饼干，干到差点噎死

### 寒冷

- 风像刀子一样割脸
- 手指冻到捏不住任何东西
- 金属表面黏住了皮肤，一扯就是一层皮
- 冷从脚底往上爬，像有东西在摸你的腿

### 炎热

- 地面烫得站不住，跳着走
- 空气像在烤箱里，吸一口气嗓子就疼
- 汗水蛰眼睛，擦了又冒
- 盔甲变成了移动烤箱

### 缺氧

- 火苗变小（蜡烛先灭）
- 太阳穴突突地跳
- 困倦感（不是想睡觉，是脑子变慢）
- 手脚发麻（血液集中在核心器官）

### 恐惧（身体反应）

- 嘴里一股铁锈味——紧张时咬破了舌头
- 腿软（血液流向四肢，准备逃跑）
- 冷汗——不是因为热，是身体在准备战斗
- 视野变窄（俗称"隧道视野"）
