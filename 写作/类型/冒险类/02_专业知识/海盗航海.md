# 海盗航海

## 核心：一艘船 + 一群性格各异的船员 + 危机四伏的大海 + 背叛与忠诚的博弈

### 五大元素

**大海——最大的角色。** 风平浪静（最危险——暴风雨前）/ 无风（帆船噩梦）/ 海雾（方向感丧失）/ 风暴（一切跟船作对）。

**船——流动的王国。** 家、堡垒、身份认同。新船光鲜有力但未经风浪。老船伤痕累累但可靠——每一道修补都是故事。沉没时读者哭了不是因为"失去了船"而是因为"失去了一段时光"。

**船员——流动的社会。** 没有出身等级 / 实力说话 / 自由高于一切 / 背叛是最大的罪。

**船上兄弟情不同于陆地友谊：** 因为你们分不开。吵架了明天还得在同一个甲板上工作。

**法外之地：** 让你觉得"这些家伙虽然不是好人，但比那些'合法'的坏人更真实"。

### 节奏模型

出航 → 航行（日常建立关系）→ 第一次危机 → 发现/到达 → 战斗/冒险 → 转折（背叛/被困）→ 绝境 → 最终突围 → 尾声

### 避坑：失败模式

- **海上生活只浪漫：** 坏血病、腹泻、孤独——这些"不酷"的细节才是真实的。
- **忽略航海技术：** 风向、洋流、帆的调整。
- **海战照搬陆战：** 船有惯性、风向影响机动性。
- **船员只是背景板：** 每个至少要有一句让人记住的台词。
- **海的"无聊"不是问题：** 连续无风无浪的无聊本身就有戏剧张力。

### 海战的独特戏剧性

**海战vs陆战的本质区别：** 延迟。转舵要等船响应，船有惯性，风有方向。

**三个阶段：** 接近（紧张克制，对话推进）→ 交火（短句，动词密集，感官爆炸）→ 接舷（回到陆战节奏，但多了"脚下不稳"）。

**示范：** "炮弹打中右舷。先是一声闷响——不是'轰'那种电影响——是沉闷的'嘭'，像巨人在船肋骨上踢了一脚。'左满舵！'舵手把舵轮打死。船开始转——不是立刻转——先是船身呻吟，然后帆吃到了新的风向，船头一寸一寸地偏。'现在！右舷炮——开火！'六门炮同时怒吼。整条船往后一退。"

### 风暴写法

**风暴前：** "空气变了。不是温度——是'密度'。海面从蓝变灰绿。鸟不见了。"

**风暴中：** "世界变成灰色。海和天分界线消失。雨横着打过来。他抓着缆绳——手没知觉了——不能松手。松手就被甩出去。"

**风暴后：** "风停了——突然的，像关了开关。松开缆绳。手指弯不回来了。有人在呻吟。有人在喊名字。有人不回答。"

### 角色模板

**魅力浪子船长：** 从不下死命令都是"我觉得这样比较好"——但所有人最后跟着他走。

**铁血军人船长：** 纪律+勇气+谋略+对船员的责任+孤独。

**复仇者船长：** 被"失去"扭曲了的好人。

**老水手：** 天气预测比仪器准（"我膝盖疼了，明天要下雨"）。

**厨师：** "别问我做的什么。能吃就行。在海上你没有挑食的权利。"

### 跨作品规律

- 船长的魅力决定一切
- 大海必须有"人格"
- 船上日常就是最好的角色塑造——谁做饭、谁值夜、谁晕船
- 海战核心不是火力是策略
- 自由是海盗故事的终极主题
