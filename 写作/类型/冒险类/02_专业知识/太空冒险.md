# 太空冒险

## 核心：航海冒险在三维空间的延伸 — 人类面对浩瀚未知时的渺小与勇气

两种方向：软科幻太空歌剧（人物为主，科技是背景）vs 硬科幻太空探险（科学可能性为主）。

### 五大元素

**飞船——太空中的孤岛。** 家、交通工具、战斗平台、社会缩影。

**宇宙环境：** 无声战斗（无空气传声）/ 三维空间（任何方向攻击）/ 时间膨胀（接近光速回家发现亲友已老去）/ 通讯延迟（距离越远延迟越大）。

**外星文明设计光谱：** 完全不可理解 / 类人但不同 / 敌对 / 共生 / 超越。真正好的外星设计是"你能理解他们在做什么但不理解为什么"——不是"长得奇怪的人类"。

**科技与伦理困境：** 克隆人/AI的权利、冷冻睡眠中的人是否还是"人"、拯救多数还是牺牲少数。

**孤独与距离：** 回家可能需要几十年。出了事没有人能来救你。这种"绝对的孤独"是太空冒险独有的情感底色。

### 节奏模型

出发 → 航行（太空日常/角色关系）→ 发现 → 接触 → 冲突 → 危机 → 抉择（伦理困境）→ 高潮 → 返回（或发现回不去了）

### 避坑：失败模式

- **科技万能：** 技术问题要有真实困难感
- **外星人只是怪皮：** 换层皮就不是外星人了
- **太空的"大"没写出来：** 如果五分钟就从A星系飞到B星系，"大"没有意义
- **科幻写成魔法：** 科技要有逻辑和限制
- **人类中心主义：** 不是所有外星故事都要以"人类拯救宇宙"结尾

### 跨作品规律

- **最好的太空冒险不在于"宇宙有多大"而在于"人在宇宙中有多渺小但依然在尝试"**
- **飞船内部的政治比星际政治更有趣。** 一条船上几十个人——权力关系、信任危机
- **技术故障比外星敌人更可怕。** 氧气系统坏了——"纯粹的物理问题"比任何战斗都窒息
- **太空的沉默本身就是美学。** 战舰爆炸但听不到声音——无声的毁灭比震耳欲聋更震撼

### 技术故障的写法

**氧气系统：** "'按当前消耗速度，还有47小时空气。'最近救援船在72小时之外。差17小时。呼吸开始变得有意识——每一口都带着'这是有限的'这个念头。之前从没注意过自己的呼吸。现在每一次都在提醒——少了一点。又少了一点。"

### 通讯与孤独

**通讯延迟：** "他说了一句话，八分钟后对方才听到。对方说一句话，八分钟后他才听到。一来一回十六分钟。在这种条件下——'对话'变成了'交换日记'。你只能对着黑暗说话，然后等待黑暗的回答。"

**绝对的孤独：** "飞船上有四十个人。但他觉得孤独。不是'没人说话'的孤独——是'就算喊了也没人能听到'的孤独。外面有无限空间但你永远不能出去。就像鱼缸里的鱼。"

### 时间膨胀的写法

"他看了看时钟：航行了四年零三个月。走出舱门。接机的人——是他女儿。但她不是十岁小女孩了。三十多岁了。旁边站着两个孩子——他的外孙。'爸爸。'他张了张嘴。'你长大了。'这是他能想到的唯一一句话。"

### 角色模板

**舰长：** 两难——"所有人活着回去但放弃任务"vs"完成任务但有人回不去"。

**科学家：** 40%好奇心 + 25%理性 + 15%执着 + 10%社交障碍 + 10%隐藏恐惧。

**普通人：** 代表"正常人在太空极端环境中的真实反应"。

**AI/机器人：** 伙伴/威胁/灰色地带。核心问题："它有感情吗？它算生命吗？"

**工程师：** "给我三十分钟能修好任何东西。如果不行——给我一个小时。"

**角色关系特殊性：** 封闭空间——不能"离开"；时间膨胀——变老了我没变——不对称是最残忍的关系杀手。太空中的爱情——脆弱也珍贵——但如果一个人老了另一个人没老——爱情在时间面前毫无胜算。

### 写作示范

**穿越小行星带：** "'小行星带。'舵手说。透过舷窗——远处有东西在漂浮——大小不一——从拳头到房子——缓缓旋转。飞船开始机动。急速下降——从两块小行星之间的缝隙穿过去——近到能看见岩石表面的纹理。一块碎片擦过船体——'嚓'——一声轻响。三分钟后穿过。太空重新空旷。他注意到舵手的手在微微发抖——刚才那三分钟里一点都没抖。"
