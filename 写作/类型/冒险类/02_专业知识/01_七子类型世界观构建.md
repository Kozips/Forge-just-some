# 冒险类 — 七子类型世界观构建指南

> 构建世界观时参考。每个子类型给出核心设定模板和"冰山原则"操作方法。

---

## 一、七子类型核心设定速查

### 1. 探险寻宝型

**核心驱动力**：好奇心 + 财富欲

**世界观三要素**：

| 层次 | 内容 | 写法 |
|------|------|------|
| 表面层 | 金银财宝 | 给读者期待，但不要太早展示 |
| 功能层 | 改变命运的力量 | 推动剧情的核心道具 |
| 象征层 | 自由 / 真相 / 救赎 | 主题升华的载体 |

**藏宝地点危险公式**：
```
危险系数 = 物理危险 × 心理压力 × 时间限制
```

**线索链公式**：
```
1个总线索 → 3-5个分线索 → 每个分线索需要一个小代价 → 最终指向总宝藏
```

**世界观冰山写法**：
- 你写了：古墓的入口、机关、宝藏的外观
- 你没写但让读者感觉到：这座墓存在了多少年、建造它死了多少人、为什么会被遗忘

---

### 2. 奇幻冒险型

**核心驱动力**：探索 + 力量 + 羁绊

**魔法/能力体系两种模式**：

| 硬系统 | 软系统 |
|--------|--------|
| 规则明确、可推演 | 神秘、不可解释 |
| 适合策略战斗 | 适合史诗氛围 |
| 例：念能力、炼金术 | 例：原力、甘道夫魔法 |

**成长阶段公式**：
```
觉醒（发现能力）→ 学习（掌握基础）→ 失败（遇到瓶颈）
→ 代价（付出代价突破）→ 精通（但发现还有更高）
```

**世界构建五层**：

| 层级 | 内容 | 写法占比 |
|------|------|---------|
| 物理规则 | 魔法如何运作 | 只在关键场景展示 |
| 地理空间 | 地图结构（线性/中心辐射/岛屿制/多层） | 写3-5个重要地点足够 |
| 社会制度 | 学院/宗派/王国/地下组织 | 只呈现跟主角相关的部分 |
| 历史传说 | 过去的重大事件 | 通过NPC对话暗示 |
| 日常生活 | 吃穿住行 | 用细节让世界"活" |

**世界观冰山写法**：
- 你写了：主角第一次施法成功
- 你没写但让读者感觉到：这套魔法体系存在了多久、有多少人练过、有多少种用法

---

### 3. 生存冒险型

**核心驱动力**：活下去

**环境威胁清单**：

| 环境类型 | 首要威胁 | 次要威胁 | 生存关键 |
|---------|---------|---------|---------|
| 荒岛 | 淡水 | 食物/孤独 | 资源管理 |
| 丛林 | 方向/疾病 | 毒虫/野兽 | 知识 + 工具 |
| 沙漠 | 脱水 | 暴晒/迷失 | 水源 + 时间 |
| 雪山 | 失温 | 雪盲/雪崩 | 庇护所 + 火 |
| 深海 | 水压/氧气 | 海洋生物 | 装备 + 冷静 |

**身体衰败时间表（无补给）**：

| 时间 | 症状 |
|------|------|
| 6小时 | 口渴、饥饿（正常反应） |
| 24小时 | 头晕、注意力下降、开始头疼 |
| 48小时 | 肌肉酸痛、判断力下降、幻觉出现 |
| 72小时 | 虚弱到行动困难、谵妄症状 |
| 5天+ | 器官衰竭、意识模糊 |

**道德困境类型**：
- 有限的水自己喝还是分给同伴？
- 同伴受伤拖累行进速度：抛弃还是坚持？
- 食物只够一个人吃：怎么分？

**世界观冰山写法**：
- 你写了：主角在荒岛上找到了一处淡水源
- 你没写但让读者感觉到：这个岛的生态系统是怎么运作的、为什么这里有淡水源

---

### 4. 海盗航海型

**核心驱动力**：自由 + 冒险

**船只功能分区**：

| 区域 | 功能 | 叙事用途 |
|------|------|---------|
| 舰桥 | 控制方向 | 决策场景 |
| 甲板 | 战斗/日常 | 群戏、打斗 |
| 船舱 | 生活/储藏 | 私下对话、密谋 |
| 底舱 | 引擎/压舱 | 最危险的地方 |
| 瞭望台 | 观察预警 | 发现新情况 |

**船员分工公式**：
```
船长（决策）+ 大副（执行）+ 舵手（方向）
+ 水手长（纪律）+ 厨师（士气）+ 船医（生存）+ 瞭望手（预警）
```

**海战三要素**：
```
风向（谁占优势）+ 船速（谁追得上）+ 火炮（谁能打到）
```

**海的状态六种**：

| 状态 | 叙事节奏 |
|------|---------|
| 风平浪静 | 暴风前的宁静 |
| 微风 | 理想航行 |
| 大风 | 速度↑ 控制↓ |
| 风暴 | 全力对抗 |
| 无风 | 帆船噩梦（被困） |
| 海雾 | 迷失（恐怖感） |

**世界观冰山写法**：
- 你写了：船长和水手长在甲板上的一次争吵
- 你没写但让读者感觉到：船上的等级制度、海上的规矩、船员们的过去

---

### 5. 太空冒险型

**核心驱动力**：好奇心 + 孤独

**飞船区域设计**：

| 区域 | 功能 | 叙事用途 |
|------|------|---------|
| 舰桥 | 指挥 | 决策场景、对外联络 |
| 生活区 | 休息/社交 | 角色互动、日常 |
| 工程区 | 引擎/维护 | 故障场景 |
| 医疗舱 | 治疗/研究 | 危机场景 |
| 货舱 | 物资/潜在威胁 | 意外发现 |
| 气闸 | 对外接口 | EVA、危险入口 |

**太空危险清单**：

| 危险 | 效果 | 紧迫程度 |
|------|------|---------|
| 真空暴露 | 15秒意识丧失 | 极高 |
| 辐射 | 长期→癌症 | 长期 |
| 失重 | 肌肉萎缩/骨质流失 | 中长期 |
| 温度极端 | +120°C / -150°C | 即时 |
| 陨石碎片 | 高速撞击 | 即时 |
| 氧气耗尽 | 窒息 | 极高 |

**太空独有叙事空间**：

| 特性 | 叙事价值 |
|------|---------|
| 无声战斗 | 太空无法传声，战斗变成无声的视觉奇观 |
| 三维空间作战 | 上下左右都能移动，战术更复杂 |
| 时间膨胀 | 接近光速时时间变慢，回家发现一切已变 |
| 通讯延迟 | 火星到地球最短3分钟，无法实时求援 |

**世界观冰山写法**：
- 你写了：飞船的人工重力突然消失
- 你没写但让读者感觉到：这套生命维持系统的复杂性、飞船设计者的考量

---

### 6. 地下城探险型

**核心驱动力**：探索 + 战利品

**层级递增公式**：

| 层数 | 怪物强度 | 陷阱复杂度 | 宝物价值 | 心理压力 |
|------|---------|-----------|---------|---------|
| 1层 | 低 | 简单 | 一般 | 低 |
| 2层 | 中低 | 中等 | 好 | 中 |
| 3层 | 中 | 复杂 | 稀有 | 高 |
| Boss层 | 高 | 机制型 | 传说级 | 极限 |

**Boss战三阶段**：

| 阶段 | 内容 |
|------|------|
| 试探 | 了解攻击模式 |
| 应对 | 找到弱点 |
| 变招 | 新机制/狂暴阶段 |

**陷阱分类**：

| 类型 | 特点 | 写作技巧 |
|------|------|---------|
| 即时型 | 踩上即触发 | 用声音/视觉暗示 |
| 延迟型 | 给虚假安全感 | 触发后才显露真面目 |
| 连锁型 | 一触触发一串 | 用环境描写铺垫连锁逻辑 |
| 环境型 | 区域本身有毒/危险 | 用感官描写让读者"感受"到危险 |
| 心理型 | 幻觉/幻术 | 用对话/记忆制造不确定性 |

**战利品层次**：
```
消耗品 → 装备 → 稀有物品 → 知识（地图/书籍/情报）→ 最终宝物
```

**世界观冰山写法**：
- 你写了：第5层的Boss房间
- 你没写但让读者感觉到：这个地下城是谁建的、为什么而建、存在了多久

---

### 7. 公路冒险型

**核心驱动力**：逃离 + 自我发现

**旅途三段式**：

| 阶段 | 位置 | 主题 | 同伴关系 |
|------|------|------|---------|
| 起点 | 出发地 | 离开的原因 | 互相试探 |
| 中点 | 旅途中 | 经历的考验 | 冲突 + 了解 |
| 终点 | 目的地 | 到达的意义 | 理解 / 改变 |

**沿途遭遇公式**：
```
每一站 = 一个有故事的人 + 一个意外事件 + 一个推动角色的信息/物品/决定
```

**交通工具气质对照**：

| 工具 | 气质 | 适合的故事类型 |
|------|------|--------------|
| 轿车 | 速度/自由 | 逃离/追凶 |
| 摩托车 | 叛逆/孤独 | 公路/流浪 |
| 房车 | 家庭/缓慢 | 公路/和解 |
| 卡车 | 蓝领/广阔 | 公路/见闻 |
| 自行车 | 慢但真实 | 成长/自我发现 |
| 步行 | 最原始/最亲密 | 心灵之旅 |

**车是移动的密室**：
你跟同行者不能离开。这种"被迫亲近"是最佳角色塑造工具。在其他冒险类型里，伙伴可以走散；在公路冒险里，矛盾必须被解决。

**世界观冰山写法**：
- 你写了：加油站便利店老板的一句话
- 你没写但让读者感觉到：这个地区的经济状况、路过这里的人的故事

## 二、世界观的"冰山理论"操作指南

**你写的只有10%**：
- 完整的历史年表 → 只透露几个关键事件
- 详细的地理图 → 只写几个重要的地点
- 复杂的政治体系 → 只呈现跟主角相关的部分
- 完整的魔法/科技规则 → 只展示在关键场景

**你没写的90%让读者"感觉到"**：
- 通过NPC的对话暗示历史
- 通过环境细节暗示地理
- 通过选择暗示道德困境
- 通过成功/失败暗示规则

**核心原则**：世界观是为故事服务的，不是故事的负担。读者不需要知道一切——只需要知道"这个故事发生在一个有重量、有细节的世界"。

## 三、七子类型速查总表

| 类型 | 核心驱动力 | 最大敌人 | 叙事节奏 | 读者爽点 |
|------|-----------|---------|---------|---------|
| 探险寻宝 | 好奇心+财富 | 机关+竞争者 | 闯关制 | 解谜的智识快感 |
| 奇幻冒险 | 探索+力量 | 黑暗势力 | 成长制 | 伙伴情+战斗爽 |
| 生存冒险 | 活下去 | 自然+身体极限 | 螺旋下降 | 小胜利的巨大满足 |
| 海盗航海 | 自由+冒险 | 大海+背叛 | 浪涌式 | 船长魅力+自由浪漫 |
| 太空冒险 | 好奇心+孤独 | 未知+故障 | 探索制 | 宇宙浩瀚感 |
| 地下城探险 | 探索+战利品 | 怪物+陷阱 | 层级制 | 策略性+奖励感 |
| 公路冒险 | 逃离+自我发现 | 沿途意外+自身 | 串珠式 | 角色变化满足 |

**"该选哪种类型"快速决策**：

- 想写探索未知 → 探险寻宝 / 奇幻冒险 / 太空冒险
- 想写人跟自然对抗 → 生存冒险
- 想写自由与反叛 → 海盗航海 / 公路冒险
- 想写打怪升级 → 地下城探险 / 奇幻冒险
- 想写人际关系+内心成长 → 公路冒险 / 生存冒险
- 想写宏大世界观 → 奇幻冒险 / 太空冒险
- 想写紧张刺激 → 探险寻宝 / 地下城探险 / 生存冒险
- 想写浪漫与诗意 → 海盗航海 / 公路冒险
