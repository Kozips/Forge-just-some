# 冒险类 — 七子类型世界观构建速查

> 每个子类型给出核心设定模板和"冰山原则"操作方法。

## 七子类型核心要点

### 1. 探险寻宝型
**核心驱动力：** 好奇心 + 财富欲
**线索链：** 1个总线索 → 3-5个分线索 → 每个需要小代价 → 最终指向总宝藏
**冰山写法：** 你写了古墓入口/机关/宝藏外观——没写但让读者感受到墓存在了多少年、为什么被遗忘。

### 2. 奇幻冒险型
**核心驱动力：** 探索 + 力量 + 羁绊
**魔法体系：** 硬系统（规则明确/适合策略战斗）vs 软系统（神秘/适合史诗氛围）
**成长阶段：** 觉醒 → 学习 → 失败 → 代价突破 → 精通
**冰山写法：** 你写了主角第一次施法——没写但让读者感受到这套魔法体系存在了多久。

### 3. 生存冒险型
**核心驱动力：** 活下去
**道德困境：** 有限的水——自己喝还是分给同伴？同伴拖累——抛弃还是坚持？
**冰山写法：** 你写了找到淡水源——没写但让读者感受到这个岛的生态运作。

### 4. 海盗航海型
**核心驱动力：** 自由 + 冒险
**船的功能分区：** 舰桥(决策)/甲板(战斗)/船舱(私下对话)/底舱(最危险)/瞭望台(预警)
**冰山写法：** 你写了船长和水手长争吵——没写但让读者感受到船的等级制度、海上规矩。

### 5. 太空冒险型
**核心驱动力：** 好奇心 + 孤独
**太空独有叙事空间：** 无声战斗 / 三维空间作战 / 时间膨胀 / 通讯延迟
**冰山写法：** 你写了重力消失——没写但让读者感受到生命维持系统的复杂性。

### 6. 地下城探险型
**核心驱动力：** 探索 + 战利品
**层级递增：** 层数↑ → 怪物↑/陷阱↑/宝物↑/压力↑
**战利品层次：** 消耗品 → 装备 → 稀有物品 → 知识 → 最终宝物
**冰山写法：** 你写了第5层Boss房间——没写但让读者感受到地下城是谁建的、为什么而建。

### 7. 公路冒险型
**核心驱动力：** 逃离 + 自我发现
**沿途遭遇公式：** 每一站 = 有故事的人 + 意外事件 + 推动角色的信息/物品/决定
**车是移动的密室：** 不能"离开"——矛盾必须被解决。
**冰山写法：** 你写了加油站老板的一句话——没写但让读者感受到这个地区的经济状况。

## 世界观冰山理论

**你写的只有10%：** 完整年表→只透露关键事件；详细地图→只写重要地点；复杂政治→只呈现跟主角相关的部分；完整规则→只在关键场景展示。

**你没写的90%让读者"感觉到"：** 通过NPC对话暗示历史、通过环境细节暗示地理、通过选择暗示道德困境、通过成功/失败暗示规则。

**核心原则：** 世界观是为故事服务的。读者不需要知道一切——只需要知道"这个故事发生在一个有重量、有细节的世界"。

## 速查总表

| 类型 | 核心驱动力 | 最大敌人 | 叙事节奏 | 读者爽点 |
|------|-----------|---------|---------|---------|
| 探险寻宝 | 好奇心+财富 | 机关+竞争者 | 闯关制 | 解谜的智识快感 |
| 奇幻冒险 | 探索+力量 | 黑暗势力 | 成长制 | 伙伴情+战斗爽 |
| 生存冒险 | 活下去 | 自然+身体极限 | 螺旋下降 | 小胜利的巨大满足 |
| 海盗航海 | 自由+冒险 | 大海+背叛 | 浪涌式 | 船长魅力+自由浪漫 |
| 太空冒险 | 好奇心+孤独 | 未知+故障 | 探索制 | 宇宙浩瀚感 |
| 地下城探险 | 探索+战利品 | 怪物+陷阱 | 层级制 | 策略性+奖励感 |
| 公路冒险 | 逃离+自我发现 | 沿途意外+自身 | 串珠式 | 角色变化满足 |

**快速决策：** 探索未知→寻宝/奇幻/太空；人跟自然对抗→生存；自由反叛→海盗/公路；打怪升级→地下城/奇幻；人际关系+内心成长→公路/生存；宏大世界观→奇幻/太空；紧张刺激→寻宝/地下城/生存；浪漫诗意→海盗/公路。
