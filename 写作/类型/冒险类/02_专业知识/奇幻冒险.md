# 奇幻冒险

## 核心：被选中的普通人 + 充满规则的魔法世界 + 从菜鸟到英雄的成长 + 伙伴 + 强敌

### 世界观五层次

1. **物理规则**——硬魔法（规则明确/有代价/适合策略战斗）vs 软魔法（神秘/不解释/适合史诗）。最怕半硬不软：前面定规则后面自己打破。
2. **地理空间**——线性/中心辐射/岛屿制/多层
3. **社会制度**——学院/宗派/王国/地下组织
4. **历史与传说**——上古大战/失落文明/预言
5. **日常生活**——货币/食物/交通/娱乐/节日

**冰山呈现法：** 角色不会解释常识。读者通过主角的"不知道"慢慢理解世界。
- ❌ "念分为六大系统……"
- ✅ 观众喊"放出系的！"主角一脸懵但不问。

**规则展示而非规则讲解：** 主角想放火球，掌心火苗被雨水浇灭——读者自己推出"雨天克火"。

### 成长阶段

觉醒 → 入门（学规则/交伙伴/初战）→ 试炼（失败/背叛/失去/发现能力局限）→ 蜕变（心态转变大于招式升级）→ 决战（胜利方式呼应主题）

### 伙伴"入队仪式"

相遇（冲突）→ 了解（发现过去和痛点）→ 共同战斗 → 认可（"我要跟你一起走"的moment）

### 战斗三层次

第一层物理——"他在干什么"；第二层策略——"他在想什么"；第三层情感——"他为什么打"。

**示范：** "索隆的第三刀被弹开了。（物理）那家伙的霸气比上次强了——不是强了，是精准了。（策略）'你以为只有你在变强吗？'索隆咬住刀柄。左眼伤疤在发疼——不是耻辱，是提醒。（情感）"

### 伙伴互动

**❌** "他看着受伤的伙伴，心里涌起暖流。他终于明白了，这才是真正的友情。"
**✅** "他把外套脱下来盖在对方身上。走到洞口坐下来，背对着里面，手放在刀柄上。整个晚上都没合眼。"

### 力量提升

三种方式：外部触发（新老师/武器）/ 内部领悟（生死关头）/ 代价换取（最有冲击力）。

**四条铁律：** 升级前必须先失败；升级要有代价；升级不能解决所有问题；升级时机要跟情感高潮重合。

### 伏笔与节奏

**三段式伏笔：** 第一次不起眼的背景 → 第二次读者开始注意 → 第三次真相揭开恍然大悟。

**节奏呼吸感：** 战斗→日常→探索→战斗→日常→大战。大战后安排1-2章轻松日常。

### 避坑：失败模式

- **魔法万能：** 必须有有限制和代价
- **世界观一次全交代：** 慢慢揭露
- **伙伴凑人数：** 每个都要有独立故事线和战斗场景
- **力量提升太随意：** 每次变强要有合理原因
- **"被选中"是唯一解释：** 胜利靠选择和努力
- **后期崩设定**

### 角色模板

**热血笨蛋：** 缺点——冲动、不顾后果、因天真伤害别人。弧线："我要变强"→"强不是一个人的事"→"为了保护重要的人我可以付出一切"。

**冷面天才：** 关系公式：互相看不惯 → 并肩认可 → 理念不同分道扬镳 → 最终决战或和解。

**被选中的普通人：** 靠"选择"而不是战斗力改变世界。

**导师：** 严师型（从不夸奖但默默关注，"最后一课"退出或牺牲）/ 疯老头型（表面不正经实际最强，教育方式是"你自己去经历"）。

**反派：** 野心家（"曾经也是好人"的堕落过去）/ 理想主义（目标可能是"好的"但手段极端——"他说的好像有道理？"）。

### 写作示范

**第一次进入精灵森林：** "当最后一棵普通的树被甩在身后，世界突然安静了。不是'声音变小了'——是某些声音消失了。光线变了——阳光被无数层叶片过滤，分解成细碎的金色微粒缓缓飘落。然后他看到第一棵树。'树'这个词不够。那是一个生物。树干粗到二十个人合不住，根从地面拱起形成天然拱门。拱门后面，有什么在发光。他知道，从这棵树开始，前面的规则不一样了。"
