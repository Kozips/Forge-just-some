# 探险寻宝

## 核心：闯关制 — 线索链 → 危险旅途 → 一系列考验 → 宝藏 → 代价

### 必备要素

**线索系统：** 地图类/文字密文/器物类/口述类/碎片推理类。每个线索的获取都要付出代价。最忌"一张图走到底"。

**机关设计四原则：** ①规则先于机关出现 ②有时间窗口才是真威胁 ③用感官替代解说（齿轮声、震动、霉味） ④机关考验角色弱点。范例："三秒内必须站到正确位置"——比"无数箭矢射出"强十倍。

**竞争者：** 专业竞争者/守护者/队伍内叛徒（前期伪装队友后期暴露）/官方势力。

**宝物三层：** 表面（金银财宝）→ 功能（长生药/终极武器）→ 象征（真相/自由/亲情）。必须落到象征层，否则十章铺垫只是一堆金币。

### 节奏模型

发现线索 → 组建团队 → 第一道关卡 → 遭遇对手 → 深入险境（裂痕）→ 重大转折/背叛 → 最终考验 → 揭开真相 → 尾声

### 避坑：失败模式

- **太容易拿到：** 第三章找到宝藏后面写什么？
- **机关不揭示什么：** 好机关揭示真相或考验角色弱点。
- **对手太蠢：** 他们应该有自己的动机和能力。
- **没有代价：** 白拿的东西没人珍惜。
- **文化不对：** 埃及写玛雅机关让读者出戏。
- **揭晓虎头蛇尾：** 意义必须配得上旅途的分量。
- **假线索太假：** 红鲱鱼要比真线索更像真——揭穿后角色是恍然大悟而不是觉得被耍。

### 悬念管理

**"二次发现"技巧：** 同一线索不同阶段不同解读——第一次是装饰，第二次是警告，第三次是逃生路线图。

**章尾钩子范例：**
- ✅ "他转过拐角，看到了不该在这里的人。" / "棺盖缓缓移开，里面是空的。不，不是空的——"
- ❌ "他们继续往前走。" / "又过了一个小时，他们终于到了。"

### 角色模板

**硬核冒险家：** 两门外语（古语更好）、绝境应变能力、某种非理性恐惧。台词："这地方风水不好。""为什么？""因为每根柱子上都刻着'死'。"

**被卷入的普通人：** 前期不能太强——菜鸟阶段越真实，后期成长越有说服力。

**老手隐退：** "这个墓……我二十年前来过一次。那次出来的只有我一个。"

**反派：** 财阀型（致命弱点：自负/偏执）/ 狂热学者型（让读者觉得"如果我是他也许也会这么做"）/ 守护者型（前期阻止，后期发现他是对的）。

### 写作示范

**专家对话：**
- ❌ "这是西汉中期青铜鎏金错银博山炉，采用了失蜡法铸造……"
- ✅ "看这个，"老胡翻个面，"西汉的。你闻闻，两千年的铜锈味。"

**紧张时刻短句：** "退后！"老胡一把拉住吴邪。石墙裂开一条缝。"有暗室。"

**宝物的写法：**
- ❌ "箱子打开，里面是金光闪闪的珠宝。"
- ✅ "那颗红宝石本身好像在呼吸，像一颗沉睡的心脏，缓慢地、微弱地、一明一暗地脉动着。"

**古墓开门：** "石门被推开一条缝。一股气流涌出来——不是风，是积蓄了千年的空气突然找到出口。带着浓重的霉味和一种说不出的甜腻。穹顶高得手电照不到头。四根石柱……蛇，很多蛇，盘旋而上。大厅中央是一个石台，上面放着一个箱子。所有人都停下了呼吸。"

**沙暴遗迹：** "能见度在三秒内从一百米降到一米。他闭着眼爬——手掌按在沙子上，感觉到下面有硬的东西。太平整了。不是石头。指尖碰到一道凹槽——人工雕刻的。沙暴持续四十分钟。一座石头拱门露了出来。拱门上方刻着一行字。笔记本里——父亲的笔迹——画着一模一样的符号。旁边只写了两个字：'入口。'"
