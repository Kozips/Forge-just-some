## 一、场景描写素材

### 1. 密林迷路

**适用**：团队在陌生环境中失去方向

> 树冠把天空遮得严严实实，连太阳在哪都看不出来。脚下的落叶又湿又滑，每一步都要抓着旁边的藤蔓才不会摔倒。走在最前面的老周突然停下来——前面的地上有一排脚印，是他们自己半小时前留下的。他们绕了一圈，又回来了。

**五感**：潮湿的腐叶味 / 鸟叫突然安静 / 皮肤上虫子爬过的触感

### 2. 发现遗迹

**适用**：角色发现未知文明或古代建筑

> 石墙上刻满了看不懂的符号，像字又像画。小林用手电照了一圈，光柱扫过的地方，壁画上的"人"全都面朝同一个方向——一个圆形的门。门是开着的。从里面吹出来的风带着一股甜腻的味道，像烧过的糖。

**五感**：石壁的冰凉 / 回声在狭窄通道中放大 / 甜腻的风

### 3. 绝境求生

**适用**：资源耗尽、体力极限

> 水壶空了。他摇了摇，只有几滴沾在嘴唇上。嘴唇已经裂开了，舌头像砂纸。前方是一片碎石坡，没有遮阴，太阳直直地砸下来。他蹲在一块石头后面，把帽子盖在脸上，想：再走十分钟。就十分钟。

### 4. 古墓主墓室开门

**适用**：推开千年封闭空间的门

> 石门被推开一条缝。一股气流从缝隙里涌出来——不是风，是积蓄了千年的空气突然找到了出口。那股气带着浓重的霉味和一种说不出的甜腻。门缝越来越大。手电筒的光挤进去，照亮了门后的空间。先是光——光打在什么东西上反射回来，晃了一下眼睛。然后是声音——他们的声音在门后变成了回声，被放大、拉长、扭曲。大厅中央是一个石台。石台上放着一个箱子。所有人都停下了呼吸。

**五感**：霉味+甜腻（嗅觉）/ 回声扭曲（听觉）/ 光反射（视觉）

### 5. 风暴前夕的海上

**适用**：航海类危险铺垫

> 风变了。不是方向——是"质地"。之前的风是均匀的——现在变得断断续续——一阵强一阵弱——像是大海在深呼吸前的犹豫。西边的天际线——半小时前还是金色的晚霞——现在是一堵墙。灰色的、翻滚的、正在以看得见的速度向这边推过来。空气变了。温度降了两度。皮肤上的汗毛竖起来了。

**五感**：风的质感变化（触觉）/ 天际线颜色变化（视觉）/ 温度下降（触觉）

### 6. 失重中的飞船

**适用**：太空冒险的日常故障

> 人工重力系统出故障的瞬间——他首先感觉到的不是"飘起来"而是"胃在翻"。内耳的平衡系统失灵了——大脑告诉他你在坠落——但眼睛说你在原地漂浮。杯子从桌上飘起来。水从杯子里飘出来——变成一个球形——晶莹的、颤动的、像一个活的水晶球。他本能地伸手去接——手碰到水球——水球碎成无数小珠子在空中飘散。

**五感**：胃的翻涌（内脏感觉）/ 水球视觉（视觉）/ 失重体感（触觉）

## 二、对话素材

### 1. 组队——用技能展示代替自我介绍

**技法要点**：冒险角色介绍自己靠做不靠说。每人一个标志性动作。

> "先自我介绍吧。"
> "程逸。盗贼。"年轻人从袖子里甩出三枚硬币，叮叮叮——分别钉在桌子三个角上。"我手快。"
> 对面的女人把硬币拔下来，翻面看了看。"纯银的。你偷的？"
> "借的。"
> "借了没还吧。"女人把硬币扔回去，"苏暮。弓箭手。"
> 她没展示什么。但桌上的人都注意到了——她的右手食指和中指第一节磨出了厚厚的茧。十几年的茧。

### 2. 被困荒岛——务实决策对话

**技法要点**：生存场景下每句话都在解决问题，没有抒情空间。

> "情况。"他掰着手指，"淡水——三个人分一壶，最多撑两天。食物——没有。救援——不知道什么时候来。"
> "有没有可能是永远不会来？"程逸看着海面。
> "那就不想了。"沈括站起来，"先解决水的问题。"
> "为什么不是找食物和留守？"程逸问。
> "因为人可以一周不吃东西，但三天不喝水就完了。"

### 3. 古墓机关前——多视角争论

**技法要点**：决策场景让每人用不同推理路径分析。

> "走左边。"程逸第一个开口，"'生'嘛，多简单。"
> "太简单了。"沈括蹲下来摸石门底部，"你看这个磨损——左边被人开过很多次。"
> "那就对了。前辈们都走左边。"
> "前辈们有没有活着出来你确定？"

### 4. 战前鼓励——用日常琐事代替口号

**技法要点**：越日常的叮嘱越像遗言。

> "没什么好说的。就一句——"
> "程逸。""在。""你欠我三顿酒钱。活着出来记得还。"
> "苏暮。""在。""你的箭——给我两支。""你又不用弓。""备用。万一你的弓弦断了，至少还有我能帮你扔。"
> "周棠。""干嘛。""你那把剑太重了。打Boss的时候注意走位，别硬扛。"
> 他用刀背敲了一下石门。石门上那张嘴裂开了。"别死。"

### 5. 幽默化解紧张

**技法要点**：越是危险的地方越要来点冷笑话——用幽默扛住恐惧。

> "我们可能要死了。"
> "可能？你什么时候变得这么乐观了。"

> "说个好消息和一个坏消息。"
> "先听坏的。"
> "我们要死了。"
> "好消息呢？"
> "死之前不用再赶路了。"

### 6. 沉默的分量

**技法要点**：用动作代替台词。用停顿代替对话。

> 他张了张嘴，最后只是把自己的水壶递了过去。

> 绳子只能承受一个人的重量。
> 他看了看她。她看了看他。
> 他把绳子的另一端系在了自己腰上。
> "你——"
> 他摇了摇头。笑了一下。
> 然后他松了手。

## 三、感官描写速查

### 视觉——广阔与危险

| 类型 | 词/句 |
|------|-------|
| 广阔 | 一望无际、地平线像被刀切过、天空大得让人发慌 |
| 危险 | 深不见底、裂缝像嘴巴一样张开、影子在动 |
| 美丽 | 光柱穿透树冠、水面碎成金子、雪地上一行孤零零的脚印 |
| 神秘 | 雾气中若隐若现的轮廓、刻痕还是新的、不该出现在这里的东西 |

### 听觉——寂静与轰鸣

| 类型 | 词/句 |
|------|-------|
| 寂静 | 安静得能听见自己的心跳、连虫子都不叫了 |
| 轰鸣 | 雷声像从地底传来、水声大到说话要用喊的 |
| 诡异 | 远处传来的金属摩擦声、脚步声但看不到人 |
| 生动 | 鸟群惊飞的声音像撕布、树枝折断的脆响 |

### 触觉——温度与质感

| 类型 | 词/句 |
|------|-------|
| 寒冷 | 风像刀子割脸、手指捏不住东西、金属黏皮肤 |
| 炎热 | 地面烫得站不住、空气像在烤、汗水蛰眼睛 |
| 粗糙 | 石壁刮破手掌、绳子磨出水泡、沙子灌进领口 |
| 湿滑 | 岩壁上一层滑腻的苔藓、脚踩上去就知道要摔 |

### 嗅觉——腐朽与新生

| 类型 | 词/句 |
|------|-------|
| 腐朽 | 潮湿的霉味、烂木头的甜腻味、尸骨的臭气 |
| 危险信号 | 硫磺味（火山）、焦味（火）、铁锈味（血） |
| 清新 | 雨后泥土的味道、高山上干净得发苦的空气 |
| 异域 | 从没闻过的花香、地下河水的矿物质味 |

### 味觉——生存的味道

| 类型 | 词/句 |
|------|-------|
| 饥渴 | 嘴唇干到舌头舔一下就裂、嗓子像在冒烟 |
| 满足 | 一口凉水下去整个人都活了、压缩饼干吃出了肉味 |
| 恐惧 | 嘴里一股铁锈味（紧张咬破了舌头）、胆汁反上来的苦 |
| 意外 | 空气里居然有甜味、水里有金属的味道 |

## 四、章尾钩子模板

> 写完一章后，从以下六种中挑一种做结尾。

1. **新信息**："他转过弯，看到了不该在这里的人。"
2. **未完成的动作**："他按下了按钮。"
3. **感官冲击**："血。到处都是血。"
4. **角色的决定**："'我去做一件事，'他说，'可能回不来。'"
5. **环境变化**："地面开始震动。"
6. **反转的暗示**："'这一切——'他的声音在发抖，'都不是真的。'"
