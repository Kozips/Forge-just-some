## 一、万能故事公式

```
出发（动机）+ 阻碍（环境+对手+自身）+ 行动（解决阻碍）+ 代价（每次行动都有）= 故事
```

主线公式：

1. 给读者一个"想去的地方"（目标/谜团/目的地）
2. 给读者一个"想跟着走的人"（主角）
3. 给主角一个"不得不去的理由"（不只是好奇心——是内心需求）
4. 让读者相信"路上会有危险"
5. 承诺"危险是值得的"（奖赏跟风险匹配）
6. 兑现承诺（让读者在高潮时得到满足）

## 二、通用节奏模板

| 位置 | 节奏 | 内容 | 字数参考 |
|------|------|------|---------|
| 0-10% | 建立 | 主角出场、目标/动机建立、规则/世界观暗示 | 3000-5000 |
| 10-25% | 出发 | 进入冒险、第一道关卡、初步成功 | 4000-6000 |
| 25-50% | 深入 | 难度升级、伙伴关系建立、外部压力增大 | 8000-12000 |
| 50-70% | 崩溃 | 重大挫折、信念动摇、团队分裂或最危险时刻 | 6000-8000 |
| 70-85% | 反击 | 找到解决方案、牺牲、最后的准备 | 4000-6000 |
| 85-100% | 高潮+收束 | 最终对决/最终选择/真相揭晓 | 3000-5000 |

**快慢交替公式**：慢 → 快 → 慢 → 更快 → 慢 → 最快 → 慢（收尾）。永远不要连续两个"快"章节。

**1个高潮章节需要3-5个日常/铺垫章节支撑。**

## 三、开头五种模板

1. **悬崖开场**：第一句话把读者扔进危险里。"绳子断的时候，他离地面还有四十米。"
2. **感官开头**：从一个强烈的感官印象开始——声音、气味、温度。
3. **对话开头**：一句有信息量的台词。让读者想知道"谁在说？为什么？"
4. **倒叙引入**：只有一个人活着回来了。记者问：到底发生了什么？
5. **反常信息**：一张地图碎片被风吹到角色脚边，背面写着"别去"。

## 四、结尾四种形态

| 形态 | 特点 | 示例 |
|------|------|------|
| 圆满 | 目标达成+关系升华 | "他们成功了，而且变得更好" |
| 留白 | 开放式结局让读者思考 | "他到了——但那是另一个故事了" |
| 反讽 | 结局与表面不同 | "他拿到了宝藏——然后发现那不是他想要的" |
| 超越 | 超越最初的期待 | "他以为自己在找宝藏——其实是找自己" |

最高级的结尾：智识满足（谜题解开）+ 情感满足（角色关系升华）+ 主题满足（让读者思考）同时给到。

## 五、七种子类型专属节奏

### 探险寻宝

闯关制——一关比一关难。

```
发现线索 → 组建团队 → 第一道机关 → 遭遇竞争者 → 深入险境
→ 背叛/内部冲突 → 终极机关（团队配合）→ 宝藏揭晓
→ 代价（得到了什么，失去了什么）→ 新的线索
```

### 奇幻冒险

成长制——从菜鸟到英雄。

```
觉醒 → 入门仪式 → 结交第一个伙伴 → 第一场真正的战斗
→ 导师的教导 → 第一次失败（重要节点）→ 突破（新力量，有代价）
→ 伙伴的危机 → 最终boss战 → 新的旅程暗示
```

### 生存冒险

螺旋下降——希望和绝望交替。

```
灾难发生 → 第一次绝望 → 找到第一个资源（小胜利）→ 生火成功
→ 自然灾害摧毁一切 → 身体开始出问题 → 精神最低点
→ 意外发现（新的希望）→ 最后的挑战 → 获救（或者没有）
```

### 海盗航海

浪涌式——平静→风暴→平静。

```
扬帆起航 → 船员亮相 → 第一次海上危机 → 港口停靠
→ 第一次海战 → 船员冲突 → 发现/到达目的地
→ 夺宝/战斗 → 背叛 → 最后的航行
```

### 太空冒险

探索制——发现→接触→冲突。

```
出航 → 飞船日常 → 发现异常信号 → 第一次跟未知接触
→ 技术故障（生死攸关）→ 跟外星文明的冲突或沟通
→ 伦理困境 → 太空战斗 → 时间膨胀的冲击 → 返回
```

### 地下城探险

层级制——逐层推进。

```
入口 → 第一批小怪 → 第一个陷阱 → 发现战利品
→ 篝火/安全区 → 精英怪/中Boss → 隐藏房间
→ Boss战 → 通向下一层的入口 → 终极Boss+终极宝物
```

### 公路冒险

串珠式——每段路一个故事。

```
出发（离开的原因）→ 车内第一次对话 → 第一次沿途遭遇
→ 美丽风景停下来看 → 车抛锚/迷路 → 搭车客/路人
→ 同行者冲突 → 深夜对话（真相时刻）→ 和解/新理解
→ 到达目的地（空虚感——答案在路上）
```

## 六、子类型决策表

想写什么主题？选类型：

- "探索未知的地方" → 探险寻宝 / 奇幻冒险 / 太空冒险
- "人跟自然对抗" → 生存冒险
- "自由与反叛" → 海盗航海 / 公路冒险
- "打怪升级" → 地下城探险 / 奇幻冒险
- "人际关系+内心成长" → 公路冒险 / 生存冒险
- "宏大世界观" → 奇幻冒险 / 太空冒险
- "紧张刺激" → 探险寻宝 / 地下城探险 / 生存冒险
- "浪漫与诗意" → 海盗航海 / 公路冒险

## 七、角色公式

### 主角五种原型

| 原型 | 核心特征 | 成长方向 | 代表角色 |
|------|---------|---------|---------|
| 热血笨蛋 | 冲动、直觉强、感染力强 | "我最强" → "我们最强" | 路飞、鸣人 |
| 冷面天才 | 天赋高、内心有创伤 | "独自承担" → "信任他人" | 佐助、酷拉皮卡 |
| 普通人 | 没有特殊能力、被卷入 | "为什么是我" → "我能行" | 吴邪、查克 |
| 老手隐退 | 曾经的传奇、被迫重出 | "逃避过去" → "面对过去" | 胡八一、琼斯博士 |
| 追寻者 | 有明确目标、充满好奇 | "找到它" → "发现路上更重要" | 小杰、格瓦拉 |

### 伙伴入队仪式

1. 相遇（有冲突或有趣的初遇）
2. 了解（发现对方的价值/弱点/过去）
3. 并肩（第一次合作）
4. 认可（"我需要你"/"我要跟你一起走"）
5. 信任（把后背交给对方）

### 反派成立三条件

1. 有合理的动机（读者可以理解）
2. 有足够的威胁（不只是"强"——是"强到改变局面"）
3. 有自己的立场（不是纯粹的恶——是"另一种选择"）
