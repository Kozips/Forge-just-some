# 冒险类 — 结构模板

> 本目录存放章节结构模板与叙事骨架。写作前用模板规划节奏。

## 单章结构（探索-危机-脱险型）

```
### 发现/推进（前20%）
- 新地标/新线索/新环境出现
- 团队做出探索决策

### 深入（30%）
- 环境描写——壮丽或危险
- 小型障碍（需要技能/体力克服）

### 危机爆发（30%）
- 核心危险出现（地形塌方/野兽/迷路/内部矛盾）
- 团队分裂或有人掉队

### 脱险/推进（20%）
- 用前文铺垫的技能或物品解决危机
- 发现新线索，为下一章铺路
```

## 多章弧线（集结-考验-终局型）

```
第1-3章：组队出发 → 发现第一处遗迹/线索
第4-6章：深入未知区域 → 遭遇重大挫折（人/环境）
第7-8章：触底反弹 → 用智慧而非武力解决问题
第9-10章：到达终点 → 发现真相/宝藏不是想象中的东西
```

## 开篇钩子模板

**类型A — 悬念开场**：一张地图碎片被风吹到角色脚边，背面写着"别去"。
**类型B — 身处险境**：角色醒来时发现自己在一个完全陌生的地方，身边没有同伴。
**类型C — 倒叙引入**：只有一个人活着回来了。记者问：到底发生了什么？他沉默了很久。
