# 冒险类 — 质检清单

> 完稿前逐项检查。打勾通过，未通过则修改。合并通用检查项与各子类型专项检查。

---

## 一、通用检查项

### 文字质量
- [ ] 无病句和错别字
- [ ] 人称一致（无视角混乱）
- [ ] 时间线连贯（无矛盾）
- [ ] 对话自然（无AI腔/书面腔）
- [ ] 无过度解释旁白（Show don't tell）

### 情节结构
- [ ] 每章有明确的情节推进（不原地踏步）
- [ ] 开篇第一句有钩子（动作/声音/异常情况）
- [ ] 每章结尾有钩子（六种章尾钩子模板中选一种）
- [ ] 节奏张弛有度（紧张→放松→更紧张的递进）
- [ ] 快慢交替（不连续两个快或两个慢章节）
- [ ] 主角的动机清晰且有内在驱动力（不只是"被卷入"）

### 角色
- [ ] 主角有弱点（不是万能的）
- [ ] 主角有成长（结尾和开头不同）
- [ ] 伙伴各有功能且有弱点（不是凑人数）
- [ ] 伙伴之间有化学反应（在一起比单独有趣）
- [ ] 团队有摩擦（不能所有人永远团结一致）
- [ ] 对手有威胁且有合理动机（不只是给主角添麻烦）
- [ ] 角色关系有变化（不是从头到尾一成不变）

### 感官
- [ ] 五感里至少用了三种
- [ ] 环境描写具体（有地名/地形特征/天气变化）
- [ ] 危险场景有感官冲击（不只是"很危险"的概括）
- [ ] 宝物/发现的描写有层次（整体→细节→触感→特别之处）

### 节奏与悬念
- [ ] 悬念管理：每章结束读者有理由翻下一页
- [ ] 信息释放节奏合理（不过密也不过稀）
- [ ] 伏笔有回收（之前提到的东西后面用上了）
- [ ] 危机有铺垫（不能突然出现，也不能突然消失）
- [ ] 结局配得上旅途的分量（不能虎头蛇尾）

---

## 二、冒险类专项检查项

### 核心铁律
- [ ] 好奇心有代价（角色为探索付出了真实代价）
- [ ] 未知大于已知（每章结束新谜比解谜多）
- [ ] 环境是最大敌人（环境威胁 > 人类对手威胁）
- [ ] 资源在消耗（水/食物/弹药/时间——每次使用后减少）
- [ ] 宝物意义 > 宝物物质（真相/自由/亲情才是真正的宝）

### 生存与环境
- [ ] 生存细节合理（水/食物/体力消耗有交代）
- [ ] 角色的技能分工明确且在情节中发挥作用
- [ ] 环境描写有变化（不能第一章和第十章是同一个地方同一个天气）
- [ ] 角色的身体状态随剧情推进有变化（受伤/疲惫/脱水/冻伤）

### 节奏
- [ ] 每3-5章有一个重大危机
- [ ] 每次大战后有1-2章喘息/日常/对话
- [ ] 短句用在紧张时刻，长句用在安全时刻
- [ ] 关键转折在全篇的25%/50%/75%位置

---

## 三、子类型专项检查

### 探险寻宝
- [ ] 线索链完整（总线索→分线索→每条线索有代价获取）
- [ ] 机关设计：规则先于机关出现（读者知道规则才能紧张）
- [ ] 机关有时间窗口（没有时间限制=普通谜题）
- [ ] 宝物揭晓不是虎头蛇尾（宝物意义配得上旅途分量）
- [ ] 竞争者/对手有合理能力和动机

### 奇幻冒险
- [ ] 魔法体系自洽（前期规则后期不打破）
- [ ] 魔法有限制和代价（不能万能）
- [ ] 力量提升有合理原因（训练/领悟/牺牲）
- [ ] 伙伴各有独立故事线和战斗场景
- [ ] 日常场景穿插在战斗之间（霍格沃茨的万圣节型）

### 海盗航海
- [ ] 航海术语用量适中（读者看得懂但不失航海味）
- [ ] 海战写法正确（延迟反应/风向影响/三维机动）
- [ ] 船上日常有描写（谁做饭/谁值夜/谁晕船）
- [ ] 船长有魅力（是叙事锚点）
- [ ] 大海有"人格"（不只是背景）

### 太空冒险
- [ ] 科技不能万能（技术问题有真实困难感）
- [ ] 外星人有"异质感"（不只是绿皮肤人类）
- [ ] 太空的"大"有体现（距离/时间/孤独）
- [ ] 科幻不等于魔法（科技有逻辑和限制）
- [ ] 飞船内部政治有刻画（权力/信任/资源分配）

### 地下城探险
- [ ] 每层有独特环境/机关/剧情（不只是"怪更强了"）
- [ ] Boss有模式化战斗（阶段一试探→阶段二反击→阶段三变招）
- [ ] 战利品配得上难度（奖励跟付出匹配）
- [ ] 篝火/安全区有叙事内容（对话/反思/准备）
- [ ] 死亡有重量（要么不复活，要么复活有巨大代价）

### 公路冒险
- [ ] 沿途遭遇都有主题关联（不能随机流水账）
- [ ] 风景有叙事功能（映射心情/标记成长）
- [ ] 同行者的变化有逻辑（需要足够共同经历支撑）
- [ ] 路上的时间没有断层（不能"三天后他们到了"全跳过）
- [ ] 到达目的地时有"空虚感"或"答案在路上"的领悟

### 生存冒险
- [ ] 科学细节经得起推敲（不喝水3天死、失温先兆等）
- [ ] "运气"不能解决太多问题（主要靠知识和意志）
- [ ] 内心独白碎片化（不是流畅散文，是断续的）
- [ ] 身体描写具体（不只写"很虚弱"，写具体症状）
- [ ] 获救后有创伤描写（PTSD/幸存者愧疚/身体后遗症）

## 四、自测评分（20分制）

| 序号 | 检查项 | 说明 | 得分 |
|------|--------|------|------|
| 1 | 有没有悬念？ | 读者有没有理由翻下一页？ | /2 |
| 2 | 感官够不够？ | 五感里至少用了三种？ | /2 |
| 3 | 对话像人话吗？ | 读出来像正常人会说的话？ | /2 |
| 4 | 危险是真的吗？ | 角色有没有真的可能死/失败？ | /2 |
| 5 | 角色有变化吗？ | 结尾和开头有什么不同？ | /2 |
| 6 | 短句用对地方了吗？ | 紧张的时候句子短了没有？ | /2 |
| 7 | 没有解释性旁白？ | 是不是在"演"而不是"讲"？ | /2 |
| 8 | 伏笔回收了吗？ | 之前提到的东西后面用上了？ | /2 |
| 9 | 伙伴有化学反应吗？ | 两个角色在一起比单独有趣？ | /2 |
| 10 | 读者有没有"操"的冲动？ | 看到某一段有没有被震到？ | /2 |

**16分以上**：能看。**12-16分**：及格，改改。**12分以下**：撕了重写。
