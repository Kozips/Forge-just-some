# 冒险类 — V2 Skill 目录

> 探索未知、解谜闯关、团队协作、奇观与危险并存的旅途

## 目录结构

| 目录 | 用途 | 内容概要 |
|------|------|---------|
| SKILL.md | 核心引擎 | 核心张力、叙事节奏、8种角色模板、15场景库、对话风格、感官词库、10技法、经典段落赏析、自测表 |
| 01_核心规则/ | 写作铁律 | 10条不可违反的规则（好奇心代价、未知>已知、资源消耗、节奏张弛等） |
| 02_专业知识/ | 领域知识库 | 七大子类型专业知识（地形/生存/航海/太空/地城/奇幻/公路）+ 生存技能 + 衰败时间表 |
| 03_素材库/ | 可复用素材 | 场景描写6段 + 对话素材6段 + 五感速查表 + 章尾钩子模板 |
| 04_写作模板/ | 结构骨架 | 万能公式 + 通用节奏 + 七种子类型专属节奏 + 开头/结尾模板 + 角色公式 |
| 05_技法研究/ | 写作技法 | 18种技法详解（环境/节奏/悬念/伙伴/动作/伏笔）+ 雷区警示 |
| 06_质检清单/ | 完稿检查 | 通用检查 + 冒险专项 + 七子类型逐项检查 + 20分自测评分 |
| references/ | 原始素材（保留） | 7个子类型深度分析（各600-1100行）+ 蒸馏精华 + 对话库 |

## 使用流程

1. **开写前** — 读 SKILL.md 了解全局，读 01_核心规则/ 确认铁律
2. **构建阶段** — 从 02_专业知识/ 获取领域知识，03_素材库/ 挑选素材
3. **写作阶段** — 按 04_写作模板/ 搭结构，参考 05_技法研究/ 提升手法
4. **完稿阶段** — 用 06_质检清单/ 逐项检查
5. **深度参考** — 需要某子类型的完整分析时，查阅 references/ 下对应文件
