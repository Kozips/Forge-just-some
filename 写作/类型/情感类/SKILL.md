## 速查表

### 核心张力公式

**情感张力 = 关系深度 x 冲突强度 x 不可逆性** — 峰终定律：读者记忆取决于峰值场景和结局。

### 单章结构（暗流-碰撞-余韵型）

| 阶段 | 占比 | 关键动作 |
|------|------|----------|
| 日常铺垫 | 20% | 生活细节引入 + 暗示情感状态 |
| 情感碰撞 | 35% | 两人互动 + 一个微小触动 |
| 内心波澜 | 25% | 独处时情感涌动，行为细节展现（不直接抒情） |
| 余韵 | 20% | 新的认知或决定 + 埋伏笔 |

### 角色压力点速查（Top 5）

1. **信任崩塌** — 最信任的人背叛/误解
2. **失去至爱** — 突然失去或缓慢告别
3. **自我认知颠覆** — 发现自己一直在骗自己
4. **道德困境** — 两个选项都有道理和代价
5. **迟来领悟** — 忽略/误解某人后发现真相

### 常见失败模式（Top 5）

1. **直说情感** — 用颤抖的手/失焦的眼神/反复抚摸遗物展示
2. **过度煽情** — 越悲痛越用平淡笔调，效果更震撼
3. **情感通货膨胀** — 张弛有度，必须有缓冲
4. **千人一面** — 有人哭/有人沉默/有人摔东西/有人打扫
5. **廉价救赎** — 救赎是漫长的、反复的、不完美的

### 结尾类型速查

| 类型 | 特征 |
|------|------|
| 释然型 | 带着伤痕前行，不完美但在呼吸 |
| 遗憾型 | 未完成的永远未完成，留白让读者填补 |
| 重逢型 | 更成熟的重新选择 |
| 永别型 | 最后一眼、最后一句话、最后的微笑 |
| 顿悟型 | 理解了之前不理解的事 |

### 开篇钩子模板

**A. 结局倒叙**：她最后一次见到他，是在机场。三年后，她在便利店闻到了他身上的洗衣液味道。
**B. 日常裂缝**：结婚纪念日，他给她买了花。她看了看花，说了句"你从来不知道我花粉过敏"。
**C. 物件寄情**：她口袋里有一颗糖。是女儿给的。她一直没舍得吃。

### 自检清单（10项必过）

- [ ] 没有直说情感？全文无"他很难过/她很愤怒"？
- [ ] 五感至少用了两种？
- [ ] 粗俗/文雅平衡？该雅不粗暴？该粗不端着？
- [ ] 身体反应先于心理描写？先写"手在抖"再写（或不写）"他害怕"？
- [ ] 高潮处克制住了？最痛的地方用了最短的句子？
- [ ] 有沉默和留白？至少一处"什么都没说"但读者能感受到的？
- [ ] 角色反应有差异？不同性格面对同一事件反应不同？
- [ ] 存在对比或反差？场景之间有情感落差？
- [ ] 细节精准且独特？有"只有这个角色才会有"的小动作？
- [ ] 读者能自己推出来？不看直白表述也能感受到核心情感？

## 角色速查（写作时直接参考）

> 完整版见 `人物/references/人物联动写作/情感类.md`

### 谢晏川（沉稳寡言·一锤定音）
- 语言："领证了？""嗯。婚礼的事，需要帮忙就说。"
- 语气：声音不大，很平，每个字都有分量，不追问细节
- 动作：三连震才接电话——不急；一个"嗯"结束话题——惜字如金
- 用法：需要写沉默型关怀、克制型亲情/友情时参考，"不说"比"说"更有感染力

### 谢怀安（隐忍恨意·暗流涌动）
- 语言："嗯。""今天下午。""临时决定的。"——恨一个人时只在心里记账
- 语气：声音平静到像在播报天气，越恨越平静
- 动作：手机三连震才拿起来看——不急不慌；恨透了但语气依旧平稳
- 用法：需要写表面平静内心翻涌的角色，用"越平静越可怕"制造读者替他记仇的效果

### 百列·加尔（善良·理想主义·隐忍）
- 语言："创建一个充满爱的组织，让每个人都感受到温暖"——长段理想宣言
- 语气：深夜独处时"低声道"——习惯压抑真实情感
- 动作：每晚独自向夜空诉说心事；单膝跪下请求脱离家族——有规矩也有底线
- 用法：需要写理想主义者在压抑环境中挣扎的孤独感，"没有一个能说真话的人"比煽情更动人
