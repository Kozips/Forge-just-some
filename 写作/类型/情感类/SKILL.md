## 速查表

### 核心张力公式

**情感张力 = 关系深度 x 冲突强度 x 不可逆性** — 峰终定律：读者记忆取决于峰值场景和结局。

### 单章结构（暗流-碰撞-余韵型）

| 阶段 | 占比 | 关键动作 |
|------|------|----------|
| 日常铺垫 | 20% | 生活细节引入 + 暗示情感状态 |
| 情感碰撞 | 35% | 两人互动 + 一个微小触动 |
| 内心波澜 | 25% | 独处时情感涌动，行为细节展现（不直接抒情） |
| 余韵 | 20% | 新的认知或决定 + 埋伏笔 |

### 角色压力点速查（Top 5）

1. **信任崩塌** — 最信任的人背叛/误解
2. **失去至爱** — 突然失去或缓慢告别
3. **自我认知颠覆** — 发现自己一直在骗自己
4. **道德困境** — 两个选项都有道理和代价
5. **迟来领悟** — 忽略/误解某人后发现真相

### 常见失败模式（Top 5）

1. **直说情感** — 用颤抖的手/失焦的眼神/反复抚摸遗物展示
2. **过度煽情** — 越悲痛越用平淡笔调，效果更震撼
3. **情感通货膨胀** — 张弛有度，必须有缓冲
4. **千人一面** — 有人哭/有人沉默/有人摔东西/有人打扫
5. **廉价救赎** — 救赎是漫长的、反复的、不完美的

### 结尾类型速查

| 类型 | 特征 |
|------|------|
| 释然型 | 带着伤痕前行，不完美但在呼吸 |
| 遗憾型 | 未完成的永远未完成，留白让读者填补 |
| 重逢型 | 更成熟的重新选择 |
| 永别型 | 最后一眼、最后一句话、最后的微笑 |
| 顿悟型 | 理解了之前不理解的事 |

### 开篇钩子模板

**A. 结局倒叙**：她最后一次见到他，是在机场。三年后，她在便利店闻到了他身上的洗衣液味道。
**B. 日常裂缝**：结婚纪念日，他给她买了花。她看了看花，说了句"你从来不知道我花粉过敏"。
**C. 物件寄情**：她口袋里有一颗糖。是女儿给的。她一直没舍得吃。

### 自检清单（10项必过）

- [ ] 没有直说情感？全文无"他很难过/她很愤怒"？
- [ ] 五感至少用了两种？
- [ ] 粗俗/文雅平衡？该雅不粗暴？该粗不端着？
- [ ] 身体反应先于心理描写？先写"手在抖"再写（或不写）"他害怕"？
- [ ] 高潮处克制住了？最痛的地方用了最短的句子？
- [ ] 有沉默和留白？至少一处"什么都没说"但读者能感受到的？
- [ ] 角色反应有差异？不同性格面对同一事件反应不同？
- [ ] 存在对比或反差？场景之间有情感落差？
- [ ] 细节精准且独特？有"只有这个角色才会有"的小动作？
- [ ] 读者能自己推出来？不看直白表述也能感受到核心情感？

## 题材铁律

# 情感类 -- 题材铁律

> 开写前必读。违反任何一条，情感感崩塌。

## 一、人物心理铁律（1-6）

1. **"心动"要有过程** — 不是第一眼就爱了。是某个瞬间的心动（一个动作/一句话/一个背影），然后反复想起。
2. **"别扭"是核心张力** — 想靠近又不敢、想说出口又咽回去、想牵手又缩回手。别扭=读者心痒。
3. **"认知差距"是戏剧来源** — A 觉得"他对我好"是客气，B 觉得"他是不是喜欢我"，认知差=误会的来源。
4. **"安全区"和"危险区"要分清** — 暧昧前是"安全区"（朋友关系），暧昧后是"危险区"（可能进可能退）。
5. **"自我否定"是常见心理** — "他不可能喜欢我" / "我配不上他" / "我现在不是谈恋爱的时候"。自我否定=真实。
6. **"身体诚实"比嘴快** — 嘴上说"我当他朋友"，但看到他/她时心跳加速/脸红/找借口。身体反应=内心真实。

---

## 二、关系递进铁律（7-12）

7. **"暧昧期"是最甜的** — 确认关系前的推拉期，读者比正式在一起时更上头。暧昧期不能太短。
8. **"第一次"是关键节点** — 第一次约会/第一次牵手/第一次接吻/第一次吵架/第一次见家长。每个"第一次"都是情节。
9. **"确认关系"要"艰难"** — 中间有误会/阻碍/第三者/现实问题。越难得到的关系越珍惜。
10. **"关系升级"有代价** — 在一起了 = 责任/期待/失望/磨合。热恋→平淡→吵架→重新理解→稳固。
11. **"分手/争吵"要有意义** — 分手/争吵必须有原因。不是"为了虐而虐"，是"价值观碰撞/现实压力"。
12. **"破镜重圆"要谨慎** — 复合必须有成长。不能同一个问题吵两次还分手。成长=复合的前提。

---

## 三、对话与潜台词铁律（13-18）

13. **"潜台词"是情感文眼** — A 说"你不用陪我"，B 说的是"你别走"。潜台词比台词更重要。
14. **"沉默"是有效台词** — 对话中30-40% 是"停顿/沉默/动作"。沉默比说满更真实。
15. **"试探"是暧昧期的语言** — "你有喜欢的人吗" / "你喜欢什么类型的" / "你今天为什么来找我"。
16. **"赌气话"不能当真** — "我不需要你" / "你走吧" / "我不在乎"——说这种话是因为在乎，不是真的不在乎。
17. **"微信消息"是现代情感载体** — 发消息不回/已读不回/回得很慢/发一个表情包——数字时代的暧昧/冷战。
18. **"内心独白"要克制** — 不要写大段心理独白。用动作/微表情/环境暗示替代。"他说完，低下头，假装看手机"比"他很紧张"更有力。

---

## 四、冲突与矛盾铁律（19-24）

19. **"误会"是经典但要用好** — 误会要"合理"。A 看到 B 和异性在一起，但不问就认为 B 出轨——这是合理误会。
20. **"原生家庭"是情感冲突根源** — 父母反对/家世差异/异地/经济条件/家族矛盾。这些是现实压力。
21. **"第三者"是张力但不能工具化** — 第三者的出现是因为"这段关系本身有问题"。第三者是问题的暴露，不是问题的制造。
22. **"三角关系"要有选择** — 主角必须在 A 和 B 之间选。选择=主角知道自己想要什么。
23. **"价值观碰撞"是深度冲突** — 两个人对"什么是重要的"理解不同。工作 vs 家庭、自由 vs 稳定、梦想 vs 现实。
24. **"不完美"是真实** — 主角不是圣人，会有自私/嫉妒/攀比/逃避。承认缺点=人物真实。

---

## 五、节奏与余韵铁律（25-30）

25. **"甜虐交织"是基调** — 甜 3 章 → 虐 1 章 → 甜 2 章 → 虐 1 章。交替投放。
26. **"日常"是情感的养料** — 一起吃饭/散步/看电影/做家务。日常=情感的日常确认。
27. **"节日/纪念日"是节奏锚** — 生日/情人节/纪念日/对方的喜好日。这些日子=关系进展的节点。
28. **"分离"是情感重塑** — 异地/出差/冷战/分手。分离期间主角的思念/成长/改变。
29. **"事后"比"当时"更动人** — 高潮场景结束后，主角/两人在安静的空间里回想/总结/流泪。
30. **"留白"是高级写法** — 最感人的场景不写完。让读者自己脑补。"他们最后有没有在一起？"——让读者自己答。

## 自检公式

```
情感感 = 心动真实性 × 推拉密度 × 潜台词深度 × 成长弧线
```

**一句话检验**：把"爱情"全换成"友情"或"亲情"，故事还成立吗？成立=靠"爱"字硬撑，失败=情感有根。

## 二、节奏检查（6项）

- [ ] **11. 情感浓度分布合理**：日常场景30-40%，转折场景60-70%，高潮场景85-95%，余韵场景40-50%
- [ ] **12. 张弛有度**：高密度情感段落后有没有安排呼吸空间？
- [ ] **13. 峰终定律**：最强情感场景是否放在了关键位置？结尾是否精心设计？
- [ ] **14. 没有情感通货膨胀**：全文不是从头到尾都是高强度情感？
- [ ] **15. 递进有层次**：情感是否逐级升高，没有突然跳级？
- [ ] **16. 章节钩子**：每章结尾是否有未解决的情感悬念？

---

## 三、角色检查（6项）

- [ ] **17. 角色有缺陷**：主角不是完美的？缺陷是否让角色更真实、更引发共情？
- [ ] **18. 依恋类型明确**：主角的情感底色是否清晰？（安全/焦虑/回避/恐惧-回避）
- [ ] **19. 暗黑角色有自洽逻辑**：反派有合理的（非"纯粹的恶"的）世界观？
- [ ] **20. 每个角色有独立欲望**：重要配角是否有自己的恐惧、欲望和性格缺陷？
- [ ] **21. 角色关系有张力**：关系张力公式是否成立？（情感张力 = 关系深度 × 冲突强度 × 不可逆性）
- [ ] **22. 角色成长有弧线**：角色从A状态到B状态的变化是否清晰？

---

## 四、结构检查（5项）

- [ ] **23. 伏笔有回收**：所有重要伏笔是否都有回收？有没有"挖坑不填"？
- [ ] **24. 伏笔回收时机恰当**：重要伏笔至少出现2-3次，间隔逐渐缩短？
- [ ] **25. 情感转折有铺垫**：转折是否"意料之外，情理之中"？
- [ ] **26. 不可逆事件存在**：是否有至少一个让角色无法回到之前状态的事件？
- [ ] **27. 选择有代价**：角色做出的选择是否都有真实的、沉重的代价？

---

## 五、表达禁区检查（8项）

- [ ] **28. 没有情感形容词堆砌**：没有"他非常非常伤心"这类表达？
- [ ] **29. 没有旁白直说情感**：没有"此刻她感到无比孤独"这类叙述？
- [ ] **30. 不是千人一面**：不同角色的情感表达方式有差异？
- [ ] **31. 没有虐无逻辑**：角色的苦难有叙事理由，不是为了惨而惨？
- [ ] **32. 没有道德说教**：没有通过旁白告诉读者"你选错了"？
- [ ] **33. 没有假选择**：不同选项是否导向不同结果？
- [ ] **34. 没有过度煽情**：情感表达是否克制，没有"写哭读者"的目的性过强？
- [ ] **35. 没有廉价救赎**：希望/救赎是否经过了真实的挣扎和代价？

---

## 六、美学检查（4项）

- [ ] **36. 美学风格一致**：整体美学基调是否统一？（侘寂/物哀/水墨/哥特等）
- [ ] **37. 意象系统协调**：使用的意象是否与情感基调匹配？（雨=悲伤？但如果是解放场景呢？）
- [ ] **38. 环境描写有情感**：环境描写是否承载了情感，而非只是"背景"？
- [ ] **39. 少即是多**：环境描写是否遵循不超过3个核心意象的原则？

---

## 七、终极检验（2项）

- [ ] **40. 读者会想起自己**：读者是否会因为这个场景想起自己的经历，而非仅仅"为角色哭"？
- [ ] **41. 悲伤不是目的，温暖才是**：哭完之后，读者感受到的是温暖/珍惜/成长，而非空虚？

---

## 快速过检流程

1. **初稿完成后**：先过基础规则检查（第1-10项），这10项是底线
2. **二稿时**：过节奏检查（第11-16项）+ 角色检查（第17-22项）
3. **三稿时**：过结构检查（第23-27项）+ 表达禁区检查（第28-35项）
4. **定稿前**：过美学检查（第36-39项）+ 终极检验（第40-41项）

> 如果基础规则检查有3项以上未通过，建议推翻重写该场景。如果表达禁区检查有2项以上未通过，说明情感表达存在根本问题，需重新审视写法。
