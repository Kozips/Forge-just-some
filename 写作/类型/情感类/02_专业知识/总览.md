# 情感创作专业知识库

> 本文件整合情感心理学理论、角色设计系统、情绪模型等核心知识，供创作参考。

## 第一章：情绪理论与情感分类

### 1.1 普鲁奇克情绪轮盘
8种基本情绪×3强度等级，可组合产生复合情绪：

| 基本情绪 | 低 | 中 | 高 | 创作应用 |
|---------|---|---|---|---------|
| 快乐 | 宁静 | 快乐 | 狂喜 | 日常场景、希望段落 |
| 信任 | 接纳 | 信任 | 钦佩 | 关系建立、师徒线 |
| 恐惧 | 忧虑 | 恐惧 | 恐怖 | 悬疑、末日、心理惊悚 |
| 悲伤 | 忧郁 | 悲伤 | 悲痛 | 离别、丧失、BE向 |
| 厌恶 | 厌烦 | 厌恶 | 憎恶 | 反派心理、道德批判 |
| 愤怒 | 烦恼 | 愤怒 | 狂怒 | 复仇、打脸、反抗 |
| 期待 | 关注 | 期待 | 警觉 | 悬念、升级、突破 |
| 惊讶 | 分心 | 惊讶 | 惊愕 | 反转、揭示、顿悟 |

**常用组合情绪**：
- 快乐+信任 = 爱
- 恐惧+惊讶 = 敬畏
- 悲伤+厌恶 = 后悔/悔恨
- 愤怒+期待 = 复仇欲
- 期待+恐惧 = 既期待又害怕

### 1.2 悲伤五阶段（DABDA）
角色面对重大丧失时的心理过程：

1. **否认**："不，这不可能发生。" 声音正常甚至偏高。
2. **愤怒**："为什么是我？这不公平！" 声音提高、语速变快。
3. **讨价还价**："如果当时我在场..." 声音颤抖、低声。
4. **抑郁**："无所谓了。" 声音极低、大量沉默。
5. **接受**："我会带着它继续前行。" 平静但有重量。

**创作应用**：
- 不必按顺序经历所有阶段，可以停留在某一阶段
- 不同角色面对同一事件可处于不同阶段（制造冲突）
- 每个阶段的外在表现应不同——否认时的麻木、愤怒时的失控、抑郁时的沉默

### 1.3 情感颗粒度
情绪粒度越高，能越精确区分不同情绪。不只是"不好受"，而是能分辨"失望/沮丧/惆怅/空虚"。

**情绪精确词汇对照**：
| 基础情绪 | 高粒度细分 |
|---------|-----------|
| 悲伤 | 哀悼、惆怅、惋惜、落寞、黯然、凄楚、悲怆、哀恸 |
| 愤怒 | 恼怒、愤慨、暴怒、怨恨、憋屈、不甘、屈辱 |
| 恐惧 | 忧虑、惊慌、惶恐、战栗、毛骨悚然、心惊肉跳 |
| 快乐 | 欣喜、满足、惬意、欣慰、陶醉、狂喜、酣畅 |
| 厌恶 | 嫌弃、鄙夷、反胃、不齿、嫌恶、唾弃 |
| 惊讶 | 讶异、震惊、愕然、目瞪口呆、不可思议 |

**文化特异性情绪词**：
- 德语Schadenfreude（幸灾乐祸）
- 日语物哀
- 中文惆怅、尴尬
- 葡萄牙语Saudade（对失去之物的怀念）

## 第二章：角色设计系统

### 2.1 角色设计三步法
**Step 1**：选择1个原型作为主型，1个作为辅型（避免单一维度）
**Step 2**：选择1种依恋类型作为情感底色
**Step 3**：选择1条成长弧线作为故事主轴

> 示例：守护者原型 + 回避型依恋 + U型觉醒弧 = 一个表面冷漠但内心深爱，经历失去后打开心扉的角色

### 2.2 12种角露骨感原型

| 原型 | 核心情感 | 情感驱动力 | 成长弧线方向 |
|------|---------|-----------|-------------|
| **英雄** | 使命感+内心恐惧的克服 | 证明自己的价值 | 逃避→接受→超越 |
| **反派/阴影** | 被伤害后的愤怒+扭曲正义感 | 报复/纠正不公 | 善良→被背叛→黑化→(可选)救赎 |
| **导师/智者** | 智慧的孤独+传承渴望 | 找到继承者 | 孤独→找到传承→放心离去 |
| **叛逆者** | 对自由的极致渴望 | 打破束缚 | 反叛→发现复杂→战斗或和解 |
| **守护者** | 深沉的爱+牺牲准备 | 保护重要之人 | 封闭→打开心扉→放手抉择 |
| **殉道者** | 超越自我的信念 | 信仰高于生命 | 承担→隐忍→释放→牺牲 |
| **幸存者** | 活着的愧疚+坚韧生命力 | 代替逝者活下去 | 麻木→新羁绊→允许自己活着 |
| **梦行者** | 对美好的执着追求 | 创造更好的世界 | 天真→被打碎→碎片中找到新美 |
| **审判者** | 绝对公正+内心孤独 | 纠正不公 | 铁面→发现灰色→理解宽恕 |
| **调和者** | 对和谐的渴望+自我压抑 | 化解矛盾 | 讨好→被忽视→为自己而活 |
| **觉醒者** | 从蒙昧到清醒的震撼 | 追寻真相 | 无知→怀疑→发现→面对新选择 |
| **守望者** | 漫长等待+不灭希望 | 守护承诺/信念 | 等待→失望→再等待→终于等到 |

### 2.3 依恋理论——角色性格底色

| 依恋类型 | 角色特征 | 情感模式 | 创作价值 |
|---------|---------|---------|---------|
| **安全型** | 信任他人，自我价值感高 | 健康表达需求，尊重边界 | 适合作为"锚点角色"，需赋予其他戏剧性特质 |
| **焦虑-矛盾型** | 渴望亲密但害怕被抛弃 | 反复试探、索取确认、容易嫉妒 | 天然的戏剧张力制造者，适合虐恋核心角色 |
| **回避型** | 重视独立，回避深层亲密 | 用距离保护自己，压抑情感表达 | 适合冷面热心型，"冰山融化"成长弧线 |
| **恐惧-回避型** | 既渴望又恐惧亲密 | 靠近→恐慌→推开→后悔→再靠近 | 最具戏剧冲突的底色，适合复杂情感故事 |

**创作提示**：童年经历决定依恋类型——被抛弃→焦虑型，被忽视→回避型，遭受虐待→恐惧-回避型。

### 2.4 成长弧线系统——五种基本弧线

| 弧线 | 路径 | 适用类型 | 经典案例 |
|------|------|---------|---------|
| **A 正向成长** | 天真→考验→挣扎→觉醒→超越 | 英雄成长、逆袭 | 萧炎、哈利波特 |
| **B 堕落弧** | 善良→诱惑→妥协→深陷→无法回头 | 反派起源、腐败 | 希斯克利夫、安纳金 |
| **C 觉醒弧（U型）** | 蒙昧→打击→谷底→发现→重生 | 顿悟、救赎 | 亚瑟·摩根、浪子回头 |
| **D 循环弧（螺旋）** | 重复→差异→发现→再重复→超越 | 轮回、时间循环 | 尼尔机械纪元 |
| **E 平线弧** | 不变→环境变化→坚持→影响世界 | 以不变应万变 | 阿甘、许三观 |

### 2.5 马斯洛需求层次——角色动机系统

| 需求层次 | 未满足时的情感 | 适用故事类型 | 典型冲突 |
|---------|--------------|------------|---------|
| **生理需求** | 焦虑、恐慌 | 末日/生存/战争 | 活下去vs道德底线 |
| **安全需求** | 恐惧、不安 | 悬疑/惊悚/冒险 | 安全vs真相 |
| **归属需求** | 孤独、空虚 | 言情/成长/校园 | 爱vs自我 |
| **尊重需求** | 自卑、羞耻 | 职场/竞技/逆袭 | 尊严vs归属 |
| **自我实现** | 迷茫、不甘 | 修仙/史诗/艺术 | 理想vs安全（"以命换道"） |

## 第三章：情感共鸣机制

### 3.1 情感共鸣的心理基础
情感共鸣（Emotional Resonance）是指个体在感知他人情感状态时，产生类似情感体验的心理现象。涉及镜像神经元、共情机制、情绪感染和社会学习等多重路径。

### 3.2 情绪感染的自动化过程
情绪感染是自动的、前意识的。人们会无意识地模仿他人的面部表情、声调和姿态，进而体验到相应的情绪。

**创作应用**：
- 高质量的表情动画和语音表演能触发更强的共情反应
- 角色的微表情和小动作比文字对话更能传递情感真实感
- 身体语言永远比台词诚实

### 3.3 共情疲劳规避
长期暴露于他人痛苦后产生的情感与身体疲惫。过多苦难叙事导致观众/读者麻木。

**"饱和点"设计原则**：
在观众即将产生疲劳之前转向希望或行动。过多苦难→麻木，过少苦难→轻浮。

**对抗疲劳的叙事策略**：
1. 赋予"行动的可能性"——不只是被动接受苦难，而是有机会改变
2. 穿插幽默和温暖——在黑暗中给予喘息
3. 展示微小的善意——证明世界未完全崩塌

## 第四章：创伤叙事设计

### 4.1 创伤记忆的不可言说性
创伤记忆是感官碎片（声音/气味/画面），不是连贯时间线叙事。文学和游戏尝试突破这种"不可言说"。

### 4.2 创伤后成长（PTG）五维度
用于角色成长后半段：

1. **对生命的重新认识**：更珍惜日常，不再纠结小事
2. **人际关系改善**：更深层的连接，选择真正重要的人
3. **个人力量感增强**："我经历过更糟的"带来的韧性
4. **发现新的可能性**：创伤打开之前不敢想的道路
5. **精神/存在层面的改变**：更深的人生理解，超越世俗

### 4.3 认知失调在创作中的运用
当个体同时持有两种矛盾的信念、态度或行为时，会产生心理不适感。

**三种消解路径决定角色成长方向**：
- 改变行为以匹配信念 → 忏悔/赎罪弧线
- 改变信念以匹配行为 → 堕落弧线
- 合理化 → 自我欺骗弧线
