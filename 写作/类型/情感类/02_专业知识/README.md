# 情感类 — 专业知识库

> 本目录存放该类型所需的领域知识。构建世界观时参考。

## 情感关系阶段

| 阶段 | 特征 | 典型表现 |
|------|------|----------|
| 暧昧期 | 不确定性、试探 | 频繁找话题/注意穿着/制造偶遇 |
| 热恋期 | 滤镜、理想化 | 觉得对方什么都好/忽略缺点 |
| 磨合期 | 差异暴露、冲突 | 争吵/冷战/第一次说"算了" |
| 平淡期 | 习惯、安全感 | 不再刻意打扮/不再秒回消息 |
| 危机期 | 质疑、动摇 | "我们还合适吗"/"我是不是不爱了" |
| 转化期 | 重选、深化 | 接受真实/重新选择在一起或分开 |

## 情感表达方式

- **行动型**：不说爱，但每天接送、记住对方的喜好
- **语言型**：善于表达，但可能"说到做不到"
- **回避型**：越在乎越躲，用冷漠掩盖紧张
- **牺牲型**：默默付出但不表达，等对方发现
- **控制型**：以爱之名做决定，"我是为你好"

## 常用术语

- **依恋类型**：安全型/焦虑型/回避型/混乱型
- **情感劳动**：关系中主动维护情绪的隐性付出
- **边界感**：亲密关系中保持独立空间的能力
- **情绪价值**：对方能让你感到被理解、被接纳
- **沉没成本**：因为已经投入太多而不舍得离开
