# 情感类 — V2 Skill 目录

> 爱情/友情/亲情、细腻心理、情感弧线、克制与爆发的情感张力

## 目录结构

| 目录 | 用途 |
|------|------|
| SKILL.md | V1 主技能文件（核心引擎） |
| references/ | V1 素材参考（保留） |
| 01_核心规则/ | 写作铁律与不可违反的规则 |
| 02_专业知识/ | 该类型的领域知识库 |
| 03_素材库/ | 场景/对话/描写素材片段 |
| 04_写作模板/ | 结构模板与章节骨架 |
| 05_技法研究/ | 写作技法分析与借鉴 |
| 06_质检清单/ | 完稿前的检查项 |

## 使用流程

1. **开写前** — 读 SKILL.md 了解全局，读 01_核心规则/确认铁律
2. **构建阶段** — 从 02_专业知识/ 获取领域知识，03_素材库/ 挑选素材
3. **写作阶段** — 按 04_写作模板/ 搭结构，参考 05_技法研究/ 提升手法
4. **完稿阶段** — 用 06_质检清单/ 逐项检查
