# 情感类 — 技法研究

> 本目录存放写作技法的分析与借鉴。用于提升手法而非直接复制。

## 技法1：以物寄情

**来源**：《情书》《秒速五厘米》
**手法拆解**：
1. 一个小物件贯穿全文（一条围巾/一首歌/一个地址）
2. 物件在不同阶段承载不同情感（暧昧时是甜蜜，分手后是刺痛）
3. 物件的"出现-消失-重新出现"暗示情感走向

## 技法2：对话中的留白

**来源**：王家卫电影台词、《花样年华》
**手法拆解**：
1. 对话只写"说出口的部分"
2. 没说出口的由读者自行脑补
3. 关键时刻角色说了一句无关紧要的话——恰恰是这句话最动人

> "你明天几点的飞机？""八点。""哦。路上注意安全。""嗯。"沉默了大概五秒。"那个……你家的WiFi密码是多少？我手机没流量了。"

## 技法3：双视角对照

**来源**：《花束般的恋爱》《那些年》
**手法拆解**：
1. 同一事件分别从A和B的视角叙述
2. 两人看到的"重点"不同——A在意的是B不在意的
3. 读者从双视角中看到双方都看不到的真相

## 技法4：时间跳跃制造沧桑感

**来源**：《大话西游》《一代宗师》
**手法拆解**：
1. 用"三年后"/"她三十岁了"等时间跳跃
2. 不交代中间发生了什么——让读者自行想象
3. 角色的变化暗示这段时间的情感重量
