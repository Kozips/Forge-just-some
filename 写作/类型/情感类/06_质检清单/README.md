# 情感类 — 质检清单

> 完稿前逐项检查。打勾通过，未通过则修改。

## 通用检查项

- [ ] 无病句和错别字
- [ ] 人称一致（无视角混乱）
- [ ] 时间线连贯（无矛盾）
- [ ] 对话自然（无AI腔/书面腔）
- [ ] 每章有明确的情节推进

## 情感类专项检查项

- [ ] 情感通过行为和细节展现（不直接抒发"我很伤心"）
- [ ] 情感变化有触发事件（不是突然转变）
- [ ] 双方付出对等但方式不同（不能一方单方面牺牲）
- [ ] 关键场景用"沉默/留白"代替"告白/哭喊"
- [ ] 暧昧/冷战/和好的节奏有起伏（不能一直甜蜜或一直虐）
- [ ] 角色有独立于感情线的个人成长
- [ ] 每段对话都有潜台词（表面意思 ≠ 实际意思）
- [ ] 分离/重逢有说服力（不是为了虐而虐/为了甜而甜）
- [ ] 没有"完美的另一半"——双方都有真实缺点
