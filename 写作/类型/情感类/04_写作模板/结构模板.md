# 情感类 — 结构模板

> 本目录存放章节结构模板与叙事骨架。写作前用模板规划节奏。

## 单章结构（暗流-碰撞-余韵型）

```
### 日常铺垫（前20%）
- 用一个生活细节引入（做饭/通勤/等消息）
- 暗示角色当下的情感状态

### 情感碰撞（35%）
- 两人互动——可能是对话、相处、或分隔两地的思念
- 一个微小的触动（对方的一句话/一个习惯/一个细节）

### 内心波澜（25%）
- 角色独处时的情感涌动
- 通过行为细节展现（不直接抒情）

### 余韵（20%）
- 一个新的认知或决定
- 为下一章的情感走向埋伏笔
```

## 多章弧线（相遇-相知-分离-重逢型）

```
第1-3章：相遇与暧昧 → 互有好感但不挑明
第4-6章：靠近与冲突 → 差异暴露，第一次大吵
第7-9章：分离与成长 → 各自面对自己的问题
第10-12章：重逢与选择 → 不是和好，而是更成熟的重新选择
```

## 开篇钩子模板

**类型A — 从结局倒叙**：她最后一次见到他，是在机场。他说了句"保重"，转身走了。她没追。三年后，她在便利店的货架前闻到了他身上的洗衣液味道。
**类型B — 日常中的裂缝**：结婚纪念日，他给她买了花。她看了看花，说了句"你从来不知道我花粉过敏"。这是结婚第五年。
