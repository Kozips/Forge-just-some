# 情感类写作模板库

> 本文件提供从单章到完整叙事的结构模板，配合各篇幅的节奏规划。04_写作模板/结构模板.md中已有基础模板保留，本文件深化补充。

## 第一章：情感弧线六种形状

Reagan大数据模型证明，故事的情感弧线主要呈现六种基本形状：

```
(1) /   从贫到富（持续上升）——逆袭爽文
(2) \   从富到贫（持续下降）——BE美学
(3) :\  堕落（先平后跌）——反派起源
(4) :/  救赎（先跌后升）——成长/救赎
(5) :/\ 灰姑娘型（升→降→升）——传播力最强，最令人难忘
(6) :\/\ 伊卡洛斯型（升→降→升→降）——宿命论/反复挣扎
```

**设计要点**：灰姑娘型传播力最强；下降段必须足够深，上升才有力量；结尾情感位置比开头更影响记忆。

## 第二章：各篇幅节奏模板

### 短篇（1-3万字）
```
[开篇] 情感基线建立 ── 快速确立角露骨感状态
[1/3处] 第一次冲击 ── 小型冲突/转折
[2/3处] 情感高峰 ── 全文最强情感场景
[结尾] 余韵收束 ── 简短但有力的收尾

伏笔数：1-2个 | 情感起伏：2-3次
```

### 中篇（3-10万字）
```
[第1-2章] 世界与角色建立 ── 日常/基线
[第3-4章] 第一波冲突 ── 裂隙出现
[第5-6章] 升级 ── 矛盾加深 + 伏笔密集期
[第7-8章] 假平静 ── 表面平息，暗流涌动
[第9章]   情感大爆发 ── 全文最高潮
[第10章]  收束与余韵 ── 结局

伏笔数：3-5个 | 情感起伏：4-6次
```

### 长篇（10万字以上）
```
[卷一·建]  世界/角色/关系建立 ── 日常积累 + 伏笔密集期
[卷一·尾]  第一次重大冲击 ── 引入核心矛盾
[卷二·起]  冲突升级 + 支线展开 ── 多条情感线交织
[卷二·中]  假平静/转折点 ── 最幸福的时刻
[卷二·尾]  剧变 ── 核心关系/世界观崩塌
[卷三·起]  低谷 + 重建 ── 角色在废墟中寻找新方向
[卷三·中]  多线汇聚 ── 所有伏笔开始回收
[卷三·尾]  终极高潮 ── 所有情感线同时引爆
[尾声]     余韵 ── 带着伤痕与成长前行

伏笔数：10-20个 | 情感起伏：8-15次
```

### 连载/网文
```
[每章钩子] 章末必须有一个情感钩子
[3-5万字一卷] 每卷有独立的完整情感弧线
[30-50万字一大卷] 一个大的情感主题贯穿

节奏原则：
- 爽感与虐感的比例：网文推荐7:3
- 每2-3章一个小高潮，每20-30章一个大高潮
- 短期伏笔（2-3章内回收）+ 中期伏笔（一卷内回收）+ 长期伏笔（跨卷回收）
```

## 第三章：10大情感场景模板

### 场景1：表白
- **含蓄型**："你放心。"——三个字胜千言
- **热烈型**："我把我整个灵魂都给你，连同它的怪癖..."
- **要点**：表白的力量在于铺垫而非言辞本身；表白前的犹豫比表白的话语更重要

### 场景2：诀别
- **主动选择型**：为保护对方主动离去，回头看最后一眼
- **被迫永别型**：来不及说出口的话，成为永远的遗憾
- **要点**：最动人的是"未完成"——没说完的话、没送出的信、没回的头

### 场景3：重逢
- **劫后重逢型**：历经生死后的相见——一个眼神就够了
- **物是人非型**：重逢了，但一切都变了
- **要点**：重逢冲击力=等待的时间x情感的浓度

### 场景4：牺牲
- **为爱牺牲型**：以命换命——"你活着就好"
- **为信念牺牲型**：明知必死——"这是我选的路"
- **要点**：最后一刻是关键——最后看一眼、最后一句话、最后的微笑

### 场景5：背叛
- **被信任者背叛型**：最信任的人是叛徒
- **自我背叛型**：角色背叛了自己的信念
- **要点**：冲击力与信任深度成正比

### 场景6：顿悟
- **真相揭露型**：所有线索串联——"原来是这样"
- **自我认知型**：发现自己一直在骗自己
- **要点**：顿悟前的线索必须已在前文埋下

### 场景7：等待
- **有期限的等待**：知道会来但时间未知——焦虑中的希望
- **无期的等待**：不知道会不会来——纯粹的信念
- **要点**：写时间的流逝——日历翻页、季节更替、容颜老去

### 场景8：失去
- **突然失去**：猝不及防——角色被击溃然后麻木
- **缓慢失去**：渐冻症式的告别——每天都在失去一点
- **要点**：失去之后的"空"比失去的瞬间更痛

### 场景9：抉择
- **两难抉择**：两个选项都有道理和代价
- **自我牺牲式**：牺牲自己还是牺牲他人
- **要点**：力量来自"读者也不知道该怎么选"

### 场景10：救赎
- **自我救赎**：战胜心魔——"我放过了自己"
- **被他人救赎**：最黑暗时有人伸出手——"我在，你不是一个人"
- **要点**：救赎不能太轻松，必须经历真实的挣扎和代价

## 第四章：五套完整叙事模板

### 模板一：武侠——侠义与情义的两难
```
开篇（少年初入江湖，怀揣理想）
→ 前期（拜师学艺+结交挚友+爱情线萌芽）——温暖上升
→ 中期（身份揭露/身世真相）——急转直下，众叛亲离
→ 后期（家国危机+个人情感终极冲突）——最高压力
→ 高潮（救苍生vs守一人，做出选择承受代价）
→ 结局（归隐/殉道/带着伤痕继续行走）
```
核心技法：环境烘托（诗词意境）+ 以武写情 + 留白克制

### 模板二：言情——从误解到理解的爱情
```
第一幕（相遇与误解——冰冷/对立）
→ 第二幕（被迫靠近+逐渐了解——温度缓慢回升）
→ 第三幕（心动确认+外部阻碍——甜蜜上升）
→ 第四幕（裂变与分离——从高处坠落）
→ 第五幕（重逢与理解——带着各自的伤，选择一起面对）
```
核心技法：心理描写+细节特写+欲言又止的对话+物象暗示

### 模板三：玄幻——凡人逆天与命运抗争
```
开篇（废物觉醒——屈辱、不甘）
→ 卷一·崛起（打脸升级循环+建立三个情感锚点）
→ 卷二·失去（核心打击——师父离去/爱人牺牲/兄弟背叛）
→ 卷三·征途（更大世界+更深孤独）
→ 终卷·超越（所有情感线同时引爆）
```
核心技法：热血宣言+师徒离别+献祭名场面+万古孤独

### 模板四：悬疑——真相的代价
```
开篇（平静被打破——谋杀/失踪/异常事件）
→ 前期调查（碎片收集+道德困境+伏笔密集期）
→ 中期转折（认知颠覆——怀疑自己判断）
→ 后期逼近（信任崩塌+生理反应描写）
→ 真相揭示（冰点——最残酷的真相指向自己）
→ 结局（带着真相活下去/毁灭/封存）
```
核心技法：好奇心缺口+碎片化叙事+多视角交织+延迟后果

### 模板五：科幻——文明尺度下的个体情感
```
开篇（日常中的裂缝——一个异变打破日常）
→ 第一幕（发现与震撼——敬畏时刻+人类渺小vs意义）
→ 第二幕（抉择与分裂——延续文明vs保护个体vs接受命运）
→ 第三幕（牺牲与传承——一封跨越光年的家书）
→ 终幕（宇宙很大，生活更大）
```
核心技法：环境叙事+哲学独白+仪式感设计+尺度反差

## 第五章：开篇钩子模板

**类型A — 从结局倒叙**：
> 她最后一次见到他，是在机场。他说了句"保重"，转身走了。她没追。三年后，她在便利店的货架前闻到了他身上的洗衣液味道。

**类型B — 日常中的裂缝**：
> 结婚纪念日，他给她买了花。她看了看花，说了句"你从来不知道我花粉过敏"。这是结婚第五年。

**类型C — 延迟反应开篇**：
> 他把杯子放下。手在抖。但他脸上没有表情。

**类型D — 信息差开篇**：
> 她发了条消息："分手吧。"他回："好。"（读者知道他回"好"是因为他确诊了，不想拖累她）

## 附录：伏笔设计速查

| 伏笔类型 | 示例 |
|---------|------|
| 物件伏笔 | 一枚不起眼的戒指，关键时刻成为救命信物 |
| 对话伏笔 | 第一章的一句玩笑话，在结局成为遗言 |
| 习惯伏笔 | 总是多摆一副碗筷——为不在的人留位 |
| 场景伏笔 | 同一地点在不同时间出现，映射角色变化 |
| 行为反差伏笔 | 角色的行为与表面情绪不一致——暗示压抑的情感 |

**伏笔回收时机**：重要伏笔至少出现2-3次，间隔逐渐缩短，在读者快要忘记时揭晓效果最佳。先设计结局再回溯埋伏笔，确保指向精确。
