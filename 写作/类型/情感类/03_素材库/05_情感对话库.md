# 情感对话数据库

> 用途：情感类创作的对话参考。每个场景含完整对话+技法要点。情感对话的核心是"说的不是想的"——潜台词比台词重要。

---

## 1. 分手后便利店偶遇

**场景设定**：分手三个月，凌晨一点的全家便利店，两个人在关东煮前撞上了。

```
她拿着一串萝卜。他拿着一瓶乌龙茶。
两个人同时看到对方。

"……"

"……"

收银台的小哥看了他们一眼，低头继续玩手机。

"你——"

"你先说。"两个人同时开口。又同时停住。

她笑了一下。不是开心的笑。是那种"好尴尬"的笑。
"你住这附近？"

"嗯。上个月搬过来的。"

"哦。"她低头翻了翻关东煮的格子，"那挺巧的。"

"是挺巧的。"

沉默。关东煮咕嘟咕嘟冒着泡。

"你瘦了。"他说。说完就后悔了。

她夹萝卜的筷子停了一下。"你倒是没变。"

"嗯。"

"还是这么不会照顾自己。"她看了一眼他手里的乌龙茶和一包薯片，"就吃这个？"

"你不也就一串萝卜。"

"我减肥。"

"你不用减。"

又沉默了。这次长了一点。

"那我……先走了。"她拿着萝卜走向收银台。

"嗯。"

她付了钱。推门。冷风灌进来。
她站在门口犹豫了一秒。回头。

"那个——你胃不好，别老喝凉的。"

门关上了。他站在原地，乌龙茶在手里握出了一手心的水雾。

他看了一眼手机。她的微信还在通讯录里。
最后一条消息是三个月前的："嗯。"

一个字的分手。
```

**技法要点**：偶遇场景的核心张力来自"想关心但没资格"——"你瘦了"说出口就后悔，因为她已经不是他可以说这话的人了；关东煮的咕嘟声填满沉默，环境音比心理独白更有效；结尾的"嗯"字呼应开头的尴尬，一字结尾比长篇大论更有余味。

## 2. 暗恋三年的表白

**场景设定**：大学图书馆闭馆，两个人被一起锁在了自习室。

```
灯灭了。整栋楼黑了。
"操，闭馆了我们没走？"她从书堆里抬起头。

"好像是。"他看了一眼手机，"九点半了。"

"门锁了？"她跑过去推了推。推不动。"我靠，真锁了。"

"打电话给保安吧。"

"行。"她翻手机，"我没保安电话。你有吗？"

"我有。"

"那你打啊。"

他没动。

"你干嘛？"

"再坐一会儿吧。"他的声音在黑暗里听起来不太一样。低了一点。

她坐回去。手机屏幕的光映着两个人的脸。

"你今天怎么了？话好少。"

"没有。"

"你从下午开始就怪怪的。一直看我。"

"我没——"

"你有。"她笑了一下，"你以为我不知道？坐我对面，平均每七分钟抬头看我一次。我数过。"

他沉默了。

"所以呢？"她的语气忽然轻了，"看了三年了，有话要说吗？"

空气安静了。黑暗里只有空调的嗡嗡声。

"我不想说了。"

"为什么？"

"说了怕你以后不坐我对面了。"

又是沉默。这次她先开口。
"你怎么知道我会不坐了？"

他抬起头。黑暗里看不见她的表情。
但她的声音在笑。

"那……"他的声音在发抖，"那你能不能——以后也坐我对面？"

"就这？"她叹了口气，"我以为你要说什么呢。坐呗，反正你帮我占座占了三年了，免费劳动力不用白不用。"

他的手心全是汗。

"还有呢？"她问。

"什么还有？"

"你就只想说这个？你看了三年，攒了三年，就这？"

他深吸一口气。
"我喜欢你。从大一开学那天你穿着军训服坐我对面开始。"

很长的安静。

"保安电话你还打不打了？"她的声音有点闷。

"打。"

"别打了。"她的手机照亮了她的脸。耳朵是红的。"再坐一会儿。"
```

**技法要点**：暗恋表白用"被困"做催化剂——黑暗、封闭、只有两个人，自然逼出真话；"平均每七分钟抬头看我一次，我数过"——她早就知道了，暗恋的微妙在于双方都心知肚明但没人先开口；"别打了，再坐一会儿"是含蓄版的"我也喜欢你"。

## 3. 异地恋视频通话的沉默

**场景设定**：异地恋第八个月，深夜视频通话，两个人都不说话。

```
屏幕亮着。她在宿舍。他在出租屋。
两个人都盯着屏幕。但眼睛没有焦点。

"你今天干嘛了？"

"上班。你呢？"

"上课。"

"嗯。"

沉默。

她翻了一页书。他刷了两下手机。

"最近忙吗？"

"还行。"

"嗯。"

又是沉默。这次长了。长到他以为网络断了。
他看了一眼。信号满格。她还在。低着头在写字。

"你在写什么？"

"论文。"

"哦。"

他靠在椅背上。天花板上有一道裂缝。他每天视频的时候都盯着那道裂缝看。
数了一下。裂缝又长了一点。

"你那边冷吗？"他问。

"嗯。暖气还没来。"

"多穿点。"

"穿了。"

他看到她裹了一件他的旧卫衣。灰色的。袖口有点长，她的手指只露出一半。
他买给自己的。走的时候留在了她那里。

"那个卫衣——"

"嗯？"

"穿着合适吗？"

"有点大。"她低头看了一眼，"我喜欢大一点的。"

"嗯。"

沉默。
她的室友开门进来，看到她在视频，识趣地又出去了。

"我室友都觉得我们不像在谈恋爱。"她忽然说。

"什么意思？"

"就是……别人视频都在聊好笑的事情，发很多消息。我们每次视频都——"

她没说下去。

"都什么？"

"都好安静。"她的声音忽然有点哑，"但我不想挂。"

他看着屏幕里的她。她低着头。刘海遮住了眼睛。

"我也不想挂。"他说。

"那你别老嗯嗯嗯的。"

"那你别老说嗯。"

她笑了。很小的笑。
"那你今天给我讲个笑话。"

"我不会讲笑话。"

"随便说点什么。说什么都行。就是——别让我觉得我们之间只剩嗯了。"

他想了很久。
"今天中午食堂的饭特别难吃，我吃了两口就扔了。然后我想起来你每次都会吃光，就算很难吃也会吃光。然后我就——"

"就什么？"

"就把那份饭捡回来了。"

"你有病吧。"

"吃了。还发了个朋友圈。"

"你发朋友圈了？！我怎么没看到？"

"秒删了。太丢人了。"

她笑出了声。真的笑。他好久没听到她这样笑了。

"你等着。"她翻手机，"我现在就去你朋友圈找。"

"删了找不到了。"

"我不信。"

两个人吵着笑着，忘了沉默的事。
```

**技法要点**：异地恋的沉默不是无话可说，是"每天都说一样的话所以不想说了"——用"嗯"字填满对话是异地恋的真实写照；旧卫衣的细节比"我想你"更深情——她每天穿着他的衣服；"别让我觉得我们之间只剩嗯了"是整个场景的情绪转折点。

## 4. 婚礼前一天新娘和前男友的最后对话

**场景设定**：婚礼前夜，新娘在酒店房间里接到一个电话。

```
手机亮了。来电显示是一个删了但记得住的号码。
她看了一眼洗手间。未婚夫在里面洗澡。

她走到阳台，关上门，接了。

"喂。"

"是我。"他的声音。三年没变了。

"我知道。"

"你明天结婚。"

"嗯。"

"我知道我不应该打这个电话。"

"那你还打。"

沉默。海风吹过来。酒店靠海。她选的。

"我就想问你一件事。"

"问。"

"当年——你走的那天，我说'你走了就别回来'。你真的就再也没回来。"

"你让我别回来的。"

"我说的是气话。"

"我知道。"她把手机换了一只手。手心出了汗，"但你没有追出来。"

"我在窗口看着你走的。"

"我知道。"她说，"我回头看了一眼。你在窗口。"

"你回头了？"

"嗯。我以为你会追出来。"

电话那边沉默了很久。

"后悔吗？"他问。

她看了一眼洗手间的门。水声还在。
"不后悔。"

"真的？"

"真的。"她的声音很稳，"他很好。"

"我知道。我看过他的朋友圈。"

"你翻我老公朋友圈？"

"……偶尔。"

她笑了。带着一点鼻音。不是哭。只是笑的时候鼻子酸了一下。

"你也会找到一个很好的人的。"

"你别用这种语气跟我说话。"

"什么语气？"

"就是……那种前女友的语气。祝福的、大度的、客客气气的。"他的声音在发抖，"你能不能——跟我说一句不客气的话？"

她握着栏杆。海风把头发吹到脸上。

"你是个混蛋。你当初不该说那句话。你追出来我们就不会是现在这样。"

"嗯。"

"但你没追。"

"嗯。"

"所以我明天结婚。"

"嗯。"

"再见。"

"等一下。"他叫住她，"你幸福吗？"

"幸福。"她没有犹豫。

"那就好。"他笑了一下。电话挂了。

她站在阳台上吹了很久的风。
回到房间的时候未婚夫已经洗完澡了。
"谁的电话？"

"推销的。"她说。
然后坐到他身边，把头靠在他肩膀上。
"明天我要迟到了你叫我。"

"嗯。"
```

**技法要点**：前任对话的核心是"未完成的遗憾"——追出来/没追出来这一件事决定了两个人的一生；"说一句不客气的话"是全场景的情感爆发点——客气的祝福比骂人更伤人；结尾靠在未婚夫肩上，用"幸福"的当下覆盖"遗憾"的过去。

## 5. 父母离婚时孩子说的话

**场景设定**：客厅里，父母正在谈离婚的细节。十四岁的儿子从房间里出来。

```
"房子归我，车归你，孩子——"

"孩子跟我。"

"凭什么跟你？"

"因为——"

"我跟谁都不跟。"

两个人同时转头。儿子站在房间门口。校服还没换。书包挂在一边肩膀上。

"小杰，你怎么——你不是说今天补课到九点吗？"

"老师请假了。"他走过来，坐在沙发扶手上，"你们继续。"

"你先回房间。"妈妈说。

"不要。"他把书包放在地上，"你们聊。我听着。"

"小杰——"

"我说了，我听着。"他的语气不像十四岁。像四十四岁。

爸爸看了妈妈一眼。"你出去跟同学玩会儿？"

"我没有同学。"

"你不是跟那个——"

"我说了我没有同学。"他重复了一遍。声音不大。但客厅安静了。

他看了一眼茶几上的离婚协议书。A4纸，好几页。
"房子归谁我不关心。车归谁我也不关心。"

"那你在乎什么？"爸爸问。

"我什么都不在乎。"

"小杰——"

"但我想问一件事。"他看着茶几上的杯子。爸爸的保温杯，妈妈的马克杯。他的——他没有杯子。他从来不在这儿喝水。

"你们什么时候开始的？"

"什么开始？"

"你们什么时候开始不想一起过的？"

沉默。

"一年前？两年前？我生日那天你们吵架的时候？"

"小杰，大人的事情——"

"我十四了。我不是小孩。"他站起来，"你们可以离。随便。但能不能告诉我一个时间？我想知道——这一年我在家吃饭的时候，你们在对面坐着，是在演戏还是真的。"

没有人说话。

"算了。"他拎起书包往房间走。走到门口停了一下。
"车给爸吧。妈你不会开车。"

然后他关上了门。

客厅里两个人对坐着。茶几上的离婚协议书被空调吹翻了一页。
```

**技法要点**：孩子的台词越冷静越让人心碎——十四岁的孩子用"我什么都不在乎"掩盖"我在乎但说出来太痛"；"演戏还是真的"是整个场景最重的一句话，直指父母最大的愧疚；最后的"车给爸吧，妈你不会开车"是用实用细节表达关心，比任何"我爱你们"都更真实。

## 对话写作核心原则总结

| 原则 | 说明 |
|------|------|
| **潜台词>直说** | "再坐一会儿"="我也喜欢你"，"别打了"="我也有话要说" |
| **沉默是台词** | 异地恋的"嗯"、便利店的无言——不说话的时候信息量更大 |
| **身体比嘴诚实** | 耳朵红、手心出汗、靠在肩上——身体在替嘴巴说话 |
| **一个字的余味** | "嗯。"的分手、"幸福"的果断——字越少情绪越重 |
| **用日常物件承载感情** | 旧卫衣、保温杯、乌龙茶——东西比文字更持久 |
