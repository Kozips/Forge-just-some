> 本文件整合情感类核心素材，包括意象速查、情感→行为速查、关系模式速查、对话技法。引用自蒸馏精华及各research子目录。

## 第一章：意象速查词典

### 1.1 自然意象表

| 意象 | 情感含义 | 使用示例 |
|------|---------|---------|
| 月亮 | 思念、孤独、团圆、阴柔 | 残月=残缺/离别；满月=团圆/圆满 |
| 雨 | 悲伤、洗涤、阻隔、净化 | 小雨=愁绪；暴雨=情感爆发；雨后=新生 |
| 雪 | 纯洁、寂寥、寒冷、遗忘 | 初雪=纯洁/新生；暴雪=隔绝/绝望 |
| 风 | 自由、无常、传递、力量 | 微风=温柔；狂风=命运力量；无风=压抑 |
| 花 | 美丽、短暂、爱情、女性 | 落花=无常/离别；花开=希望/重逢 |
| 树 | 生命、扎根、成长、庇护 | 枯树=死亡/衰败；古树=记忆/坚守 |
| 河/水 | 时间、命运、阻隔、情感流动 | 静水=深沉；激流=命运冲突；干涸=绝望 |
| 山 | 坚定、阻碍、崇高、孤独 | 高山=理想/信仰；荒山=孤寂；翻山=克服困难 |
| 海 | 广阔、未知、自由、危险 | 大海=命运/自由；海啸=毁灭；平静海=宁静 |
| 星 | 希望、指引、遥远、永恒 | 星空=无限可能；流星=短暂美好；孤星=孤独 |
| 火 | 激情、毁灭、温暖、净化 | 篝火=温暖/团聚；野火=毁灭/失控；余烬=残存 |
| 路/桥 | 选择、连接、旅程、过渡 | 岔路=人生选择；断桥=关系断裂；长路=漫长旅程 |
| 雾 | 迷茫、未知、隐藏、朦胧 | 浓雾=迷失方向；薄雾=诗意朦胧 |
| 黎明/黄昏 | 希望/结束、过渡、转变 | 黎明=新生/希望；黄昏=暮年/告别 |
| 季节 | 生命阶段、情感周期 | 春=新生/爱情；夏=热烈/激情；秋=成熟/离别；冬=死亡/考验 |

## 第三章：关系模式速查

### 爱情关系10种模式
禁忌之恋 | 误解到理解 | 错过与重逢 | 守护与被守护 | 互相救赎 | 虐恋纠葛 | 天降vs青梅 | 平行线之爱 | 面具之爱 | 超越生死

## 第四章：对话技法速查

### 4.1 潜台词对照表

| 表面 | 潜台词 |
|------|-------|
| "你走吧，我不管你了。" | 你别走。 |
| "这个便当反正做多了。" | 我特意给你做的。 |
| "谁要想你。" | 我已经在想了。 |
| "路上小心。" | 你活着回来。 |
| "嗯。" | （一万个字在心里） |
| "随便你。" | 我在乎到生气了。 |
| "你还好吗？" | 我不好，你呢？ |
| "没事，我就是问问。" | 我担心了很久终于忍不住问了。 |
| "你开心就好。" | 你开心我才能开心。 |
| "我没有生气。" | 我非常生气但我在忍。 |
| "早点回来。" | 我在家等你。 |

### 4.2 同一句话在不同情境下的含义变化

> "早点回来。"
> 恋爱时——甜蜜。
> 吵架后——和解。
> 住院时——祈求。
> 去世后——习惯。（她还是会对着空房间说"早点回来"。）

> "我在。"
> 开心时——陪伴。
> 害怕时——安心。
> 绝望时——支撑。
> 最后一次——告别。（"我在。"他说。然后电话断了。）

## 第五章：感官描写词库

### 5.1 视觉描写
1. 眼眶红了但没哭——比嚎啕大哭更让人心碎
2. 笑容到一半僵住，慢慢收回去
3. 瞳孔微微放大——震惊时身体比脑子先反应
4. 嘴唇动了动，最终什么都没说
5. 指尖轻轻碰到对方手背，又缩回去
6. 肩膀在发抖，但声音是稳的
7. 反复翻同一页书，一个字也没读进去
8. 对着镜子练习微笑，练到嘴角发酸
9. 把合影照片面朝下扣在桌上
10. 站在雨里没动，衣服全湿了才发现

### 5.2 听觉描写
1. 电话那头的沉默比哭声更刺耳
2. 呼吸突然变重了——不是累，是忍着不哭
3. 关门声。不是摔的，是轻轻带上的，反而更可怕
4. 脚步声走到门口停住了，然后往回走了
5. 笑声笑到一半断了，变成一声叹息

### 5.3 触觉描写
1. 拥抱时感受到对方的心跳——砰砰砰，比自己的快
2. 被子的温度——另一半的被窝已经凉了
3. 把脸埋进他衣服里，洗衣液的味道还没散
4. 泪水滑到嘴角——温的，咸的
5. 拇指擦过对方的眼泪，粗糙的指腹带着小心翼翼

### 5.4 嗅觉描写
1. 烟味——他戒了两年，分手后又开始抽了
2. 枕头上残留的洗发水味道，闻着闻着就哭了
3. 雨后泥土的气味——小时候妈妈在门口等他放学的记忆

### 5.5 味觉描写
1. 亲吻时尝到他嘴角的咖啡味
2. 蛋糕很甜，但她吃了一口就放下了——去年这时候是两个人吃的
3. 嘴里咬破了——吵架时咬着牙忍怒气，结果咬到了舌头
