# 情感类 — 素材库

> 本目录存放可直接复用的素材片段：场景描写、对话模板、氛围营造等。

## 暧昧试探素材

**适用**：两人关系尚未明确，充满不确定

> 他在地铁上给她发了条消息："今天冷。"她回："嗯。"他打了"多穿点"，又删掉了。然后打了"我也觉得冷"，又删掉了。最后发了个太阳的表情。她回了个笑脸。他把手机揣回口袋，觉得车厢里好像暖了一点。

## 分手余波素材

**适用**：关系结束后角色独自消化的阶段

> 她删了他的微信。然后打开朋友圈——他的朋友圈空了。不对，是她看不到他的朋友圈了。她又加了他。验证消息写了"手滑了"。他通过了。她看到他的朋友圈更新了一张猫的照片。配文是："新成员。"她退出微信，把手机扔到沙发另一头，拿起遥控器开始调台。调了三十个台，没一个能看。

## 深夜思念素材

**适用**：角色在孤独中想起对方

> 凌晨三点她醒来，习惯性地往右边看了一眼。枕头是平的。她翻了个身，拿起手机，打开相册——最近删除里还有一张合影。恢复。放大。看他的脸。看了大概两分钟，又删掉了。然后把手机扣在胸口，盯着天花板，等天亮。
