# 情感类 — 核心写作规则

> 本目录存放该类型的铁律与不可违反的规则。开写前必须阅读。

## 规则1：情感必须通过行为和细节展现，不能直接抒发

**规则**：不写"她很难过"，而是写她做了什么——反复按电梯按钮、删了又打的微信消息、半夜洗了三遍碗。
**正例**：他坐在床边，把手机解锁又锁上，解锁又锁上。聊天记录停在七天前她发的那条"嗯"。
**反例**：他非常伤心，眼泪止不住地流。他想：我好爱她啊。
**违反后果**：变成矫情文学，读者无法共情。

## 规则2：情感变化必须有触发事件

**规则**：角色的情感转折不能突然——必须有一个具体的触发点（一句话/一个画面/一个发现）。
**正例**：她一直以为他不在乎。直到她在医院醒来，看到他趴在床边睡着了，手里还攥着缴费单。
**反例**：她突然就不爱他了。她突然就原谅他了。
**违反后果**：角色像木偶，情感没有说服力。

## 规则3：双向付出必须对等但不对称

**规则**：双方都在付出，但方式不同——不能一方一直在牺牲，另一方一直在接受。
**正例**：他不会说甜言蜜语，但每天早上五点起来给她热牛奶。她不会做饭，但把他所有衣服按颜色分类叠好。
**反例**：她为他放弃了一切，他什么都没做，最后他说"我也是爱你的"。
**违反后果**：读者愤怒于不对等，无法认同这段感情。

## 规则4：沉默比对话更有力

**规则**：在关键情感场景中，"没说出口的话"比"说出来的"更动人。
**正例**：分别的时候，他想说"别走"。嘴张了张，说的是"路上小心"。
**反例**："我爱你，我不能没有你，求你不要离开我！"他哭着喊道。
**违反后果**：情感表达廉价化，失去重量。
