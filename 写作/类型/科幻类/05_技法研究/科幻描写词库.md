# 科幻描写词库

> 科幻写作的描写素材速查。按场景类型分类，包含直接可用的描写方式、词组、技法。

---

## 一、科技描写速查

### 飞船

| 部位 | 描写方式 | 词组 |
|------|---------|------|
| 外壳 | 金属质感+光线反射 | 钛合金蒙皮、碳纤维骨架、隔热瓦的蜂窝纹路、舷窗边缘的密封胶 |
| 引擎 | 声音+震动+温度 | 离子引擎的蓝色尾焰、低频震动从脚底传上来、引擎舱的灼热空气、聚变反应堆的嗡鸣 |
| 舰桥 | 光源+界面+声音 | 全息投影的蓝光映在脸上、触控屏上的数据流、警报的短促蜂鸣、指挥椅的皮革摩擦声 |
| 走廊 | 空间+光线+气味 | 金属地板的回响、LED灯带的冷白色、循环空气的臭氧味、舱门气密的嘶声 |
| 生活区 | 人造感+压抑 | 人造重力的轻微不适、隔间太小转不开身、隔音墙后面的引擎声、植物舱的泥土味 |

**描写公式**：飞船=金属+光+声音+震动。用"震动"写动力——引擎启动时牙齿在震动，这是最直接的"飞船活了"的信号。

### 机器人

| 类型 | 描写重点 | 词组 |
|------|---------|------|
| 工业机器人 | 力量+精确 | 液压关节的嘶声、伺服电机的精确转动、钛合金手指、动作没有多余 |
| 人形机器人 | 恐怖谷+模仿 | 皮肤太光滑、眨眼频率不对、笑容持续时间太长、声音没有呼吸感 |
| AI管家 | 无处不在+温和压迫 | 声音从墙壁里传出、灯光自动调节、"我注意到您最近睡眠不足"、温度恒定22度 |
| 战斗机器人 | 效率+冷酷 | 动作没有犹豫、光学传感器的红光、金属关节的咔嗒声、倒地后还在执行最后一条指令 |

**描写公式**：机器人=精确-人性。描写机器人的关键是"差一点就完美"——动作太流畅、声音太均匀、反应太快。人类的特征是有瑕疵，机器人的特征是无瑕疵。

### AI/虚拟现实

| 场景 | 描写方式 | 词组 |
|------|---------|------|
| AI对话 | 声音+节奏 | 合成音的均匀语调、回答太快没有思考停顿、"我理解您的感受"但语气没有变化 |
| 虚拟空间 | 感官过载+规则可改 | 像素边缘的锯齿、重力可以调整、颜色太鲜艳不真实、退出时的眩晕 |
| 数据流 | 视觉化+速度 | 蓝色光线从眼前流过、信息像瀑布一样倾泻、下载时太阳穴的胀痛、视野边缘的噪点 |
| 意识上传 | 身体感消失 | 手指穿过物体、感觉不到温度、视野没有边界、"我"变成了一个点 |

**描写公式**：虚拟现实=感官过载+不真实感。用"太完美"写虚拟——颜色太饱和、声音太清晰、触感太均匀。真实世界有噪点，虚拟世界没有。

### 基因改造

| 阶段 | 描写方式 | 词组 |
|------|---------|------|
| 改造前 | 身体的旧状态 | 原始基因的限制、肌肉的自然疲劳、视力的模糊、衰老的皱纹 |
| 改造中 | 疼痛+变化 | 注射时的灼烧感、骨骼在生长的胀痛、皮肤下的纳米机器蠕动、发烧般的热度 |
| 改造后 | 能力+异样 | 视力清晰到能看到毛孔、肌肉不再疲劳、伤口愈合的速度、镜子里的自己变了 |
| 代价 | 失去+异化 | 情绪波动减弱、对旧食物失去味觉、需要定期注射维持剂、"我不再完全是人类" |

**描写公式**：基因改造=获得+失去。每项增强都伴随一项丧失——看得更远但失去色彩感知、更强壮但失去痛觉。用"失去"写代价。

## 二、太空环境描写

### 失重

| 感官 | 描写方式 | 词组 |
|------|---------|------|
| 视觉 | 物体悬浮 | 头发像水草一样飘起来、咖啡从杯口溢出变成球状、工具在空中缓慢旋转 |
| 触觉 | 身体感改变 | 胃在翻腾、内脏在漂浮、抓握时身体反而飘走、脚底没有压力感 |
| 听觉 | 空间感改变 | 声音传播更均匀、没有上下之分、液体在管道里的咕噜声 |
| 心理 | 失去锚点 | 分不清上下、本能地想抓住什么、第一次吐在头盔里的恐惧 |

**描写公式**：失重=失去参照系。用"液体不落地"写物理失重，用"分不清上下"写心理失重。最恐怖的失重不是漂浮，是不知道哪边是"下"。

### 真空

| 感官 | 描写方式 | 词组 |
|------|---------|------|
| 视觉 | 无声+黑暗 | 爆炸没有声音、星星不闪烁、阴影边缘锐利、阳光直射的白和阴影的绝对黑 |
| 触觉 | 温度极端 | 阳面灼热阴面冰冷、宇航服内的汗水、面罩起雾又结冰 |
| 心理 | 绝对孤独 | 通讯延迟、呼吸声是唯一的声音、面罩里的二氧化碳味、"外面什么都没有" |

**描写公式**：真空=无声。太空中的爆炸没有声音——这是最反直觉也最震撼的描写。飞船被击中，你看到火光，但什么都没听到。三秒后震动通过船体传过来。

### 星球表面

| 环境 | 描写方式 | 词组 |
|------|---------|------|
| 荒漠星球 | 颜色+温度 | 红色沙丘、两个太阳、空气灼热、岩石表面的风蚀纹路 |
| 冰冻星球 | 白色+声音 | 白色延伸到地平线、冰层下的气泡、呼吸结冰、远处的冰裂声 |
| 沼泽星球 | 湿度+气味 | 空气黏稠、腐烂植物的甜腻味、水面下的发光生物、脚踩下去会陷 |
| 气态行星 | 颜色+风暴 | 氨气云层的黄褐色、闪电在云层中穿行、气压在增加、下降越来越快 |

**描写公式**：异星=一个感官锚点。每颗星球用一个独特的感官特征锚定——奇怪的天空颜色、奇怪的气味、奇怪的重力。读者记住的不是"荒漠"，是"红色的天空"。

### 太空站内部

| 区域 | 描写方式 | 词组 |
|------|---------|------|
| 生活区 | 拥挤+人造 | 隔间太小、人造植物、循环空气的金属味、墙壁上的家庭照片 |
| 实验室 | 洁白+精确 | 无菌灯的蓝白色、仪器的滴答声、试管架的金属光泽、手套箱里的操作 |
| 观测台 | 宏大+孤独 | 整面墙都是星空、地球在下面转动、沉默、玻璃上映出自己的脸 |
| 维修通道 | 危险+狭窄 | 只能侧身通过、管道的热气、焊花的闪光、头盔撞到管壁的闷响 |

**描写公式**：太空站=人造+封闭。用"循环空气"写封闭感——空气是被过滤过的，带着金属味和消毒水味。用"墙上的照片"写人造感——人类试图在太空中制造"家"的感觉。

## 三、科幻对话风格

### 技术术语融入对话的方式

**原则：不解释，让读者从上下文理解。**

| 方式 | 示例 | 效果 |
|------|------|------|
| 自然使用 | "把量子纠缠链路断开"——角色不会解释什么是量子纠缠 | 读者从"断开"推断这是一种通讯方式 |
| 对比解释 | "比光速快，但比虫洞慢"——用已知概念锚定未知概念 | 读者自动建立速度阶梯 |
| 错误理解 | "你管这叫AI？这顶多是个高级计算器"——用角色的争论展示技术等级 | 争论本身就是世界观展示 |
| 日常化 | "帮我查一下暗物质的汇率"——把科幻概念当日常用品用 | 暗示这个世界暗物质已经商品化 |

### 对话节奏模板

**硬科幻对话**：短句+数据
```
"轨道偏移0.3度。"她盯着屏幕。
"0.3度够吗？"
"够我们撞上月球。"
```
特征：每句话都带精确信息，没有废话。

**赛博朋克对话**：黑话+交易
```
"芯片植入费多少？"
"看你想要什么。记忆增强三千，战斗模块一万二。"
"最便宜的。"
"两千。但副作用是——"
"不关心。"
```
特征：价格驱动，副作用被忽略——穷人没有选择的余地。

**AI对话**：精确+冷漠
```
"我分析了1247种可能。"
"结果呢？"
"你们称之为'希望'的那种——概率为零。"
```
特征：数字精确，情感词汇被加引号——AI在"引用"人类的概念，而非"感受"它。

**太空歌剧对话**：格言+沉默
```
"你在害怕什么？"
"害怕有一天，我们变成我们曾经反抗的那种人。"
长久的沉默。
"那我们最好确保那一天不会来。"
```
特征：关键台词像格言，沉默比语言更有重量。

## 四、身体改造/义体描写

### 改造层级

| 层级 | 描写重点 | 词组 |
|------|---------|------|
| L1 表层植入 | 皮肤下的异物感 | 芯片植入处的微热、皮下LED灯的蓝光、接口边缘的疤痕组织 |
| L2 肢体替换 | 机械与肉体的交界 | 义肢关节的咔嗒声、金属手指的精确但冰冷、接口处的红肿、"这只手不是我的" |
| L3 器官替换 | 生命维持的依赖 | 人工心脏的稳定节拍、合成肺的呼吸声更均匀、需要定期充电/维护 |
| L4 意识改造 | 身份的边界 | 记忆可以被编辑、情感可以被调节、"我是我吗"、镜子里的脸变了 |
| L5 全身义体化 | 人的定义模糊 | 只剩大脑是原装的、身体可以更换、"我"在哪里、永生但不再是人 |

### 改造后的感官描写

| 感官 | 增强 | 代价 |
|------|------|------|
| 视觉 | 热成像、夜视、望远 | 眼睛发出红光、看到太多信息导致头痛、无法关闭增强视觉 |
| 听觉 | 超声波、频率过滤 | 能听到心跳声、噪音过载、无法"不听" |
| 触觉 | 压力感应、温度精确 | 感觉不到冷热、触感太精确反而不适、义肢没有"手感" |
| 力量 | 液压增强、合金骨骼 | 握碎杯子、拥抱时需要控制力度、动作太精确反而不自然 |

### 改造的代价描写

**身体代价**：接口感染、排异反应、电池耗尽时的无力、需要定期更换零件
**心理代价**：镜子里的陌生人、"我"的记忆还是"程序"的记忆、梦里用旧身体
**社会代价**：被视为"非人"、无法通过人类身份验证、被歧视或被崇拜

## 五、未来城市/建筑描写

### 城市分层

| 层级 | 描写重点 | 词组 |
|------|---------|------|
| 天空层 | 富人区+清洁 | 摩天大楼的玻璃幕墙、空中花园、私人飞行器、空气过滤后的清甜 |
| 地面层 | 中产+拥挤 | 全息广告的霓虹、人流密集、自动贩售机、混合现实的导航线 |
| 地下层 | 穷人+肮脏 | 潮湿的墙壁、管道的滴水声、非法改装店、空气浑浊 |
| 废墟层 | 被遗忘+危险 | 旧世界的遗迹、灰尘覆盖的街道、野草从裂缝中生长、时间停止的感觉 |

### 建筑特征

| 元素 | 描写方式 | 词组 |
|------|---------|------|
| 材质 | 未来感+冰冷 | 自修复混凝土、光敏玻璃、碳纳米管骨架、表面没有接缝 |
| 光源 | 人造+无影 | 全息投影替代路灯、建筑外墙是显示屏、没有阴影的均匀照明 |
| 交通 | 立体+自动 | 磁悬浮管道、无人机快递的嗡嗡声、自动驾驶的安静滑行 |
| 界面 | 无处不在 | 空气中的全息菜单、墙壁是触控屏、语音助手从每个角落回应 |

### 城市氛围模板

**繁华型**：霓虹灯+全息广告+人流+噪音——用过载写繁荣
**压抑型**：灰色建筑+监控摄像头+安静+巡逻无人机——用安静写控制
**衰败型**：废弃建筑+野草+灰尘+偶尔的流浪者——用静止写死亡

## 六、科幻感营造技法

### 三秒规则

**原则：科幻设定用3秒能说完，超过就是说教。**

| 场景 | 错误示范 | 正确示范 |
|------|---------|---------|
| 全息电话 | "全息投影通讯技术利用量子纠缠原理实现即时通讯" | "她的全息影像站在桌子对面，像真人一样，除了脚没有影子" |
| 义肢 | "这是第七代仿生义肢，采用纳米碳纤维材料" | "他用金属手指捏碎了核桃，然后小心地捡起碎片——怕捏碎碎片" |
| 时间旅行 | "时间旅行需要足够的负能量来维持虫洞的稳定" | "他回到昨天，发现自己已经在那里了" |

**核心技法**：用旧事物描述新事物。
- "全息影像像真人一样，除了脚没有影子"——用"影子"这个旧概念写新科技
- "AI管家的声音像一个永远不生气的妈妈"——用"妈妈"写AI的温和压迫
- "太空行走像潜水，但没有水压，也没有底"——用"潜水"写太空的失重+无边界

### 感官锚点

**原则：每颗星球/飞船/未来城市用一个独特的感官特征锚定。**

| 场景 | 锚定感官 | 示例 |
|------|---------|------|
| 火星 | 味道 | "空气有铁锈味，像舔电池" |
| 太空站 | 声音 | "永远有低频嗡鸣，像冰箱放大了一百倍" |
| 赛博城市 | 颜色 | "霓虹灯把雨天染成紫色" |
| 冰冻星球 | 触觉 | "空气冷到鼻毛结冰" |
| 气态行星 | 视觉 | "云层像被搅拌的咖啡，黄色和棕色旋在一起" |

### 日常化技术

**原则：新技术用旧方式描述，让读者瞬间理解。**

| 新技术 | 旧类比 | 效果 |
|--------|--------|------|
| 全息通话 | "像微信视频，但她的影像站在你面前" | 读者立刻理解是3D视频 |
| 记忆芯片 | "像U盘，但存的是记忆" | 读者理解是可转移的记忆 |
| 反重力 | "像磁悬浮列车，但车消失了，只剩人" | 读者理解是人悬浮 |
| 基因编辑 | "像Photoshop，但编辑的是DNA" | 读者理解是可修改的基因 |

### 数据精确化

**原则：用数字替代形容词。**

| 模糊 | 精确 | 效果 |
|------|------|------|
| 很快爆炸 | 3.7秒后爆炸 | 倒计时感 |
| 距离很远 | 4.2光年 | 宇宙尺度感 |
| 温度很低 | 零下127度 | 精确恐惧 |
| 速度很快 | 0.3倍光速 | 物理边界感 |
| 概率很小 | 0.0003% | 绝望感 |

### 代价先于能力

**原则：先展示技术的代价，再展示技术的能力。**

| 能力 | 代价（先写） | 能力（后写） |
|------|------------|------------|
| 增强视觉 | "术后三天不能见光" | "然后你能在黑暗中看到老鼠的胡须" |
| 超级力量 | "每次发力后肌肉会撕裂" | "但那一拳能打穿钢板" |
| 意识上传 | "原体会在上传后死亡" | "但你的副本可以活一万年" |
| 时间旅行 | "每次跳跃都会忘记一些记忆" | "但你能改变过去" |

## 速查：科幻描写万能公式

| 公式 | 用法 | 示例 |
|------|------|------|
| 旧物喻新 | 用旧事物描述新事物 | "全息影像像真人，除了没有影子" |
| 感官锚点 | 一个感官特征锚定整个场景 | "火星有铁锈味" |
| 数据精确 | 数字替代形容词 | "3.7秒后爆炸" |
| 代价先写 | 先写代价再写能力 | "术后三天失明，然后你能夜视" |
| 恐怖谷 | "差一点就完美"的不适 | "她的笑容持续了4.7秒——太长了" |
| 震动写活 | 用震动写飞船/机器"活着" | "引擎启动时牙齿在震动" |
| 无声爆炸 | 太空中没有声音 | "他看到火光，但什么都没听到" |
| 循环空气 | 封闭空间的气味 | "空气有金属味，被过滤了太多次" |
