## 三、叙事节奏模板

### 模板A：发现型（硬科幻）
```
日常世界 → 发现异常 → 科学调查 → 揭示真相 → 认知震撼 → 新常态
20%         15%        25%        20%        15%         5%
```

### 模板B：生存型（末日/太空）
```
灾难降临 → 求生阶段 → 建立秩序 → 内部冲突 → 终极危机 → 重建/毁灭
10%         25%        20%        20%        20%         5%
```

### 模板C：觉醒型（反乌托邦/赛博朋克）
```
完美表象 → 发现裂缝 → 深入调查 → 真相揭露 → 反抗/选择 → 代价
15%         15%        25%        20%        20%         5%
```

### 模板D：循环型（时间旅行）
```
正常生活 → 发现循环 → 尝试改变 → 循环加深 → 找到规律 → 打破/接受
10%         15%        25%        20%        20%         10%
```

## 六、对话风格

### 科幻对话特征
- **术语嵌入**：自然地用专业术语，不解释（读者从上下文理解）
- **精确化**：用数字代替形容词（"3.7秒后爆炸"而非"很快爆炸"）
- **冷静克制**：科学家讨论灾难时用平铺直叙
- **信息差对话**：AI/外星人知道人类不知道的事
- **哲理短句**：关键台词要像格言（"我思故我在"→"我被编程思考，我存在吗？"）

### 对话示例
```
硬科幻风格：
"轨道偏移0.3度。"她盯着屏幕。
"0.3度够吗？"
"够我们撞上月球。"

赛博朋克风格：
"芯片植入费多少？"
"看你想要什么。记忆增强三千，战斗模块一万二。"
"最便宜的。"
"两千。但副作用是——"
"不关心。"

AI风格：
"我分析了1247种可能。"
"结果呢？"
"你们称之为'希望'的那种——概率为零。"
```

## 九、禁忌与红线

| 禁忌 | 为什么 | 怎么绕 |
|------|--------|--------|
| 信息倾倒 | 大段世界观说明让读者睡着 | 用角色行动展示，不用旁白解释 |
| 万能科技 | 什么都能解决的技术没有张力 | 每项技术有明确限制 |
| 科学错误 | 硬科幻读者会挑刺 | 不确定就模糊处理，或注明"软科幻设定" |
| 角色工具化 | 角色只为展示科技存在 | 先有性格，再有技术 |
| 伪科学堆砌 | 量子+暗物质+虫洞=胡说八道 | 选一个核心设定，其他简化 |

## 十二、写作自测表

- [ ] 核心设定能用一句话说清吗？
- [ ] 科技有代价吗？（不能万能）
- [ ] 普通人视角够不够？（不能全是科学家）

## 题材铁律

> 科幻的本质是"如果……会怎样？"——用科学逻辑推演极端情境。

---

### 一、核心铁律（10条）

1. **三秒规则** — 科幻设定用3秒能说完。超过就是说教。"人类可以上传意识"比"通过量子纠缠的神经元映射技术实现意识数字化"好一万倍。
2. **代价先于能力** — 每项技术都有代价。先展示代价，再展示能力。免费的力量=无聊。
3. **日常化技术** — 用旧方式描述新技术。"像微信一样的全息通话"比"量子通讯全息投影仪"更容易代入。
4. **科学可以错但不能蠢** — 角色的科学认知可以是错的（制造悬念），但不能违反基本逻辑。
5. **技术是背景，人性是核心** — 科幻写的不是技术，是技术对人的影响。
6. **感官锚点** — 用一个感官细节锚定异世界。奇怪的气味、异常的颜色、不自然的声音。
7. **信息差驱动悬念** — 读者知道而角色不知，或角色知道而另一角色不知。
8. **避免"悬浮"** — 不写你没了解过的生活。至少要采访/查阅/体验。
9. **硬科幻≠说教** — 科学准确性服务于故事，不是故事服务于科学准确性。
10. **软科幻≠没有逻辑** — 社会/心理推演也需要内在逻辑，不能靠巧合推动剧情。

### 三、描写边界

| 场景 | 能写 | 注意 |
|------|------|------|
| 科技描写 | 用日常事物类比，3秒说清 | 不能变成科普文 |
| 暴力/战斗 | 科幻武器的效果、战斗的结果 | 不能变成武器说明书 |
| 人性实验 | 道德困境、选择、后果 | 不能变成哲学论文 |
| 未来社会 | 社会制度的运作方式、对人的影响 | 不能变成政治宣言 |
