# 科幻类 · 写作模板

> 科幻写作的章节结构模板。写作前用模板规划节奏。

---

## 一、单章核心张力

每章必须回答一个问题：**这一章里，主角最怕失去什么？**

| 张力类型 | 说明 | 典型场景 |
|----------|------|---------|
| 生存张力 | 不解决就死 | 飞船故障/辐射泄漏/氧气耗尽 |
| 真相张力 | 不查就永远被蒙在鼓里 | 世界真相/AI阴谋/政府隐瞒 |
| 选择张力 | 不选就有不可逆后果 | 牺牲一人救多人/保留人性还是进化 |
| 时间张力 | 不行动就来不及 | 倒计时/星球撞击/病毒扩散 |
| 身份张力 | 不面对就失去自我 | 意识上传后还是我吗/改造后还是人吗 |

## 二、单章结构模板

每章必须回答：**这一章结束时，读者对世界的认知改变了什么？**

### 模板A：发现型（硬科幻/悬疑科幻）

**适用**：角色发现某个真相/异常/秘密

```
日常锚定(10%) → 异常信号(15%) → 调查验证(25%) → 真相揭示(25%) → 认知崩塌(15%) → 新常态(10%)
```

**关键动作**：
| 阶段 | 写什么 | 注意 |
|------|--------|------|
| 日常锚定 | 用一个感官细节建立"正常" | 不超过3段，快速进入异常 |
| 异常信号 | 一个数字/现象/对话"不对" | 用精确数字制造不安（"温度偏了0.3度"） |
| 调查验证 | 角色用科学方法验证 | 读者跟着角色一起"算" |
| 真相揭示 | 真相比猜测更震撼 | 真相必须推翻角色之前的假设 |
| 认知崩塌 | 角色对世界的理解被动摇 | 不用写"他震惊了"，写他的手在抖 |
| 新常态 | 角色带着新认知继续生活 | 但生活已经不一样了 |

### 模板B：生存型（末日/太空危机）

**适用**：角色面临生死危机，必须在有限条件下求生

```
危机爆发(15%) → 资源清点(15%) → 求生行动(25%) → 代价与损失(20%) → 绝境抉择(15%) → 新平衡(10%)
```

**关键动作**：
| 阶段 | 写什么 | 注意 |
|------|--------|------|
| 危机爆发 | 用感官写灾难——声音/震动/光线 | 不要解释原因，先让读者感受到 |
| 资源清点 | 角色盘点可用资源 | 用数字制造紧迫（"氧气够3个人撑4小时"） |
| 求生行动 | 角色尝试各种方案 | 每个方案都有代价，没有完美解 |
| 代价与损失 | 有人受伤/死亡/失去资源 | 生存必须有代价，否则没有重量 |
| 绝境抉择 | 角色必须在两个坏选项中选 | 不是善vs恶，是坏vs更坏 |
| 新平衡 | 危机暂时解决，但失去了什么 | 活下来的人比之前少了 |

### 模板C：觉醒型（AI/反乌托邦/身份科幻）

**适用**：角色发现世界的真相/自己的身份/体制的谎言

```
完美表象(10%) → 第一道裂缝(15%) → 深入调查(25%) → 真相揭露(25%) → 反抗/抉择(15%) → 代价(10%)
```

**关键动作**：
| 阶段 | 写什么 | 注意 |
|------|--------|------|
| 完美表象 | 用"过度美好"制造不安 | 空气太甜、笑容太久、一切都"刚刚好" |
| 第一道裂缝 | 一个细节"不对" | 不要太大，一个小小的异常就够了 |
| 深入调查 | 角色越查越深，危险越来越大 | 有人试图阻止角色调查 |
| 真相揭露 | 真相比想象中更可怕 | 真相必须和角色的"存在"有关 |
| 反抗/抉择 | 角色必须选择：接受谎言还是面对真相 | 两个选项都有代价 |
| 代价 | 反抗成功但失去什么 | 或者——反抗失败，角色被"重置" |

### 模板D：悖论型（时间旅行/因果科幻）

**适用**：角色发现时间/因果出了问题

```
正常生活(10%) → 发现异常(15%) → 尝试修复(25%) → 悖论出现(25%) → 代价(15%) → 接受/打破(10%)
```

**关键动作**：
| 阶段 | 写什么 | 注意 |
|------|--------|------|
| 正常生活 | 角色的日常，有一个"锚点"细节 | 锚点细节会在后面重复出现 |
| 发现异常 | 某件事"不该发生"但发生了 | 用"昨天死的人今天活着"制造恐惧 |
| 尝试修复 | 角色试图"修正"时间线 | 每次修正都导致新的问题 |
| 悖论出现 | 角色发现自己就是"问题"的一部分 | 因果循环，无法逃脱 |
| 代价 | 打破悖论需要牺牲什么 | 记忆/存在/某个重要的人 |
| 接受/打破 | 角色接受命运或找到出路 | 接受比打破更难写，但更有力量 |

## 三、多章弧线模板

### 三幕式（适合单季/短篇，3-5万字）

```
第一幕：世界建立+核心冲突（25%）
  ├─ 第1-2章：建立世界+主角的日常
  ├─ 第3章：核心冲突出现
  └─ 第4章：主角被迫卷入

第二幕：调查/对抗/升级（50%）
  ├─ 第5-7章：调查真相/收集盟友
  ├─ 第8-10章：第一次重大失败
  ├─ 第11-13章：重新振作+新发现
  └─ 第14-15章：真相接近/对手反击

第三幕：最终对决+代价+新世界（25%）
  ├─ 第16-17章：最终对决
  ├─ 第18章：代价
  └─ 第19-20章：新世界/余韵
```

### 英雄之旅改编（适合长篇，10万字+）

**科幻改编要点**：
- **平凡世界**：用"日常化技术"写未来——角色觉得这些技术理所当然
- **异常召唤**：技术故障/科学发现/AI异常——不是"命运"，是"物理定律"
- **导师**：不是智者，是科学家/老飞行员/AI——用知识而非智慧引导
- **最大磨难**：不是战斗，是选择——伦理困境比物理危险更可怕
- **万灵药**：不是宝物，是真相/技术/理念——改变世界的东西

### 无限流副本型（适合连载，每卷独立）

**科幻改编要点**：
- 每个"副本"是一个星球/空间站/时间线
- "规则"是物理定律/社会制度/技术限制
- "隐藏条件"是科学原理/历史真相
- "奖励"是技术/知识/资源

### 时间循环型（适合悬疑科幻）

**设计要点**：
- 每次循环必须有**新的信息增量**——不能只是重复
- 循环中的**不变细节**是线索——为什么这件事每次都一样？
- 打破循环的**代价**是什么？——记忆/存在/某个人
- 最后一次循环和第一次循环的**对比**是最大的震撼

## 四、开篇钩子模板

### 钩子类型1：异常开场

**公式**：第一句话就是不对劲的事。

> "第三十七次醒来的时候，他发现枕头上有一根白头发。他今年十九岁。"
>
> "火星的第一场雨下了三天。但火星没有水。"
>
> "她打开门，发现门外站着自己。"

**适用**：所有科幻子类型

### 钩子类型2：数据开场

**公式**：用一组精确数据开头，制造"这是科学记录"的感觉。

> "日志编号：MK-7742。日期：2187年3月15日。氧气剩余：72小时。船员存活：2/7。我不知道还能写多久。"

**适用**：硬科幻/反乌托邦

### 钩子类型3：日常切入异常

**公式**：前100字正常，第101字开始不对。

> "他起床、刷牙、喝咖啡。咖啡的味道和昨天一样——合成豆、合成奶、合成糖。他看了一眼窗外。天空是紫色的。他放下咖啡。不对。天空应该是蓝色的。"

**适用**：软科幻/AI科幻

### 钩子类型4：倒计时开场

**公式**：从结局/灾难倒叙，制造紧迫感。

> "还有47分钟，地球就会知道它不是宇宙中唯一的文明。还有48分钟，第一颗导弹就会发射。"

**适用**：末日/太空/时间旅行

### 钩子类型5：身份反差

**公式**：角色的表面身份和真实身份形成反差。

> "我是地球上最后一个真人。其他所有人都是AI。但我不知道。直到今天。"

**适用**：赛博朋克/AI科幻

### 钩子类型6：感官锚点开场

**公式**：用一个强烈的感官细节开场，锚定整个世界观。

> "火星有铁锈味。不是比喻——是真的铁锈味。像舔电池。他来这里三年了，已经习惯了。但今天，味道变了。多了一股甜味。火星不应该有甜味。"

**适用**：所有科幻子类型

## 五、结尾钩子模板

| 类型 | 特征 | 适用 |
|------|------|------|
| 认知震撼型 | 真相让读者重新理解整个故事 | 硬科幻 |
| 道德困境型 | 没有正确答案，只有选择 | 软科幻 |
| 开放型 | 留下未解的问题 | 所有科幻 |
| 余韵型 | 安静的场景，让读者自己品味 | 文学科幻 |
| 倒计时型 | 新的威胁出现 | 太空/末日 |

## 六、科幻节奏速查

### 慢热型

**适用场景**：硬科幻、文学科幻、太空歌剧

**节奏特征**：
- 前30%用大量细节铺垫世界观和角色
- 真相/高潮在后30%爆发
- 前期的"无聊"是后期震撼的代价

**示例**：《三体》前1/3都在铺垫物理学，最后的"物理学不存在了"才有力量。

### 快节奏型

**适用场景**：赛博朋克、战斗科幻、末日生存

**节奏特征**：
- 开篇500字内必须有冲突
- 每2000字至少一个转折
- 对话>描写>心理

**示例**：《神经漫游者》开篇就进入黑市交易，节奏从不停下来。

### 波浪型

**适用场景**：太空歌剧、时间旅行、多线叙事

**节奏特征**：
- 紧张→放松→更紧张→更放松→最紧张
- 每个"放松"阶段都在为下一个"紧张"蓄力
- 高潮在最后10%

**示例**：《银河英雄传说》的战斗→政治→战斗→政治→最终决战。

## 七、章节模板自测表

- [ ] 每章回答了一个**认知改变**吗？（读者看完这章知道了什么新东西？）
- [ ] 主角在本章有**明确的目标**吗？
- [ ] 本章的**节奏**和子类型匹配吗？（硬科幻=慢热，赛博朋克=快节奏）
- [ ] 开篇钩子在**100字内**抓住读者了吗？
- [ ] 结尾钩子留下了**未解的问题**吗？
- [ ] 本章的**感官描写**用了至少两种感官吗？
- [ ] 对话**符合角色身份**吗？（科学家不会说拾荒者的话）
- [ ] 有没有**信息倾倒**？（大段世界观说明要拆散到行动中）
