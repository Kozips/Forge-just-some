# 科幻类 · 科幻世界观构建

> 世界观是科幻的地基。世界观不稳，再好的角色和剧情都会飘。
> 本文聚焦"怎么设计一个完整的科幻世界"，不重复科技体系设计（见科幻科技体系设计.md）。

---

## 一、未来社会制度模板

社会制度决定了角色的生存环境——不是选一个"好看"的制度，是选一个"有冲突"的制度。

### 模板A：乌托邦型（完美社会的裂缝）

**核心矛盾**：完美是有代价的，代价由谁承担？

| 要素 | 设计要点 |
|------|---------|
| 表面 | 没有犯罪/贫穷/疾病，一切井然有序 |
| 隐藏代价 | 失去自由/创造力/情感/选择权 |
| 维护手段 | 基因筛选/情感抑制/记忆编辑/社会信用 |
| 裂缝来源 | 一个"不该存在"的人/一件"不该发生"的事 |

**设计检查**：
- 这个社会的"完美"靠什么维持？（技术/制度/基因/药物？）
- 谁在为"完美"买单？（底层/异见者/被牺牲的少数？）
- 第一个发现裂缝的人是谁？他怎么发现的？

### 模板B：反乌托邦型（极权控制社会）

**核心矛盾**：控制越严密，反抗越激烈。

| 要素 | 设计要点 |
|------|---------|
| 控制手段 | 全民监控/思想审查/语言限制/历史篡改 |
| 权力结构 | 金字塔型：领袖→核心层→执行层→民众 |
| 宣传口号 | "稳定高于自由""安全高于隐私""集体高于个人" |
| 反抗形式 | 地下组织/信息传播/从内部瓦解 |

**设计检查**：
- 控制手段的**技术基础**是什么？（AI监控/脑机接口/基因服从？）
- 权力阶层内部有没有**分裂**？（有分裂才有故事空间）
- 民众中有没有**不被控制的人**？（孩子/外来者/技术故障者？）

### 模板C：赛博朋克型（企业寡头社会）

**核心矛盾**：企业比政府强大，技术比法律有效。

| 要素 | 设计要点 |
|------|---------|
| 权力中心 | 巨型企业集团，政府沦为傀儡 |
| 社会分层 | 天空层（企业高管）→地面层（普通员工）→地下层（失业者/黑客） |
| 经济形态 | 数据是货币，注意力是资源，身体是商品 |
| 反抗者 | 黑客/拾荒者/前企业员工/觉醒AI |

**设计检查**：
- 有哪些**企业巨头**？各自控制什么领域？
- 地下经济靠什么**运转**？（黑市芯片/非法改装/数据交易？）
- 反抗者的**技术手段**是什么？（不能只有意志力）

### 模板D：星际帝国型（多星球政治体）

**核心矛盾**：距离产生分裂，统一需要代价。

| 要素 | 设计要点 |
|------|---------|
| 统一方式 | 军事征服/经济联盟/文化同化/宗教统一 |
| 分裂来源 | 文化差异/资源争夺/独立运动/通讯延迟 |
| 治理难题 | 星球间的距离=信息延迟=地方自治=分裂风险 |
| 边疆形态 | 边缘星球自治/贸易路线争夺/海盗/走私 |

**设计检查**：
- 帝国靠什么**维持统一**？（军事/经济/文化/宗教？）
- 有哪些**分裂势力**？他们的诉求是什么？
- 星际间的**通讯和交通**速度如何影响治理？

### 模板E：无政府型（去中心化社会）

**核心矛盾**：自由的代价是混乱，秩序的代价是压迫。

| 要素 | 设计要点 |
|------|---------|
| 组织形态 | 社区自治/DAO/帮派/氏族/公司城 |
| 秩序来源 | 武力/声誉/智能合约/宗教/血缘 |
| 经济形态 | 物物交换/加密货币/资源本位 |
| 冲突来源 | 领地争夺/资源竞争/理念冲突 |

**设计检查**：
- 没有政府，**秩序靠什么维持**？
- 有没有人试图**建立新的统一**？（这本身就是冲突来源）
- 弱者在无政府社会中**如何生存**？

## 二、外星种族设计模板

外星人不是"长着触手的人类"——要从生物学推导文化。

### 设计公式：生理决定文化

```
生理特征 → 生存方式 → 社会结构 → 文化形态 → 与人类的关系
```

### 模板A：集群意识型（虫族/蜂群）

| 层级 | 设计要点 |
|------|---------|
| 生理 | 没有个体意识，只有集群意识 |
| 社会 | 女王/核心节点控制一切，个体是"零件" |
| 文化 | 没有"我"只有"我们"，没有"死亡"只有"损耗" |
| 与人类 | 无法理解人类的个体意识，人类无法理解"无我" |

**对话设计**：集群意识不会说"我"，只会说"我们"。它不理解"孤独"——因为它从未"一个人"过。

### 模板B：能量生命型（纯能量体）

| 层级 | 设计要点 |
|------|---------|
| 生理 | 没有物质身体，以能量形态存在 |
| 社会 | 通过能量频率交流，"共鸣"=社交 |
| 文化 | 不理解"物质"的概念，"拥有一具身体"是恐怖的 |
| 与人类 | 人类太"慢"了——能量生命的时间尺度不同 |

**对话设计**：能量生命不会"说话"，它会"共鸣"。翻译器能转换意思，但丢失了"频率"——就像把音乐翻译成文字。

### 模板C：长寿/永生型（千年文明）

| 层级 | 设计要点 |
|------|---------|
| 生理 | 寿命千年以上，或者意识可以转移 |
| 社会 | 决策极其缓慢（因为他们有时间），变化极其缓慢 |
| 文化 | 重视"记忆"和"传承"，不理解人类的"急躁" |
| 与人类 | 人类像"蜉蝣"——朝生暮死，不值得认真对待 |

**对话设计**：长寿种族说话慢、停顿长。它在思考的时候，人类会觉得它"卡住了"。它说"让我想想"可能意味着三天。

### 模板D：硅基/机械型（非碳基生命）

| 层级 | 设计要点 |
|------|---------|
| 生理 | 硅基/金属/晶体结构，不需要有机物 |
| 社会 | 逻辑驱动，效率至上，没有"情感"（或者情感的定义不同） |
| 文化 | 不理解"浪费"——碳基生命的食物/睡眠/繁殖都是浪费 |
| 与人类 | 人类的"非理性"是它们的研究课题 |

**对话设计**：硅基生命不会说"我觉得"，只会说"计算表明"。它不理解"直觉"——所有决策都必须有逻辑链。

### 模板E：共生/寄生型（与其他物种绑定）

| 层级 | 设计要点 |
|------|---------|
| 生理 | 两个或多个物种形成共生/寄生关系 |
| 社会 | 社会结构基于物种间的共生关系 |
| 文化 | "独立"是不可想象的——脱离共生体=死亡 |
| 与人类 | 人类的"独立个体"模式是它们无法理解的 |

**对话设计**：共生种族的对话里会有"我们"和"它们"两个主语——"我们决定……但它们不同意"。

### 外星种族自测表

- [ ] 外星种族的**生理特征**决定了什么文化特征？
- [ ] 它们的**时间尺度**和人类一样吗？
- [ ] 它们的**交流方式**和人类兼容吗？
- [ ] 它们有没有**无法理解人类**的地方？（不是弱点，是特征）
- [ ] 它们和人类的**核心分歧**是什么？（不只是"长得不一样"）

## 三、星际政治设计模板

星际政治的本质是：**距离+差异=冲突**。

### 模板A：帝国型（中央集权）

```
核心星球（首都）
  ├─ 内圈星球（忠诚/富裕/发达）
  ├─ 中圈星球（半自治/贸易枢纽）
  └─ 外圈星球（边疆/自治/不稳定）
```

**冲突来源**：
- 核心星球的资源榨取 vs 外圈星球的独立诉求
- 帝国的统一标准 vs 星球的文化差异
- 皇帝/议会的继承权争夺

### 模板B：联邦型（协商联合）

```
联邦议会
  ├─ 成员星球保留自治权
  ├─ 联邦只管外交/军事/贸易
  └─ 成员星球可以退出（但代价很高）
```

**冲突来源**：
- 大星球vs小星球的投票权争议
- 联邦权力扩张 vs 成员星球自治
- 退出联邦的法律/经济/军事后果

### 模板C：联盟型（松散联合）

```
联盟会议（没有强制力）
  ├─ 成员星球完全自治
  ├─ 联盟只是"谈事情的地方"
  └─ 成员星球之间可以开战
```

**冲突来源**：
- 联盟决议没有执行力
- 成员星球之间的战争
- 试图建立更紧密联合的势力

### 模板D：贸易路线型（经济驱动）

```
贸易路线网
  ├─ 路线节点星球（贸易枢纽/富裕）
  ├─ 路线沿线星球（过路费/服务经济）
  └─ 路线外星球（边缘化/贫困）
```

**冲突来源**：
- 贸易路线的控制权争夺
- 海盗/走私/走私
- 新路线发现后旧枢纽的衰落

### 星际政治设计检查

- [ ] 政治体的**统一靠什么维持**？（军事/经济/文化/宗教？）
- [ ] 有哪些**分裂势力**？他们的诉求是什么？
- [ ] 星球间的**通讯延迟**如何影响治理？
- [ ] 有没有**贸易路线**制造经济依赖？
- [ ] 有没有**边疆地带**制造灰色空间？

## 四、时间线设计模板

时间线决定了故事的"温度"——近未来焦虑，远未来敬畏。

### 模板A：近未来（现在→50年后）

**特征**：技术变化不大，社会变化明显。

| 要素 | 设计要点 |
|------|---------|
| 技术 | 现有技术的延伸：自动驾驶/基因治疗/初级AI |
| 社会 | 当前问题的放大：贫富分化/气候难民/数据隐私 |
| 日常 | 角色的生活和现在很像，但某个细节"不对" |
| 冲突 | 技术进步的受益者vs受害者 |

**写法要点**：用"一步之遥"制造不安——角色的生活和我们很像，但有一个关键不同。

### 模板B：中未来（50→200年后）

**特征**：技术改变生活，社会形态改变。

| 要素 | 设计要点 |
|------|---------|
| 技术 | 星际殖民/基因编辑/AI伴侣/意识上传 |
| 社会 | 新的阶级形态：基因改造者vs原生人类/数字公民vs物理公民 |
| 日常 | 角色的生活和我们完全不同，但"人"还是人 |
| 冲突 | 新技术带来的新伦理问题 |

**写法要点**：用"日常化技术"写未来——角色觉得这些技术理所当然，读者觉得震撼。

### 模板C：远未来（200→1000年后）

**特征**：文明形态改变，"人"的定义模糊。

| 要素 | 设计要点 |
|------|---------|
| 技术 | 跨维度/时间操控/物质重组/戴森球 |
| 社会 | 后人类社会：AI/改造人/纯种人类/外星种族共存 |
| 日常 | 角色的"日常"对读者来说是完全陌生的 |
| 冲突 | 文明级别的冲突：存亡/进化/宇宙命运 |

**写法要点**：用"蚂蚁视角"写宏大——角色在宇宙面前有多小，读者就有多震撼。

### 模板D：后人类时代（1000年后→文明终结）

**特征**：人类不再是宇宙的主角。

| 要素 | 设计要点 |
|------|---------|
| 技术 | 超越物理定律/改写现实/时间线操控 |
| 社会 | "人类"可能已经不存在了——进化/灭绝/融合 |
| 日常 | 叙事者可能不是人类 |
| 冲突 | 存在本身的终极问题 |

**写法要点**：用"遗迹"写衰败——曾经辉煌的文明只剩下废墟和传说。

### 时间线自测表

- [ ] 时间线设定和**技术水平**匹配吗？
- [ ] 社会制度和技术水平**同步发展**了吗？
- [ ] 角色的"日常"和读者的"日常"有多大**距离**？
- [ ] 有没有**历史事件**锚定时间线？（像我们说"二战后"一样）
- [ ] 时间线上的**关键转折点**是什么？

## 五、世界观构建自测表

写完世界观后对照检查：

- [ ] 世界观能用**一句话**说清吗？（"这是一个XXX的世界"）
- [ ] 社会制度和技术水平**匹配**吗？
- [ ] 外星种族/未来人类有**独特的生理特征**驱动文化吗？
- [ ] 星际政治有**明确的冲突来源**吗？
- [ ] 时间线有**锚定事件**吗？
- [ ] 世界观的**核心矛盾**是什么？（一句话）
- [ ] 普通人在这个世界里**怎么生活**？（不能只写主角）
- [ ] 世界观有没有**裂缝**？（完美世界没有故事）
