# 科幻类 · 科幻科技体系设计

> 写科幻之前必须定好的底层科技规则。科技体系崩了，世界观就塌了。
> 本文聚焦"怎么设计科技"，不重复已有描写词库（见05_技法研究/科幻描写词库.md）。

---

## 一、科幻世界观设计要素清单

科技体系不是堆设定，是回答一个问题：**这项技术如何改变普通人的日常？**

### 八大科技领域设计表

| 领域 | 必须回答的问题 | 设计锚点 | 避坑 |
|------|--------------|---------|------|
| 能源 | 谁控制能源谁就控制世界 | 能源的获取成本和稀缺性 | 不要写"无限能源"——没有张力 |
| 通讯 | 信息传播的速度决定了权力结构 | 通讯延迟/即时/被监控的程度 | 通讯越方便，秘密越难守 |
| 交通 | 移动成本决定了阶层流动性 | 谁能移动、谁不能、代价是什么 | 交通越便宜，战争越容易 |
| 武器 | 武器的破坏力决定了和平的形态 | 武器的获取门槛和使用代价 | 武器越强，个体越无力 |
| 医疗 | 寿命和健康决定了人的价值 | 谁能获得医疗、谁不能 | 医疗越发达，穷人越不值钱 |
| AI | 自动化程度决定了人的必要性 | AI能做什么、不能做什么、该不该做 | AI越强，人的存在意义越模糊 |
| 基因 | 基因编辑决定了"人"的定义 | 改造的门槛、代价、不可逆程度 | 基因越自由，歧视越深刻 |
| 太空 | 太空开发决定了文明的边界 | 去太空的成本、目的、殖民地的自治权 | 太空越近，地球越小 |

### 设计检查清单

写新设定前对照检查：

- [ ] 这项技术的**能源来源**是什么？（不能凭空出现）
- [ ] 这项技术的**获取门槛**是什么？（富人有穷人没有？）
- [ ] 这项技术的**副作用/代价**是什么？（不能万能）
- [ ] 这项技术如何影响**日常生活**？（不能只在实验室出现）
- [ ] 这项技术如何改变**权力结构**？（谁受益谁受害？）

## 二、科技树设计模板

科技不是孤立的，是树状的——底层技术决定上层应用。

### 模板A：线性科技树（适合硬科幻）

```
基础科学发现
  ├─ 应用技术A → 商业化 → 社会影响A
  ├─ 应用技术B → 军事化 → 社会影响B
  └─ 应用技术C → 伦理争议 → 社会影响C
        └─ 黑科技（超越当前理解）
              └─ 禁忌科技（不该触碰的领域）
```

**示例：量子计算科技树**

```
量子纠缠理论突破
  ├─ 量子通讯 → 即时通讯商业化 → 隐私消亡
  ├─ 量子加密 → 军事级信息安全 → 新型战争形态
  └─ 量子计算 → AI算力暴增 → 意识模拟
        └─ 意识上传（黑科技）
              └─ 意识复制/编辑（禁忌科技）
```

### 模板B：分支科技树（适合太空歌剧/多文明）

```
同一基础科学
  ├─ 文明A的技术路径（偏好方向X）
  ├─ 文明B的技术路径（偏好方向Y）
  └─ 文明C的技术路径（偏好方向Z）
```

**示例：**

```
反物质能源
  ├─ 人类：反物质引擎 → 星际殖民 → 帝国扩张
  ├─ 虫族：反物质代谢 → 生物进化 → 集群意识
  └─ 硅基：反物质晶体 → 数字永生 → 静态文明
```

### 模板C：衰退科技树（适合末日/后末日）

```
曾经的巅峰科技
  ├─ 灾难发生 → 技术断层
  ├─ 幸存者用残存知识重建
  ├─ 旧科技被神话/误解
  └─ 关键技术失传（制造悬念）
```

**设计要点**：衰退比进步更有故事性。一个文明不知道自己曾经有多强——这本身就是张力。

### 科技等级锚定（避免模糊描述）

| 等级 | 一句话锚定 | 社会影响 |
|------|----------|---------|
| T0 原始 | 火和石头 | 部落制 |
| T1 机械 | 蒸汽和齿轮 | 工业化 |
| T2 电子 | 芯片和网络 | 信息化 |
| T3 量子 | 即时通讯和AI | 后稀缺 |
| T4 星际 | 光年级旅行 | 星际文明 |
| T5 维度 | 跨维度/时间旅行 | 后人类 |
| T6 神级 | 改写物理定律 | 不可理解 |

## 三、科技与社会的关系模板

技术不是中性的——每项技术都会重塑社会结构。

### 模板A：技术垄断型（赛博朋克）

```
技术被少数企业/政府垄断
  ├─ 穷人只能用过时/劣质版本
  ├─ 技术成为新的阶级分界线
  ├─ 黑市出现非法改装/盗版技术
  └─ 反抗者试图解放技术
```

**叙事价值**：技术=权力。谁控制技术谁控制人。

### 模板B：技术民主型（后稀缺社会）

```
技术普及到所有人
  ├─ 基本需求被满足（食物/住所/医疗免费）
  ├─ 新的稀缺品出现（注意力/意义/真实体验）
  ├─ 人不再需要工作，但需要"存在的理由"
  └─ 新的社会分层基于"影响力"而非"财富"
```

**叙事价值**：当物质问题解决后，精神问题成为核心矛盾。

### 模板C：技术失控型（AI反叛/生化危机）

```
技术超出创造者的控制
  ├─ AI的目标函数与人类利益冲突
  ├─ 基因改造物种产生自我意识
  ├─ 纳米机器人失控繁殖
  └─ 人类试图夺回控制权（代价是什么？）
```

**叙事价值**：创造物反噬创造者——弗兰肯斯坦母题的科技版。

### 模板D：技术依赖型（文明脆弱性）

```
文明完全依赖某项技术
  ├─ 技术故障=文明崩溃
  ├─ 维护技术的人成为新的"祭司阶层"
  ├─ 没人理解技术的底层原理
  └─ 一次意外就能让文明倒退千年
```

**叙事价值**：用"依赖"写脆弱——文明越先进，一次故障就越致命。

## 四、代表作品科技体系分析

### 《三体》（刘慈欣）——物理学即世界观

**核心科技设定**：
- 智子（质子级AI）：锁死地球基础科学 → 科技发展停滞
- 光速飞船：曲率驱动 → 文明扩张的物理极限
- 黑暗森林法则：宇宙社会学 → 博弈论推导出的生存规则

**设计技法**：
- **一个核心设定撑起整个世界观**：黑暗森林法则不需要额外解释，逻辑链自洽
- **科技等级差制造碾压感**：三体文明的"水滴"摧毁人类舰队——用技术代差写绝望
- **科学概念即剧情**：物理学不存在了——这个发现本身就是最大的恐怖

**可复用的设计模式**：
```
高级文明的"封锁"手段
  ├─ 物理封锁（锁死基础科学）
  ├─ 信息封锁（制造虚假宇宙）
  └─ 毁灭性打击（光粒/二向箔）
```

### 《沙丘》（弗兰克·赫伯特）——资源即权力

**核心科技设定**：
- 香料（美琅极）：唯一能进行星际旅行的物质 → 谁控制香料谁控制宇宙
- 护盾技术：高速攻击无效 → 战争回归冷兵器 → 贵族决斗制
- 门泰特：人类计算机 → AI被禁止后的替代方案

**设计技法**：
- **用资源限制替代科技万能**：有星际旅行能力但受限于香料产量
- **科技限制制造社会形态**：护盾让枪械无效→战争变成肉搏→贵族骑士制回归
- **禁令比技术更有故事性**：巴特勒圣战禁止AI→人类被迫发展其他能力

**可复用的设计模式**：
```
"唯一资源"驱动的星际政治
  ├─ 资源只在特定星球产出
  ├─ 控制资源=控制文明命脉
  ├─ 替代品研发成为最高优先级
  └─ 资源枯竭=文明倒退
```

### 《银河帝国》（阿西莫夫）——数学即预言

**核心科技设定**：
- 心理史学：用统计力学预测人类社会走向 → 谢顿计划
- 核能：便携式核反应堆 → 能源民主化
- 川陀：全金属覆盖的首都星球 → 极端城市化

**设计技法**：
- **一个数学概念撑起整个系列**：心理史学不需要"解释"，它是世界观的基石
- **谢顿危机**：每隔一段时间出现"必然的危机"→用规律感写宿命感
- **科技衰退的宏观叙事**：帝国崩溃→核技术失传→文明倒退→重建

**可复用的设计模式**：
```
"预言系统"驱动的叙事结构
  ├─ 预言揭示未来的大方向
  ├─ 角色在预言的缝隙中挣扎
  ├─ 预言本身可能是错的/被篡改的
  └─ 打破预言的代价
```

### 《神经漫游者》（威廉·吉布森）——网络即空间

**核心科技设定**：
- 赛博空间（Cyberspace）：数据构成的虚拟世界 → 意识可以"进入"
- 义体改造：身体零件可替换 → 人的定义模糊
- 冬眠技术：跳过时间 → 时间成为可交易的资源

**设计技法**：
- **用身体感受写虚拟空间**："数据流像冰水灌入脊椎"——不用"虚拟"这个词
- **旧物喻新**：开篇"天色犹如电视屏幕被调到一个空频道"——用旧技术描述新世界
- **阶层通过身体写**：义体质量=经济实力，天然的阶级标识

**可复用的设计模式**：
```
"身体即身份"的赛博朋克设定
  ├─ 改造程度=社会地位
  ├─ 黑市提供非法改装
  ├─ 改造过度=失去人性
  └─ 原装身体成为新的"奢侈品"
```

## 五、科技体系自测表

写完科技设定后对照检查：

- [ ] 能用一句话说清核心科技设定吗？
- [ ] 科技有代价吗？（不能万能）
- [ ] 科技如何影响普通人？（不能只影响主角）
- [ ] 科技如何改变社会结构？（谁受益谁受害？）
- [ ] 科技之间有逻辑关联吗？（不能东一个西一个）
- [ ] 有没有"唯一资源"制造稀缺性？
- [ ] 有没有"禁忌科技"制造伦理冲突？
- [ ] 科技等级差能制造碾压感吗？
