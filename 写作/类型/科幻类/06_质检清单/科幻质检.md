# 科幻类 · 质检清单

> 写完后逐项检查。不达标就改。

---

## 一、核心检查（必过）

- [ ] 科幻设定能否3秒说清？（不能→太复杂，需要简化）
- [ ] 技术是否有代价？（没有→免费的力量=无聊）
- [ ] 技术是背景还是主角？（如果技术比角色更抢戏→调整重心）
- [ ] 是否有感官锚点？（至少1个感官细节锚定异世界）
- [ ] 科学逻辑是否自洽？（硬科幻必查，软科幻可以宽松）

## 二、描写检查

- [ ] 科幻描写是否用了日常化类比？（"像微信一样的全息通话"）
- [ ] 是否有"说教段落"？（大段解释技术原理→删掉或融入对话）
- [ ] 未来社会描写是否具体？（不能只说"未来很先进"，要写具体细节）
- [ ] 科幻对话是否有技术术语？（自然融入，不能像在读论文）

## 三、红线检查

- [ ] 是否违反基本科学逻辑？（科幻可以超越现实但不能违反逻辑）
- [ ] 是否变成科普文？（技术解释篇幅超过30%→太多）
- [ ] 是否变成武器说明书？（战斗描写只有技术参数→缺少人的反应）
- [ ] 是否变成政治宣言？（社会批判太直白→需要融入剧情）

## 四、质量检查

- [ ] 是否有"如果……会怎样"的核心问题？（没有→不是科幻）
- [ ] 角色是否因技术而改变？（技术对人的影响才是科幻的核心）
- [ ] 是否有认知震撼？（读者看完是否觉得"原来世界可以是这样"？）
- [ ] 结局是否有余韵？（科幻的结局应该让读者继续思考）
