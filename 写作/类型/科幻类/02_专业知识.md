## 二、科技描写模板

### 飞船/太空
- **起飞**：重力把人压进座椅，内脏像被往下拽。视野从地面建筑缩小成色块，云层穿过机翼变成白色泡沫，然后是一片纯黑。
- **失重**：胃里翻涌，像坐过山车的最高点无限延长。松开手，笔漂浮在空中旋转。水从杯口涌出来变成一个颤动的球。
- **太空行走**：面罩里只有自己的呼吸声。脚下的地球像一块蓝色的玻璃球。太阳的方向不是亮的，是刺痛的。

### AI/虚拟现实
- **AI对话**：它回答得太快了。不是人类那种"想了想再说"的快，是"答案已经在那里等你问"的快。
- **VR进入**：视网膜上的画面叠加了一层——不，是替换了一层。手指穿过虚拟桌面，没有触感，但视觉说"这是实体"。
- **意识上传**：像从一个房间走到另一个房间。但回头的时候，原来的房间已经不在了。

### 基因改造/身体改造
- **改造前**：体检报告上的数字在跳。"你的基因序列有37%会被替换。"医生顿了顿，"术后你还是你吗？这是哲学问题，不是医学问题。"
- **改造后**：伤口在48小时内愈合了。他看着自己的手，皮肤下面隐约能看到金属的颜色。能感觉到——不，是"读取到"——周围3米内所有人的体温。

## 四、科幻对话风格

### 技术术语融入对话
- **好**："别走那条路，那边的辐射值超标了。"（技术信息+日常对话）
- **坏**："根据盖革计数器的读数，该区域的电离辐射剂量率已超过安全阈值的3.7倍。"（像论文）

### 科幻感对话模板
- **信息差对话**：A知道这个世界的真相，B不知道。A说的每句话都有双重含义。
- **AI对话**：AI的回答精确但缺乏"温度"。人类在跟AI说话时会不自觉地简化自己的表达。
- **跨物种对话**：翻译器在工作，但总有些词翻译不出来。沉默比翻译更频繁。

## 五、科幻世界观构建

社会制度决定了角色的生存环境——不是选一个"好看"的制度，是选一个"有冲突"的制度。

| 类型 | 核心矛盾 | 设计检查要点 |
|------|---------|-------------|
| 乌托邦型 | 完美是有代价的，代价由谁承担？ | 完美靠什么维持？谁在买单？裂缝从哪来？ |
| 反乌托邦型 | 控制越严密，反抗越激烈 | 控制的技术基础？权力层有无分裂？有无不被控制的人？ |
| 赛博朋克型 | 企业比政府强大，技术比法律有效 | 有哪些企业巨头？地下经济靠什么运转？反抗者的技术手段？ |
| 星际帝国型 | 距离产生分裂，统一需要代价 | 靠什么维持统一？分裂势力的诉求？通讯延迟如何影响治理？ |
| 无政府型 | 自由的代价是混乱，秩序的代价是压迫 | 秩序靠什么维持？有无人试图建立统一？弱者如何生存？ |

---

### 外星种族设计

设计公式：生理特征 → 生存方式 → 社会结构 → 文化形态 → 与人类的关系

| 类型 | 对话设计要点 |
|------|-------------|
| 集群意识型（虫族） | 不会说"我"，只会说"我们"。不理解"孤独"——因为它从未"一个人"过 |
| 能量生命型 | 不会"说话"，会"共鸣"。翻译器能转换意思，但丢失了"频率"——就像把音乐翻译成文字 |
| 长寿/永生型 | 说话慢、停顿长。说"让我想想"可能意味着三天。人类会觉得它"卡住了" |
| 硅基/机械型 | 不会说"我觉得"，只会说"计算表明"。不理解"直觉"——所有决策都必须有逻辑链 |
| 共生/寄生型 | 对话里有"我们"和"它们"两个主语——"我们决定……但它们不同意" |

### 世界观构建自测表

- [ ] 世界观能用**一句话**说清吗？
- [ ] 社会制度和技术水平**匹配**吗？
- [ ] 外星种族有**生理特征驱动的独特文化**吗？**对话方式**和人类兼容吗？
- [ ] 星际政治有**明确的冲突来源**？统一靠什么维持？有无**边疆灰色地带**？
- [ ] 时间线有**锚定事件**？技术水平和社会制度**同步发展**？
- [ ] 世界观的**核心矛盾**是什么？（一句话）
- [ ] 普通人在这个世界里**怎么生活**？
- [ ] 世界观有没有**裂缝**？（完美世界没有故事）

### 星际政治设计

星际政治的本质是：**距离+差异=冲突**。

| 类型 | 冲突来源 |
|------|---------|
| 帝国型（中央集权） | 核心星球资源榨取 vs 外圈独立诉求；统一标准 vs 文化差异；继承权争夺 |
| 联邦型（协商联合） | 大小星球投票权争议；联邦权力扩张 vs 成员自治；退出代价 |
| 联盟型（松散联合） | 决议无执行力；成员间战争；试图紧密联合的势力 |
| 贸易路线型（经济驱动） | 路线控制权争夺；海盗/走私；新路线导致旧枢纽衰落 |

### 时间线设计

时间线决定了故事的"温度"——近未来焦虑，远未来敬畏。

| 时间段 | 写法要点 |
|--------|---------|
| 近未来（→50年） | "一步之遥"制造不安——角色生活和我们很像，但有一个关键不同 |
| 中未来（50→200年） | "日常化技术"写未来——角色觉得理所当然，读者觉得震撼 |
| 远未来（200→1000年） | "蚂蚁视角"写宏大——角色在宇宙面前有多小，读者就有多震撼 |
| 后人类（1000年→） | "遗迹"写衰败——曾经辉煌的文明只剩下废墟和传说 |

## 十、科幻科技体系设计

科技体系不是堆设定，是回答一个问题：**这项技术如何改变普通人的日常？**

### 八大科技领域避坑指南

| 领域 | 避坑 |
|------|------|
| 能源 | 不要写"无限能源"——没有张力 |
| 通讯 | 通讯越方便，秘密越难守 |
| 交通 | 交通越便宜，战争越容易 |
| 武器 | 武器越强，个体越无力 |
| 医疗 | 医疗越发达，穷人越不值钱 |
| AI | AI越强，人的存在意义越模糊 |
| 基因 | 基因越自由，歧视越深刻 |
| 太空 | 太空越近，地球越小 |

### 设计检查清单

写新设定前对照检查：

- [ ] 这项技术的**能源来源**是什么？（不能凭空出现）
- [ ] 这项技术的**获取门槛**是什么？（富人有穷人没有？）
- [ ] 这项技术的**副作用/代价**是什么？（不能万能）
- [ ] 这项技术如何影响**日常生活**？（不能只在实验室出现）
- [ ] 这项技术如何改变**权力结构**？（谁受益谁受害？）
- [ ] 科技之间有**逻辑关联**吗？（不能东一个西一个）
- [ ] 有没有"**唯一资源**"制造稀缺性？有没有"**禁忌科技**"制造伦理冲突？
- [ ] 科技等级差能制造**碾压感**吗？

---

### 科技树设计模板

科技不是孤立的，是树状的——底层技术决定上层应用。

- **线性科技树**（硬科幻）：基础科学发现 → 应用技术 → 商业化/军事化/伦理争议 → 黑科技 → 禁忌科技
- **分支科技树**（太空歌剧）：同一基础科学 → 不同文明的技术路径（人类走引擎，虫族走代谢，硅基走晶体）
- **衰退科技树**（末日/后末日）：巅峰科技 → 灾难 → 技术断层 → 旧科技被神话/误解

**设计要点**：衰退比进步更有故事性。一个文明不知道自己曾经有多强——这本身就是张力。

### 代表作品科技体系分析

#### 《三体》——物理学即世界观

**设计技法**：
- **一个核心设定撑起整个世界观**：黑暗森林法则不需要额外解释，逻辑链自洽
- **科技等级差制造碾压感**：三体文明的"水滴"摧毁人类舰队——用技术代差写绝望
- **科学概念即剧情**：物理学不存在了——这个发现本身就是最大的恐怖

**可复用模式**：高级文明的"封锁"手段——物理封锁（锁死基础科学）、信息封锁（制造虚假宇宙）、毁灭性打击（光粒/二向箔）

#### 《沙丘》——资源即权力

**设计技法**：
- **用资源限制替代科技万能**：有星际旅行能力但受限于香料产量
- **科技限制制造社会形态**：护盾让枪械无效→战争变成肉搏→贵族骑士制回归
- **禁令比技术更有故事性**：巴特勒圣战禁止AI→人类被迫发展其他能力

**可复用模式**："唯一资源"驱动的星际政治——资源只在特定星球产出，控制资源=控制文明命脉，替代品研发是最高优先级，资源枯竭=文明倒退

#### 《银河帝国》——数学即预言

**设计技法**：
- **一个数学概念撑起整个系列**：心理史学不需要"解释"，它是世界观的基石
- **谢顿危机**：每隔一段时间出现"必然的危机"→用规律感写宿命感
- **科技衰退的宏观叙事**：帝国崩溃→核技术失传→文明倒退→重建

**可复用模式**："预言系统"驱动的叙事结构——预言揭示大方向，角色在缝隙中挣扎，预言可能被篡改，打破预言有代价

#### 《神经漫游者》——网络即空间

**设计技法**：
- **用身体感受写虚拟空间**："数据流像冰水灌入脊椎"——不用"虚拟"这个词
- **旧物喻新**：开篇"天色犹如电视屏幕被调到一个空频道"——用旧技术描述新世界
- **阶层通过身体写**：义体质量=经济实力，天然的阶级标识

**可复用模式**："身体即身份"的赛博朋克设定——改造程度=社会地位，黑市提供非法改装，改造过度=失去人性，原装身体成为新的"奢侈品"
