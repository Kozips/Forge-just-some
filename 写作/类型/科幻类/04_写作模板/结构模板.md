# 科幻类 · 结构模板

> 科幻写作的章节结构模板。写作前用模板规划节奏。

---

## 一、单章核心张力

每章必须回答一个问题：**这一章里，主角最怕失去什么？**

| 张力类型 | 说明 | 典型场景 |
|----------|------|---------|
| 生存张力 | 不解决就死 | 飞船故障/辐射泄漏/氧气耗尽 |
| 真相张力 | 不查就永远被蒙在鼓里 | 世界真相/AI阴谋/政府隐瞒 |
| 选择张力 | 不选就有不可逆后果 | 牺牲一人救多人/保留人性还是进化 |
| 时间张力 | 不行动就来不及 | 倒计时/星球撞击/病毒扩散 |
| 身份张力 | 不面对就失去自我 | 意识上传后还是我吗/改造后还是人吗 |

## 二、单章结构模板

### 模板A：发现型（硬科幻）
```
日常世界(15%) → 发现异常(15%) → 科学调查(25%) → 揭示真相(20%) → 认知震撼(15%) → 新常态(10%)
```

### 模板B：生存型（末日/太空）
```
危机爆发(15%) → 求生行动(25%) → 资源耗尽(20%) → 绝境抉择(20%) → 代价与希望(15%) → 新秩序(5%)
```

### 模板C：觉醒型（AI/反乌托邦）
```
完美世界(10%) → 发现裂缝(15%) → 深入调查(25%) → 真相揭露(20%) → 反抗/抉择(20%) → 新世界(10%)
```

### 模板D：悖论型（时间旅行）
```
正常生活(10%) → 发现异常(15%) → 尝试修复(25%) → 悖论出现(20%) → 代价(20%) → 接受/改变(10%)
```

## 三、多章弧线模板

### 三幕式（适合单季/短篇）
- **第一幕（25%）**：建立世界+核心冲突出现
- **第二幕（50%）**：调查/对抗/升级，真相逐步揭露
- **第三幕（25%）**：最终对决+代价+新世界

### 英雄之旅改编（适合长篇）
- **平凡世界** → **异常召唤** → **拒绝召唤** → **导师出现** → **跨越门槛** → **试炼/盟友/敌人** → **接近最深洞穴** → **最大磨难** → **奖赏** → **回归之路** → **复活** → **带回万灵药**

## 四、开篇钩子模板

| 类型 | 模板 | 适用题材 |
|------|------|---------|
| 异常开场 | 第一句话就是不对劲的事 | 所有科幻 |
| 数据开场 | 用一组数据/编号开头 | 硬科幻/反乌托邦 |
| 日常切入异常 | 前100字正常，第101字开始不对 | 软科幻/AI科幻 |
| 倒计时开场 | 从结局/灾难倒叙 | 末日/太空 |
| 身份反差 | 角色的表面身份vs真实身份 | 赛博朋克/AI |

---

## 五、结尾钩子类型

| 类型 | 特征 | 适用 |
|------|------|------|
| 认知震撼型 | 真相让读者重新理解整个故事 | 硬科幻 |
| 道德困境型 | 没有正确答案，只有选择 | 软科幻 |
| 开放型 | 留下未解的问题 | 所有科幻 |
| 余韵型 | 安静的场景，让读者自己品味 | 文学科幻 |
| 倒计时型 | 新的威胁出现 | 太空/末日 |
