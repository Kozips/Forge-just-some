## 一、末日货币与经济体系

### 硬通货排序（由硬到软）

| 层级 | 物资 | 硬度原因 |
|------|------|---------|
| S | 干净水/净水设备 | 生存必需、不可替代、极易消耗 |
| A | 弹药/武器 | 自保必需、不可再生 |
| A | 抗生素/医疗品 | 救命刚需、保质期有限 |
| B | 燃料（汽油/柴油） | 机动性=生存率、会变质 |
| B | 种子/活体动物 | 长期生存投资、延迟收益 |
| C | 罐头/压缩食品 | 必需但有保质期限制 |
| C | 工具（刀/斧/绳） | 耐用、通用 |
| D | 烟酒/咖啡 | 非必需但极受欢迎（社交货币） |
| D | 信息/情报 | 价值波动大但关键时刻无价 |

### 以物易物定价参考

- 一升净水 = 五罐午餐肉 = 一盒阿莫西林
- 一发手枪弹 = 一天口粮
- 一把可用的刀 = 三天食物
- 一张安全区地图 = 一周补给
- 一个活的医生 = 无法定价（整个聚落的核心资产）

### 物资衰减曲线

- 食品：罐头（2-5年）→ 压缩饼干（1-2年）→ 干粮（6月-1年）→ 需自产
- 药品：抗生素（1-3年，效力递减）→ 止痛药（2-4年）→ 绷带（可反复使用）
- 弹药：基本不衰减但不可再生，用一发少一发
- 燃料：汽油（3-6月变质）→ 柴油（更长）→ 最终需要生物柴油/电力

**关键设计**：物资的"半衰期"决定了末日的阶段性——
- 前期（0-6月）：搜刮时代，超市/仓库是战场
- 中期（6月-2年）：抢夺时代，势力冲突为王
- 后期（2年+）：自产时代，农业/手工业=权力

## 二、生存医学速查

### 末日中最可能遇到的伤病（按频率）

| 伤病 | 发生原因 | 末日处理 | 不处理后果 |
|------|---------|---------|-----------|
| 开放性伤口感染 | 搜刮/战斗 | 清洗+消毒+包扎 | 败血症死亡 |
| 骨折 | 坠落/坍塌 | 固定+休息（无法手术） | 永久残疾 |
| 腹泻/脱水 | 脏水/过期食物 | 口服补液盐（盐+糖+水） | 脱水致死 |
| 咬伤感染 | 丧尸/人 | 截肢或等死 | 转化或死亡 |
| 肺炎 | 受寒/粉尘 | 抗生素（如有）+休息 | 死亡 |
| 坏疽 | 伤口感染恶化 | 截肢（手术风险极高） | 死亡 |
| 心理创伤 | 持续恐惧/丧失 | 无药可用，靠同伴 | 崩溃/自杀/自毁行为 |

### 末日急救黄金知识

- **止血**：直接压迫（布料/衣物）→ 止血带（每30分钟松一次）→ 高位结扎
- **感染预防**：煮沸水清洗伤口 > 酒精消毒 > 碘伏（如有）
- **退烧**：物理降温（湿毛巾+扇风）→ 布洛芬（如有）→ 危险信号：持续39度以上
- **骨折固定**：两根直树枝+布条绑缚，上下各一个关节
- **脱水判断**：捏手背皮肤，松开后回弹慢=严重脱水

### 药物替代知识

- 止痛：柳树皮煮水（含水杨酸，阿司匹林前体）
- 伤口消毒：蜂蜜（天然抗菌）/ 烧开放凉的盐水
- 退烧：冷水擦浴大血管处（脖子/腋下/腹股沟）
- 腹泻：木炭粉（吸附毒素）/ 大蒜（天然抗菌）

## 三、聚落治理速查

### 聚落规模与问题演变

| 人数 | 核心挑战 | 治理方式 |
|------|---------|---------|
| 5-15人 | 信任、分工 | 个人威信+简单规则 |
| 15-50人 | 派系、资源分配 | 领导核心+成文规则 |
| 50-200人 | 阶层分化、犯罪 | 制度化管理+司法 |
| 200+人 | 政治、经济体系 | 正式政府+法律+货币 |

### 聚落必需岗位（最低配置）

1. **领导者**（决策） — 不一定最强但能服众
2. **防御者**（安全） — 值守/巡逻/武器管理
3. **医者**（医疗） — 末日最稀缺人才
4. **猎手/农夫**（食物） — 持续食物来源
5. **工匠**（维修/建设） — 武器维护/基地加固
6. **拾荒者**（探索） — 外部物资获取

### 聚落内部冲突常见触发点

1. **食物分配不均** — "凭什么他们比我多？"
2. **危险任务轮值** — "总是我去？"
3. **新来者接纳** — "他们抢我们的资源"
4. **伤员/感染者处置** — "杀了还是留？"
5. **领导权争夺** — "你凭什么说了算？"
6. **外部势力接触** — "他们想干什么？"

### 基地防御层级

| 层级 | 设施 | 功能 |
|------|------|------|
| 外围预警 | 陷阱/绊线/铃铛 | 提前发现 |
| 第一道防线 | 围墙/铁丝网/车辆路障 | 减速+分流 |
| 第二道防线 | 射击位/高台/窄通道 | 集中火力 |
| 最后防线 | 内部安全室/地道 | 撤退掩护 |

## 四、末日交通与移动

### 不同载具末日适配度

| 载具 | 优势 | 劣势 | 燃料依赖 | 噪音 |
|------|------|------|---------|------|
| 步行 | 零燃料、安静 | 慢、体力消耗大 | 无 | 低 |
| 自行车 | 快、安静、零燃料 | 载重小、需体力 | 无 | 极低 |
| 摩托车 | 快、灵活 | 噪音大、载重小 | 高 | 高 |
| 皮卡/SUV | 载重大、可越野 | 耗油、堵塞时无用 | 高 | 中 |
| 卡车 | 载重极大 | 耗油高、笨重、需道路 | 极高 | 高 |
| 马/驴 | 可越野、可繁殖 | 需喂养、有脾气 | 草料 | 低 |
| 船 | 水路隔离安全 | 速度慢、受天气限制 | 视类型 | 低 |

### 末日路线选择原则

- 大路快但危险（开阔=暴露+可能有人设伏）
- 小路安全但慢（绕路+体力消耗+可能迷路）
- 水路最安全但限制多（需要船+逆流极难）
- 最佳策略：夜间短距移动 + 白天隐蔽休息

## 五、季节与天气对末日的影响

| 季节 | 优势 | 劣势 | 丧尸变化 |
|------|------|------|---------|
| 春 | 水源充足、可种植 | 泥泞影响移动 | 腐烂加速 |
| 夏 | 日照长、可种植 | 食物易腐败、疫病高发 | 腐臭最重、吸引动物 |
| 秋 | 收获季、温度宜人 | 准备冬季的压力 | 行动可能加速（体温调节失效） |
| 冬 | 疫病减少、食物保存久 | 低温致死、水源冻结 | 冻僵减速（但不死亡）/ 冰下仍活 |

**天气事件素材**：
- 暴风雨：掩盖行动声音（可利用），但视线归零
- 浓雾：视觉恐惧最大化，"看不见的威胁"
- 极寒：暴露=死亡，但丧尸也慢了
- 沙尘暴：呼吸困难+方向丧失
- 酸雨（设定性）：腐蚀+暴露皮肤烧灼
