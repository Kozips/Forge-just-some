# 末日世界观构建指南

> 从宏观框架到微观细节，构建可信的末日世界

---

## 一、世界观起源四类型

### 1. 病毒/感染型（科学恐慌）

**核心设定**：科学实验泄漏、生物武器失控、基因突变

**社会崩溃顺序**（重要！按此顺序写更真实）：
1. 电力中断 → 医院依赖设备停摆
2. 通信中断 → 谣言四起，真相缺失
3. 交通瘫痪 → 无法逃离，物资无法调配
4. 政府失序 → 军队各自为战
5. 货币失效 → 以物易物
6. 道德底线崩塌 → 丛林法则

**转化机制设计**：
- 咬伤转化（最经典）：被咬=感染，转化时间可长可短
- 空气传播（灾难放大器）：无法"躲在家里"，恐惧升级
- 信号触发（戏剧性强）：特定声音/画面激活丧尸
- 全民感染（最绝望）：所有人都是潜在敌人

### 2. 灾变废土型（资源争夺）

**核心设定**：核战、环境灾变、陨石撞击、资源枯竭

**废土特征**：
- 辐射/污染改变地形：河流改道、城市下沉、森林变异
- 资源极度匮乏：水、燃料、医疗品成为硬通货
- 势力割据：不同势力占据不同资源点
- 科技断层：旧技术失传，新技术萌芽

**环境描写要点**：
- 天空：铅灰色、酸雨、核冬天暗无天日
- 大地：龟裂、放射性尘埃、变异植物
- 建筑：钢筋外露、玻璃碎裂、广告牌半埋

### 3. 灵气复苏型（异能升级）

**核心设定**：灵气/能量复苏，人类觉醒异能

**与传统末日区别**：
- 不是"人和丧尸谁更强"，而是"觉醒者和未觉醒者的阶级分化"
- 升级体系成为核心驱动力
- 主角可以通过修炼变强

**能力体系设计**：
- F（普通人）→ E（初觉）→ D（较强）→ C（区域级）→ B（城市级）→ A（国家级）→ S（全球级）
- 异能六系：元素/身体强化/精神/空间/特殊/辅助

### 4. 系统降临型（游戏化）

**核心设定**：末日爆发时获得游戏化系统

**九类金手指**：
| 类型 | 核心机制 | 爽点 |
|------|---------|------|
| 随身空间 | 私人储物+种植养殖 | 物资安全+自给自足 |
| 签到系统 | 每日签到获奖励 | 稳定期待感 |
| 商城积分 | 击杀获积分兑换 | 即时正反馈 |
| 属性面板 | 数值化能力显示 | 成长可视化 |
| 任务系统 | 主线/支线任务 | 目标感+探索 |
| 重生记忆 | 知晓全部时间线 | 先知优势 |
| 吞噬进化 | 吞噬晶核获取能力 | 收集养成 |
| 模拟器 | 模拟未来场景 | 试错成本降低 |
| 召唤傀儡 | 控制丧尸 | 势力扩张 |

## 二、末日社会结构

### 六类势力

| 势力 | 特征 | 核心矛盾 |
|------|------|---------|
| 军事残余 | 秩序vs自由的二元对立 | 铁律vs人道 |
| 幸存者聚落 | 人性温暖的港湾 | 安全感vs资源匮乏 |
| 军阀帮派 | 弱肉强食丛林法则 | 暴力vs生存 |
| 宗教邪教 | 信仰极端化 | 狂热vs理性 |
| 科技组织 | 真相载体/希望 | 技术vs伦理 |
| 商人联盟 | 利益至上的中立派 | 利润vs道德 |

### 聚落选址逻辑

**学校**：围墙+教室分隔+操场开阔地
**超市**：物资集中+空间开阔+玻璃易破
**监狱**：天然防御+分区管理+阴暗压抑
**医院**：医疗价值+人群密集+病患丧尸
**船上**：隔离优势+空间独立+资源有限
**地下**：天然防护+隐蔽+幽闭恐惧

## 三、环境三层描写法

### 宏观层（奠定基调）
- 天空：灰蒙蒙的、灰尘太大遮住太阳
- 气候：酸雨、核冬天暗无光、永恒的风沙
- 大地：龟裂的柏油路、变异的河流

### 中观层（展现规模）
- 建筑：钢筋从混凝土中刺出如骨骼
- 街道：车辆骨架横陈、碎玻璃铺满路面
- 遗迹：半埋的广告牌、倒塌的电线杆

### 微观层（最具感染力）
- 涂鸦："有人活着出去了"——字迹潦草像在跑着写
- 物件：空荡荡的儿童鞋、还亮着灯的售货机
- 痕迹：地面拖痕、墙上手印、没干透的血迹

## 四、末日时间线设计

### 爆发第一天
- 信息混乱：微信群/微博→新闻不报→谣言四起
- 心理轨迹：不以为意→胃缩了一下→给家人打电话
- 核心情绪：认知崩塌，不知道发生了什么

### 爆发第一个月
- 社会秩序快速崩塌
- 幸存者开始聚集
- 势力初步形成
- 资源争夺开始

### 爆发半年到一年
- 旧秩序彻底消失
- 新的规则建立
- 基地建设/扩张成为主线
- 大规模丧尸潮首次出现

### 爆发一年以上
- 文明断层出现
- 技术失传与部分恢复
- 势力版图相对稳定
- 更大的阴谋/真相浮出水面

## 五、创作自检

写完世界观设定后，对照检查：

- [ ] **起源清晰**：读者知道末日是怎么来的吗？
- [ ] **社会崩溃逻辑**：崩塌是渐进的还是一步到位的？
- [ ] **势力分布**：世界不是只有主角一群人？
- [ ] **威胁层次**：丧尸/变异→人类敌人→人性，三层威胁都有？
- [ ] **五感调动**：不只是视觉？听觉（安静）、嗅觉（腐烂）、触觉（粗糙冰冷）？
- [ ] **微观细节**：有涂鸦、物件、痕迹等让人印象深刻的细节吗？
- [ ] **创新点**：28种细分类型中，你的设定有没有融合或创新？
