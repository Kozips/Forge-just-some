## 技法1：境界压制的恐怖感

**来源**：《遮天》《完美世界》
**手法拆解**：
1. 高境界角色出场时，用低境界角色的反应展示差距
2. 不直接描写"他有多强"——而是描写"他在他面前有多渺小"
3. 压制感来自气息/眼神/随意的一个动作

> 那个人只是看了他一眼。他的双腿就不受控制地弯了下去——不是跪拜，是身体在恐惧面前的本能反应。

## 技法2：以弱胜强的智取

**来源**：《斗破苍穹》《凡人修仙传》
**手法拆解**：
1. 主角实力不如对手，但利用环境/法宝/计谋
2. 每个"底牌"都在前文有铺垫
3. 胜利后主角也有代价（法宝损毁/重伤/暴露底牌）

## 技法3：世界观的层层展开

**来源**：《遮天》《一世之尊》
**手法拆解**：
1. 主角以为自己已经了解了世界
2. 一个事件让他发现：世界比他以为的大得多
3. 每次突破都揭开新的世界层面

## 技法4：丹药/法宝的"仪式感"

**来源**：各类修仙小说
**手法拆解**：
1. 获取宝物的过程本身就是一个小故事
2. 宝物的描述要有细节（颜色/温度/气味/反应）
3. 使用宝物时的仪式感增加期待
