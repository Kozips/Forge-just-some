# 剧情递进技法 —— 玄幻升级

> 来源：剧情递进手法_五种类型全覆盖
> 四个维度：冲突升级 · 信息解密 · 情绪压强 · 格局扩展

---

## 两种典型递进套路

**套路A：修炼→瓶颈→机缘→突破→更强的敌人**

每一次升级都伴随着一个"不可能跨越的瓶颈"。突破瓶颈的方式要么是奇遇，要么是代价极大的修炼。突破后发现上面还有更强的人等着。核心节奏是"压迫→蓄力→爆发→新压迫"。

**套路B：小世界→大世界→更大世界**

主角在小地方是最强的，进入大世界后变成蝼蚁。在大世界爬到顶端后发现还有更大的世界。每换一次地图，主角的实力和格局都被重新归零。本质是"不断刷新天花板"。

---

## 具体例子：《斗破苍穹》递进结构

> 乌坦城篇：家族废柴，被退婚。冲突是个人尊严。实力从斗之气到斗师。压迫来自家族内部和纳兰嫣然。
>
> 伽玛学院篇：进入更大舞台。冲突从个人恩怨变成学院势力争斗。实力从斗师到斗灵。压迫来自云岚宗。
>
> 中州篇：整个大陆的势力格局。冲突从区域争斗变成大陆级势力博弈。实力从斗王到斗帝。压迫来自魂殿和远古种族。

**递进拆解：**
- **冲突升级**：家族屈辱 → 宗门争斗 → 大陆势力 → 天地浩劫
- **信息解密**：个人身世 → 家族秘密 → 大陆历史 → 天地本源
- **情绪压强**：屈辱逆反 → 热血燃烧 → 孤独强者 → 苍茫大道
- **格局扩展**：一座城 → 一片区域 → 一个大陆 → 天地之间

---

## 递进节奏

核心驱动力：**实力提升**。快→慢，越到后面越慢。格局扩展路径：城→区域→大陆→天地。

| 阶段 | 占比 | 关键动作 |
|------|------|---------|
| **锚定期**（前20%） | 建立修炼体系/世界观 | 让读者以为"我懂了" |
| **递进期**（20%-60%） | 三层递进 | 每层打破一个旧认知，建立一个新认知 |
| **崩塌期**（60%-80%） | 核心反转 | 所有旧规则/认知被推翻，主角必须从零重建 |
| **终局期**（80%-100%） | 收束或开放 | 给出终极答案，或留下"真正的问题还没解决" |

## 写作检验清单

- [ ] 每一章是否比上一章的冲突强度更高？（不能原地踏步）
- [ ] 每次信息揭露是否同时制造了新悬念？（揭示≠结束）
- [ ] 情绪是否有波谷？（连续高压读者会麻木，需要喘息再加压）
- [ ] 格局扩展时主角的代价是否同步增加？（免费的升级=无聊）
- [ ] 读者在第N章是否还能回忆起第1章的悬念？（伏笔回收=信任）
- [ ] 递进是否有"不可逆"节点？（至少一个无法回头的转折点）
