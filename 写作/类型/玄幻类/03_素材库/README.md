## 一、对话素材（精选自 references/05_玄幻对话库.md）

### 退婚逆袭
- 核心节奏：忍→忍→忍→一句话怼回去→走人
- 灵气炸开只写一瞬不解释，留给后续展开
- 用物象写心理震动（苏瑶茶杯裂了一道缝）

### 拍卖竞价
- 竞价的本质是权力博弈——出价数字只是表面，威压/身份/气场才是筹码
- 微动作暗示主角遇到真正对手（从翘腿到放腿）
- "加价"两个字反将一军，把压力抛回对手

### 渡劫心魔
- 内心独白与天劫的对话——心魔用理性劝退，主角用意志回应
- 每句话不超过十个字，跟雷声节奏一样短促有力
- 对天道说"你也怕了"是最经典的情绪爆发点

### 师兄弟互怼
- 表面嘲讽底层尊重——玩笑里藏着认可
- 先铺帅再破功（"裤裆裂了你没发现吧？"），笑声是最好的节奏调节器

### 秘境信任危机
- 信息不对称：双方都有证据和理由，都没有完全坦白
- 术法验证解决表面问题，但"谁都没再说话"暗示裂痕已产生

### 对话核心原则

| 原则 | 说明 |
|------|------|
| 称谓定生死 | 从"小友"到"前辈"到"蝼蚁"，称呼变化=权力关系变化 |
| 短句=实力 | 强者说话越短，灵气/威压越足；废话越多的人往往越弱 |
| 不解释>解释 | "灵气炸了一瞬"比"他展示了筑基期的实力"有力十倍 |

## 二、场景素材（精选自 references/06_玄幻剧本库.md）

### 退婚逆袭剧本要素
- 开场：穿粗布衣进拍卖会，身份反转从一块令牌开始
- 递进式竞价制造紧张感，价格数字要具体
- 打脸要淡然——"送你了""不用了""试试"
- 留一线不杀：送回魂草给退婚前未婚妻，格局>报复
- 时间线对比产生最强冲击（三年前废物vs三年后元婴）

### 渡劫突破剧本要素
- 准备阶段：七件法宝布阵，具体数字增加真实感
- 天劫层次递进：普通雷劫→天道之眼→心魔幻境→灵魂重塑
- 心魔幻境用最私人的记忆（母亲/童年），制造选择困境
- 突破后的感受要具体（"踩在空气上，脚下生出一朵金色的云"）
- 散修身份增加难度——没有宗门护法渡过了，比有背景更震撼

### 宗门大比剧本要素
- "废物"标签先行：逆袭前提是先被踩到最低
- 碾压式节奏循环：以为能随便欺负→一招秒了→全场震惊→下一轮更强对手
- 拒绝宗主收徒比接受更有冲击力——"我师父只有一个"
- 师父遗言收束情感线（"下品灵根不可怕，可怕的是下品的心"）

## 三、描写素材（精选自 references/蒸馏精华.md + SKILL.md）

### 感官描写速查

| 感官 | 关键词 |
|------|--------|
| 视觉 | 金色流光、紫电狂舞、万剑齐发如雨、丹纹流转、符文若隐若现 |
| 听觉 | 剑吟、万剑齐鸣、雷声如鼓、药炉咕嘟、拳风呼啸 |
| 触觉 | 暖流涌遍全身、浑身骨骼在重组、入口即化、入手温凉 |
| 嗅觉 | 药香扑鼻、丹香四溢、血腥味+焦糊味 |
| 特殊 | 天道威压=灵魂颤抖、心魔=恐惧放大、顿悟=眼前豁然开朗 |

### 战斗双面写法

**文雅版**：长剑出鞘，剑意如秋水漫天。一剑斩下，虚空出现一道裂缝。
**粗俗版**：拔剑就是一顿乱砍。剑气横飞，旁边那棵树直接被劈成了两半。

- 文雅版用于：正式场合、天地异象、高境界对峙
- 粗俗版用于：市井对话、轻松场景、吐槽、打脸后反应
- 全文交替使用，根据场景切换

### 写作感悟要点（来自 references/07_小说感悟蒸馏.md）
- 修仙文核心是"资源焦虑"，主角不是在修仙，是在搞钱
- 地形即隐喻：一句话把世界观核心规则写清楚
- 世界观设定和内容天然融合，省去大量解释笔墨
- 世界观可以宏大，情感锚点必须很小——读者在乎"这对姐弟什么关系"，不在乎"玄牝大阵原理"
- 角色背景故事"一把梭"比分散多章挤牙膏更适合网文节奏
- 嘴上和心里说相反的话产生喜剧效果
