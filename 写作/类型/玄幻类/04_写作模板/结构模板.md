# 玄幻类 — 结构模板

> 章节结构模板与叙事骨架。写作前用模板规划节奏，写完后用清单自检。

---

## 一、单章核心张力

每章必须回答一个问题：**这一章里，主角最怕失去什么？**

| 张力类型 | 说明 | 典型场景 |
|----------|------|----------|
| 生存张力 | 不解决就死/重伤 | 围攻、渡劫、秘境 |
| 境界张力 | 不突破就错过唯一窗口 | 限时突破、丹药时效 |
| 面子张力 | 不赢就永远被踩 | 宗门大比、退婚打脸 |
| 关系张力 | 不行动就失去重要的人 | 兄弟重伤、师父诀别 |
| 真相张力 | 不查就永远被蒙在鼓里 | 阴谋揭露、身世之谜 |

写前问自己：本章选哪种张力？如果答不上来，说明结构松了。

---

## 二、单章结构（3级场景链）

### 开场层（前15%）— 悬置问题

- 角色带着目标或被抛入新处境
- 环境/实力落差快速交代（谁强谁弱，读者30秒内必须知道）
- 章首钩子：一句对话、一个动作、一个悬念（300字内落地）

### 中段层（30%-50%）— 升温施压

- 探索/战斗/修炼的具体过程，用五感写
- 遇到其他修炼者（盟友或对手），信息差建立
- 每800-1000字给一个微爽点或微悬念，防止读者滑走
- 配角施压：同辈嘲笑、长辈忽视、对手挑衅——至少选一种

### 转折层（35%-50%）— 压力到顶→反转或代价

- 遭遇强敌/陷入陷阱/发现秘密
- 角色用底牌、智慧或付代价脱困
- 转折后必须有**可见变化**：境界涨了、关系变了、认知翻了

### 收束层（最后15%-20%）— 结算+钩子

- 获得宝物/功法/盟友/信息，或境界接近突破
- 收束本章张力，但不收束多章弧线的张力
- 章尾钩子，让读者必须翻下一章

## 三、章节节奏曲线

```
情绪强度
  ▲
  │        ╱╲         ← 转折/危机顶点
  │       ╱  ╲
  │      ╱    ╲
  │  ╱╲╱      ╲╱╲     ← 小爽点+微伏笔
  │ ╱                  ← 开场铺垫
  │╱________________________▶ 字数
  0%   15%  40%  70%  100%
```

关键节奏点：
- **0-300字**：钩子落（读者决定看不看）
- **15%处**：目标/冲突明确（读者知道追什么）
- **40-50%处**：第一个小爽或小虐（读者确认值得追）
- **70%处**：危机/转折顶点（读者屏住呼吸）
- **最后200字**：钩子抛（读者翻页）

节奏禁忌：
- 超过1000字无冲突/无爽点 = 注水
- 超过3章无爽点 = 读者弃书
- 大爽之前不铺垫小虐 = 爽感减半

## 四、角色压力点矩阵

每章至少对主角施加2个维度的压力：

| 压力维度 | 来源 | 写法 |
|----------|------|------|
| 实力差距 | 对手境界碾压 | 用破坏力可视化，不写"他很强" |
| 时间限制 | "三天内必须突破" | 倒计时挂墙上，每段都提一笔 |
| 资源匮乏 | 丹药不够/灵石耗尽 | 具体数字，"最后一颗"比"不够了"有力 |
| 人际孤立 | 盟友离开/被陷害 | 用对话和行为展现，不用旁白说"他很孤独" |
| 内心动摇 | 自我怀疑/记忆创伤 | 闪回或内心独白，3句以内收 |
| 信息盲区 | 不知道敌人底牌 | 读者和主角一起猜，制造悬念 |

## 五、常见失败模式（踩坑即扣分）

| 失败模式 | 表现 | 补救 |
|----------|------|------|
| 无张力开局 | 主角出场没事干，环境描写500字 | 砍到100字，直接进冲突 |
| 反派降智 | 敌人明明能赢非要话多 | 给反派合理的动机（试炼/取乐/逼供） |
| 升级无代价 | 闭关三天突破元婴 | 加时间跨度+失败风险+身体代价 |
| 爽点断档 | 连续3000字纯铺垫无爽点 | 每1000字插一个微爽/微悬念 |
| 结尾无钩子 | 章末主角安全了，啥也没留 | 加一句"他不知道的是……"或新信息 |
| 设定前后矛盾 | 前面说不能飞，后面主角直接飞 | 回查前文，统一设定 |
| 感官真空 | 战斗只有招式名没有画面 | 每场战斗至少覆盖视/听/触中的两项 |

## 六、结尾类型（选一种收）

| 类型 | 效果 | 适用场景 |
|------|------|----------|
| 悬念钩 | "千里之外某人睁开了眼" | 推主线，制造期待 |
| 反转钩 | 章末揭露一个读者没想到的信息 | 高潮章、阴谋线 |
| 升级钩 | "丹田内隐隐有了裂痕——突破在即" | 修炼章、历练章 |
| 情感钩 | "她望着他的背影，轻声说了一句他没听见的话" | 感情线、师徒线 |
| 危机钩 | "第三道雷劫还在酝酿，但他的灵力已经见底" | 渡劫章、围攻章 |
| 开放钩 | 两难选择摆在面前，章末不给答案 | 道德困境、立场抉择 |

## 七、多章弧线（入世-历劫-飞升型）

```
第1-3章：进入宗门/踏上修炼之路 → 第一次展示天赋
第4-6章：宗门历练/外出任务 → 遭遇第一个真正的强敌
第7-9章：陷入更大的纷争 → 突破瓶颈
第10-12章：对决Boss → 获得阶段性胜利 → 更大的世界在等待
```

## 八、开篇钩子模板

**类型A — 废材开局**：测灵台上，他的灵根显示为"无"。所有人笑了。但测灵台裂了。
**类型B — 重生开局**：他睁开眼，回到了五百年前——他还只是个练气期小修士的时候。

## 九、写作清单（每章必查）

写完后逐条过，未达标的打回重写：

- [ ] 核心张力明确：主角怕失去什么，读者15秒内能感知
- [ ] 首钩落地：前300字有钩子（对话/动作/悬念）
- [ ] 中段有爽点：每1000字至少一个微爽或微悬念
- [ ] 转折有变化：转折后角色状态可见变化（境界/关系/认知）
- [ ] 角色有压力：至少2个压力维度同时作用
- [ ] 五感到位：战斗/突破/炼丹场景覆盖视听触至少两项
- [ ] 结尾有钩子：6种结尾类型选一种，不准平收
- [ ] 设定自洽：回查前文，无逻辑/战力矛盾
- [ ] 无注水：删掉不影响剧情的段落，删
- [ ] 对话合身份：境界高的惜字如金，境界低的不要装逼
