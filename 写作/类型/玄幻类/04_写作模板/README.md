| 文件 | 用途 |

## 核心节奏

历练-危机-收获型（外出历练→遭遇危机→突破/获得机缘）

## 多章弧线速查

### 三幕结构模板

**第一幕：蛰伏（前10%-15%）**
- 开局设定：主角的困境（废柴/被退婚/家破人亡/被逐出宗门）
- 核心冲突建立：谁在欺负主角？为什么？
- 小伏笔埋设：奇遇的预兆（老爷爷、残破玉佩、神秘血脉）
- 节奏：慢，铺垫情绪，让读者恨反派

**第二幕前半：崛起（15%-40%）**
- 奇遇降临：获得金手指（系统/传承/丹药/法宝）
- 第一次打脸：最爽的一次，往往是打脸曾经羞辱自己的人
- 身份暴露：在宗门/家族中的地位开始上升
- 节奏：快，爽点密集，每2-3章一个小高潮

**第二幕后半：升级（40%-70%）**
- 更大的世界观展开：离开原来的宗门/城池，进入更大的舞台
- 新的敌人出现：比之前强一个维度的对手
- 遭遇挫折：不是所有事都顺利，偶尔被打压一下
- 支线展开：感情线、兄弟线、师徒线
- 节奏：张弛有度，打脸和挫折交替

**第三幕：封神（70%-90%）**
- 最终Boss浮出水面
- 主角整合所有资源和力量
- 大决战：不是单打独斗，而是一场牵动整个世界观的战争
- 节奏：高潮迭起，一波未平一波又起

**尾声：新起点（最后10%）**
- 收束伏笔，交代各角色结局
- 暗示更大的世界观（续集钩子）
- 节奏：缓，给读者回味空间

### 入世-历劫-飞升型（短中篇）
```
第1-3章：进入宗门/踏上修炼之路 → 第一次展示天赋
第4-6章：宗门历练/外出任务 → 遭遇第一个真正的强敌
第7-9章：陷入更大的纷争 → 突破瓶颈
第10-12章：对决Boss → 获得阶段性胜利 → 更大的世界在等待
```

### 节奏控制红线
- **爽点间隔**：不要超过5章没有爽点（读者会弃书）
- **挫败感**：每次大爽之前，要有一次小虐（反差越大越爽）
- **信息揭露**：世界观的秘密要分批揭露，不要一次性倒出来
- **伏笔回收**：每隔20-30章回收一个早期伏笔

## 使用方法

1. 写单章前，打开 结构模板.md 找到"单章结构"，按比例分配篇幅
2. 规划多章弧线时，参考本文档"多章弧线速查"
3. 选场景时，查"场景库速查"找到对应场景的核心看点和写法要点
4. 模板是骨架，不是枷锁——根据情节需要灵活调整比例
