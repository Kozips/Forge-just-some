# 玄幻类 -- 战斗体系设计

> 写玄幻之前必须定好的底层规则。战斗体系崩了，整本书就塌了。

---

## 一、战力锚定原则

### 1. 用"可感知的破坏力"代替数字

不要写"他的战斗力是5000"。要写读者能想象的画面。

| 境界 | 破坏力锚定（一句话够用） |
|------|------------------------|
| 练气 | 一拳碎石，跑起来比马快 |
| 筑基 | 御剑飞行，一剑劈开小山丘 |
| 金丹 | 领域展开，方圆百丈内草木皆兵 |
| 元婴 | 元神出窍，一掌拍碎一座城门 |
| 化神 | 掌控法则，挥手间改变地形 |
| 渡劫 | 一击可碎山岳，天劫自动追踪 |

写法：**不要直接告诉读者"他有多强"，让读者从破坏力中自己判断。**

### 2. 越级战斗的规则

越级是爽点，但必须有代价和解释。

**允许越级的合理理由**（选一个，前文必须铺垫）：
- 法宝品质碾压（拿着仙器打灵器）
- 功法品质碾压（天级功法打黄级）
- 血脉/体质特殊（万中无一的XX体）
- 战斗经验差距（老手打新手）
- 对手有弱点被主角发现

**不允许的越级**：
- 炼气期正面硬刚元婴期（没有任何铺垫的情况下）
- 主角靠"意志力"赢（意志力不是战力）

### 3. 战力一致性清单

写新情节前对照检查：

| 检查项 | 问题 |
|--------|------|
| 境界差距 | 之前说筑基打不过金丹，这次有没有违反？ |
| 法宝限制 | 之前说这件法宝每天只能用一次，这次用了几次？ |
| 速度设定 | 之前说化神期才能瞬移，现在谁在瞬移？ |
| 寿命设定 | 之前说元婴期寿命1000年，这个角色的年龄合理吗？ |
| 恢复速度 | 受了重伤，多久能恢复？和之前一致吗？ |

## 二、战斗层次设计

### 1. 个人战 vs 团战

**个人战的核心**：底牌博弈。双方一张一张亮底牌，读者在猜"他还有什么"。

```
层次结构：
试探（出三成力）→ 认真（出七成力）→ 底牌（全力）→ 绝境（最后一张牌）→ 逆转
```

**团战的核心**：配合与分工。不是一群人各打各的，而是有战术配合。

```
层次结构：
前排抗伤 → 中排输出 → 后排辅助 → 指挥调度 → 关键时刻的配合技
```

### 2. 战斗节奏公式

**短战斗**（1章内结束）：试探→一击必杀。适合碾压局。

**中战斗**（2-3章）：试探→正面对决→底牌→逆转。适合重要对手。

**长战斗**（4-5章，不建议更长）：分阶段、有间歇、穿插回忆/对话。适合Boss战。

**铁律**：超过5章的战斗，读者会跳着看。

### 3. 战斗中的"呼吸感"

不要从头打到尾。战斗中需要"呼吸"的间隙：

- 双方对峙时的对话（展示性格和动机）
- 旁观者的反应（制造震撼感）
- 回忆闪回（增加情感厚度）
- 环境变化（天色暗了、地面裂了、风起了）

## 三、法宝与技能的克制关系

### 克制设计原则

法宝/技能之间要有克制关系，不能所有东西都靠"谁灵力多谁赢"。

```
克制关系模板：

火 克 金（熔化）  金 克 木（砍伐）  木 克 土（扎根）
土 克 水（掩埋）  水 克 火（熄灭）

阵法 克 个体（围困）   个体 克 阵眼（破阵）
精神攻击 克 肉身强者   肉身 克 精神力弱者
```

### 克制的叙事价值

- 让弱者有赢的可能（"他的火焰正好克我的金系法宝"）
- 增加战斗的策略性（不是蛮力对决）
- 给主角展示智慧的机会（发现克制关系并利用）

## 四、常见战力体系模板

### 模板A：纯境界制（斗破苍穹风格）

```
境界等级：斗之气 → 斗者 → 斗师 → 大斗师 → 斗灵 → 斗王 → 斗皇 → 斗宗 → 斗尊 → 斗圣 → 斗帝

特点：
- 每个大境界内分1-9星
- 高一个大境界 = 碾压
- 同境界内靠功法、法宝、经验分胜负
```

### 模板B：途径+序列制（诡秘之主风格）

```
特点：
- 多条修炼途径，每条途径能力不同
- 越高序列越强，但代价越大（失去人性/记忆/外貌）
- 途径之间有克制关系
- 同序列不同途径各有优劣
```

### 模板C：天赋+修炼制（斗罗大陆风格）

```
特点：
- 每个人有先天天赋（武魂/血脉/灵根）
- 天赋决定上限，修炼决定下限
- 天赋可以通过特殊方式提升（吞噬/觉醒/融合）
- 天赋差异制造天然的不平等（冲突来源）
```

### 选择建议

| 类型 | 适合的故事 | 节奏 |
|------|----------|------|
| 纯境界制 | 打脸爽文、升级流 | 快 |
| 途径序列制 | 悬疑、暗黑、策略 | 中慢 |
| 天赋修炼制 | 热血、成长、团队 | 中 |

## 五、战力描写的禁忌

1. **不要用数字代替描写**："战斗力53万"没有任何画面感
2. **不要反复用同一个描写**：每次出手都是"一拳碎山"，第三次就没人看了
3. **不要让配角的战力忽高忽低**：上一章能和Boss过招，这一章被小兵打趴
4. **不要忽略战斗的代价**：打完架活蹦乱跳，读者会觉得战斗没有重量
5. **不要让主角永远先手**：偶尔被偷袭、被围攻、被打得满地找牙
