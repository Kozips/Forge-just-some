# 玄幻类 — 质检清单

> 完稿前逐项检查。打勾通过，未通过则修改。

---

## 一、通用检查项

- [ ] 无病句和错别字
- [ ] 人称一致（无视角混乱）
- [ ] 时间线连贯（无矛盾）
- [ ] 对话自然（无AI腔/书面腔），符合角色境界身份
- [ ] 每章有明确的情节推进

---

## 二、力量体系检查项

- [ ] 力量体系等级分明且一致——前文设定不与后文矛盾
- [ ] 境界突破有明确条件和代价——不能"悟了就升级"
- [ ] 战力可视化——不写"他很强"，写"他随手一挥方圆百里树木齐根断裂"
- [ ] 资源稀缺（法宝/丹药/灵石不能无限供应）——"最后一颗"比"不够了"有力
- [ ] 战力体系自洽——前面说只有元婴期才能飞，后面不能有炼气期在飞
- [ ] 每次升级有剧情意义——不是为了升级而升级，推动了情节或角色发展
- [ ] 境界感在对话中体现——炼气期唯唯诺诺，化神期云淡风轻，渡劫期不怒自威

---

## 三、爽点与节奏检查项

- [ ] 前300字有钩子（对话/动作/悬念）
- [ ] 每1000字至少一个微爽点或微悬念——不允许连续3000字纯铺垫
- [ ] 爽点间隔不超过5章——超过5章无爽点读者会弃书
- [ ] 每次大爽之前有一次小虐——反差越大越爽
- [ ] 打脸五步法完整：受辱（蓄情绪）→ 发誓/立约 → 成长过程 → 再遇 → 打脸（爆发）
- [ ] 信息差利用到位——读者知道主角多强但书中角色不知道
- [ ] 章尾有钩子，不准平收（悬念钩/反转钩/升级钩/情感钩/危机钩选一种）
- [ ] 转折后角色状态有可见变化（境界/关系/认知）

---

## 四、世界观检查项

- [ ] 世界观不只服务主角——有独立运转的势力和事件
- [ ] 势力关系有利益博弈——不是谁拳头大谁说了算
- [ ] 世界观秘密分批揭露——不一次性倒出来
- [ ] 信息揭露有节奏——每隔20-30章回收一个早期伏笔
- [ ] 每一层世界有不同规则——凡人界讲温饱，修仙界讲资源，仙界讲法则
- [ ] 世界观有限度——不能无限堆叠境界（仙帝→神帝→至尊→……读者会审美疲劳）

---

## 五、角色检查项

- [ ] 社交关系影响剧情——不是单纯的打怪升级
- [ ] 角色有独立成长弧线——主角不只长修为不长脑子，配角也有成长
- [ ] 没有突然掏出的"底牌"——所有底牌必须前文铺垫
- [ ] 反派有自己的逻辑和立场——不是纯粹的坏人，败因合理
- [ ] 女性角色有独立力量和故事——不是花瓶/被拯救对象/后宫工具人
- [ ] 对话合身份——强者惜字如金，弱者不装逼，不同境界说话方式不同
- [ ] 战斗描写有技巧性——不是纯比数值，有策略和博弈

---

## 六、常见失败模式检测

- [ ] 没有"无张力开局"——环境描写没超过100字直接进冲突？
- [ ] 没有"反派降智"——敌人有合理动机（试炼/取乐/逼供）而非话多？
- [ ] 没有"升级无代价"——闭关有时间跨度+失败风险+身体代价？
- [ ] 没有"爽点断档"——连续铺垫不超过3000字？
- [ ] 没有"结尾无钩子"——章末留了悬念或新信息？
- [ ] 没有"强行降智"——角色行为合理，不为剧情需要做蠢事？
- [ ] 没有"感情线狗血"——三角恋不拖沓，不在两人之间反复横跳200章？
- [ ] 没有"注水"——删除某段话如果不影响剧情则已删除？

---

## 七、红线验证

- [ ] 无抄袭其他作品情节/设定
- [ ] 无无底线暴力（不写虐杀/酷刑细节）
- [ ] 不涉及未成年人的任何暧昧情节
- [ ] 宗门/宗教描写不影射现实宗教组织
- [ ] 不出现现实毒品名称
- [ ] 架空世界权力斗争不影射现实政治
