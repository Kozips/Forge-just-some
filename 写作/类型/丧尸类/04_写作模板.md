# 末世章节模板

> 末世小说的核心节奏是"紧张→喘息→更紧张"的波浪式推进。以下是四种核心章节模板和多章弧线模板。

---

## 一、末世单章结构模板

### 模板A：求生型章节

**适用场景**：资源短缺、寻找物资、躲避丧尸

```
### 伪装安全（前10%）
- 角色处于"伪安全"状态（物资还够/暂时没有丧尸）
- 通过细节暗示危机逼近（罐头越来越少/药品过期）

### 危机触发（15%）
- 触发事件：物资告罄/丧尸出现/有人受伤
- 打破伪装安全，制造即时压力

### 求生决策（30%）
- 团队讨论：留守还是转移？分头还是集中？
- 必须出现"不可能同时满足"的选择冲突
  - 例：弹药只够护送一部分人，谁留下？

### 求生行动（30%）
- 执行计划，但遭遇意外
- 意外类型：丧尸变种/敌对幸存者/地形障碍
- 核心角色面临身体/心理临界点

### 代价与喘息（15%）
- 清点损失（人员/物资/信任）
- 短暂喘息，但埋下新危机伏笔
- 损失必须改变角色关系或队伍结构
```

**节奏曲线**：
```
紧张度
  ▲        ★意外
  │      ╱ ╲
  │    ╱╱   ╲
  │  ╱╱决策   ╲代价
  │╱╱          ╲──
低│伪装安全
  └──────────────→ 时间
```

---

### 模板B：冲突型章节

**适用场景**：人vs人、势力冲突、内部矛盾

```
### 矛盾铺垫（前15%）
- 展示双方立场差异（资源分配/领导权/道德观）
- 通过对话/行为暗示冲突升级

### 冲突触发（15%）
- 触发事件：资源争夺/背叛/误杀
- 不可调和的矛盾爆发

### 对抗升级（35%）
- 从言语冲突升级到肢体/武力冲突
- 第三方介入（丧尸/其他势力）使局势更复杂
- 核心角色被迫选边站

### 冲突高潮（20%）
- 生死对决/重大牺牲/不可逆决定
- 必须有人付出代价（死亡/重伤/离开）

### 余波（15%）
- 胜利者的代价/失败者的下场
- 队伍结构永久改变
- 埋下复仇/和解的伏笔
```

**节奏曲线**：
```
紧张度
  ▲           ★高潮
  │         ╱ ╲
  │       ╱╱   ╲
  │     ╱╱对抗   ╲
  │   ╱╱           ╲余波
  │ ╱╱铺垫           ╲──
低│╱
  └──────────────→ 时间
```

---

### 模板C：探索型章节

**适用场景**：探索新区域、寻找资源、发现线索

```
### 出发准备（前10%）
- 明确目标（寻找药品/探索信号源/寻找幸存者）
- 分配任务，准备装备
- 暗示风险（"上次去的人没回来"）

### 探索过程（40%）
- 环境描写：废弃建筑/荒野/丧尸痕迹
- 发现线索：日记/地图/尸体/幸存者
- 小型危机：丧尸伏击/陷阱/迷路

### 核心发现（30%）
- 发现目标物资/真相/幸存者
- 但发现带来新问题（物资有毒/真相更残酷/幸存者有危险）
- 必须做出选择：带走什么/相信什么/救谁

### 返程危机（20%）
- 返程遭遇更大威胁
- 必须舍弃部分"收获"才能逃脱
- 回到据点后，发现据点也出了问题
```

---

### 模板D：情感型章节

**适用场景**：角色关系发展、回忆、人性探讨

```
### 伪装日常（前15%）
- 难得的平静时刻（夜晚守夜/修理装备/做饭）
- 角色之间的日常互动，展示真实性格

### 情感触发（20%）
- 触发事件：发现旧照片/角色生日/有人受伤
- 唤起角色的情感记忆或脆弱面

### 情感爆发（35%）
- 角色袒露内心（恐惧/愧疚/希望/爱情）
- 必须与末世背景结合，不能脱离生存压力
- 情感爆发可以是争吵/告白/忏悔

### 关系转变（20%）
- 关系发生质变（信任/背叛/爱情/决裂）
- 转变必须影响后续剧情

### 现实回归（10%）
- 平静被打破（丧尸出现/新危机）
- 情感线与生存线交织
```

## 二、末世多章弧线模板

### 弧线A：据点建设（5-10章）

```
第1章：发现理想地点（资源丰富/易守难攻）
第2章：清理周边丧尸，建立初步防御
第3章：内部矛盾（谁领导？规则怎么定？）
第4章：外部威胁（其他势力/尸潮接近）
第5章：危机应对（战斗/谈判/转移）
第6章：据点巩固（防御升级/人员增加）
第7章：内部危机（叛徒/疾病/资源分配）
第8章：据点危机（大规模攻击/内部崩溃）
第9章：保卫战（高潮战斗）
第10章：新据点/新开始（代价与希望）
```

### 弧线B：势力争斗（8-15章）

```
第1-2章：接触其他势力，发现资源竞争
第3-4章：小规模摩擦，谈判失败
第5-6章：间谍/情报战，内部渗透
第7-8章：冲突升级，小规模战斗
第9-10章：背叛/联盟重组
第11-12章：决战准备
第13-14章：大战（高潮）
第15章：战后清算（代价/新秩序）
```

### 弧线C：真相追寻（6-12章）

```
第1-2章：发现线索（神秘信号/异常丧尸/政府文件）
第3-4章：追寻线索，遭遇阻碍
第5-6章：发现部分真相（病毒来源/政府阴谋）
第7-8章：真相带来更大危机（被追杀/丧尸变异）
第9-10章：获取完整真相的行动
第11-12章：真相揭露（可能残酷/可能无解）
```

### 弧线D：文明重建（10-20章）

```
第1-3章：据点稳定，开始农业/手工业
第4-6章：建立规则/法律，处理纠纷
第7-9章：外部势力干扰，贸易/冲突
第10-12章：科技/文化复兴（发电/教育/艺术）
第13-15章：新旧价值观冲突
第16-18章：大规模危机（瘟疫/战争/自然灾害）
第19-20章：文明延续/新希望
```

## 三、末世开篇钩子模板

### 钩子1：灾难开场

```
开头：灾难正在发生（丧尸爆发/核弹爆炸/病毒扩散）
优势：直接进入紧张状态，快速吸引读者
示例："警报声撕裂了凌晨的宁静。窗外，邻居正在啃食他的妻子。"
```

### 钩子2：事后醒来

```
开头：主角在灾难后醒来（昏迷/躲藏/冷冻）
优势：与读者同步了解世界，代入感强
示例："他睁开眼睛，发现自己躺在医院的地板上。窗外是三个月后的世界。"
```

### 钩子3：日常被打破

```
开头：展示末世前的正常生活，然后突变
优势：对比强烈，情感冲击大
示例："她正在给孩子做早餐。收音机里传来第一条紧急广播。"
```

### 钩子4：规则建立

```
开头：直接展示末世生存规则
优势：快速建立世界观，展示主角能力
示例："规则一：永远在天黑前回到据点。规则二：永远留一颗子弹给自己。"
```

## 四、末世节奏速查

### 波浪式节奏原则

```
紧张度
  ▲
高│    ★      ★      ★★★
  │   ╱ ╲    ╱ ╲    ╱ ╲
  │  ╱   ╲  ╱   ╲  ╱   ╲
中│ ╱     ╲╱     ╲╱     ╲
  │╱   喘息  喘息   喘息   ╲
低│╱
  └────────────────────────→ 章节
     1  2  3  4  5  6  7  8
```

### 节奏控制要点

1. **每3-4章必须有一个小高潮**
   - 生存危机/战斗/重大发现
   - 不能让读者进入"安全疲劳"

2. **高潮后必须有喘息**
   - 清点损失/角色互动/情感线推进
   - 喘息不超过1章，否则节奏拖沓

3. **喘息中埋下新危机伏笔**
   - 药品快用完了/有人隐瞒伤势/发现新威胁
   - 让读者在喘息中保持不安

4. **大高潮前必须有充分铺垫**
   - 至少2-3章的矛盾积累
   - 不能突然出现Boss战

### 节奏检查清单

- [ ] 每章是否有至少一个"紧张点"？
- [ ] 连续3章是否有变化（不能都是战斗/不能都是对话）？
- [ ] 高潮后是否有喘息？
- [ ] 喘息中是否埋下伏笔？
- [ ] 大高潮前是否有充分铺垫？

## 结构模板

# 丧尸废土末日类 — 结构模板（升级版）

---

## 一、核心张力

**生存成本 vs 人性底线**：每一章必须回答一个问题——"为了活下去，角色愿意付出/牺牲什么？"物资在减少，信任在流失，决策必须有不可逆的代价。没有免费的安全。

---

## 二、单章结构（生存节奏型）

```
### 危机触发（前15%）
- 发现异常迹象（远处的声响/物资告罄/有人生病）
- 打破角色当下"伪安全"状态，制造即时威胁

### 应对决策（30%）
- 团队分歧：留守还是转移？分头还是集中？
- 准备阶段：分配武器/规划路线/分配职责
- 必须出现至少一个"不可能同时满足"的选择冲突

### 行动执行（35%）
- 遭遇意外（路被堵了/遭遇敌对幸存者/丧尸变种）
- 牺牲与抉择（谁来殿后？谁带伤员？）
- 核心角色面临身体/心理双重压力的临界点

### 余波（20%）
- 清点损失（弹药/人员/物资）
- 新信息引入（发现无线电/看到远处的光/捡到地图）
- 损失必须改变角色关系或队伍结构，不能只是数字减少
```

---

## 三、三级场景链（每章内必含三层递进）

| 层级 | 功能 | 示例 |
|------|------|------|
| **开场钩（Opening）** | 日常中的裂缝，1-2段内建立不安感 | "她数到第三遍，罐头还是少了两个。" |
| **中段拉扯（Mid）** | 行动中突发变故，节奏加速，信息反转 | 突然出现的幸存者声称知道安全区位置，但身上的血不是自己的 |
| **转折锤（Turn）** | 本章最大冲击，直接改变角色处境或认知 | 殿后者通过对讲机说的最后一句话揭示了真相 |

> **递进规则**：每一层的情绪强度必须高于上一层。钩子可以是安静的，但转折锤不能是平淡的。

---

## 四、章节节奏曲线

```
情绪
  ▲
高│                    ★转折锤
  │              ╱ ╲
  │           ╱╱    ╲
  │     中段╱╱        ╲
  │      ╱╱             ╲余波落地
  │  ╱╱╱                  ╲──────
中│╱╱  开场钩（低气压铺垫）
  │
低│
  └───────────────────────────► 时间
    15%    30%    65%   85%  100%
```

- **开头不急**：用细节暗示危险，不要一上来就逃命。
- **中段变速**：决策慢（挣扎）→ 行动快（遭遇）交错推进。
- **结尾不平**：余波要留下一个未解的钩子，推动读者翻页。

---

## 五、角色压力点（每章至少命中两项）

| 压力类型 | 触发方式 | 效果 |
|----------|---------|------|
| **物资压力** | 食物/水/弹药/药品告急 | 被迫冒险、引发内部争抢 |
| **信任压力** | 有人隐瞒伤情/说谎/立场不明 | 团队裂痕、信息不对等 |
| **道德压力** | 必须抛弃某人/对人开枪/见死不救 | 角色自我认知动摇 |
| **身体压力** | 受伤/感染/饥饿/睡眠不足 | 决策质量下降、失误 |
| **心理压力** | 目睹死亡/回忆旧世界/独处 | 幻觉、崩溃、异常行为 |

---

## 六、常见失败模式

| 失败 | 表现 | 修正方式 |
|------|------|---------|
| **免费的安全** | 角色轻松逃过危机，无人付出代价 | 每次脱险必须有可量化的损失（人/物/信任） |
| **NPC团灭主角无敌** | 配角死一片，主角毫发无伤 | 主角必须带伤或失去重要东西 |
| **决策无重量** | 两条路结果差不多 | 确保选A和选B导向截然不同的后果 |
| **丧尸当背景板** | 丧尸只是噪音，不参与剧情冲突 | 让丧尸行为触发具体情节转折（封路/暴露位置/逼出隐藏伤员） |
| **余波无意义** | 损失清点完就翻篇 | 损失必须在下一章产生连锁反应 |

---

## 七、章节结尾类型

| 类型 | 适用位置 | 示例 |
|------|---------|------|
| **悬念钩** | 章末抛出未知 | 对讲机里传来一个不该活着的人的声音 |
| **代价钩** | 以损失的重量压住读者 | 他数了数弹匣，只够一个人的份 |
| **反转钩** | 推翻本章建立的认知 | 安全区的灯火不是灯塔，是大火 |
| **静默钩** | 用安静制造不安 | 四周忽然安静了。没有虫鸣，没有风声。 |
| **抉择钩** | 在两难中戛然而止 | 门内是孩子在哭。门外是它们在逼近。 |

> **禁止用"第二天醒来"式过渡收尾**。末日题材的夜晚本身就是压力源。

## 八、写作清单（每章完稿前自检）

- [ ] 核心张力是否明确？本章为"生存 vs 什么"付出了代价？
- [ ] 三级场景链是否完整？（钩→拉扯→锤）
- [ ] 转折锤的情绪强度是否高于开场钩？
- [ ] 角色压力点是否命中至少两项？
- [ ] 是否规避了常见失败模式中的任何一项？
- [ ] 章末是否使用了有效结尾类型（非"次日过渡"）？
- [ ] 损失/代价是否会在后续章节产生连锁影响？
- [ ] 丧尸/末日环境是否参与了情节推动，而非只是背景？

## 九、多章弧线（生存→希望→破灭→重建）

```
第1-3章：日常生存 → 意外发现（无线电信号）
第4-6章：组建队伍出发 → 路途艰辛
第7-8章：到达目的地 → 发现是陷阱
第9-10章：绝境中找到真正的出路
```

> **弧线原则**：希望越高，破灭越痛。前三章可以给甜头，但必须为后续的坠落埋伏笔。重建不等于回到旧世界——是角色在废墟上找到了新的活法。
