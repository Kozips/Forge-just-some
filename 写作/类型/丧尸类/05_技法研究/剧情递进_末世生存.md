# 剧情递进技法 —— 末世生存

> 来源：剧情递进手法_规则怪谈与末世生存 + 剧情递进手法_五种类型全覆盖
> 四个维度：冲突升级 · 信息解密 · 情绪压强 · 格局扩展

---

## 两种典型递进套路

**套路A：生存威胁递进——自然→人性→文明→存在**

丧尸是第一层威胁。存活下来后发现人比丧尸更危险（抢资源、背叛、暴政）。再往后发现文明本身在崩塌（没有法律、没有秩序、没有未来）。最终问题是：活着本身还有意义吗？

**套路B：资源递进——缺食物→缺安全→缺人→缺希望**

每一次解决一个资源问题，暴露出下一个更深的稀缺。有食物了但不安全。安全了但人不够。人够了但看不到重建的可能。递进的本质是"每次以为到底了，发现还有更深的坑"。

---

## 具体例子：《末日蟑螂》递进结构

> 第一阶段（1-100章）：主角张小帅在丧尸潮中活下来。冲突是"我vs丧尸"。目标是"找到食物和武器"。信息是"丧尸怕火、怕噪音、靠气味追踪"。情绪是恐惧+求生本能。格局是一栋楼、一条街。
>
> 第二阶段（100-300章）：建立小团队。冲突变成"我们vs其他幸存者团体"。资源争夺、领地划分、信任危机。信息是"谁可以信任、谁在撒谎、谁有隐藏目的"。情绪是疲惫+道德挣扎。格局是一片城区。
>
> 第三阶段（300章+）：势力战争。冲突是"组织vs组织"。军事化、政治化。信息是"这场灾难的真正原因、政府的隐瞒、变异进化的规律"。情绪是麻木+使命感。格局是整个区域乃至全国。

**递进拆解：**
- **冲突升级**：个人生存 → 团队信任 → 势力博弈 → 文明重建
- **信息解密**：丧尸弱点 → 人心善恶 → 灾难真相 → 人类未来
- **情绪压强**：恐惧求生 → 疲惫挣扎 → 道德困境 → 存在意义
- **格局扩展**：一栋楼 → 一片城区 → 一座城市 → 整个文明

## 递进节奏

核心驱动力：**资源稀缺**。中等节奏，50-100章一层。格局扩展路径：据点→城区→区域→文明。

| 阶段 | 占比 | 关键动作 |
|------|------|---------|
| **锚定期**（前20%） | 建立末世规则/生存逻辑 | 让读者以为"我懂了" |
| **递进期**（20%-60%） | 三层递进 | 每层打破一个旧认知，建立一个新认知 |
| **崩塌期**（60%-80%） | 核心反转 | 所有旧规则/认知被推翻，主角必须从零重建 |
| **终局期**（80%-100%） | 收束或开放 | 给出终极答案，或留下"真正的问题还没解决" |

## 写作检验清单

- [ ] 每一章是否比上一章的冲突强度更高？（不能原地踏步）
- [ ] 每次信息揭露是否同时制造了新悬念？（揭示≠结束）
- [ ] 情绪是否有波谷？（连续高压读者会麻木，需要喘息再加压）
- [ ] 格局扩展时主角的代价是否同步增加？（免费的升级=无聊）
- [ ] 读者在第N章是否还能回忆起第1章的悬念？（伏笔回收=信任）
- [ ] 递进是否有"不可逆"节点？（至少一个无法回头的转折点）
