## 一、物资争夺场景

### 场景1：超市对峙
> 老程走到仓库门口——四个人。一个在搬箱子，两个在货架上翻，一个坐在箱子上抽烟。
> "路过的。"老程把手电往下移。不照脸——这是规矩：照别人的脸等于宣战。
> "这个仓库我们占了。"坐着的那个人——阿彪。"这些东西至少要搬三四趟。天一亮就危险了。我可以帮你搬——你分我一半。"

**谈判要素**：不直接开打（打=两败俱伤）；用逻辑说服；用筹码。

### 场景2：药品争夺
> 父亲把箱子护在身后。「我女儿在发烧。四十度。」
> 年轻男人的弩对准了父亲。「放下。我全家都死了。」
> 老太太翻出布洛芬。「这个是儿童的。够吃三天。」
> 「小伙子。你把弩放下来。你打死了他——你能保证晚上不做噩梦？」
>
> 弩抖了一下。「一人一半。」

**关键**：不是谁强谁赢，是谁的理由更能打动旁观者。

## 二、信任建立场景

### 场景3：废墟中相遇
> 阿亮端铁管走上六楼，手电照到一双鞋。立刻关了。
> "别紧张。我也是活人。"女声。沙哑。
> 女人掰了压缩饼干递过去。「怕我下毒？我连盐都没有。」

**信任建立阶梯**：
1. 声明无害（"我也是活人"）
2. 建立规矩（"我的东西是我的"）
3. 试探意愿（分饼干）
4. 确认底线（"各守一半夜"）
5. 小步靠近（"一起往北走"）

### 场景4：末日中的温柔
> 他把最后一罐罐头推给她。她没接。两个人看着罐头坐了十分钟。

**简写法**：不写对话，只写动作。罐头+沉默=比任何台词都有力。

## 三、道德困境模板

| 类型 | 问题 |
|------|------|
| 资源分配 | 有限药品给谁？孩子还是伤员？ |
| 信任困境 | 陌生人是敌是友？被咬的人能不能留下？ |
| 个体vs群体 | 牺牲一人当诱饵救其他人？ |
| 真相vs谎言 | 安全区已破，告不告诉？ |
| 复仇vs宽恕 | 抛弃同伴的人来求救？ |
| 过去vs未来 | 旧文明还是新秩序？坚守道德还是适应丛林法则？ |

## 四、冲突升级路径

```
意见分歧 → 小圈子形成 → 权力争夺 → 资源争夺 → 暴力冲突 → 不可逆伤害
```

每一步都可以打断，打断的难度随升级递增。

## 五、角色对话风格速查

| 角色类型 | 对话风格 | 示例 |
|---------|---------|------|
| 退伍兵 | 简洁、方位词多 | "两点钟方向。低姿。快。" |
| 沉默型 | 极少说话 | "走。""不。""嗯。" |
| 话唠型 | 不停说话对抗恐惧 | "你知道吗，末日前我最怕蜘蛛。现在想想……" |
| 老人 | 经验+沧桑 | "我活了六十年，没见过这么安静的天。" |

**对话潜台词**：
- 不说"我爱你"："弹匣给你。我只有三发了。"
- 不说"我害怕"："我们还有多远？"
