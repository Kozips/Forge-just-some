# 丧尸类 技法研究

---

## 多视角与口述体技法

---

## 一、三种视角方案

### 方案A：第一人称（单一视角）
**适用**：个人成长线、心理恐惧极致、限知悬疑

**技法要点**：
- 主角必须是"在场者"，无法用"听说"代替
- 心理活动是主要叙事资源
- 关键信息通过"文件/录音/他人转述"补足

**写作模板**：
> 我走到门口的时候停住了。不是因为看到了什么。是因为——我听到了什么。门后面有声音。不是风声。是呼吸声。

### 方案B：第三人称有限视角（推荐）
**适用**：最通用方案，代入感与叙事自由度平衡

**技法要点**：
- 每章/每节开头明确交代视角人物（名字/称谓/标志性习惯）
- 不同视角人物对同一事件的感知必须有差异
- 视角切换信号：章节标题/空白分隔/场景转换
- 3-5个核心视角足够，超过7个必然稀释代入感

**写作模板**：
> 【阿亮视角】他站在废墟顶上看着下面。那三个人——正在靠近超市入口。
>
> 【阿霞视角】超市门口有人。阿霞数了数，三个人。动作很慢——她在观察有没有陷阱。

### 方案C：多视角口述体（高级）
**适用**：宏大叙事、全球视野、拼图式真相揭露

**技法要点**：
- 每个视角人物必须有独特声音（职业/性格/口癖）
- 信息差是核心悬疑——A视角读者知道B即将踏入陷阱
- 真相通过碎片拼凑，而非直接揭露

**口述体模板**：
> "我当时在——在哪里来着——对，七楼。半夜被吵醒的。不是那种普通的吵。是——你知道那种声音吗？就是——不是尖叫。是那种——"他停了三秒。"反正你懂。"

## 二、视角切换节奏控制

| 频率 | 效果 | 适用类型 |
|------|------|---------|
| 每章一换 | 节奏稳定 | 中长篇连载 |
| 每节一换 | 节奏快，悬疑强 | 短篇/中篇 |
| 事件驱动切换 | 自然流畅 | 有明确核心事件 |

**该切换时**：视角人物即将死亡（提前切入其他视角）；真相需要拼图；情感对比
**不该切换时**：读者刚建立代入感（至少200-500字）；情绪高潮中

**视角人物功能分配**：
| 功能 | 视角人物类型 |
|------|-----------|
| 代入/成长 | 主角——从普通人到领导者 |
| 外部观察 | 新人/旁观者——带入新环境 |
| 道德镜像 | 对立立场角色——展示对立合理性 |
| 信息节点 | 情报/记录者——收集/传递信息 |
| 情感锚点 | 脆弱/核心关系——孩子/老人/伤员 |

## 三、口述体六种声音类型

**1. 士兵型**——简洁、方位词多、战术细节
> "我们在第三街区设了防线。是被堵在那儿的。丧尸从东边来。我们有十三个人。撑了四十分钟。"

**2. 平民型**——日常细节、情感丰富、时间模糊
> "那天我记得——是周三吧？不对。反正我那天在做饭。停电了。冰箱里的东西开始化。"

**3. 官员型**——宏观视角、数据引用、责任感
> "城市总人口约四百万。灾变后72小时内，有效幸存者降至约1.2%。"

**4. 医生型**——专业术语、冷静观察
> "转化时间平均四小时。从咬伤到完全转化，不到三十分钟。那个病例——我记得她的眼睛。"

**5. 孩子型**——直接、简单逻辑、残忍天真
> "妈妈不让我出去。说外面有坏人。后来我就没再见过妈妈了。"

**6. 老人型**——历史感、对比视角
> "我活了六十三年。但话说回来——我外公说过打仗的事。路断了，店关了。有些事是会重复的。"

## 四、叙事技巧

### 节奏控制
**紧张-缓和波浪式结构**：永远不让读者完全放松，也永远不让读者一直紧绷。

**六种章末钩子**：
1. 悬念中断："主角打开门。"
2. 信息揭露："她摘下面具。那张脸——已经死了三天了。"
3. 决策困境："左边弹药耗尽的队友，右边哭喊的孩子。"
4. 氛围暗示："风停了。连虫鸣也停了。"
5. 情绪转折："他笑了。在这个世界里，没有人会这样笑。"
6. 时间压力："距离防线崩溃还有六个小时。"

**节奏加减速**：加速用短句+密集动词+删除独白；减速用长句+回忆+详写小物件。

### 恐怖氛围营造
**温水煮青蛙式渐进恐怖**：
1. 细微违和："路上太安静了。"
2. 具体异常："便利店门开着，但货架食物没人动过。"
3. 威胁逼近："收银台后有一摊没干透的血。"
4. 恐怖降临："血迹延伸到仓库。仓库门从里面顶着。"

**日常物件恐怖化**：还在报时的闹钟（空房间）、秋千在无风中摇摆、手机突然响起（已死之人的号码）

### 人性冲突四维度
1. **信任与背叛**：最有力的背叛来自最亲密的同伴
2. **个体与集体**：资源只够救一部分人时谁来决定
3. **文明与野蛮**："我们和丧尸有什么区别？"
4. **过去与现在**：旧世界记忆是指引也是负担

**冲突升级**：意见分歧→小圈子→权力争夺→资源争夺→暴力冲突→不可逆伤害

### 废土美学
**五感废土化**：视觉=灰褐锈红焦黑；听觉=安静；嗅觉=锈蚀+腐烂；触觉=粗糙+冰冷

**"废墟中的花"**：废弃大楼裂缝中的野花、孩子在废墟墙上用木炭画的画

### 对话风格
**潜台词**：
- 不说"我爱你"："弹匣给你。我只有三发了。"
- 不说"我害怕"："我们还有多远？"

### 角色死亡
**死亡前**：给角色具体的个人化梦想；展示脆弱面。
**死亡写法**：最有力的死亡是极度克制的——"他倒下了。"
**死亡后**：缺席的力量——习惯坐的位置空了，多拿了一份食物

### 希望与绝望平衡
**纯粹绝望是陷阱**。**希望要小而真**——一棵树、一个发芽的种子。

**"运火"主题**：角色坚持做"没有意义"的事——埋葬死者、记录名字、教孩子读书。

## 五、剧情递进技法

### 两种递进套路

**套路A：生存威胁递进**——自然→人性→文明→存在
**套路B：资源递进**——缺食物→缺安全→缺人→缺希望

### 递进节奏

| 阶段 | 占比 | 关键动作 |
|------|------|---------|
| **锚定期**（前20%） | 建立末世规则 | 让读者以为"我懂了" |
| **递进期**（20%-60%） | 三层递进 | 每层打破旧认知，建立新认知 |
| **崩塌期**（60%-80%） | 核心反转 | 所有旧规则被推翻 |
| **终局期**（80%-100%） | 收束或开放 | 给出终极答案 |

### 写作检验清单
- [ ] 每一章是否比上一章冲突强度更高？
- [ ] 每次信息揭露是否同时制造了新悬念？
- [ ] 情绪是否有波谷？（连续高压读者会麻木）
- [ ] 格局扩展时主角的代价是否同步增加？
- [ ] 递进是否有"不可逆"节点？
