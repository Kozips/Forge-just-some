# 丧尸类 专业知识

> 末世小说的核心是"生存感"。读者必须相信角色的每一个决策都是生死攸关的。

---

## 一、末世生存要素——写作要点提炼

> 具体数值（保质期、净水方法等）属通识，不重复列出。以下只保留**能直接驱动剧情的写作要点**。

- **食物**：食物短缺是最基础的冲突触发器；食物分配问题引发团队内部矛盾；食物中毒/变质可制造意外危机；长期缺乏新鲜食物导致坏血病等疾病
- **水**：缺水比缺食物更紧迫（人3天不喝水会死）；水源争夺是势力冲突的常见导火索；水污染可导致大规模疾病爆发
- **武器**：弹药稀缺是维持紧张感的关键；噪音吸引丧尸，枪械使用需要权衡；近战武器的维护和损耗是细节真实感来源
- **医疗**：感染是末世第一杀手，抗生素=命；心理创伤（PTSD）是长期压力的必然后果
- **通讯**：通讯中断是末世的重要特征；无线电可接收外界信息推动剧情；通讯设备的电量/信号是持续紧张源
- **能源**：电力恢复是文明重建的重要标志；能源争夺可成为势力冲突的核心

## 二、丧尸类型速查

| 类型 | 特征 | 写作价值 |
|------|------|---------|
| 慢速丧尸（经典型） | 行动缓慢，数量众多 | 强调数量压迫感，考验资源管理和团队协作 |
| 快速丧尸（狂暴型） | 行动迅速，感染快 | 强调即时恐惧，考验反应和逃跑能力 |
| 智能丧尸（进化型） | 保留部分智能，会设伏/使用工具 | 模糊人与非人的界限，制造信任危机 |
| 变异丧尸（Boss型） | 特殊变异，体型/能力异常 | 制造阶段性高潮，考验战术应变 |
| 尸潮（灾难型） | 大规模丧尸群，如潮水涌来 | 制造末日感，迫使角色转移/逃亡 |
| 特殊感染（规则型） | 特殊触发条件（声音/光线/时间） | 创造独特的生存规则，考验纪律性 |

## 三、幸存者团体类型——写作价值

| 类型 | 核心写作价值 |
|------|-------------|
| 独狼 | 孤独叙事，信任主题的极端测试 |
| 小队（2-10人） | 分工协作+内部矛盾，最适合写亲密关系 |
| 社区（10-100人） | 资源管理+派系政治，权力分配叙事 |

| 邪教 | 极端信仰+领袖崇拜，非理性决策导致灾难 |
| 匪帮 | 弱肉强食的短期逻辑，不可持续的必然崩溃 |

## 四、末世社会重建阶段

### 阶段1：求生期（0-3个月）
- **冲突**：人vs丧尸，人vs人（争夺资源）
- **社会结构**：无政府状态，强者为王
- **写作重点**：生存压力，道德底线试探

### 阶段2：据点期（3-12个月）
- **冲突**：据点间摩擦，内部管理矛盾
- **社会结构**：领袖+核心团队
- **写作重点**：权力分配，规则建立

### 阶段3：势力期（1-3年）
- **冲突**：势力间战争，资源争夺升级
- **社会结构**：军事化管理，阶级分化
- **写作重点**：政治博弈，战争描写

### 阶段4：文明期（3年以上）
- **冲突**：新旧价值观冲突，身份认同
- **社会结构**：法律/政府雏形
- **写作重点**：人性回归，文明反思

## 五、感染机制六分类

### 1. 咬伤感染型（最经典）
**叙事价值**：每一次近战都是赌命，被咬=倒计时启动。
**设计要点**：
- 潜伏期长短决定剧情张力——短（分钟级=即时恐惧）vs 长（小时级=告别戏空间）
- 感染者意识残留程度：完全丧失 vs 逐渐丧失（后者更残忍）
- 关键问题：咬伤是否必然感染？如有例外概率，则产生"赌一把"的道德困境

### 2. 空气/体液传播型
**叙事价值**：无法仅靠"躲起来"生存——你呼吸的每一口空气都可能是死因。
**设计要点**：
- 传播条件限制（密闭空间更危险/特定天气加速传播）
- 防护手段设计（口罩/防毒面具/密封设施）
- "安全区"概念被根本动摇

### 3. 全民潜伏型（最绝望）
**叙事价值**：任何人都是潜在敌人——死于意外/疾病/衰老都会变丧尸。
**设计要点**：
- 死亡管理成为核心议题（死者必须被处理头部）
- 自然死亡=新的丧尸出现，聚落永无宁日
- 对人性信任的根本性打击

### 4. 信号/条件触发型
**叙事价值**：恐惧从"敌人"转向"环境"——你自身的行为可能触发灾难。
**设计要点**：
- 触发条件的发现过程本身就是悬疑线
- "禁声/禁光"规则创造独特的生存美学
- 禁忌一旦被打破的戏剧张力极高

### 5. 寄生/真菌/生物型
**叙事价值**：感染者保留部分行为模式甚至记忆，模糊"人"与"非人"的边界。
**设计要点**：
- 感染后的行为可被"编程"（如真菌控制行为模式）
- 可能存在"半感染"状态，保留部分人性
- 科学视角切入，提供解药/疫苗的可能性

### 6. 超自然/诅咒型
**叙事价值**：无法用科学理解和对抗，增加无力感和存在主义恐惧。
**设计要点**：
- 规则必须自洽（即使是超自然也要有内在逻辑）
- 传统武器可能无效，需要非常规手段
- 可以融合民俗恐怖元素

## 六、感染后意识残余度光谱

这是最有戏剧价值的设定维度——感染者"还剩多少人性"。

| 残余度 | 表现 | 叙事可能性 |
|--------|------|-----------|
| 0% | 完全野兽化 | 纯战斗/逃跑 |
| 10% | 对特定事物有反应（旧歌/旧照片） | 哀伤/日常物件恐怖化 |
| 30% | 保留本能行为（回家/找食物） | "它还认得路" |
| 50% | 能辨认亲人但无法控制攻击 | 被咬队友的告别场景 |
| 70% | 部分语言能力+意识 | "它在说话……在求救" |
| 90% | 几乎正常但身体变异 | 最残忍的道德困境——杀不杀？ |

**核心原则**：意识残余度越高，道德困境越尖锐，读者情感投入越大。

## 七、转化时间与剧情节奏

| 潜伏期 | 叙事效果 | 适用场景 |
|--------|---------|---------|
| 秒级 | 即时恐惧、零缓冲 | 战斗场景、大规模感染 |
| 分钟级 | 短促告别 | 被咬后的即时反应 |
| 小时级 | 经典告别戏空间 | 队友感染核心场景 |
| 天级 | 悬疑+猜忌 | "谁已经被咬了？" |
| 周级 | 长线伏笔 | 信任危机、阵营分裂 |
| 不确定 | 最大恐怖 | 每个人都可能是定时炸弹 |

## 八、被咬后的叙事利用

### 倒计时机制

被咬=倒计时开始，是天然的戏剧冲突来源。

**转化时间设计**：
- 10秒~1分钟：极致紧张，战斗中直接转化
- 1~6小时：标准告别时间，可写完整的被咬后对话
- 6~72小时：慢性转化，角色在清醒中逐渐失去自我
- 全员感染：随时可能转化，信任崩塌

### 被咬后对话示范

**语言退化**：从"笑"到"求"到"沉默"
- 初期："别担心，我体质好。"
- 中期："你们快走，别管我。"
- 后期："求你了，别看着我变。"

**核心恐惧**：不是怕死，是怕死后变成另一种东西

## 九、特殊感染者设计

### 设计原则

1. 每种特殊感染者解决一个特定问题（不是更强，而是"不同"）
2. 必须有弱点，否则就是无解噩梦
3. 首次出场要有戏剧性——让角色措手不及

### 常见类型

- **猎杀者**：专门追杀落单者，制造"别落单"的规则
- **潜伏者**：伪装成普通丧尸，关键时刻暴露
- **指挥者**：控制尸群战术，优先击杀
- **自爆者**：移动炸弹，制造恐慌
- **变形者**：改变形态，打破认知

## 十、创作自检

- [ ] **起源清晰**：丧尸是怎么来的？读者知道吗？
- [ ] **进化有逻辑**：变异体的出现有规律可循？
- [ ] **行为有规律**：丧尸不是随机出现，有行为模式？
- [ ] **威胁有层次**：普通丧尸→变异体→人类敌人，递进关系？
- [ ] **被咬有利用**：被咬后的倒计时写出了戏剧张力？
- [ ] **人性在撕裂**：丧尸之外的人与人冲突足够？
