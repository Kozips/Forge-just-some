## 单章结构速查（生存节奏型）

| 阶段 | 占比 | 关键动作 |
|------|------|----------|

---

## 角色压力点速查（Top 5）

1. **物资压力** — 食物/弹药/药品告急，被迫冒险
2. **信任压力** — 有人隐瞒伤情/立场不明，团队裂痕
3. **道德压力** — 必须抛弃某人/见死不救
4. **身体压力** — 受伤/感染/饥饿，决策质量下降
5. **心理压力** — 目睹死亡/回忆旧世界，崩溃与幻觉

---

## 常见失败模式速查（Top 5）

1. **免费的安全** — 无人付出代价 -> 每次脱险必须有可量化损失
2. **NPC团灭主角无敌** — 主角毫发无伤 -> 主角必须带伤或失去重要东西
3. **决策无重量** — 两条路结果差不多 -> 选A和选B导向截然不同后果
4. **丧尸当背景板** — 不参与剧情 -> 让丧尸行为触发情节转折
5. **余波无意义** — 损失清点完就翻篇 -> 损失必须产生连锁反应

## 写作自检清单

- [ ] 核心张力明确？本章为"生存 vs 什么"付出了代价？
- [ ] 三级场景链完整？（钩->拉扯->锤）
- [ ] 转折锤的情绪强度高于开场钩？
- [ ] 角色压力点命中至少两项？
- [ ] 避免了免费的安全 / 主角无敌 / 决策无重量？
- [ ] 丧尸/末日环境参与了情节推动而非只是背景？
- [ ] 五感调动到位？不只有视觉？
- [ ] 废墟中有花？至少一处美好意象制造对比？
- [ ] 章末有有效钩子（非"次日过渡"）？
- [ ] 损失/代价会在后续章节产生连锁影响？
