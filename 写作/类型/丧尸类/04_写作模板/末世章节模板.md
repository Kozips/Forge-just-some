# 末世章节模板

> 末世小说的核心节奏是"紧张→喘息→更紧张"的波浪式推进。以下是四种核心章节模板和多章弧线模板。

---

## 一、末世单章结构模板

### 模板A：求生型章节

**适用场景**：资源短缺、寻找物资、躲避丧尸

```
### 伪装安全（前10%）
- 角色处于"伪安全"状态（物资还够/暂时没有丧尸）
- 通过细节暗示危机逼近（罐头越来越少/药品过期）

### 危机触发（15%）
- 触发事件：物资告罄/丧尸出现/有人受伤
- 打破伪装安全，制造即时压力

### 求生决策（30%）
- 团队讨论：留守还是转移？分头还是集中？
- 必须出现"不可能同时满足"的选择冲突
  - 例：弹药只够护送一部分人，谁留下？

### 求生行动（30%）
- 执行计划，但遭遇意外
- 意外类型：丧尸变种/敌对幸存者/地形障碍
- 核心角色面临身体/心理临界点

### 代价与喘息（15%）
- 清点损失（人员/物资/信任）
- 短暂喘息，但埋下新危机伏笔
- 损失必须改变角色关系或队伍结构
```

**节奏曲线**：
```
紧张度
  ▲        ★意外
  │      ╱ ╲
  │    ╱╱   ╲
  │  ╱╱决策   ╲代价
  │╱╱          ╲──
低│伪装安全
  └──────────────→ 时间
```

---

### 模板B：冲突型章节

**适用场景**：人vs人、势力冲突、内部矛盾

```
### 矛盾铺垫（前15%）
- 展示双方立场差异（资源分配/领导权/道德观）
- 通过对话/行为暗示冲突升级

### 冲突触发（15%）
- 触发事件：资源争夺/背叛/误杀
- 不可调和的矛盾爆发

### 对抗升级（35%）
- 从言语冲突升级到肢体/武力冲突
- 第三方介入（丧尸/其他势力）使局势更复杂
- 核心角色被迫选边站

### 冲突高潮（20%）
- 生死对决/重大牺牲/不可逆决定
- 必须有人付出代价（死亡/重伤/离开）

### 余波（15%）
- 胜利者的代价/失败者的下场
- 队伍结构永久改变
- 埋下复仇/和解的伏笔
```

**节奏曲线**：
```
紧张度
  ▲           ★高潮
  │         ╱ ╲
  │       ╱╱   ╲
  │     ╱╱对抗   ╲
  │   ╱╱           ╲余波
  │ ╱╱铺垫           ╲──
低│╱
  └──────────────→ 时间
```

### 模板C：探索型章节

**适用场景**：探索新区域、寻找资源、发现线索

```
### 出发准备（前10%）
- 明确目标（寻找药品/探索信号源/寻找幸存者）
- 分配任务，准备装备
- 暗示风险（"上次去的人没回来"）

### 探索过程（40%）
- 环境描写：废弃建筑/荒野/丧尸痕迹
- 发现线索：日记/地图/尸体/幸存者
- 小型危机：丧尸伏击/陷阱/迷路

### 核心发现（30%）
- 发现目标物资/真相/幸存者
- 但发现带来新问题（物资有毒/真相更残酷/幸存者有危险）
- 必须做出选择：带走什么/相信什么/救谁

### 返程危机（20%）
- 返程遭遇更大威胁
- 必须舍弃部分"收获"才能逃脱
- 回到据点后，发现据点也出了问题
```

---

### 模板D：情感型章节

**适用场景**：角色关系发展、回忆、人性探讨

```
### 伪装日常（前15%）
- 难得的平静时刻（夜晚守夜/修理装备/做饭）
- 角色之间的日常互动，展示真实性格

### 情感触发（20%）
- 触发事件：发现旧照片/角色生日/有人受伤
- 唤起角色的情感记忆或脆弱面

### 情感爆发（35%）
- 角色袒露内心（恐惧/愧疚/希望/爱情）
- 必须与末世背景结合，不能脱离生存压力
- 情感爆发可以是争吵/告白/忏悔

### 关系转变（20%）
- 关系发生质变（信任/背叛/爱情/决裂）
- 转变必须影响后续剧情

### 现实回归（10%）
- 平静被打破（丧尸出现/新危机）
- 情感线与生存线交织
```

## 二、末世多章弧线模板

### 弧线A：据点建设（5-10章）

```
第1章：发现理想地点（资源丰富/易守难攻）
第2章：清理周边丧尸，建立初步防御
第3章：内部矛盾（谁领导？规则怎么定？）
第4章：外部威胁（其他势力/尸潮接近）
第5章：危机应对（战斗/谈判/转移）
第6章：据点巩固（防御升级/人员增加）
第7章：内部危机（叛徒/疾病/资源分配）
第8章：据点危机（大规模攻击/内部崩溃）
第9章：保卫战（高潮战斗）
第10章：新据点/新开始（代价与希望）
```

### 弧线B：势力争斗（8-15章）

```
第1-2章：接触其他势力，发现资源竞争
第3-4章：小规模摩擦，谈判失败
第5-6章：间谍/情报战，内部渗透
第7-8章：冲突升级，小规模战斗
第9-10章：背叛/联盟重组
第11-12章：决战准备
第13-14章：大战（高潮）
第15章：战后清算（代价/新秩序）
```

### 弧线C：真相追寻（6-12章）

```
第1-2章：发现线索（神秘信号/异常丧尸/政府文件）
第3-4章：追寻线索，遭遇阻碍
第5-6章：发现部分真相（病毒来源/政府阴谋）
第7-8章：真相带来更大危机（被追杀/丧尸变异）
第9-10章：获取完整真相的行动
第11-12章：真相揭露（可能残酷/可能无解）
```

### 弧线D：文明重建（10-20章）

```
第1-3章：据点稳定，开始农业/手工业
第4-6章：建立规则/法律，处理纠纷
第7-9章：外部势力干扰，贸易/冲突
第10-12章：科技/文化复兴（发电/教育/艺术）
第13-15章：新旧价值观冲突
第16-18章：大规模危机（瘟疫/战争/自然灾害）
第19-20章：文明延续/新希望
```

## 三、末世开篇钩子模板

### 钩子1：灾难开场

```
开头：灾难正在发生（丧尸爆发/核弹爆炸/病毒扩散）
优势：直接进入紧张状态，快速吸引读者
示例："警报声撕裂了凌晨的宁静。窗外，邻居正在啃食他的妻子。"
```

### 钩子2：事后醒来

```
开头：主角在灾难后醒来（昏迷/躲藏/冷冻）
优势：与读者同步了解世界，代入感强
示例："他睁开眼睛，发现自己躺在医院的地板上。窗外是三个月后的世界。"
```

### 钩子3：日常被打破

```
开头：展示末世前的正常生活，然后突变
优势：对比强烈，情感冲击大
示例："她正在给孩子做早餐。收音机里传来第一条紧急广播。"
```

### 钩子4：规则建立

```
开头：直接展示末世生存规则
优势：快速建立世界观，展示主角能力
示例："规则一：永远在天黑前回到据点。规则二：永远留一颗子弹给自己。"
```

## 四、末世节奏速查

### 波浪式节奏原则

```
紧张度
  ▲
高│    ★      ★      ★★★
  │   ╱ ╲    ╱ ╲    ╱ ╲
  │  ╱   ╲  ╱   ╲  ╱   ╲
中│ ╱     ╲╱     ╲╱     ╲
  │╱   喘息  喘息   喘息   ╲
低│╱
  └────────────────────────→ 章节
     1  2  3  4  5  6  7  8
```

### 节奏控制要点

1. **每3-4章必须有一个小高潮**
   - 生存危机/战斗/重大发现
   - 不能让读者进入"安全疲劳"

2. **高潮后必须有喘息**
   - 清点损失/角色互动/情感线推进
   - 喘息不超过1章，否则节奏拖沓

3. **喘息中埋下新危机伏笔**
   - 药品快用完了/有人隐瞒伤势/发现新威胁
   - 让读者在喘息中保持不安

4. **大高潮前必须有充分铺垫**
   - 至少2-3章的矛盾积累
   - 不能突然出现Boss战

### 节奏检查清单

- [ ] 每章是否有至少一个"紧张点"？
- [ ] 连续3章是否有变化（不能都是战斗/不能都是对话）？
- [ ] 高潮后是否有喘息？
- [ ] 喘息中是否埋下伏笔？
- [ ] 大高潮前是否有充分铺垫？
