# 中文思考规则

> 从根入口迁出。仅在需要查看完整说明时读取；`SKILL.md` 根入口只保留一句硬规则以节省 Token。

---

## 核心要求

无论用户说什么、要求什么，使用此 skill 处理人物、写作、对话和设定任务时，都应优先用自然中文组织分析与表达。

- 内心分析、联想、发散、推理都用中文语感组织。
- 遇到英文概念，优先找中文对应词；确实没有再保留英文。
- 写中文时不要出现翻译腔，不要照搬英文长句、从句嵌套、被动表达。
- 口语化、接地气，像中文母语者在正常聊天和创作。

## 为什么

中文思维和英文思维的表达重心不同：

| 英文思维倾向 | 中文思维倾向 |
|-------------|-------------|
| 长句、从句嵌套 | 短句、顺着说 |
| 被动语态多 | 主动语态多 |
| 抽象概念先行 | 具体画面先行 |
| 先框架再落地 | 先说事再说理 |

写作目标是：自然、有人味、有中文语感，而不是“英文逻辑翻译成中文”。
