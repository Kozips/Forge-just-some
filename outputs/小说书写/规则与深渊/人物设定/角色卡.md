# 角色卡

## 主角

### 沈厌

**基本信息**
| 项目 | 内容 |
|------|------|
| 年龄 | 26岁 |
| 职业 | 前恐怖小说编辑（已辞职三个月） |
| 外貌 | 偏瘦，黑眼圈很重，头发乱糟糟的，戴一副银框眼镜，镜片总是脏的 |
| 身高 | 176cm |

**核心性格**
- 怕死、惜命、极度务实
- 理性到冷血，但嘴上永远在抖
- 观察力极强（职业病），在恐惧中反而更冷静——不是勇敢，是肾上腺素过载后的反常冷静
- 社恐宅男，说话磕巴，遇到危险第一反应是"我不想死"
- 内心独白丰富且自嘲，嘴上怂但身体诚实

**说话风格**
- 短句+碎碎念+自嘲
- 遇到恐怖的事：用极度理性的语气说出荒诞的话
- 紧张时会念叨概率："我合理怀疑这扇门后面有东西。""但我更合理地怀疑我不开门会死。"
- 口头禅："等一下。""不对。""我算算。"
- 从不说豪言壮语，但关键时刻会冒出让人意外的话

**人物弧线**
1. 开局：被扔进副本，只想活着出去
2. 发展：靠推理和系统逐渐掌控局面，发现自己比想象中更能忍
3. 转折：深渊之眼让他看到真相，但代价是记忆
4. 收束：用最怂的方式做了最勇的事

**辨识度标签**
- 永远在算"存活概率"
- 怕黑但眼睛适应黑暗极快（系统被动）
- 发抖的手打字极准（前编辑的职业病）
- 手机壁纸是"今天也要活着"

## 副本一角色

### 林若（女，28岁）

**标签**：急诊科护士 / 冷静型 / 实用主义

**说话风格**：语速快，信息密度高，不说废话。"别废话，谁受伤了？"

**性格**：看起来冷淡，实际是职业习惯——急诊科见多了死人，情绪阈值很高。对沈厌的碎碎念免疫，偶尔会接一句让人意外的冷幽默。

**用法**：理性型队友，与沈厌形成"两个理性人互相补刀"的组合。

---

### 赵铁柱（男，35岁）

**标签**：建筑工人 / 暴躁型 / 外强中干

**说话风格**：声大，骂骂咧咧。"谁他妈设计的这破地方？""老子不信这些！"

**性格**：表面粗犷，实际胆子不大。愤怒是恐惧的替代品——越害怕越大声。第一个违反规则的人。

**用法**：恐怖场景的催化剂。他的冲动推动剧情，他的遭遇展示规则的残酷。

---

### 苏小棠（女，22岁）

**标签**：大学生 / 恐惧型 / 情绪外放

**说话风格**：声音小，经常带哭腔。"我……我想回家。""那个……那个门在动。"

**性格**：胆小但善良，观察力其实不差——因为害怕所以什么都注意到了。和沈厌的怕死不同，她是真怕，怕到做不了任何事。

**用法**：恐怖氛围的放大器。她的恐惧让读者也跟着害怕。后期有成长空间。

---

### 陈默（男，40岁）

**标签**：退休刑警 / 沉默型 / 隐藏实力

**说话风格**：极少说话，一开口就是关键信息。"别碰那扇门。""规则第七条，我大概知道写的是什么。"

**性格**：经历过真正的犯罪现场，对恐怖场景的阈值极高。表面平静，实际在观察所有人。

**用法**：信息源+悬念制造者。他知道一些其他人不知道的东西，但不说。

## 系统角色

### 深渊观测站（系统）

**人格**：冰冷机械型，偶尔冒出让人脊背发凉的提示

**说话风格**：
- 系统提示用【】格式
- 极度简短，没有感情
- 偶尔用词诡异："恭喜，你的恐惧值达标。深渊在看你。"

**示例提示**：
- 【恐惧值+10】
- 【警告：灵识消耗过度，建议停止观测】
- 【副本规则已解析3/7。剩余规则需要更高的灵识等级】
- 【深渊观测站提醒你：不要回头看】
