# 深渊道主 · 世界观设定

> 类型：恐怖 + 悬疑 + 玄幻 + 搞笑 + 热血战斗 + 异世界

---

## 世界名称：玄渊界

表面是一个标准的修仙世界——宗门林立、灵气充沛、强者为尊。

**真相**：万年前发生过「神陨之战」，旧神全部死亡。祂们的残骸化作了这个世界所谓的"灵气"。修炼者吸纳入体的，本质上是神的尸尘。

---

## 核心矛盾

**修炼 = 服毒。** 每提升一个大境界，就离旧神的残念更近一步。

| 境界 | 恐怖代价                                     |
| ---- | -------------------------------------------- |
| 引气 | 偶尔做噩梦，梦见自己不是自己                 |
| 凝脉 | 开始听到若有若无的哼唱                       |
| 结丹 | 镜子里自己的倒影偶尔会慢半拍                 |
| 元婴 | 出窍时能看到"不该存在的东西"                 |
| 化神 | 分不清幻觉和现实，开始自言自语               |
| 渡劫 | 天劫不是考验，是旧神残念在阻止你成为新的祂们 |
| 真仙 | 飞升后无人归来                               |

## 关键概念

### 裂缝

旧神死亡时撕裂现实的伤口。里面有东西在往外爬。只在夜晚出现，怕光但不怕火。

### 低语

元婴期以上修炼者开始听到的耳语。内容因人而异，但都在说同一件事："回来"。低语越来越清晰，最终让人走向裂缝。

### 异客

从异界来的穿越者。灵魂未被灵气污染，能看见原住民看不见的东西。宗门表面追捧，高层暗中抓捕——因为异客的灵魂是唯一能抵抗低语的东西。

### 镜中人

结丹期以上修炼者照镜子时，倒影偶尔会自己动。这是灵气中残存的神识在"借用"倒影。
