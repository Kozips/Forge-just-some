# 深渊道主 · 角色卡

---

## 陈渡（主角）

| 项目     | 内容                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| 年龄     | 27岁（前世）/ 身体年龄约20岁（穿越后）                                       |
| 前世身份 | 恐怖小说作家，专攻心理恐怖。代表作《三楼厕所》，卖了三千册                   |
| 穿越方式 | 连续熬夜三天猝死，一睁眼到了玄渊界                                           |
| 外貌     | 灰T恤，牛仔裤，左脚一只拖鞋。普通长相，熬夜脸，黑眼圈                        |
| 核心性格 | 怂，但不蠢。怕死，但脑子转得快                                               |
| 隐藏面   | 嘴碎，爱吐槽。最大的本事是在快被吓死的时候还能分析出怪物的行为模式           |
| 说话习惯 | 短句，问题多，偶尔冒出前世写小说的职业病（分析氛围、拆解恐怖源）             |
| 口头禅   | 无固定口头禅，但经常说"不对劲"                                               |
| 金手指   | 真实之眼——能看到灵气背后的真相。不能让他变强，只能让他知道这个世界有多不对劲 |
| 欲望     | 活下去，找到回家的路                                                         |
| 恐惧     | 发现自己回不去了。而且——他好像开始听得懂那些低语在说什么了                   |
| 角色弧光 | 起点：只想活命回家的怂人 → 转折：发现自己是唯一能看到真相的人 → 终点：？？   |

## 苏九

| 项目     | 内容                                                     |
| -------- | -------------------------------------------------------- |
| 年龄     | 15-16岁                                                  |
| 身份     | 苍云宗外门弟子，被逐出前宗门的天才                       |
| 外貌     | 脸很瘦，颧骨高，眼睛不大但亮。灰袍子洗得发白，袖子短一截 |
| 核心性格 | 冷面笑匠。话少，但句句扎心。表面无所谓，实际观察力极强   |
| 隐藏面   | 曾经突破到元婴，然后自己废了自己的修为——她看到了什么？   |
| 说话习惯 | 语气平，结论爆笑。不解释，不废话                         |
| 秘密     | 肩胛骨位置有两个补丁，补丁周围是洗不掉的血迹             |
| 关系     | 第一个发现陈渡的人，把他当"值钱的异客"                   |
