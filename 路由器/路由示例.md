# 路由示例

> 从 `forge/SKILL.md` 迁出。根入口不再内置长示例，需要理解组合加载时读取本文件。

---

## 示例1：克苏鲁 + 恋爱 + 病娇学姐

用户说："写一个克苏鲁背景的恋爱故事，女主是病娇学姐"

**路由加载**：
- 题材：克苏鲁类 + 情感/恋爱相关题材
- 世界观：克苏鲁调查或克苏鲁神话框架
- 结构：恐怖递进 + 恋爱递进
- 人物：学姐（人物类/社交型）
- 性格：病娇（性格类）
- 文风：跟随题材与人物关系
- 对话：恐怖对话 + 情感对话

## 示例2：无限流 + 规则怪谈 + 社恐程序员

用户说："写无限流，主角是社恐程序员，第一个副本是规则怪谈"

**路由加载**：
- 题材：规则怪谈类 / 无限流相关题材
- 世界观：无限流框架
- 结构：副本任务 + 生存规则 + 爽文升级
- 人物：程序员（职业类）
- 性格：社恐（性格类）
- 文风：紧张、清晰、规则驱动
- 对话：游戏 NPC 对话 + 网络梗（按需）

## 示例3：单人物 + 日常对话

用户说："扮演一个傲娇学姐和我聊天"

**路由加载**：
- 人物：学姐（人物类/社交型）
- 性格：傲娇（性格类）
- 对话：情感对话引擎 + 口头禅填充词
- 不加载：世界观、复杂题材、结构文档

## 示例4：龙国都市 + 官场 + 腹黑检察官

用户说："写一个龙国官场小说，主角是腹黑检察官"

**路由加载**：
- 题材：都市类 / 官场相关题材
- 世界观：现代都市
- 文化设定：官场体系 + 法律体系
- 国家拟实：龙国
- 人物：检察官（职业类）
- 性格：腹黑（性格类）
- 对话：官场对话 + 法律职业话语

## 示例5：泡菜国 + 韩娱 + 社恐练习生

用户说："写一个泡菜国韩娱小说，女主是社恐练习生"

**路由加载**：
- 题材：都市类 / 娱乐圈相关题材
- 世界观：现代都市
- 文化设定：娱乐圈体系 + 学校体系（如涉及练习生教育）
- 国家拟实：泡菜国
- 人物：练习生（职业类）+ 学生型（按需）
- 性格：社恐（性格类）
- 对话：韩娱对话 + 敬语体系
