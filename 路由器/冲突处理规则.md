# 冲突处理规则

> 当多个skill产生矛盾时，按以下规则处理。

---

## 冲突类型与处理

### 1. 题材冲突

| 冲突场景 | 处理方式 |
|---------|---------|
| 恐怖 + 搞笑 | 允许，以恐怖为主基调，搞笑作为调剂（参考：《惊声尖笑》） |
| 悬疑 + 恐怖 | 允许，参考"悬疑+恐怖"组合skill |
| 猎奇 + 情感 | 冲突，提示用户选择一个基调 |
| 现代都市 + 修仙 | 提示用户：是否架空？是否"都市异能"？ |

### 2. 人物+性格冲突

| 冲突场景 | 处理方式 |
|---------|---------|
| 社恐 + 话痨 | 冲突，拒绝加载，提示用户选择 |
| 腹黑 + 傻白甜 | 冲突，但可作为"表面傻白甜实际腹黑"的反差设定（需用户确认） |
| 冷血 + 恋爱脑 | 冲突，但可作为"对所有人冷血只对TA恋爱脑"的设定（需用户确认） |
| 独立女性 + 讨好型人格 | 冲突，拒绝加载 |

### 3. 世界观冲突

| 冲突场景 | 处理方式 |
|---------|---------|
| 现实世界 + 超凡体系 | 提示用户：是否"都市异能"？是否"隐藏超凡"？ |
| 龙国 + 魔法体系 | 冲突，提示用户选择架空世界观 |
| 丧尸末日 + 正常校园 | 提示用户：是否"末日前校园"或"丧尸围城校园"？ |

### 4. 国家拟实冲突

| 冲突场景 | 处理方式 |
|---------|---------|
| 龙国背景 + 泡菜国财阀 | 冲突，提示用户：龙国用"大厂/国企"，泡菜国用"财阀" |
| 绅士国 + 龙国法律 | 冲突，以国家拟实为准 |
| 多个国家同时出现 | 允许，分别加载各自的国家拟实 |

---

## 冲突处理流程

```
检测到冲突
  ↓
【1】是否用户手动指定？
  → 是：尊重用户选择，但输出警告
  → 否：进入自动处理
  ↓
【2】是否可以通过"反差设定"解决？
  → 是：提示用户确认（如"表面傻白甜实际腹黑"）
  → 否：进入强制处理
  ↓
【3】按优先级裁剪
  → 保留高优先级skill
  → 降级或移除低优先级skill
  ↓
【4】输出冲突报告
```

---

## 冲突报告格式

```
⚠️ 检测到 2 个冲突：

1. [人物冲突] 社恐 + 话痨 → 已移除"话痨"，保留"社恐"
2. [世界观冲突] 龙国 + 魔法体系 → 提示：是否改为"都市异能"？

如需调整，请告知。
```
