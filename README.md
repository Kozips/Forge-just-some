# Forge-just-some(撒撒水的写作框架)  · AI 辅助中文写作框架

> **一句话**：一套为 Agent 设计的 AI 辅助写作框架，覆盖 13 个题材的人物塑造、写作铁律、对话引擎、素材库、质检清单和设定参考。

## 这是什么

Forge-just-some 让 Agent 在写小说/扮演角色/创作内容时有章可循：
- **并非**通用提示词（几句话说完就忘掉了）
- **也不是**知识库堆砌（几千个文件每个都硬塞进 Token 预算）
- **是**路由+速查+铁律的加载框架——写之前知道读什么，写的时候有规则兜住，写完了有质检

## 能做到什么

| 能力 | 说明 |
|------|------|
| **写小说** | 13 个题材全覆盖：恐怖/悬疑/玄幻/冒险/异世界……每个有 30 条铁律 + 专业知识 + 素材库 + 质检 |
| **扮演角色** | 225 种人物原型 + 对话习惯引擎 + 多角色对话区分清单 |
| **查设定** | 13 国拟实 + 武器/载具速查表 + 文化设定 + 14 种世界观模板 |
| **多题材叠加** | 恐怖+情感、搞笑+悬疑、玄幻+战斗——按优先级组合加载 |
| **长篇维护** | 小说总谱：人物档案/时间线/伏笔追踪/章节规划，AI 自动维护 |
| **去 AI 味** | 禁用词表 + 对话五维标准 + 标杆校准样本 |
| **中文思考** | 中文思考规则 + 专业思考 + 自定义思考 |
| **写作训练** | 选主题→出题→写→逐句纠错→总结，5-15分钟练1个技法（伏笔/对话/递进/视角等） |
| **提示词** | 9类图片提示词模版（动漫肖像/写实人像/海报封面/社交媒体等）+ 双路输入 + 自查清单 |(没怎么蒸，自己拓展吧)
---

## 不能做到什么

| 限制 | 说明 |
|------|------|
| **不涉及真实人物** | 所有角色、场景、事件均为虚构创作素材 |
| **不提供违法操作指南** | 对创作内容有安全边界（核心红线不可覆盖） |
| **不是"一键写小说"** | 本框架是技法辅助工具，不是替代作者判断 |
| **不能保证"好看"** | 质检清单帮助避免错误，但"好不好看"需要人工判断 |

## 怎么用

### 用户视角：告诉 Agent 你要什么

```
"写一个恐怖短篇"                    → Agent 自动加载恐怖类铁律+素材
"扮演一个傲娇女高中生"              → Agent 加载人物原型+对话习惯
"写一个悬疑+搞笑的合租故事"         → Agent 组合加载悬疑类+搞笑类
"查一下神秘东方大国的公安司法体系"          → Agent 加载中国拟实速查
```

### Agent 视角：路由系统

阅读 `SKILL.md` 中的快速路由表，按优先级加载对应文件：

1. 题材 > 世界观 > 文化设定 > 国家拟实 > 人物 > 性格 > 职业 > 文风 > 对话习惯

## 模型适配

| 模型上下文 | 入口文件 | 加载策略 | Token消耗 |
|-----------|---------|---------|----------|
| **≤12k token** | `SKILL.md` | 加载 2-3 个文件，每次只处理一个维度 | ~5k |
| **32K-128K token** | `SKILL.md` | 加载 2-3 个文件，标准创作流程 | ~15k |
| **≥1M token** | `SKILL.md` + 全量子文件 | 可同时加载多个类型 + 设定参考 | ~40k |

> Token消耗为中文实际估算。题材铁律已合并进SKILL.md。
> 写作/类型总大小从4.5MB→1.6MB（-64%），人物/2.1MB，设定参考/892KB。
