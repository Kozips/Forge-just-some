## 描写通用要点

| 类型 | 要点 |
|------|------|
| **速度感** | 推背感、风噪、窗外模糊 |
| **操控感** | 方向盘、刹车、油门 |
| **碰撞** | 金属变形、玻璃碎裂 |
| **坦克** | 履带碾压、炮管后坐 |
| **直升机** | 螺旋桨噪音、悬停、绳降 |
| **潜艇** | 深海静默、声呐信号 |
| **战斗机** | 超音速、G力、导弹锁定 |
