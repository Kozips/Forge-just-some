## 一、中国饮食

### 八大菜系速查
| 菜系 | 特点 | 代表菜 | 描写关键词 |
|------|------|--------|-----------|
| 川菜 | 麻辣鲜香 | 火锅/回锅肉/麻婆豆腐 | 红油/花椒/辣椒/热气腾腾 |
| 粤菜 | 清鲜嫩滑 | 白切鸡/烧鹅/虾饺 | 精致/清淡/鲜美/早茶 |
| 鲁菜 | 咸鲜为主 | 糖醋鲤鱼/九转大肠 | 大气/厚重/酱香 |
| 江浙菜 | 甜糯细腻 | 东坡肉/西湖醋鱼/小笼包 | 甜/糯/精致/小碟 |
| 湘菜 | 香辣酸辣 | 剁椒鱼头/小炒黄牛肉 | 干辣/烟熏/下饭 |
| 闽菜 | 鲜香清淡 | 佛跳墙/沙茶面 | 海鲜/汤/鲜 |
| 徽菜 | 重油重色 | 臭鳜鱼/毛豆腐 | 发酵/腌制/山珍 |
| 滇菜 | 酸辣鲜香 | 过桥米线/汽锅鸡 | 酸/野味/菌类 |

### 日常饮食场景
- **早餐摊**：油条/豆浆/包子/煎饼/茶叶蛋、蒸汽升腾、塑料凳
- **火锅店**：鸳鸯锅/九宫格、涮肉/毛肚/鸭肠、啤酒碰杯、辣到冒汗
- **烧烤摊**：炭火/孜然/辣椒面、啤酒瓶/塑料杯、深夜/路边/烟火气
- **外卖**：塑料袋/一次性筷子/好评卡、独居/将就/孤独

## 六、中国街头小吃速查

> 街头小吃 = 烟火气 + 城市记忆 + 角色日常。写校园/都市/底层场景时高频出现。

### 北方小吃

| 小吃 | 描写要点 | 场景 |
|------|---------|------|
| 煎饼果子 | 铁板滋滋声、鸡蛋摊开、薄脆咔嚓碎、酱料刷上去 | 早餐摊/学校门口/排队 |
| 驴肉火烧 | 酥脆掉渣、驴肉切碎塞进去、一口咬下去油顺着手指流 | 河北/北方街头 |
| 肉夹馍 | 白吉馍烤得鼓起来、腊汁肉剁碎塞进去、油纸包着 | 西安/北方街头 |
| 烤红薯 | 焦糖香味飘很远、掰开金黄色冒热气、烫手 | 冬天/推车小贩 |
| 糖葫芦 | 红色山楂串、糖衣脆亮、咬一口嘎嘣脆 | 冬天/庙会/学校门口 |
| 煎饼卷大葱 | 煎饼摊开、大葱抹酱卷起来、一口下去辣眼睛 | 山东/北方早餐 |

### 南方小吃

| 小吃 | 描写要点 | 场景 |
|------|---------|------|
| 叉烧包 | 白色包子皮微微裂开、露出里面红亮的叉烧馅、甜咸交织 | 早茶/广东/茶楼 |
| 肠粉 | 白色粉皮裹着虾仁/猪肉、淋上酱油、滑嫩 | 早餐/广东街头 |
| 螺蛳粉 | 酸笋的臭味飘很远、粉丝滑溜、花生脆/酸豆角 | 深夜/广西/宿舍 |
| 烧麦 | 皮薄馅大、糯米/猪肉、蘸醋吃 | 早茶/早餐摊 |
| 龟苓膏 | 黑色果冻状、微苦、加蜂蜜或炼乳 | 夏天/广东/甜品店 |

### 全国通用小吃

| 小吃 | 描写要点 | 场景 |
|------|---------|------|
| 地瓜（烤红薯） | 推车上铁桶烤的、焦香、掰开冒热气、甜糯 | 冬天/街头/情侣 |
| 奶茶 | 塑料杯/纸杯、珍珠/椰果/芋圆、吸管戳进去、甜 | 校园/逛街/等人 |
| 炸鸡 | 外酥里嫩、油纸袋装着、手指油亮、可乐配 | 深夜/追剧/宿舍 |
| 臭豆腐 | 黑色方块、外酥里嫩、臭味飘很远、辣椒酱 | 夜市/长沙/路边摊 |
| 烤冷面 | 铁板上摊开、鸡蛋打上去、刷酱、切块装盒 | 学校门口/夜市 |
| 凉皮 | 白色粉皮切条、黄瓜丝/面筋/辣椒油、酸辣爽口 | 夏天/北方/快餐 |
| 鸡蛋灌饼 | 饼皮鼓起来灌鸡蛋、刷酱加生菜/火腿 | 早餐摊/学校门口 |
| 关东煮 | 竹签串着丸子/豆腐/萝卜、热汤煮着、便利店 | 冬天/便利店/深夜 |

### 写作场景速查

| 场景 | 用什么小吃 | 写法要点 |
|------|-----------|---------|
| 校园日常 | 煎饼/奶茶/烤红薯 | 学校门口的摊位、排队、AA制 |
| 约会逛街 | 奶茶/糖葫芦/炸鸡 | 两个人分着吃、喂对方、手指碰手指 |
| 深夜独处 | 螺蛳粉/泡面/外卖 | 宿舍/出租屋、一个人吃、孤独感 |
| 冬天取暖 | 烤红薯/关东煮/糖葫芦 | 推车小贩、热气、烫手、暖胃 |
| 夜市烟火 | 臭豆腐/烤冷面/炸鸡 | 人挤人、叫卖声、油烟味、热闹 |
| 异乡思乡 | 家乡小吃的回忆 | "好久没吃过XX了"、妈妈做的味道 |

| 场景 | 承载的情感 | 写法 |
|------|-----------|------|
| 一个人吃饭 | 孤独/自由/将就 | 写食物的单调、环境的安静、手机陪伴 |
| 两人对坐 | 试探/亲密/冷战 | 写谁先动筷、筷子碰碗的声音、沉默 |
| 家庭聚餐 | 温暖/矛盾/责任 | 写谁做饭、座位安排、话题回避 |
| 饭局应酬 | 权力/虚伪/疲惫 | 写敬酒顺序、座位等级、客套话 |
| 深夜独食 | 治愈/孤独/自我 | 写食物的温度、厨房的灯光、碗筷的声音 |
