# B站（bilibili）

> 中国最独特的年轻人文化社区，从二次元弹幕网站成长为泛内容生态平台。月活超3亿，Z世代浓度最高的平台，弹幕文化的精神故乡。

---

## 一、平台定位（写作用途）

- **年轻人的文化客厅**：二次元起家，现在覆盖知识、生活、游戏、音乐、影视全品类
- **用户画像**：核心用户18-35岁，一二线城市为主，大学生和年轻白领密度极高；"人均985"的自嘲背后是真实的学习/知识需求
- **气质标签**：二次元底色、弹幕社区、知识氛围、"用爱发电"、略带精英主义的亚文化认同
- **写作适用**：弹幕互动、UP主创作生态、追番文化、评论区神回复、知识区科普人设、鬼畜恶搞、拜年纪/百大UP主颁奖、年轻人的"精神角落"

---

## 二、核心功能

### 2.1 弹幕系统（B站灵魂）

弹幕是B站最核心的交互方式，也是区别于所有其他平台的根本特征。

| 弹幕类型 | 说明 | 写作用途 |
|----------|------|---------|
| **顶部/底部弹幕** | 固定位置的大字弹幕 | 重要时刻的集体刷屏 |
| **滚动弹幕** | 从右向左飘过的普通弹幕 | 实时"气氛组" |
| **彩色弹幕** | 可自定义颜色 | 标识特殊发言 |
| **高级弹幕** | 可调字号、位置、特效 | 付费/等级特权 |
| **弹幕表情** | 图片弹幕 | 视觉冲击 |

弹幕写作要点：
- **空耳**：把外语歌词听成中文谐音（如"高达"听成"肛大"），集体空耳是经典弹幕景观
- **前方高能**：提醒后面的观众有精彩/吓人内容
- **弹幕护体**：恐怖/尴尬场面时弹幕铺满屏幕"保护"观众
- **名场面刷屏**：经典片段时弹幕密度暴增，形成"弹幕墙"
- **弹幕礼仪**：不剧透、不刷屏、不引战——B站社区公约的核心

### 2.2 视频内容分区

| 分区 | 典型内容 | 写作场景 |
|------|---------|---------|
| **番剧/动画** | 日漫、国漫、新番连载 | 追番日常、弹幕狂欢、追番"等更新"的焦虑 |
| **鬼畜** | 恶搞剪辑、音MAD、鬼畜全明星 | 某角色/台词被鬼畜，"被迫成名" |
| **知识区** | 科普、人文社科、法律、医学 | 配角是知识区UP主，正经科普人设 |
| **生活区** | 日常vlog、美食、宠物、家居 | 主角是生活区UP主，记录日常 |
| **游戏区** | 游戏实况、攻略、电竞 | 游戏主播/电竞选手人设 |
| **音乐区** | 翻唱、原创、乐器演奏 | 音乐人UP主，从B站出圈 |
| **影视区** | 影评、解说、混剪 | 影视博主/二创作者 |
| **舞蹈区** | 跳舞、宅舞、编舞 | 舞见（跳舞UP主的称呼） |
| **科技区** | 数码评测、编程教程 | 科技博主/程序员人设 |

### 2.3 UP主生态

- **UP主**：B站创作者的统称（对应抖音的"达人"、YouTube的"YouTuber"）
- **投稿**：发布视频（不说"发视频"，说"投稿"）
- **三连**：点赞+投币+收藏，是支持UP主的最高礼遇
- **一键三连**：长按点赞按钮同时完成三个动作
- **下次一定**：观众说"下次一定三连"，实际意思是永远不会三连——经典敷衍梗
- **充电**：直接给UP主打钱（类似抖音的打赏），按月充电支持
- **恰饭**：UP主接广告/商单（"恰饭视频"=广告视频，观众普遍理解但会调侃）

### 2.4 会员体系

| 等级 | 权限 | 写作细节 |
|------|------|---------|
| **普通用户** | 看视频、发评论（部分视频受限） | 需要答题才能注册——B站著名的"入站考试" |
| **正式会员** | 解锁全部内容、发高级弹幕 | 通过答题考试获得，是B站"门槛"的象征 |
| **大会员** | 看番剧抢先、画质升级、专属装扮 | 月费/年费制，追番刚需 |
| **年度大会员** | 年费大会员+更多特权 | 身份标识，可在评论区"亮牌子" |

### 2.5 直播

- **直播区**：游戏直播为主，也有娱乐/学习直播
- **舰长/提督/总督**：B站直播的付费订阅等级（舰长月费约138元，提督约1998元，总督约19998元）
- **上舰**：购买舰长订阅
- **SC（Super Chat）**：付费醒目留言，在直播间高亮显示
- **小电视飞船/摩天大楼**：B站直播的高价礼物

## 四、写作场景模板（可直接嵌入剧情）

### 场景1：追番日常
> 新番更新了。她准时打开B站，大会员的标志亮在头像旁边。画面刚加载出来，弹幕就已经密密麻麻——"来了来了""前排""名场面预警"。她习惯性地关掉弹幕看一遍剧情，再打开弹幕看一遍弹幕，像两个平行宇宙。

### 场景2：被鬼畜了
> 他打开B站首页，推送第一条视频标题写着"当XXX说出那句话的时候"，播放量三百万。点进去——是他上周在年会上的演讲片段，被剪成了鬼畜，配上了电音和各种魔性表情包。评论区第一条："建议入选鬼畜全明星。"点赞两万。他哭笑不得。

### 场景3：UP主恰饭
> "这期是恰饭视频，大家理性观看。"她在视频开头坦诚地说，然后花了五分钟认真测评了那款护肤品。弹幕飘过一条"恰饭还这么真诚，爱了"，后面跟着一排"下次一定三连"。她知道，大多数人说"下次一定"的时候，意思就是永远不会。

### 场景4：入站考试
> "你要注册B站？先答题。"他把手机递过来，屏幕上是二十道题——有关于弹幕礼仪的，有考二次元常识的，还有几道完全看不懂的梗题。她答了十五分钟，73分，刚好过线。"恭喜你，正式成为B站用户。"他举着手机，像给她颁发了一个勋章。

### 场景5：拜年纪跨年
> 除夕夜，全家在看春晚，她戴着耳机看B站拜年纪。节目单上有她追了一整年的UP主合唱，有让她笑了整年的鬼畜混剪，还有弹幕里集体刷"干杯"的那一刻。零点钟声响起时，弹幕铺满了整个屏幕——"2025，bilibili干杯~"。她打了三个字："新年好。"淹没在百万条弹幕里。

### 场景6：充电风波
> 他给她发了一张截图：B站充电记录，连续充电十二个月，每个月138元。她愣住了："你给我充了一年电？""嗯，你的每期视频我都看了，觉得值。"她打开后台，充电榜上他的名字排在第一个，旁边写着"连续充电365天"。

## 五、写作避坑

1. **弹幕不是评论**——弹幕是实时飘过视频画面上的文字，评论在视频下方，两者完全不同。写"他在弹幕区写了条评论"是错误的
2. **B站用户有"入站考试"门槛**——这造成了独特的社区认同感和排外性。写一个角色"随手下个B站就能用"不够真实，除非是看视频不需要登录
3. **恰饭≠翻车**——B站观众对恰饭视频的态度是"理解但会调侃"，不是一看到广告就取关。真诚的恰饭视频反而能涨粉
4. **弹幕描写要克制**——不要把每个画面都写满弹幕，只在关键情节用弹幕推动气氛。弹幕太多反而会淹没剧情
5. **B站和抖音的用户画像重叠但不相同**——B站用户更偏"内容消费"而非"杀时间"，知识区、番剧区的存在感很强，不是纯粹的娱乐消遣平台
