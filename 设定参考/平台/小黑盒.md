# 小黑盒

> 国内最垂直的游戏/数码社区，Steam玩家的核心聚集地。用户量不大但黏性极高，社区氛围偏硬核、偏宅、偏理性。是"游戏宅"和"数码发烧友"的精神家园。

---

## 一、平台定位（写作用途）

- **游戏+数码垂直社区**：以Steam游戏为核心的讨论、评测、推荐、折扣信息平台
- **用户画像**：以18-30岁男性为主，核心用户是PC/主机游戏玩家，有一定消费能力的数码爱好者，大学生和年轻上班族居多
- **气质标签**：硬核、宅、理性消费、略带优越感、幽默自嘲、"盒友"互称、小众但忠诚
- **写作适用**：游戏玩家日常、Steam促销剁手、游戏评测争论、开箱晒货、数码产品讨论、游戏搭子/队友、游戏相关的人际关系

---

## 二、核心功能

### 2.1 游戏评测与推荐

| 功能 | 说明 | 写作场景 |
|------|------|---------|
| **游戏评测** | 用户可对游戏打分、写长评，类似Steam评测但更详细 | 主角写了一篇爆款评测被推上首页 |
| **游戏推荐** | 根据你的游戏库推荐新游戏 | "根据你玩了3000小时的XX，推荐你试试YY" |
| **游戏数据库** | 详尽的游戏信息、配置需求、发售日期、价格走势 | 查配置、查价格、查历史最低价 |
| **心愿单** | 标记想买的游戏，降价时推送通知 | 收藏了一堆游戏等打折 |
| **游戏评分** | 社区用户综合评分，分为推荐/一般/不推荐 | 评分大战——"这游戏8分太高了吧" |

### 2.2 资讯与折扣

| 功能 | 说明 | 写作场景 |
|------|------|---------|
| **游戏资讯** | 新游发布、版本更新、行业新闻 | 第一时间讨论新游戏消息 |
| **折扣信息** | Steam/EGS等平台的促销信息，历史低价提醒 | Steam夏促/冬促期间的"剁手"剧情 |
| **免费游戏** | 限时免费领取的提醒（如Epic每周免费游戏） | "快去领！白嫖了！" |
| **比价功能** | 不同平台/区服的价格对比 | 转区买游戏、跨区薅羊毛 |

### 2.3 社区动态

- **动态/圈子**：类似朋友圈的发帖功能，分享游戏截图、开箱照片、日常吐槽
- **评论/点赞/收藏**：标准互动方式
- **关注/粉丝**：用户关系链
- **热门/推荐**：算法推荐的热门内容
- **话题标签**：按游戏/话题分类讨论

### 2.4 开箱与数码

- **开箱晒单**：游戏主机、显卡、外设、游戏周边的开箱分享
- **数码讨论**：键盘、鼠标、显示器、手机等数码产品讨论
- **装机配置**：DIY电脑配置推荐和讨论
- **外设推荐**：键鼠耳机显示器的评测推荐

### 2.5 游戏工具

- **游戏时长统计**：连接Steam等平台后展示游戏时长
- **成就展示**：游戏成就完成度
- **游戏库管理**：展示你拥有和玩过的所有游戏
- **游戏日历**：新游发售日历，一目了然

## 四、写作场景模板（可直接嵌入剧情）

### 场景1：Steam大促剁手
> 夏促第三天，他的Steam购物车里已经堆了十七个游戏。盒友群里有人发消息："GTA5又打折了，史低。"他看了看库里那个吃灰两年的GTA5，默默关掉了购物车页面。五分钟后，他又打开了——"再买最后一个。"那天晚上他花了四百多，盒友问他买了啥，他说"全是骨折价，不买亏一个亿。"

### 场景2：写游戏评测
> 她花了整整一个周末打通了那个独立游戏，然后在小黑盒上写了一篇两千字的评测。从叙事结构分析到配乐解读，最后给了8.5分。第二天醒来，评测被推上了首页推荐，评论区炸了——"8.5？你在逗我？""这才是真正的评测！""盒友文学最高水平。"她盯着不断跳动的点赞数，第一次觉得自己玩游戏这件事，好像也有点意义。

### 场景3：开箱显卡
> 快递到的那天他录了一段开箱视频，刀片划开胶带的时候手都在抖。撕开防静电袋，那块4090在客厅的灯光下闪着金属光泽。他对着镜头说："盒友们，等了半年，终于等到了。"评论区第一条评论是——"大哥你电源带得动吗？"第二条是——"接下来看你打什么游戏——原神。"

### 场景4：游戏打折通知
> 凌晨一点，小黑盒推送了一条消息："你心愿单中的《XX》已降至历史最低价29元。"她看了一眼余额，犹豫了三秒，下单了。打开盒友动态，发现十分钟前已经有人发帖："快冲！史低！"回复里全是"已买""谢谢提醒""钱包在哭"。她截了图发到动态——"盒友们，又喜加一了。"

### 场景5：游戏争论大战
> 一篇标题为"这游戏到底值不值328"的帖子在盒子里挂了两天，双方吵了八百楼。支持方拿着画质截图、剧情分析、配乐鉴赏一条条摆事实；反对方翻出了发售当天的bug集锦、制作人的争议言论、以及"还不如XX"的对比视频。楼主在第五百楼之后就消失了，留下两拨人继续打。最后一条回复是——"盒友们别吵了，这游戏已经褒姒了。"

### 场景6：装机求助
> "盒友们帮我看看这配置行不行"——他把一张配置单截图发到了小黑盒，CPU是12400F，显卡是4060Ti，预算七千。评论区瞬间分成了三派：实用派推荐降级省钱，性能派建议加一千上4070，还有一派在争论AMD还是Intel。最后有人贴了一张"等等党永远不亏"的表情包，楼主回复："我先买了，等不了了。"

## 五、写作避坑

1. **不要把小黑盒写成大众平台**——小黑盒用户量远不如抖音/微博，写的时候角色应该表现出"小众圈子"的自觉。"我发在盒子里了"和"我发在抖音上"完全不是一回事
2. **盒友的语气偏自嘲+理性**——不像贴吧那么毒舌，也不像B站那么中二。盒友说话喜欢讲道理、摆数据、贴截图，吵架也是"你这数据不对"而不是直接人身攻击
3. **Steam生态是核心语境**——G胖、促销、夏促冬促、愿望单、游戏时长、背刺、喜加一，这些都是盒友的日常用语。如果角色在小黑盒上活动，这些元素应该自然出现
4. **数码/外设是第二重头戏**——键盘、显卡、显示器的讨论热度仅次于游戏。"等等党""溢价""生产力"这些梗要用得自然，别堆砌
5. **电子阳痿是真实痛点**——"库里的游戏一个不想打"是盒友最常见的自嘲，写游戏玩家日常时这个细节非常加分
