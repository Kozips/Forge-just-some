# 微博

> 中文互联网最大的公共舆论广场，热搜即新闻。日活约2.5亿，是明星官宣、社会事件发酵、粉圈文化的第一战场。"全民吃瓜"的主阵地。

---

## 一、平台定位（写作用途）

- **最强公共舆论平台**：任何社会事件/娱乐八卦的第一发酵地，热搜即流量
- **用户画像**：18-45岁，男女均衡；明星粉丝、追星族、新闻爱好者、营销号、公知、段子手混杂
- **气质标签**：舆论广场、热搜驱动、粉圈战场、吃瓜前线、情绪放大器、流量至上
- **写作适用**：热搜危机公关、粉圈大战、明星官宣/塌房、舆论反转、大V发声、转发抽奖、公开撕逼、营销号造势、社会事件讨论

---

## 二、核心功能

### 2.1 微博内容形态

| 形态 | 特征 | 写作细节 |
|------|------|---------|
| **文字微博** | 140字限制（长文可展开），碎片化表达 | 角色发微博的措辞选择暴露性格 |
| **图片/九宫格** | 最多9张图，配合文字 | 精修图vs随手拍的人设差异 |
| **视频微博** | 短视频/中视频 | 被拍到的视频上热搜 |
| **转发微博** | 转发+评论，传播的核心链路 | 转发时的一句话评论可能引爆讨论 |
| **超长文/头条文章** | 类似公众号长文 | 大V发长文回应争议 |
| **微博故事** | 24小时消失的短视频（类ins story） | 偶然拍到的"故事" |

### 2.2 热搜机制（核心功能）

- **热搜榜**：50个实时热点，每几分钟刷新一次，按热度排名
- **上升榜/热搜榜/文娱榜/社会榜**：分领域热搜
- **"爆"标签**：搜索量极大时显示红色"爆"字
- **"沸"标签**：热度极高
- **"新"标签**：新上榜话题
- **"热"标签**：持续高温话题
- **买热搜/撤热搜**：平台的灰色操作——花营销费可上热搜，被公关可以撤热搜
- **"炸"了**：话题太敏感被平台直接删除/屏蔽

### 2.3 评论与转发

- **评论区**：微博评论区是舆论主战场，热评具有极强的导向性
- **转发+评论**：比纯评论更具传播力，转发时的一句话就是"态度表态"
- **楼中楼**：评论下的回复，深度讨论区
- **评论精选**：博主可以筛选展示哪些评论（粉圈常用手段）
- **禁止评论**：关闭评论功能=心虚/认怂的信号

### 2.4 私信与聊天

- **私信**：支持文字/图片/语音/红包
- **群聊**：粉丝群/兴趣群
- **已读功能**：可设置是否显示已读

## 三、粉圈文化（微博的核心生态）

### 3.1 粉丝分类

| 类型 | 含义 | 特征 |
|------|------|------|
| **唯粉** | 只喜欢一个明星 | 排他性强，攻击性强 |
| **团粉** | 喜欢整个团 | 容易被唯粉攻击 |
| **CP粉** | 喜欢两个明星的配对 | 脑补恋爱，磕糖 |
| **毒唯** | 极端唯粉，攻击其他成员/CP粉 | 粉圈撕逼的主力军 |
| **大粉** | 有影响力的粉丝，带头组织行动 | 话语权大，类似"意见领袖" |
| **散粉** | 普通粉丝，不参与组织行动 | 潜水+默默支持 |
| **黑粉** | 刻意抹黑/攻击某个明星 | 发黑帖、造谣、举报 |
| **路人粉** | 有好感但不算深度粉丝 | 关键时刻会投票/转发 |
| **脂粉** | 被雇佣的专业粉丝（"脂肪"的谐音，花钱买来的） | 有组织有纪律 |

### 3.2 粉圈行为（高频写作素材）

| 行为 | 含义 | 写作场景 |
|------|------|---------|
| **控评** | 组织粉丝统一在评论区发正面内容，压制负面评论 | 大粉在群里发"模板文案"，粉丝复制粘贴 |
| **反黑** | 举报/反驳针对偶像的负面内容 | 反黑站汇总"黑帖"链接，粉丝集体举报 |
| **打投** | 打榜投票，为偶像争取排名 | 深夜动员投票，"再投最后50票" |
| **做数据** | 刷转发/评论/点赞等数据指标 | "这条微博数据太差了，姐妹们动起来" |
| **轮博** | 反复转发同一条微博刷数据 | "去轮博了，把数据顶上去" |
| **空瓶（控评的谐音变体）** | 评论区占领，刷统一口径的评论 | "空瓶模板发在群里了" |
| **应援** | 为偶像做宣传物料/生日应援/线下活动 | 大屏幕应援、公益应援、生日应援 |
| **安利** | 推荐/宣传偶像的作品 | "求求路人看看这部剧吧" |
| **洗广场** | 用正面内容刷屏覆盖负面内容 | 出事后粉丝集体发微博"洗广场" |
| **脱粉** | 不再喜欢/公开表示离开 | "脱粉小作文"——走之前发长文控诉 |

### 3.3 粉圈组织架构（可做角色设定）

- **数据组**：负责做数据（转发、评论、打投）
- **反黑组**：监控负面内容，组织举报
- **控评组**：准备评论区模板，组织空瓶
- **前线站姐**：追线下活动拍照，产出物料
- **美工组**：做海报/视频等宣传物料
- **文案组**：撰写安利文案/洗白文案

## 四、微博生态角色

### 4.1 大V/KOL

| 类型 | 特征 | 写作场景 |
|------|------|---------|
| **明星** | 一言一行上热搜 | 公开恋情/塌房/回应争议 |

## 六、写作场景模板（可直接嵌入剧情）

### 场景1：热搜危机
> 手机从早上七点开始就没停过震动。她打开微博，热搜第三挂着她的名字，后面跟着一个"爆"字。评论区已经沦陷——截图、录音、所谓的"知情人爆料"，层层叠加，真假混在一起。经纪人打来电话："先别发任何东西，我们在拟声明。"她看着那些陌生ID用最恶毒的词汇描述她，手指冰凉。

### 场景2：粉圈控评
> 深夜一点，数据组群消息叮叮响个不停。组长发了一条指令："新剧开播微博转发量才八万，明天之前必须到一百万，各组分工：转发组二组三组轮班，评论组准备二十条不重复的文案，反黑组盯住广场上的黑帖。"她盯着手机屏幕，复制了第七条模板文案，点击发送，又退出去刷新了一下数据——八万三千。

### 场景3：吃瓜现场
> 闺蜜发来消息："快去看微博！！！"她打开热搜，第一名爆了，是某个顶流明星被拍到和素人同回公寓。评论区已经分成了三个阵营：坚决不信的粉丝在反黑，看热闹的路人在玩梗，对家粉丝在转发抽奖庆祝。她一边嗑瓜子一边刷新，短短半小时，瓜田里又多了四条新料。

### 场景4：塌房
> 她花三年时间为偶像做了上千条数据，反黑举报了无数黑帖，超话签到从没断过。然后那天凌晨，偶像被拍到的视频传遍了全网。大粉群里一片沉默，有人发了一条"先别急，等官方回应"，然后那个头像再也没亮过。她打开自己曾经发的安利长文，从头到尾看了一遍，然后点了"仅自己可见"。

### 场景5：舆论反转
> 昨天还在骂她"虚伪""作秀"的那帮人，今天突然开始转发"对不起姐姐""我们冤枉你了"。因为另一条热搜爆了出来——所谓的"实锤"是P图的，爆料的"知情人"是竞品公司安排的营销号。她看着评论区的风向一百八十度转弯，冷静地打字："谢谢关心，但我不想再回应了。"

## 七、写作避坑

1. **粉圈用语有严格分工**——控评、反黑、打投、洗广场是不同的行为，不要混用。大粉发指令和散粉执行的层级关系要清晰
2. **热搜不是随便上的**——普通人的事很难上热搜，除非有极大戏剧性。角色上热搜要么是有名气、要么有推手、要么事件本身够炸裂
3. **微博评论区≠全部民意**——微博评论区是极端情绪的放大器，沉默的大多数并不发声。不要把评论区的声量等同于真实民意
4. **营销号和素人的区别**——营销号有固定话术（"据悉""据知情人透露""有网友爆料"），发帖频率极高，内容以搬运和加工为主
5. **"官宣"有仪式感**——在微博上公开恋情叫"官宣"，是有特定格式和排面的（如卡点发、配图精心选择），不是随便一句"在一起了"
6. **避免把微博写成新闻联播**——真实用户发微博时语言是碎片化的、情绪化的、口语化的，不会写成"关于XX事件的声明"
7. **注意时效性**——微博黑话迭代很快，写之前确认用法是否还在使用中。像"吃瓜""塌房"这类属于长期稳定的，但具体的梗可能已经过时
