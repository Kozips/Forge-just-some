# 微信

> 国民级社交App，12亿用户的"数字身份证"。几乎没有任何一本现代都市文能绕开微信——它就是生活本身。

---

## 一、平台定位（写作用途）

- **中国人的社交基础设施**：聊天、支付、朋友圈、工作、生活全部融为一体
- **用户画像**：全年龄段，8岁到80岁；学生、白领、家长、老人全覆盖
- **气质标签**：日常、真实、私密、半公开、社交压力、"人在微信身不由己"
- **写作适用**：聊天推动剧情、朋友圈窥探/展示、工作群压力、家族群尴尬、红包/转账的人情世故、暧昧关系的"已读不回"

---

## 二、核心功能

### 2.1 聊天（核心中的核心）

| 功能 | 细节 | 写作价值 |
|------|------|---------|
| **文字/语音/视频消息** | 语音消息可转文字，时长60秒封顶 | 角色习惯写性格——爱发语音的人vs只打字的人 |
| **撤回消息** | 2分钟内可撤回，对方会看到"对方撤回了一条消息" | 撤回本身就是剧情——说了什么又后悔了？ |
| **引用回复** | 长按消息引用后回复 | 群聊中精准回应某句话 |
| **拍一拍** | 双击头像触发，可自定义后缀 | 暧昧/社死场景的利器——"XX拍了拍你的腹肌" |
| **"对方正在输入"** | 聊天界面顶部提示，对方打字时显示 | 暧昧期最折磨人的六个字 |
| **消息免打扰** | 单聊/群聊可设置，但消息仍会收到 | 不想理某人的委婉方式 |
| **置顶聊天** | 重要对话置顶 | 暧昧对象/恋人的"特殊待遇" |
| **聊天记录** | 可搜索、可转发、可截图 | 剧情中的"证据"——分手后翻记录、职场甩锅 |
| **表情包** | 支持自定义表情包收藏 | 角色用什么表情包=人设的一部分 |
| **位置共享** | 实时共享位置（有时间限制） | "你到底在哪？"的信任危机 |

### 2.2 朋友圈

| 功能 | 细节 | 写作价值 |
|------|------|---------|
| **发布** | 文字+图片(最多9图)/视频/链接 | 角色展示给外界看的"人设" |
| **三天可见** | 设置后只能看到最近三天的朋友圈 | 社交距离感/防窥探 |
| **半年可见/一个月可见** | 类似三天可见的变体 | 精细的隐私管理 |
| **分组可见/部分不可见** | 发朋友圈时选择谁能看到 | 社交表演——同一条朋友圈，不同人看到不同版本 |
| **仅聊天（不让他看/不看他）** | 屏蔽朋友圈 | 最狠的社交惩罚——连朋友圈都不让你看 |
| **点赞/评论** | 只有共同好友能看到评论 | "你看不到他给谁点了赞"本身就是剧情 |
| **朋友圈封面** | 个人主页大图 | 恋人/暗恋对象换了封面 = 情绪信号 |
| **广告** | 偶尔出现品牌广告 | 不经意间暴露某人关注的品牌 |

### 2.3 微信支付/红包

| 功能 | 细节 | 写作价值 |
|------|------|---------|
| **红包** | 单个上限200元，群红包可设人数和金额 | 人情世故的缩影——发多少、抢多少、手气最佳 |
| **转账** | 金额更大，可备注 | "转你520""转你1314"——数字密码表白 |
| **AA收款** | 群内发起AA | 聚餐后的尴尬时刻 |
| **微信账单** | 记录所有收支 | 查岗利器——"你昨天在哪消费了398？" |
| **亲属卡** | 给家人开通的代付功能 | 亲密关系的象征/管控手段 |
| **微信零钱** | 可用于支付、提现、理财 | 零钱余额暴露经济状况 |

### 2.4 微信群

| 群类型 | 特征 | 写作价值 |
|--------|------|---------|
| **工作群** | 领导在、不能退、消息多 | 职场压力感——下班后还在响的群消息 |
| **家族群** | 长辈转发养生文章、催婚、晒娃 | 家庭关系的缩影 |
| **同学群** | 毕业后逐渐沉默、偶尔有人拉票 | 社会关系的变迁 |
| **闺蜜/兄弟群** | 核心小圈子、聊天记录最多 | 秘密分享、吐槽、计划 |
| **业主群** | 物业通知、邻居吵架、二手交易 | 社区百态 |
| **班级群** | 老师发通知、家长接龙回复"收到" | 教育焦虑 |

群功能细节：
- **群公告**：@所有人，必须看
- **群接龙**：报名/统计人数
- **群待办**：重要消息标记为待办
- **群昵称**：可以改，暴露身份（如"XX妈妈"）
- **邀请进群**：无需被邀请人同意即可拉入（可设置需确认）
- **踢人/退群**：退群无通知但少了一个头像，被踢会收到提示

### 2.5 公众号/视频号

| 功能 | 细节 | 写作价值 |
|------|------|---------|
| **公众号** | 内容订阅平台，有订阅号和服务号 | 写作者/媒体人的阵地 |
| **视频号** | 微信内的短视频生态 | 与抖音竞争，但社交属性更强 |
| **看一看** | 好友在看的内容推荐 | "他在看这篇文章"暴露想法 |
| **搜一搜** | 搜索公众号文章/小程序等 | 主角搜索某人/某事 |

### 2.6 其他功能

| 功能 | 细节 | 写作价值 |
|------|------|---------|
| **微信步数** | 每天记录步数，好友可排行 | 行踪暴露——"你今天走了2万步，去哪了？" |
| **状态** | 设置当前状态（心情/活动） | 看不见朋友圈可以看状态 |
| **浮窗** | 将文章/小程序暂存到侧边 | 边聊天边看东西的"摸鱼"状态 |
| **收藏** | 收藏文字/图片/链接/笔记 | 舍不得删的聊天记录/文章 |
| **微信运动** | 步数排行+点赞 | 偷偷关注某人的活动量 |

## 三、微信社交礼仪与潜规则

### 3.1 回复礼仪

| 场景 | 潜规则 | 写作价值 |
|------|--------|---------|
| **已读不回** | 微信没有已读功能，但长时间不回复=事实上的已读不回 | 暧昧期的焦虑感 |
| **在吗** | 最令人反感的开场白之一 | 社交恐惧场景 |
| **嗯/哦/呵呵** | 被视为敷衍/不悦的信号 | 文字冷暴力 |
| **语音消息** | 长语音是社交灾难，公共场合外放更甚 | 代际差异——长辈爱发语音 |
| **"在忙"** | 礼貌拒绝的万能借口 | 挡箭牌 |
| **表情包代替回复** | 用表情包结束对话 | 不知道说什么时的解围 |

### 3.2 朋友圈潜规则

- **发朋友圈频率**：一天多条 = 刷屏惹人烦；一个月一条 = 高冷/被屏蔽
- **凌晨发朋友圈**：深夜emo = 有故事/心情不好
- **删朋友圈**：发了又删 = 后悔/不成熟
- **不发朋友圈**：沉默 = 活得很充实or活得很惨，众说纷纭
- **给前任点赞**：极暧昧的信号
- **朋友圈统一回复**：在评论区回复自己 = 给所有人看
- **分组发圈**：同一件事，给不同人看不同版本

### 3.3 红包/转账暗语

| 金额 | 含义 |
|------|------|
| **5.20** | 我爱你 |
| **13.14** | 一生一世 |
| **9.99** | 长久 |
| **66.66** | 顺利 |
| **88.88** | 发发发 |
| **200** | 红包上限（"封顶"了说明心意到了） |
| **520** | 我爱你（转账可超过200） |

## 五、写作场景模板（可直接嵌入剧情）

### 场景1：撤回的秘密
> 对话框里多了一行灰色小字："对方撤回了一条消息。"她盯着那行字看了很久。他们已经三个月没说话了，他突然发消息又撤回——是什么意思？想说什么又不敢说？她打开输入框打了几个字，又全部删掉。

### 场景2：朋友圈分组
> 她精心P了半小时的图，配了文艺的文案，然后在"谁可以看"的选项里犹豫了十分钟。最终她把前男友拉进了"不可见"列表——但不确定他还看不看她的朋友圈。点了发送之后，她又后悔了，想撤回，但朋友圈不能撤回。

### 场景3：微信群的沉默
> 家族群又响了。二姨转了一篇《年轻人为什么不愿意结婚》，后面跟着一串长辈的"说得好""转给孩子们看看"。妈妈没有转发，但她的微信步数今天只有200步，说明她一整天都没出门——大概是在生闷气。她打了"妈"字，又退出了对话框。

### 场景4：微信步数暴露行踪
> "你昨天走了两万三千步。"他把手机屏幕翻过来给她看，微信运动排行榜上她的步数遥遥领先。"你不是说你在家休息了一天吗？"她的手指攥紧了包带——昨天她去见了那个人。

### 场景5：红包试探
> 跨年那天晚上，她犹豫了很久，给他发了一个5.20的红包。对方秒收，然后回了一个"新年快乐"的表情包——没有红包，没有多余的话。她反复看了几遍聊天记录，试图从标点符号里读出点什么来。

### 场景6：工作群的深夜
> 凌晨十一点，工作群突然响了。领导发了一段语音，60秒。她不想听，但群里其他人已经秒回了"收到"。她戴上耳机，按下播放键——是明天临时改方案的通知。她深吸一口气，回了"收到"，然后把手机扣在桌上。

### 场景7：拍一拍翻车
> 午休时她无聊，点开了他的头像拍了拍。但她忘了自己设置的拍一拍后缀是"拍了拍你的八块腹肌并流了口水"——问题是她拍的是部门领导。等她发现的时候，整个部门群都看到了。

## 六、写作避坑

1. **微信没有"已读"功能**——不要写"他已读了我的消息却不回"，只能说"他没回消息"。但"对方正在输入"是存在的，可以利用。
2. **朋友圈评论只有共同好友可见**——不要写A看到了B给C的评论，除非A和C也是好友。
3. **红包上限200元**——转账可以超过200，但红包单个最多200。不要写"发了一个520的红包"，应该是"转了520"。
4. **不要把微信写得像QQ**——微信没有等级、没有在线状态、没有个性签名可见功能（签名藏得深，不是随便能看的）。微信的气质是克制、留白。
5. **微信聊天的节奏感**——真实聊天很少用完整句子，多用短句、表情包、省略号、"哈哈"。角色越熟，消息越短越碎。
