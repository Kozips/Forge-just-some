# QQ

> 中国最早的国民级即时通讯软件，学生群体的"精神老家"。从1999年至今，QQ见证了一代人的青春——等级、空间、扩列，每一样都是回忆杀。

---

## 一、平台定位（写作用途）

- **学生群体的社交主场**：小学生到大学生，QQ是比微信更"年轻"的社交阵地
- **用户画像**：00后/05后为主力，学生党标配；部分90后"情怀用户"仍在使用
- **气质标签**：年轻、花哨、个性化、二次元、学生气、"QQ是青春，微信是长大"
- **写作适用**：校园文首选平台、学生社交生态、扩列/处CP、QQ空间的公开日记、群文件分享、匿名提问的暗恋与窥探、从QQ到微信的"长大"仪式

---

## 二、核心功能

### 2.1 聊天

| 功能 | 细节 | 写作价值 |
|------|------|---------|
| **文字/语音/视频/文件** | 比微信更自由——可发大文件、离线文件 | 学生之间传作业/资料的主要渠道 |
| **撤回消息** | 可撤回，对方看到提示 | 和微信一样有"撤回即心虚"效果 |
| **戳一戳** | 类似微信拍一拍，但更早出现 | 恶作剧/引起注意 |
| **消息气泡** | 可自定义气泡样式和颜色 | 个性化——用什么气泡=审美展示 |
| **字体** | 可设置聊天字体（大小、颜色、花式） | 花哨字体=中二/年轻感 |
| **GIF/表情** | 斗图文化的主战场 | 一个群999+消息可能998条是表情包 |
| **匿名消息** | 群内可匿名发言 | 暗恋表白/八卦/匿名吐槽 |
| **闪照** | 照片只能看几秒就自动销毁 | 发不能留存的照片——自拍/丑照 |
| **屏幕分享/远程协助** | 共享屏幕或远程控制对方电脑 | 帮同学远程解决电脑问题 |
| **一起听歌/一起看** | 多人同时听歌或看视频 | 暧昧期的"一起听歌" |
| **匿名提问** | 在资料卡上匿名提问 | 暗恋者的窥探渠道——"你有喜欢的人吗？" |

### 2.2 QQ空间（QQ的"朋友圈"，但更开放）

| 功能 | 细节 | 写作价值 |
|------|------|---------|
| **说说** | 类似朋友圈，但更频繁、更随意 | 学生的"公开日记"——心情、吐槽、日常 |
| **日志** | 长文发布，类似博客 | 80/90后青春记忆——"QQ空间日志" |
| **相册** | 照片存储和展示，可设权限 | 毕业照、旅行照、情侣相册 |
| **留言板** | 在对方空间留言（所有人可见） | 社交表演场——"踩踩""来过"、暗恋留言 |
| **访客记录** | 能看到谁来过你的空间 | 最恐怖的功能——"你怎么知道我来过？" |
| **黄钻** | 付费会员，可装扮空间、隐身访问 | 想看又不想被发现 = 隐身访问 |
| **空间装扮** | 背景音乐、皮肤、漂浮物、挂件 | 极致个性化——每个人的主页都不一样 |
| **被挡访客** | 非好友访问被权限挡住时仍能记录 | 设了权限也能看到谁想窥探你 |

### 2.3 QQ群（比微信群更"强功能"）

| 功能 | 细节 | 写作价值 |
|------|------|---------|
| **群文件** | 可上传/下载大文件 | 学生分享课件、资料、小说资源 |
| **群相册** | 群的公共相册 | 班级活动照片/黑历史 |
| **群公告** | @所有人，强制提醒 | 班级群通知 |
| **匿名聊天** | 群主开启后可匿名发言 | 匿名吐槽/告白/八卦 |
| **群投票** | 发起投票 | 班级选班长、聚餐选地点 |
| **群待办** | 标记重要消息 | 老师布置作业 |
| **群头衔** | 群主可设置自定义头衔 | "班长""课代表""气氛组" |
| **禁言** | 单人禁言/全体禁言 | 违反群规的惩罚 |
| **群精华** | 将消息标记为精华 | 重要信息不被刷掉 |

### 2.4 社交功能

| 功能 | 细节 | 写作价值 |
|------|------|---------|
| **扩列** | 互相加好友，扩大社交圈 | 学生群体特有的社交方式——"扩列dd" |
| **处CP/处Q友** | 寻找线上恋人/好友 | 网恋的起点——"处对象吗？" |
| **好友互动标识** | 养火花、巨轮、闺蜜标等 | 社交关系的"徽章"——标识越丰富关系越深 |
| **QQ等级** | 太阳、月亮、星星 | 青春期的"虚荣指标"——几个太阳了？ |
| **个性签名** | 展示在资料页 | 心情/态度的外显——"丧系签名""霸气语录" |
| **资料卡** | 展示年龄/星座/学校等信息 | 搭讪的第一步——先看资料 |
| **厘米秀** | 虚拟形象系统 | 年轻用户很喜欢装扮自己的虚拟形象 |
| **QQ宠物** | 虚拟宠物养成（已下线但有情怀） | 90后的回忆杀 |
| **兴趣部落** | 基于兴趣的社区（类似贴吧） | 同好交流 |
| **QQ频道** | 类似Discord的频道功能 | 游戏/兴趣社群 |

### 2.5 会员体系

| 等级 | 特权 | 写作价值 |
|------|------|---------|
| **普通用户** | 基础功能 | 普通学生 |
| **超级会员(SVIP)** | 红色铭牌、更多特权、等级加速 | 身份象征——同学间攀比 |
| **黄钻** | 空间装扮、隐身访问空间 | 偷偷看暗恋对象空间的必备 |
| **绿钻** | QQ音乐VIP | 听歌少年/少女 |
| **大会员** | 全平台特权 | 有钱任性 |

## 三、好友互动标识系统（养火花/巨轮）

这是QQ最独特、最能写进剧情的功能——通过持续互动培养"关系徽章"：

| 标识 | 条件 | 含义/写法 |
|------|------|----------|
| **火花（小火苗）** | 连续互发消息超过1天 | 基础互动标识 |
| **大火花** | 连续互发消息超过30天 | "我们养了30天的火花" |
| **巨轮** | 与对方互为最频繁聊天好友且超过7天 | 你们是最亲密的朋友——暧昧信号 |
| **闺蜜标识** | 双方互为最频繁好友 | 女生之间的亲密证明 |
| **基友标识** | 类似闺蜜 | 男生版本 |
| **恋人标识** | 设置情侣关系 | 正式官宣 |

**火花消失** = 断联 = 关系破裂的信号（非常有戏剧性："我们的火花断了"）

## 六、写作场景模板（可直接嵌入剧情）

### 场景1：火花断了
> 她打开QQ，对话框上方的小火苗图标消失了。连续养了63天的火花，断了。昨天她说"晚安"之后，他没有回最后一条消息——就差那么一条。她看着空荡荡的火苗位置，像丢了一样什么东西。她输入框里打了"在吗"，又删掉了。

### 场景2：空间访客
> 她开了黄钻，唯一的用途就是"隐身访问"。每天至少进他的空间三次——看他说说写了什么、留言板有没有人暧昧留言、相册有没有新照片。今天她发现他的留言板上多了一条："等你下课，老地方。"头像是个陌生女生。她关掉了手机。

### 场景3：扩列
> 她发了一条空间说说："扩列dd，05后，二次元，不处对象。"配图是她新买的动漫周边。半小时后收到了二十多条好友申请，验证消息五花八门——"交个朋友""同好！""看看你的空间"。她挨个通过，然后发现其中一个男生的空间封面是她最爱的那部番。

### 场景4：匿名表白墙
> 学校的QQ群"表白墙"又更新了。第42条写着："高二六班坐在窗边第三个位置的女生，我每天都会路过你们班门口，希望你有天能注意到我。"她就是高二六班窗边第三个位置。同桌把手机怼到她面前："这是不是写给你的？"她心跳加速，但评论区已经有三个人猜了不同的名字。

### 场景5：巨轮争夺
> 他和她的QQ聊天界面右上角，一个月亮图标旁多了一个"巨轮"的标识。室友看到后起哄了一整天："你俩有巨轮！有巨轮啊！"他面不改色地说"聊得多而已"，但晚上她没发消息的时候，他还是忍不住先发了一个表情包——他不想让巨轮消失。

### 场景6：等级攀比
> "你几个太阳了？"这是班里最近最流行的问题。他炫耀着自己两个太阳的等级，旁边的学霸面无表情地截图——四个太阳。"你天天学习怎么还比我等级高？""我挂Q啊。"全班沉默了两秒，然后集体笑了。

## 七、写作避坑

1. **QQ和微信的用户重叠但不完全相同**——同一个人可能在QQ上话很多但微信上很安静，QQ更"放飞自我"。角色在两个平台的表现可以有反差。
2. **空间访客记录是真的能查到的**——这是QQ最"暴露隐私"的功能，适合写偷窥/暗恋/被发现的剧情。但黄钻可以隐身访问。
3. **火花/巨轮是有视觉标识的**——不是隐性功能，聊天界面右上角就能看到，旁边的人都能瞄到。这一点可以写出很多"被发现"的戏剧冲突。
4. **扩列/处CP文化主要是学生群体**——成年职场人不太会用这些功能。如果角色是大学生以上，QQ的使用频率会降低，更多转向微信。
5. **QQ的个性化程度远高于微信**——花哨的字体、气泡、头像挂件、空间装扮，这些视觉元素可以用来做人物性格的外化（朴素的人用默认装扮，潮人用各种花哨装饰）。
