# 淘宝

> 中国最大的综合电商平台，"万能的淘宝"不是一句空话——从9.9包邮到高定礼服，什么都能找到。月活超8亿，是都市/校园文中出现频率最高的购物场景。

---

## 一、平台定位（写作用途）

- **最万能的电商平台**：只有你想不到，没有淘宝买不到
- **用户画像**：18-45岁为主，女性用户略多；从学生党到中产白领都有；消费习惯从精打细算到"买它"一掷千金全覆盖
- **气质标签**：万能、淘货、评价驱动、直播+电商双引擎、大促狂欢
- **写作适用**：网购日常、凑满减、双十一血拼、看评价种草/拔草、直播间蹲守、退货售后纠纷、商家创业故事、代购/海淘

---

## 二、核心功能

### 2.1 搜索与商品浏览

| 功能 | 描述 | 写作场景 |
|------|------|---------|
| **搜索** | 关键词搜索商品，支持图片搜同款（拍立淘） | 角色用"拍照搜索"找一件在街上看到的衣服 |
| **推荐/猜你喜欢** | 首页信息流个性化推荐 | "打开淘宝本来只买个充电线，结果刷了半小时" |
| **店铺** | 品牌旗舰店/个人C店/天猫店 | 追星女孩蹲明星同款店铺 |
| **商品详情页** | 图片+参数+评价+问大家 | 纠结症患者反复看评价的日常 |

### 2.2 评价体系（淘宝最核心的社交资产）

- **好评/中评/差评**：买家打分，直接影响店铺权重
- **带图评价**：附带实物照片，比文字更有说服力
- **追评**：使用一段时间后追加评价（"用了三个月，掉漆了，改差评"）
- **问大家**：准买家向已购买者提问——"这个真的好用吗？""170cm穿什么码？"
- **买家秀vs卖家秀**：经典梗，实物与图片的天壤之别

### 2.3 大促活动

| 活动 | 时间 | 特征 |
|------|------|------|
| **双十一（双11）** | 11月11日 | 全年最大购物节，预售+尾款+满减+红包 |
| **618** | 6月18日 | 年中大促，京东主场但淘宝也疯狂参与 |
| **年货节** | 春节前 | 置办年货 |
| **38女王节** | 3月8日 | 女性消费专场 |
| **双十二（双12）** | 12月12日 | 规模较小但折扣实在 |

### 2.4 淘宝直播

- **李佳琦式头部主播**：全网最低价、"所有女生"、OMG
- **品牌自播**：品牌官方直播间，节奏相对温和
- **中小主播**：垂直品类，如玉石、茶叶、服装档口
- **直播回放**：错过直播可以看录播下单

### 2.5 88VIP

- 年费88元（需淘气值达标），享天猫95折、优酷/饿了么/网易云等联合会员
- 写作场景：角色炫耀"我是88VIP"的优越感/被嘲笑"你也配88VIP"

### 2.6 淘宝特色功能

- **购物车**：收藏商品等大促一起下单（"购物车都溢出来了"）
- **凑单**：为了满减把不需要的东西也加进购物车
- **收藏店铺**：常逛的店加收藏
- **淘宝人生**：3D虚拟形象+消费数据回顾
- **淘友圈**：可以看到好友买了什么（社死功能）
- **菜鸟驿站/菜鸟裹裹**：快递代收点，取快递的日常

## 三、平台黑话/梗速查表

### 3.1 购物类

| 黑话 | 含义 | 用法示例 |
|------|------|---------|
| 满减 | 满一定金额减钱（如满200减30） | "差20块凑满减，随便加了个手机壳" |
| 凑单 | 为了达到满减门槛额外买的东西 | "凑单的比正经买的还贵" |
| 到手价 | 用完所有优惠后的实际价格 | "到手价才19.9，冲了" |
| 返利/返现 | 下单后返还部分金额 | "走返利链接还能再省5块" |
| 优惠券/红包 | 平台或店铺发放的折扣券 | "先领券再下单，别直接拍" |
| 跨店满减 | 不同店铺的商品凑一起满减 | "跨店满300减50，记得凑好" |
| 尾款人 | 预售后付尾款的人 | "双十一尾款人在此，花呗已阵亡" |
| 定金/预售 | 提前付定金锁定优惠，大促时付尾款 | "定金50抵100，尾款日再付" |
| 店铺粉丝价 | 关注店铺后享受专属价格 | "先关注，有粉丝价" |
| 运费险 | 退货包运费 | "有运费险就大胆买，不合适直接退" |

### 3.2 评价/社交类

| 黑话 | 含义 |
|------|------|
| 买家秀 | 买家拍的实物照片（常与卖家图形成惨烈对比） |
| 卖家秀 | 商家精修的产品图 |
| 差评师 | 故意给差评勒索商家的人 |
| 种草 | 看到好东西被安利，想要买 |
| 拔草 | 买了之后发现不值，打消购买念头 |
| 好评返现 | 商家给好评送红包（实际违规但普遍存在） |
| 淘友圈社死 | 淘友圈暴露了自己的消费记录 |

### 3.3 商家/运营类

| 黑话 | 含义 |
|------|------|
| C店/个人店 | 个人开设的淘宝店铺 |
| 天猫/旗舰店 | 品牌官方入驻的店铺（更有保障） |
| SKU | 商品的具体规格（颜色、尺码等） |
| 详情页 | 商品的详细介绍页面 |
| 转化率 | 看了商品后下单的比例 |
| DSR | 店铺动态评分（描述相符、服务态度、物流速度） |
| 刷单 | 虚假交易提高销量（平台严打但屡禁不止） |
| 淘客 | 帮商家推广赚佣金的人 |
| 淘宝联盟 | 官方推广返佣平台 |
| 皇冠/金冠 | 店铺等级，交易量越大等级越高 |

### 3.4 平台梗

| 梗 | 含义 |
|----|------|
| 万能的淘宝 | 什么都买得到 |
| 买家秀vs卖家秀 | 实物与图片的惨烈对比 |
| 淘宝给了我第二次选择的机会 | 讽刺买到了便宜但质量差的东西 |
| 双十一算术题 | 凑满减/叠优惠券的复杂计算让人头大 |
| 尾款人的悲伤 | 付完尾款看着账单想哭 |
| 打开淘宝想买东西，关上淘宝发现逛了俩小时 | 冲浪黑洞 |
| 淘宝猜你喜欢猜得真准 | 算法推荐太精准让人害怕 |

## 四、写作场景模板（可直接嵌入剧情）

### 场景1：双十一凑单
> 加完班回到家已经十一点半，离双十一开抢还有半小时。她把购物车里的东西又过了一遍：口红、面霜、加绒卫衣、暖宝宝、猫粮。满300减50，还差47块。她随手加了一包抽纸和一盒棉签，刚好302。"可以了。"她盯着屏幕倒计时，手指悬在"结算"上方，心跳莫名其妙地加快了。

### 场景2：买家秀翻车
> 快递到了，她满怀期待地拆开包裹——那件直播间里看着仙气飘飘的碎花裙，实物像从菜市场十块钱三件的地摊上捡的。颜色暗了三个度，版型完全不对，袖口的线头还没剪干净。她对着镜子拍了张照发到闺蜜群："各位，卖家秀和买家秀又开战了，这次我完败。"

### 场景3：逛淘宝停不下来
> 她本来只想买一盒面膜，点进去之后推荐栏出现了"猜你喜欢"——同风格的裙子、搭配的耳环、配套的包。每点一个进去，底下又跳出更多推荐。等她回过神来，购物车里已经躺了十七样东西，时间过了一个半小时。她叹了口气："淘宝的推荐算法到底是怎么做到比我自己还了解我的？"

### 场景4：追评的杀伤力
> 她在一家月销十万的护肤店看到一条追评："刚收到觉得还行给了好评。用了两周，脸开始起红疹，去医院查了是激素脸。已申请退款，商家不理，走12315了。姐妹们别买。"这条追评底下的回复已经盖了八百楼，全是"我也中招了"。

### 场景5：淘友圈社死
> 闺蜜发来一张截图："你昨天买的那套情趣内衣……嗯？"她愣了两秒，猛地想起淘友圈没关——所有购买记录默认对好友可见。她手忙脚乱地去设置里关掉，又一条消息弹出来："别关了，我已经截图了，准备发朋友圈。"她发出一声尖叫。

### 场景6：88VIP的优越感
> 结账的时候，同事凑过来："这个有88VIP打折吗？""有，打了九五折，省了三十七。"同事一脸羡慕："我淘气值才六百多，差远了。"她嘴上说"也没省多少"，表情却分明写着——享受这个时刻。

## 五、写作避坑

1. **不要把满减规则写得太简单**——真实的大促优惠叠加极其复杂（跨店满减+店铺券+品类券+红包+88VIP折扣），"算术题"本身就是经典梗
2. **评价区不是一片好评**——真实的商品页面永远有差评、追评反转、"问大家"里的真实吐槽，不要写成全五星好评的虚假繁荣
3. **买家秀和卖家秀的反差要具体**——不要笼统说"货不对板"，要写具体细节：色差、线头、缩水、味道刺鼻
4. **大促抢购的真实感**——零点抢购时的卡顿、库存秒没、手慢无的焦虑，以及第二天看着账单的后悔，这是完整的体验
5. **菜鸟驿站的日常**——取快递时翻半天找自己的件、短信通知"您的快递已到XX号货架"，这种生活细节比"快递送到家门口"更真实
6. **88VIP和普通用户的差异**——不要写成所有人都有88VIP，淘气值达标门槛不低，这是少数人的"特权"
