# 百度贴吧

> 中国最老牌的兴趣社区，以"吧"为单位按话题聚合用户。曾经的中文互联网第一社区，如今是情怀与衰落并存的"遗迹"，但仍有大量活跃吧和忠实用户。

---

## 一、平台定位（写作用途）

- **按兴趣建吧的老牌中文社区**：万物皆可建吧，从明星到哲学到"戒色吧"无所不包
- **用户画像**：核心用户以男性为主，年龄跨度大（80后到00后），学生党和上班族都有；曾经是最大的青年聚集地，现在用户大量流向B站/知乎/小红书，但仍有一批死忠"贴吧老哥"
- **气质标签**：草根、野生、直男、毒舌、抖机灵、曾经的互联网文化发源地、如今的"互联网活化石"
- **写作适用**：神帖传播、吧友互喷、小众圈子讨论、校园表白墙前身、网络暴力、段子文化、"挖坟"（翻老帖）、爆吧事件、怀旧情怀

---

## 二、核心功能

### 2.1 吧（贴吧的核心单元）

每个"吧"就是一个独立的小论坛，围绕一个主题（人物/事物/概念/兴趣）：

| 吧的类型 | 示例 | 特点 |
|---------|------|------|
| **明星/名人吧** | 李毅吧、某某明星吧 | 人数最多，粉丝/黑粉混战 |
| **兴趣吧** | 钓鱼吧、天文吧、手办吧 | 垂直讨论，氛围相对和谐 |
| **地域吧** | 北京吧、某某大学吧 | 校园表白墙、二手交易、本地吐槽 |
| **游戏/动漫吧** | 魔兽世界吧、某动漫名吧 | 老玩家聚集，攻略/讨论/吐槽 |
| **奇葩吧** | 戒色吧、反XX吧、XX弱智吧 | 贴吧独有现象，什么吧都有人建 |
| **综合吧** | 百度吧、贴吧吧 | 平台本身的讨论区 |

**写作要点**：每个吧都有自己的文化、黑话、潜规则。吧主/小吧主有管理权限（删帖/封禁/置顶），吧内有等级体系（签到升级）。

### 2.2 帖子

- **发帖**：在吧内发帖，支持文字+图片（早期不支持，后来逐步开放）
- **回帖/盖楼**：帖子下方回复，形成"楼"的结构，几百上千楼的帖子叫"高楼帖"
- **置顶/精品帖**：吧主可将优秀帖子置顶或标记为精品
- **删帖/封禁**：吧主有权删除违规帖子，封禁违规用户
- **签名档**：每个用户可以设置帖子底部的签名图/文字

### 2.3 等级/签到

- **贴吧等级**：在每个吧里有独立等级（1-18级），通过签到和发帖提升
- **经验值**：每日签到+发帖+回帖累积经验
- **吧主/小吧主**：吧内管理职位，吧主可通过申请或"抢吧"获得
- **大吧主**：吧的最高管理者，掌管吧内一切

### 2.4 搜索/推荐

- **贴吧首页**：推荐热帖、关注的吧的更新
- **搜索**：搜吧名进入，或搜帖子内容
- **热帖榜**：吧内/全站热门帖子排名
- **贴吧热搜**：全站热门话题

### 2.5 消息/私信

- **@某人**：在帖子里@用户
- **私信**：站内消息
- **回复通知**：有人回复你的帖子/评论会收到通知

## 三、平台黑话/梗速查表

### 3.1 社区称呼类

| 黑话 | 含义 | 用法示例 |
|------|------|---------|
| 贴吧老哥 | 对吧友的尊称，含调侃意味 | "贴吧老哥还是牛逼" |
| 度娘 | 百度的拟人化昵称 | "度娘又抽风了" |
| 吧主/大吧 | 吧的管理员 | "吧主快出来管管" |
| 楼主/LZ | 发帖人 | "楼主好人一生平安" |
| 沙发 | 第一个回帖的人 | "抢沙发！" |
| 板凳/地板 | 第二、第三个回帖 | "板凳都没抢到" |
| 吧友/吧友 | 同一个吧里的用户 | "吧友们给点建议" |
| 坟贴 | 很久以前的帖子被人挖出来回复 | "挖坟了挖坟了" |

### 3.2 内容文化类

| 黑话 | 含义 | 用法示例 |
|------|------|---------|
| 神帖 | 内容极其精彩的帖子，广为流传 | "经典神帖，建议加精" |
| 高楼帖 | 回复数极多的帖子 | "这帖子盖了五千楼" |
| 盖楼 | 不断回帖，帖子越来越长 | "大家一起盖楼" |
| 爆吧 | 大量用户涌入某个吧疯狂发帖/刷屏（通常是捣乱） | "今晚爆XX吧" |
| 考古/挖坟 | 回复很久以前的帖子，让它重新浮上来 | "别挖坟了，人楼主都跑了" |
| 水贴 | 发无意义的帖子/回复凑数刷等级 | "又来水经验了" |
| 精品帖 | 被吧主标记为精华的优质帖子 | "这帖该加精品" |
| 抢楼/占楼 | 快速回复，占据帖子前排位置 | "占楼编辑，稍后补内容" |
| 内涵 | 含蓄的讽刺/影射，不是表面意思 | "这帖子真有内涵" |
| 留名 | 回复表示到此一游，不一定有实质内容 | "留名，以后再看" |

### 3.3 事件/文化类

| 梗/黑话 | 含义 |
|---------|------|
| 李毅吧/帝吧 | "百度贴吧第一吧"，曾经最大的吧，因足球运动员李毅得名，后演变为网络文化输出地 |
| D8 | 李毅吧的别称 |
| 帝吧出征 | 帝吧用户集体行动（通常是"翻墙"去某平台刷屏） |
| 魔兽世界吧/WOW吧 | 曾经第二大吧，产出了大量经典段子 |
| 李毅大帝 | 李毅吧的精神图腾，以自恋金句闻名 |
| "我的护球像亨利" | 李毅经典语录，贴吧名梗 |
| "不要迷恋哥，哥只是个传说" | 贴吧早期名言 |
| 王自如vs老罗 | 贴吧经典讨论事件 |
| 百度贴吧被卖吧事件 | 吧主被撤、商业化入驻，引发大规模用户不满和出走 |
| 贴吧"坟贴" | 几年前甚至十几年前的帖子被挖出来，原楼主可能早已不在线 |
| 帖子被"吞" | 帖子莫名消失（审核/系统删除），吧友常说"被度娘吞了" |
| "没有人比我更懂XX" | 讽刺某人自以为是（源自特朗普梗，在贴吧大量使用） |

## 四、写作场景模板（可直接嵌入剧情）

### 场景1：神帖诞生
> 凌晨三点，他在某某大学吧随手发了个帖——"室友半夜梦游把我的期末论文格式化了，我现在该怎么办"。帖子下面的回复从"哈哈哈哈"逐渐变成"建议杀室友"，再到"我来帮你恢复数据试试"。第二天醒来，帖子已经盖了三千楼，吧主加了精品，他的手机被各种@消息卡到死机。

### 场景2：挖坟名场面
> 她点开一条回复通知，发现自己三年前在暗恋对象的表白帖下面回复的"祝你幸福"，被人翻了出来。新回复写着"三年过去了，楼主幸福了吗？回来看看啊"。她的手指悬在屏幕上方，看了看楼主最后登录时间——一年前。

### 场景3：爆吧事件
> 消息传遍了整个吧：今晚九点，帝吧出征。贴吧首页被刷屏了，满屏的帖子都在喊口号，新帖一秒就被淹没。他坐在电脑前，F5按到手指发麻，看着帖子像潮水一样涌上来，中间夹杂着几个试图讲道理的理性帖——三秒就被刷到第二页了。

### 场景4：吧主争夺
> 老吧主三个月没上线了，吧规形同虚设，广告帖满天飞。他申请了大吧主，公示期七天。有人在吧里发帖："又一个想当吧主捞好处的。"他没回，默默删掉了首页的三条卖片广告。第七天，系统通知：恭喜您成为XX吧吧主。

### 场景5：吧内暗战
> 两个小吧主闹翻了，一个在吧里置顶了"吧规更新公告"暗戳戳点名另一个，另一个在精品帖评论区贴出了前者的黑历史截图。吧友分成两派互喷，普通帖子全被淹没了。他看了看今天要找的那个攻略帖——已经被顶到了第三页。

### 场景6：校园表白墙的前身
> 大学吧里每天都有人在发"墙"——"今天在图书馆三楼靠窗位置看到一个穿白衬衫的男生，求捞"。他翻了翻，第六条就是描述他自己的。回帖里有人@了他，有人贴出了他的课表，还有人说"这人有对象了吧"。他看了眼课表——连周三下午的选修课都标对了。

## 五、写作避坑

1. **贴吧已不是当年的贴吧**——现在的贴吧活跃度远不如2010-2016年高峰期，如果写当下时间线，用户会提到"贴吧没落了""老哥们都走了"，而不是把它当主流平台
2. **吧和吧之间文化差异巨大**——李毅吧的风格和手办吧完全不同，不要用同一个"贴吧语气"套用所有吧。大吧偏毒舌/抖机灵，小吧往往更和谐
3. **吧主权力是真实存在的**——吧主可以删帖、封禁、置顶、加精，这是贴吧的核心生态。吧主更替、争夺管理权是非常真实的剧情素材
4. **"内涵"是贴吧的灵魂**——贴吧老哥不会直说，喜欢用暗讽、谐音、反话、抖机灵。写对话语气时注意这种"弯弯绕绕"的风格
5. **不要把贴吧写成干净整洁的样子**——真实的贴吧充斥着水贴、广告、复制粘贴的段子、无意义的"留名"，以及偶尔冒出的真正有价值的内容。这种混乱感才是贴吧的真实质感
