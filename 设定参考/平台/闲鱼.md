# 闲鱼

> "出闲置，找宝贝"——中国最大的二手交易平台，但远不止二手。它是人间观察站、砍价修罗场、社恐社交平台、以及当代消费主义的照妖镜。月活超1.5亿，Z世代用户占比最高。

---

## 一、平台定位（写作用途）

- **最有人间烟火气的交易平台**：表面是二手交易，实际是社会百态的显微镜
- **用户画像**：18-35岁年轻人为主，大学生和刚毕业的年轻人占比极高；卖家和买家身份随时互换
- **气质标签**：砍价文化、人间真实、故事感、奇葩文案、闲鱼小法庭、社区感、"回血"
- **写作适用**：砍价博弈、闲鱼奇遇、小法庭断案、卖家文案写作、闲置物品背后的故事、角色经济状况的侧面描写、以物易物的浪漫

---

## 二、核心功能

### 2.1 发布/卖闲置

| 功能 | 描述 | 写作场景 |
|------|------|---------|
| **一键转卖** | 淘宝/京东订单直接转到闲鱼 | "前任送的礼物一键转卖" |
| **拍照发布** | 手机拍照+写描述+定价 | 每件闲置都有一个故事 |
| **定价策略** | 全新/九成新/自定义价格 | 纠结定价时的心理活动 |
| **包邮/不包邮** | 可选邮费承担方式 | "包邮吧，不然没人买" |
| **鱼塘** | 兴趣社区/同城社区 | 在鱼塘里遇到同好/同城的人 |

### 2.2 砍价系统（闲鱼的灵魂）

- **留言砍价**：在商品下方评论"XXX出吗？"（公开砍价，所有人都能看到）
- **私聊砍价**：点"我想要"进入私聊，开始一来一回的价格拉锯
- **到手刀**：收到货后以各种理由再砍一刀——"这里有个划痕，再便宜点"
- **心理价位博弈**：卖家标价虚高等着被砍，买家往死里砍，最后在中间成交
- **"不议价"的卖家**：简介写"不议价勿扰"但依然被疯狂砍价

### 2.3 闲鱼小法庭（闲鱼最有特色的功能）

- 买卖双方发生纠纷时，由17位随机选出的"大众评审"投票裁决
- 双方各写陈词、上传证据，评审投票
- **极其有戏剧性**——因为是真人投票，结果有时出人意料
- 写作场景：角色作为评审参与断案/角色作为卖家或买家打小法庭

### 2.4 闲鱼社区

- **鱼塘**：按兴趣/地域分类的社区（如"手帐鱼塘""XX大学鱼塘"）
- **超赞**：对商品点赞（影响曝光）
- **关注/粉丝**：可以关注有趣的卖家
- **闲鱼动态**：类似朋友圈，发布生活动态（但主要用于引流卖货）

### 2.5 闲鱼特色玩法

- **拍卖**：起拍价很低，竞价决定最终价格
- **闲鱼优品**：官方验过的二手商品（质量更有保障）
- **同城交易**：面对面交易，可以先验货再付款
- **以物换物**：不需要钱，直接交换各自闲置（写作用途很浪漫）
- **"求购"功能**：发布想要买的东西，卖家主动联系

## 三、平台黑话/梗速查表

### 3.1 砍价/交易类

| 黑话 | 含义 | 用法示例 |
|------|------|---------|
| 到手刀 | 收货后找理由再砍价 | "你这个到手刀我见多了，不接受退货" |
| 屠龙刀 | 砍价幅度极大的买家 | "你这是屠龙刀啊，对半砍" |
| 小刀 | 小幅度砍价 | "可以小刀，但别太狠" |
| 大刀/对半砍 | 大幅度砍价 | "上来就对半砍，你是认真的吗" |
| 自刀 | 卖家主动降价 | "自刀50，诚心出" |
| 急出 | 急着卖掉（通常暗示可以更便宜） | "急出，价格好商量" |
| 拍下改价 | 买家下单后卖家修改价格 | "你拍下我改价" |
| 不刀不议价 | 不接受砍价 | 简介写了不刀，但依然有人来砍 |
| 出/出吗 | 这个还卖吗/愿意卖吗 | "这个1200出吗？" |
| 包邮 | 卖家承担运费 | "包邮的话我就要了" |
| 到付 | 买家承担运费 | "不包邮，到付" |

### 3.2 商品状态类

| 黑话 | 含义 |
|------|------|
| 全新未拆 | 全新没拆封 |
| 99新 | 基本没用过，接近全新 |
| 95新 | 用过但保养得好 |
| 8新/自定义 | 明显使用痕迹 |
| 传家宝 | 定价远超实际价值的商品（反讽） |
| 捡漏 | 以远低于市场价买到好东西 |
| 回血 | 卖闲置回笼资金（通常暗示经济紧张） |

### 3.3 闲鱼文化梗

| 梗 | 含义 |
|----|------|
| 闲鱼文学 | 商品描述里的魔性文案，堪比散文 |
| 前任的东西 | 分手后出闲置是闲鱼经典场景 |
| 小法庭见 | 谈不拢就上闲鱼小法庭裁决 |
| 闲鱼出的就是有故事的 | 每件闲置背后都有故事 |
| "老公不让买/老婆不让留" | 经典的卖货理由 |
| 因为穷，所以出 | 坦诚的卖货理由 |
| 验尸式检查 | 买家收到货后极其仔细地检查找瑕疵 |
| 闲鱼人均富二代 | 讽刺卖家定价离谱 |
| 面交 | 同城当面交易 |
| 传家宝价格 | 标价比新的还贵 |

## 四、写作场景模板（可直接嵌入剧情）

### 场景1：前任礼物清仓
> 她把前任送的东西一件件摆在地上：一条施华洛世奇项链（他说是"限量款"，后来发现是淘宝同款）、一只Coach包（背了两次）、三束永生花（已经积灰了）。她拍好照片，打开闲鱼，每件都标了"前任送的，不想留了，低价出"。描述写得简洁而克制，但每一件物品背后的故事，她一个字都不想多提。发布不到十分钟，第一条消息就来了——"项链多少出？可以小刀吗？"

### 场景2：砍价拉锯战
> 买家："这台相机6000出吗？"（标价8500）
> 她："最低7800，不刀了。"
> 买家："6500，诚心要。"
> 她："7500，送你个相机包。"
> 买家："6800，学生党真的没预算了。"
> 她："7300，再低我不如自己留着用。"
> 买家沉默了十分钟。
> 买家："7000，我再加个包邮，行不行？最后一次了。"
> 她盯着屏幕，咬了咬嘴唇。这台相机她确实急着出——房租月底到期。她打了三个字："拍下吧。"心里想的是：亏了五百，但至少这个月能过。

### 场景3：闲鱼小法庭
> 卖家的陈词："发货前拍了完整开箱视频，产品完好无损。买家收到后说屏幕有划痕，但照片模糊根本看不清，而且他过了五天才申请退款，明显是自己弄的。"
> 买家的陈词："收到就有划痕，第一时间拍照了。卖家说'发货前没有'就能证明不是运输中刮的？我要求退款。"
> 17位评审开始投票。投票过程比追剧还紧张——6:6的时候她的心提到了嗓子眼，最后8:9，买家险胜。卖家在评论区炸了："小法庭就是偏袒买家！"

### 场景4：闲鱼偶遇
> 他在闲鱼上搜"二手黑胶唱片"，刷到一个卖家，头像是一个女生的侧脸，简介写着"搬家清仓，唱片都是心头肉"。他点进去，唱片品味和自己几乎一模一样——Radiohead、坂本龙一、窦唯。他挑了三张，私聊问价。聊天时发现对方就住在隔壁小区。面交那天，她在咖啡馆里把唱片从纸箱里一张张拿出来，每张都包了气泡膜，边角一点折痕都没有。"你这么爱惜，怎么舍得卖？"他问。她笑了一下："搬家装不下。"但眼神分明在说——不只是唱片装不下。

### 场景5：闲鱼文学大赏
> 商品描述：
> "这把吉他跟了我三年。大一的时候攒了两个月生活费买的，以为自己会成为校园民谣歌手。结果大二考了四级就再也没碰过。琴弦有点锈了，面板有一处小磕碰——和我大一的初恋一样，留下了痕迹但不影响发声。1200出，送琴包和变调夹。希望下一任主人能弹得比我好。"
> 评论区第一条："1000出吗？"第二条："吉他出吗，前任的故事也顺便讲讲。"第三条："你这是卖琴还是写散文？"

### 场景6：面交社恐现场
> 他们约在地铁站B口见面。她远远就看到一个男生举着手机四处张望——手里拿着她要买的switch。"你好，我是闲鱼上那个……""哦哦，是你！"他递过switch，她接过来翻来覆去看了三遍，检查屏幕、手柄、充电口。"没什么问题吧？""嗯……还行。"她犹豫了一下："那我转你钱了？""好。"两人在地铁站门口站着，各自低头操作手机。转账完成，沉默了五秒。"那……再见？""再见。"她转身走了，心想：面交好尴尬，下次还是邮寄吧。

## 五、写作避坑

1. **砍价描写要真实**——闲鱼砍价不是"便宜点呗""好的"这么简单，是一来一回的拉锯战，有试探、有表演、有僵持，有时要写好几轮对话
2. **每件闲置都有故事**——不要把闲鱼只当成"卖旧东西的地方"，分手礼物、搬家清仓、冲动消费后悔、经济窘迫回血……每个商品背后都是人生片段
3. **小法庭的戏剧性**——这是闲鱼最有特色的功能，17个陌生人投票裁决，结果有时反直觉，非常适合写冲突和悬念
4. **"到手刀"是交易毒瘤**——这是闲鱼最令人痛恨的行为之一，买家收到货后找茬再砍价，描写时要体现卖家的愤怒和无奈
5. **面交的尴尬感**——两个陌生人拿着手机确认转账的画面，既真实又有喜感，适合写尴尬幽默的场景
6. **闲鱼不只是卖东西**——鱼塘社区、同城交易、以物换物，闲鱼的社交属性比其他电商平台强得多，可以写出相遇、偶遇的故事线
