# 饿了么

> 阿里巴巴旗下的外卖及本地生活平台，与美团外卖形成双寡头竞争。2018年被阿里收购后深度融入阿里生态（支付宝、淘宝、高德等入口导流），是"蓝骑士"精神的承载者。

---

## 一、平台定位（写作用途）

- **外卖赛道的蓝方选手**：以外卖配送为核心，兼有即时零售、到店业务，与美团形成直接竞争
- **用户画像**：18-35岁城市青年，大学生和白领为主，阿里系用户（淘宝/支付宝活跃用户）更容易被导流使用
- **气质标签**：蓝色品牌、阿里系生态、补贴战老兵、校园外卖、蓝骑士文化
- **写作适用**：外卖行业竞争、蓝骑士生活、骑手与算法、平台补贴大战、校园外卖文化、阿里系生态串联、深夜点外卖、商家在两平台之间的博弈

---

## 二、核心功能

### 2.1 外卖配送（核心业务）

| 模式 | 说明 | 写作细节 |
|------|------|---------|
| **饿了么外卖** | 线上下单，蓝骑士配送 | 与美团外卖功能对等 |
| **饿了么星选** | 高品质外卖（类似美团的"品牌馆"） | 筛选高品质餐厅，价格略高 |
| **超市便利** | 日用品即时配送 | 买零食、饮料、日用品 |
| **买药** | O2O药品即时配送 | 深夜买药、应急场景 |
| **跑腿代购** | 代买任意商品 | 跑腿小哥代买指定商品 |

**饿了么外卖流程**（与美团基本一致但有细节差异）：
选餐→领吃货豆/红包→下单→商家接单→蓝骑士取餐→配送→送达→评价

### 2.2 骑手体系——蓝骑士

**蓝骑士 vs 美团黄骑士**（写作核心对比点）：

| 对比项 | 饿了么蓝骑士 | 美团黄骑士 |
|--------|------------|----------|
| **工服颜色** | 蓝色 | 黄色 |
| **骑手类型** | 专送/众包/优选 | 专送/众包/乐跑 |
| **品牌归属** | 阿里本地生活 | 美团 |
| **接单App** | 蜂鸟众包 | 美团骑手 |
| **日常形象** | 蓝色头盔+蓝色工服 | 黄色头盔+黄色工服 |

**蜂鸟配送**：饿了么旗下的即时配送平台，骑手通过"蜂鸟众包"App接单。蜂鸟系统在高峰期的派单算法与美团类似，都会根据骑手位置、订单密度、配送时长等综合计算。

### 2.3 吃货豆与会员体系

- **吃货豆**：饿了么的积分体系，下单可获得吃货豆，可用于兑换红包/优惠券
- **超级会员**：付费会员（类似美团会员），每月有固定数量的无门槛红包
- **签到领红包**：每日签到可获取小额优惠券
- **吃货卡**：连续下单可升级，享受更多权益

### 2.4 阿里生态联动（饿了么的独特优势）

| 入口 | 说明 |
|------|------|
| **支付宝** | 首页直接有饿了么入口，流量巨大 |
| **淘宝** | 淘宝内可直接跳转饿了么 |
| **高德地图** | 搜餐厅可直接跳转饿了么下单 |
| **口碑** | 到店业务已并入饿了么 |

## 四、写作场景模板（可直接嵌入剧情）

### 场景1：蓝骑士的一天
> 早上七点，他换上蓝色工服，打开蜂鸟众包，开始接第一单。早高峰单量少，大多是早餐和咖啡。九点以后单量上来，他挂着三单在写字楼之间穿梭。中午十二点是最疯狂的时候——手机上的订单提示音根本停不下来，他同时带了五份外卖，电梯里和一个美团黄骑士挤在一起，两人对视了一眼，没说话。下午三点单量回落，他在路边喝了口水，看了一下今天的数据：18单，收入97块。晚上还有一波高峰。

### 场景2：校园外卖大战
> 大学食堂门口，左边一排蓝色帐篷是饿了么的，右边一排黄色帐篷是美团的。两家摆摊拉新用户，扫码下载送饮料。饿了么的地推小哥喊："新人首单一元吃炸鸡！"美团的地推立刻接："首单立减二十！"她站在中间犹豫了三秒，最后两边都扫了——手机里同时装了两个外卖App，哪边便宜用哪边。

### 场景3：支付宝里的饿了么
> 她打开支付宝交水电费，首页弹出了一个饿了么红包——满25减8。本来没想点外卖的，但这个力度实在太大了。她点进去，刷了十五分钟，最终下单了一份麻辣烫和一杯奶茶，合计23块——"再加两块钱就满减了"，于是又加了一份鹌鹑蛋。交完水电费以后，她发现自己花了比水电费还多的钱在饿了么上。

### 场景4：蓝黄骑士的街头
> 电动车从巷子里冲出来的时候，他差点撞上对面的美团骑手。两人同时急刹，保温箱里的外卖都往前倾了一下。"看着点路啊！""你才看着点！"骂完以后各自骑走，谁也没时间多停留一秒——他手里还有两单要送，对方手里还有三单。到了同一个写字楼下，两人又在电梯里碰面了。这次谁也没说话，都低头看手机上还有多少时间。

### 场景5：深夜的蜂鸟订单
> 凌晨两点，蜂鸟众包上突然跳出一单——附近三公里，配送费翻倍，备注写着"麻烦帮忙带一包烟"。他犹豫了一下，接了。到了楼下发现是个加班到崩溃的程序员，开门的时候眼睛都是红的。"不好意思啊，这么晚还点外卖。""没事，我也睡不着。"程序员接过袋子，扫码付了款，顺手打赏了五块钱。他骑车回去的路上，城市很安静，只有电动车的嗡嗡声。

## 五、写作避坑

1. **不要只写美团忽视饿了么**——很多作者写外卖场景只提美团，但实际上两家平台在大多数城市共存，很多用户两个都用，商家也通常同时入驻两个平台
2. **蓝骑士和黄骑士的关系不是敌人**——虽然平台之间是竞争关系，但骑手个体之间通常是"各跑各的"，偶尔有交集但不存在系统性对立，不要过度戏剧化
3. **吃货豆是真实存在的**——不要忽略饿了么自己的积分体系，写饿了么场景时提到"吃货豆""超级会员"比写"红包"更有平台辨识度
4. **阿里生态联动是饿了么的特色**——写"从支付宝点外卖""淘宝页面跳转到饿了么"这种场景是饿了么独有的，能和美团场景形成差异
5. **"蜂鸟"是骑手端的名字**——普通用户说"饿了么"，骑手说"跑蜂鸟"，两者不要混用。用户视角和骑手视角用词不同才真实
