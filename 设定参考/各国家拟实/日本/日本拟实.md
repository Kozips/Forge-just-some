# 日本 · 拟实写作参考

> 写日本背景的小说，这些是"不能编"的硬设定。编错了读者一秒出戏。
> **原型：日本国（日本国）**。所有设定基于现实，名称替换为虚构。

---

## 一、国家概况

| 项目 | 内容 | 写作要点 |
|------|------|---------|
| **国名** | 日本国（にほんこく/にっぽんこく） | 官方场合用「日本国」 |
| **首都** | 东京（とうきょう） | 政治、经济、文化中心 |
| **政体** | 君主立宪制（议会内阁制） | 天皇为象征，首相掌实权 |
| **面积** | 约37.8万平方公里 | 岛国，由四大岛+众多小岛组成 |
| **人口** | 约1.25亿（2024年） | 持续减少中，少子老龄化 |
| **语言** | 日语 | 关东/关西方言差异大 |
| **货币** | 日元（円/えん） | 1万円纸币最常用 |
| **时区** | UTC+9 | 无夏令时 |

## 三、人口与社会结构

### 3.1 人口分布

| 地区 | 人口占比 | 特点 |
|------|---------|------|

## 五、文化特征

### 5.1 日本人的行为模式

| 特征 | 表现 | 写作要点 |
|------|------|---------|
| **空気を読む** | 读空气/察言观色 | 不直说，靠暗示 |
| **本音と建前** | 真心话vs场面话 | 人际关系核心 |
| **遠慮** | 客气、推辞 | 拒绝时说「ちょっと...」 |
| **義理** | 人情债 | 社交义务 |
| **恥** | 耻感文化 | 不想给人添麻烦 |
| **集團主義** | 集体主义 | 不出头、随大流 |

| 关键词 | 含义 | 写作场景 |
|--------|------|---------|
| **和** | 和谐 | 人际关系 |

## 七、速查表

| 项目 | 正确写法 | 常见错误 |
|------|---------|---------|
| 自称 | 私（わたし/わたくし）、僕（ぼく）、俺（おれ） | 男性统一用「俺」（不同角色不同） |
| 敬称 | 〜さん、〜くん、〜様、〜先生 | 不要乱用「様」 |
| 鞠躬 | 15°/30°/45°对应不同场合 | 不要所有场合都90° |
| 鞋 | 进室内脱鞋，穿拖鞋 | 不要写穿着鞋进家门 |
| 电车 | 安静、优先席、女性专用车厢 | 不要在电车上大声打电话 |
| 垃圾 | 严格分类 | 不要写随手扔垃圾 |
| 约定 | 守时是基本 | 不要写日本人迟到不当回事 |
