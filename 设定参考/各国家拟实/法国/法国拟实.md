# 法棍国 · 拟实写作参考

> 写法棍国背景的小说，这些是"不能编"的硬设定。编错了读者一秒出戏。
> **原型：法兰西共和国**。所有设定基于现实，名称替换为虚构。

---

## 一、政治体制

### 半总统制（核心设定）

法棍国实行**半总统制**——全球最独特的政体之一：

- **总统**：国家元首，全民直选，任期5年可连任一次。掌握外交、国防大权，可解散议会
- **总理**：由总统任命，负责国内日常政务，对议会负责
- **左右共治**：总统与总理分属对立阵营时，彼此角力——小说中极佳的权力博弈素材

**写作提示**：法棍国政客可以精英化、傲慢，甚至公开蔑视对手——这在该国政坛是常态。

### 行政区划

| 层级 | 数量 | 说明 |
|------|------|------|
| **大区** | 18个 | 最高一级地方行政单位 |
| **省** | 101个 | 每省有省长 |
| **市镇** | 约35,000个 | 市长权力很大 |
| **巴黎** | 特殊 | 分为20个区，社会差异极大 |

**巴黎各区社会差异**：
- 1-4区：时尚和LGBTQ+社区
- 6区：文艺精英区
- 16区：富人区
- 18-20区：移民聚居区
- "住在哪个区"本身就是社会身份标志

### 主要政党（2024-2025格局）

| 光谱 | 政党 | 要点 |
|------|------|------|
| 极左 | 不屈法兰西 | 反紧缩、生态社会主义 |
| 左翼 | 社会党、共产党、绿党 | 传统左翼，影响力衰退 |
| 中间 | 复兴党 | 亲欧、自由主义改革 |
| 右翼 | 共和党 | 传统戴高乐主义 |
| 极右 | 国民联盟 | 反移民、民族主义 |

## 二、军事力量

### 外籍军团（核心设定）

全世界最富传奇色彩的军事单位：

- **成立**：1831年，允许外国人加入
- **现役**：约8,000-9,000人
- **特殊规则**：
  - 加入时使用化名（nom de guerre）
  - 服役满3年可申请法国国籍
  - 训练极为严酷，"不问来路"传统
- **成员**：约140个国家的外国人
- **标志**：白色平顶帽、绿领章

**写作提示**：一个想隐藏过去的人加入军团是经典小说桥段。军团有"荣誉法典"，强调忠诚和战友情谊。

### 核力量

- 海基：4艘战略核潜艇，射程超10,000公里
- 空基：阵风战斗机携带核巡航导弹
- 核按钮**只有总统**有权按下

### 主要装备

| 装备 | 说明 |
|------|------|
| **阵风战斗机** | 多用途四代半战斗机，出口多国 |
| **戴高乐号航母** | 核动力，法国唯一航母 |
| **勒克莱尔坦克** | 第三代主战坦克 |
| **凯旋级核潜艇** | 核威慑核心 |

## 三、司法系统

### 大陆法系发源地

法棍国是**大陆法系的发源地**——1804年拿破仑《民法典》奠定现代民法基础。

### 预审法官（核心设定）

大陆法系最独特的角色：
- 独立司法官，负责重大案件的正式侦查
- 有权搜查、监听、逮捕
- 权力极大且独立，连检察官都无权干涉
- 小说中：一个固执的预审法官追查腐败高官，是极经典的法式司法剧情

### 警察双轨制

| 警种 | 管辖区域 | 上级 |
|------|---------|------|
| **国家警察** | 城市地区 | 内政部 |
| **国家宪兵** | 农村和小城镇 | 国防部 |

两个系统之间存在管辖权之争和竞争关系——小说中可制造有趣的剧情张力。

## 四、教育体系

### 学制

| 阶段 | 年限 | 年龄 | 说明 |
|------|------|------|------|
| 幼儿园 | 3年 | 3-6岁 | 普及率极高 |
| 小学 | 5年 | 6-11岁 | — |
| 初中 | 4年 | 11-15岁 | 毕业获Brevet文凭 |
| 高中 | 3年 | 15-18岁 | 分普通、技术、职业三类 |

### 高中毕业会考（Bac）

- 通过即获高中毕业文凭，同时自动获得进入大学资格
- 分三大类：普通类、技术类、职业类
- 采用持续评估（60%）+ 期末考试（40%）
- 通过率约90%，按等级评定：très bien（优秀）、bien（良好）

### 大学校（Grande École）——精英教育核心

法棍国社会高度由教育出身决定：

| 类型 | 顶尖学校 | 培养方向 |
|------|---------|---------|
| 高等师范 | 巴黎高师 | 学术精英 |
| 工程师学校 | 综合理工 | 工程技术精英 |
| 商学院 | HEC、ESSEC、ESCP | 商业领袖 |
| 行政学院 | INSP（原ENA） | 高级公务员、政客 |
| 政治学院 | 巴黎政治学院 | 政治精英 |

**预科班（Prépa）**：通往大学校的"炼狱之路"——高中毕业后读2年预科班，学习强度极高。"你在哪读的prépa？"是法棍国社交核心问题。

**写作提示**：两个同龄人——一个Polytechnique毕业的工程师和一个普通大学毕业生——在就业、社交、婚恋市场待遇天差地别。

## 五、社会生活

### 住房

- 巴黎房价约每平米10,000欧元以上
- 租房需要完整"dossier"（申请材料）：收入证明、工作合同、身份证件、前房东推荐信
- 收入要求一般是月租金的**3倍**以上
- 需要**担保人**或**租金保险**

**写作提示**：在法棍国小说中，"找房子"是制造焦虑感和展现社会阶层的经典桥段。

### 交通

| 方式 | 说明 |
|------|------|
| **TGV高铁** | 最高试验速度574.8km/h，巴黎-里昂2小时 |
| **巴黎地铁** | 16条线路，300+车站，每日400-500万人次 |
| **RER** | 区域快速铁路，连接市中心和郊区 |
| **Navigo月票** | 约86欧元/月 |

**写作提示**：巴黎地铁每条线路都有"性格"——4号线最拥挤、6号线可看到埃菲尔铁塔、7号线小偷最出名。

### 经济

- GDP全球第7，欧洲第3
- 核心产业：奢侈品（LVMH、开云、爱马仕）、航空航天（空客）、农业（欧洲第一）、旅游、核电（75%电力来自核能）
- 35小时工作制，每年最少5周带薪年假
- 七八月是传统度假季，巴黎空了半城

### LVMH奢侈品帝国

全球最大奢侈品集团，旗下75+品牌：路易威登、迪奥、蒂芙尼、丝芙兰、Loewe、Fendi、纪梵希、赛琳等。

## 六、文化习俗

### 传统节日

| 节日 | 日期 | 习俗 |
|------|------|------|
| **劳动节** | 5月1日 | 赠送铃兰花 |
| **国庆节** | 7月14日 | 阅兵式、消防员舞会、烟火 |
| **诸圣节** | 11月1日 | 去墓地献菊花（不是玫瑰！） |
| **圣诞节** | 12月25日 | 家庭团聚、圣诞大餐 |

### 社交礼仪

**贴面礼（核心设定）**：
- 两人见面和告别时在双颊上各轻触一下
- 贴面次数因地区而异：巴黎2次、南部3-4次、布列塔尼1次
- 初次见面的异性之间、朋友之间都会行贴面礼

**称呼**：
- **"Vous"（您）** vs **"tu"（你）**：社交雷区
- 对不熟的人、长辈、上级必须用vous
- 从vous过渡到tu是关系深化的标志

**其他礼仪**：
- 进入任何场所都要说"Bonjour!"
- 吃面包直接放桌上，用手撕着吃
- 法棍国人喜欢辩论，"唱反调"是智力训练不是冒犯

### 饮食文化（UNESCO非物质文化遗产）

**正餐结构**：

| 顺序 | 说明 |
|------|------|
| 开胃酒 | 饭前饮酒社交 |
| 前菜 | 汤、沙拉、鹅肝酱 |
| 主菜 | 肉/鱼配蔬菜 |
| 奶酪 | 餐后奶酪拼盘（3-5种） |
| 甜点 | 甜点或水果 |
| 咖啡 | 饭后浓缩咖啡 |
| 消化酒 | 白兰地、干邑 |

**面包文化**：
- 法棍面包（baguette）：日常主食，每天购买
- 法律规定只能用面粉、水、酵母和盐

**葡萄酒**：
- 主要产区：波尔多、勃艮第、香槟、罗讷河谷
- 红酒配红肉、白酒配海鲜是基本规则

**奶酪**：
- 超过**1,000种**奶酪
- 奶酪柜台前讨论半小时不是夸张

**咖啡文化**：
- 只喝浓缩咖啡（express）
- 露天座位（terrasse）是标志性景观
- 萨特、波伏瓦、加缪都在咖啡馆写作

**写作提示**：法棍国人每天在食物上花的时间和心思比大多数国家都多。午餐通常是1-2小时，这在职场中是不可压缩的。

### 时尚产业

- 巴黎是**世界时尚之都**
- 顶级品牌：香奈儿、迪奥、路易威登、爱马仕、圣罗兰
- 着装哲学：**"effortless chic"**（不经意的优雅）
- 香水是时尚的一部分——几乎每人都有"签名香"

### 艺术

- **博物馆**：卢浮宫、奥赛博物馆、蓬皮杜中心
- **文学**：雨果、巴尔扎克、福楼拜、普鲁斯特、加缪、萨特
- **电影**：电影诞生地，戛纳电影节
- **音乐**：法语香颂、电子音乐（Daft Punk）、说唱

## 九、写法棍国背景的注意事项

1. **不要让所有角色都像"刻板法国人"**：巴黎人、南部人、布列塔尼人之间差异巨大
2. **不要把巴黎代表整个国家**：外省有完全不同的口音、饮食、节奏
3. **不要写"浪漫巴黎"陈词滥调**：巴黎是脏的、吵的、挤的
4. **不要忽略罢工文化**：每年至少数次全国性大罢工
5. **食物是神圣的**：午餐1-2小时不可压缩，"一起吃饭"是建立关系的起点
6. **贴面礼和称呼要准确**：Vous/tu切换反映关系变化
7. **抱怨是社交艺术**：不是真的不快乐，是文化表达方式
8. **8月巴黎空城**：度假是不可侵犯的权利
