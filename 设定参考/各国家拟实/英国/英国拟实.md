# 绅士国 · 拟实写作参考

> 写绅士国背景的小说，这些是"不能编"的硬设定。编错了读者一秒出戏。
> **原型：大不列颠及北爱尔兰联合王国**。所有设定基于现实，名称替换为虚构。

## 一、政治体制

### 君主立宪制（核心设定）

- **国王**：国家元首，权力主要是象征性和礼仪性的
- 所有国家行为以国王名义进行——"国王陛下的政府"、"国王陛下的反对党"
- 每周三接见首相听取汇报（私密会面，无会议记录）
- **首相**：由议会多数党领袖担任，掌握实际行政权

### 议会两院

| 院 | 成员 | 产生方式 | 职能 |
|------|------|---------|------|
| **上议院（贵族院）** | 约800人 | 终身贵族+世袭贵族+主教 | 审议修改法案 |
| **下议院（平民院）** | 650名议员 | 民选 | 立法权、财政权（实权） |

**首相问答（PMQs）**：每周三中午12:00-12:30，首相与反对党领袖面对面交锋，现场气氛如同角斗场——展现政治人物性格的绝佳场景。

### 四个构成国

| 构成国 | 首府 | 特点 |
|--------|------|------|
| **英兰（英格兰）** | 伦城（伦敦） | 占全国人口84%，政治经济文化中心 |
| **苏兰（苏格兰）** | 爱城（爱丁堡） | 有自己的议会和自治权，有独立运动 |
| **威兰（威尔士）** | 卡迪夫 | 威尔士语活跃，橄榄球比足球更受欢迎 |
| **北兰（北爱尔兰）** | 贝尔法斯特 | 宗教/政治分裂，"麻烦时期"历史创伤 |

### 主要政党

| 政党 | 色彩 | 传统支持者 |
|------|------|-----------|
| **保守党（托利党）** | 蓝色 | 乡绅、商界、中上层阶级 |
| **工党** | 红色 | 工会、工人阶级、城市中产 |
| **自由民主党** | 黄色 | 公民自由、环保、亲欧 |
| **苏格兰民族党** | 黄色 | 主张苏格兰独立 |

## 二、军事力量

### 军种

| 军种 | 特点 | 主要装备 |
|------|------|---------|
| **皇家海军** | 历史最悠久，曾称霸全球 | 伊丽莎白女王级航母、核潜艇 |
| **陆军** | 规模不大但精锐 | 挑战者3型坦克、近卫团 |
| **皇家空军** | 世界首支独立空军 | F-35B、台风战斗机 |

### 核力量——三叉戟（核心设定）

- 4艘前卫级核潜艇，保证至少一艘始终在海上巡逻（365天不间断）
- 核按钮由首相掌控
- **"最后手段信件"**：每任首相上任后手写这封信，密封在潜艇中，仅在政府被摧毁后才能打开——内容从未公开，前任的信会被销毁

### SAS特种空勤团（核心设定）

- 世界上最著名的特种部队之一
- 格言："Who Dares Wins"（勇者胜）
- 选拔训练极其残酷——连续数天在威尔士山区负重行军
- 标志性装备：黑色巴拉克拉瓦头套、MP5冲锋枪
- 1980年伊朗大使馆围攻事件中，SAS突击队在电视直播中从外墙突入，震惊世界
- 成员极为低调，从不公开身份，退役出书需保密审查

### 军衔

三军军衔各有名称，互不通用。军官称呼"Sir"，士兵之间不互称"Sir"。

## 三、司法系统

### 普通法系发源地（核心设定）

- **判例约束**：法官的判决对后续类似案件有约束力
- 法官"造法"——不同于大陆法系的成文法典
- 没有成文宪法，宪制由宪法惯例、成文法、判例法组成

### 警察系统

| 机构 | 职能 |
|------|------|
| **苏格兰场（伦城警察厅）** | 大伦城地区治安 |
| **地方警察局** | 43个，各管各的 |
| **国家犯罪局（NCA）** | 类似FBI |
| **MI5（军情五处）** | 国内安全情报 |
| **MI6（军情六处）** | 对外情报，007原型 |

**关键设定**：英国街头巡警传统上**不配枪**——这在全世界非常独特。警察被称为"Bobby"或"Constable"。

### 法院体系

治安法院 → 刑事法院（Crown Court）→ 高等法院 → 上诉法院 → 最高法院

**法官戴假发**：用马毛制作，刑事案件穿红色法袍，民事穿黑色。

### 陪审团制度

- 刑事案件陪审团由**12人**组成
- 裁决需要一致同意（12:0），法官可接受11:1或10:2
- 陪审员不得透露审议过程——"12个普通人决定一个人命运"的设定极具戏剧张力

## 四、教育体系

### 学制

| 阶段 | 年龄 | 说明 |
|------|------|------|
| **小学** | 5-11岁 | — |
| **中学** | 11-16岁 | 16岁参加GCSE考试 |
| **Sixth Form** | 16-18岁 | 学习A-Level（通常3-4门） |
| **大学本科** | 18岁起 | 一般3年制（苏格兰4年） |

### 公学（核心设定）

**"公学"其实是顶级私立学校**——名称源于历史，非政府出资。

| 公学 | 特点 |
|------|------|
| **伊顿公学** | 最著名，培养20位首相，年学费约5万英镑 |
| **哈罗公学** | 丘吉尔母校 |
| **温彻斯特公学** | 最古老（1382年），学术最强 |

- 寄宿制为主，穿统一校服
- 形成强大的校友网络——"Old Boys' Network"直接影响政商界
- 上公学的人说话有RP口音，形成精英循环

### 牛津与剑桥

| 大学 | 特点 |
|------|------|
| **牛津** | 英语世界最古老（1096年），38个学院，培养30位首相 |
| **剑桥** | 1209年建校，31个学院，培养15位首相 |

- 教学特色：**导师制**——每周与导师一对一深入讨论
- "May Ball"——期末后的奢华通宵舞会
- 牛剑之间有年度赛艇对抗赛

## 五、医疗体系

### NHS（核心设定）

**NHS（全民医疗服务体系）**——绅士国最引以为傲的制度：

- 1948年成立，基于临床需要而非支付能力
- 对合法居民基本**免费**
- 资金来源：国家税收

**看病流程**：先预约全科医生（GP）→ GP诊断 → 必要时转诊专科 → 专科治疗

**现实问题**：
- 非紧急手术等待数周至数月
- GP预约困难——早上8点准时打电话，当天名额几分钟内就满了
- 急诊室等待4-6小时很常见
- 资金长期不足，医护人员短缺

**写作提示**：NHS是英国人的"圣牛"——可以批评效率低，但说要废除就是挑战全国人民底线。"Clap for NHS"疫情期间每晚8点全民鼓掌致敬。

## 六、社会生活

### 阶级制度（核心设定）

绅士国阶级制度是世界上最顽固、最隐蔽的社会分层系统：

| 阶级 | 特征 | 口音 | 教育 |
|------|------|------|------|
| **上层** | 贵族、地主、旧钱 | 极纯RP | 伊顿→牛剑 |
| **上中产** | 专业人士（医生、律师） | 接近RP | 好私校→牛剑 |
| **下中产** | 普通白领、小店主 | 有差别 | 文法学校 |
| **工人阶级** | 蓝领、服务业 | 方言口音浓厚 | 综合学校 |

**阶级的隐形标记**：

1. **口音**：听30秒就能判断阶级
   - RP（女王英语）= 上层和上中产
   - Cockney（伦敦东区）= 工人阶级
   - Scouse（利物浦）、Brummie（伯明翰）、Geordie（纽卡斯尔）各有特点

2. **用词差异**：
   - 上层说"sitting room"，工人说"lounge"
   - 上层说"lavatory/loo"，工人说"toilet"
   - 上层说"pudding"，工人说"dessert"

3. **邮编**：伦敦SW1（白金汉宫附近）、W1（梅菲尔）= 上流；E1（东区）= 工人阶级

### 皇室文化

**主要建筑**：白金汉宫（775个房间）、温莎城堡、肯辛顿宫

**年度活动**：
- 国王生日阅兵（6月）
- 皇家赛马会（6月，着装要求极严）
- 圣诞致辞（12月25日下午3点）
- 皇家花园派对（夏天）

### 酒吧文化（核心设定）

酒吧是绅士国的"第二客厅"：

| 类型 | 特点 |
|------|------|
| **传统酒吧** | 木质吧台、壁炉、手泵啤酒 |
| **酒吧餐厅** | 提供高品质食物 |
| **连锁酒吧** | Wetherspoons，价格便宜 |

**酒吧礼仪**：
- **买酒轮制**：一人买一整轮，下一轮另一人买——不参与轮制是社交灾难
- 在吧台前自觉排队
- 酒保不会主动来为你服务

**酒吧问答（Pub Quiz）**：周二或周三晚上，分组竞赛回答知识问题——英国人社交生活的重要组成部分。

### 足球文化

**英超联赛**：全球最受欢迎的足球联赛，著名俱乐部：曼联、利物浦、切尔西、阿森纳、曼城

- **工人阶级的宗教**：球队代表城市/社区骄傲
- 德比大战是同城球队对决
- 球迷歌曲是流行文化的一部分
- "It's coming home"——充满自嘲和希望的矛盾情感

## 七、文化习俗

### 传统节日

| 节日 | 日期 | 习俗 |
|------|------|------|
| **圣诞节** | 12月25日 | 圣诞拉炮、圣诞布丁、国王致辞 |
| **Boxing Day** | 12月26日 | 购物打折日、看球赛 |
| **篝火之夜** | 11月5日 | 纪念火药阴谋案失败，篝火+烟花 |
| **复活节** | 春分后 | 巧克力蛋、egg rolling |
| **银行假日** | 多个 | 全国性公共假期 |

### 社交礼仪

**排队**：对排队有近乎宗教般的虔诚——插队是比偷窃更严重的社会罪行。

**礼貌与含蓄**：
- "Sorry"是说得最多的词——被人踩了脚也会说"sorry"
- 即使极度不满也会说"That's a bit unfortunate"
- 英式礼貌的反面是**被动攻击**——"With the greatest respect..."通常意味着"你说的是胡扯"

**谈话艺术**：
- 天气是万能开场白
- 自嘲式幽默是社交润滑剂
- "Not bad"可能是最高赞赏的方式
- 不要炫耀——"低调"是美德

### 饮食文化

| 食物 | 说明 |
|------|------|
| **英式早餐** | 培根、香肠、煎蛋、烤豆、吐司、蘑菇、黑布丁 |
| **下午茶** | 三层架：手指三明治→司康饼配凝脂奶油→蛋糕 |
| **炸鱼薯条** | 英国"国菜"，配盐和醋 |
| **周日烤肉** | 烤牛肉/羊腿配约克郡布丁 |
| **Haggis** | 苏格兰国菜，羊杂碎 |

**茶文化**：
- 每天喝5-6杯茶是常态
- "Cuppa"（一杯茶）是日常用语
- "Builder's Tea"：浓茶加大量牛奶和糖

### 文学

- **莎士比亚**：英语世界最伟大的剧作家，创造了1,700+单词
- **狄更斯**：维多利亚时期伦敦全景
- **简·奥斯汀**：乡绅阶层精确刻画
- **阿加莎·克里斯蒂**：推理小说女王
- **J.K.罗琳**：《哈利·波特》系列
- **柯南·道尔**：福尔摩斯系列

### 英式幽默

1. **自嘲**：以贬低自己为荣，失败比成功更好笑
2. **干幽默/冷幽默**：面无表情说荒谬的话，听众需几秒才反应过来
3. **荒诞幽默**：Monty Python式无厘头
4. **含蓄下流笑话**：表面无害但暗含双关
5. **黑色幽默**：对死亡、疾病、灾难开玩笑

## 十、写绅士国背景的注意事项

1. **口音系统是灵魂**：阶级通过口音体现，听30秒就能判断
2. **不要用美式英语**：apartment→flat, elevator→lift, fries→chips
3. **天气是永恒话题**：真正的英国人不打伞
4. **排队要精确描写**：插队是严重社会罪行
5. **茶是万能解药**：所有危机第一反应是"Shall I put the kettle on?"
6. **酒吧不是夜店**：是日常社交场所
7. **NHS看病先看GP**：不能直接去医院
8. **交通靠左**：车辆靠左行驶
9. **英国不是"一个"国家**：苏格兰、威尔士、北爱各有独立文化认同
10. **阶级问题不能回避**：渗透在生活方方面面
