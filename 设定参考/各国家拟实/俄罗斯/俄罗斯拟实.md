# 俄罗斯 · 拟实写作参考

> 写俄罗斯背景的小说，这些是"不能编"的硬设定。编错了读者一秒出戏。
> **原型：俄罗斯联邦（Российская Федерация）**。所有设定基于现实，名称替换为虚构。

---

## 七、速查表

| 项目 | 正确写法 | 常见错误 |
|------|---------|---------|
| 姓名 | 名+父称+姓（Иван Иванович Иванов） | 不要只写名+姓 |
| 称呼 | 对长辈/上级用名+父称 | 不要对所有人直呼其名 |
| 问候 | 握手（男性）、拥抱（亲密） | 不要写男女初次见面就拥抱 |
| 做客 | 带礼物（花/酒/巧克力） | 不要空手上门 |
| 饮酒 | 伏特加要一口干 | 不要写小口抿伏特加 |
| 花 | 奇数枝（丧事用偶数） | 不要送偶数花给活人 |
| 鞋 | 进室内换拖鞋 | 不要穿着鞋进家门 |
| 招财猫 | 不存在 | 不要写俄罗斯人用招财猫 |
