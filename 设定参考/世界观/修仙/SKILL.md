# 修仙世界观速查

## 一、修仙境界体系

| 境界 | 特征 | 寿命 | 突破代价 | 战力标志 |
|------|------|------|----------|----------|
| 练气 | 引气入体，感知灵气 | 100年 | 灵根觉醒或丹药辅助 | 力敌十人 |
| 筑基 | 凝聚灵力，御剑飞行 | 200年 | 筑基丹+闭关+领悟 | 碎石裂壁 |
| 金丹 | 丹田结丹，灵力质变 | 500年 | 天材地宝+心魔考验 | 移山倒海 |
| 元婴 | 元神出窍，神识覆盖 | 1000年 | 雷劫+心魔劫 | 百里杀敌 |
| 化神 | 天人合一，操控天地灵气 | 2000年 | 天道之眼+灵魂重塑 | 风云变色 |
| 渡劫 | 渡天劫，飞升在望 | 5000年 | 九道天劫+天诛 | 灭国之力 |

每个大境界分初期/中期/后期/巅峰四小阶。高一个大境界 ≈ 碾压级差距（约10倍）。

## 二、天道规则（修仙独有设定）

### 因果报应
- 杀孽过重者，天劫威力倍增
- 行善积德者，气运加身，突破顺利
- 恩怨纠缠形成"因果线"，不还清因果难以飞升
- 元婴以上修士可"看见"因果线，飞升前必须了结所有因果

### 天劫

| 天劫 | 触发 | 威力 | 生存率 |
|------|------|------|--------|
| 小天劫 | 金丹突破 | 三道雷 | 70% |
| 中天劫 | 元婴突破 | 六道雷 | 50% |
| 大天劫 | 化神突破 | 九道雷 | 30% |
| 天诛 | 渡劫飞升 | 八十一道 | 10% |

### 气运
- 气运昌隆：奇遇不断、逢凶化吉
- 气运衰败：诸事不顺、走火入魔概率增加
- 气运可转移/掠夺（邪术），但有反噬

### 天道之眼
- 化神期以上修士行事过激时降临审视
- 触发条件：大规模屠杀凡人、毁灭灵脉
- 效果：天劫威力翻三倍，几乎无法解除

## 三、正道宗门类型

| 类型 | 核心特色 | 战斗风格 | 弱点 |
|------|----------|----------|------|
| 剑修宗门 | 以剑入道，攻击力最强 | 凌厉、爆发 | 防御薄弱 |
| 丹修宗门 | 炼丹为本，丹药贸易 | 辅助、恢复 | 近战弱 |
| 符修宗门 | 符箓攻击，远程消耗 | 远程、控制 | 依赖符箓 |
| 阵修宗门 | 阵法防御，守势为主 | 防御、困敌 | 移动性差 |
| 器修宗门 | 炼器为本，法宝众多 | 法宝攻击 | 依赖法宝 |
| 体修宗门 | 以身入道，肉身无敌 | 近战、肉搏 | 法术弱 |
| 魂修宗门 | 修炼灵魂，神识攻击 | 神识攻击、幻术 | 肉身脆弱 |

宗门机构：执法殿、丹药殿、器殿、藏经阁、任务殿、外事殿、护山大阵管理处。

## 四、魔道势力

| 势力 | 核心功法 | 特色 |
|------|----------|------|
| 天魔宗 | 天魔功 | 最大魔宗 |
| 血殿 | 血煞功 | 以血修炼，残忍嗜杀 |
| 鬼谷 | 御鬼术 | 驱使鬼物 |
| 魔渊 | 混沌魔功 | 坠魔修士聚集 |
| 邪月教 | 邪月心经 | 擅长幻术魅惑 |
| 噬魂殿 | 噬魂大法 | 吞噬灵魂 |

魔道特征：进境快但走火入魔风险高；天劫威力比正道强50%-100%；大多数有"以杀证道"倾向。

## 五、核心冲突模板

### 境界压制（底层冲突动力）
- 灵压释放：高境界修士以威压压制低境界
- 以势压人：散修遇到宗门弟子被欺压
- 强买强卖：高境界强行交易
- 抢夺机缘：截胡秘境、灵草
- 同境界内初期vs巅峰差距约3倍，特殊体质可弥补1-2小阶

### 心魔与执念
- 杀戮心魔：杀人过多 → 幻觉暴躁 → 走火入魔
- 情爱心魔：情感纠葛 → 执念疯狂 → 境界停滞
- 执念心魔：未完成心愿 → 偏执固执 → 心境崩塌
- 恐惧心魔：创伤记忆 → 畏缩逃避 → 突破失败

### 因果纠缠
| 类型 | 了结方式 | 不了结后果 |
|------|----------|------------|
| 恩情因果 | 报恩 | 天道不允许飞升 |
| 仇恨因果 | 复仇或放下 | 心魔滋生 |
| 情爱因果 | 结合或斩断 | 境界停滞 |
| 杀孽因果 | 行善积德抵消 | 天劫加倍 |

### 飞升之争
- 飞升条件：渡劫巅峰 + 了结所有因果 + 渡天劫 + 心境圆满
- 冲突点：飞升资源争夺、护道者被截杀、飞升名额限制、飞升通道被破坏

## 六、禁忌规则

### 天道禁忌（触之必死）

| 禁忌 | 后果 |
|------|------|
| 篡改天道 | 天诛，形神俱灭 |
| 逆天改命 | 因果反噬，寿元大减 |
| 献祭苍生 | 天道之眼永久标记 |
| 窃取天道权柄 | 天劫永无止境 |
| 毁灭灵脉 | 业力缠身，天劫加倍 |
| 大规模屠杀凡人 | 天劫威力翻三倍 |

### 修仙禁忌（修炼风险）

| 禁忌 | 后果 |
|------|------|
| 走火入魔 | 修为倒退，沦为废人或魔修 |
| 根基受损 | 境界永远无法提升 |
| 丹田破碎 | 彻底沦为废人 |
| 灵根断裂 | 无法修炼 |
| 灵魂碎裂 | 痴呆、死亡或被夺舍 |

走火入魔征兆：灵力运转刺痛、修炼时幻觉、性情大变、眼中出现血丝或黑气。

### 社会禁忌

| 禁忌 | 后果 |
|------|------|
| 以下犯上 | 被教训、废除修为、处死 |
| 背叛宗门 | 全界通缉、废除修为 |
| 泄露功法 | 全界通缉、追杀 |
| 欺师灭祖 | 全界唾弃、追杀 |

### 力量禁忌
禁忌功法、禁术、邪术、夺舍、血祭、魔化 —— 使用者被视为异端，走火入魔+天劫加倍。

## 七、职业体系

### 修炼流派

| 流派 | 核心 | 优势 | 劣势 |
|------|------|------|------|
| 剑修 | 以剑入道 | 攻击最强、速度极快 | 防御薄弱 |
| 丹修 | 以丹入道 | 丹药自给自足 | 战斗力弱 |
| 符修 | 以符入道 | 远程控制强 | 依赖符箓储备 |
| 阵修 | 以阵入道 | 防御无敌 | 布阵需时间 |
| 器修 | 以器入道 | 法宝多样 | 依赖法宝品质 |
| 体修 | 以身入道 | 近战无敌 | 法术弱 |
| 魂修 | 以魂入道 | 神识攻击 | 肉身脆弱 |

### 炼丹师等级

| 等级 | 可炼丹药 | 对应修为 |
|------|----------|----------|
| 初级 | 一品 | 练气-筑基 |
| 中级 | 二品 | 金丹 |
| 高级 | 三品 | 元婴 |
| 大炼丹师 | 四品 | 化神 |
| 宗师级 | 五品 | 渡劫 |

辅助职业：炼丹师、炼器师、阵法师、符箓师、灵植师、鉴定师、灵厨师。

特殊职业：赏金猎人、秘境探索者、宗门执法者、灵脉勘探者。

## 八、写作注意事项

- 境界压制是底层冲突引擎 —— 散修vs宗门弟子、低辈弟子vs前辈的不平等最能引发读者共情。
- 心魔考验是角色内心戏的绝佳载体 —— 金丹初试、元婴心魔劫、化神灵魂重塑，每个阶段都可展开。
- 因果线让"恩将仇报"或"以德报怨"有了具象化表达 —— 元婴修士看见因果线，知道必须报恩却心有不甘。
- 魔道修士天劫更强 —— 这不是bug是feature，给反派一个"走投无路只能更黑化"的逻辑闭环。
- 气运可掠夺 —— "夺人气运"是邪术也是好冲突：主角被人夺了气运怎么办？
- 飞升名额限制天然制造零和博弈 —— 同时代两个天才只能飞升一个。
