## 六、职业选择建议

### 按天赋选择

| 天赋类型 | 推荐职业 | 理由 |
|----------|----------|------|
| 体能强 | 战士/格斗家 | 发挥体能优势 |
| 观察力强 | 猎人/侦察兵 | 发挥观察优势 |
| 动手能力强 | 机械师/电工 | 发挥动手优势 |
| 医学天赋 | 医生 | 发挥医学优势 |
| 社交能力强 | 交易员 | 发挥社交优势 |

### 按性格选择

| 性格类型 | 推荐职业 | 理由 |
|----------|----------|------|
| 好战 | 战士/格斗家 | 喜欢战斗 |
| 谨慎 | 医生/建筑师 | 需要细心 |
| 好奇 | 拾荒者/侦察兵 | 喜欢探索 |
| 社交 | 交易员 | 喜欢社交 |
| 独立 | 猎人/狙击手 | 喜欢独立 |

## 七、职业发展路线

### 战士路线

```
新手战士 → 资深战士 → 精英战士 → 战士大师 → 战士传说
```

### 猎人路线

```
新手猎人 → 资深猎人 → 精英猎人 → 猎人大师 → 猎人传说
```

### 机械师路线

```
学徒机械师 → 机械师 → 高级机械师 → 机械大师 → 机械传说
```

### 医生路线

```
学徒医生 → 医生 → 高级医生 → 医学大师 → 医学传说
```
