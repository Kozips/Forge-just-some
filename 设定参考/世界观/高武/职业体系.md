# 高武世界 - 职业体系

## 一、战斗类职业

### 剑客

| 要素 | 说明 |
|------|------|
| 核心能力 | 剑法精通、剑气攻击、剑意领悟 |
| 修炼要求 | 悟性极高、剑感强 |
| 战斗风格 | 灵活多变、一剑破万法 |
| 代表门派 | 华山派、峨眉派 |
| 职业优势 | 攻击力强、速度快 |
| 职业劣势 | 防御较弱 |

### 刀客

| 要素 | 说明 |
|------|------|
| 核心能力 | 刀法霸道、刀气纵横、刀意凶猛 |
| 修炼要求 | 力量强大、意志坚定 |
| 战斗风格 | 大开大合、霸道无比 |
| 代表门派 | 无特定门派 |
| 职业优势 | 攻击力极强、气势压制 |
| 职业劣势 | 速度较慢 |

### 拳师

| 要素 | 说明 |
|------|------|
| 核心能力 | 拳法精通、拳意领悟、肉身强横 |
| 修炼要求 | 肉身天赋、意志坚定 |
| 战斗风格 | 近身肉搏、刚猛无比 |
| 代表门派 | 少林寺、丐帮 |
| 职业优势 | 近战无敌、肉身强横 |
| 职业劣势 | 远程较弱 |

### 枪客

| 要素 | 说明 |
|------|------|
| 核心能力 | 枪法精通、枪意领悟、穿透力强 |
| 修炼要求 | 力量强大、眼力好 |
| 战斗风格 | 远程攻击、穿透力强 |
| 代表门派 | 杨家枪 |
| 职业优势 | 攻击距离远、穿透力强 |
| 职业劣势 | 近战较弱 |

## 六、职业选择建议

### 按天赋选择

| 天赋类型 | 推荐职业 | 理由 |
|----------|----------|------|
| 剑感强 | 剑客 | 发挥剑感优势 |
| 力量大 | 刀客/拳师 | 发挥力量优势 |
| 身体灵活 | 暗器师/杀手 | 发挥灵活性优势 |
| 悟性高 | 阵法师/隐士 | 发挥悟性优势 |
| 医术天赋 | 医者 | 发挥医术优势 |

### 按性格选择

| 性格类型 | 推荐职业 | 理由 |
|----------|----------|------|
| 好战 | 剑客/刀客 | 喜欢战斗 |
| 谨慎 | 医者/阵法师 | 需要细心 |
| 好奇 | 游侠/探子 | 喜欢探索 |
| 冷酷 | 杀手 | 适合暗杀 |
| 淡泊 | 隐士 | 适合隐居 |

## 七、职业发展路线

### 剑客路线

```
剑徒 → 剑士 → 剑师 → 剑宗 → 剑王 → 剑圣 → 剑帝
```

### 刀客路线

```
刀徒 → 刀士 → 刀师 → 刀宗 → 刀王 → 刀圣 → 刀帝
```

### 拳师路线

```
拳徒 → 拳士 → 拳师 → 拳宗 → 拳王 → 拳圣 → 拳帝
```

### 医者路线

```
药童 → 医生 → 医师 → 医宗 → 医王 → 医圣
```
