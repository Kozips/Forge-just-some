# 废土末日 · 势力结构

> 没有法律，只有力量和利益。

---

## 一、拾荒者

**特征**：独行或2-3人小队，居无定所，靠搜刮废墟生存

| 优势 | 劣势 |
|------|------|
| 灵活机动，目标小 | 无法积累资源 |
| 熟悉地形，信息灵通 | 缺乏医疗和重火力 |
| 不受势力约束 | 被所有人视为软目标 |

- **装备**：背包、撬棍、手电、简易防毒面具、匕首
- **生存率**：平均存活期2年
- **行为模式**：白天搜刮、夜间转移、避免冲突、见利忘义

---

## 二、定居点

**特征**：固定聚居地，有围墙/防御工事，自给自足型社区

| 类型 | 人口 | 特征 | 代表 |
|------|------|------|------|
| 农业聚落 | 50-200 | 靠种植和养殖，相对和平 | "绿洲镇" |
| 工业聚落 | 100-500 | 有发电机/净水厂，技术型 | "铁堡" |
| 贸易聚落 | 200-1000 | 中立交易站，货币流通 | "十字路口" |
| 军事堡垒 | 50-300 | 旧军事基地改建，重武装 | "雷区" |

| 优势 | 劣势 |
|------|------|
| 资源集中、有防御 | 目标大、易被盯上 |
| 可发展技术和贸易 | 内部权力斗争 |
| 医疗和教育相对完善 | 人口压力、资源消耗快 |

---

## 三、游牧部落

**特征**：以家族/血缘为纽带，骑乘变异兽迁徙，季节性营地

| 优势 | 劣势 |
|------|------|
| 机动性强，难以追踪 | 无法建造永久设施 |
| 精通野外生存 | 资源储备有限 |
| 战斗经验丰富 | 人口增长困难 |

- **规模**：30-150人/部落
- **生存方式**：狩猎采集 + 劫掠 + 小规模贸易
- **文化**：图腾信仰、口述历史、成年礼（独自猎杀变异兽）
- **弱点**：牲畜疫病可摧毁整个部落

---

## 四、军阀势力

**特征**：以武力控制区域，强制征税/征兵，独裁统治

| 优势 | 劣势 |
|------|------|
| 火力强大、纪律严明 | 依赖暴力维持，内部不稳 |
| 资源掠夺效率高 | 统治成本高 |
| 可迅速扩张 | 领袖死后易分裂 |

- **规模**：500-5000人
- **控制区域**：以军事据点为中心，辐射半径50-200公里
- **经济模式**：征收保护费 + 强制劳动 + 奴隶贸易
- **内部结构**：军阀 → 军官 → 士兵 → 平民 → 奴隶

---

## 五、宗教团体

**特征**：以信仰凝聚人心，提供精神慰藉，有组织的扩张

| 类型 | 信仰核心 | 特征 |
|------|---------|------|
| 末日教派 | "审判日已过，选民将重生" | 极端、排外、自我牺牲 |
| 科学复兴会 | "重建文明，复兴科技" | 理性、收集旧世界技术 |
| 自然崇拜 | "大地在惩罚人类" | 环保、反对挖掘旧世界 |
| 机械神教 | "机器是神的化身" | 维护AI/机器人、赛博风格 |

| 优势 | 劣势 |
|------|------|
| 凝聚力强、忠诚度高 | 教条限制行动 |
| 有组织的资源分配 | 与其他信仰冲突 |
| 提供心理慰藉 | 极端化风险 |

---

## 六、科技残余势力

### 旧世界军人后裔

- **特征**：保留军事训练体系，维护旧世界装备
- **优势**：纪律性强、装备精良、战术素养高
- **劣势**：人口稀少、装备老化无法补充
- **典型**：核潜艇基地幸存者、地下指挥所驻军

### 科学家后裔

- **特征**：地下实验室封闭生存，保留部分科技知识
- **优势**：可修复旧世界设备、制造药品
- **劣势**：缺乏战斗力、依赖外界物资
- **典型**：地下种子库守卫、生物实验室团队

## 八、势力规模与控制区域

| 势力类型 | 典型人口 | 控制半径 | 据点数量 | 核心资产 |
|---------|---------|---------|---------|---------|
| 拾荒者 | 1-5人 | 无固定 | 0 | 个人技能 |
| 定居点 | 50-1000 | 5-20公里 | 1 | 围墙/水源/农田 |
| 游牧部落 | 30-150 | 迁徙路线200km | 季节性营地 | 变异兽群 |
| 军阀 | 500-5000 | 50-200公里 | 5-20 | 重武器/弹药库 |
| 宗教团体 | 100-3000 | 10-100公里 | 2-10 | 信众/圣地 |
| 科技残余 | 20-200 | <5公里 | 1 | 实验室/设备 |

> 势力之间没有永恒的盟友，只有永恒的利益。资源耗尽时，任何同盟都会瓦解。
