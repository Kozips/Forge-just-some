# 仙侠世界 - 职业体系

## 一、战斗类职业

### 剑修

| 要素 | 说明 |
|------|------|
| 核心能力 | 御剑飞行、剑气攻击、剑意领悟 |
| 修炼要求 | 天灵根最佳、悟性极高 |
| 战斗风格 | 高攻高敏、一剑破万法 |
| 代表宗门 | 天剑宗、青云门 |
| 职业优势 | 攻击力最强、速度最快 |
| 职业劣势 | 防御较弱、消耗大 |

### 法修

| 要素 | 说明 |
|------|------|
| 核心能力 | 五行术法、天地法相、法力深厚 |
| 修炼要求 | 灵根属性匹配 |
| 战斗风格 | 远程攻击、范围伤害 |
| 代表宗门 | 玄天宗、逍遥派 |
| 职业优势 | 攻击范围大、手段多变 |
| 职业劣势 | 近战较弱、施法有间隙 |

### 体修

| 要素 | 说明 |
|------|------|
| 核心能力 | 肉身成圣、不坏金身、力大无穷 |
| 修炼要求 | 肉身天赋、意志坚定 |
| 战斗风格 | 近战肉搏、防御极高 |
| 代表宗门 | 武当山、少林寺 |
| 职业优势 | 防御最强、近战无敌 |
| 职业劣势 | 速度较慢、手段单一 |

### 魔修

| 要素 | 说明 |
|------|------|
| 核心能力 | 魔功诡异、嗜血狂暴、以战养战 |
| 修炼要求 | 心性坚韧、不惧杀戮 |
| 战斗风格 | 诡异莫测、杀戮果断 |
| 代表势力 | 血魔宗、天魔宫 |
| 职业优势 | 成长快、手段诡异 |
| 职业劣势 | 易入魔、被正道追杀 |

## 三、特殊职业

### 医修

| 要素 | 说明 |
|------|------|
| 核心能力 | 治疗伤势、解毒祛病、续命延寿 |
| 修炼要求 | 木灵根最佳、医术天赋 |
| 职业优势 | 各方势力争抢、地位超然 |
| 职业劣势 | 战斗力较弱、需要保护 |

### 驯兽师

| 要素 | 说明 |
|------|------|
| 核心能力 | 驯服妖兽、建立契约、妖兽进化 |
| 修炼要求 | 精神力强大、亲和力高 |
| 职业优势 | 战斗有帮手、可骑乘飞行 |
| 职业劣势 | 依赖妖兽、自身较弱 |

### 占卜师

| 要素 | 说明 |
|------|------|
| 核心能力 | 预测未来、趋吉避凶、推演天机 |
| 修炼要求 | 天眼通、灵感强 |
| 职业优势 | 知晓天机、可避灾祸 |
| 职业劣势 | 泄露天机有反噬、算不准自己 |

### 炼魂师

| 要素 | 说明 |
|------|------|
| 核心能力 | 操控魂魄、炼制魂器、灵魂攻击 |
| 修炼要求 | 精神力极强、心性坚定 |
| 职业优势 | 攻击手段诡异、防不胜防 |
| 职业劣势 | 被视为邪道、易被反噬 |

## 四、凡人职业

### 与修士相关

| 职业 | 与修士关系 | 发展前景 |
|------|------------|----------|
| 药农 | 提供灵药 | 可被宗门招募 |
| 矿工 | 挖掘灵矿 | 危险但收入高 |
| 向导 | 引路带路 | 熟悉地形 |
| 商人 | 交易物资 | 可积累财富 |
| 仆从 | 服务修士 | 有机会修炼 |

## 五、职业选择建议

### 按天赋选择

| 天赋类型 | 推荐职业 | 理由 |
|----------|----------|------|
| 天灵根 | 剑修/法修 | 修炼速度快 |
| 混沌体 | 法修 | 可融合万法 |
| 肉身强 | 体修 | 发挥肉身优势 |
| 悟性高 | 丹师/器师 | 学习能力强 |
| 精神力强 | 占卜师/炼魂师 | 精神力优势 |

### 按性格选择

| 性格类型 | 推荐职业 | 理由 |
|----------|----------|------|
| 好战 | 剑修/体修 | 战斗欲望强 |
| 谨慎 | 丹师/阵师 | 需要耐心细致 |
| 好奇 | 符师/器师 | 喜欢研究创造 |
| 野心大 | 魔修 | 快速提升实力 |
| 善良 | 医修 | 救死扶伤 |
