5. 知道何时该跑

## 五、势力关系速查

### 正魔对立

```
正道 ←→ 魔道（根本对立）
正道 ←→ 散修（中立偏正）
魔道 ←→ 散修（中立偏魔）
正道 ←→ 妖族（视情况）
魔道 ←→ 妖族（时有合作）
```

### 宗门关系

| 关系类型 | 特征 | 举例 |
|----------|------|------|
| 盟友 | 共同御敌 | 天剑宗-玄天宗 |
| 敌对 | 世仇宿敌 | 正道-魔道 |
| 中立 | 互不干涉 | 大多数宗门 |
| 依附 | 小宗依附大宗 | 三流宗门 |
| 联姻 | 结合两派 | 世家联姻 |

## 六、势力决策机制

### 宗门决策

| 决策层级 | 决策者 | 决策范围 |
|----------|--------|----------|
| 宗门大事 | 太上长老 | 宗门存亡 |
| 日常事务 | 掌门 | 宗门发展 |
| 弟子事务 | 长老 | 弟子管理 |
| 外交事务 | 外门长老 | 对外关系 |

### 正道联盟决策

- 大事由各宗掌门共同商议
- 紧急情况由轮值主席决定
- 战时由最强者统一指挥
- 重大决策需七成以上同意
