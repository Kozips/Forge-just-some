# 异世界 · 职业体系

---

## 一、战斗职业

| 职业 | 定位 | 主武器 | 核心能力 | 转职条件 |
|------|------|--------|----------|----------|
| 战士 | 坦克/近战 | 剑/斧/锤 | 高HP/防御，嘲讽 | 初始职业 |
| 骑士 | 坦克/辅助 | 剑+盾 | 守护光环，骑乘战斗 | 战士Lv20+贵族推荐 |
| 弓箭手 | 远程输出 | 弓/弩 | 远程精准，陷阱 | 初始职业 |
| 刺客 | 暗杀/爆发 | 匕首/短剑 | 潜行，背刺暴击 | 弓箭手Lv15+盗贼公会认可 |
| 狂战士 | 极限输出 | 双手武器 | 狂暴化，HP越低攻击越高 | 战士Lv25+狂暴技能觉醒 |

**战士技能树**：
```
基础剑术 → 盾击 → 嘲讽 → 铁壁 → 守护者
         → 重击 → 战吼 → 狂暴 → 狂战士
```

## 二、魔法职业

| 职业 | 定位 | 主属性 | 核心能力 | 转职条件 |
|------|------|--------|----------|----------|
| 法师 | 魔法输出 | 智力 | 元素魔法，大范围攻击 | 初始职业 |
| 牧师 | 治疗/辅助 | 信仰 | 治疗术，净化，复活 | 光明教会入教 |
| 召唤师 | 召唤/控制 | 精神 | 召唤魔物/精灵协战 | 法师Lv20+契约技能 |
| 元素使 | 元素输出 | 智力 | 单元素精通，元素融合 | 法师Lv25+单元素大师 |
| 死灵法师 | 诅咒/召唤 | 智力+暗 | 亡灵召唤，诅咒 | 法师Lv30+禁忌知识 |

**法师技能树**：
```
魔力感知 → 初级元素 → 中级元素 → 高级元素 → 元素使
                        → 魔法阵 → 召唤术 → 召唤师
                        → 暗知识 → 诅咒术 → 死灵法师
```

## 七、职业等级与转职路线

### 等级体系

| 阶段 | 等级范围 | 称号 | 对应冒险者等级 |
|------|----------|------|----------------|
| 初期 | Lv1-20 | 见习 | E-D |
| 中期 | Lv21-40 | 正式 | C-B |
| 后期 | Lv41-60 | 上级 | A |
| 顶级 | Lv61-80 | 大师 | S |
| 传说 | Lv81-100 | 传说 | S以上 |

### 主要转职路线图

```
战士 → 骑士 → 圣骑士（光属性）
     → 狂战士 → 战神（传说）
     → 武斗家 → 格斗大师

法师 → 元素使 → 大魔导师（传说）
     → 召唤师 → 精灵王（传说）
     → 死灵法师 → 不死王（传说）

弓箭手 → 刺客 → 暗影大师
       → 猎人 → 魔物猎人（传说）
       → 游侠 → 森林守护者

牧师 → 主教 → 教皇
     → 圣者 → 神使（传说）
```

## 八、职业技能获取

| 技能类型 | 获取方式 | 备注 |
|----------|----------|------|
| 基础技能 | 职业自带 | Lv1即可使用 |
| 等级技能 | 升级自动习得 | 每10级获得新技能 |
| 导师技能 | 向NPC学习 | 需付费或完成任务 |
| 卷轴技能 | 使用技能卷轴 | 一次性消耗品 |
| 觉醒技能 | 特定条件触发 | 极稀有，改变职业上限 |
| 穿越者技能 | 前世记忆觉醒 | 仅穿越者可获得 |
