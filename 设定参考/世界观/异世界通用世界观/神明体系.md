# 异世界通用世界观 · 神明体系

> 参考轻小说公共想象（异世界自查.md）中的神明/高位存在设定，整合为可直接使用的神明体系框架。

---

## 一、神明的本质与分类

异世界的神明不一定非得叫"神"——准确说是"高位存在"。他们能做到凡人做不到的事，从人类视角看就与神明无异。

### 按来源分类

| 类型 | 说明 | 代表 |
|------|------|------|
| **创世神** | 世界的创造者，通常已沉睡或退居幕后 | 光之神阿尔特、暗之神诺克斯 |
| **元素神** | 对应自然元素的神明，掌管具体领域 | 火神、水神、风神、大地女神 |
| **职能神** | 对应社会/生活各方面的神明 | 商业之神、武神、知识之神、契约之神 |
| **高位种族** | 进化到神明级别的生命体，维护宇宙秩序 | 被异世界人视为神明但本人可能不自知 |
| **智慧生命体** | 自古存在的古老存在，能力超乎理解 | 世界管理者、时空管理者 |

### 众神制 vs 一神制

| 体制 | 特点 | 适合题材 |
|------|------|----------|
| **一神制** | 唯一神创造一切，教会权力大 | 宗教压迫、信仰冲突 |
| **多神制** | 创世神领导下，各元素神/职能神分工 | 神话史诗、神代战争 |
| **众神制** | 万物皆有神，信仰之物皆可成神 | 轻喜剧、日常、万物皆可娘化 |

## 二、神明在故事中的作用

### 核心功能

1. **操作穿越** — 引导角色转生/转移到异世界
2. **赋予加护** — 给穿越者作弊技能/祝福/神装
3. **提供动因** — 给主角任务（讨伐魔王/拯救世界/带来革新）
4. **维护秩序** — 通过神罚/契约/规则约束世界运转

### 常见穿越理由（由神明提供）

| 理由 | 说明 | 例子 |
|------|------|------|
| 回报善行 | 死前做了好事，神给第二次机会 | 善人转生到和平世界 |
| 补偿失误 | 神的错误导致主角死亡，给补偿 | 神手滑了，赐你作弊技能 |
| 委托任务 | 神需要一个"外来者"改变停滞的世界 | 异世界需要新知识新血液 |
| 对抗魔王 | 经典勇者召唤 | 神谕要求召唤异世界勇者 |
| 意外发生 | 死亡超出世界运行预期 | 被动卷入，没理由 |

### 加护类型（穿越者专属）

| 加护 | 说明 | 常见表现 |
|------|------|----------|
| 异世界语言理解 | 听说读写无障碍 | 最基础的生存保障 |
| 物品箱 | 无限容量空间收纳 | 继承自游戏背包设计 |
| 状态栏可见 | 可查看自身能力数值 | 游戏化的异世界标配 |
| 作弊技能 | 超出常理的特殊能力 | 只要睡觉就升级/杀死一切/编辑技能 |
| 属性异常 | 超高能力值/成长率 | 魔力是普通人的100倍 |
| 鉴定能力 | 可查看物品/人物信息 | 战斗判断、宝物鉴定 |

## 三、神明人格化（去神性化写法）

> 将神明人格化 = 对神明"去神性化"。神明显露随性、任性、欲求 = 角色化。这是让神明不沦为工具人的关键。

### 神明的常见性格

| 性格类型 | 说明 | 写法示例 |
|----------|------|----------|
| 慈祥老者 | 温和引导，偶尔幽默 | "我看你在那边活得挺辛苦的，要不去另一个世界试试？" |
| 腹黑萝莉 | 外表可爱内心算计 | "嘻嘻，给你个好东西哦～（附带坑爹条件）" |
| 冷漠机械 | 只管执行规则，没有感情 | "编号2047，转生程序启动。目标世界：盖亚。附加技能：随机。" |
| 中二少年 | 自称最强，实则半吊子 | "吾乃掌管万物的至高之神——诶？你说这个技能太弱了？" |
| 职场打工人 | 神明也有KPI和上司 | "不好意思啊，今年的穿越指标还差一个，你来凑个数？" |

## 四、教会与神明的关系

### 核心矛盾

教会有时并不总能反映神明意志——这是一个天然的戏剧性来源。

| 状态 | 说明 | 可写冲突 |
|------|------|----------|
| 教会忠实反映神意 | 教会与神明一致 | 教会=正义，但正义可能很烦人 |
| 教会假借神意 | 教会利用神的名义谋私利 | 腐败教会 vs 真正的信仰 |
| 教会与神明分离 | 神已沉睡/不关心，教会自行其是 | 教会内部分裂、改革派vs保守派 |
| 教会人族至上 | 借神之名压迫其他种族 | 种族歧视、宗教战争 |

### 教会的常见功能

| 功能 | 说明 |
|------|------|
| 垄断治愈魔法 | 光属性治愈术由教会独占，免费或低价治疗平民以聚信仰 |
| 发放身份证 | 公会卡之外的身份证明，教会信徒专属 |
| 建立孤儿院 | 常是主角行善的契机，也可能被教会"利用" |
| 发动圣战 | 以异端/魔族为名集结军队 |
| 接收神谕 | 可真可假，假神谕是教会腐败的根源 |

## 五、神明设定速查（10问）

创建异世界时，回答以下10个问题即可确定神明体系：

1. **有没有神？** — 有/没有/不确定（信仰vs证据）
2. **几个神？** — 唯一神/多神/众神
3. **神在哪？** — 神域/天上/人群中/已沉睡/已死亡
4. **神管什么？** — 创世/元素/职能/不管事
5. **神跟人什么关系？** — 保佑/交易/利用/无视/对抗
6. **教会代表神吗？** — 是/不完全是/完全不代表
7. **神能干预人间吗？** — 直接出手/只能通过代理/只能给提示/不能
8. **穿越跟神有关吗？** — 神安排的/意外/系统/不知道
9. **加护来自神吗？** — 神赐/系统给/天生/不知道
10. **有神罚吗？** — 有（具体规则）/没有/只有传说
