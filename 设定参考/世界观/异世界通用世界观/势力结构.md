# 异世界 · 势力结构

---

## 一、人类王国

| 项目 | 内容 |
|------|------|
| 政体 | 君主立宪制，国王下设贵族议会 |
| 首都 | 王都亚尔凯迪亚 |
| 军事 | 骑士团（正规军）+ 冒险者（佣兵）+ 魔法兵团 |
| 经济 | 农业+商业为主，魔法道具贸易为支柱 |
| 文化 | 重视血统和阶级，平民可通过冒险者晋升 |

**主要城市**：
- 王都：政治中心，大教堂所在地
- 港口都市梅里迪安：海上贸易枢纽
- 边境要塞瓦尔哈拉：抵御魔族入侵前线

---

## 二、精灵王国

### 精灵分支

| 分支 | 居住地 | 特征 | 态度 |
|------|--------|------|------|
| 森林精灵 | 艾尔芙大陆中央森林 | 绿发绿瞳，亲近自然 | 排斥人类，封闭自守 |
| 暗精灵 | 地下洞穴/黑暗森林 | 灰肤红瞳，擅长暗杀 | 被正统精灵放逐，与魔族有交易 |
| 高等精灵 | 世界树周边 | 金发银瞳，魔力最强 | 自视甚高，视其他种族为低等 |

**政治结构**：
- 由精灵女王统治，女王从高等精灵中选出
- 长老议会辅政，重大决策需全体精灵投票
- 森林精灵与暗精灵长期对立，偶有小规模冲突

## 三、兽人部落

| 项目 | 内容 |
|------|------|
| 政体 | 部落联盟制，由最强大的部落首领担任大酋长 |
| 首领选拔 | 每5年举行一次比武大会，胜者为大酋长 |
| 战争文化 | 以战功为荣，战士地位最高 |
| 宗教 | 萨满教，信仰大地之神费尔娜 |
| 经济 | 狩猎+畜牧为主，与人类进行边境贸易 |

**主要部落**：
- 狼族部落：擅长侦查和游击战
- 熊族部落：力量型战士，正面战斗
- 鹰族部落：少数拥有飞行能力的兽人
- 蛇族部落：擅长暗杀和毒术，被其他部落排斥

---

## 四、矮人王国

| 项目 | 内容 |
|------|------|
| 政体 | 氏族制，各氏族族长组成议会 |
| 首都 | 地下都市尼德海姆，深入地下1000米 |
| 核心文化 | 锻造与采矿，技艺传承高于一切 |
| 经济 | 矿石/武器出口，与人类王国为主要贸易伙伴 |
| 军事 | 氏族卫队，擅长防御战和地下战 |

**主要氏族**：
- 铁锤氏族：最擅长锻造，武器工艺最高
- 石盾氏族：防御专家，守卫地下城入口
- 金钻氏族：采矿专家，掌管矿脉资源
- 火炉氏族：炼金术传承，擅长魔法武器

## 五、魔族势力

### 魔王军结构

| 阶级 | 称号 | 数量 | 能力 |
|------|------|------|------|
| 最高 | 魔王 | 1 | 统帅全军，最强战斗力 |
| 上位 | 四天王 | 4 | 各统领一军，灾厄级实力 |
| 中位 | 将军 | 20-30 | A级冒险者实力 |
| 下位 | 魔族战士 | 数万 | D-C级冒险者实力 |
| 底层 | 魔物 | 无数 | F-E级，被驱使作战 |

**入侵动机**：
- 魔界资源枯竭，需要扩张生存空间
- 魔王渴望征服现世，建立魔族帝国
- 部分魔族主张与现世共存，但被主战派压制

## 八、其他势力

### 盗贼公会
- **组织**：松散联盟，各城市有独立支部
- **业务**：情报交易、暗杀、盗窃、走私
- **规矩**：不杀12岁以下儿童，不背叛同伴

### 佣兵团
- **代表**：银狼团、铁血团、苍鹰团
- **业务**：承接公会以外的私人委托
- **特点**：比公会更灵活，但信誉参差不齐

### 商会
- **代表**：黄金商会、海上商会、冒险者商会
- **势力**：掌控贸易路线，经济影响力巨大
- **手段**：政治献金、情报网、私人武装
