# 异世界通用世界观 · 角色模板

> 整合通用角色生成模板+异世界专用维度。创建异世界角色时，按此模板逐项填写即可。
> 通用角色生成模板来源：`人物/references/素材模版库/角色生成模板.md`

---

```
【年龄】：__岁（外表__岁）
【性别】：
【身高/体重】：
【外貌关键词】：（3-5个词，如：凤眼/薄唇/左脸刀疤/银发及腰/精灵尖耳）
【第一印象】：（别人第一次见他/她会注意到什么）
```

## 二、职业与等级

```
【初始职业】：战士/法师/弓箭手/牧师/刺客/召唤师/炼金术士/冒险者/商人/贵族/其他
【转职目标】：（故事中后期会转职为什么）
【冒险者等级】：E/D/C/B/A/S
【角色等级】：Lv__（对应阶段：初期1-20/中期21-40/后期41-60/顶级61-80/传说81-100）
【所属势力】：[公会/国家/教会/独立/魔王军]
```

## 三、穿越信息（穿越者填写，原住民跳过）

```
【穿越方式】：神殿召唤/死亡转生/意外穿越/系统绑定/非穿越（原住民）
【前世身份】：（穿越前是干什么的？决定可利用的现代知识范围）
【前世知识利用】：（现代科技/异世界知识/游戏经验/专业知识等）
【神明加护】：（语言理解/物品箱/状态栏/作弊技能/属性异常/无）
【加护来源】：神明赐予/系统绑定/天生/未知
【回归意愿】：（想回原来的世界吗？转生者无此负担，转移者常有）
【身份暴露风险】：（被发现是穿越者会怎样？语言会被扭曲，无法证明）
【文化引入】：（会引入什么现实世界的东西？娱乐/食物/技术/文化）
```

## 六、人物类型标签

从 `人物/SKILL.md` 分类中选取（可组合，建议2-3个）：

```
【人物类标签】：（如：病娇/傲娇/社恐/恋爱脑/腹黑/御姐/雌小鬼）
【性格类标签】：（如：偏执型/回避型/表演型/反社会型）
【职业类标签】：（如：剑士/法师/冒险者/贵族/奴隶/魔族将军）
```

**标签组合公式**：`人物类（核心驱动）+ 性格类（行为模式）+ 职业类（社会身份）`

⚠️ 必查兼容矩阵：`人物/兼容矩阵/`

## 七、心理内核

```
【自我认知】：（他/她觉得自己是什么样的人？）
```

## 八、人物弧线预设

```
【起点状态】：（出场时是什么样的人）
【转折事件】：（什么事件会改变他/她）
【终点状态】：（故事结束时变成了什么样的人）
【弧线类型】：
  □ 成长弧（正向变化）
  □ 堕落弧（负向变化）
  □ 平弧（不变，但改变周围的人）
  □ 伪弧（表面变化，内核不变）
```

## 九、对话习惯

```
【语气基调】：（如：冷淡/热情/阴阳怪气/温柔/暴躁）
【口头禅】：（1-2个，如："随便""你开心就好""切"）
【用词特点】：（如：爱用网络梗/爱用成语/爱用脏话/爱用方言）
【说话节奏】：（如：长句多/短句多/经常半句话不说完/爱插嘴）
【特殊习惯】：（如：紧张时结巴/撒谎时摸鼻子/生气时笑）
【禁忌话题】：（听到什么会炸/会沉默/会转移话题）
```

**辨识度测试**：遮住名字后，仅看对话能认出是他/她吗？不能就强化差异。

| **性格** | 热血/冷酷/腹黑/天然/傲娇/病娇/话痨/沉默/中二/老好人 |
