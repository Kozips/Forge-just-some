# 异世界 · 常见冲突

---

## 一、种族冲突

### 人类 vs 兽人

| 项目 | 内容 |
|------|------|
| 根源 | 领土争夺，人类扩张压缩兽人生存空间 |
| 表现 | 边境战争，兽人劫掠商队，人类讨伐兽人部落 |
| 激化点 | 人类在兽人领地开采矿脉 |
| 缓和可能 | 低，双方积怨已深 |
| 典型事件 | 边境村庄屠杀事件 |

### 精灵 vs 暗精灵

| 项目 | 内容 |
|------|------|
| 根源 | 千年前暗精灵背叛精灵女王，投靠魔族 |
| 表现 | 精灵追杀暗精灵，暗精灵暗杀精灵要人 |
| 激化点 | 暗精灵与魔族联手入侵精灵森林 |
| 缓和可能 | 极低，血仇不可调和 |

### 人类 vs 魔族

| 项目 | 内容 |
|------|------|
| 根源 | 魔王军入侵现世，争夺生存空间 |
| 表现 | 全面战争，魔王军攻城略地 |
| 激化点 | 魔王觉醒/魔界裂缝扩大 |
| 缓和可能 | 极低，除非魔王更替 |

## 二、国家战争

### 领土争端

- **人类王国东部边境**：与兽人部落争夺草原/矿区
- **精灵森林边界**：人类伐木者越界砍伐神圣树木
- **矮人矿脉**：人类/兽人争夺矮人王国外层矿脉

### 资源争夺

| 资源 | 争夺方 | 价值 |
|------|--------|------|
| 魔石矿脉 | 人类/矮人/兽人 | 魔法能源，经济命脉 |
| 神圣树木 | 人类/精灵 | 附魔材料，精灵视为神圣 |
| 龙息石 | 所有种族 | 传说级锻造材料 |
| 空间裂缝 | 人类/魔族 | 连接魔界的通道 |

### 王位继承

- **人类王国**：老国王无子嗣，三名公爵争夺继承权
- **精灵王国**：女王候选人之间暗流涌动
- **矮人王国**：铁锤氏族与金钻氏族争夺议会主导权

## 三、冒险者冲突

### 任务竞争

| 冲突类型 | 表现 | 解决方式 |
|----------|------|----------|
| 抢任务 | 两支队伍同时接同一任务 | 先到先得/公会仲裁 |
| 抢功劳 | 讨伐时抢最后一击 | 伤害贡献计算 |
| 恶意竞争 | 故意破坏对手任务 | 公会处罚/禁赛 |
| 素材争夺 | 探索迷宫时争夺稀有素材 | 先到先得/实力说话 |

### 公会内斗

- **派系对立**：保守派（只接安全任务）vs 激进派（挑战高难度）
- **等级歧视**：高等级冒险者看不起新人，拒绝组队
- **腐败**：个别公会职员收受贿赂，泄露任务情报

## 四、善恶对立

### 勇者 vs 魔王

| 项目 | 勇者方 | 魔王方 |
|------|--------|--------|
| 领袖 | 勇者（神选之人） | 魔王（魔族之王） |
| 力量来源 | 神明祝福/传说武器 | 魔界之力/禁忌魔法 |
| 目标 | 守护现世和平 | 征服现世 |
| 弱点 | 使命束缚，无法自由行动 | 被光属性克制 |

### 光明教会 vs 黑暗教会

| 项目 | 光明教会 | 黑暗教会 |
|------|----------|----------|
| 信仰 | 光之神阿尔特 | 暗之神诺克斯 |
| 教义 | 慈悲、治愈、守护 | 死亡即解脱，黑暗即真理 |
| 手段 | 治疗、净化、讨伐不死 | 诅咒、亡灵召唤、暗杀 |
| 势力 | 正教，王国支持 | 地下组织，被通缉 |
| 目标 | 消灭黑暗势力 | 复活暗之神 |

## 五、阶级矛盾

### 贵族 vs 平民

| 矛盾点 | 表现 | 激化条件 |
|--------|------|----------|
| 税收 | 贵族加重赋税，平民不堪重负 | 灾年/战争时期 |
| 司法 | 贵族犯罪轻判，平民重罚 | 明显不公事件 |
| 晋升 | 平民难以提升社会地位 | 冒险者成功案例刺激 |
| 土地 | 贵族兼并土地，农民流离失所 | 大规模圈地运动 |

### 奴隶制争议

- **支持方**：战俘/罪犯应为奴，提供廉价劳动力
- **反对方**：违背人道，兽人/精灵被奴役引发种族仇恨
- **现状**：人类王国部分区域合法，精灵/矮人王国禁止

## 六、穿越者冲突

### 穿越者 vs 原住民

| 冲突点 | 表现 |
|--------|------|
| 价值观 | 穿越者带来异世界观念，与本地文化冲突 |
| 知识优势 | 穿越者利用前世知识获取不当利益 |
| 命运干预 | 穿越者改变历史线，引发连锁反应 |
| 嫉妒 | 原住民嫉妒穿越者的特权和成长速度 |

### 穿越者之间

- **竞争**：多个穿越者争夺有限资源/机遇
- **理念冲突**：守护世界 vs 利用世界 vs 毁灭世界
- **系统冲突**：不同穿越者系统任务相互矛盾
