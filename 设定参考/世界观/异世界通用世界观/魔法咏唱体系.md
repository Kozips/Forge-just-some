# 异世界通用世界观 · 魔法咏唱体系

> 覆盖：魔法本质、咏唱机制、魔导器、魔法学派/流派、天赋与魔力回路。参考轻小说公共想象中的魔法系统共识。

---

## 一、魔法本质

**魔法 = 使用魔力引发自然与超自然现象的手段。**

| 概念 | 说明 |
|------|------|
| **魔力（MP）** | 广泛存在于世界与生物体内的能量，类似空气般无处不在 |
| **魔素** | 空气中流动的魔力微粒，可通过呼吸/冥想吸收 |
| **魔力回路** | 人体内承载魔力的通道，先天决定上限，可通过训练/治疗优化 |
| **魔力耗尽** | 轻则昏迷，重则危及生命直至死亡 |
| **魔力上限** | 因人而异，可通过冥想训练缓慢提升 |

### 天赋决定论

- 人类中能使用魔法者**稀少**——魔法学院一学年约100-200人
- 能否使用魔法 = 身体可承载魔力的多寡 = 魔力回路的完整性
- 种族差异：精灵天生魔力高，兽人天生魔力低，矮人几乎不能
- **穿越者魔力觉醒概率比原住民高3倍**

## 二、咏唱机制（核心系统）

> 咏唱 = 固定对魔力操作的想象，使魔法更高效地发动。无咏唱 = 力量高强的标志。

### 咏唱层级

| 层级 | 方式 | 效率 | 说明 |
|------|------|------|------|
| **完整咏唱** | 全文念出咒语 | 100% | 标准施法，魔法学院教的正规方式 |
| **省略咏唱** | 只念关键词 | 70-80% | 高级魔法师常用，省时间 |
| **无咏唱** | 不念咒语直接发动 | 50-70% | 出神入化之境的标志，主角常用来秀操作 |
| **魔法阵** | 在地面/空中绘制符文 | 120-200% | 范围增幅，需要时间准备，常用于仪式/大规模魔法 |
| **触媒咏唱** | 通过法杖/宝石/符文石辅助 | 100-150% | 魔道具辅助，降低咏唱难度 |

### 咏唱要素

```
完整咏唱 = 召唤词（唤起注意）+ 元素名（指定系统）+ 效果词（指定行为）+ 收束词（锁定目标）

例：「伟大的火之精灵啊（召唤词），以吾之名召唤汝之力量（元素名），化为灼热之箭矢（效果词），贯穿吾之敌人（收束词）——火焰箭！」
```

### 咏唱速度与战斗|

| 场景 | 咏唱策略 |
|------|----------|
| 单挑 | 省略咏唱为主，速度>威力 |
| 团战 | 完整咏唱由后排法师输出，前排保护 |
| 绝境 | 无咏唱发动保命魔法，或使用触媒加速 |
| 大规模 | 魔法阵+多人联合咏唱，准备时间长但威力巨大 |
| 禁咒 | 完整咏唱+魔法阵+祭品，可能需要多人分担消耗 |

## 三、魔法分类体系

### 元素属性分类（最常见）

| 元素 | 属性 | 克制 | 被克 | 核心魔法 |
|------|------|------|------|----------|
| 火 | 攻击/破坏 | 风 | 水 | 火球术、陨石坠落、爆裂火焰 |
| 水 | 治愈/攻击 | 火 | 地 | 治疗术、冰枪、水盾 |
| 地 | 防御/束缚 | 水 | 风 | 石壁、地震术、土枪 |
| 风 | 速度/切割 | 地 | 火 | 风刃、飞行术、真空斩 |
| 光 | 治愈/净化 | 暗 | 无 | 圣光术、复活、神圣结界 |
| 暗 | 诅咒/削弱 | 无 | 光 | 死亡诅咒、暗影潜行、灵魂吞噬 |

### 非元素分类

| 类型 | 说明 | 代表 |
|------|------|------|
| **时空魔法** | 空间转移、时间操控，极稀有 | 传送门、时间停止、异空间收纳 |
| **召唤魔法** | 召唤魔物/精灵/元素体协战 | 精灵契约、亡灵召唤、元素召唤 |
| **治愈魔法** | 恢复伤势/疾病，教会垄断 | 初级治愈、完全治愈、复活 |
| **附魔魔法** | 给装备/物品附加魔法效果 | 火属性附魔、锋利附魔、减重附魔 |
| **生活魔法** | 日常用途，无战斗价值 | 洁净（清洁身体）、时钟、点火、照明 |
| **概念魔法** | 作用于抽象概念，最稀有 | 契约魔法、隶属魔法、因果律操作 |

### 魔法等级

| 等级 | 名称 | 魔力消耗 | 效果范围 | 学习难度 |
|------|------|----------|----------|----------|
| Lv1 | 初级魔法 | 极低 | 单体 | 1周 |
| Lv2 | 中级魔法 | 低 | 小范围 | 1月 |
| Lv3 | 高级魔法 | 中 | 中范围 | 1年 |
| Lv4 | 上级魔法 | 高 | 大范围 | 5年 |
| Lv5 | 禁咒级 | 极高 | 超大范围 | 10年以上 |

> 学习规则：角色习得"水魔法 Lv1" = 习得该级别下辖的所有具体魔法（水球、水枪、雾气等）。

## 四、魔导器

> 魔导器 = 将魔石封入器具中，赋予器具魔法功能。**大部分不要求使用者拥有魔法素质。**

### 魔石来源

| 类型 | 来源 | 稀有度 |
|------|------|--------|
| **魔核** | 魔物体内的"心脏" | 中（讨伐魔物获得） |
| **天然魔石** | 自然形成的特殊矿石 | 高（矿山开采） |
| **人工魔石** | 炼金术士人工合成 | 极高（成本巨大） |

### 常见魔导器

| 分类 | 代表 | 说明 |
|------|------|------|
| **家用** | 魔力灶台、供水口、上下水道、浴缸 | 中产以上家庭标配 |
| **照明** | 魔力灯、夜明珠 | 取代蜡烛/油灯 |
| **战斗** | 魔法炸弹、魔力护盾、追踪箭 | 军用/冒险者用 |
| **通信** | 通信水晶、传音石 | 类似电话，受距离限制 |
| **存储** | 物品箱、空间袋 | 魔法版背包 |
| **防护** | 结界石、防御护符 | 一次性或持续消耗 |

### 魔导器的价格

- 魔石稀缺+制作难度 → 魔导器**普遍昂贵**
- 封入魔石的款式更是奢侈品
- **只有魔法大量普及的世界，魔导器才可能进入寻常百姓家**

## 五、魔法学派/流派

> 不同的魔法学派代表不同的修炼理念和组织。可以是国家机构，也可以是民间团体。

### 主要学派

| 学派 | 理念 | 擅长 | 代表组织 |
|------|------|------|----------|
| **正统学院派** | 系统学习、按部就班 | 全元素基础、魔法理论 | 王立魔法学院 |
| **元素专精派** | 专精单一元素到极致 | 单元素最强 | 元素使公会 |
| **实战派** | 只练能打赢的魔法 | 战斗魔法、省略咏唱 | 冒险者公会附属魔法班 |
| **召唤派** | 与精灵/魔物缔结契约 | 召唤魔法、契约魔法 | 精灵使协会 |
| **炼金派** | 魔法即化学，配方即真理 | 药水/魔导器/附魔 | 炼金术士工房 |
| **暗黑派** | 禁忌知识也是知识 | 诅咒/死灵/禁忌魔法 | 地下结社/黑暗教会 |
| **自然派** | 与自然共感，不主动操控 | 治愈/植物/天气 | 精灵族/大地神殿 |
| **武斗派** | 魔法融入肉体战斗 | 强化魔法/魔法剑术 | 武道学院/骑士团 |

### 学派冲突

| 冲突 | 说明 |
|------|------|
| 学院派 vs 实战派 | "你的魔法太花哨了" vs "你的魔法太粗糙了" |
| 元素专精 vs 全能型 | "一招鲜吃遍天" vs "什么都会什么都不精" |
| 正统派 vs 暗黑派 | "那是禁忌！" vs "知识不分正邪" |
| 召唤派 vs 独立派 | "借外力是懒" vs "你自己打试试？" |

## 六、魔法系统速查（10问）

创建异世界魔法系统时，回答以下10个问题：

1. **魔法的本质是什么？** — 魔力/元素/精神力/科技
2. **需不需要咏唱？** — 必须/可以省略/完全不需要
3. **怎么分类？** — 元素/功能/无分类
4. **天赋决定吗？** — 天赋>努力/努力可以弥补/系统开挂绕过天赋
5. **有没有禁咒？** — 有（什么代价）/没有/禁咒即日常
6. **魔导器发达吗？** — 像手机一样普及/贵族专属/几乎没有
7. **有学派吗？** — 有（几个，什么关系）/没有
8. **治愈魔法谁控制？** — 教会垄断/人人可学/不存在
9. **穿越者怎么获得魔法？** — 天生/系统/神赐/学习
10. **魔法的上限是什么？** — 禁咒/神级/没有上限
