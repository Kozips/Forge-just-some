# SCP基金会世界观速查

核心理念：**控制、收容、保护** —— 保密至上，以科学方法研究异常，尽可能减少伤亡。

## 一、收容等级体系

| 等级 | 定义 |
|------|------|
| Safe | 容易收容 |
| Euclid | 需要持续收容 |
| Keter | 难以收容 |
| Thaumiel | 用于收容其他SCP |
| Neutralized | 已失效 |
| Apollyon | 无法收容 |
| Archon | 不应收容 |

权限等级 Level 1-5，对应普通人员到O5议会。

管理层级：O5议会（最高决策）→ 站点主管 → 部门主管 → 项目主管 → 研究员/特工。

D级人员：来源多为死刑犯，用于高风险实验，被视为消耗品。

## 二、内部派系冲突（故事金矿）

| 派系 | 理念 | 冲突焦点 |
|------|------|----------|
| 保守派 | 严格收容，绝不利用 | 收容理念 vs 改革派 |
| 改革派 | 研究并利用SCP | 收容理念 vs 保守派 |
| 鹰派 | 强硬手段，宁可牺牲 | 手段方式 vs 鸽派 |
| 鸽派 | 温和手段，重视人命 | 手段方式 vs 鹰派 |

冲突类型：收容理念（收容vs利用）、资源分配、人事安排、项目优先级。

## 七、禁忌等级速查

| 等级 | 后果 | 举例 |
|------|------|------|
| 绝对禁忌 | 立即处决 | 故意释放SCP、向敌对组织泄露信息 |
| 严重禁忌 | 严厉处分/处决 | 背叛基金会、间谍活动、私自处决人员 |
| 中等禁忌 | 处分 | 违反操作规程、篡改实验数据、未经批准研究 |
| 轻微禁忌 | 警告 | 迟到早退、工作懈怠 |

关键禁忌：故意释放SCP、背叛基金会、向公众/媒体泄露SCP存在、私自杀害人员、虐待D级人员。

## 八、冲突模板

**收容失效故事线**：设备故障/人为失误 → SCP突破 → 紧急响应 → 外部组织趁火打劫 → 高层决策分歧 → 善后与追责。

**派系斗争故事线**：发现新SCP → 保守派要收容/改革派要利用 → 各自拉拢资源 → 行动冲突 → 揭露真相或背叛。

**外部威胁故事线**：混沌分裂者渗透/蛇之手释放行动 → 基金会反击 → 间谍/反间谍 → 代价与抉择。

## 九、写作禁忌/注意事项

- 不要把基金会写成全知全能 —— 资源有限、决策失误、内部腐败都是好素材。
- D级人员不是无面NPC —— 他们有恐惧、有故事，用好这点能制造道德困境。
- O5议会不是铁板一块 —— 每个O5有自己的理念和秘密议程。
- 记忆删除不是万能工具 —— 可能失败、有副作用、被逆转。
- 保密协议被违反时的处理方式本身就是好冲突：该不该对平民动用记忆删除？
- SCP本身可以有复杂动机 —— 不全是怪物，有的比人类更值得同情。
