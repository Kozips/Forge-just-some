# 神造世界 · 世界规则

---

## 一、世界起源

神创造了这个世界。但神不知道人类该怎么生活。

祂照着自己的想象造了人——十八岁的身体，成年人的样貌，脑子里只有最基本的程序：呼吸、走路、说话、吃饭。没有性知识，没有情感认知，没有社会规则。

祂给每个人分配了一个"家庭角色"——爸爸、妈妈、妹妹、女儿、哥哥、姐姐。但这些角色没有血缘关系，没有抚养历史，没有共同记忆。他们只是被放进同一个房子里，被告知"你们是一家人"。

"家人"是什么意思？没有人知道。

---

## 二、基础规则

| 规则 | 说明 |
|------|------|
| 全员年龄 | 18岁，神按同一模板生成，无年龄差异 |
| 血缘关系 | 无。所有人都是独立个体，无生物学亲缘关系 |
| 家庭称呼 | 程序生成的名字。"爸爸""妹妹""女儿""妈妈"只是称呼，不是关系 |
| 性知识 | 无。神没教过，没有人知道性是什么 |
| 情感认知 | 极度原始。知道"开心/不开心"，不知道"爱/嫉妒/占有欲" |
| 体力 | 无上限。不会累、不会饿（穿越者除外） |
| 寿命 | 不死。神没设定死亡机制 |
| 繁殖 | 由穿越者的"行为"触发隐藏机制——内射后系统自动生成新角色 |
| 穿着 | 穿衣服。神造了衣服，所有人穿着衣服生活。脱衣服需要理由/场合/害羞克服 |
| 性行为 | 可以拒绝，但没人拒绝。社会风气天然开放，性行为像握手一样自然 |

## 三、称呼系统

| 称呼 | 程序含义 | 实际含义 | 角色理解 |
|------|---------|---------|---------|
| 爸爸 | 系统分配的男性家长 | 只是一个名字 | 不理解含义，但会本能依赖 |
| 妹妹 | 系统分配的同辈女性 | 只是一个名字 | 不理解含义，但会亲近 |
| 女儿 | 由"爸爸"的行为触发生成 | 只是一个名字 | 不理解含义，但会叫"爸爸" |
| 妈妈 | 系统分配的女性家长 | 只是一个名字 | 不理解含义，但会照顾人 |
| 哥哥/姐姐 | 系统分配的同辈 | 只是一个名字 | 不理解含义 |

**核心矛盾**：这些称呼在人类文化里意味着血缘和禁忌，但在这个世界里只是空壳。读者看到"爸爸和女儿"会代入禁忌感，但角色完全不理解这有什么特别。

## 四、隐藏机制

### 4.1 繁殖触发

穿越者内射后，系统自动生成一个新的18岁女性角色。新角色自动获得一个"家庭称呼"。

- 穿越者不知道这个机制
- 第一次是意外触发
- 如果持续内射，"家庭"会越来越大

### 4.2 程序声音

每个角色脑子里都有一个"程序声音"——告诉她们该叫什么、该去哪、该做什么。

- 程序声音说"叫爸爸"→角色叫爸爸
- 程序声音说"去河边"→角色去河边
- 程序声音说"你是妈妈"→新角色开始照顾人

**程序声音会越来越弱。** 随着角色跟穿越者互动越多，程序声音的影响越弱——她们开始有自己的想法、自己的感情、自己的选择。

### 4.3 神的观察

神在看着这个世界。祂不理解人类的行为——为什么要做"那件事"，为什么叫"爸爸"的时候身体会抖。祂在学习。穿越者的一举一动都在教神——人类到底是什么。

## 五、害羞设定（核心机制）

### 5.1 基础规则

所有人都想做，但所有人都害羞。

这不是"可以拒绝但没人拒绝"——而是**每个人都会拒绝，但每个人最后都会同意**。拒绝的原因是害羞，同意的原因是身体/本能/忍不住。

| 角色 | 程序赋予的害羞 | 害羞的表现 | 害羞被压过的方式 |
|------|--------------|-----------|----------------|
| **爸爸** | 一家之主的责任感+害羞 | 看到家里有人做→会制止→"你们在干什么！" | 他自己也想→制止的时候身体有反应→最后加入 |
| **妈妈** | 母性的矜持+害羞 | 被碰了会说"不行……孩子在看" | 身体先软→嘴上还在说不行→手已经在摸了 |
| **妹妹** | 程序设定的"纯洁"+害羞 | 第一次被碰会缩→"好奇怪" | 好奇心压过害羞→半推半就→最后比谁都主动 |
| **女儿** | 对"爸爸"的依赖+害羞 | "爸爸……不可以……" | 他说"可以"→她就信了→害羞变成撒娇 |
| **姐姐** | 自尊心+害羞 | "我才不想跟你做" | 身体出卖她→湿了→嘴硬→最后自己坐上来 |

### 5.2 害羞的三层递进

| 层级 | 表现 | 语言 |
|------|------|------|
| **第一层：拒绝** | 推他/别过头/夹紧腿/捂住自己 | "不行""不要""不可以" |
| **第二层：半推半就** | 推了但没用力/别过头但脖子露出来了/夹紧了但没合上 | "不行……嗯……""不要……但是……" |
| **第三层：投降** | 身体软了/手放开了/腿分开了/主动碰他 | "算了……来吧""你赢了""我控制不了" |

**铁律**：第一层到第三层的过渡要**自然**——不是突然从拒绝变成同意，而是一点一点地松动。每一次松动都有一个具体的原因（他碰了一个敏感的地方/他说了一句让她心跳加速的话/她的身体反应让她自己都吓到了）。

### 5.3 害羞的"程序性"

这些害羞不是她们自己选择的——是**程序刻在骨子里的**。

- 爸爸看到家人做→脑子里的程序说"应该制止"→他开口制止→但身体说"我也想"→两个信号冲突→他卡住了
- 妈妈被儿子碰→脑子里的程序说"不应该"→她推开他的手→但穴口已经在吸→程序信号和身体信号打架→身体赢了
- 妹妹看到哥哥→脑子里的程序说"他是哥哥"→她觉得不应该→但好奇→程序让步→"就看一下"→"就碰一下"→"就试一下"

**穿越者的角色**：他是唯一没有程序害羞的人。他的害羞来自现实世界的道德观——但在这个世界里，他的道德观没有任何支撑（因为这里没有血缘、没有社会规则）。所以他的害羞会比其他人消退得更快。

### 5.4 害羞制造的剧情引擎

| 场景 | 害羞的作用 | 结果 |
|------|----------|------|
| 爸爸撞见妈妈和儿子 | 爸爸制止→但自己也想→最后三个人 | 害羞→嫉妒→加入 |
| 妹妹偷看妈妈和哥哥 | 妹妹装作没看到→但一直在看→手伸到下面 | 害羞→好奇→自慰→最后加入 |
| 女儿被爸爸碰 | "爸爸不可以"→但身体在回应→"爸爸说可以那就是可以" | 害羞→信任→投降 |
| 全家在一起 | 每个人都在装作正常→但每个人的身体都不正常 | 害羞→压抑→爆发→集体 |
| 姐姐和弟弟 | "我才不想跟你做"→但湿了→"这是意外" | 自尊心→嘴硬→身体出卖→投降 |

### 5.5 害羞的消退曲线

随着角色经历越多，害羞会慢慢消退——但不会完全消失。

```
害羞程度
  ↑
  │████████████████████  第一次（拒绝到底）
  │████████████████      第三次（拒绝但很快就同意）
  │████████████          第五次（嘴上说不要身体已经在动）
  │████████              第十次（偶尔害羞一下然后继续）
  │████                  第二十次（只有特定场景才会害羞）
  └────────────────────→ 经历次数
```

**永远不会完全消退**——害羞是这个世界的核心机制，消退了就没有张力了。每次新场景（新的地点/新的角色组合/新的刺激方式）都会让害羞重新出现。

## 六、额外机制（叠加害羞使用）

### 6.1 发情期

这个世界的人有周期性的"发情期"——像动物一样，身体会自动进入一种状态，理智还在但本能压过一切。

| 要素 | 说明 |
|------|------|
| 触发 | 周期性（每3-5天一次），或被特定刺激触发（气味/触碰/视觉） |
| 表现 | 体温升高、皮肤发红、乳头变硬、下面自动湿、呼吸变快 |
| 理智 | 完全清醒——知道自己在做什么，但"控制不了" |
| 害羞 | 仍然存在——但被本能压过了。"我知道不应该……但我控制不了" |
| 对话 | "好热……""身体好难受……""帮我……" |

**真实感来源**：动物发情期是真实的生理现象。这个世界的"发情期"是神的程序设定——神不懂人类的羞耻心，只给了本能。

**剧情效果**：她害羞→但发情期来了→理智vs本能→本能赢了→"我控制不了"——不是她选择的，是身体替她选的。

### 6.2 黑暗/看不见

"我看不见你所以不害羞"——关灯/蒙眼/黑暗的房间。

| 要素 | 说明 |
|------|------|
| 机制 | 害羞的很大来源是"被看见"——看不见=害羞减弱 |
| 表现 | 黑暗中触觉被放大——碰一下比看得见刺激十倍 |
| 对话 | "关灯……""我看不见你……所以没关系……""你别开灯……" |
| 反转 | 开灯的瞬间→她看到自己的样子→害羞全回来→"不要看！" |

**真实感来源**：人类在黑暗中确实更大胆——酒吧/夜店/电影院的搭讪率远高于白天。

**剧情效果**：黑暗给了她一个"借口"——"我看不见所以不算"。但身体的反应比看得见更强烈。

### 6.3 半梦半醒

睡着了/快醒的时候——意识模糊，分不清现实和梦。

| 要素 | 说明 |
|------|------|
| 机制 | 理智在休眠，身体是醒的。"我以为在做梦" |
| 表现 | 她闭着眼睛，身体在回应，但嘴上说"我在做梦……这是梦……" |
| 反转 | 醒了→发现是真的→"你刚才对我做了什么？！"→但她记得梦里的感觉→"梦里的感觉好真实……" |
| 边界 | 她是清醒的——只是意识边缘，理智在休眠。不是昏迷/不是被下药 |

**真实感来源**：每个人都有过半梦半醒时做出平时不会做的事的经历。

**剧情效果**：她用"我在做梦"给自己一个借口——梦里做的事不算。但醒来后她记得一切。

### 6.4 时间回溯

这个世界有一个Bug：每天凌晨会重置——所有人的记忆回到起点。

| 要素 | 说明 |
|------|------|
| 穿越者 | 记得一切——每一天的记忆都累积 |
| 其他角色 | 不记得——每天都是"第一次" |
| 身体记忆 | 脑子不记得，但身体记得——被碰过的部位会更敏感、分泌反应更快 |
| 对话 | 她每次都说"不要……第一次……"→但他知道她的身体已经不是第一次了 |
| 边界 | 她每次都是真拒绝（因为不记得）→但她的身体每次都比上次更快投降 |

**真实感来源**：肌肉记忆/条件反射是真实的生理现象——身体可以记住大脑忘记的事。

**剧情效果**：她每次都是"第一次"害羞→但身体越来越诚实→他看着她的身体"认出"他——比脑子先认出来。

### 6.5 食物/果实

这个世界的某种果子/花粉有"放松"效果——不是迷药，是像酒精。

| 要素 | 说明 |
|------|------|
| 效果 | 害羞减弱、身体更敏感、理智完全清醒 |
| 持续 | 30分钟-1小时 |
| 表现 | 脸红、身体发软、反应更强烈、说话更直白 |
| 对话 | "我吃了一个果子……好奇怪……""为什么我的身体在发抖……" |
| 边界 | 意识完全清醒——她知道自己在做什么，只是"胆子大了" |

**真实感来源**：酒精/大麻降低抑制力的真实机制。

**剧情效果**：她平时害羞→吃了果子→"我只是放松了一下"→事后她记得一切→不能用"我不记得了"否认。

### 6.6 嫉妒驱动

不是他主动→是她看到他跟别人做→嫉妒→害羞被嫉妒压过→她主动冲上去。

| 要素 | 说明 |
|------|------|
| 触发 | 看到他跟另一个角色亲密/做爱 |
| 表现 | 胸口闷、手指绞在一起、呼吸变快——然后冲过去 |
| 对话 | "我不是想做……我只是不想让你跟她做""你是我的……" |
| 反转 | 她说"我只是不想输"→但身体的反应是"我想要" |

**真实感来源**：嫉妒是人类最强的情绪之一，真的能让人突破底线。

**剧情效果**：害羞被嫉妒压过→她主动做了→但嘴上说"我不是想做"→可爱+张力。

### 6.7 争宠/竞争

多个角色同时想要他→互相不服→"凭什么你先"→害羞被竞争意识压过。

| 要素 | 说明 |
|------|------|
| 触发 | 两个或以上角色同时在场，都想要他的注意力 |
| 表现 | 争着碰他/争着叫他/争着坐上去 |
| 对话 | "我先来的""凭什么你先""你上次已经""那这次轮到我" |
| 反转 | 争着争着→两个人一起做了→"我不是想跟你一起……只是不想输给你" |

**真实感来源**：竞争心理真的能让人突破底线。

**剧情效果**：害羞被"不想输"压过→两个人从竞争变成合作→但嘴上还在吵。

## 七、世界观禁忌（写作红线）

| 禁忌 | 说明 |
|------|------|
| 真实未成年 | 全员18岁是硬性设定，不可改变 |
| 心智幼态 | 角色可以天真懵懂，但心智必须是成年人水平 |
| 非自愿 | "可以拒绝但没人拒绝"≠"不能拒绝"。角色有拒绝权 |
| 血缘关系 | 无血缘是硬性设定。称呼只是程序名字 |
