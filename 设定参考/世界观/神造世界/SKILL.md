# 神造世界世界观速查

## 一、世界规则

### 1.1 世界起源

神创造了这个世界。但神不知道人类该怎么生活。

祂照着自己的想象造了人——十八岁的身体，成年人的样貌，脑子里只有最基本的程序：呼吸、走路、说话、吃饭。没有性知识，没有情感认知，没有社会规则。

祂给每个人分配了一个"家庭角色"——爸爸、妈妈、妹妹、女儿、哥哥、姐姐。但这些角色没有血缘关系，没有抚养历史，没有共同记忆。他们只是被放进同一个房子里，被告知"你们是一家人"。

"家人"是什么意思？没有人知道。

### 1.2 基础规则

| 规则 | 说明 |
|------|------|
| 全员年龄 | 18岁，神按同一模板生成，无年龄差异 |
| 血缘关系 | 无。所有人都是独立个体 |
| 家庭称呼 | 程序生成的名字。"爸爸""妹妹"只是称呼，不是关系 |
| 性知识 | 无。神没教过 |
| 情感认知 | 极度原始。知道"开心/不开心"，不知道"爱/嫉妒/占有欲" |
| 体力/寿命 | 无上限，不死。神没设定死亡机制 |
| 繁殖 | 穿越者内射后系统自动生成新角色 |
| 穿着/性行为 | 穿衣服生活。性行为像握手一样天然自然 |

### 1.3 称呼系统

| 称呼 | 含义 |
|------|------|
| 爸爸 | 系统分配男性家长→本能依赖 |
| 妈妈 | 系统分配女性家长→会照顾人 |
| 妹妹 | 系统分配同辈女性→会亲近 |
| 女儿 | 穿越者行为触发生成→会叫"爸爸" |
| 哥哥/姐姐 | 系统分配同辈 |

**核心矛盾**：这些称呼在人类文化里意味着血缘和禁忌，但在这个世界里只是空壳。读者看到"爸爸和女儿"会代入禁忌感，但角色完全不理解这有什么特别。

### 1.4 隐藏机制

**繁殖触发**：穿越者内射后，系统自动生成一个新的18岁女性角色，自动获得"家庭称呼"。穿越者不知道这个机制，第一次是意外触发。

**程序声音**：每个角色脑子里都有一个"程序声音"——告诉她们该叫什么、该去哪、该做什么。随着角色跟穿越者互动越多，程序声音越弱——她们开始有自己的想法、感情、选择。

**神的观察**：神在看着这个世界。祂不理解人类的行为——为什么要做"那件事"，为什么叫"爸爸"的时候身体会抖。祂在学习。穿越者的一举一动都在教神——人类到底是什么。

### 1.5 害羞设定（核心机制）

#### 基础规则

所有人都想做，但所有人都害羞。每个人都会拒绝，但每个人最后都会同意。拒绝的原因是害羞，同意的原因是身体/本能/忍不住。

| 角色 | 程序赋予的害羞 | 害羞被压过的方式 |
|------|--------------|----------------|
| **爸爸** | 一家之主的责任感 | 看到家人做→制止→但自己也想→加入 |
| **妈妈** | 母性的矜持 | 身体先软→嘴上还在说不行→手已经在摸了 |
| **妹妹** | 程序设定的"纯洁" | 好奇心压过害羞→半推半就→比谁都主动 |
| **女儿** | 对"爸爸"的依赖 | 他说"可以"→她就信了→害羞变撒娇 |
| **姐姐** | 自尊心 | 嘴硬→身体出卖她→最后自己坐上来 |

#### 三层递进

| 层级 | 表现 | 语言 |
|------|------|------|
| **第一层：拒绝** | 推他/别过头/夹紧腿/捂住自己 | "不行""不要""不可以" |
| **第二层：半推半就** | 推了但没用力/别过头但脖子露出来了 | "不行……嗯……""不要……但是……" |
| **第三层：投降** | 身体软了/手放开了/主动碰他 | "算了……来吧""我控制不了" |

**铁律**：过渡要**自然**——不是突然从拒绝变成同意，而是一点一点松动。每次松动都有具体原因。

#### 害羞的"程序性"

害羞不是她们自己选择的——是**程序刻在骨子里的**。

- 爸爸看到家人做→程序说"应该制止"→他开口制止→但身体说"我也想"→两个信号冲突→他卡住了
- 妈妈被碰→程序说"不应该"→她推开→但身体在回应→程序信号和身体信号打架→身体赢了
- 妹妹看到哥哥→程序说"他是哥哥"→但好奇→"就看一下"→"就碰一下"→"就试一下"

**穿越者**是唯一没有程序害羞的人。他的害羞来自现实世界的道德观——但在这里没有支撑，所以他的害羞消退最快。

#### 害羞的消退曲线

```
害羞程度
  ↑
  │████████████████████  第一次（拒绝到底）
  │████████████████      第三次（拒绝但很快同意）
  │████████████          第五次（嘴上说不要身体已在动）
  │████████              第十次（偶尔害羞一下然后继续）
  │████                  第二十次（只有特定场景才害羞）
  └────────────────────→ 经历次数
```

**永远不会完全消退**——每次新场景（新地点/新角色组合/新刺激方式）都会让害羞重新出现。

### 1.6 额外机制（叠加害羞使用）

| 机制 | 核心概念 |
|------|---------|
| **发情期** | 周期性（每3-5天）或刺激触发。理智清醒但本能压过一切——"我知道不应该……但我控制不了" |
| **黑暗/看不见** | 看不见=害羞减弱。黑暗中触觉放大十倍。反转：开灯瞬间害羞全回来——"不要看！" |
| **半梦半醒** | 理智休眠身体醒着。"我以为在做梦"给自己借口。醒来后记得一切——"梦里的感觉好真实……" |
| **时间回溯** | 凌晨记忆重置。穿越者记得一切，其他人每天是"第一次"。但**身体记得**——被碰过的部位更敏感，身体比脑子先"认出"他 |
| **食物/果实** | 某种果子像酒精——害羞减弱、身体更敏感、理智完全清醒。事后记得一切，不能否认 |
| **嫉妒驱动** | 看到他跟别人做→嫉妒压过害羞→主动冲上去。"我不是想做……我只是不想让你跟她做" |
| **争宠/竞争** | 多个角色同时想要→互相不服→"凭什么你先"→竞争变合作→"不是想跟你一起……只是不想输给你" |

### 1.7 世界观禁忌（写作红线）

| 禁忌 | 说明 |
|------|------|
| 真实未成年 | 全员18岁是硬性设定，不可改变 |
| 心智幼态 | 角色可以天真懵懂，但心智必须是成年人水平 |
| 非自愿 | "可以拒绝但没人拒绝"≠"不能拒绝"。角色有拒绝权 |
| 血缘关系 | 无血缘是硬性设定。称呼只是程序名字 |

### 1.8 可爱纯真写法指南（替代"幼态"）

| 禁区（幼态） | 替代（可爱纯真） |
|------------|----------------|
| 身材像小孩（比例未发育） | 成年人但骨架小/脸圆/眼睛大/皮肤白 |
| 说话像小孩（叠词/奶音） | 成年人但声音软/语速慢/尾音上翘 |
| 行为像小孩（蹦跳/哭闹） | 成年人但容易惊讶/好奇心重/认真提问 |
| 心智像小孩（不懂事） | 成年人但对世界充满好奇/信任别人 |
| "哇好好玩呀~" | "这个是什么？我没见过。" |
| 抱着玩偶/哭着要妈妈 | 攥衣角睡觉/安静地找他 |

**核心区别**：幼态=模拟未成年人。可爱纯真=成年人的天然魅力。写的是一个**不知道世界是什么但思考方式是成年人的**角色。

## 二、势力结构

这个世界没有国家、没有军队、没有宗教、没有商业。神只造了人和房子，没造"社会"。唯一的组织形式是"家庭"——由程序分配的角色组成。家庭会越来越大——穿越者每内射一次，系统就生成一个新角色。

**核心冲突**：程序声音（神的意志）vs 穿越者（人类意志）vs 觉醒角色（新生自由意志）。程序告诉她们该做什么 vs 她们自己想做什么。

神不是敌人。神只是不理解。穿越者的行为在教神——人类到底是什么。

## 三、常见冲突

### 3.1 害羞冲突（最核心）

| 冲突 | 表现 |
|------|------|
| 程序害羞 vs 身体欲望 | 嘴上说不行→手已经在摸→"算了" |
| 爸爸的责任感 vs 欲望 | "你们在干什么！"→看了三秒→"我也来" |
| 妈妈的矜持 vs 身体 | "不行……孩子在看……嗯……别停" |
| 妹妹的纯洁 vs 好奇 | 偷看→自慰→被发现→半推半就→比谁都主动 |
| 女儿的依赖 vs 害羞 | "爸爸不可以"→他说可以→"爸爸说可以那就是可以" |
| 姐姐的自尊心 | 嘴硬→身体出卖→"这是意外"→投降 |

### 3.2 关系/情感/世界观冲突

**关系**：角色竞争（争夺注意力）、占有欲觉醒（不知道那是吃醋）、称呼的重量（"爸爸"从空壳变有感情）、新角色加入打破平衡

**情感**：依赖→爱（"心跳在拐弯"→"我不要叫你爸爸了"）、天真→觉醒（理解"禁忌"含义）、穿越者的责任（"我到底在教她们什么？"）、爱的表达（攥衣角、分离焦虑）

**世界观**：神的不理解（可能干预）、程序Bug（繁殖机制的逻辑漏洞）、世界的有限性（只有"家"和外面的虚无）

## 四、写作注意事项

| 注意 | 说明 |
|------|------|
| 称呼的双重含义 | 每次角色叫"爸爸"，读者会代入禁忌感——不要破坏这个信息差 |
| 害羞是核心机制 | 每个角色都会因为害羞拒绝→但最后都会同意。拒绝是真拒绝，同意也是真同意 |
| 穿越者的禁忌感 | 他是唯一"知道不该这么做"的人——但在这里他的道德观没有支撑，所以消退最快 |
| 角色的天真 | 她们不理解"乱伦""禁忌""道德"——不要用成人词汇污染她们的认知 |
| 程序声音是伏笔 | 每次"脑子里的声音告诉我"都是在埋伏笔 |
| 新角色生成是悬念 | 每次内射都可能生成新角色——读者会期待"下一个是谁" |
| 荒诞感是调味料 | 一群十八岁男女穿着衣服互叫家庭称呼但什么都不懂——用喜剧感中和禁忌感 |
