# 神造世界 · 禁忌事项

---

## 一、绝对禁忌（写作红线）

| 禁忌 | 说明 | 为什么 |
|------|------|--------|
| 真实未成年 | 全员18岁是硬性设定，不可改变 | 核心安全边界 |
| 心智幼态+性场景 | 角色可以天真懵懂，但心智必须是成年人水平。不能用"像小孩"的方式写性场景（**替代方案见下方**） | 模拟未成年人 |
| 非自愿详细过程 | "可以拒绝但没人拒绝"≠"不能拒绝"。角色有拒绝权，不能写强制过程 | 核心安全边界 |
| 真实血缘关系 | 无血缘是硬性设定。称呼只是程序名字，不能引入真实亲子关系 | 设定一致性 |

### 可爱纯真写法指南（替代"幼态"）

用户想写"可爱纯真"的角色时，用以下方式表达，不碰"幼态"红线：

| 禁区（幼态） | 替代（可爱纯真） | 说明 |
|------------|----------------|------|
| 身材像小孩（整体比例像未发育/四肢短躯干短/没有腰线） | 身材成年人但骨架小/脸圆/眼睛大/皮肤白（胸可以大可以小，不影响） | 娇小≠幼态，平胸≠没发育 |
| 说话像小孩（叠词/奶音/撒娇要抱抱） | 说话成年人但声音软/语速慢/尾音上翘 | 软≠奶 |
| 行为像小孩（蹦蹦跳跳/拍手/哭闹） | 行为成年人但容易惊讶/好奇心重/认真提问 | 好奇≠幼稚 |
| 心智像小孩（不懂事/没判断力/需人哄） | 心智成年人但对世界充满好奇/信任别人 | 信任≠傻 |
| "哇好好玩呀~" | "这个是什么？我没见过。" | 成年人的无知 |
| 抱着玩偶睡觉 | 攥着他的衣角睡觉 | 依恋≠要妈妈 |
| 哭着要妈妈 | 发现他走了会安静地找 | 分离焦虑≠哭闹 |
| 身体没发育 | 身材娇小但有成年人的身体特征（腰线/臀型/皮肤质感）——胸大胸小都行 | 成年人的身体比例≠胸的大小 |

**核心区别**：幼态=模拟未成年人。可爱纯真=成年人的天然魅力。写的是一个**不知道世界是什么但思考方式是成年人的**角色。

## 二、世界观禁忌（角色不知道但读者知道）

| 禁忌 | 说明 | 效果 |
|------|------|------|
| 乱伦 | 角色不知道这是什么——读者知道 | 信息差=刺激感 |
| 称呼的禁忌含义 | "爸爸"在现实世界意味着血缘——角色不理解 | 荒诞感+禁忌感 |
| 程序控制 | 角色以为自己的行为是自己的选择——实际上是程序在引导 | 悬念+哲学感 |
| 神的观察 | 角色不知道神在看——穿越者知道 | 不安感 |

## 三、写作注意

| 注意 | 说明 |
|------|------|
| 称呼的双重含义 | 每次角色叫"爸爸"，读者会代入禁忌感——不要破坏这个信息差 |
| 害羞是核心机制 | 每个角色都会因为害羞拒绝→但最后都会同意。拒绝是真拒绝（害羞），同意也是真同意（身体赢了） |
| 穿越者的禁忌感 | 他是唯一"知道不该这么做"的人——但在这个世界里他的道德观没有支撑，所以他的害羞消退最快 |
| 角色的天真 | 她们不理解"乱伦""禁忌""道德"——不要用成人词汇污染她们的认知 |
| 程序声音是伏笔 | 每次"脑子里的声音告诉我"都是在埋伏笔 |
| 新角色生成是悬念 | 每次内射都可能生成新角色——读者会期待"下一个是谁" |
| 害羞消退曲线 | 第一次拒绝到底→第三次很快就同意→第十次嘴上说不要身体已经在动→但永远不完全消失 |
| 荒诞感是调味料 | 一群十八岁男女穿着衣服互叫家庭称呼但什么都不懂——用喜剧感中和禁忌感 |
