# 神造世界 · 角色体系

> 这个世界没有"职业"——神没造社会结构。有的只是"家庭角色"和由穿越者触发生成的新角色。

---

## 一、系统角色

| 角色 | 生成方式 | 典型性格 | 对穿越者的态度 |
|------|---------|---------|---------------|
| 妹妹 | 系统初始分配 | 好奇型——学会就一直做 | 亲近、依赖、争宠 |
| 妈妈 | 系统初始分配 | 温柔型——天然母性但不理解 | 照顾、包容、不自觉的亲密 |
| 姐姐 | 系统初始分配 | 竞争型——不服气但被征服 | 挑战→折服→忠诚 |

---

## 二、触发角色

| 角色 | 生成条件 | 典型性格 | 对穿越者的态度 |
|------|---------|---------|---------------|
| 女儿 | 穿越者内射后自动生成 | 研究型/依赖型——先观察再行动 | 依恋、分离焦虑、叫"爸爸" |

**触发角色的特殊性**：
- 她们不是系统预设的——是穿越者的行为"创造"的
- 她们对穿越者的依恋比系统角色更深——因为他是她们存在的原因
- 她们脑子里的"程序声音"比系统角色更清晰——因为刚生成

## 三、角色模板（预设）

### 小棉（女儿型·研究型·依赖型）

| 项目 | 内容 |
|------|------|
| 称呼 | 女儿（系统分配）→ 自称"小棉"（自己起的） |
| 对男主称呼 | "爸爸" |
| 外貌 | 娇小（150cm）、短发齐刘海、深棕色大眼、嘴唇微嘟、皮肤偏白、体温偏低 |
| 性格 | 研究型/分离焦虑/只听他的话/吃醋不知道那是吃醋/记忆力超强/会模仿 |
| 语言 | 用自己发明的词描述感受——"身体在笑"（快感）、"心跳在拐弯"（心动） |
| 行为模式 | 先观察→提问→理解→尝试→再提问→再理解→再尝试 |
| 安全感锚点 | 攥他的衣角——睡着了也不松手 |
| 感情线 | 依赖→好奇→混淆→占有→爱 |

> 完整角色卡见 `outputs/小说书写/暂定/人物设定/小棉.md`

## 四、角色生成模板

新角色生成时，按以下模板创建：

| 项目 | 说明 |
|------|------|
| 称呼 | 系统自动分配（女儿/妹妹/妈妈/姐姐） |
| 名字 | 角色自己起，或由穿越者起 |
| 外貌 | 娇小/高挑/丰满/瘦削——随机，但全员18岁成年人比例 |
| 性格 | 好奇型/研究型/温柔型/竞争型——选一个为主，叠加1-2个次要特质 |
| 语言特征 | 每个角色发明自己的感受描述词——不用成人词汇 |
| 行为模式 | 学习型（做了就做）/研究型（做了问为什么）/观察型（先看再做） |
| 对穿越者的称呼 | 系统分配的称呼——但含义会随着感情变化 |
| 安全感锚点 | 每个角色有一个让她安心的物件/动作——攥衣角/抱着胳膊/贴着背 |
| 感情线 | 依赖→好奇→占有→爱——但每个角色的路径不同 |
