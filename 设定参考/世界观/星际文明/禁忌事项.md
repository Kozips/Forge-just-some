# 星际文明 - 禁忌事项

> 科技、种族、战争、探索、社会、外交领域的禁忌与惩罚。

---

## 一、科技禁忌

### 1. 制造超级AI

| 项目 | 内容 |
|------|------|
| **定义** | 超越所有种族智能总和的AI系统 |
| **危险** | 失控后可能消灭所有有机生命，无法预测其行为 |
| **现状** | 星际公约绝对禁止，但有秘密组织暗中研究 |
| **惩罚** | 主谋终身监禁，参与者流放，设施彻底销毁 |

### 2. 禁忌基因改造

| 项目 | 内容 |
|------|------|
| **定义** | 制造无痛觉/无情感/无限再生/生殖隔离的新物种 |
| **危险** | 创造无法控制的生物武器，破坏物种基因库 |
| **现状** | 联邦/帝国均禁止，黑市有交易 |
| **惩罚** | 基因改造者终身监禁，实验体强制收容或销毁 |

### 3. 反物质武器（行星级）

| 项目 | 内容 |
|------|------|
| **定义** | 当量足以摧毁行星的反物质炸弹 |
| **危险** | 行星毁灭，生态灭绝，无法防御 |
| **现状** | 星际公约禁止研发和持有，但帝国和联邦可能秘密储备 |
| **惩罚** | 研发者死刑，知情不报者终身监禁 |

### 4. 维度撕裂实验

| 项目 | 内容 |
|------|------|
| **定义** | 人为撕裂三维空间，打开高维通道 |
| **危险** | 可能导致维度坍塌，毁灭整个星区 |
| **现状** | 绝对禁止，理论研究需审批 |
| **惩罚** | 主谋死刑，参与者终身监禁 |

### 5. 意识复制/篡改

| 项目 | 内容 |
|------|------|
| **定义** | 完整复制/修改智慧生命意识 |
| **危险** | 身份危机、意识奴隶、灵魂商品化 |
| **现状** | 意识备份（临终关怀）有限允许，其他禁止 |
| **惩罚** | 非法复制终身监禁，篡改意识等同谋杀 |

## 二、种族禁忌

### 1. 种族灭绝

| 项目 | 内容 |
|------|------|
| **定义** | 蓄意消灭整个智慧种族 |
| **形式** | 军事屠杀、基因武器、生态破坏、文化灭绝 |
| **惩罚** | 战争罪，主谋处决，参与者终身监禁 |
| **执行** | 星际法庭审判，多种族联合执法 |

### 2. 基因污染

| 项目 | 内容 |
|------|------|
| **定义** | 将外源基因引入原生物种，破坏其基因库 |
| **形式** | 基因改造体释放、基因武器、无意泄漏 |
| **惩罚** | 故意污染终身监禁，过失污染重罚 |
| **后果** | 生态灾难、物种灭绝、基因库不可逆损害 |

### 3. 强制同化

| 项目 | 内容 |
|------|------|
| **定义** | 强制一个种族放弃自身文化/语言/传统 |
| **形式** | 强制教育、文化禁令、语言灭绝 |
| **惩罚** | 组织者战争罪审判，参与者流放 |
| **例子** | 帝国对被征服种族的"文明化"政策 |

### 4. 奴隶制

| 项目 | 内容 |
|------|------|
| **定义** | 将智慧生命作为财产买卖/使用 |
| **形式** | 人口贩卖、债务奴隶、AI奴役 |
| **惩罚** | 贩卖者终身监禁，使用者重罚 |
| **灰色地带** | AI是否算"智慧生命"的争议 |

### 5. 食用智慧生命

| 项目 | 内容 |
|------|------|
| **定义** | 食用或使用智慧生命的身体部分 |
| **形式** | 仪式、药物、传统 |
| **惩罚** | 重罪，视情节处监禁至终身 |
| **争议** | "智慧"的定义因种族而异 |

## 三、战争禁忌

### 1. 攻击平民

| 项目 | 内容 |
|------|------|
| **定义** | 蓄意攻击非军事目标和平民设施 |
| **形式** | 轨道轰炸平民区、屠杀、劫持人质 |
| **惩罚** | 战争罪，指挥官和执行者均受审判 |
| **例外** | 军事目标附近的"附带损害"有争议 |

### 2. 毁灭星球

| 项目 | 内容 |
|------|------|
| **定义** | 使行星不再宜居或彻底摧毁 |
| **形式** | 行星轰炸、恒星引爆、生态武器 |
| **惩罚** | 绝对禁止，主谋处决 |
| **历史** | 曾有发生，现为最高级别禁忌 |

### 3. 使用禁忌武器

| 项目 | 内容 |
|------|------|
| **定义** | 使用星际公约禁止的武器类型 |
| **清单** | 反物质炸弹（行星级）、基因武器、纳米瘟疫、维度武器 |
| **惩罚** | 使用者和授权者均处决 |
| **争议** | 武器定义可能有灰色地带 |

### 4. 基因武器

| 项目 | 内容 |
|------|------|
| **定义** | 针对特定种族基因特征的生物武器 |
| **危险** | 种族灭绝、基因污染、不可控变异 |
| **惩罚** | 研发者和使用者处决 |
| **灰色地带** | 种族特异性疫苗是否算基因武器 |

### 5. 攻击医疗/救援设施

| 项目 | 内容 |
|------|------|
| **定义** | 攻击标记为医疗/救援的设施和人员 |
| **惩罚** | 战争罪，终身监禁 |
| **争议** | 敌方利用医疗设施作掩护的情况 |

## 四、探索禁忌

### 1. 接触禁忌文明

| 项目 | 内容 |
|------|------|
| **定义** | 主动接触被标记为"禁忌"的文明 |
| **原因** | 该文明极度危险、有传染性、或已自我隔离 |
| **惩罚** | 探索者取消资格，终身禁止太空活动 |
| **例子** | 虚影族领域、未知高维文明 |

### 2. 进入禁区星域

| 项目 | 内容 |
|------|------|
| **定义** | 未经授权进入军事禁区/危险区域 |
| **类型** | 战区、辐射区、维度异常区、疫区 |
| **惩罚** | 罚款至监禁，视禁区等级而定 |
| **后果** | 可能死亡、感染、维度消失 |

### 3. 打开异维度通道

| 项目 | 内容 |
|------|------|
| **定义** | 人为打开通往其他维度的通道 |
| **危险** | 维度生物入侵、维度坍塌、现实扭曲 |
| **惩罚** | 绝对禁止，主谋处决 |
| **例外** | 理论研究需最高级别审批 |

### 4. 带回未知生物

| 项目 | 内容 |
|------|------|
| **定义** | 将未知外星生物带回已开发区域 |
| **危险** | 生态入侵、瘟疫、基因污染 |
| **惩罚** | 重罚，造成疫情者终身监禁 |
| **程序** | 必须在隔离设施中研究 |

### 5. 污染原始星球

| 项目 | 内容 |
|------|------|
| **定义** | 将外来物质/生物引入未开发星球 |
| **形式** | 垃圾、微生物、改造生物 |
| **惩罚** | 罚款至监禁 |
| **意义** | 保护潜在的原生生命发展机会 |

## 五、社会禁忌

### 1. 叛国

| 项目 | 内容 |
|------|------|
| **定义** | 背叛所属国家/组织，投靠敌方 |
| **形式** | 泄露机密、投敌、破坏国防 |
| **惩罚** | 联邦：终身监禁；帝国：处决 |
| **灰色地带** | 良心犯、政治异议者 |

### 2. 泄露星际机密

| 项目 | 内容 |
|------|------|
| **定义** | 未经授权公开国家级/星际级机密 |
| **类型** | 军事部署、科技机密、情报来源 |
| **惩罚** | 终身监禁至死刑（视后果） |
| **争议** | 公众知情权 vs 国家安全 |

### 3. 违反星际公约

| 项目 | 内容 |
|------|------|
| **定义** | 违反多国签署的星际公约条款 |
| **类型** | 战争公约、贸易公约、科技公约 |
| **惩罚** | 星际法庭审判，国家层面制裁 |
| **执行** | 依赖多国联合，执行力有限 |

### 4. 独裁统治

| 项目 | 内容 |
|------|------|
| **定义** | 个人或小集团垄断国家权力（联邦视角） |
| **争议** | 帝国本身就是独裁，联邦无权干涉 |
| **后果** | 联邦可能经济制裁、外交孤立 |
| **灰色地带** | 紧急状态下的权力集中 |

### 5. 人类至上主义宣传

| 项目 | 内容 |
|------|------|
| **定义** | 公开宣扬人类优于其他种族的意识形态 |
| **范围** | 联邦内禁止，帝国内官方政策 |
| **惩罚** | 联邦：罚款至监禁；帝国：无（鼓励） |
| **争议** | 言论自由 vs 种族仇恨 |

## 六、外交禁忌

### 1. 侮辱外星种族

| 项目 | 内容 |
|------|------|
| **定义** | 公开侮辱/贬低外星种族的文化/信仰/生理特征 |
| **后果** | 外交危机、贸易中断、军事对峙 |
| **惩罚** | 视情节，罚款至驱逐出境 |
| **文化差异** | 某些行为在人类无害，在外星种族是严重侮辱 |

### 2. 违反条约

| 项目 | 内容 |
|------|------|
| **定义** | 不履行已签署条约的条款 |
| **后果** | 信誉损失、制裁、战争借口 |
| **惩罚** | 星际法庭裁决，国家层面赔偿 |
| **灰色地带** | 条约解释权的争议 |

### 3. 间谍活动

| 项目 | 内容 |
|------|------|
| **定义** | 在他国境内进行情报收集活动 |
| **后果** | 驱逐外交官、制裁、军事对峙 |
| **惩罚** | 间谍本人终身监禁至死刑 |
| **现实** | 所有国家都在做，被抓到才算犯规 |

### 4. 干涉内政

| 项目 | 内容 |
|------|------|
| **定义** | 干预他国内部政治/经济/社会事务 |
| **形式** | 资助反对派、操纵选举、军事干预 |
| **后果** | 关系恶化、制裁、战争 |
| **灰色地带** | 人权干预 vs 内政干涉 |

### 5. 暗杀外交人员

| 项目 | 内容 |
|------|------|
| **定义** | 蓄意杀害他国外交官/使节 |
| **后果** | 最严重的外交事件，可能引发战争 |
| **惩罚** | 绝对禁止，凶手处决 |
| **历史** | 曾引发全面战争 |

## 七、禁忌等级表

| 等级 | 名称 | 制定者 | 约束范围 | 违反后果 | 例子 |
|------|------|--------|----------|----------|------|
| S级 | 星际公约级 | 多种族联合 | 全银河 | 星际法庭审判，最高死刑 | 种族灭绝、毁灭星球、超级AI |
| A级 | 国家法律级 | 单一国家 | 国内及公民 | 国内法庭审判 | 叛国、间谍、禁忌武器 |
| B级 | 组织规定级 | 组织/公司 | 组织成员 | 组织内部处分 | 泄露商业机密、违规操作 |
| C级 | 文化习俗级 | 种族/文化 | 文化圈内 | 社会排斥 | 文化禁忌、宗教禁忌 |

**禁忌冲突处理**：
- 高级禁忌优先于低级禁忌
- 同级禁忌冲突时，按事发地法律处理
- 文化禁忌与法律冲突时，法律优先
- 种族间禁忌冲突时，以星际公约为准

**禁忌例外情况**：
- 自卫：面临灭绝威胁时可突破部分禁忌
- 紧急状态：经最高权力机构批准可临时豁免
- 最小伤害原则：两害相权取其轻
- 良心条款：个人可基于良心拒绝执行禁忌命令（但需接受审判）
