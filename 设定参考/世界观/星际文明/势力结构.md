# 星际文明 - 势力结构

> 联邦、帝国、商业联盟、外星种族及各势力之间的关系网络。

---

## 一、星际联邦

| 项目 | 内容 |
|------|------|
| **政体** | 议会制共和国 |
| **组成** | 127个成员国（人类殖民星系）+ 18个外星种族观察员 |
| **首都星** | 新地球（New Terra），半人马座α星系 |
| **立法机构** | 星际议会（1200席，按人口分配） |
| **行政机构** | 联邦总统（任期6年，可连任1次） |
| **军事力量** | 联邦星际舰队（12支主力舰队） |
| **核心价值观** | 民主、自由、多元、和平探索 |

**联邦法律要点**：
- 《星际人权宪章》：保障所有智慧生命基本权利
- 《殖民法》：殖民地自治权、资源分配规则
- 《AI权利法案》：高级AI享有有限公民权
- 《星际贸易法》：自由贸易、反垄断、关税协定

**军事编制**：
- 主力舰队：航母+战列舰+巡洋舰+驱逐舰+护卫舰
- 陆战队：行星登陆战、空间站夺控
- 特种部队：情报、渗透、暗杀
- 后勤舰队：补给、维修、医疗

## 二、星际帝国

| 项目 | 内容 |
|------|------|
| **政体** | 君主立宪制（实为军事独裁） |
| **最高统治者** | 皇帝（终身制，血统继承） |
| **贵族阶层** | 大公爵（星区总督）、侯爵（星系总督）、伯爵以下 |
| **首都星** | 帝都星（Imperial Throne），天狼星系 |
| **军事力量** | 帝国皇家舰队（20支主力舰队+禁卫军） |
| **核心价值观** | 秩序、荣耀、扩张、人类至上 |

**帝国贵族体系**：
| 等级 | 称号 | 权力 | 数量 |
|------|------|------|------|
| 一等 | 大公爵 | 星区总督，掌管数个星系 | 12 |
| 二等 | 侯爵 | 星系总督 | 约200 |
| 三等 | 伯爵 | 行星总督 | 约2000 |
| 四等 | 子爵 | 大型殖民地管理者 | 约10000 |
| 五等 | 男爵 | 小型殖民地/空间站管理者 | 约50000 |

**扩张主义政策**：
- "文明使命"：帝国自认为有义务"开化"落后种族
- 强制殖民：对资源星球的原住民进行迁移或同化
- 军事威慑：边境永远保持舰队存在

## 三、商业联盟

| 项目 | 内容 |
|------|------|
| **性质** | 跨国商业集团联合体 |
| **总部** | 自由港（Freeport），独立空间站群 |
| **领导层** | 商业议会（按股权投票） |
| **私军规模** | 约联邦舰队的40% |
| **核心利益** | 利润、贸易垄断、资源控制 |

**贸易垄断领域**：
- 稀有矿产开采权（反物质矿、零点能晶体）
- 跃迁引擎制造（垄断核心技术）
- 星际通讯网络（中继站所有权）
- 基因改造技术（高端改造市场）

**影响力手段**：
- 经济制裁：切断贸易路线
- 代理人战争：资助叛军/海盗
- 政治游说：在联邦议会和帝国宫廷都有代理人
- 信息控制：掌控主要星际媒体

## 四、外星种族

### 1. 赛隆人（Theron）

| 项目 | 内容 |
|------|------|
| **生理** | 硅基生命，身高3-4米，晶体外壳，寿命2000+年 |
| **文化** | 哲学至上，崇尚知识和永恒，集体决策 |
| **科技** | II级文明，晶体科技，能量操控 |
| **政体** | 长老议会（最年长者决策） |
| **与人类关系** | 中立偏友好，贸易伙伴，警惕人类扩张 |

### 2. 克萨人（Kessa）

| 项目 | 内容 |
|------|------|
| **生理** | 碳基昆虫形态，六肢，群居，单体寿命30年，蜂后寿命300年 |
| **文化** | 集体意识，个体无自我，服从蜂后 |
| **科技** | I-II级过渡，生物技术与机械混合 |
| **政体** | 蜂群体制（蜂后绝对统治） |
| **与人类关系** | 敌对，领土争端频繁，不可信任（无法理解个体） |

### 3. 维恩人（Vien）

| 项目 | 内容 |
|------|------|
| **生理** | 碳基水生，类人但有鳃和蹼，可在水中和陆地生存 |
| **文化** | 艺术至上，情感丰富，重视美学和音乐 |
| **科技** | I级文明，海洋工程和声波技术 |
| **政体** | 联邦制海洋城邦 |
| **与人类关系** | 友好，文化交流密切，部分加入联邦 |

### 4. 塔克人（Tark）

| 项目 | 内容 |
|------|------|
| **生理** | 碳基爬行类，强壮，鳞甲覆盖，再生能力 |
| **文化** | 战士文化，荣誉至上，弱肉强食 |
| **科技** | I级文明，武器和装甲技术先进 |
| **政体** | 军事独裁（最强者为王） |
| **与人类关系** | 敌对，视人类为弱者，边境冲突不断 |

### 5. 阿尔法人（Alpha）

| 项目 | 内容 |
|------|------|
| **生理** | 能量生物，无固定形态，可依附机械体 |
| **文化** | 神秘主义，很少与其他种族交流 |
| **科技** | III级文明（远超其他种族），维度操控 |
| **政体** | 未知（可能无政体概念） |
| **与人类关系** | 冷漠，偶尔出手干预，动机不明 |

### 6. 诺瓦人（Nova）

| 项目 | 内容 |
|------|------|
| **生理** | 碳基，类人，但有生物发光器官，寿命150年 |
| **文化** | 科学至上，理性，追求真理 |
| **科技** | II级文明，量子计算和生物科技 |
| **政体** | 科学家委员会（学术精英统治） |
| **与人类关系** | 合作，联合科研项目多，技术交换 |

### 7. 格里克人（Grik）

| 项目 | 内容 |
|------|------|
| **生理** | 碳基，矮壮，适应高重力，寿命80年 |
| **文化** | 工匠文化，重视手艺和实业，保守 |
| **科技** | I级文明，重工业和矿业技术 |
| **政体** | 行会制（各行业行会联合执政） |
| **与人类关系** | 贸易伙伴，提供矿产和工业品 |

### 8. 虚影族（Phantom）

| 项目 | 内容 |
|------|------|
| **生理** | 维度生物，可在三维和四维间切换，形态不定 |
| **文化** | 无法理解，行为模式与三维生命完全不同 |
| **科技** | IV级文明，维度技术 |
| **政体** | 未知 |
| **与人类关系** | 未知/威胁，偶尔出现造成维度异常 |
