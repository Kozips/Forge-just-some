# 星际文明 - 职业体系

> 军事、探索、科技、商业、特殊及社会职业的完整体系。

---

## 一、军事职业

### 1. 星际舰队指挥官

| 项目 | 内容 |
|------|------|
| **职责** | 指挥舰队作战、制定战略、舰队管理 |
| **晋升路径** | 军校→少尉→中尉→上尉→少校→中校→上校→将官 |
| **核心技能** | 战术指挥、舰队编成、超光速作战、政治敏感度 |
| **收入** | 军饷+战功奖金+贵族封地（帝国）/退休金（联邦） |
| **装备** | 指挥舰、战术AI辅助、量子通讯权限 |

### 2. 陆战队

| 项目 | 内容 |
|------|------|
| **职责** | 行星登陆战、空间站夺控、反海盗、维和 |
| **分类** | 突击兵/重装兵/侦察兵/工兵/医疗兵 |
| **核心技能** | 零重力作战、外骨骼操作、外星环境生存 |
| **收入** | 军饷+战斗津贴+伤残补偿 |
| **装备** | 动力装甲、能量武器、战术无人机 |

### 3. 战斗机飞行员

| 项目 | 内容 |
|------|------|
| **职责** | 舰载机空战、轰炸、侦察、护航 |
| **核心技能** | 高G机动、武器系统操作、编队战术 |
| **收入** | 军饷+飞行津贴+击坠奖金 |
| **装备** | 舰载战斗机/轰炸机/侦察机 |

### 4. 情报官

| 项目 | 内容 |
|------|------|
| **职责** | 情报收集、分析、反间谍、秘密行动 |
| **核心技能** | 信息分析、密码学、外星语言、伪装 |
| **收入** | 高薪+秘密行动奖金 |
| **装备** | 加密通讯器、伪装设备、神经接口 |

### 5. 后勤官

| 项目 | 内容 |
|------|------|
| **职责** | 补给线管理、舰队维护、物资调配 |
| **核心技能** | 物流管理、供应链优化、资源核算 |
| **收入** | 稳定军饷+管理津贴 |
| **装备** | 物流管理系统、自动化仓库 |

## 二、探索职业

### 1. 星际探险家

| 项目 | 内容 |
|------|------|
| **职责** | 探索未知星系、绘制星图、发现新资源/生命 |
| **核心技能** | 导航、生存、外星生物学基础、急救 |
| **收入** | 发现奖金（按发现价值）+赞助 |
| **风险** | 极高（失踪率约15%） |

### 2. 异星考古学家

| 项目 | 内容 |
|------|------|
| **职责** | 发掘先行者遗迹、解读古代科技、文物保护 |
| **核心技能** | 考古学、古代语言、异星文化、化学分析 |
| **收入** | 学术薪资+发现奖金+基金会资助 |
| **装备** | 非侵入式扫描仪、保护性挖掘工具 |

### 3. 殖民者

| 项目 | 内容 |
|------|------|
| **职责** | 建设新殖民地、开发资源、建立社区 |
| **核心技能** | 工程、农业、领导力、外星环境适应 |
| **收入** | 殖民地份额+资源分红 |
| **风险** | 中高（环境灾害、原住民冲突） |

### 4. 导航员

| 项目 | 内容 |
|------|------|
| **职责** | 超光速航线计算、引力场分析、安全跃迁 |
| **核心技能** | 天体力学、引力数学、量子计算、直觉判断 |
| **收入** | 高薪（稀缺人才） |
| **装备** | 导航AI、星图数据库、引力感应器 |

| **收入** | 中高薪（稀缺语言更高） |
| **装备** | 翻译AI辅助、语言数据库 |

## 七、职业等级与收入对照表

| 等级 | 典型职位 | 年收入（星币） | 社会地位 |
|------|----------|----------------|----------|
| 顶级 | 舰队司令、总督、大商人 | 1000万+ | 上层 |
| 高级 | 舰长、高级工程师、外交官 | 100-1000万 | 中上 |
| 中级 | 飞行员、工程师、商人 | 10-100万 | 中层 |
| 初级 | 陆战队员、矿工、殖民者 | 1-10万 | 中下 |
| 底层 | 劳工、流放者 | <1万 | 底层 |

**注**：1星币 ≈ 联邦核心星区1天基本生活费。帝国使用帝国马克，商业联盟使用信用点。
