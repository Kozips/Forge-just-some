# 无限流世界 - 职业体系

## 一、战斗职业

### 战士系

| 职业 | 特点 | 优势 | 劣势 |
|------|------|------|------|
| 剑士 | 近战输出 | 高攻击力 | 远程弱 |

| 狂战士 | 狂暴输出 | 爆发高 | 控制弱 |
| 骑士 | 攻防兼备 | 均衡 | 不突出 |

| 职业 | 特点 | 优势 | 劣势 |

## 五、职业搭配

### 最佳队伍配置

| 位置 | 职业 | 作用 | 重要性 |
|------|------|------|--------|
| 坦克 | 盾战士 | 吸收伤害 | 极高 |
| 输出 | 剑士/法师 | 造成伤害 | 极高 |
| 治疗 | 牧师 | 恢复生命 | 极高 |
| 辅助 | 吟游诗人 | 增益队友 | 高 |
| 控制 | 幻术师 | 控制敌人 | 高 |

### 职业克制关系

```
战士 → 射手（近身优势）
射手 → 法师（远程压制）
法师 → 战士（范围控制）
刺客 → 法师（脆皮目标）
治疗 → 刺客（持续恢复）
```

```
新手 → 剑士 → 剑圣 → 剑神
新手 → 盾战士 → 守护者 → 圣骑士
新手 → 狂战士 → 狂暴者 → 战神
```

```

新手 → 幻术师 → 幻术大师 → 幻神
```

### 刺客路线

```
新手 → 刺客 → 暗影大师 → 暗影之神
```
