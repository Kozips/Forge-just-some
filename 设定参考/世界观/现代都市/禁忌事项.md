# 现代都市禁忌事项

> 小说写作参考 | 以当代中国（2020s）为基准 | 具体可操作

---

## 一、政治禁忌

### 绝对红线（碰了就完）

| 禁忌 | 后果 | 案例 |
|------|------|------|
| 公开质疑国家体制 | 删帖/封号/约谈/刑拘 | 网络发言被请喝茶 |
| 发布涉军涉警负面信息 | 迅速删帖+追查来源 | 侮辱军人/警察被拘留 |
| 传播政治谣言 | 行政拘留/刑事追责 | 编造领导人谣言被判刑 |
| 涉及民族宗教敏感 | 删帖+约谈+可能刑拘 | 地域歧视引发民族问题 |
| 翻墙传播/接收敏感信息 | 罚款/警告/刑拘（极端） | VPN使用者被处罚 |

### 高风险话题（小说中需谨慎处理）

- **台湾问题**：必须称"台湾地区"，不能称"国家"
- **西藏/新疆**：涉及人权、民族政策的话题极度敏感
- **六四/文革**：历史敏感话题，公开讨论受限
- **领导人**：不能直呼其名做调侃，需用"某领导"代替
- **党媒/官媒**：不能质疑其公信力

### 小说处理建议

- 用虚构地名、虚构机构名
- 政治背景做模糊化处理
- 不要写正面批判体制的内容
- 可以写"社会现象"但不要写"政治原因"

## 二、社会禁忌

### 地域歧视

| 歧视内容 | 后果 | 敏感度 |
|---------|------|--------|
| "河南人都是骗子" | 舆论反噬、被举报 | 极高 |
| "东北人都是黑社会" | 地域骂战 | 高 |
| "上海人排外" | 本地人反击 | 中高 |
| "广东人什么都吃" | 被骂刻板印象 | 中 |
| 城乡歧视（看不起农村人） | 道德谴责 | 高 |

### 性别歧视

| 行为 | 后果 | 敏感度 |
|------|------|--------|
| "女司机就是不行" | 被骂性别歧视 | 高 |
| "剩女/圣女"标签 | 女性群体反感 | 高 |
| 职场性别歧视（招聘限男性） | 被举报/舆论攻击 | 极高 |
| 家暴合理化 | 社会性死亡 | 极高 |
| "男人哭什么哭" | 被骂有毒男子气概 | 中 |

### 职业歧视

- **外卖员/快递员**：不能看不起，社会同情度高
- **清洁工/保安**：不能侮辱，会被拍视频传播
- **农民**：可以调侃但不能侮辱，涉及"忘本"
- **性工作者**：极度敏感，小说中需模糊处理

### 年龄歧视

- 称呼女性年龄需谨慎（不能叫"大妈""阿姨"，除非对方明显年长）
- "35岁以上不要"是职场潜规则但不能明说
- 调侃老人"碰瓷"会被骂不尊老

## 三、职场禁忌

### 法律级（可能坐牢）

| 行为 | 后果 | 例子 |
|------|------|------|
| 泄露商业机密 | 刑事责任+赔偿 | 程序员带走代码被判刑 |
| 职务侵占 | 刑事责任 | 会计挪用公款 |
| 受贿/行贿 | 刑事责任 | 采购吃回扣 |
| 竞业违约 | 高额赔偿（年薪数倍） | 跳槽到竞争对手被起诉 |

### 道德级（社死级别）

| 行为 | 后果 | 例子 |
|------|------|------|
| 办公室恋情（上下级） | 被辞退/调岗 | 总监和下属被发现 |
| 越级汇报 | 被直属领导针对 | 绕过主管直接找VP |
| 抢功/甩锅 | 失去信任、被孤立 | 把团队成果说成自己的 |
| 背后议论领导 | 被传话、穿小鞋 | 茶水间吐槽被录音 |
| 离职前挖人 | 破坏关系、可能被起诉 | 走的时候带走整个团队 |

### 潜规则级（不明说但重要）

- 不要问同事工资
- 不要在领导面前表现得比领导强
- 不要拒绝领导的饭局邀请（至少前几次）
- 不要在群里发与工作无关的内容
- 不要轻易站队（办公室政治）

## 四、社交禁忌

### 当众揭短

- **场景**：聚会时提及别人的失败经历、缺点、糗事
- **后果**：关系破裂、被记恨、社交圈传开
- **正确做法**：给人留面子，私下沟通

### 借钱不还

- **中国式借贷**：朋友借钱=考验友谊，不还=友谊终结
- **潜规则**：借钱时是孙子，还钱时是大爷
- **后果**：关系破裂、被朋友圈曝光、法律诉讼
- **正确做法**：不借或写借条，金额控制在"丢了不心疼"的范围

### 炫耀攀比

- **禁忌**：朋友圈炫富、饭局上吹嘘收入、比较孩子成绩
- **后果**：招人嫉妒、被借钱、被疏远
- **正确做法**：低调、自嘲、把优越感藏起来

### 背后议论

- **规则**：议论别人的话，一定会传到当事人耳朵里
- **后果**：信任崩塌、关系破裂、被反噬
- **正确做法**：当面沟通或闭嘴

## 五、家庭禁忌

### 干涉隐私

| 行为 | 后果 | 频率 |
|------|------|------|
| 翻看子女手机/日记 | 亲子关系破裂 | 极高 |
| 催婚催生到极端 | 子女断联/过年不回家 | 高 |
| 过问收入/存款 | 尴尬、被借钱 | 高 |
| 干涉子女恋爱对象 | 私奔/决裂 | 中高 |
| 未经同意安排相亲 | 反感、抗拒 | 高 |

### 重男轻女

- **表现**：家产给儿子不给女儿、女儿出嫁后就是"外人"、儿子犯错女儿买单
- **后果**：女儿怨恨、家庭关系破裂、被社会舆论谴责
- **现代转变**：城市地区大幅减少，农村/小城市仍存在

### 啃老

- **定义**：成年后仍依赖父母经济支持
- **现实**：房价高企导致"六个钱包"买房是常态，但"啃老不工作"是禁忌
- **灰色地带**：父母资助首付可以接受，但长期不工作靠父母养会被鄙视

### 不孝

- **底线**：不赡养老人是违法（可被起诉）
- **社会标准**：常回家看看、生病照顾、经济支持
- **禁忌**：把老人送养老院不管不问（社会舆论谴责）
- **现实矛盾**：工作忙+距离远+经济压力=有心无力

## 八、小说写作中的禁忌处理建议

1. **政治敏感**：用虚构背景，不直接映射现实事件
2. **地域歧视**：可以用角色表达歧视观点，但需要有反对声音或后果展示
3. **性别议题**：避免单方面站队，展示多方视角
4. **暴力/犯罪**：可以写但不要美化，需要有代价展示
5. **露骨内容**：点到为止，用暗示和留白，不要露骨描写
6. **宗教话题**：谨慎处理，避免贬低任何宗教
7. **真实人名/地名**：尽量避免，用虚构替代
