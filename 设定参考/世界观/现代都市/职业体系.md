## 一、体制内职业

### 公务员

| 层级 | 月薪（元） | 福利 | 晋升路径 |
|------|-----------|------|---------|
| 中央部委 | 8000-12000 | 分房/配租、子女入学 | 科员→处级→厅级（极难） |
| 省级机关 | 6000-10000 | 配租、稳定 | 科员→处级（天花板） |
| 市级机关 | 5000-8000 | 公积金高、医疗好 | 科员→科级→处级 |
| 县级/乡镇 | 4000-6000 | 基层补贴、工作强度大 | 晋升难、调动难 |

**特点**：铁饭碗、社会地位高、婚恋市场吃香、灰色收入（部分岗位）、工作稳定但枯燥

### 事业编

- **教师**：寒暑假、稳定、社会尊重，但工资低（尤其小学/初中）
- **医生**：收入高（含灰色）、培养周期长（本硕博10年+）、压力大
- **科研院所**：论文压力、项目经费、非升即走（高校）

### 国企

| 类型 | 月薪（元） | 特征 |
|------|-----------|------|
| 央企总部 | 10000-20000 | 名校门槛、福利好、晋升靠关系 |
| 央企地方 | 6000-12000 | 稳定、福利好、天花板明显 |
| 地方国企 | 5000-10000 | 区域垄断、关系更重要 |
| 国企劳务派遣 | 3000-6000 | 同工不同酬、无归属感 |

## 八、职业发展路径速查

**体制内路径**：
```
考公/考编 → 基层锻炼 → 科员 → 副科 → 正科 → 副处 → 正处（大多数人止步正科）
```

**互联网路径**：
```
校招/社招 → 初级岗 → 中级岗（3年）→ 高级岗（5年）→ 管理岗/专家岗（8年+）→ 被裁或转行
```

**金融路径**：
```
名校硕士 → 实习 → 初级分析师 → 高级分析师 → VP → ED → MD（极少数人）
```

**创业路径**：
```
打工积累 → 找到方向 → 融资/自筹 → 烧钱扩张 → 盈利或倒闭（90%失败率）
```

**蓝领路径**：
```
学徒/入行 → 熟练工 → 带徒弟/小包工头 → 转行或回老家（大多数人的归宿）
```
