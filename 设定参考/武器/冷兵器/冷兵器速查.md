# 冷兵器 · 写作参考

| 名称 | 产地 | 年代 | 写作场景 | 描写要点 |
|------|------|------|----------|----------|
| **太刀/打刀** / 日本刀(Katana) | 日本 | 平安-江户 | 武士道、忍者、日系奇幻 | 弯曲单刃，双手握持。折叠锻打钢，刀纹（刃文）独特。拔刀斩、居合一刀流。速度与优雅的象征 |
| **苗刀** / 中国苗刀 | 中国 | 明代 | 抗倭、中国武术、武侠小说 | 长柄长刃，兼具刀的劈砍和枪的长度。戚继光抗倭利器，双手使用 |
| **唐横刀** / 唐刀 | 中国 | 唐代 | 古装、武侠、唐代军事 | 直刃单面开锋，唐帝国军事力量的象征。影响了日本刀的设计，被称为日本刀的"祖先" |
| **长剑** / Longsword (Hand-and-a-half Sword) | 欧洲 | 中世纪 | 骑士、中世纪奇幻（《指环王》《权力的游戏》） | 双手或一手半握，十字护手，可劈砍可突刺。骑士精神的象征，铠甲缝隙攻击术 |
| **刺剑/细剑** / Rapier | 欧洲/西班牙 | 文艺复兴 | 决斗、佐罗、绅士暗杀 | 细长轻巧，以刺击为主，追求精准而非力量。16-17世纪欧洲绅士佩剑 |
| **大剑/阔剑** / Claymore | 苏格兰 | 15-17世纪 | 苏格兰勇士、高地战斗 | 双手巨剑，120-140cm长，四瓣护手。力量型战士的武器，一击可劈开盾牌 |
| **弯刀/弯刃大刀** / Scimitar/Saber | 中东/欧洲 | 中世纪-近代 | 骑兵、海盗、中东勇士 | 弯曲单刃，适合骑马劈砍。土耳其弯刀、骑兵军刀皆属此类 |
| **双节棍** / Nunchaku | 中国/日本 | 明代-现代 | 李小龙电影、武术表演 | 两节短棍以链条/绳连接，速度极快，近身格斗利器。李小龙使之闻名全球 |
| **匕首/短剑** / Dagger | 各文明 | 远古至今 | 暗杀、自卫、近身格斗 | 最古老的冷兵器之一，单手握持，便于隐藏。军刀/格斗刀/飞刀各类变体 |
| **长枪/矛** / Spear | 各文明 | 远古至今 | 古代战场、武侠小说 | "百兵之王"，战场主力武器。一寸长一寸强，可刺可扫。便宜易造，数量决定胜负 |
| **战斧** / Battle Axe | 北欧/各国 | 维京-中世纪 | 维京人、狂战士、矮人（奇幻） | 单手/双手大斧，劈砍破坏力强。维京人的标志武器，野蛮力量感 |
| **弓** / Longbow / Recurve Bow | 英国/蒙古/各国 | 中世纪 | 弓箭手、游侠、草原骑兵 | 英格兰长弓（射程200米+）、蒙古复合弓（骑射）。安静致命的远程冷兵器 |
| **弩** / Crossbow | 中国/欧洲 | 春秋-中世纪 | 狙击手前身、暗杀、城堡攻防 | 机械上弦精度高，可击穿轻甲。中国战国时期已有，欧洲中世纪严格管制（教廷曾试图禁用） |
| **战锤/钉头锤** / War Hammer/Mace | 欧洲/中东 | 中世纪 | 对抗重甲骑士、野蛮人 | 钝击武器，对铠甲有效。一锤下去震碎铠甲下的骨骼，不需要破甲即可杀伤 |
| **流星锤/连枷** / Morning Star/Flail | 欧洲 | 中世纪 | 暴力场面、宗教裁判所 | 链接铁球的武器，轨迹不可预测，可绕过盾牌。暴力美学，但实际使用困难 |
| **爪/手甲** / Katar/Push Dagger | 印度 | 中世纪 | 暗杀、印度武术、刺客 | 握把与刀刃成T形，直刺发力。手刺类暗器，隐蔽致命 |

## 写作要点

- **重量**：传递角色气质
- **温度**：金属的冰冷感
- **刃口反光**：视觉描写
- **握感**：皮革/木材/金属的触感
- **声音**：刀剑碰撞声、出鞘声
