## 描写通用要点

| 类型 | 要点 |
|------|------|
| **枪声** | 手枪"砰"、步枪"哒哒哒"、霰弹枪"轰"、机枪撕裂声 |
| **后坐力** | 肩膀被推、枪口跳动 |
| **硝烟** | 火药味、白烟、呛人 |
| **弹壳** | 叮当落地、金属光泽 |
| **爆炸** | 先看到光后听到声、冲击波、碎片飞溅 |
| **导弹** | 尾焰、音速、命中爆炸 |

## ⚠️ 时代红线（写作必查）

> 选武器前先看这条。写了不符合时代的武器=出戏。

| 武器 | 最早出现年代 | 时代红线 |
|------|-------------|----------|
| 手枪（格洛克/M1911） | 1980s/1911 | 二战及以前小说不能用格洛克 |
| 突击步枪（AK-47/M4/95式） | 1947/1960s/1995 | 二战/抗日小说不能用AK/M4；只能用栓动步枪 |
| 狙击步枪（巴雷特/AWM） | 1980s/1990s | 1980年前的小说不能用 |
| 冲锋枪（MP5/乌兹） | 1960s/1950s | 二战及以前不能用MP5 |
| 霰弹枪（雷明顿870） | 1950s | 二战及以前用猎枪替代 |
| 火箭筒（RPG-7） | 1961 | 越战及以前不能用RPG-7 |
| 反坦克导弹（标枪） | 1996 | 1996年前小说不能用 |
| C4炸药 | 1950s | 二战及以前不能用 |
| 巡航导弹（战斧） | 1983 | 1983年前小说不能用 |
| 狙击步枪（巴雷特M82） | 1982 | 1982年前小说不能用 |
| 反舰导弹（鹰击-62） | 2000 | 2000年前小说不能用 |
| 消音器（现代型） | 1940s | 古代/武侠小说绝对不能用 |
| 电击器（泰瑟枪） | 1990s | 1990年前小说不能用 |
| 防弹插板（现代陶瓷） | 1990s | 1980年前小说只能用钢制 |

**通用原则**：
- 抗日/二战 → 栓动步枪（98K/三八式）+驳壳枪 + 手榴弹 + 土炮
- 越战 → AK-47 出现，RPG-7 出现，M16 出现
- 现代 → 全部可用
- 未来 → 按设定调整，优先用"更新换代"的型号
