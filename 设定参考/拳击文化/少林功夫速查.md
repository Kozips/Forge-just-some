# 少林功夫 · 写作速查

> 中国武术的发源地——嵩山少林寺。
> 风格：内外兼修、硬功+套路+内功、佛门武学。

---

## 一、基本功

### 桩功

| 桩功 | 动作 | 作用 |
|------|------|------|
| 马步桩 | 双脚开立，膝盖弯曲，蹲马步 | 腿力+定力 |
| 弓步桩 | 前腿弓后腿蹬 | 前进发力 |
| 虚步桩 | 前脚虚点地面 | 轻灵 |
| 独立桩 | 单脚站立 | 平衡+腿力 |

### 腿功

| 腿法 | 动作 | 描写要点 |
|------|------|----------|
| 正踢腿 | 腿从正面踢起 | 基本功 |
| 侧踢腿 | 腿从侧面踢起 | 柔韧性 |
| 里合腿 | 腿画内弧踢 | 像画圈 |
| 外摆腿 | 腿画外弧踢 | 像画圈 |
| 旋风腿 | 跳起360°横扫 | 华丽 |

## 二、少林七十二绝技（部分）

| 绝技 | 类型 | 描写要点 |
|------|------|----------|
| 铁砂掌 | 手部硬功 | 手掌劈砖/碎石 |
| 金钟罩 | 全身硬功 | 抗击打——挨打不疼 |
| 铁头功 | 头部硬功 | 用头撞碎东西 |
| 鹰爪功 | 指力 | 手指能插木头 |
| 一指禅 | 单指 | 一根手指支撑全身重量 |
| 轻功 | 身法 | 跑墙、跳高、落地无声 |
| 铁臂功 | 手臂硬功 | 手臂能挡刀剑 |
| 排打功 | 全身 | 用木棍/铁棍打身体→身体变硬 |

**描写要点**：少林硬功的训练过程非常有画面感——"他的手掌反复插入滚烫的铁砂里，皮肤从红变黑，从黑变厚。三年后，他的手掌像一块铁板。"

## 三、少林拳法套路

| 套路 | 特点 | 写作用途 |
|------|------|----------|
| 少林长拳 | 大开大合、刚猛 | 展示力量 |
| 小洪拳 | 紧凑、快速 | 展示速度 |
| 大洪拳 | 刚猛有力 | 展示实力 |
| 通背拳 | 手臂伸展、远距离 | 展示臂力 |
| 罗汉拳 | 模仿罗汉姿态 | 展示佛门身份 |
| 醉拳 | 模仿醉酒状态 | 华丽+迷惑对手 |

## 四、兵器

少林兵器是武术描写的重要元素。

| 兵器 | 特点 | 描写要点 |
|------|------|----------|
| 少林棍 | 少林最著名——"天下武功出少林，少林武功棍为先" | 棍花、扫、劈、点 |
| 少林刀 | 刚猛劈砍 | 劈、砍、撩 |
| 少林剑 | 轻灵快速 | 刺、点、撩 |
| 少林枪 | 长距离 | 扎、拦、拿 |
| 禅杖 | 重兵器 | 横扫千军 |

## 五、少林文化

| 元素 | 描写 |
|------|------|
| 寺庙 | 嵩山深处、古木参天、晨钟暮鼓 |
| 僧人 | 光头、僧袍、念珠、戒律 |
| 晨练 | 凌晨4点起床、跑步、站桩、练拳 |
| 戒律 | 不杀生、不饮酒、不食荤腥 |

---

## 六、写作要点

1. **硬功训练最有画面感**——铁砂掌、金钟罩的训练过程很残酷
2. **少林棍是灵魂**——少林最标志性的兵器
3. **晨钟暮鼓的氛围**——少林寺的描写要有佛门意境
4. **俗家弟子**——不出家但学武，是常见的角色设定
5. **武侠必备**——少林是武侠小说的常客
