# 太极拳 · 写作速查

> 中国传统武术，以柔克刚、借力打力、四两拨千斤。
> 风格：慢中有快、柔中有刚、以静制动、后发先至。

---

## 一、基本站姿与步法

### 站姿

- 双脚与肩同宽，膝盖微弯
- 重心均匀分布或微偏后脚
- 双手自然抬起，掌心朝内，如抱球
- 含胸拔背，沉肩坠肘，虚灵顶劲

### 步法

| 步法 | 动作 | 用途 |
|------|------|------|
| 弓步 | 前腿弯曲，后腿伸直 | 前进发力 |
| 虚步 | 前脚虚点地面 | 试探+随时变向 |
| 马步 | 双脚开立，膝盖弯曲 | 稳固重心 |
| 侧行步 | 横向移动 | 调整角度 |
| 碾步 | 脚掌碾地旋转 | 转身借力 |

## 二、核心理念（写作时必须体现）

| 理念 | 含义 | 写作体现 |
|------|------|----------|
| 以柔克刚 | 不硬碰，用巧劲化解 | 对手一拳打来→他轻轻一拨→对手的力量打空了 |
| 借力打力 | 用对手的力量打对手 | 对手推他→他顺势一拉→对手自己飞出去了 |
| 四两拨千斤 | 用很小的力化解很大的力 | 对手全力一拳→他只用两根手指一拨→拳路偏了 |
| 后发先至 | 等对手先动→后出手但先到 | 对手出拳的瞬间他动了→后发的拳先到了 |
| 以静制动 | 不动→等对手动→再动 | 他站在那里不动→对手不知道该打哪里→一出手就破绽百出 |

## 三、核心招式

### 太极推手（实战核心）

推手是太极的双人训练/实战形式——两人手臂接触→感知对方力量→借力打力。

| 推手技法 | 动作 | 描写要点 |
|----------|------|----------|
| 掤 | 双手向外撑开 | 像一个气球膨胀 |
| 捋 | 顺着对方的力量往旁边带 | 像水流引导 |
| 挤 | 双手向前挤推 | 像挤牙膏 |
| 按 | 双手向下按压 | 像按住水面 |
| 采 | 抓住对方某个点往下拽 | 像摘果子 |
| 捌 | 横向扭转对方 | 像拧毛巾 |
| 肘 | 用肘攻击 | 近距离 |
| 靠 | 用肩膀/后背撞击 | 类似八极的铁山靠但更柔和 |

**描写要点**：推手的精髓是"听劲"——通过手臂接触感知对方力量方向→然后做出反应。"他的手搭上对方的手臂，像蜻蜓点水——但对方每一丝力量的变化他都知道。"

### 太极套路（招式名非常有意境）

| 招式 | 动作 | 写作意境 |
|------|------|----------|
| 野马分鬃 | 双手上下分开→像拨开马鬃 | 大开大合 |
| 白鹤亮翅 | 双臂展开→像白鹤展翅 | 优雅从容 |
| 搂膝拗步 | 一手搂膝→一手前推 | 攻防一体 |
| 云手 | 双手在胸前画圆→像云在飘 | 行云流水 |
| 单鞭 | 一手前推→一手后展 | 像鞭子甩出 |
| 高探马 | 单手前探→像骑在马上探路 | 试探 |
| 玉女穿梭 | 左右穿插→像织女穿梭 | 灵活 |
| 海底针 | 身体下沉→手指向下 | 像从海底捞针 |

## 四、太极的"刚柔并济"

太极不是纯柔——**柔是手段，刚是目的**。

| 阶段 | 表现 | 描写 |
|------|------|------|
| 化劲（柔） | 消解对手的攻击力量 | "他像棉花一样，打上去没有着力点" |
| 引劲（过渡） | 引导对手失去平衡 | "他轻轻一拉，对手不由自主地往前倾" |
| 发劲（刚） | 瞬间爆发出全部力量 | "他突然一掌推出——刚才的棉花变成了铁锤，对手飞了出去" |

**关键**：太极的发力是**瞬间爆发**——平时慢悠悠的→突然一下极快极猛。"他一直在画圈→突然直线一掌→速度快到对手眼睛都没跟上→掌根撞上胸口→对手飞出三步。"

## 五、太极拳的文化意境

| 元素 | 描写 |
|------|------|
| 太极图 | 阴阳鱼——柔中有刚，刚中有柔 |
| 晨练 | 清晨公园/广场，老人打太极，鸟叫声，阳光 |
| 道家哲学 | 无为而治、以柔克刚、上善若水 |
| 养生 | 太极有养生的一面——呼吸调节、气血运行 |
| 内功 | 太极讲"内劲"——力量从体内发出，不是肌肉的力量 |

## 六、太极拳流派

| 流派 | 特点 | 写作适用 |
|------|------|----------|
| 陈式太极拳 | 有快有慢、有发劲、刚柔并济 | 最适合写实战——有爆发力 |
| 杨式太极拳 | 舒展大方、匀速柔和 | 适合写养生/意境 |
| 吴式太极拳 | 小巧紧凑、柔化为主 | 适合写技巧型高手 |
| 武式太极拳 | 小架、紧凑、重内劲 | 适合写内家高手 |
| 孙式太极拳 | 步法灵活、开合转换 | 适合写灵活型角色 |

## 七、太极拳 vs 其他武术

| 对抗 | 太极优势 | 太极劣势 |
|------|----------|----------|
| vs 八极拳 | 化力借力 | 被铁山靠硬冲难接 |
| vs 形意拳 | 后发先至 | 形意速度太快 |
| vs 泰拳 | 借力打力 | 泰拳力量太大不好化 |
| vs 散打 | 近身推手技术 | 散打距离控制好 |
| vs 咏春 | 以柔克刚 | 咏春快拳不好防 |

## 八、写作常见错误

1. **太极只慢不快**——太极套路是慢的，但实战发力是瞬间爆发的
2. **太极没有攻击力**——太极的"发劲"威力极大，一掌能打飞人
3. **太极只适合老人**——年轻高手也可以用太极，陈式太极有快有慢
4. **以柔克刚=不用力**——四两拨千斤是用巧劲，不是不用力
5. **忽略"听劲"**——太极的核心是感知对方力量，不接触就打不了
