# 跆拳道 · 写作速查

> 韩国国技，以腿法为主的竞技武术。"手是两扇门，全凭腿打人"。
> 风格：华丽、快速、高踢为主、观赏性强。

---

## 一、基本站姿与步法

### 站姿

- 双脚与肩同宽，前脚微内扣
- 重心居中或略偏后脚（方便起高腿）
- 双手举在胸前，比拳击低（跆拳道手部攻击有限）
- 膝盖微弯，随时准备起腿

### 步法

| 步法 | 动作 | 用途 |
|------|------|------|
| 前滑步 | 前脚滑进 | 拉近距离出腿 |
| 后滑步 | 后脚后撤 | 退出攻击范围 |
| 侧滑步 | 横向移动 | 调整角度 |
| 跳换步 | 双脚跳起交换前后 | 换势（Orthodox↔Southpaw） |
| 垫步 | 后脚蹬地前冲 | 突然拉近距离+出腿 |

## 二、腰带等级体系（角色背景速查）

| 腰带 | 韩语 | 含义 | 角色定位 |
|------|------|------|----------|
| 白带 | 白 | 新手，什么都不会 | 刚入门的角色 |
| 黄带 | 黄 | 学会基本动作 | 学了几个月 |
| 绿带 | 绿 | 能实战了 | 有一定基础 |
| 蓝带 | 蓝 | 技术成型 | 中级选手 |
| 红带 | 红 | 接近黑带 | 高手 |
| 黑带一段 | 一品 | 入门高手 | 黑带才是真正的开始 |
| 黑带三段 | 三品 | 教练级 | 可以开道馆 |
| 黑带五段+ | 五品 | 大师级 | 宗师/前辈角色 |

**描写要点**：腰带颜色是最直观的实力标签。"他腰间系着红带——再升一级就是黑带了。"

## 三、核心攻击技术

### 腿法（跆拳道的灵魂）

| 腿法 | 韩语 | 动作 | 描写要点 |
|------|------|------|----------|
| 前踢 | Ap Chagi | 膝盖抬高→脚面弹出 | 最基本的腿法，像弹射 |
| 横踢 | Dollyo Chagi | 转腰→小腿横扫 | 跆拳道最常用——速度快、角度刁 |
| 侧踢 | Yop Chagi | 侧身→脚底蹬出 | 力量大，像一堵墙推过来 |
| 下劈 | Naeryeo Chagi | 脚从上往下劈 | 目标头顶，像斧头劈柴 |
| 后踢 | Dwi Chagi | 转身→脚跟后蹬 | 转身借力，距离远 |
| 旋风踢 | Dwi Huryeo Chagi | 360°转身→横踢 | 跆拳道招牌——华丽、高风险、KO技 |
| 双飞踢 | Narae Chagi | 腾空连踢两脚 | 跳起来左右连踢 |

**横踢（最核心）**：提膝→转腰→小腿像鞭子一样甩出→用脚背击打。"他的横踢像鞭子一样甩出，啪的一声打在对手护具上。"

**旋风踢（招牌技）**：跳起→身体360°旋转→空中横踢→落地。"他整个人腾空旋转，像一阵旋风，脚背啪地抽在对手头部。对手摇晃了两下，倒了。"

### 手法（有限但存在）

| 手法 | 动作 | 场景 |
|------|------|------|
| 直拳 | 前手直线推出 | 竞技跆拳道只允许打躯干 |
| 手刀 | 掌缘劈击 | 传统跆拳道/品势 |
| 肘击 | 肘部攻击 | 传统跆拳道（竞技不允许） |
| 掌根击 | 手掌根部推击 | 防守反击 |

**竞技限制**：现代竞技跆拳道**不能用手打头**，只能踢。所以手的作用主要是防守和控制距离。

## 四、防守技术

| 技术 | 动作 | 描写要点 |
|------|------|----------|
| 格挡 | 用小臂/小腿挡住来腿 | 硬碰硬 |
| 闪躲 | 侧身/后仰 | 跆拳道闪躲幅度比拳击大 |
| 提膝格挡 | 提膝用小腿挡住低踢 | 类似泰拳 |
| 推挡 | 用手推开对手 | 制造出腿距离 |

## 五、竞技跆拳道 vs 传统跆拳道

| 方面 | 竞技跆拳道（WT） | 传统跆拳道（ITF） |
|------|-----------------|-----------------|
| 重点 | 踢击得分 | 拳脚并用 |
| 手法 | 只能打躯干，不能打头 | 可以打头 |
| 护具 | 电子护具（踢中自动计分） | 传统护具 |
| 风格 | 快进快出、点到为止 | 更接近实战 |
| 腿法 | 高腿为主（头部=3分） | 中低腿更多 |
| 品势 | 有 | 有（套路表演） |

**写作选择**：写竞技比赛用WT规则（电子护具、踢头3分）；写街头实战/传统用ITF风格。

## 六、跆拳道文化

| 元素 | 描写 |
|------|------|
| 道馆 | 脱鞋进入、木地板/垫子、墙上挂国旗和师范照片、喊声震天 |
| 道服 | 白色道服+腰带，赤脚，道服很宽松 |
| 敬礼 | 进出道馆要鞠躬、对练前后要鞠躬、对师范要敬礼 |
| 品势 | 固定套路表演——太极一章到八章，高段有高丽、金刚等 |
| 击破 | 用拳/脚打破木板/砖头——考试必考 |
| 喊声 | 出招时大喊"啊"（기합/气合）——气势+发力 |

## 七、跆拳道 vs 其他流派

| 对抗 | 跆拳道优势 | 跆拳道劣势 |
|------|-----------|-----------|
| vs 拳击 | 腿比拳远，控制距离 | 近身拳法差距大 |
| vs 泰拳 | 高腿速度快 | 近身肘膝完全打不过 |
| vs 散打 | 腿法更华丽更灵活 | 被接腿摔克制 |
| vs 空手道 | 腿法更全面 | 手法不如空手道 |
| vs 街头打架 | 踢腿速度快 | 没有地面战能力 |

## 八、写作常见错误

1. **忽略竞技限制**——竞技跆拳道手不能打头，只能踢
2. **腿法描写太暴力**——跆拳道是"点到为止"的计分运动，不是往死里踢
3. **没有品势/礼仪**——跆拳道非常注重礼仪，写比赛不写敬礼就少了味道
4. **忽略电子护具**——现代跆拳道用电子护具自动计分，不是裁判肉眼看
5. **高踢太容易**——现实中华丽的高踢很容易被接腿摔倒
