# 八卦掌 · 写作速查

> 中国传统武术，以游走绕圈为核心，身法诡异。
> 风格：走为上计、身法诡异、以走为攻、绕着打。

---

## 一、基本站姿与步法

### 站姿

- 双脚略宽于肩，膝盖微弯
- 双手举在胸前，掌心朝外（八卦掌用掌不用拳）
- 上身微含，重心略低

### 步法（八卦掌的灵魂）

| 步法 | 动作 | 描写要点 |
|------|------|----------|
| 蹚泥步 | 脚贴着地面走，像在泥里趟 | 脚不离地——稳且快 |
| 扣步 | 脚尖内扣 | 转弯时用 |
| 摆步 | 脚尖外摆 | 转弯时用 |
| 走圈 | 绕着对手/中心走圆圈 | 八卦掌标志——一直在转 |

**走圈（八卦掌核心）**：八卦掌一切技术都在"走圈"中完成——绕着对手转→找角度→出手。

"他像陀螺一样绕着对手转。对手转身想面对他——他已经到了背后。对手再转身——他又到了侧面。"

## 二、核心招式

### 八掌（八卦掌八个基本掌型）

| 掌型 | 动作 | 描写要点 |
|------|------|----------|
| 单换掌 | 一手前伸→换方向时换手 | 基本走圈时的掌型 |
| 双换掌 | 双手交换 | 走圈变化 |
| 顺势掌 | 顺着身体方向出掌 | 借力 |
| 背身掌 | 背对对手出掌 | 背后一掌——出其不意 |
| 翻身掌 | 翻身出掌 | 身法+掌法 |
| 磨身掌 | 贴着对手身体转→出掌 | 像磨盘一样转 |
| 三穿掌 | 连续三次穿掌 | 快速连击 |
| 回身掌 | 转身回手一掌 | 回马枪 |

### 八卦掌的"游身"

八卦掌的特点是**一直在移动**——不是站住了打，是走着打。

| 身法 | 描写 |
|------|------|
| 穿 | 从对手防守缝隙中穿过 |
| 闪 | 侧身闪避+同时出掌 |
| 翻 | 翻身改变方向 |
| 转 | 绕着对手转圈 |
| 走 | 一直在走，不站住 |

## 五、八卦掌 vs 其他武术

| 对抗 | 八卦优势 | 八卦劣势 |
|------|----------|----------|
| vs 八极拳 | 身法诡异，打不到 | 被铁山靠冲到就完 |
| vs 形意拳 | 走位绕开直线攻击 | 形意速度太快 |
| vs 太极 | 游走不给太极接触 | 太极听劲克制近身 |
| vs 泰拳 | 身法闪避 | 近身肘膝太猛 |

## 六、写作要点

1. **走圈是灵魂**——八卦掌的描写必须有"一直在转"的感觉
2. **身法诡异**——对手永远打不到他，因为他不在你以为的位置
3. **用掌不用拳**——八卦掌用手掌，不用拳头
4. **武侠适用**——八卦掌的身法非常适合写武侠中的轻功高手
5. **以走为攻**——八卦掌不是正面硬打，是绕着打
