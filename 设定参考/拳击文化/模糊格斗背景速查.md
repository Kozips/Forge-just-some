# 模糊格斗背景速查

> 当角色设定是"会一些拳脚""能打""练过"但没有指定具体流派时，用这个文件。
> 核心：现实中"会打"的人往往不是正经学过某一门派，而是各种东西混着来。

---

## 一、"会一些拳脚"的真实来源

| 来源 | 程度 | 描写要点 |
|------|------|----------|
| **体校练过** | 中等 | 散打/跆拳道/柔道练过几年，没到专业水平，但比普通人强很多 |
| **当兵退伍** | 中高 | 军体拳+格斗训练，招式简单但实用，抗击打强 |
| **街头打出来的** | 不稳定 | 没有系统训练，但经验丰富，出手没规矩但有效 |
| **健身房练的** | 低中 | 有力量但没技术，出拳全靠蛮力 |
| **小时候学过** | 低 | 跆拳道/空手道练过一两年，会基本动作但实战少 |
| **看视频自学** | 很低 | 动作看起来像那么回事但经不起实战 |
| **农村/工地** | 中等 | 体力活练出来的力量+经常打架的经验 |

## 二、不同"会打"程度的写法

### 1. "稍微练过"（打普通人够用）

**典型特征**：
- 出拳有章法——知道打哪里、怎么发力
- 但只有几招——翻来覆去就那么两三下
- 遇到真正的练家子就不够看了

**写法**：
> 他一拳打过来，角度不错——是从体校学的刺拳，直奔鼻梁。但后面的衔接就乱了——没有组合拳，打完一拳就愣在那里。
>
> 对付普通人够用。但对手侧身一闪，反手一个直拳打在他下巴上——专业和业余的差距，一拳就看出来了。

### 2. "当过兵/练过散打"（比普通人强一个档次）

**典型特征**：
- 招式简洁实用——没有花架子
- 抗击打能力强——挨了一拳还能站住
- 有摔法意识——近身会本能地去摔
- 体力好——能打持久战

**写法**：
> 他没有摆什么架势——就是双脚前后分开，双手举起来护住脸。军人的习惯，简单、直接、不浪费。
>
> 对手冲过来，他侧身让过，顺势搂住对方脖子往下一按——膝盖撞上腹部。对手弯下腰，他又补了一肘。
>
> 不好看。但有效。

### 3. "街头打架厉害"（经验丰富但没技术）

**典型特征**：
- 出手没有章法——但狠、快、不讲规矩
- 善用环境——抄凳子、推桌子、利用地形
- 第一下最猛——先手打完后面就乱了
- 被打了不怕——抗击打靠的是"习惯挨打"

**写法**：
> 他没有架势——直接抄起桌上的啤酒瓶砸过去。瓶子碎了，对手捂着头，他又是一脚踹在膝盖上。
>
> 没有章法。没有招式。但每一招都是在巷子里打出来的——够狠、够快、够不要命。

### 4. "练过但很久没练了"（生疏但底子在）

**典型特征**：
- 身体还记得——肌肉记忆没完全消失
- 但反应慢了——以前能躲的现在躲不开
- 体力不够——打两下就喘
- 关键时刻会本能地用出以前的招式

**写法**：
> 他已经十年没练了。但当拳头飞过来的瞬间，他的身体自己动了——左手格挡，右拳冲出。散打的底子，十年没练，但身体记得。
>
> 只是太慢了。以前这一拳能打中，现在对手轻松躲开了。他喘着气，膝盖在发抖——不是怕，是体力跟不上了。

### 5. "跟XX学过几招"（半吊子）

**典型特征**：
- 会几个特定招式——但不会连贯
- 知道原理——但做不到位
- 打得出第一下——第二下就断了

**写法**：
> "我跟一个退伍的哥们儿学过。"他说完，摆出一个歪歪扭扭的格斗架势。
>
> 第一拳打得还不错——直拳，角度对了，力量也够。但打完之后他就卡住了——不知道接下来该做什么。
>
> 那个哥们儿教他的第二招是什么来着？他想不起来了。

## 三、"会打"的通用描写库

### 出拳时

| 水平 | 描写 |
|------|------|
| 不会打 | 一拳挥过去——全靠手臂力量，拳头打在对手肩膀上 |
| 稍微练过 | 一拳打出去——转腰送肩，拳头正中鼻梁 |
| 练过几年 | 刺拳试探→直拳跟进→摆拳收尾，三拳一气呵成 |
| 专业级 | 前手虚晃一下→对手防守偏移→后手直拳从缝隙中穿过→正中下巴 |

### 挨打时

| 水平 | 描写 |
|------|------|
| 普通人 | 挨了一拳→眼前一黑→腿软→坐到地上 |
| 稍微练过 | 挨了一拳→晃了一下→咬牙站住→还能打 |
| 练过几年 | 挨了一拳→本能地抱住对手→拖入近战→用摔法 |
| 专业级 | 挨了一拳→身体自动后仰卸力→反手就是一拳→以牙还牙 |

### 被摔倒时

| 水平 | 描写 |
|------|------|
| 普通人 | 被摔→后背砸地→喘不上气→爬不起来 |
| 稍微练过 | 被摔→本能地用手撑地→减少伤害→还能爬起来 |
| 练过几年 | 被摔→空中本能地缩成一团→落地时用肩膀滚翻→立刻站起来 |

## 四、不同身份的格斗风格速查

| 身份 | 风格 | 典型动作 |
|------|------|----------|
| 退伍军人 | 简洁实用 | 直拳+膝撞+肘击，不花哨 |
| 体校生 | 有章法但单一 | 会一两种技术，翻来覆去用 |
| 健身教练 | 力量大技术差 | 一拳很猛但出拳慢，不会防守 |
| 工地工人 | 力气大+不怕打 | 出拳全靠蛮力但真的疼 |
| 校园混混 | 先手猛后面弱 | 第一下冲劲十足，打完就懵 |
| 黑道打手 | 狠+脏 | 专打要害，咬人、插眼、踢裆都来 |
| 保安/警察 | 控制为主 | 不想打伤你，只想按住你 |
| 街头rapper | 嘴比拳头硬 | 嘴上很凶，真打起来一般 |

## 五、"会打"vs"不会打"的差距描写

写格斗场景时，体现差距比描写技术更重要。

| 差距类型 | 描写 |
|----------|------|
| 速度差距 | "他的拳头还没伸出去，对手的拳已经收回来了" |
| 力量差距 | "他用尽全力打了一拳——对手晃了一下。对手回了一拳——他飞了出去" |
| 技术差距 | "他出了三拳，一拳都没打中。对手只出了一拳——正中下巴" |
| 经验差距 | "他以为对手要出拳——对手出了腿。他以为对手要进攻——对手在等他进攻" |
| 体力差距 | "前三回合他还能打。第四回合开始喘。第五回合站都站不住了" |
