# 空手道 · 写作速查

> 日本传统武术，"空手"=不用武器。
> 风格：刚猛正面对决、一击必杀、注重礼仪和精神修行。

---

## 一、基本站姿与步法

### 站姿

- 三战步（前虚后实，重心偏后）
- 双拳握紧，护在腰间或胸前
- 下巴收紧，肩膀下沉
- 脚趾抓地

### 步法

| 步法 | 动作 | 用途 |
|------|------|------|
| 前进/后退 | 前脚进后脚跟 | 基本移动 |
| 送足 | 后脚推前脚前进 | 快速推进 |
| 引足 | 前脚拉后脚后退 | 快速后撤 |
| 回旋步 | 转身移动 | 换角度/接回旋踢 |

## 三、核心攻击技术

### 手技

| 技法 | 日语 | 动作 | 描写要点 |
|------|------|------|----------|
| 正拳突 | Seiken Tsuki | 直拳（握拳方式不同——拇指压在食指中指上） | 像子弹射出 |
| 里拳 | Uraken | 用拳背横扫 | 近距离甩击 |
| 手刀 | Shuto | 掌缘劈击 | 像刀一样劈下去 |
| 贯手 | Nukite | 手指直刺 | 目标眼睛/喉咙 |
| 肘击 | Empi | 肘部攻击 | 近距离 |

### 足技

| 技法 | 日语 | 动作 | 描写要点 |
|------|------|------|----------|
| 前踢 | Mae Geri | 脚掌/脚尖前蹬 | 控制距离 |
| 横踢 | Yoko Geri | 侧身用脚底踹 | 像推墙 |
| 回旋踢 | Mawashi Geri | 转腰横踢 | 空手道招牌 |
| 后回旋踢 | Ushiro Mawashi Geri | 转身360°回旋踢 | 华丽KO技 |
| 下段踢 | Gedan Geri | 低踢小腿/膝盖 | 消耗对手 |

### 空手道的"型"（套路）

| 型 | 特点 | 写作用途 |
|----|------|----------|
| 平安型 | 基础型，初学者必练 | 角色入门场景 |
| 铁骑型 | 力量型，动作刚猛 | 展示力量 |
| 披塞型 | 速度型，动作快速 | 展示速度 |
| 观空大 | 高级型，动作优美 | 展示修为 |

## 五、空手道文化

| 元素 | 描写 |
|------|------|
| 道场 | 木地板、正面挂道场旗/师范照片、两侧挂流派旗帜 |
| 道服 | 白色空手道服（比跆拳道服厚）、腰带、赤脚 |
| 行礼 | 进出道场鞠躬、对练前后鞠躬、对师范敬礼 |
| 气合 | 出招时大喊"呀""嘶"——发力+气势 |
| 一击必杀 | 空手道的理念——一拳解决战斗 |
| 组手 | 对练/实战练习 |
| 型 | 套路表演——一个人做固定招式组合 |

## 六、空手道 vs 其他武术

| 对抗 | 空手道优势 | 空手道劣势 |
|------|-----------|-----------|
| vs 跆拳道 | 手法更强 | 腿法不如跆拳道华丽 |
| vs 泰拳 | 腿法速度 | 近身肘膝差距大 |
| vs 拳击 | 有腿法 | 拳法精细度不如 |
| vs 柔道 | 站立打击 | 被摔法克制 |
| vs 咏春 | 力量更大 | 近身快拳不好防 |

## 七、写作要点

1. **一击必杀的哲学**——空手道追求一招制敌，不打消耗战
2. **礼仪是核心**——空手道极度注重礼仪，写空手道不能不写敬礼
3. **型的美感**——套路表演有独特的美感，可以写角色修行场景
4. **日式道场**——木地板、行礼、师范、道服=典型的日式武道氛围
5. **回旋踢**——空手道最华丽的招式，适合写KO场景
