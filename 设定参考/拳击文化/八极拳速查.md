# 八极拳 · 写作速查

> 中国传统武术，"文有太极安天下，武有八极定乾坤"。
> 风格：贴身短打、刚猛暴烈、发力整、速度快、一步到位。

---

## 一、基本站姿与步法

### 站姿

- 四六步（前四后六），重心偏后
- 双脚略宽于肩，膝盖内扣（八极特有）
- 双手举在胸前，掌心朝内（不是拳，是掌）
- 含胸拔背，沉肩坠肘

### 步法

| 步法 | 动作 | 用途 |
|------|------|------|
| 闯步 | 后脚蹬地→前脚大步踏入 | 八极标志——一步跨进贴身距离 |
| 碾步 | 脚掌碾地旋转 | 发力时的转腰借力 |
| 震脚 | 脚跺地 | 借地面反作用力发力+气势压制 |
| 后撤步 | 前脚后撤 | 退后调整 |

**闯步（八极灵魂）**：后脚猛蹬→整个人像一发炮弹冲进对手怀里→贴身出招。"他一个闯步冲进对手怀里，还没等对方反应，肘已经撞上了胸口。"

## 二、核心攻击技术

### 八极拳核心招式

| 招式 | 动作 | 描写要点 |
|------|------|----------|
| 金刚八式 | 八个基本动作的组合 | 八极入门套路 |
| 马步冲拳 | 马步站稳→直拳冲出 | 稳如泰山的一拳 |
| 铁山靠 | 用肩膀/后背撞击对手 | 八极招牌——像一堵墙撞过来 |
| 跺子脚 | 脚跺地+出招 | 借震脚的反作用力打出重击 |
| 顶心肘 | 肘尖从下往上顶 | 目标胸口/下巴，近距离爆发 |
| 抱肘 | 双手抱臂→肘尖外露→向前冲撞 | 用肘尖当武器冲锋 |
| 两仪顶 | 双掌向前推出 | 八极的发力方式——整劲 |

### 铁山靠（八极拳的灵魂招式）

**动作**：沉身→后脚蹬地→肩膀/后背撞向对手→同时手臂张开护住两侧

**描写**：
- "他沉身蓄力，后脚猛蹬，整个人像一辆卡车撞了过来——肩膀正中对方胸口，对手腾空飞出。"
- "他的后背像一堵墙撞上来，对方被弹开三步，捂着胸口喘不上气。"

**发力原理**：力从脚底→经腿→传到腰→传到肩→瞬间爆发。全身力量集中在一个点。

### 八极拳发力特点

- **整劲**：全身力量集中在一个点爆发，不是单靠手臂
- **十字劲**：力的方向是十字形——上下+前后同时发力
- **沉坠劲**：力量向下沉，像一块石头砸下去
- **劲短力猛**：发力距离极短但力量极大——贴身就能打

## 三、八极拳与其他武术的关系

### 八极+劈挂

传统上八极拳和劈挂掌是搭配练的——八极主近战，劈挂主远攻。

- 八极：贴身短打、刚猛
- 劈挂：手臂伸展、大开大合、远距离抽打

### 八极+擒拿

八极拳的近身技术可以接擒拿——贴身后抓住对手关节/要害。

## 四、八极拳文化

| 元素 | 描写 |
|------|------|
| 训练方式 | 打桩（用身体撞木桩）、震脚、站桩、打沙袋 |
| 铁砂掌 | 用手反复插铁砂→手掌变硬→能劈砖 |
| 硬功 | 八极拳注重硬功——抗击打能力极强 |
| 师承 | 八极拳非常讲究师承，有明确的门派传承 |
| 河北沧州 | 八极拳发源地，可以做角色背景 |

## 五、八极拳 vs 其他武术

| 对抗 | 八极优势 | 八极劣势 |
|------|----------|----------|
| vs 太极 | 铁山靠硬冲 | 被太极化力借力 |
| vs 咏春 | 力量碾压 | 咏春快拳连击不好防 |
| vs 形意拳 | 都是刚猛，八极更爆 | 形意更简洁直接 |
| vs 泰拳 | 贴身肘法互有胜负 | 泰拳膝法更凶 |
| vs 散打 | 贴身爆发极强 | 散打距离控制更好 |

## 六、写作要点

1. **闯步+铁山靠=八极标志**——一步冲进贴身距离→肩膀撞飞对手
2. **震脚是气势**——八极的跺脚声像战鼓，有压迫感
3. **贴身短打**——八极不跟你打远距离，直接冲进怀里
4. **力量感**——八极的描写要突出"重""沉""猛"
5. **武侠适用**——八极拳的刚猛风格非常适合武侠/仙侠中的武者角色
