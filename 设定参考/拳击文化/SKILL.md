## 如何选择流派

### 按角色气质选

| 气质 | 推荐流派 | 原因 |
|------|----------|------|
| 暴躁/暴力 | 泰拳、八极拳 | 肘膝近战、刚猛短打 |
| 冷静/精准 | 截拳道、空手道 | 一击必杀、截击反击 |
| 聪明/技术型 | 巴西柔术、咏春 | 以小博大、技术压制力量 |
| 飘逸/神秘 | 太极拳、八卦掌 | 以柔克刚、身法诡异 |
| 全面/职业 | MMA、散打 | 站立+地面全覆盖 |
| 年轻/热血 | 跆拳道、空手道 | 华丽腿法、道场修行 |
| 老成/沉稳 | 形意拳、太极拳 | 简洁直接、大巧若拙 |

### 按场景选

| 场景 | 推荐流派 |
|------|----------|
| 职业赛事 | 拳击、MMA、散打、跆拳道 |
| 地下黑拳 | 泰拳、MMA、散打 |
| 街头实战 | 散打、截拳道、泰拳 |
| 武侠/仙侠 | 八极拳、太极、八卦掌、形意拳、少林 |
| 学校/社团 | 跆拳道、空手道、柔道 |
| 女性格斗家 | 咏春、跆拳道、巴西柔术 |

### 流派对抗速查

| 对抗组合 | 优劣 | 写作张力 |
|----------|------|----------|
| 泰拳 vs 拳击 | 泰拳肘膝占优，拳击步法占优 | 近战vs距离控制 |
| 八极拳 vs 太极 | 八极硬冲，太极化力 | 刚vs柔的经典对决 |
| 巴西柔术 vs 泰拳 | 地面战泰拳完全劣势 | 站立之王被拖入地面 |
| 咏春 vs 空手道 | 咏春快拳连击vs空手道一击重拳 | 速度vs力量 |
| 散打 vs 跆拳道 | 散打摔法克制腿法 | 踢腿被接住摔倒 |
| 形意拳 vs 八极拳 | 都是刚猛路线 | 硬碰硬的对轰 |

## 格斗描写通用公式

```
站姿/距离 → 试探（刺拳/假动作）→ 一方出击 → 防守/反击 → 
节奏变化（快慢交替）→ 关键一击 → 身体反应 → 结果
```

### 通用五感速查

| 感官 | 格斗场景 |
|------|----------|
| 视觉 | 汗珠飞溅、血丝、拳头在眼前放大、视线模糊 |
| 听觉 | 拳头撞击声、骨裂声、喘息声、观众声、耳鸣 |

| 味觉 | 嘴里的铁锈味（血）、口干舌燥 |
