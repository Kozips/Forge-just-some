# 黑帮体系

> 跨国家复用——龙国用"黑社会"，泡菜国用"组织暴力团"，绅士国用"gangster"。

---

## 一、组织结构

| 层级 | 职位 | 职能 |
|------|------|------|
| **顶层** | 老大/教父/会长 | 最高决策者 |
| **高层** | 堂主/头目/干部 | 管理一个区域或业务线 |
| **中层** | 打手头目/小队长 | 带领手下执行任务 |
| **底层** | 小弟/马仔/古惑仔 | 执行具体任务 |

---

## 二、业务类型

| 业务 | 说明 | 写作场景 |
|------|------|---------|
| **保护费** | 收取商户"保护费" | 底层冲突 |
| **赌博** | 地下赌场、网络赌博 | 金钱诱惑 |
| **毒品** | 贩毒、运毒 | 禁忌+危险 |
| **高利贷** | 放贷、催收 | 经济压迫 |
| **走私** | 走私货物、偷渡 | 边境题材 |
| **洗钱** | 把黑钱洗白 | 金融犯罪 |
| **卖淫** | 组织卖淫 | 社会黑暗面 |

---

## 三、黑道规矩

| 规矩 | 说明 | 写作场景 |
|------|------|---------|
| **义气** | 兄弟情义是核心 | 背叛+忠诚 |
| **等级** | 尊卑分明，不能越级 | 权力关系 |
| **保密** | 不能出卖兄弟 | 背叛题材 |
| **报仇** | 有仇必报 | 复仇题材 |
| **洗白** | 有钱后想洗白上岸 | 转型题材 |

---

## 四、经典黑帮场景

| 场景 | 描写要点 |
|------|---------|
| **入会仪式** | 喝血酒、发誓、拜关公 |
| **收保护费** | 威胁、恐吓、暴力 |
| **火拼** | 两帮人马对砍 |
| **谈判** | 在茶楼/酒吧谈判 |
| **背叛** | 出卖兄弟、卧底暴露 |
| **洗白** | 开公司、做慈善、转型 |

---

## 五、禁忌

- 不要美化黑帮生活
- 不要美化暴力犯罪
- 黑帮题材要有"因果报应"的基调
