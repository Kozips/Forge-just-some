### 训练类型

| 类型 | 内容 | 写作场景 |
|------|------|---------|
| **体能** | 跑步、俯卧撑、障碍跑 | 新兵连 |
| **射击** | 实弹射击、战术射击 | 专业训练 |
| **格斗** | 军体拳、擒拿格斗 | 近身战斗 |
| **战术** | 班组战术、城市作战 | 实战演练 |
| **野外** | 拉练、野外生存、武装越野 | 极限挑战 |

## 四、经典军事场景

| 场景 | 描写要点 |
|------|---------|
| **新兵入伍** | 离别、适应、蜕变 |
| **班长带兵** | 严厉+关爱、老兵欺负新兵 |
| **实弹射击** | 紧张、后坐力、成绩 |
| **演习** | 红蓝对抗、胜负心理 |
| **战友离别** | 退伍、调岗、牺牲 |
| **战场** | 恐惧、勇气、兄弟情 |
| **军嫂** | 等待、孤独、坚强 |

---

## 五、禁忌

- 不要丑化军队形象
- 不要涉及军事机密
- 不要美化战争暴力
- 军旅题材要体现"战友情"和"保家卫国"
