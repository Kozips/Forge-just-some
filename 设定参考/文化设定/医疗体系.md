# 医疗体系

> 跨国家复用——龙国用"三甲医院"，泡菜国用"大学附属医院"，绅士国用"NHS"。

---

## 一、医院等级

### 龙国

| 等级 | 说明 | 写作场景 |
|------|------|---------|
| **三甲** | 最高等级医院 | 大城市、疑难杂症 |
| **三乙/二甲** | 中等医院 | 地级市 |
| **社区医院** | 基层医疗 | 常见病、慢病 |
| **乡镇卫生院** | 农村医疗 | 乡村题材 |

### 泡菜国

| 等级 | 说明 | 写作场景 |
|------|------|---------|
| **大学附属医院** | 最高等级 | 大城市 |
| **综合医院** | 大型医院 | 区域医疗 |
| **诊所** | 小型私人诊所 | 基层医疗 |

### 绅士国

| 等级 | 说明 | 写作场景 |
|------|------|---------|
| **NHS** | 全民免费医疗 | 等待时间长 |
| **GP诊所** | 全科医生 | 首诊 |
| **私人诊所** | 快捷但贵 | 高端医疗 |

---

## 二、科室分类

| 科室 | 职能 | 写作场景 |
|------|------|---------|
| **急诊科** | 急救、危重症 | 紧急场景 |
| **外科** | 手术、创伤 | 手术场景 |
| **内科** | 内科疾病 | 慢病管理 |
| **妇产科** | 分娩、妇科 | 生育题材 |
| **儿科** | 儿童疾病 | 家庭题材 |
| **精神科** | 心理疾病 | 心理题材 |
| **ICU** | 重症监护 | 生死场景 |
| **手术室** | 手术进行 | 手术场景 |

---

## 三、医护人员

| 角色 | 职能 | 写作场景 |
|------|------|---------|
| **主任医师** | 科室负责人 | 权威+责任 |
| **主治医师** | 主要诊治 | 日常医疗 |
| **住院医师** | 值班、写病历 | 新手成长 |
| **护士** | 护理、执行医嘱 | 护患关系 |
| **麻醉师** | 麻醉 | 手术场景 |

---

## 四、医患关系

| 场景 | 矛盾 | 写作场景 |
|------|------|---------|
| **看病难** | 挂号难、排队久 | 社会焦虑 |
| **看病贵** | 自费药、进口药 | 经济压力 |
| **医闹** | 医疗纠纷、暴力伤医 | 社会矛盾 |
| **红包** | 送红包求安心 | 社会潜规则 |

---

## 五、经典医疗场景

| 场景 | 描写要点 |
|------|---------|
| **急诊抢救** | 紧张、时间紧迫、生死一线 |
| **手术** | 无影灯、手术刀、团队配合 |
| **诊断** | 检查、分析、确诊 |
| **告知病情** | 坏消息、患者反应、家属崩溃 |
| **值班** | 通宵值班、疲惫、紧急呼叫 |
| **医患冲突** | 误解、投诉、暴力 |

---

## 六、禁忌

- 不要丑化医护人员
- 不要美化医疗事故
- 医疗题材要体现"救死扶伤"的正面
