## 一、企业类型

### 龙国

| 类型 | 特点 | 写作场景 |

| **国企** | 铁饭碗、论资排辈、关系网 | 体制内题材 |

| 类型 | 特点 | 写作场景 |
|------|------|---------|

| **创业公司** | 江南区创业热 | 创业题材 |

### 绅士国

| 类型 | 特点 | 写作场景 |
|------|------|---------|
| **金融城** | 全球金融中心 | 金融题材 |

## 二、职场生态

### 晋升路径

```
实习生 → 初级员工 → 高级员工 → 主管 → 经理 → 总监 → VP → CEO
```

### 职场潜规则

| 规则 | 说明 | 写作场景 |
|------|------|---------|
| **站队** | 跟对领导比能力重要 | 派系斗争 |
| **汇报** | 向上管理是核心技能 | 权力互动 |
| **加班** | 龙国996、泡菜国会食文化 | 职场压力 |
| **裁员** | 末位淘汰、优化、毕业 | 社会焦虑 |
| **办公室政治** | 表面和气背后捅刀 | 人际博弈 |

## 四、财富阶层

| 层级 | 资产 | 生活方式 |
|------|------|---------|
| **顶级富豪** | 100亿+ | 私人飞机、游艇、豪宅 |
| **富豪** | 10-100亿 | 别墅、豪车、国际学校 |
| **高净值** | 1-10亿 | 多套房产、海外投资 |
| **中产** | 1000万-1亿 | 房贷、学区车、焦虑 |
| **普通白领** | 100-1000万 | 租房/房贷、通勤 |
| **底层** | 100万以下 | 生存压力、月光 |
