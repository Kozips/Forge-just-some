# 官场体系

> 跨国家复用——龙国用"官场"，泡菜国用"政界"，绅士国用"议会"，底层逻辑相通。

---

## 一、权力结构

### 龙国（中国原型）

| 层级 | 机构 | 关键职位 | 写作场景 |
|------|------|---------|---------|
| **中央** | 党中央、国务院 | 总理、部长、省委书记 | 大官场、反腐 |
| **省级** | 省委、省政府 | 省委书记、省长 | 省级博弈 |
| **市级** | 市委、市政府 | 市委书记、市长 | 市级斗争 |
| **县级** | 县委、县政府 | 县委书记、县长 | 基层官场 |
| **乡镇** | 乡党委、乡政府 | 乡党委书记、乡长 | 最基层 |

### 泡菜国（韩国原型）

| 层级 | 机构 | 关键职位 | 写作场景 |
|------|------|---------|---------|
| **总统** | 青瓦台/龙山 | 总统、总理 | 最高层博弈 |
| **部级** | 各部委 | 部长、次官 | 政策斗争 |
| **地方** | 道/市 | 知事、市长 | 地方选举 |

### 绅士国（英国原型）

| 层级 | 机构 | 关键职位 | 写作场景 |
|------|------|---------|---------|
| **中央** | 议会、内阁 | 首相、大臣 | PMQs交锋 |
| **地方** | 郡议会 | 郡长、议员 | 地方政治 |

## 二、晋升路径

### 龙国标准路径

```
科员 → 副科 → 正科 → 副处 → 正处 → 副厅 → 正厅 → 副部 → 正部
```

**关键节点**：
- **副科到正科**：基层干部的第一道坎
- **正处到副厅**：县级到市级的跨越，最难的一步
- **正厅到副部**：地方到中央的跨越

**晋升要素**：
- 政绩（GDP、项目、亮点工程）
- 关系（上级赏识、同级支持、下级拥护）
- 时机（换届、空缺、政策风口）
- 学历（党校进修、在职研究生）
- 年龄（35岁正处、45岁正厅是关键线）

### 写作要点

- **开会是权力展示**：座次、发言顺序、谁先谁后都是信号
- **批示是权力语言**：领导的批示风格反映性格
- **调研是表态**：去哪个地方调研、不去哪个都是态度
- **人事是核心**：谁能升、谁被调、谁被查是官场小说的核心矛盾

## 三、官场潜规则

| 规则 | 说明 | 写作场景 |
|------|------|---------|
| **站队** | 跟对人比能力重要 | 派系斗争 |
| **表态** | 关键时刻必须表明立场 | 政治站队 |
| **汇报** | 向上级汇报是核心工作 | 权力互动 |
| **保密** | 不该说的绝不说 | 信息博弈 |
| **面子** | 给领导面子是基本功 | 社交场景 |
| **人情** | 人情债是最难还的债 | 利益交换 |

---

## 四、经典官场场景

| 场景 | 描写要点 |
|------|---------|
| **饭局** | 座次、敬酒顺序、谁买单 |
| **开会** | 发言顺序、表态方式、沉默的含义 |
| **调研** | 前呼后拥、汇报材料、照片站位 |
| **送礼** | 时机、方式、金额、禁忌 |
| **被查** | 双规/留置、心理崩溃、供出同伙 |
| **换届** | 站队、拉票、明升暗降 |

## 五、禁忌

- 不要直接描写真实政治人物
- 不要涉及敏感历史事件
- 反腐题材要符合"正义终将胜利"的基调
- 不要美化腐败行为
