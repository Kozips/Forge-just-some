# 娱乐圈体系

> 跨国家复用——龙国用"内娱"，泡菜国用"韩娱"，绅士国用"West End"。

---

## 一、行业结构

### 经纪公司

| 类型 | 特点 | 写作场景 |
|------|------|---------|
| **大型经纪公司** | 资源多、控制强、抽成高 | 偶像题材 |
| **独立工作室** | 自由度高、资源少 | 独立艺人 |
| **个人工作室** | 自己当老板 | 顶流艺人 |

### 经纪人

| 角色 | 职能 | 写作场景 |
|------|------|---------|
| **经纪人** | 接活、谈判、危机公关 | 幕后操控 |
| **助理** | 日常事务、行程安排 | 生活细节 |
| **宣传** | 媒体关系、热搜管理 | 舆论战 |

---

## 二、艺人类型

| 类型 | 特点 | 写作场景 |
|------|------|---------|
| **演员** | 演技、作品、奖项 | 影视题材 |
| **歌手** | 唱功、专辑、演唱会 | 音乐题材 |
| **偶像** | 人设、粉丝、流量 | 韩娱/内娱 |
| **网红** | 短视频、直播、带货 | 现代题材 |
| **综艺咖** | 搞笑、话题、曝光 | 综艺题材 |

---

## 三、娱乐圈生态

### 上位路径

```
练习生/龙套 → 出道/配角 → 走红 → 顶流 → 过气/转型
```

### 潜规则

| 规则 | 说明 | 写作场景 |
|------|------|---------|
| **陪酒** | 应酬、饭局、资源交换 | 黑暗面 |
| **潜规则** | 性交易换资源 | 社会题材 |
| **炒CP** | 制造绯闻、捆绑营销 | 流量操作 |
| **买热搜** | 花钱上热搜、控评 | 舆论战 |
| **撕番位** | 争一番、争C位 | 粉丝战争 |

---

## 四、粉丝文化

| 类型 | 特点 | 写作场景 |
|------|------|---------|
| **真爱粉** | 无条件支持 | 粉丝题材 |
| **黑粉** | 恶意攻击 | 网络暴力 |
| **私生饭** | 跟踪、偷拍、骚扰 | 恐怖题材 |
| **站姐** | 追行程、拍照、修图 | 粉丝生态 |
| **CP粉** | 配对、磕糖 | 同人题材 |

---

## 五、经典娱乐圈场景

| 场景 | 描写要点 |
|------|---------|
| **选秀** | 练习生、淘汰、出道 |
| **红毯** | 礼服、拍照、抢C位 |
| **颁奖典礼** | 获奖、落选、感言 |
| **演唱会** | 排练、舞台、粉丝互动 |
| **塌房** | 丑闻曝光、人设崩塌、粉丝脱粉 |
| **解约** | 打官司、赔偿、冷藏 |
| **复出** | 洗白、转型、重新出发 |

## 六、禁忌

- 不要直接使用真实艺人姓名
- 不要美化潜规则
- 不要涉及未成年人
- 粉丝题材注意不要过度极端化
