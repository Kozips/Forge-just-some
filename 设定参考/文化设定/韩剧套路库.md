## 七、经典韩剧套路公式

### 公式1：三集定律

```
第1集：相遇（冲突+误会）
第2集：靠近（甜蜜+暧昧）
第3集：分离（误会+错过）
→ 循环
```

### 公式2：四角恋

```
男主 ↔ 女主
  ↑        ↑
女配 ↔ 男配
→ 复杂关系网
```

### 公式3：身份反转

```
表面：普通人 ↔ 普通人
真相：财阀 ↔ 财阀
→ 身份揭露后的关系变化
```

### 公式4：契约关系

```
契约结婚/恋爱 → 假戏真做 → 误会分离 → 真心告白
```

### 公式5：复仇线

```
被伤害 → 隐忍成长 → 回来复仇 → 发现真相 → 和解
```
