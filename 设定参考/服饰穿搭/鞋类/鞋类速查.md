# 鞋类速查

> 脚上穿的。鞋是整套搭配的锚点——同一身衣服换双鞋，风格完全不同。

---

## 女鞋

| 单品 | 款式细分 | 情感暗示 |
|------|---------|---------|
| **高跟鞋** | 细跟=性感/气场、粗跟=稳重、坡跟=休闲 | 女人味、权力、身高 |
| **尖头鞋** | 高跟/平跟 | 干练、职场、攻击性 |
| **圆头鞋** | 玛丽珍、平底 | 少女、学院、可爱 |
| **乐福鞋** | 平底、金属扣 | 学院、知性、英伦 |
| **帆布鞋** | 高帮/低帮 | 文艺、学生、随性 |
| **小白鞋** | 运动/板鞋 | 百搭、清爽、基础 |
| **运动鞋** | AJ/老爹鞋/跑鞋 | 潮流、运动、活力 |
| **靴子** | 短靴=日常、过膝靴=气场、马丁靴=酷 | 秋冬、硬朗、酷 |
| **凉鞋** | 绑带=度假、夹趾=休闲 | 夏天、清爽 |
| **拖鞋** | 毛绒=居家、人字拖=随意 | 居家、放松 |
