# 头戴速查

> 头上戴的所有东西——帽子、发饰、头巾、耳机。

---

## 帽子

| 单品 | 情感暗示 | 场景 |
|------|---------|------|
| **棒球帽** | 随性、年轻、遮脸 | 日常、运动、明星遮面 |
| **贝雷帽** | 文艺、法式、复古 | 秋冬、学院、凹造型 |
| **渔夫帽** | 日系、休闲、遮阳 | 夏天、文艺 |
| **礼帽** | 正式、绅士、复古 | 西装、英伦 |
| **针织帽** | 保暖、可爱、随性 | 冬天、学生、情侣 |
| **大檐帽** | 度假、优雅、遮阳 | 海边、夏天、贵妇 |
| **厨师帽** | 职业、高度=级别 | 厨房 |
| **安全帽** | 工地、建筑 | 职业 |
| **贝雷帽（军）** | 军装、颜色代表兵种 | 军事 |
| **凤冠/花冠** | 古典、婚礼、华丽 | 传统婚礼、古装 |

| **发箍** | 宽/窄、珍珠/蝴蝶结 | 少女、甜美 |
| **发带** | 运动/丝绒/缎面 | 运动/复古/日常 |
| **头绳** | 简单/蝴蝶结/珍珠 | 扎头发、日常 |
| **发圈** | 大肠发圈=复古、丝绒=精致 | 日常、少女 |

## 头巾/面纱

| 单品 | 场景 | 描写要点 |
|------|------|---------|
| **头巾（阿拉伯）** | 中东传统 | 白色/格纹、黑绳固定 |
| **面纱** | 婚礼、神秘、中东 | 透不透、长度、掀起的动作 |
| **丝巾包头** | 复古、时尚 | 花色、系法、露出的发尾 |
| **蒙眼布** | 角色扮演、cosplay | 材质（丝绸/蕾丝）、紧度 |
