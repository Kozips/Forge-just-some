# 内衣速查

> 贴身衣物。可以跟任何外衣组合。

---

## 日常内衣

| 单品 | 款式细分 | 情感暗示 |
|------|---------|---------|
| **文胸** | 无钢圈=舒适、有钢圈=塑形、三角杯=少女、全罩杯=大胸、半罩杯=聚拢 | 看款式和颜色 |
| **内裤** | 三角=基础、平角=舒适、丁字=无痕、高腰=收腹 | 看款式和材质 |
| **背心/吊带** | 棉质=日常、蕾丝=精致、丝绸=高级 | 内搭、居家 |
| **塑身衣** | 连体/分体、收腹/提臀 | 功能性、身材管理 |

### 颜色暗示
- 黑色 = 性感/神秘
- 白色 = 清纯/基础
- 粉色 = 少女
- 红色 = 热情
- 肤色 = 日常/隐形
- 蓝色/紫色 = 精致/冷淡

### 材质暗示
- 棉质 = 日常/朴实/舒适
- 蕾丝 = 精致/性感/浪漫
- 丝绸 = 贵气/柔媚/诱惑
- 网纱 = 大胆/挑逗
- 缎面 = 高级/柔美

### 描写要点
- 肩带是否露出、勒痕
- 边缘花纹（蕾丝边/光面）
- 透不透、包裹程度
- 新旧状态（新=紧致 vs 旧=松弛）
