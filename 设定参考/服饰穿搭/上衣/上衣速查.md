# 上衣速查

> 上半身衣物。可跨风格搭配——西装外套+汉服内搭、旗袍外披风衣，合理就好。

---

## 现代上衣

| 单品 | 款式细分 | 情感暗示 |
|------|---------|---------|
| **T恤** | 圆领/V领/亨利领、纯色/印花/Logo | 基础、随性、年轻 |
| **衬衫** | 白色=基础、蓝=商务、格子=程序员、牛仔=休闲 | 看颜色决定正式度 |
| **卫衣** | 连帽=随性、圆领=基础、Oversized=慵懒 | 年轻、休闲、遮身材 |
| **毛衣** | 高领=知性、圆领=基础、V领=显瘦、开衫=文艺 | 温暖、秋冬、质感 |
| **夹克** | 皮衣=酷、牛仔=休闲、飞行员=帅气、棒球服=运动 | 外套决定整体风格 |
| **西装** | 单排扣=基础、双排扣=正式、Oversized=时尚 | 职场、气场、权力 |
| **风衣** | 卡其=经典、黑色=酷、长款=气场、短款=干练 | 秋冬、气质、走路带风 |
| **大衣** | 羊毛=质感、驼色=高级、黑色=百搭、长款=气场 | 冬天、质感、成熟 |
| **羽绒服** | 长款=保暖、短款=利落、亮面=潮、哑光=低调 | 冬天、实用 |

## 传统上衣

| 单品 | 描写要点 | 搭配建议 |
|------|---------|---------|
| **汉服上襦** | 交领右衽、大袖/窄袖、系带、面料光泽 | 搭配下裙、可外穿现代外套 |
| **旗袍** | 立领、盘扣、修身、开衩高度 | 单穿/外披西装/风衣均可 |
| **和服上着** | 宽袖、腰带位置、花纹 | 搭配半衿、可现代混搭 |
| **赤古里（韩）** | 短上衣、胸口蝴蝶结、袖子撞色 | 搭配高腰长裙 |

## 职业上衣

| 单品 | 描写要点 |
|------|---------|
| **白大褂** | 长度（短=医生/长=教授）、胸牌、口袋里的笔 |
| **手术服** | 蓝/绿、V领套头、宽松、收口袖 |
| **护士服** | 粉色/白色、护士帽级别横杠、胸牌 |
| **军装上衣** | 肩章军衔、臂章部队、领花、贝雷帽/大檐帽 |
| **厨师服** | 白色双排扣、厨师帽高度=级别 |
| **工装** | 反光背心、连体服、安全帽 |
