# 配饰速查

> 除衣服鞋帽之外的所有装饰物——项链、戒指、手链、包、手表等。

---

## 首饰

| 单品 | 描写要点 | 情感暗示 |
|------|---------|---------|
| **项链** | 长短（锁骨链=精致/长链=文艺）、材质、吊坠 | 信仰/纪念/装饰 |
| **耳钉** | 单耳=叛逆、双耳=精致、材质（金/银/宝石） | 个性/叛逆/精致 |
| **耳环** | 垂坠=优雅、圈圈=时尚、大小 | 女人味/气场 |
| **唇钉** | 下唇=叛逆/性感、唇角=挑逗/亚文化、材质（银/金色）| 亚文化/叛逆/性感/坏女孩 |
| **鼻钉/鼻环** | 单侧鼻翼=印度风/异域、鼻 septum=暗黑/牛仔 | 异域/叛逆/暗黑 |
| **舌钉** | 隐藏的、说话时偶尔闪光 | 极端叛逆/隐秘的性感/亚文化 |
| **眉钉** | 眉骨上方、金属杆 | 朋克/亚文化/视觉冲击 |
| **脐钉** | 肚脐、露腰时可见 | 性感/年轻/自信 |
| **乳钉** | 乳头穿孔、单侧或双侧、金属环/杆状 | 极致叛逆/隐秘的性感/亚文化/身体自主 |
| **戒指** | 戴哪根手指（无名指=婚姻/食指=单身）、材质 | 婚姻/承诺/权力 |
| **手链** | 金/银/皮/绳、珠子 | 祈愿/纪念/装饰 |
| **脚链** | 金/银、铃铛 | 异域/装饰 |
| **胸针** | 位置/大小/造型 | 身份/品位/暗号 |
| **choker/颈环** | 蕾丝=少女、皮革=酷、金属=暗黑 | 性感/叛逆/控制 |

### choker/颈环描写速查
- **蕾丝choker**：少女、精致、锁骨上方
- **皮革颈环**：酷、叛逆、摇滚
- **金属颈环**：暗黑、控制
- **丝绒choker**：复古、优雅
- **珍珠颈链**：复古、贵气
- **系带choker**：蝴蝶结系在后面、可拉扯

## 描写技巧速查

### 通过服饰暗示性格
- 扣子扣到最上面 → 保守/自律/压抑
- 袖子随意卷起 → 随性/不拘小节
- 衣服大一码 → 没安全感/遮身材
- 衣服小一码 → 自信/展示身材
- 全身黑白灰 → 低调/冷漠
- 颜色鲜艳 → 外向/自信

### 通过服饰暗示关系
- 穿对方的衣服 → 亲密/占有/标记
- 交换配饰 → 承诺/羁绊
- 衣服上残留对方气味 → 暧昧
- 帮对方整理衣领 → 亲密/照顾

### 通过服饰推动情节
- 扣子崩开 → 意外/身材变化
- 丝袜勾丝 → 尴尬/事后
- 衣服撕裂 → 暴力/激情
- 换装 → 角色转变/新阶段
