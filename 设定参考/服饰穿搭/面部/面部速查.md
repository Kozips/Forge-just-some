# 面部速查

> 脸上戴/挂的——眼镜、口罩、面纱、面具。

---

| 单品 | 款式细分 | 情感暗示 |
|------|---------|---------|
| **眼镜** | 金丝=书生、黑框=严肃、无框=商务、圆框=文艺、猫眼=复古/性感 | 看款式 |
| **墨镜** | 飞行员=酷、方框=正式、圆框=文艺、猫眼=复古 | 明星、神秘、遮面 |
| **口罩** | 医用=日常、黑色=时尚、布面=文艺、N95=防护 | 遮脸、神秘、防尘 |
| **面纱** | 婚纱=婚礼、黑纱=神秘、蕾丝=复古 | 若隐若现、神秘 |
| **面具** | 半脸=舞会、全脸=角色扮演、威尼斯=奢华 | 匿名、伪装、派对 |
| **眼罩** | 黑色=海盗、丝绸=优雅、蕾丝=洛丽塔 | 独眼、遮蔽、装饰 |
| **鼻环** | 单颗=印度、链式=朋克 | 异域、叛逆 |
| **唇钉/舌钉** | 金属 | 叛逆、暗黑 |

### 眼镜描写速查

- **材质**：金属框=精致、塑料框=年轻、醋酸框=高级
- **镜片**：透明=日常、变色=户外、有色=时尚
- **佩戴**：推眼镜=思考、摘眼镜=放松/颜值释放、镜片反光=遮掩情绪
- **搭配反差**：西装+圆框=学术、辣妹+黑框=禁欲系、校服+金丝=书呆子
