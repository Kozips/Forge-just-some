# 连体服饰速查

> 一件搞定全身的单品——裙子、连体衣、旗袍、和服等。不需要上下搭配。

---

## 现代连体

| 单品 | 款式细分 | 情感暗示 |
|------|---------|---------|
| **连衣裙** | 碎花=田园、纯色=简约、吊带=性感、衬衫裙=知性 | 看款式决定风格 |
| **连体裤** | 工装款=酷、针织款=温柔、牛仔款=休闲 | 方便、利落 |
| **睡裙** | 吊带=性感、棉质=舒适、丝绸=高级 | 居家、私密 |
| **浴袍/睡袍** | 棉质=舒适、丝绸=诱惑、白色=酒店感 | 出浴、私密 |

## 传统连体

| 单品 | 描写要点 | 搭配建议 |
|------|---------|---------|
| **旗袍** | 立领、盘扣、开衩、修身、丝绸/棉质 | 外披风衣/西装、配丝袜+高跟 |
| **汉服曲裾** | 缠绕式深衣、庄重、多层 | 配发簪+步摇 |
| **汉服齐胸襦裙** | 高腰线在胸下、飘逸、大袖 | 配团扇+披帛 |
| **和服** | 宽袖、腰带（帯）、花纹、后颈露出 | 配木屐+足袋+簪子 |
| **纱丽（印度）** | 6码布缠绕、露腰、金线刺绣 | 配鼻环+手镯+脚链 |
| **维多利亚长裙** | 束腰+撑裙+高领+蕾丝 | 配手套+珍珠项链 |

## 职业连体

| 单品 | 描写要点 |
|------|---------|
| **空姐制服套裙** | 丝巾、盘发、丝袜+中跟鞋、名牌 |
| **女仆装** | 黑色连衣裙+白色围裙+头饰 |
| **军装连体服** | 战术背心、靴子、装备腰带 |
| **工装连体服** | 多口袋、拉链、反光条 |
