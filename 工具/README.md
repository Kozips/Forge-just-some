# 工具箱

> 写作造物Agent，用于生成新Skill和地图编辑。

## 工具清单

| 工具 | 类型 | 用途 |
|------|------|------|
| `写作造物/SKILL.md` | Agent | 输入题材名→自动调研→蒸馏→生成写作题材Skill |
| `地图编辑器/地图编辑器.html` | 工具 | 可视化地图编辑器，支持自定义世界观地图 |

## 写作造物Agent

输入题材名，自动完成全流程：
1. 调研经典作品
2. 提炼写作技法
3. 建立角色与场景库
4. 生成SKILL.md + 4个素材文件

### 使用方式
```
蒸馏XX题材
做个XX写作skill
更新XX的写作skill
```
