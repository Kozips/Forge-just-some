# 末班地铁 · NPC角色卡

## NPC1：戴耳机的女大学生

| 项目 | 内容 |
|------|------|
| 外貌 | 20出头，卫衣帽子压得很低，戴白色无线耳机，手机屏幕的光打在脸上 |
| 位置 | 车厢左侧，靠窗坐着 |
| 表面 | 一直在刷手机，看起来完全不在乎周围 |
| 秘密 | 她的手机没有信号，屏幕上反复播放同一段视频——一个她不认识的人在对着镜头哭 |
| 说话风格 | 简短、不看人、带刺。"嗯。""不知道。""你别问我。" |
| 触发条件 | 如果有人提到"第三节车厢"，她会突然摘下耳机 |

## NPC2：抱公文包的中年男人

| 项目 | 内容 |
|------|------|
| 外貌 | 40多岁，西装领带，头发油亮但眼神发直，怀里的公文包抱得很紧 |
| 位置 | 车厢右侧，离你两个座位 |
| 表面 | 看起来像加班到深夜的上班族 |
| 秘密 | 公文包里装的不是文件，是一叠照片——每张照片上都是这班地铁的不同站点 |
| 说话风格 | 客套、不安、话多。"你也加班啊？""这站名……我没见过。""你说这车是不是坐错了？" |
| 触发条件 | 如果有人问他的公文包，他会慌 |

## NPC3：穿校服的男孩

| 项目 | 内容 |
|------|------|
| 外貌 | 十六七岁，校服外套披着，靠在门边打瞌睡，书包放在脚边 |
| 位置 | 车厢门口，靠着扶手杆 |
| 表面 | 像补课补到很晚的学生 |
| 秘密 | 他的校服是这所学校的，但这所学校十年前就拆了 |
| 说话风格 | 迷糊、少话、偶尔冒出一句让人脊背发凉的话。"嗯……几点了？""这车好慢。""……又来了。" |
| 触发条件 | 如果有人叫醒他，他会问一句"到第几站了"，语气像在等一个结果 |

## NPC4：车厢尽头的女人

| 项目 | 内容 |
|------|------|
| 外貌 | 看不清脸。长发，深色大衣，一直坐在最远的角落，一动不动 |
| 位置 | 车厢最远端，角落座位 |
| 表面 | 什么都不做，像一尊雕塑 |
| 秘密 | 她不是乘客，是这班地铁的一部分。每到一站，她会抬头——抬头的时候，规则就生效了 |
| 说话风格 | 不说话。但如果你靠近，会听到她在低声数数。 |
| 触发条件 | 不要主动靠近她。靠近她的代价取决于你在哪一站 |
