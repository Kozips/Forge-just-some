# 末班地铁 · 世界观设定

## 核心规则

- 这是一班不存在于任何地铁线路图上的列车
- 每一站都有自己的规则，违反规则的人"下车"后不会回来
- 广播会给出每站的规则，但规则可能是假的
- 车厢里的人各怀秘密，没人是完全无辜的
- 真相：这班地铁在"收人"——每站收集一个灵魂，集满才能到终点

## 地铁规则

| 项目 | 内容 |
|------|------|
| 运行时间 | 23:47 - ? |
| 始发站 | 你上车的那站（正常的地铁站） |
| 当前站 | 未知（站名不在线路图上） |
| 终点站 | "归墟" |
| 广播 | 每到一站播报一条规则，规则可能是真或假 |
| 下车条件 | 完成规则考验 / 找到真相 / 被"收走" |

## 车厢结构

- 第一节车厢：你所在的车厢（4个NPC）
- 第二节车厢：空的（但灯在闪）
- 第三节车厢：有人坐着（不能对视）

## 站点设计

| 站名 | 规则 | 真相 |
|------|------|------|
| 永安 | 不要和第三节车厢的人对视 | 对视后会看到自己的死亡方式 |
| 忘川 | 不要回答广播的提问 | 回答=同意被收走 |
| 灯灭 | 如果灯灭了，闭眼数到十 | 灯灭时睁眼会看到真相，但会被标记 |
| 归墟 | （终点站，无规则） | 选择：下车=死亡，留在车上=永远困在这里 |
