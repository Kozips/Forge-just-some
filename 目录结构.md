# forge 目录结构

> 从 `forge/SKILL.md` 迁出。此文件用于人工导航，不参与每次 skill 激活。

---

## 顶层入口

```text
forge/
├── SKILL.md                 # 低 Token 薄入口：路由、硬规则、快速路由表
├── 人物/                    # 人物、性格、职业、对话习惯
├── 写作/                    # 题材、文风、剧情结构、写作流程
├── 设定参考/                # 武器、载具、国家拟实、文化设定、世界观
├── 路由器/                  # 路由规则、加载优先级、冲突处理、加载限制
├── 小说总谱/                # 长篇项目状态维护模板
├── 工具/                    # 写作造物工具/地图编辑器
├── 规则/                    # 通用创作规则与声明
├── 输出规则.md              # 输出目录与命名规则
└── 目录结构.md              # 本文件
```

---

## 六大核心模块

| 模块 | 入口 | 职责 |
|------|------|------|
| 人物 | `人物/SKILL.md` | 角色扮演、人物原型、性格、职业、对话习惯 |
| 写作 | `写作/SKILL.md` | 题材选择、写作铁律、题材子 Skill 导航 |
| 设定参考 | `设定参考/SKILL.md` | 武器、载具、国家拟实、文化设定、世界观 |
| 路由器 | `路由器/路由规则.md` | 用户意图解析与组合加载 |
| 工具 | `工具/README.md` | 写作造物、地图编辑器 |
| 小说总谱 | `小说总谱/README.md` | 长篇项目状态维护 |

---

## 题材模块内部结构（01-06）

每个题材子目录结构统一（恐怖类已精简，去掉02_专业知识，其余12个题材保留完整六模块）：

```text
写作/类型/[题材名]/
├── SKILL.md               # 题材入口
├── README.md              # 题材概览
├── 01_核心规则/            # 该题材的硬性写作规则
├── 02_专业知识/            # 领域知识参考
├── 03_素材库/              # 可复用素材
├── 04_写作模板/            # 结构化模板
├── 05_技法研究/            # 高级技法
└── 06_质检清单/            # 写完后自查项
```
