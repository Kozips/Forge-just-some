# forge 目录结构

> 从 `forge/SKILL.md` 迁出。此文件用于人工导航，不参与每次 skill 激活。

---

## 顶层入口

```text
forge/
├── SKILL.md                 # 低 Token 薄入口：硬规则、模型适配、路由指针（~60行）
├── 人物/                    # 人物、性格、职业、对话习惯
├── 写作/                    # 题材、文风、剧情结构、写作流程
├── 设定参考/                # 武器、载具、国家拟实、文化设定、世界观
├── 小说总谱/                # 长篇项目状态维护模板
├── 工具/                    # 写作造物工具/地图编辑器
├── 规则/                    # 通用创作规则与声明
├── 输出规则.md              # 输出目录与命名规则
└── 目录结构.md              # 本文件
```

---

## 六大核心模块

| 模块 | 入口 | 职责 |
|------|------|------|
| 人物 | `人物/SKILL.md` | 角色扮演、人物原型、性格、职业、对话习惯 |
| 写作 | `写作/SKILL.md` | 题材选择、写作铁律、题材子 Skill 导航 |
| 设定参考 | `设定参考/SKILL.md` | 武器、载具、国家拟实、文化设定、世界观 |
| 工具 | `工具/README.md` | 写作造物、地图编辑器 |
| 小说总谱 | `小说总谱/README.md` | 长篇项目状态维护 |

---

## 题材模块内部结构

每个题材采用扁平化结构（01_核心规则和06_质检清单已合并进SKILL.md）：

```text
写作/类型/[题材名]/
├── SKILL.md               # 题材入口（含速查表+铁律+质检清单）
├── 02_专业知识.md          # 领域知识参考（或 02_专业知识/ 目录）
├── 03_素材库.md            # 可复用素材（合并版）
├── 05_技法研究.md          # 高级技法（合并版）
└── 04_写作模板.md          # 结构化模板（部分题材）
```

│   ├── 英国/
│   ├── 法国/
