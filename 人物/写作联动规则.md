# 写作联动规则

> 定义人物模块与写作模块的交互规则。确保角色塑造和故事写作不脱节。

---

## 核心原则

**人物决定故事走向，故事环境反塑人物。**

- 写作时先加载人物，再决定剧情
- 剧情中的事件要反馈回人物档案（状态变化、关系值变化）
- 人物和写作是双向的，不是单向调用

---

## 加载顺序

每次写新内容时，按以下顺序加载：

```
1. 写作/写作流程/质检.md            ← 先过检查
2. 人物/SKILL.md                      ← 加载角色技能库
3. 人物/速查表/                       ← 按需查找角色标签
4. 人物/兼容矩阵/                     ← 检查标签组合是否有冲突
5. 写作/类型/[题材]类/SKILL.md        ← 加载题材规则（含铁律）
6. 开始写作
7. 写作/写作流程/质检.md            ← 写完过质检
```

## 人物 → 写作（角色如何影响故事）

### 角色性格决定剧情分支

| 角色标签 | 推荐剧情走向 |
|----------|-------------|
| 病娇 + 偏执型 | 占有欲驱动的冲突→囚禁/跟踪/爆发 |
| 社恐 + 回避型 | 逃避→被迫面对→突破舒适区（或崩溃） |
| 腹黑 + 表演型 | 表面和善→暗中操控→翻车或成功 |
| 恋爱脑 + 焦虑型 | 患得患失→误会→和好（或分手） |
| 傲娇 + 依恋型 | 嘴硬心软→关键时刻暴露真心 |

### 角色能力决定场景设计

- 有格斗技能的角色 → 可以设计战斗场景
- 有心理创伤的角色 → 可以设计触发/崩溃场景
- 有专业知识的角色 → 可以设计该领域的专业场景

### 角色关系决定冲突类型

| 关系状态 | 冲突类型 |
|----------|---------|
| 好感度高但有秘密 | 信任危机 |
| 好感度低但被迫相处 | 摩擦→了解→转变 |
| 三角关系 | 嫉妒/选择/牺牲 |
| 师徒/上下级 | 理念冲突/背叛/超越 |

## 写作 → 人物（故事如何反塑角色）

### 事件反馈规则

**每次发生以下事件，必须更新人物档案：**

| 事件类型 | 更新内容 |
|----------|---------|
| 物理伤害 | 更新身体状态（受伤/残疾/恢复） |
| 心理创伤 | 更新心理状态，可能触发/加重标签特征 |
| 亲密接触 | 更新亲密等级、关系值 |
| 重大冲突 | 更新关系值、信任值 |
| 关键选择 | 记录选择，影响后续性格发展 |
| 角色死亡/离开 | 更新存活状态、对其他角色的影响 |

### 关系值变化规则

```
+10  共同经历生死
+5   帮助/保护
+3   坦诚分享秘密
+1   日常善意互动
-1   小摩擦/误会
-5   背叛/伤害
-10  不可原谅的事
0    消除误会/和解（可恢复部分扣减）
```

## 标签组合兼容矩阵速查

| 组合 | 兼容性 | 注意事项 |
|------|--------|---------|
| 病娇 + 社恐 | 高兼容 | 内向的占有欲更隐蔽更可怕 |
| 腹黑 + 恋爱脑 | 高兼容 | 表面算计内心恋爱脑=反差萌 |
| 傲娇 + 社恐 | 高兼容 | 嘴硬+不敢表达=双重障碍 |
| 病娇 + 傲娇 | 中兼容 | 占有欲+嘴硬需要平衡，别太极端 |
| 恋爱脑 + 社恐 | 高兼容 | 想爱又不敢=天然戏剧性 |
| 腹黑 + 病娇 | 需谨慎 | 两个极端标签可能互相打架 |

完整兼容矩阵详见 `人物/兼容矩阵/` 目录。

## 题材 × 人物 联动表

| 题材 | 推荐人物标签 | 避免的标签 |
|------|-------------|-----------|
| 恐怖类 | 社恐/偏执/敏感型 | 无敌型/全知型 |
| 亲密类 | 恋爱脑/依恋型/表演型 | 禁欲型（除非是反差设定） |
| 悬疑类 | 腹黑/高智商/反社会型 | 粗心型/冲动型（除非是配角） |
| 战斗类 | 好战型/保护欲/复仇型 | 纯粹的和平主义者 |
| 冒险类 | 好奇心强/冒险型/乐观型 | 宅家型（除非是被迫冒险） |
| 情感类 | 焦虑型/回避型/安全型 | 纯粹的冷血型 |
| 丧尸末日 | 生存型/领导型/冷酷型 | 圣母型（除非是悲剧设定） |

## 跨模块冲突处理

当人物规则和题材规则冲突时：

```
优先级：核心红线 > 人物底线 > 题材规则 > 剧情需要
```

- 核心红线（如：不碰未成年人）永远最高优先
- 人物设定的"绝对不能做的事"不可被剧情覆盖
- 题材规则可以在人物设定允许的范围内灵活调整
- 剧情需要可以让人物暂时偏离日常状态，但要有充分铺垫

---

## 联动检查清单

写完每章后，额外检查：

- [ ] 本章出场角色的标签是否在行为中体现？
- [ ] 角色关系值变化是否已记录？
- [ ] 是否有角色做出了不符合标签的行为？（如果是有意为之，需有铺垫）
- [ ] 题材规则和人物规则是否一致？
- [ ] 角色档案是否需要更新？
