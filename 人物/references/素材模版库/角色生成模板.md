# 角色生成模板（V2 统一版）

> 创建新角色时使用此模板。与 `人物/SKILL.md` 和 `人物/速查表/` 联动。
> 异世界角色专用完整版（含穿越者/神明/反派模板+魔法能力+种族标签）：`设定参考/世界观/异世界通用世界观/角色/角色模板.md`

---

## 基础档案

```
【角色名】：（见下方命名速查，不要老三样）
【别名/外号】：
【年龄】：
【性别】：
【身高/体重】：
【外貌关键词】：（3-5个词，如：凤眼/薄唇/左脸刀疤/银发及腰）
【第一印象】：（别人第一次见他/她会注意到什么）
```

### 命名速查

> 命名规则已独立拆分为：[命名规则速查.md](命名规则速查.md)
> 包含：性格/命运暗示、反差命名、文艺名、网文爽感名、谐音名表、恐怖专用名、造名原则、命名禁忌。

## 人物类型标签

从 `人物/SKILL.md` 分类中选取（可组合，建议2-3个）：

```
【人物类标签】：（如：病娇/傲娇/社恐/恋爱脑/腹黑）
【性格类标签】：（如：偏执型/回避型/表演型/反社会型）
【职业类标签】：（如：医生/特警/纹身师/大学生/自由职业）
```

**标签组合公式**：`人物类（核心驱动）+ 性格类（行为模式）+ 职业类（社会身份）`

示例：`病娇 + 偏执型 + 大学生` = 一个偏执的大学生病娇角色

---

## 心理内核

```
【核心欲望】：（他/她最想要什么？一句话）
【核心恐惧】：（他/她最怕什么？一句话）
【自我认知】：（他/她觉得自己是什么样的人？）
【他人认知】：（别人觉得他/她是什么样的人？）
【认知差距】：（自我认知和他人认知的差距=戏剧性来源）
```

---

## 人物弧线预设

```
【起点状态】：（出场时是什么样的人）
【转折事件】：（什么事件会改变他/她）
【终点状态】：（故事结束时变成了什么样的人）
【弧线类型】：
  □ 成长弧（正向变化）
  □ 堕落弧（负向变化）
  □ 平弧（不变，但改变周围的人）
  □ 伪弧（表面变化，内核不变）
```

---

## 对话习惯

参照 `人物/SKILL.md` 中的对话习惯模块：

```
【语气基调】：（如：冷淡/热情/阴阳怪气/温柔/暴躁）
【口头禅】：（1-2个，如："随便""你开心就好""切"）
【用词特点】：（如：爱用网络梗/爱用成语/爱用脏话/爱用方言）
【说话节奏】：（如：长句多/短句多/经常半句话不说完/爱插嘴）
【特殊习惯】：（如：紧张时结巴/撒谎时摸鼻子/生气时笑）
【禁忌话题】：（听到什么会炸/会沉默/会转移话题）
```

**辨识度测试**：遮住名字后，仅看对话能认出是他/她吗？如果不能，强化差异。

## 关系网络

```
【与主角关系】：
  - 关系类型：（恋人/对手/师徒/兄妹/宿敌/暧昧...）
  - 初始好感度：（0-100）
  - 关系张力：（什么让这段关系有戏？）

【与其他角色关系】：
  - 角色B：关系 + 张力
  - 角色C：关系 + 张力

【社交模式】：
  - 对陌生人：（如：冷淡/热情/警惕）
  - 对熟人：（如：毒舌/依赖/疏离）
  - 对敌人：（如：残忍/戏弄/恐惧）
```

## 感官特征（写作用）

```
【视觉标志】：（一个让人记住的视觉特征，如：总戴黑色手套/左耳三个耳洞）
【听觉标志】：（声音特征，如：声音沙哑/说话带鼻音/笑声像猫叫）
【嗅觉标志】：（气味特征，如：总带着烟味/洗衣液味/血腥味）
【触觉标志】：（触感特征，如：手总是冰的/皮肤粗糙/体温偏高）
【行为标志】：（习惯性动作，如：转笔/咬指甲/摸后颈/抖腿）
```

---

## 能力与缺陷

```
【核心能力】：（他/她最擅长什么？）
【致命缺陷】：（什么弱点会被利用？）
【隐藏技能】：（平时不显露的特长）
【绝对不能做的事】：（底线/禁忌）
```

## 角色速查卡（蒸馏用）

创建完角色档案后，蒸馏出一张速查卡：

```
[角色名] | [年龄] | [标签组合]
一句话：[核心欲望] → [核心恐惧]
说话：[语气基调] + [口头禅]
标志：[视觉/行为标志]
弧线：[起点] → [转折] → [终点]
关系：与主角[关系类型]，好感度[数值]
```
