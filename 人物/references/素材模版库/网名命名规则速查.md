# 网名命名规则速查

> 来源：B站真实用户名总结。LLM读取后可自行生成符合中国互联网风格的网名。
> 核心原则：**真人网名≠精心设计，真人网名=随手一打+偶尔灵光。**

---

# 一、铁律（生成网名前必读）

1. **禁止正能量**："星辰大海""追梦少年""阳光男孩""向阳而生"全是AI腔，真人不用
2. **禁止感叹句当ID**："绷不住了""笑死我了""绝了"是弹幕用语不是网名
3. **随机>合理**：真人网名经常毫无逻辑——"豌豆肆王八""雜莯"没有任何意义，但这就是真实
4. **混搭>纯中文**：中英混搭、中文+数字、中文+符号非常常见——"弹珠biidama""Melody-iii""_初空月二四_"
5. **越长越离谱越真实**："纯欲母蟑螂夜袭老汉""良子本入胃袋这一块"——越荒诞越像真人
6. **食物是万能素材**："雪糕千層餅""-酸菜五花肉-""蒜香奶油蘑菇烩牛肉"
7. **不要怕看不懂**："蜉立叶瓯纹""阿勒_穆罕喀诃"——真人网名经常就是看不懂的

# 二、14种真实网名模式 + 1种独立子系统

## 1. 动漫/游戏角色型

直接用或改动漫角色名。B站最常见。

**真实案例**：
- __Sephiroth（最终幻想）
- 澤村英绿绿（动漫角色+颜色重复）
- 蝙蝠侠的父母（DC漫画，地狱梗：蝙蝠侠父母双亡）
- 蜘蛛侠的本叔叔（同理：本叔死了）
- 莱伊兰佩路基（动漫角色音译）
- 天际乌木勇士（上古卷轴+奇幻风格）

**生成公式**：`[动漫/游戏角色名]` 或 `[角色名]+[随机修饰]`

---

## 2. 古风/文学型

用古典诗词意境或文言文风格。看起来有文化。

**真实案例**：
- 北地玉箫声
- 墨染红尘夜
- 犹如黎明
- 思念成风倚楼听雨
- -青语提灯-
- 贫僧不羡仙
- 靖又破防了（古典+现代梗混搭）

**生成公式**：`[2-4字古典意象]+[2-3字意境词]` 或 `-[文艺短语]-`

---

## 3. 食物型

用食物名当网名。中国人真的很爱用食物。

**真实案例**：
- 雪糕千層餅
- -酸菜五花肉-
- 蒜香奶油蘑菇烩牛肉
- 超级茶叶蛋丶

**生成公式**：`[食物名]` 或 `-[食物名]-` 或 `[食物名]+丶` 或 `[夸张形容]+[食物名]`

---

## 4. 荒诞/无厘头型

毫无逻辑的词组合，越荒诞越真实。B站特色。

**真实案例**：
- 纯欲母蟑螂夜袭老汉（每个词都很正常，组合在一起就离谱）
- 侏儒哥布林
- 豌豆肆王八
- 超绝旋风贝利亚大王
- 你算什么最高发圆
- 折大戟把
- 夏波利利的内痔真好吃

**生成公式**：`[形容词]+[不相关的名词]+[不相关的动词]+[不相关的名词]`，或者`[A]的[B]真好吃`（B是完全不搭的东西）

---

## 5. 自嘲/自黑型

用自嘲表达幽默。中国网民最爱的身份标签。

**真实案例**：
- 没事就睡觉辣
- 不灭小天尊（反讽：叫"小天尊"但实际很弱）
- 階不欺负老实人嘛
- 良子本入胃袋这一块
- 可以叫一声木琴么

**生成公式**：`[自嘲行为]+[语气词]` 或 `[反讽身份]+[梗]`

---

## 6. 随机看不懂型

没有意义的字符组合。真人就是会起这种名字。

**真实案例**：
- 杂莯
- 烟波-_-
- 龙猴123
- IIIQuQII
- 蜉立叶瓯纹
- 弹珠biidama
- Kar7hus

**生成公式**：`[2-3个随机汉字]` 或 `[随机字母]+[数字]` 或 `[中文]+[英文混搭]`

---

## 7. 真名/半真名型

用真名的一部分+修饰。

**真实案例**：
- 李耀林XingBro
- 鄙人梁某
- 北平王守义

**生成公式**：`[姓+名]+[英文后缀]` 或 `鄙人[X]某` 或 `[地名]+[名]`

---

## 8. 英文/混搭型

英文单词、英文+数字、英文+符号。

**真实案例**：
- ultimate_freedom
- Melody-iii
- light-back
- GlamJ128
- Sir-Archer
- Giggityy

**生成公式**：`[英文词]_[英文词]` 或 `[英文词]-[罗马数字]` 或 `Sir-[英文词]`

---

## 9. 互联网梗/自造梗型

把网络梗用进网名。关键不是直接用梗，而是**用梗的句式给自己起名**。

> 梗素材库详见：[设定参考/梗文化/梗库.md](../../../设定参考/梗文化/梗库.md)

**真实案例**：
- 靖又破防了（"又"字暗示经常破防，自嘲）
- 階不欺负老实人嘛（"嘛"字=撒娇式自辩）
- 良子本入胃袋这一块（良子=B站吃播博主，标志台词"味真足""XX这一块"。"胃袋"是他形容自己能装的胃。整句=自比良子的胃，用良子的句式包装自己。属于"用博主的梗给自己起名"）
- 球伯逆转许昊龙（梗的变体+人名混搭）
- 给他鱼不如教他钓鱼（俗语改写）
- 可以叫一声木琴么（"木琴"="母亲"的谐音梗，地狱笑话）

**生成公式**：`[网红/博主名]+[该网红的标志性台词/梗的变体]` 或 `[梗的变体]` 或 `[人物]+[梗的行为]`

**进阶手法**：用某个网红/博主的**说话句式**给自己起名，而不是直接用网红的名字。"良子本入胃袋这一块"比"我是良子"高级十倍——因为前者是"懂梗的人"才会起的名字。

**错别字梗**：故意用错别字替代正确字，致敬经典梗。比如"入"代替"人"（起源：小孩玩和平精英说"666，这个入开桂"——"入"="人"的错别字，"桂"="挂"的错别字，后来网友故意用"入"代替"人"当梗）。所以"良子本入"="良子本人"，但写错致敬这个梗。

## 13. 谐音（跨类型修饰，可叠加到任何模式上）

谐音不是独立分类，是**万能调料**——可以叠加到上面任何一种模式里，让网名多一层意思。

### 谐音的6个层次

1. **成语谐音**：骑乐无穷（其乐无穷）、一派胡言（一派湖言）
2. **人名谐音**：秦寿生（禽兽生）、范统（饭桶）、杜子腾（肚子疼）、史珍香（屎真香）、蔡道（菜到）、梅良心（没良心）、赖月京（来月经）
3. **脏话谐音**：用看似正常的字组合，实际读出来是脏话
4. **器官谐音**：用兵器/植物/食物等正常词谐音器官，常配合倒装（折大戟把=这大鸡巴）
5. **方言谐音**：用方言发音制造双关
6. **外语谐音**：用英文/日文等外语的音译当"安全词"，实际意思大家心知肚明。迪克=dick、法克=fuck、沙碧=shabi。看起来像洋气网名，实际在开车

### 谐音叠加示例（谐音+其他模式）

| 叠加组合 | 示例 | 解读 |
|----------|------|------|
| 谐音+食物型 | 梅柿仁（没事人） | 表面是食物相关，实际是"没事人" |
| 谐音+古风型 | 郝建仁（好人→表面是好名字实际在骂人） | 表面文艺，读出来是脏话 |
| 谐音+自嘲型 | 蔡道巅峰（菜到巅峰） | 自嘲菜但用古风包装 |
| 谐音+互联网梗 | 这个入不太冷（人→入，错别字梗+电影梗） | 双重梗叠加 |
| 谐音+荒诞型 | 折大戟把（这大鸡巴+倒装） | 表面兵器名，实际器官谐音 |
| 谐音+英文混搭 | Dick丶迪克 | 外语谐音+中文音译双保险 |

**铁律**：谐音要"看不出来但读出来就懂"。太明显的谐音不好笑，太隐晦的没人get到。最好的谐音网名是"不知道的人觉得正常，知道的人笑出声"。

### 谐音的"洋葱层"手法

好的谐音网名不止一层——表面正常，深想一层变味，再深一层更炸裂。

**案例解析：梅事别慌**
- 第一层：表面=没事别慌（安慰语）
- 第二层：谐音=梅毒+没逝（梅毒没事→得了梅毒但没死）
- 第三层：暗示=得了梅毒还很淡定（自嘲+地狱笑话）

**案例解析：郝建仁**
- 第一层：表面=好人（郝建仁=好人）
- 第二层：谐音=好贱人（郝建仁→好贱人）
- 第三层：故意用"看起来像夸人"的名字实际在骂自己

**案例解析：这个入不太冷**
- 第一层：表面=致敬电影《这个杀手不太冷》
- 第二层：错别字梗=入→人（和平精英梗）
- 第三层：暗示=这个人不太冷→这个人有点热（冲浪梗）

**生成公式**：先起一个表面正常的词，检查有没有至少1层隐藏解读。有1层=及格，有2层=优秀，有3层=神级。

### 谐音的"打散重组"手法

比洋葱层更隐蔽——把目标词**拆成单字**，**打乱顺序**，**逐字谐音**，包装成一个看起来完全无关的名字。

**案例解析：杯姬菲**
- 目标词：飞机杯（成人用品）
- 拆字：飞→菲，机→姬，杯→杯
- 打乱：菲姬杯→杯姬菲
- 表面：杯姬菲=可爱日系女名（姬=公主，菲=芬芳）
- 实际：重排还原→菲姬杯→飞姬杯→飞机杯

**案例解析：迪克丶本**
- 目标词：迪克本→dick本→dick本身
- 拆字：迪克=dick（外语谐音），本=本体
- 表面：像个外国人的中文名
- 实际：在说自己是dick

**生成公式**：`[目标词拆成单字]→[每字找谐音]→[打乱顺序]→[组合成看似正常的名字]`

**铁律**：打散后表面必须像一个**真实存在的名字/词语**，不能看起来就是乱码。"杯姬菲"像日系名所以成功，"菲杯姬"像乱码所以失败。顺序很重要——要选一个**读起来最顺口**的排列。

## 14. 黄梗谐音型（独立子系统）

把性相关的词用谐音/倒装/拆字包装成"看起来正常"的网名。是谐音的**最大应用场景**。

### 器官谐音命名法

| 原词 | 包装手法 | 网名示例 | 表面意思 |
|------|----------|----------|----------|
| 鸡巴 | 倒装+谐音 | 折大戟把 | 兵器名（戟=古代兵器） |
| 鸡巴 | 反切/拆音 | 吧唧 | 拟声词（亲嘴的声音） |
| 鸡巴 | 拆字谐音 | 又鸟八 | 表面像生僻字组合 |
| 阴茎 | 谐音替换 | 银荆 | 植物名（银荆树） |
| 乳房 | 谐音替换 | 如方 | 像正常人名 |
| 乳房 | 量词包装 | 一对木瓜 | 水果名 |
| 乳头 | 谐音替换 | 如豆 | 像食物名 |
| 阴部 | 谐音替换 | 音步 | 像音乐术语 |
| 屁股 | 谐音/量词 | 两瓣苹果 | 水果描述 |
| 蛋蛋 | 谐音替换 | 弹弹 | 拟声词/游戏名 |

### 动作谐音命名法

| 原词 | 包装手法 | 网名示例 | 表面意思 |
|------|----------|----------|----------|
| 做爱 | 拆字谐音 | 作艾 | 像中药名（艾草） |
| 口交 | 谐音替换 | 口焦 | 像形容口渴 |
| 手淫 | 谐音替换 | 守银 | 像守财的意思 |
| 插入 | 谐音替换 | 茶入 | 像茶道术语 |
| 高潮 | 语义双关 | 浪尖冲浪手 | 冲浪运动 |
| 射 | 谐音/语义 | 射手座本座 | 星座梗 |

### 谐音层数（从浅到深）

| 层级 | 示例 | 解读 |
|------|------|------|
| **1层·直接谐音** | 吧唧 | 鸡巴→吧唧，换个顺序 |
| **2层·倒装谐音** | 折大戟把 | 这大鸡巴→折大戟把，字面是兵器 |
| **3层·拆字谐音** | 又鸟八 | 鸡→又+鸟，巴→八，拆散了 |
| **4层·洋葱层** | 杯姬菲 | 飞机杯→飞姬杯→菲姬杯→杯姬菲，要反向还原才懂 |
| **5层·跨语言** | 迪克丶本 | dick→迪克（音译）+本（本体），中外双保险 |

### 生成公式

```
第1步：确定原词（器官/动作/场景）
第2步：找谐音字（同音或近音的"正常"字）
第3步：选包装手法（倒装/拆字/跨语言/量词+名词）
第4步：组装成看起来像正常网名的东西
第5步：验证——不认识的人觉得正常，认识的人一秒get
```

**铁律**：
- 包装后必须**像一个真实存在的词/名字**——"折大戟把"像兵器名才成功，"把戟折大"像乱码就失败
- 谐音不能太明显——"鸡八"太直白，"又鸟八"才够隐蔽
- 越正经越好笑——表面越正经（兵器名/植物名/人名），实际含义越炸裂

## 15. 地狱笑话/黑色幽默型

用已死角色、悲剧事件、禁忌话题当网名。B站年轻人最爱的"冒犯型幽默"。

**真实案例**：
- 蝙蝠侠的父母（父母双亡梗）
- 蜘蛛侠的本叔叔（本叔被杀梗）
- 泰坦尼克号的瞭望员（没看到冰山）
- 泰山的父母（被豹子吃了）

**核心手法**：找一个"悲剧角色/事件"，用"正常身份"的方式说出来。表面看是正常称呼，知道背景的人会觉得地狱。

**生成公式**：`[作品名]的[已死/悲剧角色]` 或 `[悲剧事件]的[相关身份]`

**铁律**：地狱笑话的核心是"知道的人才懂"——太直白就不是地狱笑话了，是冒犯。要的是"想了一下才反应过来"的延迟伤害。

# 三、网名的"真实感"指标

| 指标 | AI腔 | 真人 |
|------|------|------|
| 长度 | 2-4字，精准 | 随机，2-15字都有 |
| 含义 | 有意义、正能量 | 经常无意义、自嘲 |
| 语言 | 纯中文 | 中英混搭、日文混搭 |
| 符号 | 无符号 | 常用`-` `_` `丶` `ゞ` |
| 数字 | 无或有意义数字 | 随机数字（123/666） |
| 食物 | 不用 | 超爱用 |
| 荒诞 | 不敢荒诞 | 越荒诞越真实 |
| 逻辑 | 有逻辑 | 经常没逻辑 |

# 五、生成网名的自检清单

- [ ] 这个网名像真人会随手打出来的吗？（不要精心设计感）
- [ ] 有没有犯"正能量"错误？（星辰大海/追梦少年/向阳而生=AI腔）
- [ ] 有没有犯"感叹句"错误？（绷不住了/笑死我了=弹幕用语不是ID）
