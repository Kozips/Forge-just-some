# 概念艺术与奇幻 · 提示词库
> 概念艺术、超现实、奇幻场景。

---

## 通用写法

### 灵魂
用**现实媒介的物理缺陷**作为锚点，让AI在"不完美"中创造出比"完美"更有叙事深度的奇幻真实感。

### 通用结构（8步）
- ① 媒介视角伪装：开头声明"这不是画，而是某种真实媒介拍摄的"——"iPhone拍摄的幕后照片""闪光灯自拍"
- ② IP世界观锚定：指定已知世界观（"火影忍者""美少女战士"）作为视觉语言参照系
- ③ 角色融合到环境：角色嵌入复杂场景中，角色与环境的关系比角色本身更重要
- ④ 物理混乱感：强调"不完美"——"电缆、灯架、沙袋""凌乱床品""杂乱小桌子"
- ⑤ 技术限制模拟：指定物理设备缺陷——"运动柔化""智能手机传感器颗粒感""略微倾斜"
- ⑥ 涂鸦叠加层：在真实照片上叠加手绘元素——"马克笔艺术""粗糙草图线条""白色贴纸边框"
- ⑦ 精确数量装饰："5个红色爱心涂鸦、7个黄色星星"——数量精确到个位
- ⑧ 反完美收尾：主动排斥AI默认美化——"没有精心修饰的摄影棚风格""保持手工感"

### 关键技法
- **媒介伪装法**：不说"画一个幕后场景"，说"用iPhone 17拍摄的非专业幕后照片"——AI自动模拟iPhone的景深、噪点、色彩科学
- **前景监视器嵌套**：在画面前景放一台播放同一场景的监视器，形成"画中画"——同时传达"这是片场"和"这是电影"
- **反AI美学指令**：主动对抗美化本能——"略显粗糙""略带运动柔化""略微倾斜"——"略"字是关键词
- **涂鸦层与真实层分离**：涂鸦叠加层和照片层有不同的质感规则（照片=闪光灯摄影，涂鸦=马克笔艺术），两层独立描述再合并
- **IP改编伪装**：不直接要求画IP，而是要求画"真人版电影的拍摄现场"——AI同时生成IP元素和电影工业元素

### 通用填空模版
```
这是{伪装媒介}拍摄的{照片类型}，取景于{世界观/IP}的{具体场景}。
{角色描述}正在{动作}，周围环绕着{环境细节}。
前景：{前景物体}，{技术设备/道具}。
背景：{大环境}，{远景元素}。
叠加层：{手绘/涂鸦/特效元素}，{质感描述}，{精确数量的装饰元素}。
物理特征：{模拟设备的缺陷}，{镜头特性}，{颗粒感/畸变/模糊}。
收尾：{反完美指令}，{禁止美化清单}。
```

## 火影忍者真人版电影幕后拍摄场景
这是一张用iPhone 17拍摄的非专业幕后照片，取景于一部真人版《火影忍者》奇幻忍者战争电影的拍摄现场。第七班和一小队忍者战士小心翼翼地包围着十尾——这头巨大的远古巨兽矗立在一片被摧毁的战场中央。鸣人身着橙黑色战斗服，佐助身穿深色忍者服，小樱身穿红色战斗服，卡卡西则顶着一头银发和面具，他们围绕着这头巨兽保持着一定的距离，为最终决战做准备。十尾体型庞大，面目狰狞，皮肤粗糙，长着多条尾巴、一只恐怖的独眼、锋利的利爪，脚下扬起漫天尘土，烟雾中隐隐透出类似查克拉的能量。 小队成员们在巨兽周围组成了一个紧张的战术圈，手中拿着苦无、刀剑、卷轴道具、破损的旗帜和发光的特效道具。气氛仿佛一场大战即将爆发的最后一秒。前景中，一台专业电影监视器正播放着同样的对峙场景，部分遮挡了画面下方。监视器周围是电影摄影机、三脚架、电缆、灯架、吊杆麦克风、沙袋、特技协调员、戴着耳机的剧组人员、一位导演正指向那头怪兽，以及特效人员正在释放尘土、烟雾和橙蓝色光效。 背景是一个大型户外战争电影片场：龟裂的土地、断裂的石柱、残破的忍者旗帜、人造碎片、脚手架、烟雾机、远处逼真的爆炸声，以及上方壮观的暴风云。影片采用iPhone手持拍摄，画面构图略显粗糙，略带运动柔化效果，保留了真实的智能手机传感器颗粒感和自然的镜头光晕，展现了略显混乱的制作细节，呈现出电影般的质感，但又明显带有幕后拍摄的痕迹，超写实的实景拍摄，细节丰富，营造出可信的电影片场氛围，没有经过精心修饰的摄影棚风格，也没有动漫插画的痕迹。
