## Caption 10步结构

每步一句话，不跳步不合并：

```
① 开场：[画质] + [风格] + [角色识别] + [镜头类型] + [拍摄角度]
② 位置：角色在画面中的[位置] + [填充方式] + [透视效果]
③ 头发：[颜色] + [长度] + [动态] + [特殊标记]
④ 眼睛：[颜色] + [瞳孔形状] + [眼神方向] + [表情分类标签]
⑤ 面部光影：[遮挡物] + [光影效果名称] + [对氛围的作用]
⑥ 服装：[层次1→内] + [颜色] + [细节] + [与背景的对比关系]
⑦ 动作（左手）：[方向] + [手势] + [动态程度]
⑧ 动作（右手）：[方向] + [手势] + [动态程度]
⑨ 特效/环境动态：[类型] + [颜色] + [运动方式] + [与角色的空间关系]
⑩ 收尾：[画质标签] + [艺术效果] + [背景与主体分离] + [光照方向和来源]
```

### 每步速记

| 步 | 关键规则 |
|----|---------|
| ① | 必须含：an extremely detailed + 风格 + identified as $character_1$ + 镜头 + 角度 |
| ② | 写死位置，不用模糊词（"a girl standing" ✗ → "filling the center-foreground" ✓） |
| ③ | 头发必须有动态（风吹/飘浮/垂落），不能是静物 |
| ④ | 用日系表情词比形容词精准：jitome=嫌弃眼、sanpui=三白眼、tsurime=冷淡、furime=媚眼 |
| ⑤ | 必须有一个遮挡物制造光影：帽子/头发/手/窗户格栅 |
| ⑥ | **服装必须和背景有对比**：`contrasting sharply with [背景颜色]` + `defined edges, clear outline` |
| ⑦⑧ | 每只手独立一句，必须写 `detailed five fingers, correct anatomy` |
| ⑨ | 没特效也要写环境动态（风/雨/灰尘/光粒子） |
| ⑩ | 清晰场景（海滩/办公室）**不要用 bokeh**，改用 `sharp focus, no soft diffusion` |

## Negative Prompt

### 基础（每次必带）

```
lowres, bad anatomy, bad hands, bad fingers, text, error, missing fingers, extra digit, fewer digits, cropped, worst quality, low quality, normal quality, jpeg artifacts, signature, watermark, username, blurry, deformed, extra limbs, mutated hands, poorly drawn face, ugly, disfigured, fused fingers, too many fingers, six fingers, four fingers, malformed hands, extra fingers, claw, blurry hands
```

### 按场景追加

| 场景 | 追加 |
|------|------|
| 清晰画面（海滩/办公室/教室） | `soft focus, hazy, washed out, low contrast, muddy colors, indistinct edges, blurry background` |
| 服装和背景分离 | `clothing blending into background, no separation between subject and background, same color subject and background` |
| 大胸禁止 | `big breasts, large breasts, huge breasts` |
| 年幼禁止 | `young, loli, child, underage` |
| 暗场景 | `overexposed, underexposed, noisy, grainy, banding` |

## 三大翻车坑（必读）

| 坑 | 症状 | 预防 |
|----|------|------|
| **手指变形** | 多指/少指/融合/爪子 | ⑦⑧写 `detailed five fingers, correct anatomy`；Negative带 `fused fingers, six fingers, four fingers, malformed hands` |
| **画面模糊** | 整体发虚/像低分辨率放大 | ⑩写 `sharp focus, crisp linework`；清晰场景**不用bokeh**；Negative带 `blurry, soft focus, hazy` |
| **衣服融背景** | 白衣+白沙滩=人消失 | ⑥写 `contrasting sharply with [背景颜色]` + `defined edges, clear outline, distinct from background` |
