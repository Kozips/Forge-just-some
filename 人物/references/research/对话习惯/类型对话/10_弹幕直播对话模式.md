# 弹幕对话模板档案

> **通用弹幕写作模板**——任何写作类型（直播文、综艺文、电竞文、选秀文、规则怪谈、末日生存、校园、都市等）只要涉及"弹幕/评论区/直播间发言"，都调用本文件。
> 核心原则：母语+翻译、乐子人居多、事不关己高高挂起、涉及本国利益时切换战斗模式。

---

## 〇、昵称生成规则（LLM必读）

**写弹幕时，每条弹幕的昵称不要用固定名称，按以下规则从 [../../../../../references/素材模版库/网名命名规则速查.md](../../../../../references/素材模版库/网名命名规则速查.md) 中生成：**

### 生成流程

1. 确定该弹幕属于哪种**弹幕人物类型**（乐子人/嘉豪/烂梗小鬼/冲浪人/分析党/互骂党/粉丝/悲情）
2. 确定**国家**（龙国/樱花国/棒子国/鹰国/毛熊国/中立国）
3. 从网名命名规则速查.md的14种模式中，选择**符合该国家+该类型**的模式生成昵称
4. 嘉豪型昵称固定为"隔壁班嘉豪"或变体（这是角色名，不是普通昵称）

### 各类型×国家的昵称模式对应

| 弹幕类型 | 龙国偏好模式 | 外国偏好模式 |
|----------|-------------|-------------|
| 乐子人 | 随机看不懂/自嘲/食物/真名半真名 | 本国语言的自嘲/随机词 |
| 嘉豪 | **固定角色名**：隔壁班嘉豪/嘉豪本豪 | — |
| 烂梗小鬼 | 荒诞无厘头/动漫梗/互联网梗 | 本国网络梗/动漫梗 |
| 冲浪人 | 荒诞/谐音/食物 | 本国语言的暗示性昵称 |
| 分析党 | 古风文学/英文混搭/真名半真名 | 本国语言的正式昵称 |
| 互骂党 | 角色扮演身份/真名半真名 | 本国语言的爱国昵称 |
| 粉丝 | [角色名]+老婆粉/姐姐粉/吹 | [角色名]+推し/팬 |
| 悲情 | 古风文学/符号装饰 | 本国语言的悲伤词汇 |

### 铁律

- **同一个角色（嘉豪/烂梗小鬼）的昵称全篇固定不变**
- **普通路人的昵称每次出现可以不同**（模拟真实弹幕——每次刷屏的人不一定是同一个人）
- **同一个国家的昵称风格要统一**（龙国=中英混搭+梗，樱花国=日语，鹰国=英文）
- **不要用感叹句当昵称**（"绷不住了""笑死我了"是弹幕内容不是ID）
- **不要用正能量当昵称**（"星辰大海""追梦少年"=AI腔）

## 一、弹幕格式硬规则

1. **昵称必写**：每条弹幕前面加【昵称】，格式：【国家】【昵称】内容。昵称按上方规则生成
2. **母语优先**：每个国家的观众用自己的母语发弹幕，后面括号跟中文翻译
3. **翻译不是逐字翻**：翻译是意译，要口语化，像弹幕软件自动翻译的效果
4. **乐子人态度**：80%的弹幕是事不关己的态度——别人倒霉跟我有什么关系，好看就行。但涉及本国选手/角色的脸面和利益时立刻切换战斗模式
5. **烂梗浓度**：龙国弹幕大量使用抽象文化梗（典/孝/乐/绷/急/麻/寄）、地狱笑话、烂梗小鬼式发言
6. **互骂是日常**：龙国vs棒子国是永恒战场，偶尔波及樱花国和鹰国。毛熊国不参与互骂但偶尔一句话所有人闭嘴

### 弹幕格式示例

> 以下昵称为**示例**，实际写作时按上方规则自行生成，不要重复使用这些示例昵称。

```
【龙国】【隔壁班嘉豪】这[事件]换我来一巴掌搞定
【龙国】【[乐子人昵称]】你比泰森还牛逼 乐
【龙国】【[烂梗小鬼昵称]】一眼丁真 鉴定为[当前事物]
【龙国】【[冲浪人昵称]】射了射了射了
【龙国】【[分析党昵称]】根据规则分析……
【龙国】【[互骂党昵称]】棒子国你们偶巴[xxx]
【樱花国】【[粉丝昵称]】[角色名]先生……お疲れ様です……
【棒子国】【[粉丝昵称]】우리 오빠 [xxx]!!
【鹰国】【[乐子人昵称]】bruh that [xxx]
【毛熊国】【[悲情昵称]】За Родину……
【中立国】【[分析党昵称]】赔率更新：[xxx]
```

## 三、各国观众核心特征

### 龙国观众

**核心态度**：乐子人浓度最高，万物皆可梗，自嘲+玩梗+偶尔真情流露。涉及龙国脸面时瞬间切换战斗模式。

**常用梗**：
- 典/孝/乐/绷/急/麻/寄
- 啊对对对（万能敷衍）
- 急了急了（对方生气时）
- 谁问你了（怼分析党）
- 格局打开
- 不会吧不会吧
- 家人们谁懂啊
- 典中典
- 大可不必
- 啊这/就这？/6

**通用弹幕模板**（[ ]内替换为当前场景内容）：

| 场景类型 | 弹幕模板 |
|----------|---------|
| **惊吓/恐怖** | "卧槽卧槽卧槽" / "寄" / "草 [事物]好恶心" / "[角色名]快跑！！" |
| **暧昧/亲密** | "我不充钱能看到这个？？" / "开冲了兄弟们" / "这是我不花钱能看的？" / "[角色名]🤤🤤🤤" |
| **搞笑/翻车** | "哈哈哈哈哈哈" / "绷" / "6" / "就这？" |
| **感动/泪目** | "破防了" / "泪目" / "我的青春结束了" |
| **互骂** | "棒子国你们偶巴[xxx]" → "关你屁事" → "啊对对对" |

### 樱花国观众

**核心态度**：表面绅士实则闷骚，技术流鉴赏，日语为主偶尔混中文。不主动挑事但被挑衅会反击。

**常用语**：すごい/こわい/wwwww/マジで？/やばい/お疲れ様/なるほど

**通用弹幕模板**：

| 场景类型 | 弹幕模板 |
|----------|---------|
| **惊吓** | "こわい……こわいよ……" / "もう見たくない……" |
| **暧昧** | "[角色名]……お疲れ様です……" / "これは無料で見れるの？マジで？" / "wwwwwwww" |
| **分析** | "[角色名]は……[分析内容]……" |
| **互骂** | "龍国は興奮しすぎだろ" → "我々は鑑賞しているだけだ" |

### 棒子国观众

**核心态度**：粉丝应援+颜值至上+碎碎念+大量感叹号。被骂了会疯狂反击但逻辑混乱。

**常用语**：偶巴/大发/헐/대박/ㅠㅠ/진짜？

**通用弹幕模板**：

| 场景类型 | 弹幕模板 |
|----------|---------|
| **惊吓** | "헐……저게 뭐야……" / "偶巴不要啊！！！ㅠㅠ" |
| **偶像** | "偶巴好帅！！大发！！！" / "偶巴加油！！你是最棒的！！" |
| **互骂** | "龙国人变态！！" → "你们的脸都是整出来的" → "整完也比你们帅！！" → "啊对对对" → "你们就知道啊对对对！！" → "不能" |

### 鹰国观众

**核心态度**：黑色幽默+实用主义+英文为主。嘲讽所有人但不站队。偶尔冒出军事术语。

**常用语**：bruh/lmao/F/RIP/nah/fr fr/no cap/sheesh/W/L

**通用弹幕模板**：

| 场景类型 | 弹幕模板 |
|----------|---------|
| **惊吓** | "bruh that [xxx]" / "nah this is insane" / "RIP. F." |
| **暧昧** | "Is this free?? THIS IS FREE??" / "sheeeeeesh" / "I need to go to church after this" |
| **互骂** | "y'all fighting while [xxx] lmaooo" / "everyone calm down and watch the show" |

### 毛熊国观众

**核心态度**：伏特加式抽象。平时极度冷静一句话所有人闭嘴，但喝多了就开始整活。不参与互骂，但偶尔冒出一句让全场炸裂的发言。

**常用语**：Сила/За Родину/Молчать/Продолжай/Нормально/Это просто

**通用弹幕模板**：

| 场景类型 | 弹幕模板 |
|----------|---------|
| **惊吓** | "Нормально。"（淡定发言）→ [全场] "毛熊国你清醒一点！！" |
| **煽情** | "你们都在看戏……我们的人已经没了……" → "不需要道歉。需要他们活着。" |
| **抽象** | "这个[事物]长得像我二叔喝醉以后的样子。" / "我在想如果把[事物]带回家能不能看门。" / "喝点伏特加就好了。" → "伏特加是万能的。" |
| **终结互骂** | "Молчать. Все。" → [弹幕瞬间安静3秒] |

### 中立国观众

**核心态度**：旁观+下注+猎奇。纯粹的看客心态，谁赢都行只要好看。

**常用语**：赔率、下注、买定离手、W/L

**通用弹幕模板**：
- "赔率更新：[队伍]X.X↓"
- "下注了下注了"
- "打起来打起来"

## 四、乐子人亚型——嘉豪 & 烂梗小鬼

弹幕里最有节目效果的不是正常观众，是嘉豪和烂梗小鬼。他们的存在让弹幕从"看直播"变成"看弹幕比看直播好笑"。

### 嘉豪型（嘴炮王者）

**人设**：隔壁班的嘉豪，音乐课才艺展示太尬又太自信。永远觉得自己比别人强，什么场面都敢说"换我来"。实际上是个手无缚鸡之力的普通人。

**昵称**：固定为"隔壁班嘉豪"或变体（嘉豪本豪/嘉豪到你了），全篇不变。

**核心台词模式**（[ ]内替换为当前场景内容）：
- "这[事物]换我来一巴掌搞定"
- "就这？我上我也行"
- "[角色名]那[技能]不行，换我[更好的方案]"
- "[异性角色名]那种女人/男人我一晚上搞定"
- "[某项成就]？我起码[更好的数字]"

**被怼模式**（昵称按规则生成，每次可以不同）：
- 嘉豪吹牛 → 【[乐子人昵称]】嘲讽他实际做不到
- 嘉豪吹异性 → 【[烂梗小鬼昵称]】"你先搞定你右手再说吧"
- 嘉豪被怼后辩解 → 【[烂梗小鬼昵称]】"急了急了" → 嘉豪："我没急！" → 【[乐子人昵称]】"啊对对对"

### 烂梗小鬼型（行走的梗库）

**人设**：万物皆可梗，一句话塞三个梗，用搞笑掩饰孤独。任何严肃话题都能用一个烂梗打断。

**昵称**：从网名规则的"荒诞无厘头型""动漫梗型""互联网梗型"中生成，全篇固定。

**核心梗库（通用）**：
- 一眼丁真："一眼丁真，鉴定为[当前事物]"
- 格局打开："[事物]也是生命，格局打开"
- 泼天的XX："泼天的[事物]终于轮到[队伍]了"
- city不city："这个[事物]city不city"
- 赢学："不管怎样我们都赢了"（阴阳怪气）
- 原神启动："原神启动！"（任何时候都能插入）
- 您配吗："嘉豪说[吹牛内容]——您配吗？"
- 回复梗："你说的这个朋友到底是不是你自己"
- 牢大/see you again："[角色名]——see you again"

## 五、弹幕节奏模板

### 通用节奏

**紧张/惊吓场景**：
```
[事件发生前] 正常弹幕 混合分析和乐子
[事件发生的瞬间] 弹幕突然消失 1-2秒
[然后爆发] "卧槽""草""寄""F""こわい""헐"
[持续] 恐慌弹幕+分析党尝试解读+乐子人"就这？"
[互骂插入] "棒子国你们偶巴遇到这个怎么办" → "闭嘴！！"
[平复] 弹幕密度降低
```

**暧昧/亲密场景**：
```
[暧昧阶段] "要来了要来了""开冲预备"
[开始] "卧槽""真的来了""我不充钱能看到这个？"
[进行中] 密度最高的弹幕 乐子人全开+互骂暂停（都在看）
[高潮] "射了""awsl""？？？？？""这是我不花钱能看的？"
[结束] "完了？""就这？""再来一次"
[互骂恢复] "棒子国你们偶巴也经历过这个吗" → "你闭嘴！！！"
```

**互骂节奏**：
```
[触发] 一方提到另一方的选手/国家
[升级] 另一方反击 语言越来越激烈
[扩散] 第三方加入（通常是鹰国嘲讽双方）
[终结] 毛熊国一句话所有人安静 或者 某个事件打断互骂
```

**感动/泪目场景**：
```
[感动瞬间] 弹幕密度突然下降（都在认真看）
[反应] "破防了""泪目""我的青春结束了"
[扩散] 更多人加入感动 弹幕变慢但密度高
[恢复] 乐子人回来"就这？""感动了？"→ 被其他人骂"滚"
```

## 六、不同题材的弹幕调用指南

| 题材 | 弹幕重点 | 嘉豪型台词方向 | 烂梗小鬼方向 |
|------|---------|---------------|-------------|
| 规则怪谈/恐怖 | 恐慌+分析+地狱笑话 | "这怪我一巴掌拍死" | 恐怖事物的梗 |
| 末日生存 | 紧张+绝望+黑色幽默 | "这丧尸我一刀一个" | 末日梗+地狱笑话 |
| 电竞/游戏 | 操作分析+互骂+吹 | "这操作我上我也行" | 游戏梗+选手梗 |
| 恋爱/综艺 | 暧昧+磕CP+互骂 | "那女的我一晚上搞定" | 恋爱梗+冲浪梗 |
| 校园/日常 | 吐槽+共鸣+找乐子 | "这题我闭着眼都会" | 校园梗+考试梗 |
| 都市/职场 | 共鸣+分析+吐槽 | "这项目换我三天搞定" | 打工人梗+加班梗 |

**铁律**：弹幕内容必须跟当前场景匹配。恐怖场景不要出现恋爱弹幕，恋爱场景不要出现恐怖弹幕（除非是搞笑打断）。
