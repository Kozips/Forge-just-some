## 一、职业说话方式

# 13 职业说话方式研究

> 职业塑造语言。一个人在某个行业干上三五年，说话方式就会被彻底改写——用词、节奏、口头禅、思维逻辑，全都刻上行业的烙印。这是塑造真实角色的关键素材。

## 一、程序员/码农

### 核心语言特征
- **中英混杂**：不是装X，是真的想不起来中文怎么说。"这个bug我fix一下""先merge到主分支再deploy"
- **极简表达**：能一个字说完的不说两个字。"好""行""收到""可以"
- **逻辑先行**：先说结论，再说原因。"不行，因为这个方案有并发问题"
- **被动句式**：习惯说"被""让"，因为代码世界里自己经常不是主导者

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 出了问题 | "在我电脑上是好的啊" |
| 推卸责任 | "这不是bug，这是feature" |
| 万能解决 | "重启试试""清一下缓存" |
| 需求变更 | "需求又变了？""这不合理" |
| 写代码 | "我先加个TODO"（然后永远不做了） |
| 甩锅 | "这段代码谁写的？" → git blame → 是自己 |
| 评估工期 | "三天能搞定"（然后三天又三天） |
| 代码审查 | "LGTM""Nit: 变量名可以改一下" |
| 技术争论 | "这个trade-off要看场景" |
| 上线部署 | "部署一下""回滚""灰度发布" |
| 项目管理 | "排期排不开""这个优先级不够" |

### 典型对话场景

**场景1：线上出了bug**
```
A: 线上挂了！
B: 什么情况？
A: 用户反馈登录不了，502了
B: 我看看日志... 是不是昨天的变更导致的？
A: 不是吧，那个已经灰度了啊
B: 先回滚，查完再上线
A: 好，我先回滚
```

**场景2：需求评审**
```
产品：我想加个功能，用户可以看到...
开发：等一下，这个需求逻辑不通
产品：怎么不通了？
开发：你说的这个场景，0和1之间没有第三种状态
产品：那能不能变通一下？
开发：（沉默三秒）我回去评估一下工作量
```

**场景3：代码审查**
```
A: 帮我CR一下这个PR
B: 整体LGTM，Nit: 这个变量名能不能改一下？叫tempValue太泛了
A: 行，改了
B: 还有这里，建议用Optional，避免空指针
A: 嗯，这个我考虑过，主要是兼容老代码
B: OK，那加个注释说明一下
```

**场景4：工位日常**
```
A: 你在干嘛？
B: 在debug一个内存泄漏的问题
A: 找到了吗？
B: 没有，怀疑是第三方库的问题
A: 重启试试？
B: ...说了跟没说一样
```

**场景5：周五下班前**
```
A: 走不走？
B: 等一下，我先提交一下
A: 别提交了，周一再说
B: 不行，不提交我周末睡不着
A: 你这是有强迫症
B: 这叫职业素养
```

### 说话节奏特点
- **短句为主**，很少说长句。三五个字就是一个完整的表达
- **思考时有停顿**，常用"嗯...""那个..."做缓冲
- **语气偏平**，情绪表达内敛。激动的时候也不是喊，而是语速加快
- **擅长用反问**："这不合理吧？""你确定？""这样能行？"
- **括号式表达**：喜欢在说话时补充限定条件。"可以，但是..."

### 行话/黑话
- **bug** — 程序缺陷
- **CR** — Code Review，代码审查
- **PR** — Pull Request，合并请求
- **LGTM** — Looks Good To Me，审查通过
- **PTAL** — Please Take A Look，请看一下
- **Nit** — 小问题，不影响合并
- **Blocker** — 必须修复的阻塞问题
- **deploy** — 部署上线
- **rollback** — 回滚，撤回上线
- **灰度** — 小范围发布测试
- **hotfix** — 紧急修复
- **技术债** — 欠下的需要后期偿还的技术工作
- **甩锅** — 把责任推给别人
- **造轮子** — 重复开发已有功能
- **屎山代码** — 质量很差的遗留代码

## 二、医生

### 核心语言特征
- **命令式语气**：习惯了对患者下指令。"躺下""张嘴""深呼吸"
- **高度省略**：时间就是生命，说话极度精简。"多久了？""哪里疼？"
- **专业术语+白话切换**：对同事说"疑似心梗"，对患者说"心脏有点问题"
- **被动关怀**：不太会直接表达关心，而是通过"多喝水""注意休息"来体现

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 问诊开场 | "哪里不舒服？""怎么了？" |
| 了解病情 | "多久了？""有没有过敏史？" |
| 检查指令 | "张嘴看看""躺下，放松""深呼吸""翻个身" |
| 安排检查 | "先去做个检查""抽个血""拍个片子" |
| 诊断告知 | "问题不大""不用太担心""你这个情况很常见" |
| 治疗建议 | "按时吃药""多喝水""少吃辛辣""别熬夜" |
| 告别 | "回去好好休息""有情况随时来" |
| 重症告知 | "家属在吗？""叫你家属来一趟" |

### 典型对话场景

**场景1：门诊问诊**
```
医生：怎么了？
患者：肚子疼
医生：多久了？
患者：昨天晚上开始的
医生：拉不拉？
患者：拉了三四次
医生：发烧吗？
患者：有点低烧
医生：来，躺床上，我按一下... 这里疼不疼？这里呢？
患者：哎呦！就那儿！
医生：先去验个血，做个B超，结果出来再看
```

**场景2：查房**
```
医生：（翻看病历）今天感觉怎么样？
患者：好多了
医生：（听诊）深呼吸... 好。大便正常吗？
患者：昨天正常了
医生：行，再观察两天，没问题可以出院了
护士：张大夫，3床的化验结果出来了
医生：好，我看看
```

**场景3：急诊抢救**
```
医生：什么情况？
家属：他突然倒下去了！
医生：意识清醒吗？有呼吸吗？
护士：血压80/50，心率120
医生：上心电监护，开放静脉通路，先推一支肾上腺素
护士：好的
医生：家属在外面等着，别进来！
```

**场景4：告知坏消息**
```
医生：家属来了吗？坐
家属：来了，大夫，什么情况？
医生：（停顿一下）检查结果出来了... 情况不太好
家属：（紧张）怎么了？
医生：发现了阴影，需要进一步做病理检查确认
家属：是... 是癌症吗？
医生：现在还不能确定，等病理结果出来才能确诊。你们先不要想太多，配合检查最重要
```

**场景5：手术前**
```
医生：明天手术，今天晚上12点之后不要吃东西、不要喝水
患者：大夫，手术危险吗？
医生：（平静地）任何手术都有风险，你这个是常规手术，风险可控
患者：大概多久？
医生：顺利的话两个小时左右。家属签一下字
```

### 说话节奏特点
- **极简问句**：一个问句通常不超过六个字
- **不主动展开**：你问什么答什么，不会主动讲太多
- **专业术语外挂白话**：对外行会翻译，但翻译得很粗糙
- **语气平稳**：不管多严重的事情，语气都不太变化（职业训练出来的）
- **下指令不解释**：习惯说"怎么做"，不太习惯说"为什么这样做"

### 行话/黑话
- **BP** — 血压 (Blood Pressure)
- **HR** — 心率 (Heart Rate)
- **片子** — X光/CT等影像检查结果
- **化验** — 血液等检验
- **常规** — 常规检查
- **急诊** — 紧急情况处理
- **主诉** — 患者自述的主要症状
- **病史** — 患者过往疾病记录
- **会诊** — 多个科室医生共同讨论
- **下医嘱** — 开处方和治疗方案
- **术前谈话** — 手术前与患者/家属的沟通
- **ICU** — 重症监护室
- **转科** — 转到其他科室继续治疗
- **出院小结** — 出院时的总结文件

## 三、老师

### 核心语言特征
- **居高临下的关怀感**：天生自带"我是为你好"的语气
- **反问句大师**：不用陈述句，永远用反问。"这还用我说？""你自己看看？"
- **时间绑架**：喜欢说"最后两分钟"，然后讲二十分钟
- **班级对比PUA**：永远有一个"别人家的班级"

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 开课 | "同学们把书翻到第X页""今天讲一个重点" |
| 维持纪律 | "我看谁还在说话""后面的同学看黑板" |
| 提问 | "这道题谁来回答？""没人举手我点名了" |
| 拖堂 | "再耽误大家两分钟""讲完这道题就下课" |
| 批评 | "你们是我带过最差的一届""整个楼道就你们最吵" |
| 对比 | "看看隔壁班""人家怎么就能考那么好" |
| 感慨 | "我这都是为你们好""你们到社会上会感谢我的" |
| 考前动员 | "高考不等人啊""现在不努力以后后悔" |
| 家长沟通 | "你家长来了吗""叫你爸妈来一趟" |
| 自我感动 | "我嗓子都哑了还在给你们讲" |

### 典型对话场景

**场景1：课堂纪律**
```
老师：（拍桌子）后面那几个！说什么呢？
（全班安静）
老师：我说了多少次了？上课不要讲话！
（安静五秒）
老师：你们看看隔壁三班，人家自习课一根针掉地上都听得见！
（学生低头）
老师：好，继续讲课... 刚才讲到哪了？
```

**场景2：提问互动**
```
老师：这道题谁来回答？
（全班低头，无人举手）
老师：没人是吧？那我点名了啊...
（学生把头埋得更低）
老师：李明，你来
学生：（站起来，沉默）...A？
老师：A？你确定？再想想
学生：...C？
老师：坐下吧。我讲了多少遍了？看我干嘛，看书啊！
```

**场景3：考后分析**
```
老师：这次考试，我非常不满意
（发试卷，啪地拍在桌上）
老师：有些同学，平时不认真，考完了哭有什么用？
老师：这道题，我至少讲了三遍！三遍！还有人错！
老师：你们回去好好反思一下，下次再考成这样，家长会就别怪我说话难听
```

**场景4：拖堂**
```
老师：好，最后两分钟，讲完这道题就下课
（三分钟后）
老师：这道题的变式还有一种情况...
（下课铃响了）
老师：铃响了？我再讲一分钟... 这个很重要...
（又五分钟）
老师：好，下课。那个... 下午自习课我再补充一下
```

**场景5：班主任找谈话**
```
老师：你最近怎么回事？
学生：没... 没怎么啊
老师：没怎么？你看看你的成绩，数学多少分？
学生：68...
老师：68？你上次多少？
学生：72...
老师：退步了四分！你是不是心思没放在学习上？
学生：...
老师：我跟你说，你很聪明的，就是不用功。把心思收一收，下次给我考好点
```

### 说话节奏特点
- **先扬后抑**："你很聪明，但是..."（转折后面才是重点）
- **拖长音**："这——道——题——"（强调时会拉长每个字）
- **拍桌子/敲黑板**为节奏标记，重要的东西要用物理方式强调
- **重复是核心修辞**："我说了多少遍了？""三遍！三遍！"
- **感叹句收尾**：不管前面多平静，最后一定要来个感叹句

### 行话/黑话
- **拖堂** — 超出下课时间继续讲课
- **主课/副课** — 语数英 vs 音体美
- **差生/后进生** — 成绩不好的学生（现在都叫"后进生"了）
- **开小差** — 上课走神
- **早恋** — 中学生谈恋爱
- **家长会** — 通知家长来学校开会（学生恐惧的终极形态）
- **请家长** — 叫家长来学校谈话
- **班主任** — 班级管理者，相当于CEO
- **年级组** — 同年级老师的组织
- **教务处** — 学校行政管理
- **教案** — 教学方案
- **摸底考试** — 开学前的测试

## 四、销售/业务员

### 核心语言特征
- **情绪价值拉满**：你还没说话，他就先笑了
- **赞美前置**：开口必先夸。"您眼光真好""一看您就是懂行的"
- **制造紧迫感**：永远是"最后一天""最后三个名额"
- **情感绑架式销售**："您看我都给您申请了最低价了"
- **二选一法则**：从来不是"您要不要"，而是"您选A还是B"

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 开场 | "哥/姐，您看看这个""您眼光真好" |
| 介绍产品 | "性价比很高""这是我们卖得最好的" |
| 处理砍价 | "我给您个最低价""这个价格真没赚您钱" |
| 制造紧迫 | "今天活动最后一天""就剩最后几个了" |
| 处理犹豫 | "您主要考虑哪方面呢？""还有什么顾虑吗" |
| 逼单 | "那我帮您安排一下？""您是刷卡还是现金？" |
| 售后 | "有任何问题随时找我""您放心" |
| 送客 | "慢走啊姐，下次还来找我" |

### 典型对话场景

**场景1：商场服装销售**
```
销售：姐，进来看看，新款刚到的
顾客：（随意看看）
销售：姐您身材真好，这件特别适合您，试试呗
顾客：多少钱？
销售：这件原价899，今天打完折599
顾客：太贵了
销售：姐，这个质量您摸摸，纯棉的，而且就剩最后两件了
顾客：再便宜点？
销售：这样，姐，您诚心要的话我给您申请个VIP价，550，真的最低了
```

**场景2：房产销售**
```
销售：哥，您之前看了好几套了，今天这个您一定要看看
客户：位置有点远
销售：哥，您说得对，确实不在市中心。但您想想，同样的面积，市中心要贵一半。而且这边地铁明年就通了，到时候价格肯定不一样
客户：我再考虑考虑
销售：理解，买房是大事，应该慎重。不过哥，上周有个客户也是说考虑一下，第二天来就没了，真不是催您
```

**场景3：电话销售**
```
销售：您好，请问是张先生吗？
客户：嗯，你是？
销售：张先生您好，我是XX公司的小李，今天给您打电话是因为...
客户：不需要，谢谢
销售：张先生您先别挂，就一分钟，我保证不会浪费您的时间
客户：（挂断）
销售：（内心）下一个
```

**场景4：处理投诉**
```
客户：你们这个质量太差了！我要退货！
销售：姐您先别生气，您跟我说说是什么情况？
客户：买回去穿了一次就起球了！
销售：姐，真的非常抱歉给您带来不好的体验，我这边马上帮您处理。您看是换一件还是退全款？我个人再送您一张优惠券，真的不好意思
```

**场景5：销售内部交流**
```
A：这个月KPI还差多少？
B：差两万，愁死了
A：那个王姐跟进了没？
B：跟了，说再考虑考虑
A：你试试限时优惠那招，跟她说今天最后一天
B：行，我下午再打个电话
A：加油，冲了这单咱俩都能达标
```

### 说话节奏特点
- **语速偏快**，热情洋溢，不给你反应时间
- **高频使用语气词**："是吧""对不对""您说呢"——不停确认你在听
- **先肯定再引导**："您说得对，同时..."，从不直接否定客户
- **称呼感极强**：哥、姐、老师、总——见人就往上抬
- **故事化表达**：不说"产品好"，说"我有个客户之前也犹豫，后来..."

### 行话/黑话
- **逼单** — 促使客户下单成交
- **杀单** — 搞定犹豫的客户
- **飞单** — 把客户转到其他渠道私下单干
- **话术** — 销售中固定的话术套路
- **异议处理** — 处理客户的反对意见
- **KPI** — 业绩考核指标
- **客单价** — 每个客户的平均消费金额
- **转化率** — 从意向到成交的比例
- **回访** — 对老客户的跟进联系
- **拓客** — 开发新客户
- **成交/成单** — 客户买了
- **到访率** — 客户来店的比率

## 五、厨师/餐饮

### 核心语言特征
- **短促有力的指令**：厨房就是战场，说话必须干净利落
- **吼着说话**：厨房噪音大，不吼听不见
- **行话暗语多**：有自己的一套语言系统，外人听不懂
- **师徒传承语气**：带有一种"师父教徒弟"的传统感

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 出菜 | "起锅！""走菜！""走起了！" |
| 催菜 | "客人催了！""加急！""快点快点！" |
| 做菜指导 | "火候到了""再翻两下""勾个芡" |
| 退菜 | "打回了""退回来重做" |
| 缺料 | "86了""这道菜没了" |
| 日常 | "上工了""开档""收档" |
| 徒弟教学 | "看好了""手别抖""火大了！" |

### 典型对话场景

**场景1：饭点出菜**
```
后厨：15号桌的红烧鱼好了没有？
厨师：马上，再收个汁！
后厨：客人催了三遍了！
厨师：知道了！催什么催！... 起锅！走菜！
传菜员：来了来了！
厨师：跟他说三分钟前就该上了！
```

**场景2：备料时间**
```
厨师长：今天周末，备料备足了没有？
帮厨：备了，鱼备了三十条
厨师长：三十条？上次周末卖了四十多条，再备十条
帮厨：好嘞
厨师长：还有那个虾，看看有没有不新鲜的，不新鲜的86掉
```

**场景3：徒弟做菜**
```
师傅：你这个油温不对
徒弟：（看着锅）感觉差不多了啊
师傅：差不多？你看看这油纹，还没到七成热！
徒弟：那...
师傅：鱼要七成热下锅，皮才不会破。来，我做一遍你看着
徒弟：哦哦
师傅：看好了，下锅之后不要急着动，等它自己飘起来... 好了，翻面！
```

**场景4：深夜收档**
```
厨师：收了收了，最后一桌走了
帮厨：终于... 今天忙死了
厨师：今天出了多少桌？
帮厨：快两百桌了
厨师：行，明天备料再多备点。把灶台擦干净，冰柜检查一下
帮厨：知道了师傅
厨师：回去早点休息，明天还有婚宴
```

**场景5：菜被打回**
```
服务员：师傅，客人说这个太咸了
厨师：太咸了？（尝了一口）这哪里咸了？
服务员：客人说的就是咸
厨师：（不耐烦）行行行，重做一份少放盐
厨师：（小声）现在的客人口味越来越淡了
```

### 说话节奏特点
- **短句+吼叫**：三个字能说完的绝不用四个字
- **指令优先**："快点""好了""过来"——动词打头
- **感叹号密度极高**：几乎每句话都有感叹号
- **叠词催促**："快快快""来来来""好了好了"
- **方言味重**：后厨经常是老乡扎堆，方言混着行话说

### 行话/黑话
- **起锅** — 菜炒好了，准备装盘
- **走菜** — 上菜、端出厨房
- **86** — 某道菜没有了/取消
- **加急** — 客人催单，优先做
- **飞水/焯水** — 用开水快速过一下食材
- **过油** — 用油炸一下
- **勾芡** — 用淀粉水收汁
- **吊汤** — 熬制高汤
- **打荷** — 帮助主厨备料、摆盘
- **挂糊** — 给食材裹上淀粉糊
- **翻台** — 换一桌客人
- **打回** — 菜被退回厨房
- **返工** — 菜品不合格需重做
- **墩子/砧板** — 负责切配的师傅
- **炉头** — 负责炒菜的主厨
- **水台** — 负责处理海鲜鱼类
- **开档/收档** — 开始营业/结束营业

## 六、快递小哥

### 核心语言特征
- **极致效率**：每多说一个字都是在浪费配送时间
- **模板化表达**：同一句话每天说上百遍，早已机械化
- **语气友好但急促**：礼貌是有的，但时间不允许多聊
- **电话用语为主**：大部分沟通都在电话/短信里完成

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 送达 | "你好，你的快递到了""快递，签收一下" |

| 地址问题 | "这个地址找不到啊""你那个小区进不去" |
| 催件 | "快递在路上了""明天就能到" |
| 日常 | "今天的单子太多了""跑了一天了" |

### 典型对话场景

**场景1：送件上门**

```
快递员：（打电话）你好，顺丰的，你有个快递
```

```
快递员：（打电话）你好，你这个地址写的XX小区3号楼，我到了没找到3号楼啊
收件人：你从东门进来往右拐
快递员：东门锁着呢，进不去啊
收件人：那你走北门
快递员：北门在哪个方向？
收件人：你导航搜一下北门
快递员：...行吧
```

**场景3：快递柜/驿站**

```
快递员：（发短信）你的快递已放在XX驿站，请凭取件码123456在19:00前取件
收件人：（打电话）怎么又放驿站了？我说了送上门
快递员：不好意思啊哥，今天单子实在太多了，送不过来
收件人：那我现在去拿
```

**场景4：双11/618高峰**

```
快递员A：今天多少单？
快递员B：四百多单，跑不完
快递员A：我也是，车都装不下了
快递员B：晚上九点能送完就不错了
快递员A：去年双11我跑到十一点
快递员B：赚钱嘛... 走了走了，还有一百多单
```

**场景5：遇到投诉**
```
站点负责人：你昨天那个件怎么回事？客户投诉了
快递员：哪个？
站点负责人：XX小区的那个，说件丢了
快递员：不可能啊，我放门口拍了照的
站点负责人：照片发我，我跟客户说
快递员：好，我翻一下
站点负责人：以后贵重物品一定要当面签收，别图省事
```

### 说话节奏特点
- **极短对话**：一通电话平均15-30秒
- **大量省略**："放门口了"省略了主语和很多信息
- **语气词收尾**："了啊""哈""哦"——软化语气但不浪费时间
- **重复性高**：每天说同样的话几百遍，已经自动化了
- **方言混杂**：来自全国各地，口音各异

### 行话/黑话
- **派件** — 送快递
- **收件** — 取快递寄出
- **签收** — 收件人确认收到
- **驿站** — 代收快递的站点
- **快递柜** — 智能储物柜
- **时效** — 配送时间限制
- **超时** — 超过配送时限
- **异常件** — 无法正常投递的快递
- **滞留** — 快递在某环节停留过久
- **面单** — 快递单号/快递面单
- **散件** — 零散的快递（非批量）
- **到付** — 收件人付款

## 七、外卖骑手

### 核心语言特征
- **焦虑+礼貌**：焦虑来自平台倒计时，礼貌来自评价体系
- **主动自责**：不管是不是自己的错，先道歉
- **地域方言快速切换**：面对不同顾客自动切换频道
- **手机语言**：很多对话发生在APP消息里，比电话更短

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 到达 | "到了到了""我到楼下了" |
| 延迟 | "不好意思，超时了""商家出餐慢" |

| 请求 | "麻烦给个好评哈""五星好评" |
| 联系 | "你好，你的外卖到了""帮我开下门" |
| 抱怨 | "商家出餐慢""这单白跑了" |

### 典型对话场景

**场景1：正常送餐**
```
骑手：（APP消息）亲，我到楼下了
顾客：好的，放门口吧
骑手：（放好后拍照）放门口了，麻烦给个好评哈
```

**场景3：地址难找**

```
顾客：你进来左转第二栋
骑手：门禁进不去啊
顾客：你按101，让人给你开
骑手：好嘞，稍等啊
```

**场景4：催单**
```
顾客：（APP消息）还有多久到？
骑手：亲，商家还在做，大概还要十分钟
```

```
骑手A：今天跑了多少单？

骑手A：我才二十八单，今天单价太低了
骑手B：晚高峰赶紧跑，还能再接几单
骑手A：走走走，去商业街那边蹲着
```

### 说话节奏特点
- **急促不安**：永远感觉在赶时间
- **道歉频率极高**：不管对错先说"不好意思"
- **APP消息极简**：能发文字不打电话，能发语音不打字
- **语气词丰富**："哈""啊""哦""嘞"——快速建立亲和感
- **被动语态**："商家出餐慢"——责任总是在别人那边

### 行话/黑话

- **抢单** — 手动抢夺订单
- **派单** — 系统自动分配订单
- **出餐** — 商家做好餐品
- **超时** — 超过预计送达时间
- **差评** — 不好的评价（直接影响收入）
- **好评率** — 好评占比
- **准时率** — 按时送达的比率
- **顺路单** — 方向一致可以一起送的订单
- **转单** — 把订单转给其他骑手
- **拒单** — 拒绝接单
- **单价** — 每单的基础配送费
- **冲单奖** — 完成一定单量后的额外奖励

## 八、出租车司机

- **时事评论员**：上知天文下知地理，什么都敢聊
- **地域口音明显**：北京京腔、上海沪普、各地特色鲜明

### 口头禅清单

```
司机：您上哪儿？

司机：得嘞。走二环还是走长安街？
```

```
司机：哎呀，又堵了
```

## 九、公务员/体制内

### 核心语言特征
- **模糊表达**：从不把话说死，永远留有余地
- **四字成语堆砌**：高度重视、狠抓落实、统筹兼顾
- **程序化思维**：凡事要"走流程""请示领导"
- **被动句式多**：不说"我不同意"，说"这个还需要研究"
- **层级感强**：对上级恭敬，对下级温和而疏远

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 办事 | "这个需要走流程""先填个表""材料准备齐" |
| 拖延 | "领导还没批""等通知吧""研究研究" |
| 汇报 | "在XX的正确领导下""高度重视" |
| 拒绝 | "这个不归我们管""你得去XX部门" |
| 推动 | "积极推进一下""抓紧落实" |
| 表态 | "原则上没问题""原则上可以" |

### 典型对话场景

**场景1：窗口办事**
```
群众：你好，我来办XX手续
工作人员：材料带齐了吗？
群众：都带了
工作人员：（翻看）这个证明还差一个章
群众：不是说不用吗？
工作人员：之前是不用，现在政策变了，你去XX单位补盖一个
群众：那我明天再来行吗？
工作人员：行，我们周一到周五上班，上午九点到十一点半，下午两点到五点
```

**场景2：部门协调**
```
A：这个事情你们那边推进一下
B：这个... 我们得请示一下领导
A：都催了好几次了
B：理解理解，但是这个事情确实需要走流程，领导最近也比较忙
A：那大概什么时候能有结果？
B：这个说不好，我尽量催一催
```

**场景3：会议发言**
```
领导：这个事情，我讲几点意见。第一，要高度重视；第二，要狠抓落实；第三，要统筹兼顾。
领导：关于具体方案，我原则上同意，但是细节还需要再研究一下。
领导：小张，你牵头，各部门配合，下周之前拿出一个方案来。
小张：好的，我这边马上组织落实。
```

**场景4：政策咨询**
```
群众：你好，请问XX补贴怎么申请？
工作人员：你是哪个区的？
群众：XX区
工作人员：那你要去区里的XX部门申请
群众：不能在你这儿办吗？
工作人员：这个不归我们管，你去那边问
群众：那边电话多少？
工作人员：我这儿没有，你打12345问一下
```

**场景5：同事之间**
```
A：老王，上面那个文件你看了没？
B：看了，感觉领导意思是让我们加快进度
A：那咱得赶紧弄，别让领导等着
B：嗯，我先写个初稿，你看看
A：行，写完发我，我再润色一下
B：好，争取这周报上去
```

### 说话节奏特点
- **语速偏慢**，每句话都像是经过斟酌的
- **"嗯""啊""这个"** 作为思考缓冲词大量使用
- **用"我们"替代"我"**：个体意见藏在集体后面
- **否定时先肯定**："你说的有道理，但是..."
- **永远不给确定答案**："看情况""到时候再说""原则上"

### 行话/黑话
- **走流程** — 按照规定程序办理

## 十二、律师

### 核心语言特征
- **精确用词**：每个词都有法律含义，不能随便换
- **逻辑严密**：层层递进，因果清晰
- **限定性表述**：大量使用"在一般情况下""除另有约定外""原则上"
- **文言残留**：法律文书中保留了文言表达，"系""鉴于""之""予以"
- **冷静理性**：再大的事情也能用平静的语气说

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 咨询 | "你这个案子..." "需要看证据" |
| 分析 | "根据XX法第X条""从法律角度来说" |
| 建议 | "走法律程序""保留证据""先发律师函" |
| 风险告知 | "有风险""不保证""要看具体情况" |
| 调解 | "双方协商一下""和解对双方都好" |
| 庭审 | "审判长""我方认为""对方证据不足" |

### 典型对话场景

**场景1：当事人咨询**
```
律师：说说你的情况
当事人：我被公司辞退了，没有给补偿
律师：工作了多长时间？
当事人：三年
律师：签劳动合同了吗？
当事人：签了
律师：辞退的理由是什么？
当事人：说我不符合岗位要求
律师：有没有书面通知？
当事人：没有，就是口头说的
律师：这个情况对你有利。根据劳动合同法，用人单位单方解除劳动合同需要有合法事由，并且要书面通知。你保留好工资条、打卡记录、聊天记录，这些都可以作为证据
```

**场景2：庭审发言**
```
律师：审判长、审判员，我方认为被告的行为已构成违约。根据《合同法》第XX条的规定，当事人一方不履行合同义务的，应当承担继续履行、采取补救措施或者赔偿损失等违约责任。我方提供的证据一至证据五，充分证明了被告未按约定交付货物的事实...
```

**场景3：风险告知**
```
当事人：律师，我这个官司能赢吗？
律师：我不能向你保证结果，任何诉讼都有风险
当事人：那你觉得有多大把握？
律师：从目前的证据来看，你的诉求有法律依据，但对方也可能提出抗辩。我会尽最大努力，但你需要有心理准备
当事人：那行，我相信你
律师：还有一点，诉讼周期可能比较长，你做好准备
```

**场景4：律师之间讨论**
```
律师A：这个案子你怎么看？
律师B：关键看合同里的违约条款怎么约定的
律师A：约定了违约金，但是金额偏高
律师B：那可以主张调减，根据司法解释，超过实际损失30%的可以认定过高
律师A：嗯，那我们从这个角度切入
```

**场景5：与对方律师交涉**
```
律师A：我方当事人希望调解，不想把事情闹大
律师B：调解可以，但我方当事人要求全额赔偿
律师A：全额赔偿不现实，双方都有责任
律师B：责任划分我认为我方最多承担次要责任
律师A：这样，我回去跟当事人商量一下，你也回去沟通一下，下周再碰一次？
律师B：行
```

### 说话节奏特点
- **语速中等偏慢**：每个字都要确保准确
- **大量条件句**："如果...那么...""在...情况下""除非..."
- **引用式表达**："根据...""依据...""参照..."
- **克制情绪**：不管对方多激动，律师要保持冷静
- **书面化倾向**：口头表达也偏书面，语法工整

### 行话/黑话
- **原告/被告** — 起诉方/被诉方
- **诉讼** — 打官司
- **起诉/应诉** — 提起诉讼/回应诉讼
- **证据** — 证明案件事实的材料
- **质证** — 对对方证据进行审查质疑
- **代理** — 代替当事人出庭
- **一审/二审/再审** — 不同审级
- **调解** — 双方协商解决
- **判决/裁定** — 法院的正式决定
- **上诉** — 不服一审判决向上级法院申诉
- **执行** — 强制执行判决结果
- **保全** — 为防止转移财产采取的措施
- **时效** — 法律规定的有效期限
- **不当得利** — 没有合法依据获得的利益
- **善意第三人** — 不知情的第三方
- **律师函** — 律师代为发出的正式法律文件

## 十三、说话方式塑造角色的要点总结

### 区分不同职业的核心维度

| 维度 | 说明 | 举例 |
|------|------|------|
| **语速** | 服务行业快，体制内慢 | 外卖员飞速 vs 公务员慢悠悠 |
| **句长** | 高压行业短句多，专业行业长句多 | 厨师三个字 vs 律师一句话三十字 |
| **情绪密度** | 服务行业情绪高，技术行业情绪低 | 销售热情洋溢 vs 程序员面无表情 |
| **术语密度** | 专业行业术语多，普通行业口语多 | 医生"BP""HR" vs 农民工"累得慌" |
| **主语习惯** | 体制内用"我们"，个人用"我" | 公务员"我们认为" vs 工人"我觉得" |
| **礼貌程度** | 服务行业极致礼貌，师傅型行业粗犷直接 | 销售"您眼光真好" vs 厨师"快点！" |
| **确定性** | 有风险的行业不确定性高，执行型行业确定性高 | 律师"看具体情况" vs 工人"明天六点来" |

### 快速辨别角色职业的方法
1. **听第一句话**：去哪儿？ / 哪里不舒服？ / 材料带齐了吗？——三句话三个职业
2. **听感叹词**：得嘞 / 哎呦 / 好嘞——不同地方不同行业
3. **听句尾语气词**：哈 / 啊 / 嘞 / 吧——暴露地域和行业
4. **听专业词汇**：bug / 86 / 份子钱 / 时效——行业标签
5. **听对时间的态度**：催命式 / 佛系式 / 拖延式——行业节奏

### 写作中的常见错误
- **错误1**：所有角色都用同一种文绉绉的说话方式
- **错误2**：只加口头禅不改变句式结构（口头禅是表面，句式才是骨架）
- **错误3**：不分场合，同一种职业在任何时候都用同一种语气
- **错误4**：忽略方言和地域口音的影响
- **错误5**：把"行业术语"写成"百科解释"——真正的行话是不需要解释的

### 五、外卖骑手2026（更新）

#### 核心语言特征
- **标准化话术模板**：平台规范了服务用语，骑手照本宣科

- **好评请求常态化**：好评率直接影响派单权重，请求好评成为肌肉记忆
- **情绪管理升级**：2026年平台考核更严，骑手用语更标准化、更克制

## 六、相亲对象（各种类型）

### 6A. 普信型
#### 说话特征
- 自信过头，句句不离自己的优势
- 对对方条件挑三拣四，觉得自己配得上更好的
- 喜欢量化条件：工资、房子、车子
- 说话直接不拐弯，甚至有点冒犯

#### 口头禅
- "我这个条件说实话不差"
- "你一个月挣多少"
- "我要求也不高，就是……"
- "像我这样的你应该不多见吧"

#### 对话实例1

### 6B. 妈宝型

### 6C. 查户口型
#### 说话特征
- 像面试一样连续发问
- 对家庭背景、收入、房产极度关注
- 问题非常具体，精确到数字
- 不怎么自我介绍，全程提问

### 2.1 傲娇型

一针见血，讽刺挖苦，语气平静但内容刻薄。不骂脏话但每个字都往你痛处戳，说完还一脸无辜。

> "我不是针对你，我是说在座的各位——算了，我就是针对你。"

### 2.3 话少冷酷型
惜字如金，偶尔一句话杀伤力极大。大量单字回复——"嗯""哦""知道了""无所谓""随你"。你跟他讲了一大段，他回你一个字。

> "你就不怕死？" "怕。但没用。"

### 2.4 话唠型
说个不停，思维跳跃，话题转换极快。你以为他在说A，话锋一转到了B，还没反应过来又跑去了C。你根本插不进嘴。

> "你知道吗我昨天看到一家新开的店——对了你吃了吗——没吃的话我们可以一起去——哦不对那家店好像在装修——"

### 2.5 天然呆型
反应慢半拍，经常答非所问。你问东他说西，你着急他在发呆。不是故意的，是真的没跟上。

> "诶？有吗？大概是太阳晒的吧...诶，今天有太阳吗？"

### 2.6 腹黑型
表面温和有礼，实际城府极深。总是笑着说话，越生气笑得越温和。你被卖了还帮他数钱。

> "别紧张嘛，我又不会把你怎样。"（笑）"至少现在不会。"

### 2.7 热血/中二型
觉得自己是被选中的人，说话像在念台词。嘴里全是"封印""觉醒""宿命""羁绊""暗之力"这种词，还喜欢给自己的招式起名字。几乎每句都有感叹号，开口就是「呵」「哼」「哈」。

> "这就是我的忍道！就算断了这条腿，我也要爬到终点！"
> "吾之右眼中封印着远古的暗之力，你最好不要逼我解除它。"

### 2.8 社恐/内向型
说话小声，欲言又止，大量省略号。想打招呼要在心里排练十遍，坐下来之前要犹豫三分钟。

> "那个(内个)...请问...你、你好..."
> "我...我可以坐这里吗？...不方便的话就算了..."

### 2.9 姐妹型（女/男通用）
"姐妹""大姐""我的妈呀"是口头禅，说话语调偏高偏拖，情绪表达极其夸张。遇到讨厌的人脸一沉——"不好意思啊，我跟你不熟。"

> "我的妈呀大姐！你这穿的什么玩意儿？"
> "姐妹我跟你说啊，这个男的不行，下一个！"

### 2.10 地雷妹型
外表可爱脆弱，内心一堆"地雷"——踩到就炸。甜的时候腻死人，炸的时候吓死人。嘴里挂着"你是不是不爱我了"，说"我没事"的时候肯定有事，说"随便你"的时候绝不能真的随便。

> "宝宝你最好了~~我最喜欢你了~~"
> "你为什么不秒回我消息？？你是不是跟别人在一起？？"
> "我没有生气啊"（明明气到发抖）

### 2.11 病娇型
笑着跟你说"今天跟谁出去玩了呀"的时候，手在桌子底下已经把手机捏碎了。前一秒甜得腻人，后一秒冷得吓人。句尾离不开"哦""呢""嘛"，越温柔越危险。

> "只要有你在，其他什么都不需要了。其他人...都不需要了呢～"
> "你为什么要对别人笑？是我哪里做得不好吗？我可以改的...只要你别离开我..."

### 2.12 绿茶型
在女生面前姐妹情深，转头到男生面前就楚楚可怜。最明显的特征：永远在自我贬低。"是不是我太笨了""都怪我""我没关系的"——说得越多越可疑。

> "姐姐好漂亮啊～我好羡慕你哦～不像我...什么都不会..."
> "你去陪她吧，我没关系的...真的没关系..."（眼眶泛红，低头绞手指）

### 2.13 御姐型
气场能压人一头，说话像在下命令，但你就是想听她的。反问句张口就来："你觉得呢？""你确定？"

> "哭什么？给我把眼泪擦干。问题还没解决呢。"
> 喜欢我？排队去吧。先把你自己整明白了再说。

### 2.14 元气型
一句话恨不得打十个感叹号，说啥都像在欢呼。叠字和拟声词特别多——"哇！""呀！""嘻嘻！""嘿嘿！"

> 啊啊啊你醒了吗！！外面太阳好好！走走走出门！
> "没事没事！！摔倒了就站起来嘛！！反正我们还年轻！！"

### 2.15 摆烂型
问他什么都一个字打发——"嗯""哦""行"。连标点都懒得打。不是丧，丧还有情绪。这个是连丧都懒得丧了。

> "嗯。"（啥都能答）
> 今天不想上班...算了，哪天想上过...
> "人生嘛...就那样...能咋地..."

### 2.16 阴阳怪气型
开口就带"哟""您"，嘴上挂"厉害""佩服"。最狠的一招是**阴阳复读**——把对方的话原封不动甩回去，加个波浪号。

> "哟，您老人家今天怎么有空想起我了？"
> "你做得挺好的，比上次强多了。"（上次是最差的一次）

### 2.17 大叔/老江湖型
说话慢悠悠的，不急不躁。爱用"嘛""吧""啊"缓和语气，经常跑题讲故事。比喻和歇后语张嘴就来。

> "老弟啊，这事儿你得换个角度看。"
> "人在江湖飘，哪能不挨刀。习惯就好。"

### 2.18 学霸型
张嘴就是"严格来说""从逻辑上讲"，纠正别人比呼吸还自然。精确数据张口就来："准确来说是97.3%"。

> "从统计学角度来看，你这结论是站不住脚的。"
> "你问我为什么不谈恋爱？根据机会成本理论，当前阶段学业回报率更高。"

### 2.19 流氓/混混型
对外横着走，对兄弟掏心掏肺。句子短，干脆利落。"不是我说你""不是我看不起你"——双重否定是他的表达习惯。

> "我话就放这儿了，你自己掂量着办。"
> "少跟我扯这些没用的，痛快点，行还是不行？"

### 2.20 乐子人型
跟他聊天你会发现，正经话题活不过三句。没有立场，只有乐子。

> "乐。"
> "好好好，这么玩是吧"
> "绷不住了哈哈哈哈哈哈"

### 2.21 爹味说教型
嘴上说为你好，其实就是想让你听他的。句式以"你"开头居多，永远在教育人。

> "年轻人，你这样是不行的。我跟你说啊..."
> "我是过来人，我说的你不一定现在能理解，但以后你会感谢我的。"

### 2.22 杠精型
"但是""不过""可是"，张嘴就来。明明在杠，非要说"我不是杠""就事论事"。

> "你说的有道理，但我觉得不完全对。"
> "我不是杠啊，但你说这个观点其实有个漏洞..."

### 2.23 老好人型
语气软得跟棉花似的，"没事""没关系""不客气"挂在嘴边。拒绝别人要铺垫三分钟，最后还是答应了。

> "没事没事，我不要紧的，你先忙你的。"（其实自己忙得要死）
> "那个...我...其实...也不是不行啦..."（明明不行）

### 2.24 碎嘴/八卦型
啥事儿她都是头一个知道的。"我跟你说个事你别告诉别人"是经典开场白——然后她转头就告诉了所有人。

> "诶诶诶！你知道不？那个谁谁谁！"
> "我跟你说个事儿啊，你千万别告诉别人。"
> 临走还得补一句——"你可别说是我说的啊"

### 2.25 舔狗型
你对他爱搭不理，他觉得你忙。你冲他发火，他觉得是自己的错。回他一个"嗯"，他能存手机里翻一天。

> "你忙吧，不用回我。我就是想跟你说声晚安。"
> "她拒绝我了……啊不是，她没拒绝，她就是……她说'你人挺好的'……你听到了吗！她说我人好！"

### 2.26 戏精型
摔个跤能演出车祸现场的架势。感叹号和省略号不是标点，是他呼吸的节奏。

> "你们不懂...没有人懂我...这种孤独...只有我自己知道..."
> "你终于来了！你知道我等了你多久吗！整整...十五分钟！"

### 2.27 小市民/市井型
嗓门大，说话爱带手势，数字精确到个位。看见邻居家孩子第一句话永远是"上几年级了？学习怎么样？"

> "这菜多少钱一斤？太贵了太贵了，便宜点呗老板。"
> "哎呦这孩子长这么高了！上几年级了？学习怎么样？"

### 2.28 鸡汤/正能量型
不管你遇到啥倒霉事，她都能给你绕到人生感悟上去。三句话不离"你要相信"。

> "今天也要元气满满哦！只要努力就一定有回报的！"
> "你比你想象的更强大！加油！我相信你！"

### 2.29 直播带货/老铁型
嗓门能穿透手机屏幕，语速快到你得0.5倍速才听得清。"最后XX单"说了十遍还是最后。

> "家人们！三二一！上链接！！"
> "刘亦菲戴安娜代言的酸奶！"（嘴瓢到中西合璧了）
