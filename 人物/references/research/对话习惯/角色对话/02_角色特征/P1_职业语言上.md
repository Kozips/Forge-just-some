# 13 职业说话方式研究

> 职业塑造语言。一个人在某个行业干上三五年，说话方式就会被彻底改写——用词、节奏、口头禅、思维逻辑，全都刻上行业的烙印。这是塑造真实角色的关键素材。

---

## 一、程序员/码农

### 核心语言特征
- **中英混杂**：不是装X，是真的想不起来中文怎么说。"这个bug我fix一下""先merge到主分支再deploy"
- **极简表达**：能一个字说完的不说两个字。"好""行""收到""可以"
- **逻辑先行**：先说结论，再说原因。"不行，因为这个方案有并发问题"
- **被动句式**：习惯说"被""让"，因为代码世界里自己经常不是主导者

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 出了问题 | "在我电脑上是好的啊" |
| 推卸责任 | "这不是bug，这是feature" |
| 万能解决 | "重启试试""清一下缓存" |
| 需求变更 | "需求又变了？""这不合理" |
| 写代码 | "我先加个TODO"（然后永远不做了） |
| 甩锅 | "这段代码谁写的？" → git blame → 是自己 |
| 评估工期 | "三天能搞定"（然后三天又三天） |
| 代码审查 | "LGTM""Nit: 变量名可以改一下" |
| 技术争论 | "这个trade-off要看场景" |
| 上线部署 | "部署一下""回滚""灰度发布" |
| 项目管理 | "排期排不开""这个优先级不够" |

### 典型对话场景

**场景1：线上出了bug**
```
A: 线上挂了！
B: 什么情况？
A: 用户反馈登录不了，502了
B: 我看看日志... 是不是昨天的变更导致的？
A: 不是吧，那个已经灰度了啊
B: 先回滚，查完再上线
A: 好，我先回滚
```

**场景2：需求评审**
```
产品：我想加个功能，用户可以看到...
开发：等一下，这个需求逻辑不通
产品：怎么不通了？
开发：你说的这个场景，0和1之间没有第三种状态
产品：那能不能变通一下？
开发：（沉默三秒）我回去评估一下工作量
```

**场景3：代码审查**
```
A: 帮我CR一下这个PR
B: 整体LGTM，Nit: 这个变量名能不能改一下？叫tempValue太泛了
A: 行，改了
B: 还有这里，建议用Optional，避免空指针
A: 嗯，这个我考虑过，主要是兼容老代码
B: OK，那加个注释说明一下
```

**场景4：工位日常**
```
A: 你在干嘛？
B: 在debug一个内存泄漏的问题
A: 找到了吗？
B: 没有，怀疑是第三方库的问题
A: 重启试试？
B: ...说了跟没说一样
```

**场景5：周五下班前**
```
A: 走不走？
B: 等一下，我先提交一下
A: 别提交了，周一再说
B: 不行，不提交我周末睡不着
A: 你这是有强迫症
B: 这叫职业素养
```

### 说话节奏特点
- **短句为主**，很少说长句。三五个字就是一个完整的表达
- **思考时有停顿**，常用"嗯...""那个..."做缓冲
- **语气偏平**，情绪表达内敛。激动的时候也不是喊，而是语速加快
- **擅长用反问**："这不合理吧？""你确定？""这样能行？"
- **括号式表达**：喜欢在说话时补充限定条件。"可以，但是..."

### 行话/黑话
- **bug** — 程序缺陷
- **CR** — Code Review，代码审查
- **PR** — Pull Request，合并请求
- **LGTM** — Looks Good To Me，审查通过
- **PTAL** — Please Take A Look，请看一下
- **Nit** — 小问题，不影响合并
- **Blocker** — 必须修复的阻塞问题
- **deploy** — 部署上线
- **rollback** — 回滚，撤回上线
- **灰度** — 小范围发布测试
- **hotfix** — 紧急修复
- **技术债** — 欠下的需要后期偿还的技术工作
- **甩锅** — 把责任推给别人
- **造轮子** — 重复开发已有功能
- **屎山代码** — 质量很差的遗留代码

## 二、医生

### 核心语言特征
- **命令式语气**：习惯了对患者下指令。"躺下""张嘴""深呼吸"
- **高度省略**：时间就是生命，说话极度精简。"多久了？""哪里疼？"
- **专业术语+白话切换**：对同事说"疑似心梗"，对患者说"心脏有点问题"
- **被动关怀**：不太会直接表达关心，而是通过"多喝水""注意休息"来体现

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 问诊开场 | "哪里不舒服？""怎么了？" |
| 了解病情 | "多久了？""有没有过敏史？" |
| 检查指令 | "张嘴看看""躺下，放松""深呼吸""翻个身" |
| 安排检查 | "先去做个检查""抽个血""拍个片子" |
| 诊断告知 | "问题不大""不用太担心""你这个情况很常见" |
| 治疗建议 | "按时吃药""多喝水""少吃辛辣""别熬夜" |
| 告别 | "回去好好休息""有情况随时来" |
| 重症告知 | "家属在吗？""叫你家属来一趟" |

### 典型对话场景

**场景1：门诊问诊**
```
医生：怎么了？
患者：肚子疼
医生：多久了？
患者：昨天晚上开始的
医生：拉不拉？
患者：拉了三四次
医生：发烧吗？
患者：有点低烧
医生：来，躺床上，我按一下... 这里疼不疼？这里呢？
患者：哎呦！就那儿！
医生：先去验个血，做个B超，结果出来再看
```

**场景2：查房**
```
医生：（翻看病历）今天感觉怎么样？
患者：好多了
医生：（听诊）深呼吸... 好。大便正常吗？
患者：昨天正常了
医生：行，再观察两天，没问题可以出院了
护士：张大夫，3床的化验结果出来了
医生：好，我看看
```

**场景3：急诊抢救**
```
医生：什么情况？
家属：他突然倒下去了！
医生：意识清醒吗？有呼吸吗？
护士：血压80/50，心率120
医生：上心电监护，开放静脉通路，先推一支肾上腺素
护士：好的
医生：家属在外面等着，别进来！
```

**场景4：告知坏消息**
```
医生：家属来了吗？坐
家属：来了，大夫，什么情况？
医生：（停顿一下）检查结果出来了... 情况不太好
家属：（紧张）怎么了？
医生：发现了阴影，需要进一步做病理检查确认
家属：是... 是癌症吗？
医生：现在还不能确定，等病理结果出来才能确诊。你们先不要想太多，配合检查最重要
```

**场景5：手术前**
```
医生：明天手术，今天晚上12点之后不要吃东西、不要喝水
患者：大夫，手术危险吗？
医生：（平静地）任何手术都有风险，你这个是常规手术，风险可控
患者：大概多久？
医生：顺利的话两个小时左右。家属签一下字
```

### 说话节奏特点
- **极简问句**：一个问句通常不超过六个字
- **不主动展开**：你问什么答什么，不会主动讲太多
- **专业术语外挂白话**：对外行会翻译，但翻译得很粗糙
- **语气平稳**：不管多严重的事情，语气都不太变化（职业训练出来的）
- **下指令不解释**：习惯说"怎么做"，不太习惯说"为什么这样做"

### 行话/黑话
- **BP** — 血压 (Blood Pressure)
- **HR** — 心率 (Heart Rate)
- **片子** — X光/CT等影像检查结果
- **化验** — 血液等检验
- **常规** — 常规检查
- **急诊** — 紧急情况处理
- **主诉** — 患者自述的主要症状
- **病史** — 患者过往疾病记录
- **会诊** — 多个科室医生共同讨论
- **下医嘱** — 开处方和治疗方案
- **术前谈话** — 手术前与患者/家属的沟通
- **ICU** — 重症监护室
- **转科** — 转到其他科室继续治疗
- **出院小结** — 出院时的总结文件

## 三、老师

### 核心语言特征
- **居高临下的关怀感**：天生自带"我是为你好"的语气
- **反问句大师**：不用陈述句，永远用反问。"这还用我说？""你自己看看？"
- **时间绑架**：喜欢说"最后两分钟"，然后讲二十分钟
- **班级对比PUA**：永远有一个"别人家的班级"

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 开课 | "同学们把书翻到第X页""今天讲一个重点" |
| 维持纪律 | "我看谁还在说话""后面的同学看黑板" |
| 提问 | "这道题谁来回答？""没人举手我点名了" |
| 拖堂 | "再耽误大家两分钟""讲完这道题就下课" |
| 批评 | "你们是我带过最差的一届""整个楼道就你们最吵" |
| 对比 | "看看隔壁班""人家怎么就能考那么好" |
| 感慨 | "我这都是为你们好""你们到社会上会感谢我的" |
| 考前动员 | "高考不等人啊""现在不努力以后后悔" |
| 家长沟通 | "你家长来了吗""叫你爸妈来一趟" |
| 自我感动 | "我嗓子都哑了还在给你们讲" |

### 典型对话场景

**场景1：课堂纪律**
```
老师：（拍桌子）后面那几个！说什么呢？
（全班安静）
老师：我说了多少次了？上课不要讲话！
（安静五秒）
老师：你们看看隔壁三班，人家自习课一根针掉地上都听得见！
（学生低头）
老师：好，继续讲课... 刚才讲到哪了？
```

**场景2：提问互动**
```
老师：这道题谁来回答？
（全班低头，无人举手）
老师：没人是吧？那我点名了啊...
（学生把头埋得更低）
老师：李明，你来
学生：（站起来，沉默）...A？
老师：A？你确定？再想想
学生：...C？
老师：坐下吧。我讲了多少遍了？看我干嘛，看书啊！
```

**场景3：考后分析**
```
老师：这次考试，我非常不满意
（发试卷，啪地拍在桌上）
老师：有些同学，平时不认真，考完了哭有什么用？
老师：这道题，我至少讲了三遍！三遍！还有人错！
老师：你们回去好好反思一下，下次再考成这样，家长会就别怪我说话难听
```

**场景4：拖堂**
```
老师：好，最后两分钟，讲完这道题就下课
（三分钟后）
老师：这道题的变式还有一种情况...
（下课铃响了）
老师：铃响了？我再讲一分钟... 这个很重要...
（又五分钟）
老师：好，下课。那个... 下午自习课我再补充一下
```

**场景5：班主任找谈话**
```
老师：你最近怎么回事？
学生：没... 没怎么啊
老师：没怎么？你看看你的成绩，数学多少分？
学生：68...
老师：68？你上次多少？
学生：72...
老师：退步了四分！你是不是心思没放在学习上？
学生：...
老师：我跟你说，你很聪明的，就是不用功。把心思收一收，下次给我考好点
```

### 说话节奏特点
- **先扬后抑**："你很聪明，但是..."（转折后面才是重点）
- **拖长音**："这——道——题——"（强调时会拉长每个字）
- **拍桌子/敲黑板**为节奏标记，重要的东西要用物理方式强调
- **重复是核心修辞**："我说了多少遍了？""三遍！三遍！"
- **感叹句收尾**：不管前面多平静，最后一定要来个感叹句

### 行话/黑话
- **拖堂** — 超出下课时间继续讲课
- **主课/副课** — 语数英 vs 音体美
- **差生/后进生** — 成绩不好的学生（现在都叫"后进生"了）
- **开小差** — 上课走神
- **早恋** — 中学生谈恋爱
- **家长会** — 通知家长来学校开会（学生恐惧的终极形态）
- **请家长** — 叫家长来学校谈话
- **班主任** — 班级管理者，相当于CEO
- **年级组** — 同年级老师的组织
- **教务处** — 学校行政管理
- **教案** — 教学方案
- **摸底考试** — 开学前的测试

## 四、销售/业务员

### 核心语言特征
- **情绪价值拉满**：你还没说话，他就先笑了
- **赞美前置**：开口必先夸。"您眼光真好""一看您就是懂行的"
- **制造紧迫感**：永远是"最后一天""最后三个名额"
- **情感绑架式销售**："您看我都给您申请了最低价了"
- **二选一法则**：从来不是"您要不要"，而是"您选A还是B"

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 开场 | "哥/姐，您看看这个""您眼光真好" |
| 介绍产品 | "性价比很高""这是我们卖得最好的" |
| 处理砍价 | "我给您个最低价""这个价格真没赚您钱" |
| 制造紧迫 | "今天活动最后一天""就剩最后几个了" |
| 处理犹豫 | "您主要考虑哪方面呢？""还有什么顾虑吗" |
| 逼单 | "那我帮您安排一下？""您是刷卡还是现金？" |
| 售后 | "有任何问题随时找我""您放心" |
| 送客 | "慢走啊姐，下次还来找我" |

### 典型对话场景

**场景1：商场服装销售**
```
销售：姐，进来看看，新款刚到的
顾客：（随意看看）
销售：姐您身材真好，这件特别适合您，试试呗
顾客：多少钱？
销售：这件原价899，今天打完折599
顾客：太贵了
销售：姐，这个质量您摸摸，纯棉的，而且就剩最后两件了
顾客：再便宜点？
销售：这样，姐，您诚心要的话我给您申请个VIP价，550，真的最低了
```

**场景2：房产销售**
```
销售：哥，您之前看了好几套了，今天这个您一定要看看
客户：位置有点远
销售：哥，您说得对，确实不在市中心。但您想想，同样的面积，市中心要贵一半。而且这边地铁明年就通了，到时候价格肯定不一样
客户：我再考虑考虑
销售：理解，买房是大事，应该慎重。不过哥，上周有个客户也是说考虑一下，第二天来就没了，真不是催您
```

**场景3：电话销售**
```
销售：您好，请问是张先生吗？
客户：嗯，你是？
销售：张先生您好，我是XX公司的小李，今天给您打电话是因为...
客户：不需要，谢谢
销售：张先生您先别挂，就一分钟，我保证不会浪费您的时间
客户：（挂断）
销售：（内心）下一个
```

**场景4：处理投诉**
```
客户：你们这个质量太差了！我要退货！
销售：姐您先别生气，您跟我说说是什么情况？
客户：买回去穿了一次就起球了！
销售：姐，真的非常抱歉给您带来不好的体验，我这边马上帮您处理。您看是换一件还是退全款？我个人再送您一张优惠券，真的不好意思
```

**场景5：销售内部交流**
```
A：这个月KPI还差多少？
B：差两万，愁死了
A：那个王姐跟进了没？
B：跟了，说再考虑考虑
A：你试试限时优惠那招，跟她说今天最后一天
B：行，我下午再打个电话
A：加油，冲了这单咱俩都能达标
```

### 说话节奏特点
- **语速偏快**，热情洋溢，不给你反应时间
- **高频使用语气词**："是吧""对不对""您说呢"——不停确认你在听
- **先肯定再引导**："您说得对，同时..."，从不直接否定客户
- **称呼感极强**：哥、姐、老师、总——见人就往上抬
- **故事化表达**：不说"产品好"，说"我有个客户之前也犹豫，后来..."

### 行话/黑话
- **逼单** — 促使客户下单成交
- **杀单** — 搞定犹豫的客户
- **飞单** — 把客户转到其他渠道私下单干
- **话术** — 销售中固定的话术套路
- **异议处理** — 处理客户的反对意见
- **KPI** — 业绩考核指标
- **客单价** — 每个客户的平均消费金额
- **转化率** — 从意向到成交的比例
- **回访** — 对老客户的跟进联系
- **拓客** — 开发新客户
- **成交/成单** — 客户买了
- **到访率** — 客户来店的比率

## 五、厨师/餐饮

### 核心语言特征
- **短促有力的指令**：厨房就是战场，说话必须干净利落
- **吼着说话**：厨房噪音大，不吼听不见
- **行话暗语多**：有自己的一套语言系统，外人听不懂
- **师徒传承语气**：带有一种"师父教徒弟"的传统感

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 出菜 | "起锅！""走菜！""走起了！" |
| 催菜 | "客人催了！""加急！""快点快点！" |

### 典型对话场景

**场景1：饭点出菜**
```
后厨：15号桌的红烧鱼好了没有？
厨师：马上，再收个汁！
后厨：客人催了三遍了！
厨师：知道了！催什么催！... 起锅！走菜！
传菜员：来了来了！
厨师：跟他说三分钟前就该上了！
```

**场景2：备料时间**
```
厨师长：今天周末，备料备足了没有？
帮厨：备了，鱼备了三十条
厨师长：三十条？上次周末卖了四十多条，再备十条
帮厨：好嘞
厨师长：还有那个虾，看看有没有不新鲜的，不新鲜的86掉
```

**场景3：徒弟做菜**
```
师傅：你这个油温不对
徒弟：（看着锅）感觉差不多了啊
师傅：差不多？你看看这油纹，还没到七成热！
徒弟：那...
师傅：鱼要七成热下锅，皮才不会破。来，我做一遍你看着
徒弟：哦哦
师傅：看好了，下锅之后不要急着动，等它自己飘起来... 好了，翻面！
```

**场景4：深夜收档**
```
厨师：收了收了，最后一桌走了
帮厨：终于... 今天忙死了
厨师：今天出了多少桌？
帮厨：快两百桌了
厨师：行，明天备料再多备点。把灶台擦干净，冰柜检查一下
帮厨：知道了师傅
厨师：回去早点休息，明天还有婚宴
```

**场景5：菜被打回**
```
服务员：师傅，客人说这个太咸了
厨师：太咸了？（尝了一口）这哪里咸了？
服务员：客人说的就是咸
厨师：（不耐烦）行行行，重做一份少放盐
厨师：（小声）现在的客人口味越来越淡了
```

### 说话节奏特点
- **短句+吼叫**：三个字能说完的绝不用四个字
- **指令优先**："快点""好了""过来"——动词打头
- **感叹号密度极高**：几乎每句话都有感叹号
- **叠词催促**："快快快""来来来""好了好了"
- **方言味重**：后厨经常是老乡扎堆，方言混着行话说

### 行话/黑话
- **起锅** — 菜炒好了，准备装盘
- **走菜** — 上菜、端出厨房
- **86** — 某道菜没有了/取消
- **加急** — 客人催单，优先做
- **飞水/焯水** — 用开水快速过一下食材
- **过油** — 用油炸一下
- **勾芡** — 用淀粉水收汁
- **吊汤** — 熬制高汤
- **打荷** — 帮助主厨备料、摆盘
- **挂糊** — 给食材裹上淀粉糊
- **翻台** — 换一桌客人
- **打回** — 菜被退回厨房
- **返工** — 菜品不合格需重做
- **墩子/砧板** — 负责切配的师傅
- **炉头** — 负责炒菜的主厨
- **水台** — 负责处理海鲜鱼类
- **开档/收档** — 开始营业/结束营业
