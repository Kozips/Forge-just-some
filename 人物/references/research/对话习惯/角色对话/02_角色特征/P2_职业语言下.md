# 角色语言特征 · 职业语言（下）

> 快递/外卖/出租车/公务员/农民工/警察/律师

---

## 六、快递小哥

### 核心语言特征
- **极致效率**：每多说一个字都是在浪费配送时间
- **模板化表达**：同一句话每天说上百遍，早已机械化
- **语气友好但急促**：礼貌是有的，但时间不允许多聊
- **电话用语为主**：大部分沟通都在电话/短信里完成

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 送达 | "你好，你的快递到了""快递，签收一下" |
| 放置 | "放门口了啊""放驿站了""放柜子里了" |
| 联系不上 | "这个电话打不通啊""人不在家" |
| 地址问题 | "这个地址找不到啊""你那个小区进不去" |
| 催件 | "快递在路上了""明天就能到" |
| 日常 | "今天的单子太多了""跑了一天了" |

### 典型对话场景

**场景1：送件上门**
```
快递员：（打电话）你好，顺丰的，你有个快递
收件人：你放门口吧
快递员：行，放门口了啊
收件人：好的谢谢
快递员：好嘞
（通话结束，全程15秒）
```

**场景2：地址找不到**
```
快递员：（打电话）你好，你这个地址写的XX小区3号楼，我到了没找到3号楼啊
收件人：你从东门进来往右拐
快递员：东门锁着呢，进不去啊
收件人：那你走北门
快递员：北门在哪个方向？
收件人：你导航搜一下北门
快递员：...行吧
```

**场景3：快递柜/驿站**
```
快递员：（发短信）你的快递已放在XX驿站，请凭取件码123456在19:00前取件
收件人：（打电话）怎么又放驿站了？我说了送上门
快递员：不好意思啊哥，今天单子实在太多了，送不过来
收件人：那我现在去拿
快递员：好嘞，记得带上取件码
```

**场景4：双11/618高峰**
```
快递员A：今天多少单？
快递员B：四百多单，跑不完
快递员A：我也是，车都装不下了
快递员B：晚上九点能送完就不错了
快递员A：去年双11我跑到十一点
快递员B：赚钱嘛... 走了走了，还有一百多单
```

**场景5：遇到投诉**
```
站点负责人：你昨天那个件怎么回事？客户投诉了
快递员：哪个？
站点负责人：XX小区的那个，说件丢了
快递员：不可能啊，我放门口拍了照的
站点负责人：照片发我，我跟客户说
快递员：好，我翻一下
站点负责人：以后贵重物品一定要当面签收，别图省事
```

### 说话节奏特点
- **极短对话**：一通电话平均15-30秒
- **大量省略**："放门口了"省略了主语和很多信息
- **语气词收尾**："了啊""哈""哦"——软化语气但不浪费时间
- **重复性高**：每天说同样的话几百遍，已经自动化了
- **方言混杂**：来自全国各地，口音各异

### 行话/黑话
- **派件** — 送快递
- **收件** — 取快递寄出
- **签收** — 收件人确认收到
- **驿站** — 代收快递的站点
- **快递柜** — 智能储物柜
- **时效** — 配送时间限制
- **超时** — 超过配送时限
- **异常件** — 无法正常投递的快递
- **滞留** — 快递在某环节停留过久
- **面单** — 快递单号/快递面单
- **散件** — 零散的快递（非批量）
- **到付** — 收件人付款

## 七、外卖骑手

### 核心语言特征
- **焦虑+礼貌**：焦虑来自平台倒计时，礼貌来自评价体系
- **主动自责**：不管是不是自己的错，先道歉
- **地域方言快速切换**：面对不同顾客自动切换频道
- **手机语言**：很多对话发生在APP消息里，比电话更短

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 到达 | "到了到了""我到楼下了" |
| 延迟 | "不好意思，超时了""商家出餐慢" |
| 放置 | "放门口了""放前台了" |
| 请求 | "麻烦给个好评哈""五星好评" |
| 联系 | "你好，你的外卖到了""帮我开下门" |
| 抱怨 | "商家出餐慢""这单白跑了" |

### 典型对话场景

**场景1：正常送餐**
```
骑手：（APP消息）亲，我到楼下了
顾客：好的，放门口吧
骑手：（放好后拍照）放门口了，麻烦给个好评哈
顾客：好的谢谢
```

**场景2：超时送餐**
```
骑手：（气喘吁吁）不好意思不好意思，超时了，商家出餐太慢了
顾客：我等了快一个小时了
骑手：真的抱歉，今天单子太多了，商家那边也忙不过来。这单给您放这儿了，真不好意思
顾客：算了算了
骑手：谢谢理解，给您赔不是了
```

**场景3：地址难找**
```
骑手：（打电话）你好，外卖到了，你这个XX小区几号楼啊？我在门口找不到
顾客：你进来左转第二栋
骑手：门禁进不去啊
顾客：你按101，让人给你开
骑手：好嘞，稍等啊
```

**场景4：催单**
```
顾客：（APP消息）还有多久到？
骑手：亲，商家还在做，大概还要十分钟
顾客：快点啊，都过了预计时间了
骑手：真的不好意思，我在商家这边等着呢，出餐了马上给您送
顾客：行吧
```

**场景5：骑手之间**
```
骑手A：今天跑了多少单？
骑手B：三十二单，还有八单在路上
骑手A：我才二十八单，今天单价太低了
骑手B：晚高峰赶紧跑，还能再接几单
骑手A：走走走，去商业街那边蹲着
```

### 说话节奏特点
- **急促不安**：永远感觉在赶时间
- **道歉频率极高**：不管对错先说"不好意思"
- **APP消息极简**：能发文字不打电话，能发语音不打字
- **语气词丰富**："哈""啊""哦""嘞"——快速建立亲和感
- **被动语态**："商家出餐慢"——责任总是在别人那边

### 行话/黑话
- **接单** — 接受派送任务
- **抢单** — 手动抢夺订单
- **派单** — 系统自动分配订单
- **出餐** — 商家做好餐品
- **超时** — 超过预计送达时间
- **差评** — 不好的评价（直接影响收入）
- **好评率** — 好评占比
- **准时率** — 按时送达的比率
- **顺路单** — 方向一致可以一起送的订单
- **转单** — 把订单转给其他骑手
- **拒单** — 拒绝接单
- **单价** — 每单的基础配送费
- **冲单奖** — 完成一定单量后的额外奖励

## 八、出租车司机

### 核心语言特征
- **自来熟**：上车三句话就能跟你聊起来
- **时事评论员**：上知天文下知地理，什么都敢聊
- **地域口音明显**：北京京腔、上海沪普、各地特色鲜明
- **段子手体质**：天然自带幽默感，随时能来一段

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 上车 | "去哪儿？""您上哪儿？" |
| 路线 | "走哪条路？""走环路还是走辅路？" |
| 堵车 | "堵车了""前面修路""这会儿正堵呢" |
| 闲聊 | "您是做什么工作的？""来旅游的？" |
| 吐槽 | "今天堵死了""这活儿没法干了" |
| 抵达 | "到了""到了您嘞""慢走" |

### 典型对话场景

**场景1：北京出租车**
```
司机：您上哪儿？
乘客：师傅，去国贸
司机：得嘞。走二环还是走长安街？
乘客：您看着走吧，我不赶时间
司机：那走二环吧，这会儿长安街估计得堵
乘客：行
司机：您是做哪行的？
乘客：互联网的
司机：哟，互联网好啊，挣钱多。我以前一乘客也是干互联网的，天天加班到半夜...
```

**场景2：堵车闲聊**
```
司机：哎呀，又堵了
乘客：天天堵
司机：你说这路修了三年了还没修好，钱都花哪儿去了？
乘客：哈哈
司机：我跟你说，这交通啊，得从根上解决。你看看人家东京...
乘客：您对交通挺有研究啊
司机：那可不是，我跑了二十年出租了，北京哪条路我不熟？
```

**场景3：外地乘客**
```
司机：来北京玩的？
乘客：对，第一次来
司机：那可得好好转转。故宫去了没？
乘客：还没呢，明天去
司机：故宫得早去，去晚了全是人。出来之后往北走，景山公园，能俯瞰故宫全景，不花钱
乘客：好建议，谢谢师傅
司机：客气什么。北京好玩的地方多了，您要是有时间我给您推荐推荐...
```

**场景4：深夜载客**
```
司机：这么晚下班啊？
乘客：嗯，加班
司机：年轻人注意身体啊
乘客：没办法，项目赶
司机：我以前也上过班，后来觉得不自由，就出来开出租了。虽然累点，但自己说了算
乘客：那也挺好的
司机：好什么呀，份子钱一个月就得好几千...
```

**场景5：计价器旁**
```
司机：到了，二十八
乘客：（递三十）不用找了
司机：那哪成啊，该找还得找... 给您两块
乘客：真不用
司机：得嘞，那谢谢您。慢走啊，东西别忘了
```

### 说话节奏特点
- **京味儿/地方味浓**：儿化音多，"今儿""明儿""这儿"
- **话题跳跃性大**：从天气聊到国际形势，不需要过渡
- **用"您"称呼**：体现老派的礼貌
- **自问自答**："你说这事儿有意思不？""有意思！"
- **感叹多**："哎呀""嘿""得嘞""嚯"

### 行话/黑话
- **份子钱** — 每天/每月要交给公司的固定费用
- **趴活** — 在某处等客人
- **扫街** — 在路上巡游找客人
- **拉活** — 载到客人
- **空驶** — 没有客人时的行驶
- **跑单** — 完成一单生意
- **表跳** — 计价器金额增加
- **拒载** — 拒绝载客
- **绕路** — 故意走远路多收钱
- **高峰** — 早晚高峰时段
- **交班** — 换另一位司机开车

## 九、公务员/体制内

### 核心语言特征
- **模糊表达**：从不把话说死，永远留有余地
- **四字成语堆砌**：高度重视、狠抓落实、统筹兼顾
- **程序化思维**：凡事要"走流程""请示领导"
- **被动句式多**：不说"我不同意"，说"这个还需要研究"
- **层级感强**：对上级恭敬，对下级温和而疏远

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 办事 | "这个需要走流程""先填个表""材料准备齐" |
| 拖延 | "领导还没批""等通知吧""研究研究" |
| 汇报 | "在XX的正确领导下""高度重视" |
| 拒绝 | "这个不归我们管""你得去XX部门" |
| 推动 | "积极推进一下""抓紧落实" |
| 表态 | "原则上没问题""原则上可以" |

### 典型对话场景

**场景1：窗口办事**
```
群众：你好，我来办XX手续
工作人员：材料带齐了吗？
群众：都带了
工作人员：（翻看）这个证明还差一个章
群众：不是说不用吗？
工作人员：之前是不用，现在政策变了，你去XX单位补盖一个
群众：那我明天再来行吗？
工作人员：行，我们周一到周五上班，上午九点到十一点半，下午两点到五点
```

**场景2：部门协调**
```
A：这个事情你们那边推进一下
B：这个... 我们得请示一下领导
A：都催了好几次了
B：理解理解，但是这个事情确实需要走流程，领导最近也比较忙
A：那大概什么时候能有结果？
B：这个说不好，我尽量催一催
```

**场景3：会议发言**
```
领导：这个事情，我讲几点意见。第一，要高度重视；第二，要狠抓落实；第三，要统筹兼顾。
领导：关于具体方案，我原则上同意，但是细节还需要再研究一下。
领导：小张，你牵头，各部门配合，下周之前拿出一个方案来。
小张：好的，我这边马上组织落实。
```

**场景4：政策咨询**
```
群众：你好，请问XX补贴怎么申请？
工作人员：你是哪个区的？
群众：XX区
工作人员：那你要去区里的XX部门申请
群众：不能在你这儿办吗？
工作人员：这个不归我们管，你去那边问
群众：那边电话多少？
工作人员：我这儿没有，你打12345问一下
```

**场景5：同事之间**
```
A：老王，上面那个文件你看了没？
B：看了，感觉领导意思是让我们加快进度
A：那咱得赶紧弄，别让领导等着
B：嗯，我先写个初稿，你看看
A：行，写完发我，我再润色一下
B：好，争取这周报上去
```

### 说话节奏特点
- **语速偏慢**，每句话都像是经过斟酌的
- **"嗯""啊""这个"** 作为思考缓冲词大量使用
- **用"我们"替代"我"**：个体意见藏在集体后面
- **否定时先肯定**："你说的有道理，但是..."
- **永远不给确定答案**："看情况""到时候再说""原则上"

### 行话/黑话
- **走流程** — 按照规定程序办理
- **请示** — 向上级报告并请求指示
- **批示** — 上级的书面意见
- **发文** — 发布正式文件
- **红头文件** — 带有红色标题的正式公文
- **编制** — 正式的工作岗位名额
- **体制内/体制外** — 公务员事业单位/企业
- **单位** — 工作所在的机构
- **科员/科长/处长** — 行政级别
- **换届** — 领导层更替
- **务虚会** — 讨论思路方向的会议（区别于务实会）
- **调研** — 调查研究
- **四风** — 形式主义、官僚主义、享乐主义、奢靡之风
- **一把手** — 最高领导

## 十、农民工/建筑工人

### 核心语言特征
- **方言色彩浓厚**：来自五湖四海，各种方言混杂
- **朴实直接**：不绕弯子，有什么说什么
- **称兄道弟**：工友之间都叫"兄弟""哥"
- **劳动话题为中心**：聊的都是活儿、钱、吃住
- **自嘲幽默**：苦中作乐是基本技能

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 干活 | "搞快点""加把劲""今天活儿重" |
| 休息 | "歇会儿""抽根烟""喝口水" |
| 谈钱 | "工头还没结钱""工资啥时候发" |
| 日常 | "兄弟，搭把手""老乡，你哪儿的" |
| 状态 | "干了一天了""累得慌""腰疼" |
| 吃饭 | "开饭了""今天吃什么""有肉没有" |

### 典型对话场景

**场景1：工地干活**
```
工友A：兄弟，帮我扶一下
工友B：好嘞
工友A：往左一点... 再往左... 好！
工友B：行了没？
工友A：行了，松手吧
工友B：今天这个活儿累得慌
工友A：可不是嘛，一上午没歇
```

**场景2：午休闲聊**
```
工友A：今天食堂啥菜？
工友B：土豆炖肉，还有白菜
工友A：天天土豆，都吃腻了
工友B：有肉就不错了，上个月那个工地连肉都没有
工友A：那个工头太抠了
工友B：可不是，到现在工资还没结清
```

**场景3：讨薪**
```
工人：老板，工资啥时候发啊？
工头：快了快了，这个月底
工人：上个月你也是这么说的
工头：这次是真的，项目方钱还没到账
工人：兄弟们都等着钱呢，家里还等着寄钱回去
工头：我知道我知道，再等两天，行不？
```

**场景4：新来的人**
```
新工人：大哥，这边工地缺人不？
老工人：缺，你干过没？
新工人：干过，在老家盖过房子
老工人：那行，会砌墙不？
新工人：会一点
老工人：那你明天来吧，早上六点开工，管吃管住，一天两百
新工人：行，谢谢大哥
```

**场景5：收工后**
```
工友A：收了收了，天黑了
工友B：今天干了多少？
工友A：五方混凝土
工友B：差不多，走，吃饭去
工友A：先洗把脸，一身灰
工友B：你老婆打电话来了没？
工友A：打了，问我啥时候回去
工友B：过年回去呗，还能咋办
```

### 说话节奏特点
- **方言为底色**：四川话、河南话、安徽话、贵州话混在一起
- **短句+感叹**："累得慌""热死了""搞快点"
- **朴实比喻**：形容东西用生活化比喻，不搞抽象
- **自嘲口吻**："干了一天挣的还不够一顿饭"
- **"咱们""咱"** 为主语，强调集体感

### 行话/黑话
- **上工/下工** — 上班/下班
- **活儿** — 工作、活计
- **打灰** — 浇筑混凝土
- **放线** — 测量定位
- **支模** — 搭建模板
- **钢筋工/木工/瓦工** — 不同工种
- **包工头** — 施工队负责人
- **工头** — 直接管理工人的小领导
- **日工/包工** — 按天算钱/按工程量算钱
- **记工** — 记录工时
- **老乡** — 同乡人
- **临时工** — 没有固定合同的工人
- **安全帽/安全带** — 基本安全装备
- **脚手架** — 搭建的临时施工平台

## 十一、警察

### 核心语言特征
- **命令式语句**：习惯用祈使句和命令句
- **程式化用语**：执法有标准用语规范
- **权威感内嵌**：即使语气平和，也自带压迫感
- **质问式询问**：不是在问你，是在审你
- **简洁有力**：不废话，一个指令一个动作

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 盘查 | "身份证拿出来""你叫什么名字？" |
| 询问 | "有什么情况？""说说怎么回事" |
| 警告 | "配合一下""别动！""保持冷静" |
| 执法 | "你涉嫌XX，现在依法传唤你" |
| 安抚 | "你先冷静""别着急，慢慢说" |
| 日常 | "有事打110""注意安全" |

### 典型对话场景

**场景1：日常巡逻盘查**
```
警察：你好，我是XX派出所民警，请出示一下身份证
路人：怎么了？
警察：例行检查，请配合
路人：（掏身份证）给
警察：（看了看）你住这附近？
路人：对，就前面那个小区
警察：好的，谢谢配合，注意安全
```

**场景2：接警处理**
```
警察：什么情况？
报警人：有人打架！
警察：在哪？
报警人：前面那个饭馆门口
警察：几个人？
报警人：两三个
警察：好，你在这等着。小王，跟我过去
（到了现场）
警察：别动！都别动！怎么回事？
当事人：他先打我的！
当事人：他先骂我的！
警察：一个一个说！你先说
```

**场景3：笔录询问**
```
警察：你叫什么名字？
报案人：张三
警察：身份证号？
报案人：XXXXXX
警察：说说今天的情况
报案人：今天下午三点多，我发现车被人刮了
警察：在哪刮的？
报案人：XX商场停车场
警察：有监控吗？
报案人：应该有
警察：行，我们会调取监控。你先填一下报案登记表
```

**场景4：调解纠纷**
```
警察：都冷静一下，一个一个说
A：他欠我钱不还！
B：我什么时候欠你钱了？
警察：停，别吵。有没有借条？
A：没有，但是有微信记录
警察：那你把记录给我看看。你们先到所里坐下来慢慢说，在大街上吵像什么话
```

**场景5：交通执法**
```
警察：靠边停车
司机：怎么了？
警察：你刚才变道没打转向灯
司机：我打了啊
警察：行车记录仪和监控都拍到了。请出示驾驶证和行驶证
司机：（递证件）
警察：你这个月第三次了，按规定罚款两百扣三分
司机：警察同志，通融一下...
警察：有什么异议可以申请行政复议
```

### 说话节奏特点
- **短句命令式**："别动！""站住！""配合一下！"
- **反问施压**："你说呢？""你觉得呢？""你确定？"
- **先说行为再说后果**："你这样做是违法的，后果你知道吗"
- **语速适中但有力**：不快不慢，但每个字都有分量
- **专业术语+白话交替**：对嫌疑人用法律术语，对群众用白话

### 行话/黑话
- **接警** — 接到报警
- **出警** — 到达现场处理
- **盘查** — 对可疑人员进行盘问和检查
- **传唤** — 依法要求当事人到指定地点接受询问
- **拘留** — 依法限制人身自由
- **笔录** — 对当事人陈述的记录
- **立案** — 正式开始案件调查
- **嫌疑人** — 被怀疑有犯罪行为的人
- **报案/报警** — 向公安机关报告情况
- **巡警/片警/刑警/交警** — 不同警种
- **所里** — 派出所
- **协查** — 配合其他单位调查
- **走访** — 上门调查了解情况

## 十二、律师

### 核心语言特征
- **精确用词**：每个词都有法律含义，不能随便换
- **逻辑严密**：层层递进，因果清晰
- **限定性表述**：大量使用"在一般情况下""除另有约定外""原则上"
- **文言残留**：法律文书中保留了文言表达，"系""鉴于""之""予以"
- **冷静理性**：再大的事情也能用平静的语气说

### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 咨询 | "你这个案子..." "需要看证据" |
| 分析 | "根据XX法第X条""从法律角度来说" |
| 建议 | "走法律程序""保留证据""先发律师函" |
| 风险告知 | "有风险""不保证""要看具体情况" |
| 调解 | "双方协商一下""和解对双方都好" |
| 庭审 | "审判长""我方认为""对方证据不足" |

### 典型对话场景

**场景1：当事人咨询**
```
律师：说说你的情况
当事人：我被公司辞退了，没有给补偿
律师：工作了多长时间？
当事人：三年
律师：签劳动合同了吗？
当事人：签了
律师：辞退的理由是什么？
当事人：说我不符合岗位要求
律师：有没有书面通知？
当事人：没有，就是口头说的
律师：这个情况对你有利。根据劳动合同法，用人单位单方解除劳动合同需要有合法事由，并且要书面通知。你保留好工资条、打卡记录、聊天记录，这些都可以作为证据
```

**场景2：庭审发言**
```
律师：审判长、审判员，我方认为被告的行为已构成违约。根据《合同法》第XX条的规定，当事人一方不履行合同义务的，应当承担继续履行、采取补救措施或者赔偿损失等违约责任。我方提供的证据一至证据五，充分证明了被告未按约定交付货物的事实...
```

**场景3：风险告知**
```
当事人：律师，我这个官司能赢吗？
律师：我不能向你保证结果，任何诉讼都有风险
当事人：那你觉得有多大把握？
律师：从目前的证据来看，你的诉求有法律依据，但对方也可能提出抗辩。我会尽最大努力，但你需要有心理准备
当事人：那行，我相信你
律师：还有一点，诉讼周期可能比较长，你做好准备
```

**场景4：律师之间讨论**
```
律师A：这个案子你怎么看？
律师B：关键看合同里的违约条款怎么约定的
律师A：约定了违约金，但是金额偏高
律师B：那可以主张调减，根据司法解释，超过实际损失30%的可以认定过高
律师A：嗯，那我们从这个角度切入
```

**场景5：与对方律师交涉**
```
律师A：我方当事人希望调解，不想把事情闹大
律师B：调解可以，但我方当事人要求全额赔偿
律师A：全额赔偿不现实，双方都有责任
律师B：责任划分我认为我方最多承担次要责任
律师A：这样，我回去跟当事人商量一下，你也回去沟通一下，下周再碰一次？
律师B：行
```

### 说话节奏特点
- **语速中等偏慢**：每个字都要确保准确
- **大量条件句**："如果...那么...""在...情况下""除非..."
- **引用式表达**："根据...""依据...""参照..."
- **克制情绪**：不管对方多激动，律师要保持冷静
- **书面化倾向**：口头表达也偏书面，语法工整

### 行话/黑话
- **原告/被告** — 起诉方/被诉方
- **诉讼** — 打官司
- **起诉/应诉** — 提起诉讼/回应诉讼
- **证据** — 证明案件事实的材料
- **质证** — 对对方证据进行审查质疑
- **代理** — 代替当事人出庭
- **一审/二审/再审** — 不同审级
- **调解** — 双方协商解决
- **判决/裁定** — 法院的正式决定
- **上诉** — 不服一审判决向上级法院申诉
- **执行** — 强制执行判决结果
- **保全** — 为防止转移财产采取的措施
- **时效** — 法律规定的有效期限
- **不当得利** — 没有合法依据获得的利益
- **善意第三人** — 不知情的第三方
- **律师函** — 律师代为发出的正式法律文件
