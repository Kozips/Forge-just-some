## 十三、说话方式塑造角色的要点总结

### 区分不同职业的核心维度

| 维度 | 说明 | 举例 |
|------|------|------|
| **语速** | 服务行业快，体制内慢 | 外卖员飞速 vs 公务员慢悠悠 |
| **句长** | 高压行业短句多，专业行业长句多 | 厨师三个字 vs 律师一句话三十字 |
| **情绪密度** | 服务行业情绪高，技术行业情绪低 | 销售热情洋溢 vs 程序员面无表情 |
| **术语密度** | 专业行业术语多，普通行业口语多 | 医生"BP""HR" vs 农民工"累得慌" |
| **主语习惯** | 体制内用"我们"，个人用"我" | 公务员"我们认为" vs 工人"我觉得" |
| **礼貌程度** | 服务行业极致礼貌，师傅型行业粗犷直接 | 销售"您眼光真好" vs 厨师"快点！" |
| **确定性** | 有风险的行业不确定性高，执行型行业确定性高 | 律师"看具体情况" vs 工人"明天六点来" |

### 快速辨别角色职业的方法
1. **听第一句话**：去哪儿？ / 哪里不舒服？ / 材料带齐了吗？——三句话三个职业
2. **听感叹词**：得嘞 / 哎呦 / 好嘞——不同地方不同行业
3. **听句尾语气词**：哈 / 啊 / 嘞 / 吧——暴露地域和行业
4. **听专业词汇**：bug / 86 / 份子钱 / 时效——行业标签
5. **听对时间的态度**：催命式 / 佛系式 / 拖延式——行业节奏

### 写作中的常见错误
- **错误1**：所有角色都用同一种文绉绉的说话方式
- **错误2**：只加口头禅不改变句式结构（口头禅是表面，句式才是骨架）
- **错误3**：不分场合，同一种职业在任何时候都用同一种语气
- **错误4**：忽略方言和地域口音的影响
- **错误5**：把"行业术语"写成"百科解释"——真正的行话是不需要解释的

## 2026.06 新增职业蒸馏

### 一、AI时代程序员

#### 核心语言特征
- **AI工具深度嵌入日常表达**：GPT、Copilot已替代"百度一下"成为默认动词
- **Prompt思维外溢**：用prompt逻辑描述一切需求，"我给它一个约束条件"
- **Token焦虑症**：随时关注API用量，token成了新的度量衡
- **人机协作口吻**：不再是"我写的"，而是"我跟Copilot一起写的"

#### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 解决问题 | "问一下GPT""让Copilot写" |
| 吐槽工具 | "这个prompt不行""模型幻觉了" |
| 工作流程 | "跑一下pipeline""CI挂了" |
| 成本管控 | "token不够了""API又被限流了" |
| 代码审查 | "这段是AI生成的，你review一下" |
| 技术选型 | "用大模型做""跑个embedding" |
| 甩锅 | "不是我写的，是Copilot生成的" |
| 赶工 | "先让AI出个初稿我再改" |

#### 典型对话场景
```
A: 这个需求怎么搞？
B: 我先问一下GPT，让它给个方案
A: 上次那个prompt你还有吗？
B: 有，但是那个模型幻觉太严重了，回答的都不对
A: 换个模型试试？Claude可能好一点
B: 行，token够不够啊？
A: 够，刚充了值。先跑一下pipeline看看
B: 好，我这边Copilot写的单元测试先放上去
```

### 二、直播带货主播

#### 核心语言特征
- **情绪永远在高点**：语速快、音量高、感叹号是标配
- **制造稀缺与紧迫**："最后XX单"是核心话术引擎
- **亲缘化称呼**："家人们""宝子们"拉近距离，降低防备
- **数字轰炸**：价格、销量、优惠力度全部数字化

#### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 开场 | "家人们，今天给大家炸个福利" |
| 上架 | "三二一，上链接！" |
| 逼单 | "最后五十单！拍完就没了！" |
| 优惠 | "拍一发三！买一送一！" |
| 售后保障 | "运费险""七天无理由""不喜欢直接退" |
| 互动 | "想要的扣1""觉得值不值？" |
| 催促 | "手慢无！""犹豫就没了！" |
| 信任建设 | "我自己天天在用""回购八百次了" |

#### 典型对话场景
```
主播：家人们！今天这款面霜，专柜三百九，我直播间只要九十九！
助理：姐，库存只有一百单了
主播：听到没有！只有一百单！三二一，上链接！
（弹幕滚动）
主播：哇，已经抢了六十单了！再补五十单，最后五十单！拍一发三！还有运费险，七天无理由退！不喜欢直接退，一分钱不用出！
助理：姐，又没了
主播：没了没了，下次再来！家人们点个关注，明天还有更大的福利！
```

### 三、短视频博主

#### 核心语言特征
- **固定句式模板化**：开场白和结尾有标准化公式
- **强共情开头**：先制造情绪共鸣再输出内容
- **号召式结尾**：每条视频都以行动号召收尾
- **口语碎片化**：不讲长句，单句不超过十个字

#### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 共情开场 | "家人们谁懂啊""大数据听我说" |
| 推荐安利 | "建议全国推广""不允许有人没用过" |
| 收藏号召 | "码住""先收藏再看" |
| 情绪推动 | "冲冲冲""闭眼入" |
| 制造好奇 | "看到最后有惊喜""千万别划走" |
| 社交货币 | "发给你最好的朋友""艾特你那个冤种闺蜜" |

#### 典型对话场景
```
博主：家人们谁懂啊，这个拖把真的绝了
博主：我已经用了三个月了，今天必须安利给你们
博主：你看这个效果，一拖就干净，不弯腰不费力
博主：建议全国推广！二十块钱的东西做到这个程度
博主：链接放购物车了，码住！冲冲冲！
博主：大数据听我说，把这条推给需要的人，爱你们，么么哒
```

### 四、新能源车销售

#### 核心语言特征
- **技术参数前置**：开口就是续航、智驾、激光雷达，用数据说话
- **传统车vs新能源对比**：每个卖点都锚定油车痛点
- **科技感包装**：把汽车说成"智能终端""移动第三空间"
- **降低焦虑话术**：续航焦虑、充电焦虑每个都有对应话术

#### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 续航 | "续航七百公里""实打实的，不虚标" |
| 智驾 | "城市NOA全场景覆盖""智驾天花板" |
| 硬件 | "激光雷达""800V平台""碳化硅" |
| 补能 | "家充桩一度电几毛钱""换电三分钟" |
| 性能 | "百公里加速三点几秒""比油车快一倍" |
| 升级 | "OTA升级常用常新""买了等于没买完" |
| 省钱 | "一公里几分钱""保养几乎不用花钱" |
| 逼单 | "这个月有置换补贴""限时优惠最后三天" |

#### 典型对话场景
```
销售：哥，您之前开油车是吧？一年油费多少？
客户：两万多吧
销售：电车的话，一年电费顶多两千。光油费一年省一万八，五年省九万
客户：续航够不够用啊？
销售：这款七百公里续航，日常通勤一周一充。而且家里装个家充桩，晚上充白天用，跟手机一样方便
客户：电池衰减怎么办？
销售：现在技术很成熟了，而且支持OTA升级，系统不断迭代，常用常新。哥您要不要试驾一下？百公里加速三点八秒，您感受一下就知道了
```

### 五、外卖骑手2026（更新）

#### 核心语言特征
- **标准化话术模板**：平台规范了服务用语，骑手照本宣科
- **实时状态播报**：每一步都通过APP主动推送，减少电话沟通
- **好评请求常态化**：好评率直接影响派单权重，请求好评成为肌肉记忆
- **情绪管理升级**：2026年平台考核更严，骑手用语更标准化、更克制

#### 口头禅清单
| 场景 | 口头禅 |
|------|--------|
| 到店 | "已到店取餐" |
| 配送中 | "正在火速配送" |
| 预计送达 | "预计XX分钟送达" |
| 请求好评 | "麻烦给个好评" |
| 异常通知 | "商家出餐稍有延迟，给您说声抱歉" |
| 送达确认 | "已送达，祝您用餐愉快" |
| 联系顾客 | "您好，您的餐品已放在指定位置" |
| 催促应对 | "非常理解您着急，正在尽快送达" |

#### 典型对话场景
```
（APP消息）
骑手：亲，已到店取餐，正在火速配送，预计8分钟送达
顾客：好的快点啊我饿了
骑手：收到！马上就到

（送达后）
骑手：您好，餐品已放在门口，祝您用餐愉快
顾客：好的
骑手：麻烦给个好评，感谢支持！
```
