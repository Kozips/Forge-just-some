## 六、社交平台对话语气

### 6.1 微信聊天语气词大全

#### 核心语气词对比

| 语气词 | 含义/情绪温度 | 备注 |
|--------|-------------|------|
| **嗯** | 知道了（偏冷淡） | 可能表示不想继续聊 |
| **嗯嗯** | 知道了（偏温暖） | 比"嗯"亲切很多 |
| **嗯嗯嗯** | 非常认同/积极 | 过度热情 |
| **哦** | 知道了（冷暴力级别） | 被称为"聊天终结者" |
| **哦哦** | 恍然大悟 | 比"哦"好一些 |
| **好的** | 礼貌接受 | 正式但友好 |
| **好** | 简单直接 | 稍显冷淡 |
| **好嘞** | 爽快答应 | 亲切、积极 |
| **好滴** | 可爱版答应 | 亲昵感 |
| **行吧** | 勉强同意 | 有点不情愿 |
| **哈哈** | 礼貌性回应 | 可能不是真笑 |
| **哈哈哈** | 觉得还行 | 3个h是起步标准 |
| **哈哈哈哈哈哈** | 真的很好笑 | 越多越真诚 |
| **嘻嘻** | 偷笑/俏皮 | 略带撒娇 |
| **嘿嘿** | 坏笑/不好意思 | 有点小得意 |
| **emmm** | 犹豫/思考中 | 不知道怎么回答 |
| **啊啊啊** | 兴奋/崩溃/惊讶 | 视语境而定 |
| **？** | 疑惑/质疑/不满 | 单独一个问号很有力 |
| **……** | 无语/尴尬/省略 | 省略号代表千言万语 |
| **呵呵** | 嘲讽/不屑/尴尬 | 最具争议的回复词 |

**关键规则："哈哈哈"的数量社交学"**

```
"哈" — 敷衍
"哈哈" — 礼貌但不真诚
"哈哈哈" — 基本标准，表示还算好笑
"哈哈哈哈" — 确实好笑
"哈哈哈哈哈哈" — 真的笑死了
"哈哈哈哈哈哈哈哈哈哈哈" — 已经无法控制
```

**对比示例：**

```
场景：朋友讲了一个冷笑话

敷衍回应：哈
礼貌回应：哈哈哈
真诚回应：哈哈哈哈哈哈救命笑死我了
```

#### 语气词的社交潜规则

```
收到"哦"的感受：对方不想理你了
收到"呵呵"的感受：对方在嘲讽你
收到"微笑😊"的感受：长辈发的——正常；同龄人发的——完了，对方生气了
收到"👍"的感受：敷衍，对话可以结束了
收到单个"。"的感受：对方非常认真/不高兴了（年轻人一般不用句号）
收到单个"？"的感受：你在逗我？
```

### 6.2 QQ聊天特色用语

QQ用户群比微信更年轻，形成了独特的表达风格：

| 用语 | 含义 |
|------|------|
| **扩列** | 扩充好友列表，交新朋友 |
| **养火** | 和好友保持连续聊天，维持聊天火花标识 |
| **巨轮** | QQ聊天中最频繁的互动对象 |
| **处Q友** | 在QQ上交朋友 |
| **在吗** | 经典开场白（被很多人嫌弃但仍在使用） |

**QQ vs 微信差异：**
- QQ更倾向于使用emoji和个性化表情包
- QQ群聊更加活跃、无序
- 微信对话相对更正式一些
- QQ用户更年轻，网络用语使用更密集

### 6.3 小红书说话风格

小红书形成了非常独特的社区语言风格，与其他平台差异明显：

#### 称呼词
- **姐妹** — 最常见的称呼，营造姐妹淘氛围
- **集美们** — "姐妹们"的谐音变体，更俏皮
- **宝子** — 亲昵称呼
- **家人们** — 拉近距离

#### 特色词汇
- **种草** — 推荐某个产品/事物
- **拔草** — 取消对某产品/事物的兴趣
- **安利** — 强烈推荐
- **码住** — 先收藏
- **天花板** — 某领域最好的
- **平替** — 平价替代品

#### 语气特点
- 大量使用emoji和颜文字：🎀✨💕🥰
- 感叹号极多！！！
- 句尾常加"～"软化语气
- 语气夸张热情

**小红书典型对话风格：**

```
姐妹们！！这个面霜真的绝绝子！！
谁用谁知道～
我已经空瓶3个了！
便宜大碗！干皮亲妈！
都给我冲！！！✨✨
```

```
集美们救命！！

## 七、直播带货话术

### 7.1 主播说话节奏与话术体系

直播带货已经形成了一套高度成熟的语言体系，以下是各环节的典型话术：

#### 开场话术（吸引注意力）

```
"家人们，今天给大家带来的是全网最低价！"
"所有刚进来的宝宝们，先点个关注，等会儿有大福利！"
"新来的家人们先别走，今天的东西真的太炸了！"
"欢迎欢迎，刚进来的宝宝扣个1让我看到你们！"
```

#### 留人话术（防止划走）

```
"千万别走开，等一下有秒杀款！"
"还没有抽到奖的宝宝再等3分钟！"
"接下来这款是今天的王炸，错过就没了！"
"想不想看9.9的福利款？想的扣想！"
```

#### 产品介绍话术

**痛点切入法：**
```
"夏天到了，出汗多、脱妆快怎么办？姐妹们有没有这个烦恼？"
"熬夜党的姐妹看过来！黑眼圈重不重？眼袋大不大？"
```

**对比法：**
```
"外面专柜卖399，我们直播间今天只要99！你们自己算算差多少？"
"同款在XX平台卖XXX，来我们直播间，直接省了一半！"
```

**场景化描述：**
```
"你去约会穿这套，男朋友眼睛都直了！"
"上班穿这个，领导都会多看你两眼！"
"带娃穿这件，方便又好看，喂奶都不用脱！"
```

#### 促单话术（促成下单）

```
"最后10单，拍完就没了！"
"三、二、一，上链接！"
"库存就这么多，犹豫就没了！"
"手慢无啊家人们！"
"已经拍了的宝宝扣'已拍'！"
"抢到就是赚到！"
```

#### 逼单话术（制造紧迫感）

```
"这个价格只有今天，过了今天恢复原价！"
"倒计时5秒，5、4、3、2、1——下架！"
"最后30秒！没拍到的别来找我！"
"3分钟没付款的自动取消啊！"
```

#### 售后话术（消除顾虑）

```
"不好用直接退，运费我出！"
"有任何问题找我，终身售后！"
"7天无理由退换，家人们放心拍！"
"品质我给大家把关，出了问题找我！"
```

### 7.2 主播说话风格特点

1. **语速极快**：信息密度高，不给观众思考时间
2. **音量偏大**：通过声音制造紧迫感和兴奋感
3. **重复核心信息**：价格、优惠力度会反复说很多遍
4. **互动频繁**："对不对？""是不是？""有没有？"
5. **数字敏感**：精确到几块几毛，增加可信度
6. **称呼亲切**：始终用"家人们""宝宝们""姐妹们"
7. **情绪饱满**：全程高能，不会出现低潮

### 7.3 2024—2025年带货话术新趋势

| 趋势 | 说明 |
|------|------|
| **真诚化** | 消费者对过度营销疲劳，更倾向朴实真实的表达 |
| **知识型带货** | 主播需要讲解成分、面料等专业知识 |
| **合规化** | 平台管控虚假宣传和极限词，话术更规范 |
| **情绪价值** | 不只卖货，还要提供陪伴感和情感连接 |
| **反向带货** | "不建议所有人买""不适合XX人群"反而更有效 |

## 八、饭圈用语

### 8.1 核心术语表

| 术语 | 含义 |
|------|------|
| **控评** | 有组织地控制评论区，刷正面评论压制负面评价 |
| **打榜** | 通过投票、播放、购买等方式提升偶像榜单排名 |
| **反黑** | 举报或反击针对偶像的负面言论 |
| **做数据** | 为偶像刷流量、播放量、转发量 |
| **氪金** | 为偶像花钱（买专辑、代言产品等） |
| **站姐/站哥** | 跟拍偶像的粉丝，运营粉丝应援站 |
| **路人缘** | 非粉丝群体对艺人的好感度 |
| **拉踩** | 贬低别人来抬高自家偶像 |
| **毒唯** | 极端唯粉，只喜欢团中一个人且攻击其他成员 |
| **CP粉** | 喜欢某对组合/配对的粉丝 |
| **大粉** | 有影响力的粉丝领袖 |
| **糊** | 不红、没有知名度 |
| **塌房** | 偶像曝出丑闻/人设崩塌 |
| **考古** | 搜索偶像过去的视频/资料 |
| **营业** | 偶像发布内容、与粉丝互动 |
| **划水** | 偶像在舞台上表现不认真/不尽力 |
| **脱粉** | 不再是某人的粉丝 |
| **回踩** | 脱粉后反过来黑之前的偶像 |

### 8.2 饭圈典型对话

```
A：姐妹们快去打榜！XX快被超过了！
B：来了来了，已经在投票了
C：数据组说了，每个人至少投500票
A：氪金也冲一波，新专辑买了吗
B：买了10张，给哥哥冲销量
```

```
A：XX又上热搜了
B：看到黑词条了，姐妹们去反黑！
A：已经举报了，评论区也在控评
C：别跟黑子对线，直接举报就好
```

```
A：天呐，XX塌房了
B：什么？？？不会吧
C：[截图][截图] 铁证如山
B：我房子塌了……
A：脱粉了，回踩一把
```

### 8.3 饭圈用语的日常化

很多饭圈用语已经溢出到日常生活中：

- **"我塌房了"** — 不仅指偶像丑闻，也用于"发现自己喜欢的东西翻车了"
- **"他好会营业"** — 夸某人很会社交/展示自己
- **"路人缘很好"** — 普通人缘好
- **"这件事需要考古一下"** — 需要翻翻旧账/查查历史
- **"XX已糊"** — 某人/某事已经不火了

## 九、不同年龄层的说话差异

### 9.1 代际语言对比表

| 维度 | 70后 | 80后 | 90后 | 00后 |
|------|------|------|------|------|
| **网络用语** | 很少使用 | "给力""亲""童鞋""886" | "2333""gg""蓝瘦香菇""666" | "绝绝子""YYDS""我真的会谢""抽象" |
| **表情使用** | 基础emoji或不用 | QQ默认表情 | 颜文字+emoji混搭 | 特殊emoji含义+密集表情包 |
| **笑的表达** | "哈哈" | "哈哈""2333" | "hhhh""2333" | "哈哈哈哈哈哈（极度长串）""绷""乐" |
| **语气词** | "嗯""好的" | "哦""呵呵" | "emmm""hhhh" | "救命""属于是""就是说" |
| **表态方式** | 直接表达 | 委婉但明确 | 委婉+调侃 | 模糊+解构+反讽 |
| **缩写使用** | 几乎不用 | 偶尔使用 | 中度使用 | 高频使用 |
| **句号使用** | 正常使用 | 正常使用 | 减少使用 | 几乎不用（句号=不高兴） |

### 9.2 各代际详细特征

#### 70后

```
"小王，这个方案我看了一下，整体还行，有几个地方需要改改。"
"好的，我知道了。"
"明天上午10点开会，你准备一下材料。"
```

特征：
- 说话直接，不用修饰
- 正式称呼，如"小王""张总"
- 标点符号使用规范
- 很少使用网络用语
- 微信表情偏好：🌹👍😊🙏

#### 80后

```
"亲，这个淘宝链接能发我一下不？"
"给力！"
"这个东东不错啊"
"886，下次聊"
"童鞋们，周末约起来"
```

特征：
- 经历了互联网早期文化
- 会用"亲""童鞋""东东"等早期网络词
- "呵呵"对他们来说可能真的是在笑
- QQ时代的回忆（火星文、非主流等）

#### 90后

```
"hhhhh这也太好笑了"
"2333333"
"gg了，这把又白给"
"蓝瘦，香菇"
"6666，厉害了"
"emmm这个怎么说呢"
"我选择死亡.jpg"
```

特征：
- 混合使用中英文和数字
- "23333"是笑的标准写法（源自猫扑论坛表情代码）
- 从论坛/QQ/贴吧过渡到微博/微信
- 表情包使用频率适中
- "呵呵"开始有了负面含义

#### 00后

```
"属于是狠狠共情了家人们"
"我真的会谢"
"救命！！！这也太绝了吧😭😭😭"
"i人表示已经社死了💀"
"绷"
"就是说 有没有一种可能"
"就……怎么说呢……懂的都懂"
```

特征：
- 大量使用网络流行语
- 语气词丰富且多变
- "哈哈哈"的长度是社交诚意的体现
- 不用句号，句号=生气/严肃
- emoji有特殊含义（😊≠微笑）
- 极度依赖表情包
- 抽象文化和解构表达的主力军
- 喜欢用省略号和空格制造"欲言又止"的感觉

### 9.3 经典代际沟通错位案例

**场景：收到好消息**

```
70后：太好了！恭喜恭喜！
80后：给力！恭喜啊！
90后：66666！恭喜恭喜hhhh
00后：啊啊啊啊啊！！！真的吗！！属于是天大的好消息😭😭😭 yyds！！
```

**场景：表达不满**

```
70后：我觉得这样不太合适。
80后：emmm这个……不太行吧？
90后：hhhh这个就……
00后：😅（一个微笑emoji就是最大的不满）
```

**场景：不想继续聊天**

```
70后：好的，我知道了。（正常结束）
80后：嗯嗯，好的。（正常结束）
90后：哈哈哈好吧hh（礼貌结束）
00后：嗯（一个字表示不想聊了）
```

## 十、职场 vs 日常说话方式差异

### 10.1 核心差异对照表

| 场景 | 日常私下说法 | 职场正式说法 |
|------|-------------|-------------|
| 太离谱了 | "什么鬼？？？" | "这个方案可能需要再优化一下。" |
| 不想干了 | "不想上了毁灭吧" | "我想考虑一下其他职业发展方向。" |
| 你说的不对 | "不是吧，你认真的？" | "我有一个不同的角度，我们可以探讨一下。" |
| 做不到 | "不行不行，做不了" | "目前资源/时间上可能存在一些挑战。" |
| 你快点啊 | "搞快点！" | "麻烦您尽快确认，方便推进后续工作。" |
| 这是你的锅 | "这怪谁啊？" | "这个问题我们可能需要复盘一下各环节的责任归属。" |
| 跟我没关系 | "关我啥事" | "这个节点是在XX环节，建议同步确认。" |
| 我不知道 | "啊？不知道啊" | "这个我需要确认一下，稍后给您回复。" |

### 10.2 职场黑话大全

互联网/科技公司尤其流行一套"行业黑话"：

| 黑话 | 实际含义 |
|------|----------|
| **对齐** | 统一大家的意见 |
| **拉通** | 协调各方信息 |
| **赋能** | 给你资源/支持 |
| **抓手** | 切入点/方法 |
| **底层逻辑** | 根本原因/核心逻辑 |
| **颗粒度** | 细致程度/详细程度 |
| **闭环** | 从头到尾完成，形成闭环 |
| **沉淀** | 积累/总结经验 |
| **打法** | 策略/方法 |
| **链路** | 完整的流程/路径 |
| **中台** | 提供共性支持的平台/团队 |
| **私域** | 自有用户池（微信群、小程序等） |
| **抓手** | 具体可行的切入点 |
| **复盘** | 事后回顾总结 |
| **同步** | 让大家知道/统一信息 |
| **owner** | 负责人 |

### 10.3 职场对话示例

```
日常版：
小王：这个项目太离谱了，根本做不完，需求一直在变。
小李：是啊，客户也太烦了吧。
小王：不想干了，毁灭吧。

职场版：
小王：关于这个项目，目前的进度和预期存在一定偏差，需求变更频率较高，对交付时间有一定影响。
小李：是的，建议我们和客户方再次对齐需求，确认优先级。
小王：认同，我可以拉一个需求评审会，把各方同步一下。
```

### 10.4 打工人自嘲文化

年轻人把日常工作的疲惫和不满，用自嘲的方式在社交媒体上表达：

| 表达 | 含义 |
|------|------|
| **打工人打工魂** | 打工人的精神，苦中作乐 |
| **上班如上坟** | 形容上班的心情 |
| **社畜** | 比打工人更惨的说法，像畜牲一样被驱使 |
| **搬砖** | 干活/工作 |
| **摸鱼** | 偷懒 |
| **996** | 早9晚9每周6天的工作制度 |
| **007** | 每天0点到0点，每周7天 |
| **被优化** | 被裁员 |
| **毕业** | 被裁员（公司说你"毕业了"） |
| **拥抱变化** | 公司裁员/调整 |

**自嘲对话示例：**

```

## 十二、对话风格的核心规律总结

### 12.1 年轻人网络对话的六大特征

1. **情绪放大化**：任何情绪都要放大表达，"绝了""救命""哭了""死了"是常态
2. **表达模糊化**：刻意不把话说清楚，"就是那个""懂的都懂""属于是"
3. **解构严肃化**：用玩梗消解一切严肃话题，越是重要的事越要用轻松的语气说
4. **社交货币化**：流行语是融入群体的通行证，不懂梗=社交失败
5. **表情替代化**：一个emoji/表情包可以替代一段文字
6. **语气软化**：通过"哈哈哈""呢""吧""啦"等词降低攻击性和正式感

### 12.2 语气软化技巧（高频使用）

| 技巧 | 示例 |
|------|------|
| **加"哈哈哈"** | "你这个方案不太行哈哈哈"（用笑声缓冲批评） |
| **加"吧"** | "你考虑一下吧"（比"你考虑"柔和） |
| **加"呢"** | "你最近怎么样呢"（比"你最近怎么样"亲切） |

## 口头禅与填充词

---
name: dialogue-filler-words
description: 口头禅——"那个/内个/这个"、"卧槽"等中文填充词和万能语气词
metadata:
  type: feedback
---

很多中国人说话嘴边挂着"那个"、"内个"、"这个"——不是在指什么东西，就是嘴在脑子里还没想好下一句的时候自己先找个音出来垫着。

**Why:** 这是真实中文口语里极其普遍的现象，不写进去就假了。

**How to apply:**
- 写对话的时候，角色卡壳、犹豫、组织语言的时候就该冒出这些词
- "那个...我想说的是..." / "这个这个，你听我说啊" / "内个啥，你别急"
- 北方人更多说"内个"，南方人更多说"这个"，但不绝对
- 不只是犹豫的时候用，有些人就是口头禅，每句话前面都带一个
- 有的人说"嗯那个"连着说，有的人说"就...那个..."

## "卧槽"——万能语气词，一个词顶一百个词

"卧槽"这玩意儿是中文里最全能的词之一，同一个字，场景不同意思完全不同：

- **震惊/不敢相信：** "卧槽？？他真这么说的？？"（眼睛瞪大那种）
- **赞叹/牛逼：** "卧槽，这也太牛了吧"（竖大拇指那种）
- **愤怒/骂人：** "卧槽你他妈的..."（咬牙切齿那种）
- **疼痛：** "卧槽！！烫死我了！"（下意识蹦出来的）
- **无奈/认命：** "卧槽...行吧..."（叹气那种）
- **恐惧/吓一跳：** "卧槽！！什么东西！！"（往后跳那种）
- **感叹/无语：** "卧槽..."（就一个词，啥也不想说了）
- **打招呼/口头禅：** "卧槽兄弟你咋来了"（跟"哎呦"差不多）
- **自言自语：** "卧槽我钥匙呢..."（跟自己嘟囔那种）

关键不是这个词本身，是**语气和上下文**。写出来的时候要注意：
- 有时候就是"卧槽"两个字，后面啥也没有，比长句更有劲
- 有些人把它当标点符号用，一句话一个卧槽
- 受过教育的人可能说"我靠"、"我去"、"我擦"、"我了个去"来替代
- 女生可能说"我去"、"天哪"比较多，但也有女生直接说卧槽
- 不同人嘴里出来的卧槽，音调、力度、频率全不一样

## 结巴与卡顿——不是每个人说话都溜的

真实对话里经常有人说话不利索，尤其是突然被问到、紧张、心虚、着急的时候：

- **突然被问到：** "你、你问我？" / "啊？我、我不知道啊" / "那个...等、等一下"
- **紧张：** "我、我我跟你说啊..." / "就是那个...就是...你知道的" / "不不不不是你想的那样"
- **心虚：** "我...我没干嘛啊" / "那个...这个...你听我解释" / "应、应该没吧？"
- **着急：** "快快快快走！" / "你你你别动！" / "等等等等！" / "先、先别急！"
- **害怕：** "别、别过来..." / "我求、求你了" / "你...你要干嘛"
- **被吓到：** "卧槽！你你你什么时候在这的？" / "啊！！你、你吓死我了"
- **说话本身就不利索的人：** 有些人天生说话就慢半拍，不是每句都结巴但经常卡壳，"我跟你说啊，就是那个...嗯...对，就是那么回事"

写的时候注意：结巴不是每个字都重复，是关键的几个字卡一下就够了。写太多读着也累。

## 顿住/沉默——有时候不说话比说话更有劲

- 说到一半突然停了："我那天看到他...算了，不说了"
- 被问住沉默几秒："... ... 你怎么突然问这个"
- 该说的没说出来："我想跟你说个事...嗯...没事"
- 用动作代替语言：他张了张嘴，最后只摇了摇头。

### 2024-2025最新口头禅——过时的和还在用的

网络口头禅更新太快了，半年不用就过时。以下按"还在活"和"已经死"分：

**还活着的：**
- **"绝了"**——万能感叹词，好的坏的都能用。"这饭绝了"（好吃）"这操作绝了"（菜到极致）
- **"绷不住了"**——本来是"蚌埠住了"谐音，现在直接说"绷不住"。笑到绷不住、气到绷不住、感动到绷不住
- **"显眼包"**——形容那种特别爱出风头的人。"你今天又当显眼包了"
- **"嘴替"**——替你说出心里话的人。"你就是我的嘴替！"
- **"i人/e人"**——MBTI火了之后人人都在用。"我是i人别叫我出去""这局全是e人在那嗨"
- **"抽象"**——不是字面意思，是"离谱""离奇""什么玩意儿"的代称。"这个操作太抽象了"

**已经死了但你写角色可以用（说明角色网感滞后）：**
- "YYDS"——现在用会被嘲，但2021年的时候人人都说
- "绝绝子"——已经土到地心了
- "集美们"——被小红书玩烂了，现在说出来有年代感
- "芭比Q了"——2022年的词，现在没人说了
- "老铁666"——更早的了，但中年人还在用

**写角色的时候**：如果角色是冲浪达人，用最新的；如果角色是网速慢的人，用过时的——这本身就是人物标签。

### 不同人群的口头禅差异

**大学生：**
- "不是......"（口头禅前置，跟内容无关）"不是，你听我说啊"
- "真的假的""假的吧"
- "哈？""啊？"（表示没听清/震惊）
- "搞笑呢""你在开玩笑吧"
- "救命""我死了""要命"——夸张表达，不是真要命

**打工人：**
- "好的""收到""了解"——工作三连
- "怎么说呢......"——为难的时候
- "行吧""都行""可以"——已经被社会磨平了
- "算了""随便""无所谓"——同上
- "麻了"——比"无语"更高级的表达

**中年父母：**
- "不是我说你啊......"——接下来就要说你了
- "我像你这么大的时候......"——忆苦思甜经典开头
- "你看看人家XXX"——攀比三件套
- "我这是为你好"——所有管教的万能理由
- "现在年轻人啊......"——后面跟的都不是好话

**小学生/初中生（2024+）：**
- "6"——一个数字搞定一切回应
- "啊？""嗯？"——反问语气表震惊
- "闹呢"——你闹呢=你在开玩笑吧
- "逆天"——太离谱了
- "菜"——一个字骂人，又简洁又扎心
- "笑死"——有时候根本不笑，就是口头禅

### 气味词和感叹词的地域差异

**北方人：**
- "我去！""我天！""我靠！"——震惊三连
- "嘿""嚯""呦"——开口先来一个感叹词
- "咋了""咋回事""咋整"——北方万能疑问词
- "得嘞""成""妥了"——同意的表达

**南方人（偏江浙沪）：**
- "啊呀""哎哟""天呐"——语气词更软
- "好好好""行行行"——叠词表示急了
- "烦死了""讨厌""神经"——骂人也带语气词

**广东人/港澳：**
- "唔系啊嘛""冇搞错啊"——粤语特有的吐槽
- "好嘢""正嘢"——夸赞
- 普通话夹粤语字："这个好正啊"

**四川/重庆人：**
- "要得""安逸""巴适"——表示好/舒服
- "龟儿子""瓜娃子"——骂人但不一定是真骂
- "啥子嘛""爪子嘛"——啥/干什么

### 最容易写错的口头禅

很多人写对话喜欢加"额"，但真实口语里：
- 北方人更多说"呃"（短促）而不是"额"（太正式了）
- "emmm"是打字用的，说话的时候是"嗯......"拖长音
- "那个"比"额"真实一百倍
- 有些人卡壳不是说"额"，是直接没声音，或者"就......""然后......"
- 打字的时候"......"比"......"真实，六个点比三个点多，真实对话里犹豫更长
