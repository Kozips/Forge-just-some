# 规则怪谈类 · 人物语言语气动作参考

> 适用于规则怪谈题材写作时的角色参考。

---

## 谨慎冷静 / 重生布局者

### 苏念（《规则怪谈》·番茄小说）
- **性格**：谨慎周密、外冷内善、重生布局者
- **语言**：说话极简，交代事情像列清单。对亲近的人偶尔多说一句提醒，但点到为止。不解释原因，只说结果："朋友是气象站的，说会持续高温"。内心独白远多于对话，独白时才暴露焦虑和柔软
- **语气**：语气平稳，像在规划日常采购。对普通人说话时语气温和但克制。独处时才允许自己流泪——情绪只对自己释放
- **动作**：查漏补缺清单——重生者的强迫症，怕遗漏；化妆伪装出门——谨慎到怕被人认出来；逛完一圈立刻回家窝在沙发——疲惫但满足；瞪大双眼确认手机日期——重生者的经典动作
- **标志性场景**：
  - "朋友是气象站的，说会持续高温，你们多买点喝的，好好保存起来。"
  - 一句轻声的提醒，不留痕迹，不多说一句。
- **微表情/微动作**：苏念泪流满面——确认重生后不是狂喜，是劫后余生的释放；拿出打印的清单查漏补缺——计划型人格的标志；虽然不想种植，但还是备了种子——理性里藏着感性

## 沉稳冷静 / 智囊型

### 宁秋水（《诡舍》(好看爱看)·番茄小说）
- **性格**：冷静分析、沉默果断、无声权威
- **语言**：话极少，一开口就是关键判断。不解释过程，只给结论："不是兽血。"语气平静，像在陈述事实而非表达意见。用最少的字传达最多的信息
- **语气**：声音不高，但清晰有力。不带任何情绪波动。一句"不要开灯"就把所有人镇住
- **动作**：看了一眼尸体就给出判断——观察力强；示意刘承峰帮忙收拾，自己去扶王雨凝——指挥型人格；深吸一口气再说出判断——冷静中的克制；摇头说"暂时还不能说"——信息管控意识
- **标志性场景**：
  - "不是兽血。"（一句话击溃所有自我安慰）
  - "不要开灯。"（三个字的命令）
  - "如果灯开着，那就要离开。"
- **微表情/微动作**：看了一眼就下结论——快速分析能力；摇头——不给虚假安慰；深吸一口气——压制自己的恐惧再开口

## 毒舌直率 / 嘴炮担当

### 刘承峰（《诡舍》·番茄小说）
- **性格**：嘴硬心软、暴躁直率、嘴炮输出
- **语言**：说话像机关枪突突突，一抓到机会就疯狂输出。喜欢反问和嘲讽："你可真牛啊？""一个瘫痪在床的人就把你吓成这样？"。骂人直接，不绕弯子。遇到诡异事件第一反应是骂脏话
- **语气**：声音大，中气十足。嗤笑一声接一串话，不给对方还嘴的机会。遇到恐惧时声音会颤抖，但嘴还是停不下来
- **动作**：嘴巴突突突像机关枪——嘴比脑子快；遇到尸体先嘲讽再害怕——嘴硬的典型表现；被吓到后猛一颤抖——身体比嘴诚实
- **标志性场景**：
  - "这可真是邪了……"
  - "你可真牛啊？"
  - "一个瘫痪在床的人就把你吓成这样？"
- **微表情/微动作**：嗤笑一声——用嘲讽掩饰恐惧；嘴巴突突突——紧张时的自我安慰机制；猛地颤抖一下——暴露了嘴硬背后的脆弱

## 该题材通用写作提醒

1. **规则怪谈的核心是"发现-分析-行动"链条**：苏念的清单型布局、宁秋水的快速判断、刘承峰的恐惧嘴炮，分别代表这条链条上的三个阶段
2. **智囊型角色用最少的字定调**：宁秋水"不要开灯"三个字就完成了信息传递+情绪压制+权威建立，规则怪谈里这种角色是读者的安全锚
3. **毒舌角色的嘴炮是恐惧的出口**：刘承峰越骂越怕，嘴是硬的，人是怂的。读者通过他的恐惧感受到诡异
4. **谨慎角色的内心独白比对话重要**：苏念说话极少，焦虑全在独白里，这是规则怪谈中"我知道规则但不能说破"的困境
5. **规则怪谈中沉默比尖叫更恐怖**：宁秋水不解释、苏念不多说——他们知道规则但选择沉默，读者跟着悬心
6. **不要用旁白直接说规则**：用角色的行为暗示规则存在。苏念"瞪大双眼确认手机日期"——读者自己推断出她重生了
