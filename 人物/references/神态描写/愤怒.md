# 愤怒 · 神态描写

> 愤怒是热的。但最可怕的愤怒看起来是冷的。

---

## 轻度——还在压着

**关掉表情的瞬间：**
> 他没说话。嘴角往下压了一下，下颌骨的轮廓从脸上凸出来了。手指在膝盖上敲了两下，停了。再敲第三下——手指自己收回来了。

**那种笑：**
> 她笑了。嘴角是翘的，眼睛没动——眼珠停在一个位置，一眨不眨。你看不出来她在笑还是下一秒就要把杯子摔过来。"行，你说得对。"

**鼻子出的一声：**
> 他鼻子里出了一口气。嘴是闭着的，鼻翼撑了一下就收回去了。就那一声，比"操你妈"还重。

---

## 中度——压不住了，漏了

**血管在说话：**
> 他太阳穴的青筋跳了一下。不是那种"突突突"连续跳——就一下。然后喉咙里发出一声很短的"嗯"，像吞了什么东西没吞下去。手攥成拳。指节比手背先白了一截。

**咬出来的句子：**
> 她把杯子往桌上一顿——没摔。但那个声音比摔还响。手指离开杯身的时候僵了一下——再多一秒她就真的把它砸墙上了。"你再说一遍。"声音很平，每个字都咬在牙根上。

**吃东西的样子不像在吃：**
> 他把叉子放下。放的动作很正常——太正常了。擦了擦嘴。"所以。"两个字。然后停了。过了大概五秒，他把盘子往旁边推了半寸，站了起来。没有摔东西，没有拍桌子。就站了起来。

---

## 重度——出去了

**临界点之后：**
> 他一把抓住对方领子，把人提起来怼到墙上。没吼。两下呼吸从鼻子里喷出来，又急又烫。眼珠盯着对方，一眨不眨。过了三秒才开口，声音是挤出来的，哑的："你他妈知不知道你干了什么。"

**只用了一个字：**
> 她声音突然拔高了一截——从喉咙最深处刮出来的，像碎玻璃擦过的地方。"滚。"就是这一个字。说完转身走了。没有多一句。步子又快又重，鞋跟砸在地板上，咚咚咚的——那个节奏是踹。

**哭了的愤怒：**
> 他在骂你。越骂越快，声音越来越大。忽然嗓子劈了——没控制住。然后眼眶红了。他自己也不知道是气的还是难过，但停不下来手在抖、声音在抖。"你觉得你做得对？你觉得——"他张着嘴，后面的话没出来。眼珠看了一眼天花板，又看回来了。有水光。

---

## 静态愤怒——这才是最恐怖的那种

**临走前的平静：**
> 她没骂你。没摔东西。把杯子放进水槽，椅子推回桌下，手机装进口袋。一样一样收拾好，动作又慢又稳。收拾完了站在门口看了你一眼。那个眼神里面没有愤怒——空得更彻底。"钥匙我放桌上了。"声音没有任何起伏。门是轻轻带上的。

**笑了但没有笑：**
> 他看着你。什么都不说。脸上什么表情都没有。过了五秒，他笑了——嘴角往上，眼睛死了，眼珠上的光像被什么东西吸走了。"我知道了。"说完就走了。

## 愤怒的递进信号（写作时对照）

| 信号 | 说明 |
|------|------|
| 嘴角往下压 | 最早期——在忍 |
| 鼻子里出气不张嘴 | 压到嗓子眼了 |
| 太阳穴跳一下 | 临界点前 |
| 指节发白 | 手在给自己加压 |
| 东西放下/推走 | 代替打人 |
| 每个字都咬在牙根上 | 咬着牙说的 |
| 嗓子劈了 | 彻底失控的前一秒 |
| 眼眶红了 | 愤怒和别的情绪撞了 |
| 门轻轻带上 | 没声音的把一切毁了 |
