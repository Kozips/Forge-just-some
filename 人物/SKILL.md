# 人物Skills · 角色扮演技能库

> 详细说明见上级 `forge/SKILL.md`。

## 快速使用

1. 确定人物类型（人物类/性格类/职业类）
2. 加载 `references/research/对话习惯/核心/00_核心理论.md`
3. 加载对应的 `[类型]_skill参考.md`
4. 融合输出

### 命名规则

**默认主角名（硬性规则）：**
- **男主默认名：天蕴**
- **女主默认名：邵阳**
- 用户明确提出名字时，以用户为准，覆盖默认名
- 其他配角按下方命名速查走

创建新角色时，不要用"陈默林念苏晴"老三样。命名速查见 [references/素材模版库/命名规则速查.md](references/素材模版库/命名规则速查.md)：
- 性格/命运直给（沈倦、许长生）
- 反差命名（王大锤是冷静杀手、苏小棉是特种兵）
- 文艺名（沈知否、江晚橘）
- 谐音梗（秦狩=禽兽、贾正经=假正经）
- 恐怖专用（张伟、023号、陈忘）
- 造名原则：先有人设再有名字，名字嘴里过三遍

## 多角色对话区分清单

多人同场对话时，逐条检查：

1. **语气基线不同** — 每个角色的默认语调（冷/热/阴阳/平）必须有差异，同场景不能所有人一个调
2. **句式长度分层** — 话痨用长句流水账，话少用一字应答，腹黑用短句反问；同一段对话里句式长短要形成对比
3. **口头禅/语气词不重叠** — A用"切"、B用"嘛"、C用"那个"；禁止两人共用同一个标志性语气词
4. **信息量不对称** — 腹黑角色知道的比说的多，傻白甜说的比知道的多；不同角色对同一事件的反应要体现认知差
5. **遮名测试** — 遮住所有角色名，仅看对话内容能否区分谁在说话；分不出来就强化差异

## 角色规则 vs 题材规则冲突处理

```
优先级（从高到低）：
  1. 核心红线（不碰未成年人等）         ← 绝对不可覆盖
  2. 人物底线（"绝对不能做的事"字段）    ← 不可被剧情覆盖
  3. 题材硬规则（`写作/类型/核心规则/`） ← 可在人物允许范围内灵活
  4. 剧情需要                          ← 需充分铺垫才能让人物偏离日常
```

常见冲突场景：

| 冲突 | 处理方式 |
|------|---------|
| 角色设定"绝不撒谎" + 题材需要卧底 | 改标签或加"被逼到绝境"的铺垫，不能直接跳过 |
| 角色社恐 + 题材需要大量社交场景 | 社恐本身是戏剧性来源，写焦虑而非跳过 |
| 角色底线与关键剧情矛盾 | 剧情让步，或设计角色主动突破底线的弧线转折（需高代价） |

## ⚠️ 建角前必查兼容矩阵（硬性规则）

> 每次创建新角色，标签组合必须过兼容矩阵检查。
> 不过=角色会在写作时产生OOC冲突，读者会感到"这个人设对不上"。

**检查步骤**：
1. 选完人物类+性格类+职业类标签后
2.查 `人物/兼容矩阵/人物兼容矩阵.md`（人物类冲突）
3. 查 `人物/兼容矩阵/性格兼容矩阵.md`（性格类冲突）
4. 查 `人物/兼容矩阵/职业兼容矩阵.md`（职业类冲突）
5. 有冲突 → 调整标签或加"为什么会这样"的铺垫
6. 无冲突 → 继续建档

**典型冲突示例**：
- 病娇 + 纯善 → 冲突（病娇的本质是"控制+偏执"，纯善人设对不上）
- 反社会 + 圣母 → 冲突（反社会的本质是"无共情"，圣母人设对不上）
- 社恐 + 表演型 → 冲突（社恐回避关注，表演型渴望关注，互相矛盾）

> 完整冲突表见 `人物/兼容矩阵/` 三个文件。

## 快速建角清单（10项）

> 精简自 `人物/references/素材模版库/角色生成模板.md`，创建角色时逐项过：

1. **基础档案** — 名字 + 年龄 + 外貌关键词（3-5个）+ 第一印象
2. **标签组合** — 人物类 + 性格类 + 职业类（各1个，共2-3个）→**⚠️ 必查兼容矩阵**
3. **核心欲望与恐惧** — 各一句话，二者之间的张力 = 角色动力
4. **认知差距** — 自我认知 vs 他人认知的差异 = 戏剧性来源
5. **对话辨识度** — 语气基调 + 口头禅 + 用词特点，遮名后能认出吗？
6. **感官标志** — 至少3个感官（视觉/听觉/嗅觉/触觉/行为）
7. **关系网络** — 与至少1个角色有明确张力（好感度 + 矛盾点）
8. **人物弧线** — 起点 → 转折事件 → 终点（成长/堕落/平弧/伪弧四选一）
9. **能力与缺陷** — 核心能力 + 致命缺陷 + 绝对底线
10. **蒸馏速查卡** — 一行概括：名字|年龄|标签 → 欲望→恐惧 → 说话方式 → 标志

## 人物-写作联动速查（5条规则）

> 精简自 `人物/写作联动规则.md`，写作过程中随时对照：

1. **先人后剧情** — 加载顺序：角色档案 → 速查表 → 兼容矩阵 → 题材规则 → 开始写
2. **性格决定走向** — 病娇偏执→占有冲突；社恐回避→被迫突破；腹黑表演→操控翻车；标签即剧情引擎
3. **事件反哺档案** — 每次物理伤害/心理创伤/亲密接触/重大冲突/关键选择后，更新角色档案
4. **关系值量化** — 生死与共+10 / 帮助保护+5 / 坦诚秘密+3 / 日常善意+1 / 小摩擦-1 / 背叛-5 / 不可原谅-10
5. **写后联动检查** — 本章标签是否体现？关系值是否更新？有无违背标签的行为（需铺垫）？档案是否需要更新？
6. **禁用"不是A，是B"** — 叙述/描写硬禁用（对话中角色可以说，那是人话）。替代技法见 `写作/写作流程/对话/对比与否定写法.md`
