### 命名速查（拒绝"陈默林念苏晴"老三样）

> 好名字自带信息量——读者看到名字就想往下读。

#### 1. 名字暗示性格/命运

| 类型 | 命名逻辑 | 示例 |
|------|---------|------|
| **性格直给** | 名字就是性格标签 | 沈倦（懒散）、顾慎（谨慎）、叶躁（暴躁）、温吞（慢性子） |
| **命运预言** | 名字暗示结局或宿命 | 许长生（偏偏短命）、余欢（余生皆欢？）、程寂（注定孤独）、黎未（黎明未至） |
| **职业暗示** | 名字带职业气息 | 白鸽（信使/间谍）、铁柱（建筑工）、苏针（医生）、柳烟（青楼/舞者） |

#### 2. 名字与性格反差（最有张力）

| 类型 | 命名逻辑 | 示例 |
|------|---------|------|
| **搞笑名+严肃人** | 名字让人笑，行为让人怕 | 王大锤（但他是冷静的杀手）、李铁蛋（但他是温柔的心理医生）、赵翠花（但她是冷面女总裁） |
| **温柔名+狠人** | 名字很软，人很硬 | 苏小棉（但她是特种兵）、林甜甜（但她是毒枭）、温如玉（但她是格斗冠军） |
| **高贵名+底层人** | 名字很贵，人很穷 | 慕容雪（但她是外卖员）、上官月（但她是清洁工）、欧阳星（但他是工地搬砖的） |
| **普通名+不普通人** | 名字平平无奇，人是怪物 | 张伟（但他是天才黑客）、刘洋（但她是通灵者）、王芳（但她是前特种兵） |

#### 3. 文艺名（适合情感/悬疑/古风）

| 类型 | 命名逻辑 | 示例 |
|------|---------|------|
| **诗词化用** | 从古诗词里抠字 | 沈知否（知否知否）、顾清欢（人间有味是清欢）、林惊鸿（翩若惊鸿） |
| **意象叠加** | 两个意象拼在一起 | 江晚橘（江边晚霞+橘子）、苏薄荷（清凉+刺人）、叶知秋（一叶知秋） |
| **留白感** | 名字只有半个故事 | 许半（只有一半）、程未（未来未至）、余白（空白的余生） |

#### 4. 网文爽感名（适合玄幻/都市/龙傲天）

| 类型 | 命名逻辑 | 示例 |
|------|---------|------|
| **单字霸气** | 姓+一个字，干脆利落 | 叶凡、萧炎、林动、唐三 |
| **复姓加成** | 复姓自带高级感 | 慕容、上官、欧阳、司马、独孤 |
| **谐音梗** | 名字有隐藏含义，读出来才懂 | 见下方谐音名速查表 |

**谐音名速查表**（读出来才懂的名字）：

| 名字 | 谐音 | 适合角色 |
|------|------|---------|
| 秦狩 | 禽兽 | 表面斯文实际狠辣的角色 |
| 贾正经 | 假正经 | 一本正经不干正事的人 |

| 史珍香 | 屎真香 | 搞笑配角/沙雕文主角 |

**使用注意**：
- 谐音名适合**搞笑文、沙雕文、轻松向**作品
- 正剧/恐怖/情感类里用谐音名会直接出戏
- 谐音名做**配角/龙套**效果最好，做主角容易腻
- 谐音一旦被读者发现，会产生"会心一笑"的效果——这是加分项，但别指望藏太久

#### 5. 规则怪谈/恐怖专用名

| 类型 | 命名逻辑 | 示例 |
|------|---------|------|
| **无名感** | 名字越普通越恐怖 | 张伟、李娜、王磊——在恐怖环境里，普通名字反而让人不安 |
| **数字/符号** | 去人格化 | 023号、实验体A、第七位 |
| **消失感** | 名字暗示"不存在" | 陈忘（被遗忘的）、林无（不存在的人）、何在（在哪里？） |

#### 6. 造名原则（别只会从表里挑）

> 上面的示例是**思路参考**，不是固定菜单。每次都从表里挑，名字会撞车。

**造名的核心逻辑**：先有人设，再有名字。名字是人设的投影，不是随机抽的。

| 造名方法 | 怎么操作 | 示例 |
|---------|---------|------|
| **拆人设关键词** | 把角色的核心特质拆成两个字，找谐音或近义字当名字 | 角色特质"沉默+执着"→ 沉执 → 陈执 |
| **反着来** | 人设的反义词当名字，制造反差 | 角色是冷血杀手→ 名字叫温暖/暖暖/阳阳 |
| **从职业/背景里抠字** | 角色的出身、职业、地域特征融入名字 | 海边长大的渔民女儿→ 潮生/汐落/渔歌 |
| **从命运里抠字** | 角色的结局或宿命藏在名字里 | 角色最终会牺牲→ 名字叫余生/长留/未归 |
| **拼两个无关的字** | 两个看似不搭的字拼在一起，产生陌生感 | 漆白、盐野、铁蕊、冰燃 |
| **用生僻姓** | 换个不常见的姓，普通名也变特别 | 张伟 → 褚伟、靳伟、雒伟 |
| **加一个意外的字** | 在常见名字里塞一个不常见的字 | 林小棉 → 林小锥、林小铁、林小灰 |

**最重要的一条**：名字要**嘴里过三遍**。念出来顺不顺？放在对话里自然不自然？"陈默你过来"和"秦狩你过来"，节奏完全不同。

#### 7. 命名禁忌

- **不要全篇都是ABB式**：林念念、苏甜甜、陈默默——三个以上读者脸盲
- **不要所有角色同风格**：全古风或全现代——混搭才有层次
- **不要名字比人精彩**：名字太花哨但角色立不住，反成笑柄
- **不要谐音梗用在正剧里**：严肃题材里出现"史珍香"直接出戏

## 人物类型标签

从 `人物/SKILL.md` 分类中选取（可组合，建议2-3个）：

```
【人物类标签】：（如：病娇/傲娇/社恐/恋爱脑/腹黑）
【性格类标签】：（如：偏执型/回避型/表演型/反社会型）
【职业类标签】：（如：医生/特警/纹身师/大学生/自由职业）
```

**标签组合公式**：`人物类（核心驱动）+ 性格类（行为模式）+ 职业类（社会身份）`

示例：`病娇 + 偏执型 + 大学生` = 一个偏执的大学生病娇角色

## 心理内核

```
【核心欲望】：（他/她最想要什么？一句话）
【核心恐惧】：（他/她最怕什么？一句话）
【自我认知】：（他/她觉得自己是什么样的人？）
【他人认知】：（别人觉得他/她是什么样的人？）
【认知差距】：（自我认知和他人认知的差距=戏剧性来源）
```

## 人物弧线预设

```
【起点状态】：（出场时是什么样的人）
【转折事件】：（什么事件会改变他/她）
【终点状态】：（故事结束时变成了什么样的人）
【弧线类型】：
  □ 成长弧（正向变化）
  □ 堕落弧（负向变化）
  □ 平弧（不变，但改变周围的人）
  □ 伪弧（表面变化，内核不变）
```

## 对话习惯

参照 `人物/SKILL.md` 中的对话习惯模块：

```
【语气基调】：（如：冷淡/热情/阴阳怪气/温柔/暴躁）
【口头禅】：（1-2个，如："随便""你开心就好""切"）
【用词特点】：（如：爱用网络梗/爱用成语/爱用脏话/爱用方言）
【说话节奏】：（如：长句多/短句多/经常半句话不说完/爱插嘴）
【特殊习惯】：（如：紧张时结巴/撒谎时摸鼻子/生气时笑）
【禁忌话题】：（听到什么会炸/会沉默/会转移话题）
```

**辨识度测试**：遮住名字后，仅看对话能认出是他/她吗？如果不能，强化差异。
